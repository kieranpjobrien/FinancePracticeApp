# Question

The short-run aggregate supply curve is upward sloping because:

## Options

- **A**: All input prices adjust immediately to output price changes
- **B**: Some input prices are sticky, so higher output prices increase profit margins
- **C**: The economy always operates at full capacity

## Answer

B

## Explanation

In the short run, some input prices (especially wages) are sticky due to contracts, money illusion, or adjustment lags. When output prices rise, firms' profit margins expand because costs don't rise proportionally, incentivizing higher production. In the long run, input prices fully adjust, and the aggregate supply curve becomes vertical at potential output.

**Common mistakes:**
- Option A describes long-run conditions where SRAS becomes vertical
- Option C ignores that output can vary around potential in the short run
