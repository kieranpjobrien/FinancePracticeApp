# Question

Fixed costs differ from variable costs in that fixed costs:

## Options

- **A**: Change with the level of output
- **B**: Remain constant regardless of output level in the short run
- **C**: Only exist in the long run

## Answer

B

## Explanation

Fixed costs (rent, salaries of permanent staff, depreciation) don't change with output in the short run—they're incurred even at zero production. Variable costs (raw materials, hourly wages, utilities) change with output. In the long run, all costs become variable as firms can adjust all inputs. The distinction is crucial for shutdown decisions.

**Common mistakes:**
- Option A describes variable costs
- Option C reverses the relationship—all costs are variable in the long run
