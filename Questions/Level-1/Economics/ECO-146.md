# Question

If EUR/USD moves from 1.10 to 1.00, the euro has:

## Options

- **A**: Appreciated by approximately 10%
- **B**: Depreciated by approximately 9.1%
- **C**: Remained unchanged

## Answer

B

## Explanation

When EUR/USD falls from 1.10 to 1.00, fewer dollars are needed to buy a euro, so the euro has depreciated (weakened).

Percentage change = (1.00 - 1.10) / 1.10 = -0.0909 = -9.1%

The euro depreciated by approximately 9.1% against the dollar.

**Common mistakes:**
- Option A reverses both direction and calculation
- Option C ignores the 10-cent change
