# Question

Non-fungible tokens (NFTs) differ from cryptocurrencies like Bitcoin because:

## Options

- **A**: NFTs have no value whatsoever
- **B**: Each NFT is unique and not interchangeable, while cryptocurrencies are fungible
- **C**: NFTs are only used for payments

## Answer

B

## Explanation

NFT characteristics:

**Fungibility explained:**
- Fungible: One unit equals another
- Bitcoin: Each BTC identical
- NFT: Each token unique

**NFT features:**
- Unique identifier
- Provable ownership
- Scarcity verified
- Metadata attached

**Use cases:**
- Digital art
- Collectibles
- Gaming items
- Music/media rights
- Real-world asset tokenization

**How NFTs work:**
- Minted on blockchain
- Ownership recorded
- Transferable/tradeable
- Royalties programmable

**Investment considerations:**
- Highly speculative
- Illiquid markets
- Valuation challenges
- Copyright vs ownership

**Common mistakes:**
- Option A: NFTs can have significant value
- Option C: Used for ownership, not payments
