# Question

Survivorship bias in alternative investment databases causes:

## Options

- **A**: Reported average returns to be understated
- **B**: Reported average returns to be overstated
- **C**: No impact on reported performance statistics

## Answer

B

## Explanation

Survivorship bias explained:

**What happens:**
- Failed funds disappear from databases
- Only successful funds remain
- Average calculated from survivors only
- Performance appears better than reality

**Magnitude:**
- Hedge funds: 2-4% annual overstatement
- Private equity: Similar issues
- Significant for investment decisions

**Example:**
- 100 funds start with $100M each
- 20 fail and close (poor returns)
- 80 survive with good returns
- Database shows only 80 survivors
- Average return inflated

**Other biases in alternatives:**
- Backfill bias: Adding historical returns
- Selection bias: Voluntary reporting
- Stale pricing: Smoothed returns

**Common mistakes:**
- Option A: Bias overstates, not understates
- Option C: Bias significantly impacts statistics
