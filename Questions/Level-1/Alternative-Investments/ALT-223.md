# Question

Non-recourse debt in real estate means the lender:

## Options

- **A**: Can pursue all of the borrower's assets if the loan defaults
- **B**: Can only recover from the specific property securing the loan
- **C**: Has no security interest in any collateral

## Answer

B

## Explanation

Non-recourse debt:

**Definition:**
- Claim limited to collateral
- Property is sole security
- No personal guarantee
- Borrower protected

**How it works:**
- Borrower defaults
- Lender takes property
- Cannot pursue other assets
- Loss limited to property value

**Carve-outs:**
- "Bad boy" guarantees
- Fraud exclusions
- Environmental issues
- Misrepresentation

**Recourse comparison:**
- Recourse: Personal guarantee
- Full borrower liability
- Other assets at risk
- More borrower risk

**Why non-recourse common:**
- Property-specific risk
- Asset-backed lending
- Borrower protection
- Industry standard (institutional)

**Common mistakes:**
- Option A: This describes recourse debt
- Option C: Property is collateral
