# Question

Public Market Equivalent (PME) analysis compares private equity returns to:

## Options

- **A**: Other private equity funds of the same vintage
- **B**: What would have been earned investing the same cash flows in public markets
- **C**: The risk-free rate over the investment period

## Answer

B

## Explanation

PME methodology:

**Concept:**
- Hypothetical public market investment
- Uses actual PE cash flow timing
- Capital calls invested in index
- Distributions withdrawn from index

**Calculation:**
- Each capital call: Buy index
- Each distribution: Sell index
- Compare ending values
- PME > 1: PE outperformed

**Why useful:**
- Addresses timing differences
- Apples-to-apples comparison
- Accounts for opportunity cost
- Market conditions reflected

**Example:**
- PE fund returns 2.0x
- PME analysis: Public would have returned 1.6x
- PE outperformed by 0.4x

**Variations:**
- Kaplan-Schoar PME
- Long-Nickels PME+
- Direct Alpha

**Common mistakes:**
- Option A: This is peer comparison, not PME
- Option C: PME uses market returns, not risk-free
