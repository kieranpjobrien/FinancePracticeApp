# Question

The preferred return (hurdle) in private equity fund terms:

## Options

- **A**: Guarantees a minimum return to investors
- **B**: Is the return LPs receive before the GP earns carried interest
- **C**: Is paid only to the general partner

## Answer

B

## Explanation

Preferred return explained:

**Definition:**
- Minimum return to LPs first
- Before GP receives carry
- Typically 8% compounded
- Priority in distribution

**Purpose:**
- Protect LP interests
- Ensure alignment
- Base return hurdle
- Industry standard

**How it works:**
- Fund generates returns
- First 8% goes to LPs
- Then catch-up or splits begin
- GP incentivized above hurdle

**Not a guarantee:**
- Not promised return
- Just priority in waterfall
- If fund loses, no pref paid
- Performance dependent

**Common structures:**
- 8% compounded annually
- 7-10% range
- May be simple or compound
- Varies by fund terms

**Common mistakes:**
- Option A: Priority, not guarantee
- Option C: Goes to LPs, not GP
