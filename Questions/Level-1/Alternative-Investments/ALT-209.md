# Question

A rent roll in real estate provides:

## Options

- **A**: Only the total annual rent for a property
- **B**: Detailed information about each tenant, lease terms, and rental rates
- **C**: Architectural drawings of the building

## Answer

B

## Explanation

Rent roll contents:

**Key information:**
- Tenant names
- Unit/space identification
- Square footage
- Rent per square foot
- Lease start and end dates
- Rent escalations

**Additional details:**
- Security deposits
- Options (renewal, expansion)
- Expense reimbursements
- Rent concessions

**Why important:**
- Income verification
- Tenant quality assessment
- Lease expiration analysis
- Rollover risk evaluation

**Due diligence use:**
- Verify reported income
- Analyze tenant concentration
- Project future cash flows
- Identify risks

**Analysis enabled:**
- Weighted average lease term
- Tenant diversification
- Expiration schedule
- Mark-to-market potential

**Common mistakes:**
- Option A: Detailed tenant-level data
- Option C: Financial/lease document, not architectural
