# Question

In real estate investing, "location, location, location" emphasizes that:

## Options

- **A**: All locations offer identical investment returns
- **B**: Geographic factors significantly impact property value and performance
- **C**: Location is irrelevant for commercial properties

## Answer

B

## Explanation

Location importance in real estate:

**Why location matters:**
- Cannot be changed
- Supply constrained in prime areas
- Demand driven by access
- Long-term value driver

**Location factors:**
- Employment centers
- Transportation access
- Demographics
- Quality of life
- Regulatory environment

**Micro-location:**
- Street presence
- Visibility
- Parking
- Neighboring uses
- Building position

**Macro-location:**
- Metropolitan area
- Submarket
- Economic drivers
- Growth trends

**Property type considerations:**
- Retail: Traffic, visibility
- Office: Labor access, amenities
- Industrial: Transportation, labor
- Multifamily: Schools, amenities

**Common mistakes:**
- Option A: Returns vary dramatically by location
- Option C: Location critical for all property types
