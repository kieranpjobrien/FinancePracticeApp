# Question

Layer 2 scaling solutions for blockchain:

## Options

- **A**: Replace the underlying blockchain entirely
- **B**: Process transactions off the main chain to increase throughput and reduce costs
- **C**: Only apply to Bitcoin and no other blockchains

## Answer

B

## Explanation

Layer 2 solutions:

**Purpose:**
- Increase transaction speed
- Reduce fees
- Scale without changing Layer 1
- Maintain security

**How they work:**
- Bundle transactions off-chain
- Settle periodically on main chain
- Inherit security from Layer 1
- Process many transactions cheaply

**Examples:**
- Lightning Network (Bitcoin)
- Optimistic rollups (Ethereum)
- ZK-rollups (Ethereum)
- Sidechains

**Benefits:**
- Thousands of TPS possible
- Lower gas fees
- Better user experience
- Enable new use cases

**Trade-offs:**
- Some trust assumptions
- Complexity
- Fragmented liquidity
- User experience challenges

**Investment relevance:**
- Layer 2 tokens tradeable
- Infrastructure investment theme

**Common mistakes:**
- Option A: Complements, not replaces Layer 1
- Option C: Multiple blockchain ecosystems use L2
