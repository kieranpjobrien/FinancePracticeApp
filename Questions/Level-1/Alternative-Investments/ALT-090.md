# Question

Activist hedge fund strategies involve:

## Options

- **A**: Passive index investing with no company engagement
- **B**: Taking significant stakes and pushing for changes to unlock shareholder value
- **C**: Only investing in companies with perfect governance

## Answer

B

## Explanation

Activist investing:

**Approach:**
- Build significant stake (5-10%+)
- Engage with management/board
- Push for strategic changes
- Unlock hidden value

**Common activist demands:**
- Board seats
- Management changes
- Strategic alternatives (sale, spinoff)
- Capital allocation changes
- Cost cutting

**Value creation levers:**
- Operational improvements
- Capital structure optimization
- M&A activity
- Governance improvements

**Process:**
- Accumulate position (13D filing)
- Engage privately first
- Public campaign if needed
- Proxy contest possible

**Notable activists:**
- Carl Icahn
- Bill Ackman
- Elliott Management
- ValueAct

**Common mistakes:**
- Option A: Very active, not passive
- Option C: Target underperforming companies
