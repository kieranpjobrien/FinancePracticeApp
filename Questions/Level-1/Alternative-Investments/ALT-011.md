# Question

The J-curve effect in private equity refers to:

## Options

- **A**: The tendency for early returns to be negative before turning positive
- **B**: The shape of the fee structure over time
- **C**: The correlation pattern with public markets

## Answer

A

## Explanation

J-curve effect explained:

**Pattern:**
- Early years: Negative returns
- Middle years: Returns improve
- Later years: Positive cumulative returns
- Graph looks like letter "J"

**Why negative early returns:**
- Management fees paid immediately
- Investments made at cost
- No realized gains yet
- Fund expenses reduce NAV

**Why returns improve:**
- Portfolio companies grow
- Value creation realized
- Exits begin (IPOs, sales)
- Gains offset early losses

**Timeline (typical 10-year fund):**
- Years 1-3: J-curve trough
- Years 4-6: Value creation
- Years 7-10: Harvesting period

**Implications:**
- Patience required
- Commitment period planning
- Benchmark timing issues

**Common mistakes:**
- Option B: Fees don't follow J-curve
- Option C: Correlation pattern is separate concept
