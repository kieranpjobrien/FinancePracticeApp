# Question

Total Value to Paid-In (TVPI) multiple measures:

## Options

- **A**: Only realized returns from distributions
- **B**: Total value (realized + unrealized) relative to capital invested
- **C**: The fund's annual return percentage

## Answer

B

## Explanation

TVPI multiple explained:

**Formula:**
TVPI = (Distributions + NAV) / Paid-In Capital

**Components:**
- Distributions: Cash returned to LPs
- NAV: Remaining value in fund
- Paid-In: Capital actually invested

**Interpretation:**
- TVPI of 1.0x: Break-even
- TVPI of 2.0x: Doubled money
- TVPI of 0.8x: Lost 20%

**Related multiples:**
- DPI (Distributions/Paid-In): Realized only
- RVPI (Residual Value/Paid-In): Unrealized only
- TVPI = DPI + RVPI

**Advantages:**
- Simple to understand
- Intuitive wealth measure
- Comparable across funds

**Limitations:**
- Ignores time value
- Residual value subjective
- Doesn't capture cash flow timing

**Common mistakes:**
- Option A: This describes DPI, not TVPI
- Option C: Multiples aren't annual percentages
