# Question

Same-store NOI growth in real estate measures:

## Options

- **A**: Growth from acquiring new properties
- **B**: Organic growth in income from properties owned throughout both periods
- **C**: Changes in property values

## Answer

B

## Explanation

Same-store NOI growth:

**Definition:**
- Comparable property performance
- Properties owned in both periods
- Excludes acquisitions/dispositions
- Organic performance measure

**What it captures:**
- Rental rate growth
- Occupancy changes
- Expense management
- Operational improvements

**Calculation:**
- Same properties in both periods
- Compare NOI year-over-year
- Typically same quarter prior year

**Why important:**
- True operational performance
- Not influenced by transactions
- Management effectiveness
- Market conditions impact

**REIT reporting:**
- Common disclosure metric
- Investor expectations
- Trend analysis
- Peer comparison

**Common mistakes:**
- Option A: Excludes new properties
- Option C: Income metric, not values
