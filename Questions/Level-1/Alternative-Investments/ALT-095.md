# Question

In hedge fund analysis, "alpha" refers to:

## Options

- **A**: Returns attributable to market movements
- **B**: Returns from manager skill after accounting for market exposure
- **C**: The fund's total return

## Answer

B

## Explanation

Alpha explained:

**Definition:**
- Excess return above benchmark
- After adjusting for risk/beta
- Represents manager skill
- Not market exposure

**Alpha vs beta:**
- Beta: Market-related return
- Alpha: Skill-related return
- Total return = Alpha + Beta return

**Why alpha matters:**
- What you pay fees for
- Scarce and valuable
- Justifies hedge fund fees
- Not available from index

**Measurement challenges:**
- Which benchmark to use?
- Multiple factor exposures
- Changing beta over time
- Style drift

**Alpha generation:**
- Security selection
- Market timing
- Strategy expertise
- Information advantage

**Common mistakes:**
- Option A: This describes beta
- Option C: Total return includes both alpha and beta
