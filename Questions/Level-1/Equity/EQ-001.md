# Question

A stock just paid a dividend of $2.00 per share. Dividends are expected to grow at a constant rate of 5% per year. If the required return is 12%, the intrinsic value of the stock using the Gordon Growth Model is closest to:

## Options

- **A**: $28.57
- **B**: $30.00
- **C**: $33.33

## Answer

B

## Explanation

The Gordon Growth Model (constant growth DDM) formula is:
V₀ = D₁ / (r - g)

Where:
- D₁ = Next year's dividend = D₀ × (1 + g) = $2.00 × 1.05 = $2.10
- r = Required return = 12% = 0.12
- g = Growth rate = 5% = 0.05

V₀ = $2.10 / (0.12 - 0.05)
V₀ = $2.10 / 0.07
V₀ = $30.00

**Common mistakes:**
- Option A uses D₀ ($2.00) instead of D₁ ($2.10): $2.00 / 0.07 = $28.57
- Option C uses only the required return as denominator: $2.00 / 0.06 = $33.33
