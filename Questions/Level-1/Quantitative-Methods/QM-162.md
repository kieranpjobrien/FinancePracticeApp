# Question

The efficient frontier represents:

## Options

- **A**: All possible portfolios that can be constructed from available assets
- **B**: Portfolios offering the highest return for each level of risk
- **C**: The portfolio with the absolute highest return

## Answer

B

## Explanation

The efficient frontier is the set of optimal portfolios that offer the highest expected return for each level of risk (standard deviation). Portfolios on the efficient frontier dominate those below it—for any given risk level, no portfolio offers higher expected return. Rational investors should hold portfolios on the efficient frontier.

**Common mistakes:**
- Option A describes the feasible set, which includes all possible portfolios including inefficient ones
- Option C describes only one point; the efficient frontier is a curve of many optimal portfolios
