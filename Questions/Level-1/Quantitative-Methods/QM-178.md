# Question

M-squared (M²) is a risk-adjusted performance measure that:

## Options

- **A**: Uses beta as the measure of risk
- **B**: Expresses performance in the same units as the benchmark return
- **C**: Can only be calculated for individual securities

## Answer

B

## Explanation

M-squared adjusts portfolio risk to match the benchmark's risk, then compares returns. It scales the portfolio to have the same standard deviation as the market, providing a direct return comparison. M² is expressed in percentage return units, making it easier to interpret than the Sharpe ratio:

M² = (Sharpe ratio of portfolio × σmarket) + Rf

A positive M² means the portfolio outperformed on a risk-adjusted basis.

**Common mistakes:**
- Option A describes the Treynor ratio, which uses beta
- Option C is incorrect; M² is used for portfolio evaluation
