# Question

Bootstrap resampling involves:

## Options

- **A**: Collecting new data from the population
- **B**: Drawing repeated samples with replacement from the existing data
- **C**: Removing outliers from the dataset before analysis

## Answer

B

## Explanation

Bootstrap resampling is a technique that treats the observed sample as if it were the population. Multiple samples are drawn randomly with replacement from the original sample, each the same size as the original. Statistics are calculated for each bootstrap sample, allowing estimation of the sampling distribution and confidence intervals without relying on theoretical assumptions about the population distribution.

**Common mistakes:**
- Option A describes traditional sampling; bootstrap specifically uses only existing data
- Option C describes a data cleaning technique, not resampling
