# Question

In a confusion matrix, precision is calculated as:

## Options

- **A**: True Positives / (True Positives + False Negatives)
- **B**: True Positives / (True Positives + False Positives)
- **C**: (True Positives + True Negatives) / Total Observations

## Answer

B

## Explanation

Precision measures how many of the positive predictions were actually correct:

Precision = TP / (TP + FP)

This is important when false positives are costly. For example, in fraud detection, precision tells us what proportion of flagged transactions are actually fraudulent.

Recall (Option A) = TP / (TP + FN) - measures what proportion of actual positives were identified
Accuracy (Option C) = (TP + TN) / Total

**Common mistakes:**
- Option A defines recall (sensitivity), not precision
- Option C defines overall accuracy, not precision
