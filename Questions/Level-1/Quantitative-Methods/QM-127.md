# Question

A 180-day investment has a holding period yield of 3%. The bond-equivalent yield (annualized using a 365-day year with simple interest) is closest to:

## Options

- **A**: 6.00%
- **B**: 6.08%
- **C**: 6.14%

## Answer

B

## Explanation

Bond-equivalent yield annualizes the holding period return using simple interest and a 365-day year:

BEY = HPY × (365 / Days held)
BEY = 3% × (365 / 180)
BEY = 3% × 2.0278
BEY = 6.08%

**Common mistakes:**
- Option A uses a 360-day year: 3% × (360/180) = 6%
- Option C may result from compounding instead of simple interest annualization
