# Question

If the sample size is quadrupled while the population standard deviation remains constant, the standard error of the mean will:

## Options

- **A**: Decrease by half
- **B**: Decrease by three-quarters
- **C**: Remain unchanged

## Answer

A

## Explanation

Standard error = σ/√n

If sample size increases from n to 4n:
New standard error = σ/√(4n) = σ/(2√n) = (1/2) × (σ/√n)

The standard error decreases by half when the sample size quadruples. This demonstrates the diminishing returns to increasing sample size—to halve the standard error, the sample size must be quadrupled.

**Common mistakes:**
- Option B would require a 16-fold increase in sample size, not a quadrupling
- Option C ignores the inverse relationship between standard error and the square root of sample size
