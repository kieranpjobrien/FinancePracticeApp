# Question

Monte Carlo simulation is most useful for:

## Options

- **A**: Calculating exact analytical solutions to complex problems
- **B**: Generating multiple scenarios based on random sampling from probability distributions
- **C**: Eliminating all uncertainty from investment analysis

## Answer

B

## Explanation

Monte Carlo simulation uses random sampling from specified probability distributions to generate a large number of possible outcomes. This allows analysts to model complex situations where analytical solutions are impractical, understand the range of possible outcomes, and assess the probability of different results. It is particularly valuable for option pricing, risk analysis, and portfolio stress testing.

**Common mistakes:**
- Option A is incorrect because Monte Carlo provides approximate solutions through simulation, not exact analytical solutions
- Option C is incorrect because simulation quantifies and illustrates uncertainty rather than eliminating it
