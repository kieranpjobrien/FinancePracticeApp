# Question

A regression analysis yields a slope coefficient of 1.5 with a standard error of 0.5. To test whether the slope is significantly different from zero at the 5% level (critical t-value ≈ 2.0), the analyst should conclude that:

## Options

- **A**: The slope is statistically significant because the t-statistic exceeds the critical value
- **B**: The slope is not statistically significant because the t-statistic is below the critical value
- **C**: Significance cannot be determined without the R-squared value

## Answer

A

## Explanation

The t-statistic for testing whether the slope differs from zero is:

t = (b - 0) / SE(b) = 1.5 / 0.5 = 3.0

Since 3.0 > 2.0 (the critical value), we reject the null hypothesis that the slope equals zero. The slope is statistically significant at the 5% level, indicating a meaningful linear relationship between X and Y.

**Common mistakes:**
- Option B incorrectly states the comparison; 3.0 exceeds 2.0
- Option C is incorrect because significance testing for the slope coefficient requires only the coefficient estimate and its standard error, not R-squared
