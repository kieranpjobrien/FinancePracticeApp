# Question

Scenario analysis involves:

## Options

- **A**: Randomly sampling from probability distributions thousands of times
- **B**: Defining specific sets of input values to examine possible outcomes
- **C**: Finding the input values that produce a target output

## Answer

B

## Explanation

Scenario analysis examines what happens to outputs when inputs are changed to represent specific alternative situations. Common scenarios include base case, best case, and worst case, or economic scenarios like recession, normal growth, and boom. Unlike Monte Carlo simulation, scenario analysis uses discrete, predefined scenarios rather than random sampling.

**Common mistakes:**
- Option A describes Monte Carlo simulation
- Option C describes goal seek or optimization analysis
