# Question

An investment earns a nominal return of 8% during a year when inflation is 3%. The approximate real return is closest to:

## Options

- **A**: 4.9%
- **B**: 5.0%
- **C**: 11.0%

## Answer

B

## Explanation

Approximate Real Return ≈ Nominal Return - Inflation Rate
Approximate Real Return ≈ 8% - 3% = 5%

The exact calculation using the Fisher equation would be:
(1 + Real) = (1 + Nominal) / (1 + Inflation)
(1 + Real) = 1.08 / 1.03 = 1.0485
Real Return = 4.85%

Both 4.9% and 5.0% are close; the approximation of 5% is acceptable.

**Common mistakes:**
- Option C adds the rates instead of subtracting, which is conceptually wrong
