# Question

A stock is expected to pay a dividend of $3 next year. Dividends are expected to grow at 4% per year indefinitely. If the required return is 10%, the present value of all future dividends is closest to:

## Options

- **A**: $30.00
- **B**: $50.00
- **C**: $75.00

## Answer

B

## Explanation

For a growing perpetuity:
PV = D₁ / (r - g)
PV = $3 / (0.10 - 0.04)
PV = $3 / 0.06
PV = $50.00

This is the Gordon Growth Model, which values a stream of cash flows growing at a constant rate forever.

**Common mistakes:**
- Option A uses the required return only: $3 / 0.10 = $30 (ignores growth)
- Option C may result from using the wrong formula or rates
