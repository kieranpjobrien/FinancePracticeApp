# Question

Hyperparameters in machine learning are:

## Options

- **A**: Parameters learned automatically from the training data
- **B**: Settings chosen by the analyst before training that control model complexity
- **C**: The final predictions output by the model

## Answer

B

## Explanation

Hyperparameters are configuration settings specified before training begins—examples include the learning rate, number of trees in a random forest, or regularization strength. Unlike model parameters (like regression coefficients) that are learned from data, hyperparameters must be chosen by the analyst, often through cross-validation or grid search. They control model complexity and learning behavior.

**Common mistakes:**
- Option A describes model parameters, not hyperparameters
- Option C describes model output, not configuration settings
