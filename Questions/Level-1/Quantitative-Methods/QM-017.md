# Question

An analyst estimates three scenarios for next year's stock return: bull market (probability 30%, return 25%), normal market (probability 50%, return 10%), and bear market (probability 20%, return -15%). The expected return is closest to:

## Options

- **A**: 6.7%
- **B**: 9.5%
- **C**: 12.0%

## Answer

B

## Explanation

Expected Return = Σ(Probability × Return)
E(R) = (0.30 × 25%) + (0.50 × 10%) + (0.20 × -15%)
E(R) = 7.5% + 5.0% + (-3.0%)
E(R) = 9.5%

Expected value is the probability-weighted average of all possible outcomes.

**Common mistakes:**
- Option A may result from a calculation error
- Option C might result from incorrect probability weighting
