# Question

Natural language processing (NLP) in finance is most commonly used for:

## Options

- **A**: Predicting exact stock prices based on company name length
- **B**: Extracting sentiment and information from text documents like news and filings
- **C**: Replacing all quantitative analysis with word-based models

## Answer

B

## Explanation

NLP enables analysis of unstructured text data such as news articles, social media, earnings call transcripts, and regulatory filings. Applications include sentiment analysis (determining if text is positive/negative), topic extraction, and identifying material information. NLP complements traditional quantitative analysis by adding an information source previously difficult to systematically analyze.

**Common mistakes:**
- Option A is not a realistic NLP application
- Option C overstates NLP's role; it supplements rather than replaces quantitative analysis
