# Question

An investor buys $100,000 of stock using $60,000 of equity and $40,000 of borrowed funds at 5% interest. If the stock rises 15% over one year, the return on equity is closest to:

## Options

- **A**: 15.0%
- **B**: 21.7%
- **C**: 25.0%

## Answer

B

## Explanation

Gain on stock = $100,000 × 15% = $15,000
Interest cost = $40,000 × 5% = $2,000
Net profit = $15,000 - $2,000 = $13,000

Return on equity = Net profit / Equity invested
Return on equity = $13,000 / $60,000 = 21.67%

Leverage amplifies returns (both up and down) relative to the unlevered return.

**Common mistakes:**
- Option A ignores the leveraging effect and interest cost
- Option C calculates ($15,000/$60,000) without subtracting interest expense
