# Question

A test statistic yields a p-value of 0.03. At a significance level of 5%, the analyst should:

## Options

- **A**: Reject the null hypothesis because the p-value is less than the significance level
- **B**: Fail to reject the null hypothesis because the p-value is greater than zero
- **C**: Suspend judgment until more data is collected

## Answer

A

## Explanation

The p-value represents the probability of obtaining a test statistic at least as extreme as the observed result, assuming the null hypothesis is true. The decision rule is: reject H₀ if p-value ≤ α (significance level). Since 0.03 < 0.05, the null hypothesis is rejected at the 5% significance level. The result is statistically significant—the observed data is unlikely to have occurred if the null hypothesis were true.

**Common mistakes:**
- Option B misunderstands the decision rule; the comparison is between p-value and significance level, not whether p-value is positive
- Option C is not a standard statistical decision; hypothesis testing provides a framework for decision-making based on available evidence
