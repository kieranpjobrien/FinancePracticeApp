# Question

In simple linear regression, the F-test is used to test whether:

## Options

- **A**: The intercept equals zero
- **B**: The slope coefficient equals zero (model has explanatory power)
- **C**: The residuals are normally distributed

## Answer

B

## Explanation

The F-test in regression analysis tests whether the regression model as a whole has explanatory power. In simple linear regression with one independent variable, the F-test is equivalent to testing whether the slope coefficient equals zero (H₀: β = 0). If we reject this null hypothesis, we conclude that the independent variable has a statistically significant relationship with the dependent variable.

**Common mistakes:**
- Option A describes a different hypothesis test; the intercept is typically not the focus of the F-test
- Option C describes a diagnostic check for regression assumptions, not what the F-test evaluates
