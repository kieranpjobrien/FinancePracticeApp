# Question

Compared to an ordinary annuity with the same payment, interest rate, and number of periods, an annuity due has:

## Options

- **A**: A lower present value and a lower future value
- **B**: A higher present value and a higher future value
- **C**: The same present value but a higher future value

## Answer

B

## Explanation

An annuity due has payments at the beginning of each period instead of the end. Since payments are received (or made) earlier, each payment has more time value:

PV annuity due = PV ordinary annuity × (1 + r)
FV annuity due = FV ordinary annuity × (1 + r)

Both values are higher by one period's interest factor because all payments shift one period earlier in time.

**Common mistakes:**
- Option A reverses the relationship
- Option C incorrectly suggests only future value differs
