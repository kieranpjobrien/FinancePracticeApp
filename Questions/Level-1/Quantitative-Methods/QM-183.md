# Question

A parameter is:

## Options

- **A**: A numerical value calculated from sample data
- **B**: A numerical characteristic of a population
- **C**: An estimate that varies from sample to sample

## Answer

B

## Explanation

A parameter is a fixed numerical characteristic of a population (e.g., population mean μ, population standard deviation σ). Parameters are typically unknown and are estimated using sample statistics. A statistic is calculated from sample data and is used to estimate the corresponding parameter.

**Common mistakes:**
- Option A describes a statistic, not a parameter
- Option C also describes a statistic, which varies across samples
