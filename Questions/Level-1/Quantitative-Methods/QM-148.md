# Question

Feature engineering in machine learning refers to:

## Options

- **A**: Selecting the best pre-existing features from the dataset
- **B**: Creating new input variables from raw data to improve model performance
- **C**: Removing all outliers from the training data

## Answer

B

## Explanation

Feature engineering is the process of creating new variables (features) from raw data that better capture the underlying patterns relevant to prediction. Examples include creating ratios (P/E ratio from price and earnings), transformations (log returns from prices), interaction terms, or domain-specific metrics. Good feature engineering often has more impact on model performance than the choice of algorithm.

**Common mistakes:**
- Option A describes feature selection, which is related but different
- Option C describes outlier treatment, a separate data preprocessing step
