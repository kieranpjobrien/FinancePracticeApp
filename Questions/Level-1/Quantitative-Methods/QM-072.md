# Question

An observation at the 75th percentile indicates that:

## Options

- **A**: 75% of observations fall below this value
- **B**: 75% of observations fall above this value
- **C**: The observation equals 75% of the maximum value

## Answer

A

## Explanation

A percentile indicates the percentage of observations that fall at or below that value. The 75th percentile (also called the third quartile, Q3) means that 75% of the data falls below this point and 25% falls above it. Percentiles are used to describe the relative position of a data point within a distribution.

**Common mistakes:**
- Option B reverses the interpretation; it describes the 25th percentile, not the 75th
- Option C confuses percentiles with a proportion of the maximum value, which is unrelated
