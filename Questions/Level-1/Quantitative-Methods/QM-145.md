# Question

The Sortino ratio differs from the Sharpe ratio by:

## Options

- **A**: Using geometric mean instead of arithmetic mean return
- **B**: Using downside deviation instead of total standard deviation in the denominator
- **C**: Using the benchmark return instead of the risk-free rate

## Answer

B

## Explanation

The Sortino ratio uses downside deviation (standard deviation of only negative returns relative to a minimum acceptable return) instead of total standard deviation. This addresses a criticism of the Sharpe ratio: that it penalizes upside volatility equally with downside volatility. The Sortino ratio focuses only on harmful volatility, which may be more relevant for investors concerned with downside protection.

Sortino = (Return - MAR) / Downside Deviation

**Common mistakes:**
- Option A describes a different return calculation method
- Option C describes the M-squared measure or other benchmark-relative metrics
