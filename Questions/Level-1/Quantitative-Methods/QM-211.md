# Question

The required rate of return on an investment typically consists of:

## Options

- **A**: Only the risk-free rate
- **B**: The risk-free rate plus a risk premium
- **C**: Only the expected inflation rate

## Answer

B

## Explanation

The required return compensates investors for: (1) the time value of money (real risk-free rate), (2) expected inflation, and (3) risk. This is often expressed as:

Required return = Risk-free rate + Risk premium

Where the risk-free rate itself includes the real rate and expected inflation. Riskier investments require higher risk premiums.

**Common mistakes:**
- Option A ignores the risk premium required for risky investments
- Option C ignores the real return and risk premium components
