# Question

Target semideviation is calculated relative to:

## Options

- **A**: The sample mean return
- **B**: A minimum acceptable return (MAR) specified by the investor
- **C**: The median return

## Answer

B

## Explanation

Target semideviation measures downside risk relative to a specific target or minimum acceptable return (MAR) chosen by the investor. Only returns below this target are included in the calculation. This is useful because different investors may have different thresholds for what constitutes unacceptable performance—a pension fund might use their required actuarial return, while an individual might use inflation.

**Common mistakes:**
- Option A describes below-mean semideviation, a related but different measure
- Option C is not typically used as the threshold for semideviation
