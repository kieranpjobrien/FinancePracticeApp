# Question

K-means clustering is an example of:

## Options

- **A**: Supervised learning because it uses labeled training data
- **B**: Unsupervised learning because it finds patterns without labeled outcomes
- **C**: Reinforcement learning because it learns from feedback

## Answer

B

## Explanation

K-means clustering is an unsupervised learning algorithm that groups observations into K clusters based on similarity (minimizing within-cluster variance). There is no target variable or labeled outcomes—the algorithm discovers structure in the data. Applications include customer segmentation, stock grouping, and anomaly detection. The analyst specifies K (number of clusters) but the algorithm determines group membership.

**Common mistakes:**
- Option A is incorrect; clustering does not use labeled data or a target variable
- Option C describes a different learning paradigm involving rewards and penalties
