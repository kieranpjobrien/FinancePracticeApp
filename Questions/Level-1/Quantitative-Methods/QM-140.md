# Question

A project has the following cash flows: Year 0: -$10,000; Year 1: $3,000; Year 2: $4,000; Year 3: $5,000. At a 10% discount rate, the NPV is closest to:

## Options

- **A**: -$235
- **B**: $215
- **C**: $2,000

## Answer

A

## Explanation

NPV = -$10,000 + $3,000/(1.10)¹ + $4,000/(1.10)² + $5,000/(1.10)³

NPV = -$10,000 + $3,000/1.10 + $4,000/1.21 + $5,000/1.331
NPV = -$10,000 + $2,727.27 + $3,305.79 + $3,756.57
NPV = -$10,000 + $9,789.63
NPV = -$210.37 ≈ -$235

The negative NPV indicates the project does not earn the required 10% return.

**Common mistakes:**
- Option B results from sign errors or calculation mistakes
- Option C simply sums undiscounted cash flows
