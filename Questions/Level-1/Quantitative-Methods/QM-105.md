# Question

A correlation coefficient of -0.85 indicates:

## Options

- **A**: A weak negative relationship
- **B**: A strong negative relationship
- **C**: The variables are unrelated

## Answer

B

## Explanation

The correlation coefficient ranges from -1 to +1. A value of -0.85 indicates a strong negative linear relationship—as one variable increases, the other tends to decrease. The absolute value of 0.85 is close to 1, indicating strength, while the negative sign indicates the inverse direction. In a scatter plot, points would cluster tightly around a downward-sloping line.

**Common mistakes:**
- Option A misinterprets the magnitude; values above 0.7 (in absolute value) typically indicate strong relationships
- Option C ignores the high absolute value of the correlation
