# Question

According to the Security Market Line (SML), a security with a beta greater than 1:

## Options

- **A**: Should earn a return less than the market return
- **B**: Should earn a return greater than the market return
- **C**: Has no systematic risk

## Answer

B

## Explanation

The SML shows the linear relationship between expected return and beta:

E(Ri) = Rf + βi[E(Rm) - Rf]

A beta greater than 1 means higher systematic risk than the market, requiring a higher expected return to compensate. If market return exceeds the risk-free rate, the security's expected return should exceed the market return.

**Common mistakes:**
- Option A reverses the relationship between beta and expected return
- Option C is incorrect; beta > 1 indicates above-average systematic risk
