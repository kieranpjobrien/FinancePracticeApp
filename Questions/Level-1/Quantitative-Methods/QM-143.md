# Question

Holding significance level constant, increasing sample size will:

## Options

- **A**: Increase both Type I and Type II error probabilities
- **B**: Decrease Type I error probability but increase Type II error probability
- **C**: Increase statistical power and decrease Type II error probability

## Answer

C

## Explanation

Increasing sample size improves the precision of estimates (smaller standard errors), making it easier to detect true effects. With a fixed significance level, larger samples increase statistical power (probability of correctly rejecting a false null) and reduce Type II error probability (probability of failing to reject a false null). Type I error probability is determined by the significance level, not sample size.

**Common mistakes:**
- Option A incorrectly suggests both errors increase
- Option B incorrectly links sample size to Type I error; Type I error is fixed by the significance level
