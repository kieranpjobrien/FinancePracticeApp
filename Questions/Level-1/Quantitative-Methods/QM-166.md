# Question

Winsorizing data involves:

## Options

- **A**: Removing outliers from the dataset entirely
- **B**: Replacing extreme values with less extreme boundary values
- **C**: Transforming all values using a logarithm

## Answer

B

## Explanation

Winsorizing replaces extreme values at specified percentiles with values at those percentile boundaries. For example, 5% Winsorizing replaces values below the 5th percentile with the 5th percentile value and values above the 95th percentile with the 95th percentile value. This reduces outlier influence while keeping all observations in the dataset, unlike trimming which removes outliers entirely.

**Common mistakes:**
- Option A describes trimming, not Winsorizing
- Option C describes a log transformation, a different technique for handling skewed data
