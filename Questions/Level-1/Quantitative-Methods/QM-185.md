# Question

Parametric VaR differs from historical simulation VaR in that parametric VaR:

## Options

- **A**: Uses actual historical returns without distributional assumptions
- **B**: Assumes returns follow a specific probability distribution (typically normal)
- **C**: Always produces more conservative (higher) risk estimates

## Answer

B

## Explanation

Parametric (analytical) VaR assumes returns follow a specified distribution, usually normal. VaR is calculated using the distribution's properties:

VaR = μ - zα × σ

This approach is computationally efficient but may underestimate risk if returns have fat tails (are not normally distributed). Historical simulation uses actual past returns without distributional assumptions.

**Common mistakes:**
- Option A describes historical simulation, not parametric VaR
- Option C is not generally true; parametric VaR often underestimates tail risk
