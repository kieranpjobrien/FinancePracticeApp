# Question

A key limitation of Monte Carlo simulation is that:

## Options

- **A**: It can only model normally distributed variables
- **B**: Results are highly dependent on the assumed input distributions and parameters
- **C**: It cannot handle more than two random variables

## Answer

B

## Explanation

Monte Carlo simulation results are only as good as the inputs used. The "garbage in, garbage out" principle applies—if the probability distributions, correlations, or parameters are incorrectly specified, the simulation outputs will be misleading. This makes careful estimation and validation of inputs critical. The model's sensitivity to assumptions is a fundamental limitation.

**Common mistakes:**
- Option A is incorrect; Monte Carlo can model any probability distribution, not just normal distributions
- Option C is incorrect; Monte Carlo simulation can handle many variables simultaneously, which is one of its key advantages
