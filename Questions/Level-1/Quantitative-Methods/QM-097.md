# Question

A prediction interval for an individual observation is wider than a confidence interval for the mean because:

## Options

- **A**: Prediction intervals use a smaller sample size
- **B**: Individual observations have more uncertainty than means due to random variation
- **C**: Confidence intervals are always set at higher confidence levels

## Answer

B

## Explanation

A prediction interval estimates where a single new observation will fall, while a confidence interval estimates where the mean response lies. Individual observations vary around the regression line due to the random error term, adding uncertainty beyond the uncertainty in estimating the mean. The prediction interval must account for both the uncertainty in the estimated mean and the variance of individual observations around that mean.

**Common mistakes:**
- Option A is incorrect; both intervals use the same regression analysis
- Option C is incorrect; the confidence level can be set the same for both intervals
