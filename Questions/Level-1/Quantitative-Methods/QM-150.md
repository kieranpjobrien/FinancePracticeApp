# Question

An investment has a 70% probability of gaining $10,000 and a 30% probability of losing $5,000. The expected value is:

## Options

- **A**: $5,000
- **B**: $5,500
- **C**: $7,000

## Answer

B

## Explanation

Expected value = Σ(Probability × Outcome)

E(V) = (0.70 × $10,000) + (0.30 × -$5,000)
E(V) = $7,000 + (-$1,500)
E(V) = $5,500

**Common mistakes:**
- Option A might result from averaging $10,000 and $0
- Option C only considers the gain scenario without accounting for the loss probability
