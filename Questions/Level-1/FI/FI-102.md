# Question

In the structural model of credit risk (Merton model), default occurs when:

## Options

- **A**: The company misses an interest payment
- **B**: The value of company assets falls below the value of its liabilities
- **C**: Credit rating agencies downgrade the bond

## Answer

B

## Explanation

Merton model: treats equity as call option on firm assets. Default when asset value < debt value at maturity. Probability of default derived from asset volatility, leverage, time. Structural approach uses economic fundamentals rather than statistical default rates.

**Common mistakes:**
- Option A describes technical default event
- Option C describes rating methodology, not structural model
