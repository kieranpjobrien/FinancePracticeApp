# Question

A cross-default provision means:

## Options

- **A**: Default on one debt obligation triggers default on other specified obligations
- **B**: Bonds are secured by multiple types of collateral
- **C**: Multiple bonds have identical credit ratings

## Answer

A

## Explanation

Cross-default: default on any debt (specified threshold) triggers default on bond with cross-default provision. Protects bondholders from subordination—if issuer defaults elsewhere, these bondholders don't wait. Cross-acceleration is similar—acceleration elsewhere triggers acceleration here.

**Common mistakes:**
- Option B describes cross-collateralization
- Option C describes rating parity, unrelated to cross-default
