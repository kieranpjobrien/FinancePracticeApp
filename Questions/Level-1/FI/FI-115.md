# Question

Option-adjusted spread (OAS) is best described as:

## Options

- **A**: The spread over the Treasury curve after removing the value of embedded options
- **B**: The spread that includes the value of embedded options
- **C**: The difference between nominal and real yields

## Answer

A

## Explanation

OAS: spread after adjusting for embedded option value. For callable bond: OAS = Z-spread - option cost. Allows comparison of bonds with different embedded options on level basis. OAS reflects pure credit/liquidity spread without option distortion.

**Common mistakes:**
- Option B describes nominal spread which includes option value
- Option C describes breakeven inflation
