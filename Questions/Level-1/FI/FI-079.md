# Question

The "haircut" in a repurchase agreement represents:

## Options

- **A**: The repo rate charged on the transaction
- **B**: The difference between collateral value and cash borrowed
- **C**: The term of the repo agreement

## Answer

B

## Explanation

Haircut (margin): percentage by which collateral value exceeds cash lent. Protects lender if collateral value declines. Higher haircuts for riskier/less liquid collateral. Example: $100 securities posted for $98 cash = 2% haircut.

**Common mistakes:**
- Option A describes interest rate, not margin
- Option C describes duration, not collateral buffer
