# Question

A yield curve steepening trade would involve:

## Options

- **A**: Buying long-term bonds and selling short-term bonds
- **B**: Selling long-term bonds and buying short-term bonds
- **C**: Buying bonds of the same maturity

## Answer

B

## Explanation

Steepener: profit if curve steepens (long rates rise vs. short). Sell long-dated bonds (lose value if long rates rise), buy short-dated (less affected). Flattener trade is opposite. Duration-neutral steepener matches DV01 on each leg.

**Common mistakes:**
- Option A describes flattener trade
- Option C provides no curve exposure
