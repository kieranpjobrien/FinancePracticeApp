# Question

Binomial interest rate trees are used in bond valuation to:

## Options

- **A**: Calculate simple present value of cash flows
- **B**: Value bonds with embedded options by modeling possible rate paths
- **C**: Determine the bond's credit rating

## Answer

B

## Explanation

Binomial trees: model interest rate as moving up or down each period. At each node, evaluate option exercise. Work backward from maturity. Captures rate volatility's impact on option value. Used for callable, putable, convertible bonds.

**Common mistakes:**
- Option A describes basic DCF, doesn't need tree
- Option C describes rating agency function
