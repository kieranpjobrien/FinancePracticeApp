# Question

Option-adjusted spread (OAS) represents:

## Options

- **A**: The spread over Treasury after removing the value of embedded options
- **B**: The nominal spread including option value
- **C**: Only the credit spread component

## Answer

A

## Explanation

OAS: constant spread over benchmark after modeling option impact. For callable bond: OAS = Z-spread - option cost. Allows comparison of bonds with different embedded options. More accurate measure of credit compensation than nominal or Z-spread for optioned bonds.

**Common mistakes:**
- Option B describes nominal spread which includes option effects
- Option C oversimplifies—OAS represents full spread after option adjustment
