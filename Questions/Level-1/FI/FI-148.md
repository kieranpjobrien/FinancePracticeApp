# Question

The key difference between incurrence and maintenance covenants is:

## Options

- **A**: Incurrence covenants are tested only when the issuer takes a specific action
- **B**: Maintenance covenants are only tested at bond maturity
- **C**: Incurrence covenants apply only to investment-grade bonds

## Answer

A

## Explanation

Incurrence covenants: tested when issuer acts (e.g., incurs new debt, makes acquisition). If action would breach, action prohibited. Maintenance covenants: tested periodically regardless of action. High-yield bonds often have weaker incurrence-only covenants.

**Common mistakes:**
- Option B—maintenance tested periodically, not just at maturity
- Option C—incurrence covenants more common in high-yield
