# Question

For a callable bond, the relationship between values is:

## Options

- **A**: Value of callable = Value of straight bond + Value of call option
- **B**: Value of callable = Value of straight bond - Value of call option
- **C**: Value of callable = Value of call option only

## Answer

B

## Explanation

Callable bond value = Straight bond value - Call option value. Issuer owns call option (can call bond). Option has value to issuer, cost to bondholder. Higher volatility increases call option value, reducing callable bond price.

**Common mistakes:**
- Option A adds option value (would be putable bond)
- Option C ignores straight bond component
