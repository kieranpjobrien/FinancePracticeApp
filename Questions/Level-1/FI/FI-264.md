# Question

Linear interpolation of the yield curve:

## Options

- **A**: Uses a straight line between observed points to estimate intermediate yields
- **B**: Fits a polynomial through all observed points
- **C**: Ignores observed data points

## Answer

A

## Explanation

Linear interpolation: estimate yield for maturity between two observed points using straight line. Simple but may not capture curve shape well. More sophisticated methods: spline fitting, Nelson-Siegel. Used when quick estimate needed.

**Common mistakes:**
- Option B describes polynomial/spline methods
- Option C—interpolation uses observed data
