# Question

The "roll-down return" refers to:

## Options

- **A**: Price appreciation from a bond's maturity shortening along an upward-sloping yield curve
- **B**: Return from reinvesting coupon payments
- **C**: Capital loss from rising interest rates

## Answer

A

## Explanation

Rolldown: with upward-sloping curve, as time passes bond "rolls down" curve to lower yields, gaining price (if curve unchanged). A 5-year bond becomes 4-year; if 4-year yield lower, price rises. Component of carry/rolldown analysis.

**Common mistakes:**
- Option B describes reinvestment return
- Option C describes rate change effect, not rolldown
