# Question

The Nelson-Siegel model represents the yield curve using:

## Options

- **A**: Only two parameters for level and slope
- **B**: Three parameters capturing level, slope, and curvature
- **C**: A separate parameter for each maturity point

## Answer

B

## Explanation

Nelson-Siegel: parsimonious model with level (long-term rate), slope (short vs. long difference), and curvature (hump) factors. Smooth curve fitting. Extended by Svensson with second curvature term. Used by central banks for yield curve estimation.

**Common mistakes:**
- Option A missing curvature parameter
- Option C describes non-parametric approach
