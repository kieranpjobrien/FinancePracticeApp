# Question

The value of an option consists of:

## Options

- **A**: Intrinsic value only
- **B**: Time value only
- **C**: Intrinsic value plus time value

## Answer

C

## Explanation

Option value components:

**Total Option Value:**
Option Price = Intrinsic Value + Time Value

**Intrinsic value:**
- Value if exercised immediately
- Call: Max(0, S - K)
- Put: Max(0, K - S)
- Cannot be negative
- Zero for OTM options

**Time value (extrinsic value):**
- Value from time remaining
- Possibility of favorable moves
- Always ≥ 0 before expiration
- Decays as expiration approaches

**At expiration:**
- Time value = 0
- Option value = Intrinsic value only
- No time left for changes

**Example:**
- Stock = $55, Strike = $50
- Call trades at $7
- Intrinsic = $55 - $50 = $5
- Time value = $7 - $5 = $2

**Common mistakes:**
- Option A: Ignores time value
- Option B: Ignores intrinsic value
