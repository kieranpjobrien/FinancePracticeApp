# Question

An off-market forward contract is one where:

## Options

- **A**: The contract trades on an exchange rather than OTC
- **B**: The forward price is set different from the no-arbitrage price, requiring an upfront payment
- **C**: The contract has already expired

## Answer

B

## Explanation

Off-market forwards:

**Standard (on-market) forward:**
- Price set by no-arbitrage
- Zero value at inception
- No upfront payment

**Off-market forward:**
- Price set different from fair value
- Non-zero value at inception
- Requires upfront payment

**Example:**
- Fair forward price = $105
- Off-market forward price = $100
- Long position would pay upfront
- Compensates for favorable price

**Why use off-market:**
- Match specific hedge needs
- Accounting considerations
- Restructuring existing positions
- Specific payoff requirements

**Upfront payment:**
= PV(Fair price - Off-market price)

**Example calculation:**
- Fair F = $105, Off-market F = $100
- Difference = $5
- Upfront = $5/(1+r)^T

**Common mistakes:**
- Option A: Off-market refers to pricing, not venue
- Option C: Refers to active contracts with non-standard pricing
