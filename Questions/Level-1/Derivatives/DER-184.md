# Question

In swap terminology, the "payer" in an interest rate swap refers to the party that:

## Options

- **A**: Receives fixed rate payments
- **B**: Pays the fixed rate
- **C**: Pays the higher of the two rates

## Answer

B

## Explanation

Swap party terminology:

**Payer:**
- Pays FIXED rate
- Receives floating rate
- Benefits if rates rise

**Receiver:**
- Receives FIXED rate
- Pays floating rate
- Benefits if rates fall

**Memory aid:**
- "Payer" = Pays fixed
- "Receiver" = Receives fixed
- Named for fixed leg position

**Common language:**
- "Pay fixed, receive floating"
- "Receive fixed, pay floating"
- Always refer to fixed rate

**Example:**
- 5-year swap at 4% fixed
- Payer: Pays 4%, receives SOFR
- Receiver: Receives 4%, pays SOFR

**View implication:**
- Payer: Bearish on bonds (expects rate rises)
- Receiver: Bullish on bonds (expects rate falls)

**Common mistakes:**
- Option A: This is the receiver
- Option C: Terminology based on fixed leg, not rate level
