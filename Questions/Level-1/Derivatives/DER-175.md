# Question

An out-of-the-money option has:

## Options

- **A**: Positive intrinsic value and positive time value
- **B**: Zero intrinsic value and positive time value (before expiration)
- **C**: Negative intrinsic value

## Answer

B

## Explanation

OTM option value composition:

**Out-of-the-money definition:**
- Call: Stock < Strike (S < K)
- Put: Stock > Strike (S > K)

**Intrinsic value for OTM:**
- Cannot be negative
- Minimum is zero
- OTM intrinsic = $0

**Time value for OTM:**
- Positive before expiration
- Possibility of becoming ITM
- Decays toward expiration

**Total value:**
Option value = 0 + Time value = Time value

**Example (OTM call):**
- Stock = $45, Strike = $50
- Intrinsic = Max(0, 45-50) = $0
- If option trades at $2
- All $2 is time value

**Why OTM has value:**
- Time for favorable moves
- Volatility creates possibility
- Optionality worth something

**Common mistakes:**
- Option A: OTM has zero intrinsic value
- Option C: Intrinsic can't be negative
