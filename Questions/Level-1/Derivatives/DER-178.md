# Question

A put option has a strike of $75 and the stock trades at $68. The intrinsic value is:

## Options

- **A**: $0
- **B**: $7
- **C**: $68

## Answer

B

## Explanation

Put intrinsic value:

**Formula:**
Intrinsic Value = Max(0, K - S)

**Given:**
- Strike (K) = $75
- Stock (S) = $68

**Calculation:**
IV = Max(0, $75 - $68)
IV = Max(0, $7)
IV = $7

**Interpretation:**
- Put is in the money by $7
- Right to sell at $75
- Stock only worth $68
- Value of right = $7

**Status:** In-the-money put
- K > S for puts = ITM
- Exercise would yield profit
- Before considering premium

**Time value:**
- If put trades at $9
- Time value = $9 - $7 = $2
- Total = Intrinsic + Time

**Common mistakes:**
- Option A: Would be true if stock > strike
- Option C: Stock price, not intrinsic value
