# Question

Put-call parity can also be expressed in terms of forward prices as:

## Options

- **A**: C - P = S
- **B**: C - P = PV(F - K)
- **C**: C + P = F

## Answer

B

## Explanation

Put-call parity with forwards:

**Standard parity:**
C - P = S - PV(K)

**Forward relationship:**
F = S × (1 + r)^T
So: S = PV(F)

**Substituting:**
C - P = PV(F) - PV(K)
C - P = PV(F - K)

**Interpretation:**
- Call minus put equals PV of forward minus strike
- If F > K: C > P
- If F < K: C < P
- If F = K: C = P (at-the-money forward)

**ATM forward options:**
- Strike = Forward price
- Call = Put
- Symmetric exposure

**Practical use:**
- Price options from forward prices
- Understand relationship between options
- Arbitrage detection

**Common mistakes:**
- Option A: Missing present value and strike
- Option C: Addition doesn't give correct relationship
