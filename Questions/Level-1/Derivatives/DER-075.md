# Question

In an equity swap, one party typically:

## Options

- **A**: Exchanges a fixed rate for the total return on an equity index
- **B**: Exchanges physical shares for a bond
- **C**: Receives dividends only without price appreciation

## Answer

A

## Explanation

Equity swap structure:

**Basic structure:**
- One party pays equity return
- Other party pays fixed or floating rate
- Based on notional principal
- Periodic settlements

**Total return includes:**
- Price appreciation/depreciation
- Dividends
- Full economic exposure

**Example:**
- Notional: $10 million
- Party A pays: S&P 500 total return
- Party B pays: SOFR + 50 bps
- Quarterly settlements

**Uses:**
- Gain equity exposure without owning
- Tax efficiency
- Avoid transaction costs
- Regulatory considerations

**Variations:**
- Fixed vs. equity return
- Floating vs. equity return
- Equity vs. equity (different indexes)

**Common mistakes:**
- Option B: No physical exchange in swaps
- Option C: Total return includes dividends AND price changes
