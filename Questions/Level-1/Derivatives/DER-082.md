# Question

At expiration, an out-of-the-money option is:

## Options

- **A**: Worth its intrinsic value
- **B**: Worthless and expires without exercise
- **C**: Automatically exercised by the exchange

## Answer

B

## Explanation

Out-of-the-money at expiration:

**OTM option status:**
- No intrinsic value
- Would lose money if exercised
- Rational holder lets it expire

**Call option OTM:**
- Strike > Spot price
- No reason to pay more than market
- Intrinsic value = $0

**Put option OTM:**
- Strike < Spot price
- No reason to sell below market
- Intrinsic value = $0

**What happens:**
- Option expires unexercised
- Holder loses premium paid
- Writer keeps premium collected

**Example:**
- Stock at $45
- $50 call is OTM
- Would pay $50 for $45 stock? No
- Option expires worthless

**Common mistakes:**
- Option A: OTM options have zero intrinsic value
- Option C: Only ITM options auto-exercise
