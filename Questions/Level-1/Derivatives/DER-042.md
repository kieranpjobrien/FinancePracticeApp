# Question

Time decay (theta) for a long option position is typically:

## Options

- **A**: Positive, meaning the option gains value over time
- **B**: Negative, meaning the option loses value over time
- **C**: Zero, as time has no effect on option value

## Answer

B

## Explanation

Time decay in options:

**Theta defined:**
- Rate of time value decay
- Change in option value per day
- Usually negative for long positions

**Why options lose value over time:**
- Less time for favorable moves
- Time value erodes
- At expiration, time value = 0
- Probability of exercise changes

**Theta characteristics:**
- Accelerates near expiration
- Greatest for ATM options
- Lower for deep ITM/OTM
- Weekends and holidays count

**Who benefits:**
- Option sellers (short): Collect theta
- Option buyers (long): Pay theta
- Time works against buyers

**Strategic implications:**
- Long positions: Time is enemy
- Short positions: Time is friend
- Spread strategies can modify theta

**Common mistakes:**
- Option A: Long options lose value over time
- Option C: Time is critical factor in option pricing
