# Question

A stock trades at $50. A call with strike $50 expiring in 1 year is priced at $5. The risk-free rate is 4%. Using put-call parity, the put price is closest to:

## Options

- **A**: $3.08
- **B**: $5.00
- **C**: $6.92

## Answer

A

## Explanation

Put-call parity calculation:

**Formula:**
C + PV(K) = P + S

**Solving for P:**
P = C + PV(K) - S

**Given:**
- C = $5
- K = $50
- S = $50
- r = 4%
- T = 1 year

**Step 1: Calculate PV(K)**
PV(K) = K / (1 + r)^T
PV(K) = $50 / (1.04)^1
PV(K) = $48.08

**Step 2: Calculate P**
P = $5 + $48.08 - $50
P = $3.08

**Interpretation:**
- ATM call worth $5
- ATM put worth $3.08
- Difference due to time value of money
- Call more valuable (can defer purchase)

**Common mistakes:**
- Option B: Assumed put = call (ignores rates)
- Option C: Added instead of subtracted
