# Question

The maximum value of a European put option is:

## Options

- **A**: The stock price
- **B**: The strike price
- **C**: The present value of the strike price

## Answer

C

## Explanation

Put option upper bound:

**European put maximum:**
P ≤ PV(K)

**Reasoning:**
- Maximum payoff = K (if stock goes to $0)
- This is received at expiration
- Today's value = PV(K)
- Can't be worth more than this

**Why not K today:**
- Money has time value
- K received in future
- Discount to present

**American put difference:**
- Can exercise anytime
- Maximum = K (not PV(K))
- Could get K today if exercised

**Example:**
- K = $50, T = 1 year, r = 5%
- European put max = $50/1.05 = $47.62
- American put max = $50

**Lower bound:**
- European put: Max(0, PV(K) - S)
- American put: Max(0, K - S)

**Common mistakes:**
- Option A: Stock price isn't put maximum
- Option B: This is American put maximum
