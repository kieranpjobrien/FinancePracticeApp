# Question

A call option with a strike price of $50 when the underlying is trading at $55 is:

## Options

- **A**: Out of the money
- **B**: At the money
- **C**: In the money

## Answer

C

## Explanation

Option moneyness:

**Call option moneyness:**
- In the money (ITM): Underlying > Strike
- At the money (ATM): Underlying ≈ Strike
- Out of the money (OTM): Underlying < Strike

**This example:**
- Strike = $50
- Underlying = $55
- $55 > $50 → In the money
- Intrinsic value = $55 - $50 = $5

**Put option (opposite):**
- ITM: Underlying < Strike
- ATM: Underlying ≈ Strike
- OTM: Underlying > Strike

**Intrinsic value:**
- Call: Max(0, S - K)
- Put: Max(0, K - S)
- ITM options have positive intrinsic value

**Exercise consideration:**
- ITM options have exercise value
- OTM options would not be exercised
- ATM is the breakeven point

**Common mistakes:**
- Option A: Would be true if underlying < strike
- Option B: Would be true if underlying = strike
