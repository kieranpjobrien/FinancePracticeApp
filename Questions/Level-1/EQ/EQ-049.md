# Question

Index reconstitution refers to:

## Options

- **A**: Calculating daily index values
- **B**: Periodically adding or removing securities from an index based on selection criteria
- **C**: Adjusting for stock splits

## Answer

B

## Explanation

Reconstitution: periodic review (often quarterly or annually) to update index membership. Add securities meeting criteria; remove those no longer qualifying. Ensures index remains representative. Creates trading activity as index funds adjust holdings. Separate from rebalancing (adjusting weights of existing components).

**Common mistakes:**
- Option A describes index calculation, not reconstitution
- Option C describes divisor adjustments for corporate actions
