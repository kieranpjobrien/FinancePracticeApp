# Question

Sensitivity analysis in equity valuation is used to:

## Options

- **A**: Determine a single precise intrinsic value
- **B**: Understand how value changes with different input assumptions
- **C**: Eliminate all uncertainty from valuations

## Answer

B

## Explanation

Sensitivity analysis: varies key inputs (growth rate, discount rate, margins) to see impact on value. Produces value range rather than point estimate. Identifies which assumptions matter most. Critical for managing model risk. Often displayed in data tables showing value across input combinations.

**Common mistakes:**
- Option A—sensitivity shows range, not single value
- Option C—uncertainty cannot be eliminated, only understood
