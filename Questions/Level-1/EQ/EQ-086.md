# Question

Survivorship bias in index data occurs when:

## Options

- **A**: All historical constituents are properly included
- **B**: Failed or delisted companies are removed from historical data, overstating past returns
- **C**: The index includes too many companies

## Answer

B

## Explanation

Survivorship bias: using only currently existing companies in historical analysis. Excludes failed, merged, or delisted companies—typically poor performers. Overstates historical returns and understates risk. Important to use point-in-time data that includes all companies that were in the index at each historical date.

**Common mistakes:**
- Option A describes proper methodology that avoids bias
- Option C is not related to survivorship bias
