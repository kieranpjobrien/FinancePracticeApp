# Question

Tracking error measures:

## Options

- **A**: The absolute return of an index fund
- **B**: The deviation of a portfolio's returns from its benchmark index returns
- **C**: The total expense ratio of a fund

## Answer

B

## Explanation

Tracking error: standard deviation of difference between portfolio and benchmark returns. Lower tracking error = closer replication. Sources: transaction costs, cash drag, sampling (not holding all constituents), timing differences. Target is minimal tracking error for index funds.

**Common mistakes:**
- Option A describes absolute, not relative performance
- Option C describes costs, not return deviation
