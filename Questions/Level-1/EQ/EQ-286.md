# Question

A company with expected EPS of $5.00 and required return of 10% has a no-growth value per share of:

## Options

- **A**: $50.00
- **B**: $5.00
- **C**: $0.50

## Answer

A

## Explanation

No-growth value = E₁ / r = $5.00 / 0.10 = $50.00. Assumes all earnings paid as dividends with zero growth. Represents value as perpetuity of current earnings. If stock trades above $50, difference is PVGO—market's valuation of growth opportunities.

**Common mistakes:**
- Option B confuses EPS with value
- Option C incorrectly calculates the relationship
