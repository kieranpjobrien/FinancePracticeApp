# Question

Tangible book value differs from book value by excluding:

## Options

- **A**: Cash and cash equivalents
- **B**: Intangible assets like goodwill and patents
- **C**: Property, plant, and equipment

## Answer

B

## Explanation

Tangible book value = Book value - Intangible assets (goodwill, patents, trademarks, etc.). More conservative measure focusing on physical, liquidatable assets. Important for financial firms and M&A analysis. Goodwill particularly excluded as it's vulnerable to write-downs and hard to realize in liquidation.

**Common mistakes:**
- Option A—cash is very tangible and included
- Option C—PP&E is tangible and included
