# Question

Financial leverage magnifies:

## Options

- **A**: Only positive returns to equity holders
- **B**: Both positive and negative returns to equity holders
- **C**: Only operating income volatility

## Answer

B

## Explanation

Financial leverage: using debt to increase returns. Magnifies ROE in both directions—enhances gains when ROA exceeds cost of debt, amplifies losses when ROA falls below cost. Debt is fixed obligation, so equity holders bear all residual volatility. DFL = % change EPS / % change EBIT.

**Common mistakes:**
- Option A ignores downside magnification
- Option C describes operating leverage, not financial leverage
