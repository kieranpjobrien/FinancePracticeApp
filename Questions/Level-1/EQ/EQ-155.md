# Question

A total return index differs from a price return index by including:

## Options

- **A**: Only capital gains
- **B**: Dividends and other income reinvested in the index
- **C**: Trading costs and fees

## Answer

B

## Explanation

Total return index: includes both price appreciation and income (dividends, interest) assuming reinvestment. Price return index only captures capital gains/losses. Total return more accurately reflects actual investor experience. Difference can be significant for high-dividend indices over time.

**Common mistakes:**
- Option A describes price return index, not total return
- Option C is not included in either index type
