# Question

The value effect suggests that stocks with:

## Options

- **A**: High P/E and P/B ratios outperform
- **B**: Low P/E and P/B ratios (value stocks) have historically outperformed growth stocks
- **C**: Average valuations perform best

## Answer

B

## Explanation

Value effect: value stocks (low P/E, P/B, high dividend yield) historically outperformed growth stocks. Fama-French HML factor. Explanations: value stocks riskier, behavioral overreaction to growth, extrapolation errors. Value premium varies cyclically—growth outperformed value in certain periods.

**Common mistakes:**
- Option A describes growth stocks, which underperformed historically
- Option C misses the value vs. growth distinction
