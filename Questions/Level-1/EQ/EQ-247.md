# Question

Rising input costs for an industry typically:

## Options

- **A**: Always increase industry profits
- **B**: Compress margins unless companies can pass costs to customers
- **C**: Have no effect on competitive dynamics

## Answer

B

## Explanation

Input cost impact: depends on pricing power. If costs can be passed through (low buyer power, inelastic demand), margins preserved. If not (competitive markets, price-sensitive customers), margins compress. Low-cost producers gain advantage when industry faces rising inputs.

**Common mistakes:**
- Option A ignores margin compression possibility
- Option C understates importance of cost structure
