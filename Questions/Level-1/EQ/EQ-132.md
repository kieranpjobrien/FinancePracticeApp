# Question

Semi-strong form market efficiency implies that:

## Options

- **A**: Only historical price data is reflected in stock prices
- **B**: All publicly available information is reflected in stock prices
- **C**: All information, including insider information, is reflected in prices

## Answer

B

## Explanation

Semi-strong efficiency: prices reflect all publicly available information—historical prices, financial statements, analyst reports, news, macroeconomic data. Fundamental analysis should not generate abnormal returns. Insider trading could still profit. Tests include event studies around public announcements.

**Common mistakes:**
- Option A describes weak-form efficiency only
- Option C describes strong-form efficiency
