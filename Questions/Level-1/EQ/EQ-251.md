# Question

The purpose of an index divisor is to:

## Options

- **A**: Increase the index value over time
- **B**: Maintain index continuity when stocks split or constituents change
- **C**: Determine which stocks are included in the index

## Answer

B

## Explanation

Index divisor: ensures index value continuity despite corporate actions or composition changes. Adjusted when: stocks split, dividends paid (for total return), constituents added/removed. Without adjustment, these events would artificially change index level. Divisor calculation varies by weighting method.

**Common mistakes:**
- Option A—divisor maintains continuity, doesn't inflate
- Option C—selection criteria determine inclusion, not divisor
