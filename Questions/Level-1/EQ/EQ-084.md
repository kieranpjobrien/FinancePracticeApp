# Question

A stock market index serves as:

## Options

- **A**: A guarantee of investment returns
- **B**: A benchmark for measuring portfolio performance and market movements
- **C**: The actual portfolio investors must hold

## Answer

B

## Explanation

Market indices: measure market or segment performance, serve as benchmarks for active managers. Used for: performance comparison, market sentiment indicators, basis for index products (ETFs, futures). Not directly investable—index funds approximate index returns minus costs.

**Common mistakes:**
- Option A overstates index function
- Option C confuses index with index fund
