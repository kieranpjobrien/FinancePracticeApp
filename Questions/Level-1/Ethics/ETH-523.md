# Question

GIPS-compliant presentations must be provided to:

## Options

- **A**: Only clients who specifically request them
- **B**: All prospective clients
- **C**: Only institutional prospective clients

## Answer

B

## Explanation

GIPS requires that compliant presentations be made available to all prospective clients. This ensures consistent, standardized performance information reaches anyone considering the firm's services.

**Common mistakes:**
- Option A limits distribution too narrowly
- Option C excludes retail prospects
