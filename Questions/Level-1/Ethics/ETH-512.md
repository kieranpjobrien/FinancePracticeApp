# Question

A vendor provides an analyst with advanced software tools for free in exchange for the firm's trading business. Standard IV(B):

## Options

- **A**: Does not apply to non-monetary benefits
- **B**: Requires disclosure to the employer as this constitutes compensation
- **C**: Only applies if the analyst personally uses the software

## Answer

B

## Explanation

Standard IV(B) covers compensation in any form, not just cash. Free software that benefits the analyst or firm in exchange for business constitutes compensation that should be disclosed to the employer.

**Common mistakes:**
- Option A narrowly defines compensation as monetary
- Option C ignores benefits that flow to the firm
