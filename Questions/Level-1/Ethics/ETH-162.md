# Question

When using the CFA designation, it should be written:

## Options

- **A**: In lowercase letters (cfa) to show humility
- **B**: As a noun ("she is a CFA")
- **C**: As an adjective ("CFA charterholder")

## Answer

C

## Explanation

The CFA designation should be used as an adjective, not a noun. Correct usage is "CFA charterholder" or "Jane Doe, CFA." Saying "she is a CFA" uses it incorrectly as a noun.

**Common mistakes:**
- Option A violates formatting guidelines; CFA should be capitalized
- Option B uses the designation as a noun, which is improper
