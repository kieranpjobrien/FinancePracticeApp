# Question

A firm presents hypothetical back-tested returns alongside GIPS-compliant actual returns. Under GIPS:

## Options

- **A**: Hypothetical returns cannot be shown with GIPS presentations
- **B**: Supplemental information may be included if clearly labeled and not misleading
- **C**: Hypothetical returns are treated the same as actual returns

## Answer

B

## Explanation

GIPS allows supplemental information including hypothetical returns if clearly labeled as such and not presented in a way that could mislead. The distinction between actual and hypothetical results must be clear.

**Common mistakes:**
- Option A is too restrictive regarding supplemental information
- Option C fails to distinguish hypothetical from actual performance
