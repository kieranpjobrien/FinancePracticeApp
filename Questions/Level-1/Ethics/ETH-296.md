# Question

A trading error results in purchasing the wrong security for a client. Under Standard III(A):

## Options

- **A**: The error should be hidden if the security later profits
- **B**: The client should be made whole and the error disclosed appropriately
- **C**: Errors are an accepted part of trading and need not be addressed

## Answer

B

## Explanation

Standard III(A) requires acting in clients' best interests, which includes addressing errors promptly and fairly. Clients should be made whole for losses from firm errors, and transparency is essential regardless of the error's ultimate outcome.

**Common mistakes:**
- Option A prioritizes concealment over client interest
- Option C fails to protect client from preventable harm
