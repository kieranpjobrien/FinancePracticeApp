# Question

A CFA charterholder also holds other professional credentials. When listing them:

## Options

- **A**: The CFA designation must always appear first
- **B**: All credentials should be accurately represented without overstating the CFA designation's scope
- **C**: Only the CFA designation should be listed

## Answer

B

## Explanation

Standard VII(B) requires accurate representation of all credentials. Multiple designations may be listed together provided each is accurately represented and none are overstated in scope or meaning.

**Common mistakes:**
- Option A invents a priority requirement
- Option C unnecessarily restricts credential disclosure
