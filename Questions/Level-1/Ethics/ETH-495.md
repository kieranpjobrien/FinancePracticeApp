# Question

A GIPS-compliant firm wants to show pre-GIPS historical performance that does not meet all requirements. This data:

## Options

- **A**: Cannot be shown under any circumstances
- **B**: May be shown as supplemental information with clear disclosure of non-compliance
- **C**: Must be restated to comply with GIPS before presentation

## Answer

B

## Explanation

GIPS allows pre-compliance historical performance as supplemental information if clearly identified as non-compliant. This provides useful context while maintaining transparency about what meets GIPS standards.

**Common mistakes:**
- Option A prohibits useful historical context
- Option C requires potentially impossible restatement
