# Question

A GIPS-compliant firm includes trading costs within their management fee (bundled fee). The presentation must:

## Options

- **A**: Unbundle all costs for presentation
- **B**: Disclose that fees include trading costs and the approximate trading cost component
- **C**: Present only gross returns

## Answer

B

## Explanation

GIPS accommodates bundled fee arrangements but requires disclosure. Firms must indicate that fees are bundled and provide sufficient information about the trading cost component for meaningful comparison.

**Common mistakes:**
- Option A requires impractical unbundling
- Option C ignores the net return disclosure requirement
