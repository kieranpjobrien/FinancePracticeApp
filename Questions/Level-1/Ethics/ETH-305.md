# Question

A brief investment recommendation lacks space for full disclosure. Under Standard V(B):

## Options

- **A**: Brief formats are prohibited
- **B**: The report should indicate where complete information is available
- **C**: Disclosure requirements don't apply to brief formats

## Answer

B

## Explanation

Standard V(B) recognizes practical format constraints. When full disclosure isn't possible in a brief format, the communication should direct recipients to where complete information, including disclosures, is available.

**Common mistakes:**
- Option A is impractically restrictive
- Option C incorrectly exempts brief formats from disclosure
