# Question

Misrepresentation under Standard I(C) includes:

## Options

- **A**: Only written false statements
- **B**: Any untrue statement or omission of fact, or false impression through any means
- **C**: Only statements about investment performance

## Answer

B

## Explanation

Standard I(C) broadly defines misrepresentation to include any untrue statement, misleading omission, or action that creates a false impression. This applies to written, oral, and electronic communications across all professional activities.

**Common mistakes:**
- Option A too narrowly limits to written statements
- Option C ignores misrepresentation about qualifications, services, etc.
