# Question

A GIPS-compliant firm discovers an error in a previously reported composite return. The firm should:

## Options

- **A**: Only correct the error if it exceeds a predetermined materiality threshold and disclose the correction
- **B**: Ignore the error if the composite no longer exists
- **C**: Retroactively change all historical presentations without disclosure

## Answer

A

## Explanation

GIPS requires firms to have error correction policies that include materiality thresholds. When material errors are discovered, firms must correct the performance history and disclose the nature and impact of the correction. Immaterial errors may not require correction but should still be evaluated under the firm's policies.

**Common mistakes:**
- Option B ignores the need to maintain accurate historical records
- Option C fails to require disclosure of corrections, which is essential for transparency
