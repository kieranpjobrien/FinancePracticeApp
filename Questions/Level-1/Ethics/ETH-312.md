# Question

A firm sets a minimum portfolio size for composite inclusion. Under GIPS:

## Options

- **A**: Minimum thresholds are prohibited
- **B**: Minimums are permitted if consistently applied and disclosed
- **C**: Only very large minimums require disclosure

## Answer

B

## Explanation

GIPS permits minimum asset level thresholds for composite inclusion, provided they are applied consistently and disclosed in presentations. This allows firms to exclude portfolios too small to implement the strategy fully.

**Common mistakes:**
- Option A is too restrictive
- Option C incorrectly links disclosure to threshold size
