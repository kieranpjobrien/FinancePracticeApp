# Question

A firm presents model portfolio returns that do not reflect actual trading costs or fees. Standard III(D):

## Options

- **A**: Permits this if labeled as model results
- **B**: Requires clear disclosure that results exclude transaction costs and fees
- **C**: Prohibits showing model results entirely

## Answer

B

## Explanation

Standard III(D) requires fair and accurate presentation. Model results can be shown but must clearly disclose limitations including exclusion of trading costs and fees, as actual performance would differ.

**Common mistakes:**
- Option A treats labeling as sufficient without specific disclosures
- Option C prohibits useful illustrative information
