# Question

"Fair dealing" under Standard III(B) applies to:

## Options

- **A**: Only research recommendations
- **B**: Both recommendations and investment actions
- **C**: Only trade execution

## Answer

B

## Explanation

Standard III(B) requires fair dealing in both dissemination of recommendations and taking investment action. Clients should be treated fairly across all aspects of the investment relationship.

**Common mistakes:**
- Options A and C each capture only part of the Standard's scope
