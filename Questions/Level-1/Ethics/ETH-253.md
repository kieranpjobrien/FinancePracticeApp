# Question

Due to time pressure, an analyst issues a recommendation based on incomplete research. Under Standard V(A):

## Options

- **A**: Time constraints excuse abbreviated analysis
- **B**: Recommendations should only be made when adequate basis exists, regardless of deadlines
- **C**: Issuing caveated recommendations eliminates liability

## Answer

B

## Explanation

Standard V(A) requires reasonable and adequate basis for recommendations. Time pressure does not excuse insufficient research. If adequate analysis cannot be completed, the recommendation should be delayed or declined.

**Common mistakes:**
- Option A uses deadlines to excuse inadequate work
- Option C suggests disclaimers substitute for proper analysis
