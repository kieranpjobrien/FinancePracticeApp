# Question

A portfolio manager makes an investment decision that loses money for clients despite using a thorough, well-documented research process. Under the Code of Ethics:

## Options

- **A**: The manager has violated fiduciary duty because clients lost money
- **B**: The manager has likely met obligations if reasonable care was exercised
- **C**: Investment losses always indicate ethical violations

## Answer

B

## Explanation

The Code of Ethics requires reasonable care and independent judgment, not guaranteed outcomes. If the manager used a thorough, appropriate process, investment losses alone do not constitute an ethical violation.

**Common mistakes:**
- Option A conflates poor outcomes with breach of duty
- Option C ignores that investment inherently involves risk
