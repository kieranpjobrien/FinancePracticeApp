# Question

A charterholder's LinkedIn profile states "I am a CFA." Under Standard VII(B):

## Options

- **A**: This is correct usage as a personal identifier
- **B**: This incorrectly uses CFA as a noun rather than an adjective
- **C**: Social media profiles are exempt from designation rules

## Answer

B

## Explanation

Standard VII(B) specifies that CFA must be used as an adjective, not a noun. Correct usage would be "I am a CFA charterholder." Social media and all communications must follow proper designation usage.

**Common mistakes:**
- Option A accepts improper usage
- Option C incorrectly exempts social media from the Standard
