# Question

According to Standard I(A) Knowledge of the Law, when a member or candidate faces a conflict between local law and the Code and Standards, the member should:

## Options

- **A**: Always follow the Code and Standards, regardless of local law
- **B**: Follow the stricter of the local law or the Code and Standards
- **C**: Follow local law, as it takes precedence over the Code and Standards

## Answer

B

## Explanation

Standard I(A) requires members to comply with all applicable laws, rules, and regulations, including the Code and Standards. When there is a conflict, members must follow the **stricter** of the applicable law or the Code and Standards.

**Common mistakes:**
- Option A is incorrect because members cannot violate local law, even if the Code and Standards are stricter
- Option C is incorrect because if the Code and Standards are stricter than local law, members must still follow the Code and Standards
