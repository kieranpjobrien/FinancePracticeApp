# Question

A portfolio manager's investment style has drifted from the strategy described to clients. Under Standard III(A):

## Options

- **A**: Style drift is acceptable if performance improves
- **B**: Material changes to investment approach should be communicated to clients
- **C**: Clients only need notification of losses, not strategy changes

## Answer

B

## Explanation

Standard III(A) requires acting in clients' interests, which includes managing assets consistent with the agreed approach. Material style drift without disclosure may violate the duty to clients who selected the manager based on the original strategy.

**Common mistakes:**
- Option A prioritizes returns over adherence to client expectations
- Option C ignores the importance of strategy transparency
