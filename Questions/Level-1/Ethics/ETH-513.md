# Question

A firm implements automated monitoring tools to detect suspicious trading patterns. A supervisor:

## Options

- **A**: Can rely entirely on the automated tools for supervision
- **B**: Should use the tools as part of a comprehensive supervision system with human oversight
- **C**: Has no supervisory obligations if automated tools are in place

## Answer

B

## Explanation

Standard IV(C) requires reasonable supervision. Automated tools are valuable but require human oversight, interpretation of alerts, and follow-up investigation. Technology supplements but does not replace supervisory responsibility.

**Common mistakes:**
- Option A over-relies on automation
- Option C incorrectly transfers responsibility to technology
