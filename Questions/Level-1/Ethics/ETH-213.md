# Question

Under GIPS, composite returns should be presented alongside:

## Options

- **A**: Only absolute return targets
- **B**: A relevant benchmark that reflects the investment strategy
- **C**: No benchmark is required

## Answer

B

## Explanation

GIPS requires presentation of a benchmark that reflects the investment mandate, objective, or strategy of the composite. This allows prospective clients to evaluate relative performance appropriately.

**Common mistakes:**
- Option A limits comparison to absolute targets only
- Option C ignores the benchmark requirement
