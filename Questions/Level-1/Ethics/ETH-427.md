# Question

An Investment Policy Statement (IPS) should be updated:

## Options

- **A**: Only at account opening
- **B**: Periodically and when material changes in client circumstances occur
- **C**: Only when the client requests changes

## Answer

B

## Explanation

Standard III(C) requires ongoing review of client suitability. IPS documents should be reviewed periodically and updated when material changes occur in client circumstances, objectives, or constraints.

**Common mistakes:**
- Option A treats the IPS as a static document
- Option C places all responsibility on the client to initiate updates
