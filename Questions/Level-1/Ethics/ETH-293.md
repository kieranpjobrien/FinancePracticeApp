# Question

An analyst uses a statistical concept from an academic paper without citation but phrases it differently. Under Standard I(C):

## Options

- **A**: Rephrasing eliminates the need for attribution
- **B**: The concept should still be attributed to its original source
- **C**: Only direct quotes require citation

## Answer

B

## Explanation

Standard I(C) prohibits plagiarism, which includes using another's ideas without attribution. Rephrasing doesn't eliminate the obligation to credit the source of concepts, methodologies, or findings.

**Common mistakes:**
- Option A allows plagiarism through paraphrasing
- Option C too narrowly defines what requires citation
