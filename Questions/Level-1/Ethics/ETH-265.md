# Question

When Code of Ethics principles appear to conflict in a specific situation, members should:

## Options

- **A**: Always prioritize the most recently added principle
- **B**: Exercise judgment considering all stakeholders and the spirit of the Code
- **C**: Defer to employer preference

## Answer

B

## Explanation

When ethical principles seem to conflict, members must exercise thoughtful judgment, considering all affected stakeholders and the underlying purpose of the Code. There is no mechanical hierarchy of principles—context and judgment matter.

**Common mistakes:**
- Option A applies an arbitrary ordering
- Option C substitutes employer direction for personal ethical judgment
