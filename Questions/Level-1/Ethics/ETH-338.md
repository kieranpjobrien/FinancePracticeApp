# Question

GIPS-compliant presentations must include:

## Options

- **A**: Only composite returns
- **B**: Composite returns, assets, and the number of portfolios, among other disclosures
- **C**: Only benchmark comparisons

## Answer

B

## Explanation

GIPS requires comprehensive disclosure including composite returns, assets under management, number of portfolios in the composite, and various other disclosures to provide context for the performance information.

**Common mistakes:**
- Options A and C each provide only partial information
