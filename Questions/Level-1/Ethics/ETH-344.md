# Question

Under Standard I(C), misrepresentation includes:

## Options

- **A**: Only affirmative false statements
- **B**: Both false statements and material omissions
- **C**: Only written communications

## Answer

B

## Explanation

Standard I(C) covers both affirmative misstatements and material omissions that create false impressions. Failing to disclose material information can be as misleading as stating falsehoods.

**Common mistakes:**
- Option A ignores the impact of omissions
- Option C incorrectly limits to written communications
