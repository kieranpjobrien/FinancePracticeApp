# Question

GIPS requires performance to be presented:

## Options

- **A**: Only in monthly intervals
- **B**: In annual periods at a minimum
- **C**: Only when performance is positive

## Answer

B

## Explanation

GIPS requires presentation of annual returns for each year. While more frequent returns may be shown, annual presentation is the minimum standard required for GIPS-compliant reports.

**Common mistakes:**
- Option A sets an incorrect interval requirement
- Option C allows cherry-picking of performance periods
