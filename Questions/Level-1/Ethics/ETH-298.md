# Question

Changes in a client's financial circumstances:

## Options

- **A**: Rarely affect portfolio suitability
- **B**: Should trigger review of portfolio suitability
- **C**: Are the client's responsibility to manage

## Answer

B

## Explanation

Standard III(C) requires ongoing suitability assessment. Changes in client circumstances—such as job loss, inheritance, or retirement—may materially affect what investments are suitable and should trigger portfolio review.

**Common mistakes:**
- Option A underestimates the impact of circumstance changes
- Option C abdicates the advisor's ongoing responsibility
