# Question

A financial advisor receives a fee for referring clients to a specific external asset manager. The advisor recommends this manager frequently. Under Standard VI(C):

## Options

- **A**: The referral arrangement must be disclosed to clients before any recommendation
- **B**: As long as the manager performs well, no disclosure is required
- **C**: Disclosure is only required if clients ask about compensation

## Answer

A

## Explanation

Standard VI(C) requires disclosure of referral fee arrangements before making recommendations. Clients must understand the financial incentive that may influence referrals, regardless of the external manager's performance.

**Common mistakes:**
- Option B incorrectly links disclosure to performance outcomes
- Option C fails the proactive disclosure requirement
