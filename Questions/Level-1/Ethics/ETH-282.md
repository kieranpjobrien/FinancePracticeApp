# Question

The correct priority for investment transactions under Standard VI(B) is:

## Options

- **A**: Personal, employer, then client
- **B**: Client, employer, then personal
- **C**: All transactions should be simultaneous

## Answer

B

## Explanation

Standard VI(B) establishes that client transactions take priority over employer (proprietary) transactions, which take priority over personal transactions. This hierarchy protects client interests.

**Common mistakes:**
- Option A completely reverses the required priority
- Option C ignores the established priority hierarchy
