# Question

An analyst issues a buy recommendation based on a model developed by her colleague without reviewing the model's assumptions. This:

## Options

- **A**: Is acceptable if the colleague is more experienced
- **B**: May violate Standard V(A) if the analyst cannot justify the recommendation independently
- **C**: Is required since collaboration is encouraged

## Answer

B

## Explanation

Standard V(A) requires each analyst to have reasonable basis for recommendations they issue. While using colleagues' work is acceptable, the analyst must understand and be able to defend the recommendation. Blind reliance on others is insufficient.

**Common mistakes:**
- Option A substitutes seniority for independent analysis
- Option C confuses collaboration with abdication of responsibility
