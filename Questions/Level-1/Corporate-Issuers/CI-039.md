# Question

An investment in net working capital for a project:

## Options

- **A**: Is a sunk cost that should be ignored
- **B**: Is a cash outflow at project start, typically recovered at project end
- **C**: Does not affect project NPV

## Answer

B

## Explanation

Projects requiring inventory, receivables, and payables invest in working capital. This cash outflow at project start reduces NPV. At project end, working capital is liquidated (inventory sold, receivables collected), recovering the investment. Working capital changes during the project also affect cash flows.

**Common mistakes:**
- Option A ignores working capital investment
- Option C ignores timing effects on NPV
