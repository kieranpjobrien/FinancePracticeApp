# Question

Under capital rationing, a company should:

## Options

- **A**: Accept all positive NPV projects
- **B**: Select the combination of projects that maximizes total NPV within the budget
- **C**: Choose projects with the shortest payback periods

## Answer

B

## Explanation

With limited capital, not all positive NPV projects can be funded. The goal is maximizing total value creation within constraints. Profitability index helps rank projects by NPV per dollar invested. Integer programming may be needed for optimal selection with indivisible projects.

**Common mistakes:**
- Option A ignores the budget constraint
- Option C uses payback, which doesn't maximize value
