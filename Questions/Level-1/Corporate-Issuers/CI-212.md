# Question

Scenario analysis differs from sensitivity analysis in that scenario analysis:

## Options

- **A**: Changes only one variable at a time
- **B**: Examines the effect of simultaneous changes in multiple variables
- **C**: Ignores the range of possible outcomes

## Answer

B

## Explanation

Scenario analysis: creates comprehensive cases (best, base, worst) changing multiple variables consistently. Reflects that variables often move together. Sensitivity analysis changes one variable while holding others constant. Scenario analysis captures interactions; sensitivity identifies key value drivers.

**Common mistakes:**
- Option A describes sensitivity analysis
- Option C is opposite—scenario analysis explicitly considers outcome ranges
