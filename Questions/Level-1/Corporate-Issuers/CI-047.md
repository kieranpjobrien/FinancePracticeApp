# Question

WACC calculations should use:

## Options

- **A**: Book value weights from the balance sheet
- **B**: Target capital structure weights based on market values
- **C**: Last year's capital structure weights

## Answer

B

## Explanation

WACC should use target (or current market value) capital structure weights—reflecting what the company will use going forward. Book values reflect historical costs, not current economic values. Target weights represent optimal long-run structure. Market values reflect investor expectations.

**Common mistakes:**
- Option A uses historical book values, not current market
- Option C may not reflect optimal or forward-looking structure
