# Question

A staggered (classified) board:

## Options

- **A**: Allows all directors to be elected annually
- **B**: Elects only a portion of directors each year
- **C**: Requires unanimous board approval for all decisions

## Answer

B

## Explanation

Staggered boards divide directors into classes, with one class elected each year (e.g., three classes, three-year terms). This makes hostile takeovers more difficult—acquirers can't gain board control in one election. Critics argue it entrenches management; supporters cite continuity benefits.

**Common mistakes:**
- Option A describes annual election of all directors (de-staggered)
- Option C describes a different governance mechanism
