# Question

A project requires an initial investment of $100,000 and is expected to generate cash flows of $40,000 per year for 4 years. If the required rate of return is 12%, the NPV of the project is closest to:

## Options

- **A**: $21,494
- **B**: $60,000
- **C**: -$1,506

## Answer

A

## Explanation

NPV = -Initial Investment + PV of Cash Flows

Using the annuity formula:
PV of annuity = PMT × [(1 - (1 + r)^-n) / r]
PV = $40,000 × [(1 - (1.12)^-4) / 0.12]
PV = $40,000 × [(1 - 0.6355) / 0.12]
PV = $40,000 × [0.3645 / 0.12]
PV = $40,000 × 3.0373
PV = $121,494

NPV = -$100,000 + $121,494 = $21,494

**Common mistakes:**
- Option B simply sums cash flows without discounting ($40,000 × 4 - $100,000)
- Option C uses an incorrect discount rate or calculation error
