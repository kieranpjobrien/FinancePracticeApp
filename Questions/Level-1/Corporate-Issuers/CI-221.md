# Question

The payback period method's main limitation is that it:

## Options

- **A**: Is too complex to calculate
- **B**: Ignores cash flows after the payback period and the time value of money
- **C**: Always gives the same result as NPV

## Answer

B

## Explanation

Payback period: time to recover initial investment. Ignores profitability beyond payback, ignores time value of money, and doesn't measure value creation. Simple and useful for liquidity assessment but shouldn't be primary criterion. Discounted payback addresses one limitation but not the other.

**Common mistakes:**
- Option A is opposite—payback is very simple to calculate
- Option C is incorrect—payback and NPV can rank projects differently
