# Question

The Adjusted Present Value (APV) method values a project by:

## Options

- **A**: Using a single WACC for all cash flows
- **B**: Valuing the unlevered project plus the present value of financing side effects
- **C**: Ignoring the effects of financing decisions

## Answer

B

## Explanation

APV: value project as if all-equity financed (using unlevered cost of capital), then add present value of financing effects (tax shields, flotation costs, subsidized loans). Useful when financing changes over time or project leverage differs from company. APV = Unlevered NPV + PV(Financing effects).

**Common mistakes:**
- Option A describes the WACC approach
- Option C ignores valuable financing effects
