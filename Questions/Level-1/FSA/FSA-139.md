# Question

Equipment has an original cost of $120,000 and accumulated depreciation of $45,000. The net book value is:

## Options

- **A**: $45,000
- **B**: $75,000
- **C**: $165,000

## Answer

B

## Explanation

Net book value = Original cost - Accumulated depreciation = $120,000 - $45,000 = $75,000. Accumulated depreciation is a contra-asset representing total depreciation expense recognized over the asset's life. Book value declines as the asset ages.

**Common mistakes:**
- Option A is just the accumulated depreciation
- Option C adds instead of subtracts
