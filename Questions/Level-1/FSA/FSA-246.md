# Question

Under IFRS, investment property can be measured using:

## Options

- **A**: Cost model only
- **B**: Fair value model only
- **C**: Either cost model or fair value model

## Answer

C

## Explanation

IFRS allows choice between cost model (depreciated cost, tested for impairment) or fair value model (changes in fair value through profit or loss). Once chosen, the policy applies to all investment property. US GAAP generally requires the cost model for real estate. Investment property is held for rental income or capital appreciation.

**Common mistakes:**
- Option A ignores the fair value option
- Option B ignores the cost model option
