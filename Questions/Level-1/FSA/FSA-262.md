# Question

Historical cost is preferred for measuring PPE because:

## Options

- **A**: It better reflects current economic value
- **B**: It is objective, verifiable, and based on actual transactions
- **C**: It requires regular revaluation

## Answer

B

## Explanation

Historical cost is reliable and verifiable—based on arm's length transactions with supporting documentation. It's objective, reducing subjectivity and manipulation risk. However, it may become less relevant over time as values change. IFRS allows revaluation as an alternative; US GAAP requires historical cost.

**Common mistakes:**
- Option A is an argument for fair value, not historical cost
- Option C describes fair value/revaluation approach
