# Question

After a 2-for-1 stock split, reported EPS for prior periods is:

## Options

- **A**: Unchanged
- **B**: Restated to reflect the split retroactively
- **C**: Doubled

## Answer

B

## Explanation

Stock splits and stock dividends require retroactive adjustment of all per-share data for comparability. A 2-for-1 split doubles shares, so prior EPS is halved to be comparable with current EPS. Without adjustment, EPS would appear to have declined dramatically, misleading users about actual performance trends.

**Common mistakes:**
- Option A would destroy comparability
- Option C reverses the adjustment direction
