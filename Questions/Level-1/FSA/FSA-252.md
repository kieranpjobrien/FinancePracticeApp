# Question

A perpetual inventory system:

## Options

- **A**: Calculates COGS only at year-end
- **B**: Updates inventory and COGS with each transaction
- **C**: Requires annual physical counts to determine ending inventory

## Answer

B

## Explanation

Perpetual systems maintain continuous inventory records, updating quantity and cost with each purchase and sale. This provides real-time inventory information and immediate COGS calculation. Physical counts still occur for verification but don't determine ending inventory. Periodic systems (Options A and C) update only at period end.

**Common mistakes:**
- Option A describes periodic inventory system
- Option C applies to periodic systems—perpetual uses records, counts verify
