# Question

Component depreciation requires:

## Options

- **A**: Using the same depreciation method for all assets
- **B**: Separately depreciating significant parts of an asset with different useful lives
- **C**: Depreciating assets only when they are in use

## Answer

B

## Explanation

Component depreciation (required under IFRS, permitted under US GAAP) requires identifying and separately depreciating significant parts of an asset that have different useful lives. For example, an aircraft's engines, fuselage, and interior may be depreciated separately. This better matches expense with the consumption of each component's economic benefits.

**Common mistakes:**
- Option A misunderstands the requirement
- Option C describes units-of-production method
