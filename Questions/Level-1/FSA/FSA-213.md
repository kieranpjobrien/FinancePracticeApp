# Question

Component depreciation under IFRS requires:

## Options

- **A**: Using the same depreciation method for all asset components
- **B**: Depreciating significant components with different useful lives separately
- **C**: Annual revaluation of each component

## Answer

B

## Explanation

IFRS requires component depreciation: if an asset has parts with significantly different useful lives, each component is depreciated separately. For example, an aircraft's engines, airframe, and interior might have different lives. US GAAP permits but doesn't require component depreciation. This improves matching of expense with benefit.

**Common mistakes:**
- Option A contradicts the purpose of component depreciation
- Option C confuses depreciation with revaluation requirements
