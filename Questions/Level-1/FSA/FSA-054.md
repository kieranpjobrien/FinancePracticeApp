# Question

Equipment with a cost of $80,000 and accumulated depreciation of $50,000 is sold for $35,000. The gain or loss on sale is:

## Options

- **A**: $5,000 gain
- **B**: $5,000 loss
- **C**: $15,000 loss

## Answer

A

## Explanation

Book value = Cost - Accumulated depreciation = $80,000 - $50,000 = $30,000. Gain/loss = Sale proceeds - Book value = $35,000 - $30,000 = $5,000 gain. The asset sold for more than its book value, resulting in a gain reported on the income statement.

**Common mistakes:**
- Option B reverses the calculation
- Option C uses cost instead of book value
