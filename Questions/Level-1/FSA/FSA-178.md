# Question

A defined benefit pension plan is underfunded when:

## Options

- **A**: The fair value of plan assets exceeds the projected benefit obligation
- **B**: The projected benefit obligation exceeds the fair value of plan assets
- **C**: Pension expense exceeds cash contributions

## Answer

B

## Explanation

Funded status = Fair value of plan assets - Projected benefit obligation (PBO). When PBO > assets, the plan is underfunded, reported as a net pension liability. When assets > PBO, it's overfunded, shown as a net pension asset. This funded status appears on the balance sheet.

**Common mistakes:**
- Option A describes an overfunded plan
- Option C relates to cash flow, not funded status
