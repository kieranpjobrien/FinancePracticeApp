# Question

For a manufacturer, inventory includes:

## Options

- **A**: Only finished goods ready for sale
- **B**: Raw materials, work-in-process, and finished goods
- **C**: Only purchased merchandise

## Answer

B

## Explanation

Manufacturing inventory comprises three stages: raw materials (inputs awaiting production), work-in-process (partially completed goods), and finished goods (completed, ready for sale). Retailers have only merchandise inventory. The inventory note discloses amounts in each category, providing insight into production cycle and potential obsolescence.

**Common mistakes:**
- Option A omits raw materials and WIP
- Option C describes retail/merchandising inventory only
