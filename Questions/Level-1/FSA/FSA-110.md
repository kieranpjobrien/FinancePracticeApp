# Question

Under IFRS, if the circumstances that caused an inventory writedown no longer exist:

## Options

- **A**: No adjustment is made; writedowns are permanent
- **B**: The writedown can be reversed up to the original cost
- **C**: The inventory is revalued to current market price

## Answer

B

## Explanation

IFRS allows reversal of inventory writedowns if NRV recovers, but only up to original cost—never above. This differs from US GAAP, which prohibits reversals (writedowns are permanent). Reversal is recognized in profit or loss, reducing COGS in the period of reversal.

**Common mistakes:**
- Option A describes US GAAP treatment
- Option C would allow writeup above cost, which is prohibited
