# Question

Accumulated depreciation is:

## Options

- **A**: An expense account
- **B**: A contra-asset account
- **C**: A liability account

## Answer

B

## Explanation

Accumulated depreciation is a contra-asset that offsets the related asset account. It represents total depreciation taken since acquisition. Net book value = Cost - Accumulated depreciation. While depreciation expense reduces income, accumulated depreciation reduces assets. The credit balance is "contra" to the asset's debit balance.

**Common mistakes:**
- Option A confuses stock (accumulated) with flow (expense)
- Option C misclassifies the balance sheet location
