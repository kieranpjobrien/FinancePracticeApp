# Question

A company has COGS of $720,000, beginning inventory of $90,000, and ending inventory of $110,000. Inventory turnover is:

## Options

- **A**: 6.5 times
- **B**: 7.2 times
- **C**: 8.0 times

## Answer

B

## Explanation

Average inventory = (Beginning + Ending) / 2 = ($90,000 + $110,000) / 2 = $100,000. Inventory turnover = COGS / Average inventory = $720,000 / $100,000 = 7.2 times. Higher turnover generally indicates more efficient inventory management.

**Common mistakes:**
- Option A uses ending inventory only: $720,000 / $110,000 = 6.5
- Option C uses beginning inventory only: $720,000 / $90,000 = 8.0
