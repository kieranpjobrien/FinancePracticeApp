# Question

Sensitivity analysis in capital budgeting:

## Options

- **A**: Determines the single most likely outcome
- **B**: Shows how NPV changes when one input variable is changed
- **C**: Eliminates all project risk

## Answer

B

## Explanation

Sensitivity analysis varies one input at a time (sales volume, price, costs, discount rate) to see how NPV changes. It identifies which variables have the greatest impact—key risk drivers deserve more attention in forecasting and management. It doesn't show joint variable effects or probabilities.

**Common mistakes:**
- Option A describes point estimation, not sensitivity
- Option C overstates the benefit—it identifies, not eliminates, risk
