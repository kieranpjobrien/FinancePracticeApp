# Question

Preferred stock pays a $5 annual dividend and is priced at $62.50. The cost of preferred stock is:

## Options

- **A**: 5%
- **B**: 8%
- **C**: 12.5%

## Answer

B

## Explanation

Cost of preferred stock = Dividend / Price = $5 / $62.50 = 8%. Preferred stock is perpetual, so the cost equals the dividend yield. Unlike debt, preferred dividends aren't tax-deductible, so no tax adjustment is needed.

**Common mistakes:**
- Option A uses an incorrect calculation
- Option C reverses the ratio
