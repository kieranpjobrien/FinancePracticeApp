# Question

An option to expand is most valuable when:

## Options

- **A**: Project outcomes are certain
- **B**: There is high uncertainty with potential upside
- **C**: The project has a negative NPV

## Answer

B

## Explanation

Expansion options have value when future demand is uncertain—if demand materializes, the company can invest more; if not, it limits exposure. Higher uncertainty increases option value (similar to financial options). Certain outcomes don't benefit from flexibility; negative NPV projects may still have option value if upside exists.

**Common mistakes:**
- Option A makes flexibility valueless—no benefit from waiting
- Option C doesn't guarantee option value—depends on upside potential
