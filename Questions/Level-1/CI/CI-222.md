# Question

When evaluating a new product that will reduce sales of existing products, the lost contribution margin should be:

## Options

- **A**: Ignored because it's an existing product issue
- **B**: Included as a negative incremental cash flow in the new project analysis
- **C**: Added to the new product's revenues

## Answer

B

## Explanation

Cannibalization/erosion: new project reduces cash flows from existing operations. The lost contribution margin is an incremental effect of the new project decision and must be deducted from project cash flows. Exception: if competitors would capture those sales anyway (defensive investment).

**Common mistakes:**
- Option A ignores relevant incremental effects
- Option C would double-count or incorrectly add
