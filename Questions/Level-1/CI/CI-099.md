# Question

Monte Carlo simulation in capital budgeting:

## Options

- **A**: Provides a single point estimate of NPV
- **B**: Generates a probability distribution of outcomes by randomly sampling input variables
- **C**: Eliminates the need for sensitivity analysis

## Answer

B

## Explanation

Monte Carlo simulation: assigns probability distributions to input variables, randomly samples from each, calculates NPV, and repeats thousands of times. The result is an NPV distribution showing probability of various outcomes, providing richer risk information than point estimates or scenarios.

**Common mistakes:**
- Option A describes traditional NPV analysis
- Option C—simulation complements, doesn't replace, sensitivity analysis
