# Question

Financial leverage amplifies:

## Options

- **A**: Only positive returns to shareholders
- **B**: Both positive and negative returns to shareholders
- **C**: Operating income variability

## Answer

B

## Explanation

Financial leverage: fixed interest payments create leverage on equity returns. When EBIT exceeds interest, shareholders benefit disproportionately. When EBIT falls, shareholders bear concentrated losses. Amplifies both upside and downside—increases return volatility and equity risk.

**Common mistakes:**
- Option A ignores downside amplification
- Option C describes operating leverage, not financial leverage
