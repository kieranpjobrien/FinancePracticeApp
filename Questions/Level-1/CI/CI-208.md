# Question

A target payout ratio approach to dividend policy:

## Options

- **A**: Keeps dividends constant regardless of earnings
- **B**: Adjusts dividends gradually toward a long-term payout percentage of earnings
- **C**: Pays out all earnings as dividends each period

## Answer

B

## Explanation

Target payout ratio: company has long-term dividend/earnings target but adjusts gradually. Avoids large dividend changes that might be reversed. Lintner model: dividends change slowly toward target, with partial adjustment each period. Balances signaling concerns with target payout objectives.

**Common mistakes:**
- Option A describes stable dollar dividend policy
- Option C describes 100% payout with no smoothing
