# Question

Lease financing affects capital structure analysis because:

## Options

- **A**: Operating leases have no financial impact
- **B**: Finance leases create debt-like obligations that should be included in leverage calculations
- **C**: All leases are treated as equity

## Answer

B

## Explanation

Finance leases (and now operating leases under new standards) create right-of-use assets and lease liabilities—debt-like obligations. Analysts should include capitalized lease obligations in leverage ratios for accurate comparisons. This affects debt capacity and coverage ratios.

**Common mistakes:**
- Option A ignores operating lease obligations
- Option C misclassifies lease obligations
