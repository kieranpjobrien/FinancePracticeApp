# Question

According to Modigliani-Miller with taxes, firm value increases with debt because:

## Options

- **A**: Debt is riskier than equity
- **B**: Interest payments create a tax shield
- **C**: Debt holders require lower returns

## Answer

B

## Explanation

With corporate taxes, interest is tax-deductible, creating a tax shield = Tax rate × Interest. The present value of tax shields increases firm value. MM with taxes: VL = VU + (T × D). However, this ignores financial distress costs that offset tax benefits at high leverage.

**Common mistakes:**
- Option A describes risk, not value creation
- Option C is true but doesn't explain the value increase from leverage
