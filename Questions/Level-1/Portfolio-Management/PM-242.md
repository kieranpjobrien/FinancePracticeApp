# Question

Total portfolio variance can be decomposed into:

## Options

- **A**: Only systematic variance
- **B**: Systematic variance plus unsystematic variance
- **C**: Expected return plus unexpected return

## Answer

B

## Explanation

Variance decomposition:

**Total variance = Systematic + Unsystematic**

**Systematic (market) variance:**
- β² × σ²market
- Non-diversifiable
- Driven by market factors

**Unsystematic (residual) variance:**
- σ²(ε)
- Diversifiable
- Firm-specific factors

**Formula:**
σ²total = β²σ²m + σ²ε

**Diversification impact:**
- As portfolio diversifies, unsystematic → 0
- Total variance approaches systematic variance

**Common mistakes:**
- Option A: Ignores unsystematic component
- Option C: Returns, not variances
