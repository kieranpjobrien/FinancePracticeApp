# Question

Portfolio A has a return of 14%, beta of 1.2, and standard deviation of 22%. The risk-free rate is 4% and market return is 11%. Portfolio A's Treynor ratio is closest to:

## Options

- **A**: 0.045
- **B**: 0.083
- **C**: 0.455

## Answer

B

## Explanation

The Treynor ratio measures excess return per unit of systematic risk:

Treynor Ratio = (Rp - Rf) / βp

Where:
- Rp = Portfolio return = 14%
- Rf = Risk-free rate = 4%
- βp = Portfolio beta = 1.2

Treynor Ratio = (14% - 4%) / 1.2
Treynor Ratio = 10% / 1.2
Treynor Ratio = 0.0833 ≈ 0.083

Compare to market's Treynor ratio:
Market Treynor = (11% - 4%) / 1.0 = 7% = 0.07

Portfolio A outperformed on a systematic risk-adjusted basis (0.083 > 0.070).

**Common mistakes:**
- Option A: Uses standard deviation instead of beta (Sharpe ratio approach)
- Option C: Uses percentage values incorrectly in calculation
