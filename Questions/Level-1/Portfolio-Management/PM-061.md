# Question

A foundation has assets of $50 million and must distribute 5% annually to maintain tax-exempt status. Administrative costs are 0.5% annually. Expected inflation is 2.5%. The foundation's required nominal return to maintain real purchasing power is closest to:

## Options

- **A**: 5.5%
- **B**: 8.0%
- **C**: 8.2%

## Answer

C

## Explanation

Required nominal return must cover:
1. Distribution requirement: 5.0%
2. Administrative costs: 0.5%
3. Inflation adjustment: 2.5%

Method 1 - Additive approximation:
Required return ≈ 5.0% + 0.5% + 2.5% = 8.0%

Method 2 - Compounding (more precise):
(1 + spending)(1 + costs)(1 + inflation) - 1
(1.05)(1.005)(1.025) - 1
= 1.0823 - 1
= 8.2%

The foundation needs 8.2% nominal return to:
- Meet 5% distribution requirement
- Cover 0.5% costs
- Preserve real purchasing power (2.5% inflation)

**Common mistakes:**
- Option A: Only includes distribution and costs, ignores inflation
- Option B: Uses additive method (less precise)
