# Question

The Security Market Line (SML) plots the relationship between:

## Options

- **A**: Expected return and total risk (standard deviation)
- **B**: Expected return and systematic risk (beta)
- **C**: Historical return and historical volatility

## Answer

B

## Explanation

The Security Market Line (SML):

- Graphical representation of CAPM
- X-axis: Beta (systematic risk)
- Y-axis: Expected return
- Y-intercept: Risk-free rate
- Slope: Market risk premium [E(Rm) - Rf]

Equation: E(Ri) = Rf + βi[E(Rm) - Rf]

The SML applies to all securities and portfolios, not just efficient ones. Securities above the SML are undervalued (positive alpha), below are overvalued (negative alpha).

**Common mistakes:**
- Option A: This describes the Capital Market Line (CML), not the SML
- Option C: SML uses expected (forward-looking) values, not historical data
