# Question

A key difference between the Capital Market Line (CML) and Security Market Line (SML) is that:

## Options

- **A**: CML uses beta while SML uses standard deviation
- **B**: CML applies only to efficient portfolios while SML applies to all securities
- **C**: CML is used for individual securities while SML is for portfolios only

## Answer

B

## Explanation

CML vs. SML:

**Capital Market Line (CML):**
- X-axis: Total risk (standard deviation)
- Y-axis: Expected return
- Applies to: Efficient portfolios only
- Portfolios combining risk-free asset and market portfolio
- Inefficient portfolios plot below the CML

**Security Market Line (SML):**
- X-axis: Systematic risk (beta)
- Y-axis: Expected return
- Applies to: All securities and portfolios
- Securities above SML are undervalued (positive alpha)
- Securities below SML are overvalued (negative alpha)

**Key insight:**
- Only efficient portfolios lie on CML
- All fairly priced securities lie on SML

**Common mistakes:**
- Option A: Reversed; CML uses standard deviation, SML uses beta
- Option C: Reversed; SML applies to individual securities
