# Question

Model risk in portfolio management arises from:

## Options

- **A**: Accurate models being applied correctly
- **B**: Errors in model design, implementation, or inappropriate application
- **C**: Market prices moving against portfolio positions

## Answer

B

## Explanation

Model risk:

**Definition:**
- Risk of loss from using flawed models
- Includes design errors, implementation bugs, misuse

**Sources:**
1. **Model design:** Wrong assumptions, oversimplification
2. **Implementation:** Programming errors, data issues
3. **Application:** Using models outside intended scope

**Examples:**
- VaR model assuming normal returns (fat tails ignored)
- Correlation assumptions breaking during crises
- Valuation models with incorrect inputs

**Mitigation:**
- Independent model validation
- Backtesting and stress testing
- Documentation and review
- Model governance framework

**Common mistakes:**
- Option A: Correct application doesn't create model risk
- Option C: This describes market risk, not model risk
