# Question

Jensen's alpha measures:

## Options

- **A**: The portfolio's excess return relative to its total risk
- **B**: The portfolio's actual return minus its CAPM expected return
- **C**: The portfolio's return divided by its beta

## Answer

B

## Explanation

Jensen's alpha (αp):

αp = Rp - [Rf + βp(Rm - Rf)]
αp = Actual return - CAPM expected return

Interpretation:
- α > 0: Portfolio outperformed CAPM expectation (skilled management or luck)
- α = 0: Portfolio performed as expected given its beta
- α < 0: Portfolio underperformed CAPM expectation

Jensen's alpha:
- Based on systematic risk (beta)
- Appropriate for diversified portfolios
- Absolute measure (in percentage terms)

**Common mistakes:**
- Option A: This describes the Sharpe ratio concept (return per unit of total risk)
- Option C: Dividing return by beta doesn't account for the risk-free rate
