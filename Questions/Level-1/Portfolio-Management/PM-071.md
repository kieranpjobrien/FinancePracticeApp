# Question

Scenario analysis differs from sensitivity analysis in that scenario analysis:

## Options

- **A**: Changes only one variable at a time
- **B**: Examines the impact of simultaneous changes in multiple variables
- **C**: Only considers best-case scenarios

## Answer

B

## Explanation

Scenario vs. Sensitivity Analysis:

**Sensitivity analysis:**
- Changes one variable at a time
- Holds other variables constant
- Shows impact of individual factors
- Example: How does a 1% interest rate change affect portfolio value?

**Scenario analysis:**
- Changes multiple variables simultaneously
- Models coherent scenarios (recession, inflation shock, etc.)
- More realistic as variables often move together
- Example: Recession scenario with falling GDP, rising unemployment, lower interest rates, widening credit spreads

**Stress testing:**
- Extreme scenario analysis
- Tests portfolio under severe conditions
- Often uses historical crisis scenarios (2008, COVID)

**Common mistakes:**
- Option A: This describes sensitivity analysis
- Option C: Scenario analysis includes adverse scenarios, not just positive ones
