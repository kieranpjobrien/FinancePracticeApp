# Question

The risk-free rate is 4% and the expected market return is 11%. Using CAPM, the required return for a stock with a beta of 1.5 is closest to:

## Options

- **A**: 14.5%
- **B**: 15.5%
- **C**: 20.5%

## Answer

A

## Explanation

Using the Capital Asset Pricing Model (CAPM):

E(Ri) = Rf + βi × [E(Rm) - Rf]

Where:
- Rf = Risk-free rate = 4%
- E(Rm) = Expected market return = 11%
- βi = Beta = 1.5
- Market risk premium = E(Rm) - Rf = 11% - 4% = 7%

E(Ri) = 4% + 1.5 × 7%
E(Ri) = 4% + 10.5%
E(Ri) = 14.5%

**Common mistakes:**
- Option B: May add the risk-free rate incorrectly or use wrong market risk premium
- Option C: Multiplies beta by the total market return (1.5 × 11% + 4%) instead of the market risk premium
