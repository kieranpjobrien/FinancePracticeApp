# Question

An appropriate benchmark for a small-cap value equity manager would be:

## Options

- **A**: S&P 500 Index
- **B**: Russell 2000 Value Index
- **C**: Bloomberg Aggregate Bond Index

## Answer

B

## Explanation

Benchmark selection:

**Matching criteria:**
- Same asset class (equities)
- Same style (value)
- Same capitalization (small-cap)
- Investable

**Why Russell 2000 Value:**
- Small-cap universe
- Value-screened constituents
- Matches manager's mandate

**Why others fail:**
- S&P 500: Large-cap, not style-specific
- Bond index: Wrong asset class

**Good benchmark properties:**
- Unambiguous
- Investable
- Measurable
- Appropriate
- Reflective of current views
- Specified in advance
- Owned by manager

**Common mistakes:**
- Option A: Different size and style
- Option C: Different asset class entirely
