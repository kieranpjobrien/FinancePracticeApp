# Question

A stock has a market model regression equation: Rᵢ = 2% + 1.3Rₘ. If the market returns 10%, the stock's expected return is:

## Options

- **A**: 12%
- **B**: 13%
- **C**: 15%

## Answer

C

## Explanation

Market model calculation:

**Market model:**
Rᵢ = α + βRₘ

**Given:**
- Alpha (α) = 2%
- Beta (β) = 1.3
- Market return (Rₘ) = 10%

**Calculation:**
Rᵢ = 2% + 1.3 × 10%
Rᵢ = 2% + 13%
Rᵢ = 15%

**Interpretation:**
- 2% alpha: Excess return independent of market
- 1.3 beta: 130% market sensitivity
- 13%: Return from market exposure
- 15%: Total expected return

**Note:**
- Alpha represents skill or mispricing
- Beta captures systematic risk

**Common mistakes:**
- Option A: Forgot alpha component
- Option B: Used beta of 1.1 or miscalculated
