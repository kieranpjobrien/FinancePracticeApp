# Question

An investment earned a nominal return of 8% during a year when inflation was 3%. The real return is closest to:

## Options

- **A**: 4.9%
- **B**: 5.0%
- **C**: 11.0%

## Answer

A

## Explanation

Real return calculation:

**Exact formula (Fisher equation):**
(1 + Nominal) = (1 + Real) × (1 + Inflation)

Solving for real return:
Real = [(1 + Nominal) / (1 + Inflation)] - 1
Real = (1.08 / 1.03) - 1
Real = 1.0485 - 1
Real = 0.0485 = 4.85% ≈ 4.9%

**Approximation:**
Real ≈ Nominal - Inflation
Real ≈ 8% - 3% = 5%

The approximation is close but slightly overstates real return.

**Common mistakes:**
- Option B: Uses the approximation (less precise)
- Option C: Adds instead of subtracting inflation
