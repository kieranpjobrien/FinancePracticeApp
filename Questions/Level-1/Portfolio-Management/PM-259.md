# Question

An asset has a 60% chance of returning 15% and a 40% chance of returning -5%. The expected return is:

## Options

- **A**: 5%
- **B**: 7%
- **C**: 10%

## Answer

B

## Explanation

Expected return calculation:

**Formula:**
E(R) = Σ(Probability × Return)

**Calculation:**
E(R) = (0.60 × 15%) + (0.40 × -5%)
E(R) = 9% + (-2%)
E(R) = 7%

**Interpretation:**
- Probability-weighted average outcome
- Not guaranteed return
- Useful for comparing investments

**Properties:**
- Accounts for all scenarios
- Weights by likelihood
- Central tendency measure

**Common mistakes:**
- Option A: May have calculation error
- Option C: Simple average (15-5)/2, ignoring probabilities
