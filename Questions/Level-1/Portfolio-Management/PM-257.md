# Question

Stress testing a portfolio involves:

## Options

- **A**: Only using historical average returns
- **B**: Evaluating portfolio performance under extreme but plausible scenarios
- **C**: Testing software for bugs

## Answer

B

## Explanation

Stress testing:

**Purpose:**
- Assess impact of extreme events
- Identify vulnerabilities
- Prepare for crisis scenarios
- Complement VaR analysis

**Types of scenarios:**
- Historical: Past crises (2008, COVID)
- Hypothetical: Constructed scenarios
- Reverse: What could cause specific loss

**Scenario elements:**
- Large market moves
- Correlation changes
- Liquidity stress
- Multiple simultaneous shocks

**Uses:**
- Risk limits
- Capital planning
- Contingency planning
- Regulatory requirement

**Common mistakes:**
- Option A: Stress tests use extreme, not average, scenarios
- Option C: Different type of testing
