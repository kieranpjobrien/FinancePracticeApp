# Question

Active return is defined as:

## Options

- **A**: The portfolio's absolute return
- **B**: The portfolio return minus the benchmark return
- **C**: The portfolio return minus the risk-free rate

## Answer

B

## Explanation

Active return:

**Definition:**
Active Return = Portfolio Return - Benchmark Return

**Also known as:**
- Excess return (relative to benchmark)
- Alpha (in some contexts)
- Value added

**Interpretation:**
- Positive: Outperformed benchmark
- Negative: Underperformed benchmark
- Zero: Matched benchmark (passive approach)

**Uses:**
- Evaluate active manager skill
- Calculate information ratio
- Justify active management fees

**Distinction:**
- Active return: vs. benchmark
- Excess return (absolute): vs. risk-free rate

**Common mistakes:**
- Option A: Absolute return doesn't reference benchmark
- Option C: This is excess return over risk-free, used in Sharpe ratio
