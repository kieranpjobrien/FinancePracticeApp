# Question

The global minimum variance portfolio is best described as the portfolio:

## Options

- **A**: With the highest Sharpe ratio on the efficient frontier
- **B**: At the leftmost point of the Markowitz efficient frontier
- **C**: That consists entirely of risk-free assets

## Answer

B

## Explanation

The global minimum variance (GMV) portfolio:

- Has the lowest possible variance (risk) among all portfolios of risky assets
- Located at the leftmost point of the minimum variance frontier
- Marks the boundary between the efficient frontier (upper portion) and inefficient frontier (lower portion)
- Contains only risky assets (no risk-free asset)

Key characteristics:
- All portfolios above GMV on the frontier are efficient
- All portfolios below GMV have the same risk but lower return (inefficient)
- GMV is not necessarily the optimal portfolio for any investor

**Common mistakes:**
- Option A: The highest Sharpe ratio portfolio is the tangency portfolio (market portfolio), not GMV
- Option C: GMV consists of risky assets; the risk-free asset is not part of the Markowitz efficient frontier
