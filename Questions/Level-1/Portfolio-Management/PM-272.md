# Question

A security plotting above the Security Market Line (SML) is:

## Options

- **A**: Overvalued and should be sold
- **B**: Undervalued and should be bought
- **C**: Fairly valued based on its risk

## Answer

B

## Explanation

Security Market Line analysis:

**Above SML:**
- Expected return > Required return
- Positive alpha
- Undervalued (buy signal)
- Price should rise to fair value

**Below SML:**
- Expected return < Required return
- Negative alpha
- Overvalued (sell signal)
- Price should fall to fair value

**On SML:**
- Expected return = Required return
- Zero alpha
- Fairly valued

**Trading implication:**
- Buy securities above SML
- Sell securities below SML
- Arbitrage drives prices to equilibrium

**Common mistakes:**
- Option A: Above SML means undervalued, not overvalued
- Option C: Fairly valued securities plot ON the SML
