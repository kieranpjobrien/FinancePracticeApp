# Question

An asset with a negative beta would be valuable in a portfolio because it:

## Options

- **A**: Always generates positive returns
- **B**: Tends to rise when the overall market falls
- **C**: Has higher expected returns than the market

## Answer

B

## Explanation

Negative beta:

**Definition:**
- Negative correlation with market returns
- Moves opposite to market direction

**Examples (rare):**
- Gold (sometimes slightly negative)
- Certain hedge fund strategies
- Put options

**Portfolio benefit:**
- Provides hedging during market downturns
- Reduces portfolio beta
- Diversification benefit

**Expected return consideration:**
- CAPM implies negative beta = lower expected return than Rf
- Investors "pay" for insurance via lower return
- E(R) = Rf + (-β) × MRP < Rf

**Common mistakes:**
- Option A: Negative beta doesn't guarantee positive returns
- Option C: Negative beta implies lower, not higher, expected returns
