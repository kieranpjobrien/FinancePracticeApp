# Question

The tangency portfolio is significant because it:

## Options

- **A**: Has zero risk
- **B**: Maximizes the Sharpe ratio among all risky portfolios
- **C**: Contains only risk-free assets

## Answer

B

## Explanation

Tangency portfolio:

**Definition:**
- Point where CML touches efficient frontier
- Optimal risky portfolio
- Combines with Rf for all efficient portfolios

**Properties:**
- Maximum Sharpe ratio
- Best risk-return trade-off
- In CAPM, equals market portfolio
- All investors hold this (two-fund separation)

**Why maximum Sharpe:**
- CML slope = Sharpe ratio
- Tangency maximizes slope from Rf to frontier

**Allocation:**
- Conservative: Tangency + Rf
- Aggressive: Leveraged tangency

**Common mistakes:**
- Option A: Tangency has risk (it's on risky frontier)
- Option C: Contains only risky assets
