# Question

Extreme Value Theory (EVT) is used in risk management to:

## Options

- **A**: Model average returns over normal periods
- **B**: Model the tails of return distributions for extreme event estimation
- **C**: Calculate standard portfolio variance

## Answer

B

## Explanation

Extreme Value Theory:

**Purpose:**
- Model extreme events (tail behavior)
- Estimate probabilities of rare events
- Improve VaR for extreme quantiles

**Why needed:**
- Normal distribution underestimates tails
- Extreme events more common than normal suggests
- Standard VaR inadequate for tail risk

**Approaches:**
- Block maxima (GEV distribution)
- Peaks-over-threshold (GPD)
- Focus on observations beyond threshold

**Applications:**
- Tail VaR estimation
- Stress testing
- Insurance pricing
- Natural disaster modeling

**Limitations:**
- Requires sufficient tail data
- Parameter estimation uncertainty
- Model risk

**Common mistakes:**
- Option A: EVT focuses on extremes, not averages
- Option C: Variance doesn't capture tail behavior adequately
