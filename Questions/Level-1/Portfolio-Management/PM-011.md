# Question

A portfolio has an expected return of 12%, a standard deviation of 20%, and the risk-free rate is 3%. The Sharpe ratio of this portfolio is closest to:

## Options

- **A**: 0.45
- **B**: 0.60
- **C**: 4.00

## Answer

A

## Explanation

The Sharpe ratio measures excess return per unit of total risk:

Sharpe Ratio = (Rp - Rf) / σp

Where:
- Rp = Portfolio return = 12%
- Rf = Risk-free rate = 3%
- σp = Portfolio standard deviation = 20%

Sharpe Ratio = (12% - 3%) / 20%
Sharpe Ratio = 9% / 20%
Sharpe Ratio = 0.45

A Sharpe ratio of 0.45 means the portfolio earns 0.45% of excess return for each 1% of risk taken.

**Common mistakes:**
- Option B: Uses incorrect values or calculation errors
- Option C: Divides return by risk-free rate (12% / 3%) instead of using the correct formula
