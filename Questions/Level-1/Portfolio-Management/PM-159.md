# Question

The Black-Litterman model improves on traditional mean-variance optimization by:

## Options

- **A**: Eliminating the need for expected return estimates
- **B**: Combining market equilibrium returns with investor views
- **C**: Using only historical data without any forward estimates

## Answer

B

## Explanation

Black-Litterman model:

**Problems with traditional MVO:**
- Extreme sensitivity to expected return inputs
- Unintuitive, concentrated portfolios
- Estimation error maximized

**Black-Litterman approach:**
1. Start with equilibrium (implied) returns from market portfolio
2. Investor expresses views with confidence levels
3. Blends equilibrium and views mathematically
4. Results in more stable, intuitive portfolios

**Key features:**
- Equilibrium as neutral starting point
- Views can be absolute or relative
- Confidence weighting on views
- Reduces input sensitivity

**Benefits:**
- More intuitive portfolios
- Can incorporate views systematically
- Better behaved optimization

**Common mistakes:**
- Option A: Still uses expected returns (equilibrium + views)
- Option C: Combines equilibrium with forward views
