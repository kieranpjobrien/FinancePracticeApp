# Question

Portable alpha involves:

## Options

- **A**: Holding a single diversified index fund
- **B**: Separating alpha generation from market exposure using derivatives
- **C**: Investing only in the highest-alpha stocks

## Answer

B

## Explanation

Portable alpha:

**Concept:**
- Separate alpha from beta
- Generate alpha from one source
- Get beta exposure from another (derivatives)

**Implementation:**
1. Invest with alpha-generating manager
2. Overlay with futures/swaps for desired beta
3. Combine for total return

**Example:**
- $100M to market-neutral hedge fund (alpha only)
- Buy $100M S&P futures (beta exposure)
- Result: S&P return + hedge fund alpha

**Benefits:**
- Access best alpha generators
- Choose any beta exposure
- More efficient alpha extraction

**Requirements:**
- Skill in identifying alpha sources
- Derivatives expertise
- Manage cash/collateral

**Common mistakes:**
- Option A: Index funds provide only beta, no alpha
- Option C: This is concentrated stock picking, not portable alpha
