# Question

Conflict of interest disclosures should be made:

## Options

- **A**: Only annually in formal documents
- **B**: When conflicts arise and updated as circumstances change
- **C**: Only when clients specifically request them

## Answer

B

## Explanation

Standard VI(A) requires disclosure of conflicts in a timely manner. Initial disclosure when conflicts arise and updates as circumstances change ensures affected parties have current, relevant information.

**Common mistakes:**
- Option A may allow stale or delayed disclosures
- Option C fails the proactive disclosure requirement
