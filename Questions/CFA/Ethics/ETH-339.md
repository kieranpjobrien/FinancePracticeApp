# Question

Valuing illiquid alternative investments for GIPS purposes:

## Options

- **A**: Should use purchase price indefinitely
- **B**: Requires fair value estimates updated at least annually
- **C**: Is exempt from GIPS valuation requirements

## Answer

B

## Explanation

GIPS requires fair value for all assets, including illiquid investments. Fair value must be estimated using reasonable methods and updated regularly, at least annually, even when market prices are unavailable.

**Common mistakes:**
- Option A fails to update valuations
- Option C incorrectly exempts illiquid assets
