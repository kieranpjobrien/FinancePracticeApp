# Question

A client with a conservative IPS insists on purchasing a highly speculative stock. The portfolio manager should:

## Options

- **A**: Execute the trade because the client has the final decision
- **B**: Refuse to execute the trade under any circumstances
- **C**: Document the discussion, explain the unsuitability, and if the client persists, consider updating the IPS or declining the instruction

## Answer

C

## Explanation

When clients request unsuitable investments, managers should document concerns, explain the risks, and attempt to update the IPS if appropriate. Simply refusing or blindly executing without proper process fails to properly address the suitability conflict.

**Common mistakes:**
- Option A ignores the manager's suitability obligations
- Option B may not allow for legitimate IPS updates when client circumstances change
