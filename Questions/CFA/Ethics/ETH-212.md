# Question

A portfolio significantly underperformed and the client terminated the relationship. Under GIPS:

## Options

- **A**: The portfolio may be excluded from composite returns
- **B**: The portfolio's historical returns must remain in the composite for the period it was managed
- **C**: The composite should be disbanded if any portfolio is terminated

## Answer

B

## Explanation

GIPS prohibits survivorship bias by requiring that terminated portfolios' historical returns remain in composite calculations for the periods they were managed. Cherry-picking by excluding poor performers is strictly prohibited.

**Common mistakes:**
- Option A allows cherry-picking that would inflate historical performance
- Option C overreacts to normal client turnover
