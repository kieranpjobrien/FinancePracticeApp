# Question

Under GIPS, portfolio valuations should be based on:

## Options

- **A**: Original cost or book value to maintain consistency
- **B**: Fair value as of the valuation date
- **C**: The higher of cost or market value

## Answer

B

## Explanation

GIPS requires portfolios to be valued at fair value. Fair value represents the price that would be received to sell an asset in an orderly transaction between market participants. This provides the most accurate representation of portfolio value for performance calculation purposes.

**Common mistakes:**
- Option A uses historical cost, which may not reflect current values
- Option C creates an upward bias by always selecting the higher value
