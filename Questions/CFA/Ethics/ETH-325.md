# Question

A manager compares equity portfolio returns to a bond index. Under Standard III(D):

## Options

- **A**: Any benchmark comparison provides useful context
- **B**: Benchmark comparisons should be to relevant and appropriate benchmarks
- **C**: Benchmark comparisons are optional

## Answer

B

## Explanation

Standard III(D) requires fair and accurate performance presentation. Using an inappropriate benchmark creates a misleading impression of relative performance and violates the Standard.

**Common mistakes:**
- Option A ignores the importance of benchmark appropriateness
- Option C understates the value of proper benchmarking
