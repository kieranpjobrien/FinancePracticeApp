# Question

A portfolio manager receives an all-expenses-paid trip to an exclusive resort from a broker seeking increased trading business. Under Standard I(B):

## Options

- **A**: This is acceptable if disclosed to the employer
- **B**: This likely compromises independence and should be declined
- **C**: The trip can be accepted if no explicit quid pro quo is discussed

## Answer

B

## Explanation

Standard I(B) requires members to maintain independence and objectivity. Lavish gifts such as expensive trips are likely to create actual or perceived bias and should generally be declined. Disclosure alone does not cure the compromise to independence.

**Common mistakes:**
- Option A assumes disclosure remedies independence concerns
- Option C ignores that influence can occur without explicit agreements
