# Question

A composite's investment strategy changes significantly. Under GIPS:

## Options

- **A**: Historical returns can be restated to reflect the new strategy
- **B**: A new composite should typically be created for the new strategy
- **C**: Strategy changes have no effect on composite construction

## Answer

B

## Explanation

Significant strategy changes typically require creating a new composite because the historical returns no longer represent the new approach. Restating history would misrepresent what the old strategy actually achieved.

**Common mistakes:**
- Option A allows misleading restatement
- Option C ignores the impact of strategy changes
