# Question

The Code of Ethics requires members to use:

## Options

- **A**: Only employer-approved investment approaches
- **B**: Independent professional judgment in conducting activities
- **C**: Whatever methods generate the highest returns

## Answer

B

## Explanation

The Code of Ethics emphasizes using independent professional judgment. Members must exercise their own analytical skills rather than simply following others' directives or pursuing returns without ethical consideration.

**Common mistakes:**
- Option A subordinates judgment to employer dictates
- Option C prioritizes returns over ethical judgment
