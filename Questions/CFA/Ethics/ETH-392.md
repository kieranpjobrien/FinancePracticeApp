# Question

A member practicing in multiple countries finds that local law in one jurisdiction is less strict than the Code and Standards. The member should:

## Options

- **A**: Follow the less strict local law to maintain competitive advantage
- **B**: Follow the stricter Code and Standards
- **C**: Average the requirements between jurisdictions

## Answer

B

## Explanation

Standard I(A) requires following the stricter of applicable law or the Code and Standards. When local law is more permissive, members must still adhere to the higher ethical requirements of the Code and Standards.

**Common mistakes:**
- Option A uses local law to justify lower standards
- Option C creates an undefined middle ground
