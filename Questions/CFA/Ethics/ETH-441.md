# Question

A portfolio manager moves to a new firm and wants to bring her performance track record. Under GIPS portability requirements:

## Options

- **A**: The track record can be used if the manager was primarily responsible and proper documentation exists
- **B**: Past performance can never be ported to a new firm
- **C**: The manager can claim the track record regardless of her role

## Answer

A

## Explanation

GIPS allows performance portability if the manager was primarily responsible for performance, decision-making processes are substantially the same, documentation supports the claims, and GIPS requirements are met.

**Common mistakes:**
- Option B incorrectly prohibits all portability
- Option C ignores the requirement for primary responsibility
