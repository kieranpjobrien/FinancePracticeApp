# Question

The Code of Ethics applies to members' conduct:

## Options

- **A**: Only during trading hours
- **B**: In all professional activities, including advisory, educational, and volunteer roles
- **C**: Only when managing money for fees

## Answer

B

## Explanation

The Code of Ethics applies broadly to all professional activities, not just fee-generating services. This includes advisory roles, educational activities, and volunteer positions where members represent the investment profession.

**Common mistakes:**
- Option A incorrectly limits by time
- Option C incorrectly limits by compensation structure
