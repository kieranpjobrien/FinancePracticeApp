# Question

A manager presents only accounts with the best performance when meeting with prospective clients. This practice:

## Options

- **A**: Is acceptable marketing highlighting the manager's capabilities
- **B**: Violates Standard III(D) by presenting misleading performance
- **C**: Is permitted if the accounts shown are real

## Answer

B

## Explanation

Standard III(D) requires fair, accurate, and complete performance presentation. Cherry-picking best-performing accounts misrepresents typical results and misleads prospective clients about expected outcomes.

**Common mistakes:**
- Option A treats cherry-picking as legitimate marketing
- Option C ignores that true but selectively presented data can mislead
