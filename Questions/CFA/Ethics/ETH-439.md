# Question

CFA charterholders must annually:

## Options

- **A**: Retake the Level III exam
- **B**: Sign the Professional Conduct Statement affirming compliance
- **C**: Complete a minimum number of continuing education credits

## Answer

B

## Explanation

CFA charterholders must annually sign the Professional Conduct Statement, affirming they have complied with the Code and Standards. While continuing education is encouraged, it is not a mandatory annual requirement.

**Common mistakes:**
- Option A imposes a non-existent retesting requirement
- Option C describes a practice that is encouraged but not mandated
