# Question

Under Standard I(C), misrepresentation includes:

## Options

- **A**: Only false statements about investment performance
- **B**: False or misleading statements about qualifications, services, or performance
- **C**: Only intentional deception

## Answer

B

## Explanation

Standard I(C) broadly prohibits misrepresentation about qualifications, services, performance, and other matters. Both intentional and negligent misstatements are covered.

**Common mistakes:**
- Option A too narrowly limits covered topics
- Option C incorrectly requires intent for all violations
