# Question

The GIPS compliance statement:

## Options

- **A**: Can be modified to fit the firm's marketing language
- **B**: Must follow a specific prescribed format
- **C**: Is optional for verified firms

## Answer

B

## Explanation

GIPS requires a specific compliance statement format that cannot be altered. This ensures consistency and prevents firms from implying partial compliance or modifying the statement's meaning.

**Common mistakes:**
- Option A allows unauthorized modifications
- Option C incorrectly suggests verification eliminates statement requirement
