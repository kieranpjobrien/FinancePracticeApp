# Question

According to professional standards, how should a professional designation typically be displayed after a person's name?

## Options

- **A**: Using capital letters without periods, following the person's name (e.g., "John Smith, ABC")
- **B**: Using periods between letters following the name (e.g., "John Smith, A.B.C.")
- **C**: Placing the designation before the name as a title (e.g., "ABC John Smith")

## Answer

A

## Explanation

Professional designations should generally be used as adjectives that follow a person's name, using capital letters without periods. This format clearly identifies the credential while maintaining proper usage conventions.

**Common mistakes:**
- Option B is typically incorrect because periods between letters is outdated style for most modern designations
- Option C is incorrect because professional designations generally follow the name rather than precede it; they are credentials, not titles like "Dr."
