# Question

A compliance officer implements a reasonable supervision system but an employee still manages to violate the Standards. The compliance officer:

## Options

- **A**: Has violated Standard IV(C) because a violation occurred
- **B**: Has not violated the Standard if reasonable procedures were in place and properly implemented
- **C**: Must resign due to the supervisory failure

## Answer

B

## Explanation

Standard IV(C) requires reasonable, not perfect, supervision. If a supervisor establishes and implements reasonable procedures but violations still occur, the supervisor is not automatically in violation. The Standard focuses on the adequacy of the supervisory system.

**Common mistakes:**
- Option A imposes strict liability for subordinate violations
- Option C overreacts to an incident that may not indicate supervisory failure
