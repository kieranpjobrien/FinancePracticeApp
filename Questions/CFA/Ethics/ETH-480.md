# Question

During a secondary offering, a manager pre-sells shares to select clients before the offering is announced. This practice:

## Options

- **A**: Is rewarding loyal clients appropriately
- **B**: Likely violates Standard III(B) by providing unfair access to investment opportunities
- **C**: Is acceptable if the selected clients have large accounts

## Answer

B

## Explanation

Standard III(B) requires fair dealing in investment opportunities. Giving select clients pre-announcement access to secondary offerings disadvantages other clients and violates fair dealing principles.

**Common mistakes:**
- Option A justifies preferential treatment inappropriately
- Option C uses account size to justify unfair access
