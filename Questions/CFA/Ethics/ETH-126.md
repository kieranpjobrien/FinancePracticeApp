# Question

A portfolio manager presents performance results showing only the top-performing years while omitting years of poor performance. This most likely violates:

## Options

- **A**: Standard I(C) because selective presentation misrepresents actual performance
- **B**: Standard III(D) because it affects trade allocation
- **C**: Standard V(B) because it fails to distinguish fact from opinion

## Answer

A

## Explanation

Selectively presenting only favorable performance data while omitting unfavorable periods constitutes misrepresentation under Standard I(C). This creates a misleading impression of the manager's track record, even if the individual data points shown are accurate.

**Common mistakes:**
- Option B relates to fair dealing in trade allocation, not performance presentation
- Option C addresses distinguishing facts from opinions in communications
