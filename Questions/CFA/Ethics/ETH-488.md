# Question

An analyst provides detailed reports to institutional clients but only summary recommendations to retail clients. Standard V(B):

## Options

- **A**: Requires identical reports for all clients
- **B**: Permits format differences if core information and reasoning are communicated to all
- **C**: Only applies to written communications

## Answer

B

## Explanation

Standard V(B) requires communicating material factors to clients but allows format flexibility. Institutional and retail clients may receive different formats as long as essential information about risks and rationale is provided to all.

**Common mistakes:**
- Option A requires unnecessary uniformity
- Option C excludes oral and other communication forms
