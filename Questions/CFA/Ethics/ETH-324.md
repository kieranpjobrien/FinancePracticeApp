# Question

A firm offers pre-packaged model portfolios to retail clients with minimal customization. Under Standard III(C):

## Options

- **A**: Model portfolios eliminate suitability requirements
- **B**: The firm must still assess that the model is suitable for each client
- **C**: Suitability applies only to customized portfolios

## Answer

B

## Explanation

Standard III(C) requires suitability assessment even when using model portfolios. The firm must determine that the selected model is appropriate for the client's situation, not just offer it without evaluation.

**Common mistakes:**
- Option A incorrectly suggests standardization eliminates suitability duty
- Option C creates an unjustified distinction
