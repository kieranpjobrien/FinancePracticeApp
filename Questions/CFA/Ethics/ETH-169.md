# Question

An investment manager based in Country A manages portfolios for clients in Country B, where securities laws are less strict. Under Standard I(A):

## Options

- **A**: The manager should follow Country B's less strict laws since clients are located there
- **B**: The manager must follow the stricter of Country A's laws, Country B's laws, and the Code and Standards
- **C**: Only the Code and Standards apply in cross-border situations

## Answer

B

## Explanation

Standard I(A) requires following the stricter of applicable laws or the Code and Standards in all jurisdictions where the member conducts business. Members cannot choose less strict requirements simply because clients are in another jurisdiction.

**Common mistakes:**
- Option A allows regulatory arbitrage, which is prohibited
- Option C ignores that local laws still apply alongside the Code and Standards
