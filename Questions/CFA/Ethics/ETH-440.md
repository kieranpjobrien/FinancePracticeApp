# Question

A charterholder's business card reads "John Smith, C.F.A." with periods between letters. This usage:

## Options

- **A**: Is acceptable as a personal style preference
- **B**: Violates Standard VII(B) by using the designation incorrectly
- **C**: Is required to distinguish from other certifications

## Answer

B

## Explanation

Standard VII(B) requires proper use of the CFA designation. The correct format is "CFA" without periods, used as an adjective (John Smith, CFA) or noun (CFA charterholder). Adding periods misrepresents the mark.

**Common mistakes:**
- Option A treats trademark usage as personal preference
- Option C incorrectly justifies improper formatting
