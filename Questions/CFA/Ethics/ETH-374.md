# Question

Standard III(D) Performance Presentation requires that members:

## Options

- **A**: Only show periods of positive performance
- **B**: Present performance information that is fair, accurate, and complete
- **C**: Use any calculation method as long as it is disclosed

## Answer

B

## Explanation

Standard III(D) mandates fair, accurate, and complete performance presentation. Members cannot selectively present only favorable periods or use misleading calculation methods, even with disclosure.

**Common mistakes:**
- Option A allows cherry-picking favorable periods
- Option C suggests disclosure excuses misleading methodologies
