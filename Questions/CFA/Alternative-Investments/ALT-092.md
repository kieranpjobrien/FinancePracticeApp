# Question

A high-water mark provision in hedge fund fee structures:

## Options

- **A**: Guarantees minimum returns to investors
- **B**: Prevents managers from charging performance fees until prior losses are recovered
- **C**: Sets a maximum limit on fund returns

## Answer

B

## Explanation

High-water mark explained:

**How it works:**
- Tracks highest NAV achieved
- Performance fees only on new profits
- Must exceed prior peak
- Protects against "double dipping"

**Example:**
- NAV peaks at $120
- Falls to $100
- Rises to $115
- No performance fee yet
- Fee only above $120

**Investor protection:**
- No fee for recovering losses
- Manager must create new value
- Aligns incentives
- Standard in industry

**Manager implications:**
- Must recover losses first
- Can be significant hurdle
- May take years to recover
- Talent retention challenge

**Reset provisions:**
- Some funds have modified HWM
- Time-based resets
- Investor-specific marks

**Common mistakes:**
- Option A: No return guarantee
- Option C: Not a return limit
