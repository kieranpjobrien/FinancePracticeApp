# Question

High autocorrelation in alternative investment returns suggests:

## Options

- **A**: Strong investment manager skill
- **B**: Returns may be smoothed or subject to stale pricing
- **C**: Low correlation with traditional asset classes

## Answer

B

## Explanation

Autocorrelation in alternatives:

**What autocorrelation means:**
- Current return correlates with past returns
- Returns are serially dependent
- Not independently distributed

**Why it occurs:**
- Appraisal-based valuations
- Infrequent marking to market
- Manager discretion in valuation
- Illiquid asset pricing

**Implications:**
- True volatility understated
- Risk measures biased downward
- Sharpe ratio overstated
- Portfolio optimization distorted

**Detection:**
- Autocorrelation tests
- Ljung-Box statistic
- Serial correlation analysis

**Adjustment approaches:**
- Unsmoothing techniques
- Geltner/Okunev methods
- Use lagged correlations

**Common mistakes:**
- Option A: Autocorrelation isn't skill indicator
- Option C: Describes correlation with other assets
