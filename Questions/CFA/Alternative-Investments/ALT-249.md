# Question

The application of GIPS (Global Investment Performance Standards) to alternative investments:

## Options

- **A**: Is mandatory for all hedge funds globally
- **B**: Provides voluntary performance reporting standards that enhance comparability
- **C**: Only applies to mutual funds

## Answer

B

## Explanation

GIPS for alternatives:

**GIPS purpose:**
- Standardized reporting
- Performance comparability
- Investor protection
- Voluntary compliance

**Alternative-specific provisions:**
- Private equity requirements
- Real estate standards
- Pooled fund reporting
- Composite construction

**Key requirements:**
- Fair value standards
- Return calculation methods
- Disclosure requirements
- Verification options

**Benefits of compliance:**
- Credibility enhancement
- Investor confidence
- Industry best practice
- Competitive advantage

**Challenges:**
- Complex structures
- Illiquid valuations
- Data availability
- Cost of compliance

**Common mistakes:**
- Option A: Voluntary, not mandatory
- Option C: Applies broadly to alternatives
