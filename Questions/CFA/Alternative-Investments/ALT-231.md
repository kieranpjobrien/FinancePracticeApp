# Question

Project finance structures for infrastructure typically feature:

## Options

- **A**: Full recourse to the sponsors' balance sheets
- **B**: Non-recourse or limited recourse debt with the project as primary security
- **C**: No contractual arrangements with off-takers

## Answer

B

## Explanation

Project finance characteristics:

**Non-recourse structure:**
- Debt secured by project only
- Limited sponsor recourse
- Project cash flows repay debt
- Ring-fenced SPV

**Special Purpose Vehicle (SPV):**
- Separate legal entity
- Owns project assets
- Isolates risks
- Single purpose

**Contractual framework:**
- Offtake agreements
- Construction contracts
- O&M agreements
- Supply contracts

**Lender protections:**
- Cash flow waterfall
- Reserve accounts
- DSCR covenants
- Step-in rights

**Why used:**
- Large capital needs
- Risk allocation
- Off-balance sheet (sometimes)
- Match asset life to debt

**Common mistakes:**
- Option A: Limited, not full recourse
- Option C: Contracts essential for bankability
