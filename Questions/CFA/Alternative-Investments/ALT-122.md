# Question

Time-weighted returns (TWR) differ from money-weighted returns (IRR) in that TWR:

## Options

- **A**: Is more appropriate when investors control cash flow timing
- **B**: Eliminates the impact of cash flow timing on performance measurement
- **C**: Always produces higher return figures than IRR

## Answer

B

## Explanation

TWR vs IRR comparison:

**Time-weighted returns:**
- Removes cash flow impact
- Measures manager skill
- Geometric linking of periods
- Standard for public managers

**Money-weighted returns (IRR):**
- Reflects cash flow timing
- Measures investor experience
- Weighted by capital at work
- Standard for private markets

**When to use TWR:**
- Manager doesn't control flows
- Compare managers fairly
- Public market benchmarking
- Client-directed cash flows

**When to use IRR:**
- Manager controls timing
- Private equity/real estate
- Investor experience matters
- Cash flow timing is skill

**Example difference:**
- Large inflow before poor period: IRR < TWR
- Large inflow before good period: IRR > TWR

**Common mistakes:**
- Option A: IRR appropriate when investor controls
- Option C: Can be higher or lower
