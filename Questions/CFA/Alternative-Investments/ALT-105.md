# Question

Blockchain technology provides:

## Options

- **A**: A centralized database controlled by one entity
- **B**: A distributed, immutable ledger of transactions
- **C**: Unlimited transaction reversal capabilities

## Answer

B

## Explanation

Blockchain fundamentals:

**Key features:**
- Distributed across nodes
- Immutable once recorded
- Transparent and auditable
- Cryptographically secured

**How it works:**
- Transactions grouped into blocks
- Blocks linked chronologically
- Consensus mechanism validates
- No single point of control

**Benefits:**
- Trustless transactions
- Censorship resistance
- Transparent history
- Reduced intermediaries

**Types:**
- Public (Bitcoin, Ethereum)
- Private/permissioned
- Consortium

**Applications beyond crypto:**
- Supply chain tracking
- Identity verification
- Financial settlement
- Record keeping

**Common mistakes:**
- Option A: Distributed, not centralized
- Option C: Immutability is key feature
