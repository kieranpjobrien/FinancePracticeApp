# Question

Wrapped tokens in cryptocurrency represent:

## Options

- **A**: Tokens that can only be traded on one blockchain
- **B**: Tokens pegged to assets on other blockchains, enabling cross-chain functionality
- **C**: Encrypted tokens that cannot be transferred

## Answer

B

## Explanation

Wrapped tokens:

**Definition:**
- Represent assets from other chains
- Pegged 1:1 to underlying
- Enable cross-chain use
- Tokenized version

**How they work:**
- Lock original asset
- Mint wrapped version
- Use on different blockchain
- Redeem by burning wrapped

**Common examples:**
- WBTC (Wrapped Bitcoin on Ethereum)
- WETH (Wrapped Ether)
- Various bridge tokens

**Purpose:**
- Cross-chain interoperability
- Use Bitcoin in DeFi
- Access different ecosystems
- Expand utility

**Risks:**
- Custodian risk
- Bridge vulnerabilities
- Peg stability
- Smart contract risk

**Common mistakes:**
- Option A: Enables cross-chain, not single-chain
- Option C: Transferable by design
