# Question

The Sortino ratio differs from the Sharpe ratio by:

## Options

- **A**: Using total volatility in the denominator
- **B**: Only penalizing downside volatility
- **C**: Ignoring the risk-free rate

## Answer

B

## Explanation

Sortino ratio explained:

**Formula:**
Sortino = (Return - Target) / Downside Deviation

**Key difference from Sharpe:**
- Sharpe uses total volatility
- Sortino uses only downside volatility
- Upside volatility not penalized

**Why this matters:**
- Investors care more about losses
- Upside volatility is "good"
- Better for asymmetric return distributions
- More relevant for alternatives

**Downside deviation:**
- Standard deviation of returns below target
- Target often 0% or risk-free rate
- Only negative surprises count

**Application to alternatives:**
- Hedge funds: Often asymmetric returns
- Options strategies: Skewed distributions
- Better risk-adjusted measure

**Common mistakes:**
- Option A: Sharpe uses total volatility
- Option C: Sortino still uses risk-free/target
