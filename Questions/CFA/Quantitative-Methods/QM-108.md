# Question

The difference between gross return and net return is:

## Options

- **A**: Net return includes the effects of inflation
- **B**: Net return reflects deductions for management fees and expenses
- **C**: Gross return is always lower than net return

## Answer

B

## Explanation

Gross return is the return before any fees or expenses are deducted. Net return is the actual return an investor receives after management fees, trading costs, and other expenses are subtracted. The difference matters significantly over time due to compounding—even small annual fees can substantially reduce terminal wealth over long investment horizons.

**Common mistakes:**
- Option A describes real vs nominal return, not gross vs net return
- Option C is incorrect; gross return is always higher than or equal to net return (fees cannot be negative)
