# Question

Random forest improves upon a single decision tree by:

## Options

- **A**: Using fewer features and a smaller training set
- **B**: Building multiple trees with random subsets of data and features, then averaging predictions
- **C**: Requiring the user to specify the exact structure of the tree

## Answer

B

## Explanation

Random forest is an ensemble method that builds many decision trees, each trained on a random bootstrap sample of the data and using a random subset of features at each split. The final prediction averages across all trees (for regression) or uses majority voting (for classification). This reduces overfitting and variance compared to a single tree, improving generalization to new data.

**Common mistakes:**
- Option A is the opposite; random forest uses more resources to build multiple trees
- Option C describes the opposite of what random forest does; it automates structure selection
