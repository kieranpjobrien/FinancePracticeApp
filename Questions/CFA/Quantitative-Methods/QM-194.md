# Question

An investor with utility function U = E(r) - 0.5Aσ² has risk aversion coefficient A = 4. For a portfolio with expected return 10% and variance 0.04, the utility is:

## Options

- **A**: 0.02
- **B**: 0.06
- **C**: 0.18

## Answer

A

## Explanation

U = E(r) - 0.5Aσ²

U = 0.10 - 0.5(4)(0.04)
U = 0.10 - 0.08
U = 0.02

Higher utility indicates a more preferred portfolio. Higher risk aversion (A) reduces utility for a given level of risk.

**Common mistakes:**
- Option B forgets to multiply by 0.5: 0.10 - (4)(0.04) = 0.10 - 0.16 = wrong sign
- Option C adds rather than subtracts the risk penalty
