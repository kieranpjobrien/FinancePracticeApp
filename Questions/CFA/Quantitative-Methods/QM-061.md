# Question

An analyst estimates three scenarios for a stock's return: 30% probability of 20% return, 50% probability of 8% return, and 20% probability of -10% return. The expected return is:

## Options

- **A**: 6.0%
- **B**: 8.0%
- **C**: 10.0%

## Answer

B

## Explanation

Expected return = Σ (Probability × Return)

E(R) = (0.30 × 20%) + (0.50 × 8%) + (0.20 × -10%)
E(R) = 6% + 4% + (-2%)
E(R) = 8%

The expected value is the probability-weighted average of all possible outcomes.

**Common mistakes:**
- Option A results from arithmetic errors in the weighted calculation
- Option C uses a simple average of returns (20% + 8% - 10%)/3 = 6% then adds to something, ignoring probability weights
