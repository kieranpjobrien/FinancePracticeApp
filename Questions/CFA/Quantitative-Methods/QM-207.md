# Question

Before running a regression, examining a scatter plot of X and Y is useful to:

## Options

- **A**: Calculate the exact regression coefficients
- **B**: Visually assess whether a linear relationship is appropriate
- **C**: Determine the p-value for the slope coefficient

## Answer

B

## Explanation

A scatter plot displays each observation as a point with X and Y coordinates. Visual inspection helps assess: (1) whether the relationship appears linear or curved, (2) the direction and strength of association, (3) the presence of outliers, and (4) whether variance appears constant. This preliminary analysis informs model specification before formal statistical analysis.

**Common mistakes:**
- Option A requires computation, not visual inspection
- Option C requires hypothesis testing after fitting the model
