# Question

For a two-tailed test at the 5% significance level, the critical z-values are approximately:

## Options

- **A**: ±1.28
- **B**: ±1.65
- **C**: ±1.96

## Answer

C

## Explanation

For a two-tailed test at 5% significance, 2.5% falls in each tail of the distribution. The z-value that leaves 2.5% in the upper tail (or 97.5% below it) is approximately 1.96. Thus the critical values are ±1.96.

Common critical values:
- 10% two-tailed: ±1.645 (5% in each tail)
- 5% two-tailed: ±1.96 (2.5% in each tail)
- 1% two-tailed: ±2.576 (0.5% in each tail)

**Common mistakes:**
- Option A is for 20% significance (10% in each tail)
- Option B is for 10% significance (5% in each tail)
