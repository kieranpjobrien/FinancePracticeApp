# Question

Holding sample size constant, if an analyst increases the confidence level from 90% to 99%, the confidence interval will:

## Options

- **A**: Become narrower because higher confidence means more precision
- **B**: Become wider because greater confidence requires a larger range
- **C**: Remain unchanged because sample size determines interval width

## Answer

B

## Explanation

A higher confidence level requires a larger critical value (z or t), which widens the confidence interval. For example, the z-value for 90% confidence is 1.645, while for 99% confidence it is 2.576. There is a trade-off between confidence level and precision—to be more confident that the interval contains the true parameter, the interval must be wider.

**Common mistakes:**
- Option A confuses confidence with precision; higher confidence means wider, not narrower, intervals
- Option C ignores the effect of confidence level; while sample size affects width, so does the chosen confidence level
