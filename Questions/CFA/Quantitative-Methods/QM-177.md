# Question

Adjusted R-squared differs from R-squared because it:

## Options

- **A**: Is always higher than R-squared
- **B**: Adjusts for the number of independent variables in the model
- **C**: Can exceed 100%

## Answer

B

## Explanation

Adjusted R² penalizes for adding variables that don't improve the model. R² always increases (or stays the same) when variables are added, even if they have no explanatory power. Adjusted R² can decrease when adding irrelevant variables, making it better for comparing models with different numbers of predictors.

Adjusted R² = 1 - [(1 - R²)(n - 1)/(n - k - 1)]

**Common mistakes:**
- Option A is incorrect; adjusted R² is typically lower than R², especially with more variables
- Option C is incorrect; adjusted R² is bounded, like R²
