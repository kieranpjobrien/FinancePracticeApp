# Question

Component VaR measures:

## Options

- **A**: The VaR of each asset in isolation
- **B**: The contribution of each position to the portfolio's total VaR
- **C**: The change in VaR when removing an asset

## Answer

B

## Explanation

Component VaR decomposes portfolio VaR into contributions from each position. It accounts for correlation effects—the sum of component VaRs equals total portfolio VaR. This is useful for risk budgeting and identifying which positions contribute most to overall risk. Component VaR = Weight × Marginal VaR × Portfolio VaR.

**Common mistakes:**
- Option A describes standalone VaR, which ignores diversification effects
- Option C describes incremental VaR, which measures the impact of removing a position
