# Question

When testing a hypothesis about a single population variance, the appropriate test statistic follows:

## Options

- **A**: The t-distribution
- **B**: The chi-square distribution
- **C**: The F-distribution

## Answer

B

## Explanation

For hypothesis tests about a single population variance, the test statistic is:

χ² = (n-1)s²/σ₀²

where s² is the sample variance and σ₀² is the hypothesized population variance. This statistic follows a chi-square distribution with n-1 degrees of freedom. The chi-square distribution is right-skewed and bounded at zero, appropriate for testing variance which cannot be negative.

**Common mistakes:**
- Option A (t-distribution) is used for testing population means, not variances
- Option C (F-distribution) is used for comparing two variances, not testing a single variance against a hypothesized value
