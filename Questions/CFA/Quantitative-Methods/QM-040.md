# Question

The power of a statistical test is defined as:

## Options

- **A**: The probability of rejecting the null hypothesis when it is true
- **B**: The probability of correctly rejecting a false null hypothesis
- **C**: The probability of failing to reject a true null hypothesis

## Answer

B

## Explanation

Statistical power equals 1 - β, where β is the probability of a Type II error. Power represents the probability of correctly rejecting the null hypothesis when it is false—that is, correctly detecting an effect when one exists. Higher power means the test is more likely to detect true differences. Power can be increased by increasing sample size, using a larger significance level, or when the true effect size is larger.

**Common mistakes:**
- Option A describes a Type I error (α), not power
- Option C describes the correct decision when the null is true, not power (which relates to correctly rejecting false nulls)
