# Question

According to Chebyshev's inequality, for any distribution, at least what percentage of observations must fall within 3 standard deviations of the mean?

## Options

- **A**: 75%
- **B**: 89%
- **C**: 99.7%

## Answer

B

## Explanation

Chebyshev's inequality states that at least (1 - 1/k²) of observations fall within k standard deviations of the mean for any distribution.

For k = 3:
Minimum percentage = 1 - 1/3² = 1 - 1/9 = 8/9 = 88.9% ≈ 89%

This is a lower bound that applies to all distributions, unlike the 99.7% rule which applies only to normal distributions.

**Common mistakes:**
- Option A uses k = 2: 1 - 1/4 = 75%
- Option C applies the empirical rule for normal distributions (68-95-99.7), which doesn't apply to all distributions
