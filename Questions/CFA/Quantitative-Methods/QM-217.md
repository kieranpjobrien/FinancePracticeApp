# Question

Cluster sampling differs from stratified sampling in that:

## Options

- **A**: Cluster sampling selects all members from some clusters, while stratified sampling selects some members from all strata
- **B**: Cluster sampling requires larger sample sizes
- **C**: Stratified sampling is only used for qualitative research

## Answer

A

## Explanation

In stratified sampling, the population is divided into strata and samples are taken from every stratum. In cluster sampling, the population is divided into clusters, some clusters are randomly selected, and all (or most) members within selected clusters are sampled. Cluster sampling is efficient when naturally occurring groups exist and visiting all locations is impractical.

**Common mistakes:**
- Option B is not necessarily true; sample size depends on study design
- Option C is incorrect; stratified sampling is used in quantitative research
