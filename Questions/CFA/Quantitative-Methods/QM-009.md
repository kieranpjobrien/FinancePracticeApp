# Question

An investor will receive $50,000 in 5 years. If the discount rate is 8% per year, the present value of this future payment is closest to:

## Options

- **A**: $31,250
- **B**: $34,029
- **C**: $36,751

## Answer

B

## Explanation

PV = FV / (1 + r)^n
PV = $50,000 / (1.08)^5
PV = $50,000 / 1.46933
PV = $34,029.16

Present value discounts future cash flows back to today's dollars at the required rate of return.

**Common mistakes:**
- Option A incorrectly uses simple discounting: $50,000 / 1.40 = $35,714 or another error
- Option C may result from using a lower discount rate or fewer periods
