# Question

Given: Cov(X,Y) = 60, Var(X) = 100, Var(Y) = 144. The slope (b₁) of the regression of Y on X is:

## Options

- **A**: 0.42
- **B**: 0.60
- **C**: 1.44

## Answer

B

## Explanation

The slope of the regression of Y on X is:

b₁ = Cov(X,Y) / Var(X)

b₁ = 60 / 100 = 0.60

Note: The variance of Y is not used in calculating the slope of Y on X.

**Common mistakes:**
- Option A uses Cov(X,Y)/Var(Y) = 60/144, which would be the slope of X on Y
- Option C uses Var(Y)/Var(X), which is not the correct formula
