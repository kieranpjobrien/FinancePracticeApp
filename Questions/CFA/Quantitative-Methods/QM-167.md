# Question

Misspecification error in regression occurs when:

## Options

- **A**: The sample size is too small
- **B**: The functional form of the model is incorrect or relevant variables are omitted
- **C**: The data contains measurement errors

## Answer

B

## Explanation

Model misspecification includes: (1) using the wrong functional form (e.g., linear when relationship is nonlinear), (2) omitting relevant variables (causes omitted variable bias), (3) including irrelevant variables, or (4) using the wrong variable proxies. Misspecification can bias coefficient estimates and invalidate inference. Diagnostic tests and theoretical knowledge help detect and correct specification errors.

**Common mistakes:**
- Option A describes a sample size issue, not specification error
- Option C describes errors-in-variables, a different problem from misspecification
