# Question

Sampling error refers to:

## Options

- **A**: Mistakes made during data collection
- **B**: The difference between a sample statistic and the true population parameter
- **C**: Bias introduced by non-random sampling methods

## Answer

B

## Explanation

Sampling error is the natural variation that occurs because a sample is only a subset of the population. Even with perfect random sampling, the sample statistic will typically differ from the true population parameter. Sampling error decreases as sample size increases and is quantified by the standard error. It is not the result of mistakes or bias—it is inherent to the sampling process.

**Common mistakes:**
- Option A describes measurement errors or data collection mistakes, not sampling error
- Option C describes sampling bias, which is a systematic error different from random sampling error
