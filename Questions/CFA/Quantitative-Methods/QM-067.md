# Question

Compared to parametric tests, nonparametric tests:

## Options

- **A**: Require assumptions about the population distribution
- **B**: Are generally less powerful but make fewer distributional assumptions
- **C**: Can only be used with interval or ratio scale data

## Answer

B

## Explanation

Nonparametric tests make minimal assumptions about the underlying population distribution (they are "distribution-free"). This flexibility comes at a cost: nonparametric tests typically have less statistical power than their parametric counterparts when parametric assumptions are met. They are useful when data is ordinal, when sample sizes are small, or when the population is known to deviate significantly from normality.

**Common mistakes:**
- Option A describes parametric tests, not nonparametric tests
- Option C is incorrect; nonparametric tests can be used with ordinal data, which is an advantage over parametric tests
