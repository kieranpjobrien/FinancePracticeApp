# Question

The Treynor ratio uses which measure of risk in the denominator?

## Options

- **A**: Standard deviation (total risk)
- **B**: Beta (systematic risk)
- **C**: Tracking error (active risk)

## Answer

B

## Explanation

The Treynor ratio measures excess return per unit of systematic (market) risk:

Treynor Ratio = (Portfolio Return - Risk-Free Rate) / Beta

Unlike the Sharpe ratio which uses total risk (standard deviation), the Treynor ratio focuses only on systematic risk. This is appropriate for evaluating diversified portfolios where unsystematic risk has been eliminated.

**Common mistakes:**
- Option A describes the denominator of the Sharpe ratio
- Option C describes the denominator of the Information ratio
