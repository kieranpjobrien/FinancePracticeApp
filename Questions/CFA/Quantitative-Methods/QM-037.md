# Question

A Type I error occurs when:

## Options

- **A**: The null hypothesis is rejected when it is actually true
- **B**: The null hypothesis is not rejected when it is actually false
- **C**: The sample size is too small to draw conclusions

## Answer

A

## Explanation

A Type I error (false positive) occurs when we incorrectly reject a true null hypothesis. The probability of a Type I error is denoted by alpha (α) and equals the significance level of the test. For example, at a 5% significance level, there is a 5% probability of rejecting the null hypothesis when it is actually true.

**Common mistakes:**
- Option B describes a Type II error (false negative), which occurs when we fail to reject a false null hypothesis
- Option C describes a limitation related to statistical power, not a specific type of error in hypothesis testing
