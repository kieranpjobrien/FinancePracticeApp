# Question

A decision tree in machine learning:

## Options

- **A**: Can only be used for regression problems, not classification
- **B**: Partitions the feature space into regions using a series of rules based on feature values
- **C**: Requires all input features to be continuous numerical variables

## Answer

B

## Explanation

A decision tree uses a series of if-then rules to partition the data into increasingly homogeneous groups. At each node, the algorithm selects a feature and split point that best separates the data according to some criterion (like Gini impurity for classification or variance for regression). The result is a tree-like structure that is highly interpretable. Decision trees can handle both classification and regression, and can use both categorical and continuous features.

**Common mistakes:**
- Option A is incorrect; decision trees work for both classification and regression
- Option C is incorrect; decision trees naturally handle both categorical and continuous variables
