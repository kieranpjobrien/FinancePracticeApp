# Question

Increasing the number of trials in a Monte Carlo simulation will:

## Options

- **A**: Change the underlying probability distributions used
- **B**: Improve the precision of the estimated statistics
- **C**: Eliminate model risk from incorrect assumptions

## Answer

B

## Explanation

As the number of simulation trials increases, the law of large numbers ensures that sample statistics (mean, standard deviation, percentiles) converge toward their true values based on the assumed distributions. More trials reduce sampling error and provide more precise estimates. However, additional trials do not correct for incorrectly specified distributions or parameters—they only reduce random variation in the simulation output.

**Common mistakes:**
- Option A is incorrect; the input distributions are specified before the simulation and do not change with more trials
- Option C is incorrect; more trials reduce sampling error but do not address model risk from wrong assumptions
