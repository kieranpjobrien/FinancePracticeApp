# Question

The key difference between supervised and unsupervised machine learning is that:

## Options

- **A**: Supervised learning requires more computational power
- **B**: Supervised learning uses labeled data with known outcomes to train the model
- **C**: Unsupervised learning always produces more accurate predictions

## Answer

B

## Explanation

In supervised learning, the model is trained on labeled data where the outcome variable (target) is known. The algorithm learns the relationship between input features and the outcome to make predictions on new data. Examples include regression and classification. In unsupervised learning, there is no target variable—the algorithm finds patterns, clusters, or structures in the data without predefined outcomes. Examples include clustering and dimensionality reduction.

**Common mistakes:**
- Option A is incorrect; computational requirements depend on the specific algorithm and data size, not the learning type
- Option C is incorrect; accuracy depends on the problem, data quality, and algorithm choice, not whether learning is supervised or unsupervised
