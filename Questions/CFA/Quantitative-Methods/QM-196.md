# Question

The use of leverage in a portfolio:

## Options

- **A**: Reduces both potential gains and losses
- **B**: Amplifies both potential gains and losses
- **C**: Only affects gains, not losses

## Answer

B

## Explanation

Leverage magnifies returns in both directions. If an investor uses borrowed money to increase exposure, gains are amplified when investments perform well, but losses are also amplified when they perform poorly. The return on equity increases with leverage for positive returns but becomes more negative for losses. Leverage increases portfolio risk (volatility).

**Common mistakes:**
- Option A describes the opposite effect of leverage
- Option C ignores that leverage affects both sides equally
