# Question

A portfolio has the following annual returns over four years: 5%, 10%, -2%, and 15%. The arithmetic mean return is closest to:

## Options

- **A**: 6.0%
- **B**: 7.0%
- **C**: 8.0%

## Answer

B

## Explanation

Arithmetic Mean = Sum of Returns / Number of Returns
Arithmetic Mean = (5% + 10% + (-2%) + 15%) / 4
Arithmetic Mean = 28% / 4
Arithmetic Mean = 7.0%

The arithmetic mean is the simple average of all returns.

**Common mistakes:**
- Options A and C may result from calculation errors in adding the returns
