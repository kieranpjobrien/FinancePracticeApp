# Question

Survivorship bias in a mutual fund database would most likely cause:

## Options

- **A**: Average historical returns to appear lower than the true average
- **B**: Average historical returns to appear higher than the true average
- **C**: No systematic effect on reported average returns

## Answer

B

## Explanation

Survivorship bias occurs when only surviving entities are included in a dataset, while those that failed or were terminated are excluded. In mutual fund databases, funds that performed poorly and were subsequently merged or liquidated are often removed. The remaining funds are the "survivors" with better performance, biasing the average return upward. Historical analysis using such data overstates expected future returns.

**Common mistakes:**
- Option A is the opposite of the actual effect
- Option C ignores the systematic upward bias caused by excluding poor performers
