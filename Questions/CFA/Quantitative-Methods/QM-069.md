# Question

Sensitivity analysis differs from scenario analysis in that sensitivity analysis:

## Options

- **A**: Changes multiple input variables simultaneously to examine combined effects
- **B**: Changes one input variable at a time while holding others constant
- **C**: Uses random sampling from probability distributions

## Answer

B

## Explanation

Sensitivity analysis examines how the output changes when a single input variable is changed, holding all other inputs constant. This isolates the impact of each variable. Scenario analysis, in contrast, changes multiple variables simultaneously to represent coherent alternative futures (e.g., "recession scenario" with higher unemployment, lower growth, and wider credit spreads). Monte Carlo simulation involves random sampling from distributions.

**Common mistakes:**
- Option A describes scenario analysis, not sensitivity analysis
- Option C describes Monte Carlo simulation, not sensitivity analysis
