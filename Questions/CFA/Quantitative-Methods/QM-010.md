# Question

An ordinary annuity pays $1,000 per year for 4 years. If the discount rate is 6%, the present value of the annuity is closest to:

## Options

- **A**: $3,465
- **B**: $3,673
- **C**: $4,000

## Answer

A

## Explanation

PV of Ordinary Annuity = PMT × [(1 - (1 + r)^-n) / r]
PV = $1,000 × [(1 - (1.06)^-4) / 0.06]
PV = $1,000 × [(1 - 0.7921) / 0.06]
PV = $1,000 × [0.2079 / 0.06]
PV = $1,000 × 3.4651
PV = $3,465.11

**Common mistakes:**
- Option B might result from using an annuity due formula (payments at beginning of period)
- Option C ignores the time value of money entirely
