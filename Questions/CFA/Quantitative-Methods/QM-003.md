# Question

An investor buys a stock for $50 and sells it one year later for $55. During the year, the stock paid a $2 dividend. The holding period return is closest to:

## Options

- **A**: 10.0%
- **B**: 12.0%
- **C**: 14.0%

## Answer

C

## Explanation

Holding Period Return = (Ending Price - Beginning Price + Income) / Beginning Price

HPR = ($55 - $50 + $2) / $50 = $7 / $50 = 0.14 = 14.0%

The total return includes both capital appreciation ($5) and dividend income ($2).

**Common mistakes:**
- Option A only considers the capital gain and ignores the dividend
- Option B may result from a calculation error
