# Question

A fraud detection model identifies 80 of 100 actual fraud cases, while flagging 20 non-fraud transactions. The recall (sensitivity) is:

## Options

- **A**: 75%
- **B**: 80%
- **C**: 20%

## Answer

B

## Explanation

Recall (Sensitivity) = True Positives / (True Positives + False Negatives)

True Positives = 80 (correctly identified fraud)
False Negatives = 20 (missed fraud: 100 - 80 = 20)

Recall = 80 / (80 + 20) = 80 / 100 = 80%

Recall measures what proportion of actual positive cases (fraud) were correctly identified.

**Common mistakes:**
- Option A calculates 80/(80+20+20) mixing different metrics
- Option C confuses with the number of non-fraud falsely flagged
