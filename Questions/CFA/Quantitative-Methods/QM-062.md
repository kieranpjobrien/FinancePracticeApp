# Question

The probability of a recession is 25%. If there is a recession, the probability that a stock declines is 70%. If there is no recession, the probability the stock declines is 20%. What is the unconditional probability that the stock declines?

## Options

- **A**: 32.5%
- **B**: 45.0%
- **C**: 90.0%

## Answer

A

## Explanation

Using the total probability rule:

P(Decline) = P(Decline|Recession) × P(Recession) + P(Decline|No Recession) × P(No Recession)

P(Decline) = (0.70 × 0.25) + (0.20 × 0.75)
P(Decline) = 0.175 + 0.15
P(Decline) = 0.325 = 32.5%

**Common mistakes:**
- Option B incorrectly averages the conditional probabilities (70% + 20%)/2 = 45%
- Option C adds the conditional probabilities (70% + 20% = 90%) without considering the probability weights
