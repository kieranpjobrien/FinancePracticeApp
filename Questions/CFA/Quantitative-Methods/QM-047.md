# Question

Overfitting in a machine learning model occurs when:

## Options

- **A**: The model performs well on training data but poorly on new data
- **B**: The model is too simple to capture the underlying patterns
- **C**: The model uses too few features or variables

## Answer

A

## Explanation

Overfitting occurs when a model learns the training data too well, including its noise and random fluctuations, rather than the underlying pattern. The model becomes overly complex and performs excellently on training data but generalizes poorly to new, unseen data. This is a key concern in machine learning and is addressed through techniques such as cross-validation, regularization, and limiting model complexity.

**Common mistakes:**
- Option B describes underfitting, the opposite problem where the model is too simple
- Option C also describes underfitting; overfitting typically involves too many features or excessive complexity
