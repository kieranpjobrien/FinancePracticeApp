# Question

Systematic sampling selects every kth element from a population. A potential problem with this method is:

## Options

- **A**: It requires too much computational power
- **B**: It may produce biased samples if there is a periodic pattern in the data
- **C**: It can only be used with very small populations

## Answer

B

## Explanation

Systematic sampling is efficient but can be biased if the population has periodicity that aligns with the sampling interval. For example, if sampling every 7th day from sales data that has weekly patterns, the sample might overrepresent one day of the week. When no such patterns exist, systematic sampling produces results similar to simple random sampling.

**Common mistakes:**
- Option A is incorrect; systematic sampling is computationally simple
- Option C is incorrect; systematic sampling works with large populations
