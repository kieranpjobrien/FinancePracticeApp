# Question

An ordinary annuity pays $500 per year for 5 years. If the discount rate is 8%, the present value of the annuity is closest to:

## Options

- **A**: $1,996
- **B**: $2,157
- **C**: $2,500

## Answer

A

## Explanation

For an ordinary annuity, PV = PMT × [(1 - (1 + r)^(-n)) / r]

PV = $500 × [(1 - (1.08)^(-5)) / 0.08]
PV = $500 × [(1 - 0.6806) / 0.08]
PV = $500 × [0.3194 / 0.08]
PV = $500 × 3.9927
PV = $1,996

**Common mistakes:**
- Option B uses the annuity due formula (payments at beginning of period) instead of ordinary annuity
- Option C simply multiplies the payment by the number of years, ignoring the time value of money
