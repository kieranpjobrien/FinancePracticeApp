# Question

Stratified random sampling is most appropriate when:

## Options

- **A**: The population is homogeneous
- **B**: The population contains distinct subgroups with different characteristics
- **C**: A very large sample size is available

## Answer

B

## Explanation

Stratified random sampling divides the population into mutually exclusive subgroups (strata) based on shared characteristics, then randomly samples from each stratum. This ensures representation from all important subgroups and can reduce sampling error when strata differ in their characteristics. Examples include stratifying by market cap, sector, or credit rating when constructing a bond index sample.

**Common mistakes:**
- Option A is incorrect; if the population is homogeneous, simple random sampling is sufficient
- Option C is not a reason for stratification; stratification improves efficiency regardless of sample size
