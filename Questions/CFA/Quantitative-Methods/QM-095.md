# Question

Data mining bias (data snooping) occurs when:

## Options

- **A**: The sample data is collected using non-random methods
- **B**: Researchers test many hypotheses on the same data until finding significant results
- **C**: Important variables are omitted from the analysis

## Answer

B

## Explanation

Data mining bias arises when researchers repeatedly test hypotheses on the same dataset until finding statistically significant results. With enough tests, some will appear significant by chance alone (false positives). Results may not replicate on new data. This is particularly problematic in finance where historical data is limited and researchers may unknowingly or knowingly exploit patterns that are spurious. Out-of-sample testing helps mitigate this risk.

**Common mistakes:**
- Option A describes sample selection bias, not data mining bias
- Option C describes omitted variable bias, a different type of specification error
