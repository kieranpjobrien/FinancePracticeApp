# Question

A screening test for recession correctly identifies 90% of actual recessions (sensitivity) and correctly identifies 85% of non-recessions (specificity). If the unconditional probability of a recession is 20%, and the test predicts a recession, what is the probability that a recession will actually occur?

## Options

- **A**: 60.0%
- **B**: 54.5%
- **C**: 90.0%

## Answer

A

## Explanation

Using Bayes' theorem:
- P(Recession) = 0.20, P(No Recession) = 0.80
- P(Positive | Recession) = 0.90 (sensitivity)
- P(Positive | No Recession) = 1 - 0.85 = 0.15 (false positive rate)

P(Positive) = P(Positive | Recession) × P(Recession) + P(Positive | No Recession) × P(No Recession)
P(Positive) = (0.90 × 0.20) + (0.15 × 0.80) = 0.18 + 0.12 = 0.30

P(Recession | Positive) = [P(Positive | Recession) × P(Recession)] / P(Positive)
P(Recession | Positive) = (0.90 × 0.20) / 0.30 = 0.18 / 0.30 = 0.60 = 60%

**Common mistakes:**
- Option B confuses the calculation by using incorrect conditional probabilities
- Option C incorrectly assumes the test's sensitivity is the posterior probability
