# Question

A continuous uniform distribution between 0 and 10 has a probability of:

## Options

- **A**: 0.1 for any single point
- **B**: 0.3 for values between 2 and 5
- **C**: Higher probability at the endpoints

## Answer

B

## Explanation

For a continuous uniform distribution on [a, b], the probability of falling in any subinterval [c, d] is:

P(c ≤ X ≤ d) = (d - c) / (b - a)

P(2 ≤ X ≤ 5) = (5 - 2) / (10 - 0) = 3/10 = 0.3

For continuous distributions, the probability at any single point is zero.

**Common mistakes:**
- Option A is incorrect; continuous distributions have zero probability at single points
- Option C is incorrect; uniform distributions have equal probability density throughout
