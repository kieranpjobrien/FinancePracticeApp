# Question

Principal Component Analysis (PCA) is primarily used to:

## Options

- **A**: Predict future values of a target variable
- **B**: Reduce dimensionality by transforming correlated variables into uncorrelated components
- **C**: Classify observations into predefined categories

## Answer

B

## Explanation

PCA is an unsupervised technique that transforms many correlated variables into a smaller number of uncorrelated principal components. Each component captures a portion of total variance, with the first component capturing the most. PCA is useful for: (1) reducing data dimensions while preserving information, (2) visualizing high-dimensional data, and (3) addressing multicollinearity in regression.

**Common mistakes:**
- Option A describes supervised learning techniques like regression
- Option C describes classification methods
