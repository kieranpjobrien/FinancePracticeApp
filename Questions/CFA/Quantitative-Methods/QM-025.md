# Question

In a two-asset portfolio where the assets have equal standard deviations of 20% and a correlation of -0.5, the minimum variance portfolio would allocate:

## Options

- **A**: 100% to the lower-returning asset
- **B**: 50% to each asset
- **C**: More than 50% to the higher-returning asset

## Answer

B

## Explanation

The minimum variance portfolio weight for asset 1 in a two-asset portfolio is:

w₁ = (σ₂² - ρσ₁σ₂) / (σ₁² + σ₂² - 2ρσ₁σ₂)

When both assets have equal standard deviations (σ₁ = σ₂ = σ):

w₁ = (σ² - ρσ²) / (2σ² - 2ρσ²)
w₁ = σ²(1 - ρ) / 2σ²(1 - ρ)
w₁ = 1/2 = 50%

With equal standard deviations, the minimum variance portfolio always allocates 50% to each asset, regardless of the correlation or expected returns. The minimum variance portfolio is determined solely by risk characteristics, not expected returns.

**Common mistakes:**
- Option A confuses minimum variance optimization with a strategy based on expected returns
- Option C incorrectly assumes expected returns affect the minimum variance allocation
