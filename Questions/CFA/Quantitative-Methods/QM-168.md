# Question

The purpose of splitting data into training and test sets is to:

## Options

- **A**: Increase the total amount of data available
- **B**: Evaluate model performance on data not used in training
- **C**: Reduce computational time

## Answer

B

## Explanation

Splitting data into training and test sets allows for honest evaluation of model performance. The model is built using training data, then its predictions are tested on the held-out test set that was not used in fitting. This helps detect overfitting—a model that performs well on training data but poorly on test data is likely overfit.

**Common mistakes:**
- Option A is incorrect; splitting reduces available training data
- Option C is incorrect; splitting doesn't significantly affect computational time
