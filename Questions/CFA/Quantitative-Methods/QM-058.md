# Question

A portfolio has annual returns of 10%, 15%, 5%, and 20% over four years. The sample standard deviation of returns is closest to:

## Options

- **A**: 5.59%
- **B**: 6.45%
- **C**: 7.07%

## Answer

B

## Explanation

Step 1: Calculate the mean
Mean = (10% + 15% + 5% + 20%) / 4 = 50% / 4 = 12.5%

Step 2: Calculate squared deviations
(10% - 12.5%)² = 6.25
(15% - 12.5%)² = 6.25
(5% - 12.5%)² = 56.25
(20% - 12.5%)² = 56.25
Sum = 125

Step 3: Sample variance (divide by n-1)
Variance = 125 / 3 = 41.67

Step 4: Standard deviation
SD = √41.67 = 6.45%

**Common mistakes:**
- Option A uses population standard deviation (dividing by n instead of n-1)
- Option C results from calculation errors in the squared deviations
