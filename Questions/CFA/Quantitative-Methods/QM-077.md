# Question

Regularization techniques such as LASSO and Ridge regression help address:

## Options

- **A**: Insufficient data for model training
- **B**: Overfitting by penalizing large coefficient values
- **C**: Missing values in the dataset

## Answer

B

## Explanation

Regularization adds a penalty term to the regression objective function that discourages large coefficient values. LASSO (L1 regularization) can shrink some coefficients exactly to zero, effectively performing variable selection. Ridge regression (L2 regularization) shrinks coefficients toward zero but does not eliminate them. Both techniques reduce model complexity and help prevent overfitting, improving the model's ability to generalize to new data.

**Common mistakes:**
- Option A is incorrect; regularization does not address data quantity issues
- Option C is incorrect; missing value imputation is a separate data preprocessing step
