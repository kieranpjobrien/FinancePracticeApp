# Question

A preferred stock pays a constant annual dividend of $3.00. If the required return is 6%, the value of this perpetuity is:

## Options

- **A**: $18.00
- **B**: $50.00
- **C**: $180.00

## Answer

B

## Explanation

For a perpetuity (constant payments forever):

PV = Payment / Discount Rate
PV = $3.00 / 0.06
PV = $50.00

The present value of a perpetuity is simply the payment divided by the required rate of return.

**Common mistakes:**
- Option A multiplies the dividend by the rate instead of dividing
- Option C multiplies the dividend by too large a factor, perhaps confusing with an annuity formula
