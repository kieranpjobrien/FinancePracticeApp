# Question

A portfolio manager must select 3 stocks from a list of 10 candidates. How many different combinations of stocks can be formed?

## Options

- **A**: 30
- **B**: 120
- **C**: 720

## Answer

B

## Explanation

This is a combination problem (order does not matter). The number of combinations is:

C(n,r) = n! / [r!(n-r)!]

C(10,3) = 10! / [3!(10-3)!]
C(10,3) = 10! / (3! × 7!)
C(10,3) = (10 × 9 × 8) / (3 × 2 × 1)
C(10,3) = 720 / 6
C(10,3) = 120

**Common mistakes:**
- Option A incorrectly calculates using 10 × 3 = 30
- Option C calculates permutations (where order matters) rather than combinations: P(10,3) = 10 × 9 × 8 = 720
