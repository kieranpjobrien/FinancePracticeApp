# Question

An investment earns a nominal return of 8% during a period when inflation is 3%. Using the exact formula, the real return is closest to:

## Options

- **A**: 4.85%
- **B**: 5.00%
- **C**: 5.24%

## Answer

A

## Explanation

The exact formula for real return is:

(1 + Real Return) = (1 + Nominal Return) / (1 + Inflation Rate)

Real Return = [(1 + 0.08) / (1 + 0.03)] - 1
Real Return = (1.08 / 1.03) - 1
Real Return = 1.0485 - 1
Real Return = 0.0485 = 4.85%

**Common mistakes:**
- Option B uses the approximation (Nominal - Inflation = 8% - 3% = 5%), which slightly overstates the real return
- Option C results from adding instead of dividing in the formula
