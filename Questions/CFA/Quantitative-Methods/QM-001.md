# Question

An investor will receive $10,000 in 3 years. If the discount rate is 8% per year, what is the present value of this payment?

## Options

- **A**: $7,938
- **B**: $8,573
- **C**: $9,259

## Answer

A

## Explanation

The present value formula is: PV = FV / (1 + r)^n

PV = $10,000 / (1.08)^3
PV = $10,000 / 1.2597
PV = $7,938.32 ≈ $7,938

**Common mistakes:**
- Option B ($8,573) results from using only 2 years instead of 3
- Option C ($9,259) results from using only 1 year instead of 3
