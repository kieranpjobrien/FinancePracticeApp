# Question

A portfolio consists of 60% in Asset A (standard deviation 15%) and 40% in Asset B (standard deviation 25%). If the correlation between the assets is 0.3, the portfolio standard deviation is closest to:

## Options

- **A**: 13.5%
- **B**: 15.3%
- **C**: 19.0%

## Answer

B

## Explanation

Portfolio variance = w₁²σ₁² + w₂²σ₂² + 2w₁w₂ρσ₁σ₂

= (0.6)²(0.15)² + (0.4)²(0.25)² + 2(0.6)(0.4)(0.3)(0.15)(0.25)
= (0.36)(0.0225) + (0.16)(0.0625) + 2(0.24)(0.3)(0.0375)
= 0.0081 + 0.01 + 0.0054
= 0.0235

Portfolio standard deviation = √0.0235 = 0.153 = 15.3%

**Common mistakes:**
- Option A results from calculation errors or using incorrect formula
- Option C represents the simple weighted average of standard deviations (0.6×15% + 0.4×25% = 19%), which ignores diversification benefits from imperfect correlation
