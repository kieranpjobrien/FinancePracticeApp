# Question

A weakness of using the range as a measure of dispersion is that it:

## Options

- **A**: Is difficult to calculate
- **B**: Uses only the two extreme values and ignores all other observations
- **C**: Cannot be used with ordinal data

## Answer

B

## Explanation

The range is simply the difference between the maximum and minimum values. While easy to calculate and interpret, it uses only two data points and is therefore highly sensitive to outliers. A single extreme value can dramatically change the range while the bulk of the data remains unchanged. More robust measures like standard deviation or interquartile range use information from all or most observations.

**Common mistakes:**
- Option A is incorrect; the range is one of the simplest statistics to calculate
- Option C is incorrect; the range can be used with ordinal data that has meaningful order
