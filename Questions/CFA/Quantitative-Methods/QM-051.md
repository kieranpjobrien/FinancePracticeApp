# Question

An investor purchases a stock for $50 and sells it one year later for $55, having received a $2 dividend during the year. The holding period return is:

## Options

- **A**: 10%
- **B**: 14%
- **C**: 4%

## Answer

B

## Explanation

Holding period return = (Ending Price - Beginning Price + Income) / Beginning Price

HPR = ($55 - $50 + $2) / $50
HPR = $7 / $50
HPR = 0.14 = 14%

The total return includes both capital appreciation ($5) and income ($2).

**Common mistakes:**
- Option A only considers capital appreciation, ignoring the dividend income
- Option C only considers dividend yield, ignoring capital appreciation
