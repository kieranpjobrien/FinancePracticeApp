# Question

In Monte Carlo simulation, the random numbers generated by computers are typically:

## Options

- **A**: Truly random numbers from physical processes
- **B**: Pseudo-random numbers generated by deterministic algorithms
- **C**: Taken directly from historical market data

## Answer

B

## Explanation

Computer-generated random numbers used in Monte Carlo simulation are pseudo-random—they are produced by mathematical algorithms that generate sequences appearing random but are actually deterministic. Given the same starting seed, they will produce identical sequences. For most simulation purposes, high-quality pseudo-random number generators are sufficient and allow for reproducibility when the same seed is used.

**Common mistakes:**
- Option A describes true random number generators using physical processes, which are rarely used in financial simulations
- Option C describes historical simulation (bootstrapping), not the random number generation used in standard Monte Carlo
