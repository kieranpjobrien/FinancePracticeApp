# Question

Unstructured data differs from structured data in that unstructured data:

## Options

- **A**: Is organized in rows and columns like a spreadsheet
- **B**: Includes formats like text, images, and audio that lack predefined organization
- **C**: Is always smaller in volume

## Answer

B

## Explanation

Structured data fits neatly into tables with defined fields (like databases or spreadsheets). Unstructured data lacks such organization—examples include social media posts, news articles, images, videos, and audio files. Big data often involves large volumes of unstructured data, requiring specialized techniques (NLP, computer vision) to analyze.

**Common mistakes:**
- Option A describes structured data
- Option C is incorrect; unstructured data often comprises the majority of data volume
