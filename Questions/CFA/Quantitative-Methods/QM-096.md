# Question

The critical value in hypothesis testing represents:

## Options

- **A**: The calculated test statistic from the sample data
- **B**: The boundary value beyond which the null hypothesis is rejected
- **C**: The p-value of the test

## Answer

B

## Explanation

The critical value is determined by the significance level (α) and the test distribution. It defines the rejection region boundary. If the calculated test statistic exceeds the critical value (in absolute terms for two-tailed tests), the null hypothesis is rejected. The critical value is determined before examining the data, based on the chosen significance level and whether the test is one-tailed or two-tailed.

**Common mistakes:**
- Option A describes the test statistic, which is compared to the critical value
- Option C describes the p-value, which is a probability, not a boundary value
