# Question

Events are exhaustive if:

## Options

- **A**: They cannot occur simultaneously
- **B**: They cover all possible outcomes
- **C**: They have equal probabilities

## Answer

B

## Explanation

Exhaustive events collectively cover the entire sample space—at least one of them must occur. The sum of probabilities of exhaustive events equals 1. For example, for a stock: {up, down, unchanged} is exhaustive because one of these must happen. "Exhaustive" means no possible outcome is left out.

**Common mistakes:**
- Option A describes mutually exclusive events
- Option C is not required for exhaustiveness; exhaustive events can have unequal probabilities
