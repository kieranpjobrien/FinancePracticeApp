# Question

An analyst wants to test whether a portfolio manager's returns are different from the benchmark (either higher or lower). The appropriate test is:

## Options

- **A**: A one-tailed test because we want to detect outperformance
- **B**: A two-tailed test because deviations in either direction are of interest
- **C**: A one-tailed test because we want to detect underperformance

## Answer

B

## Explanation

A two-tailed test is appropriate when the alternative hypothesis is that the parameter differs from a specified value in either direction (H₁: μ ≠ μ₀). Since the analyst wants to know if returns are "different" from the benchmark—whether higher or lower—a two-tailed test is correct. A one-tailed test would be used only if the analyst specifically wanted to test for outperformance (or specifically for underperformance), not both.

**Common mistakes:**
- Options A and C each represent a one-tailed test appropriate for testing directional hypotheses, but the question asks about any difference
