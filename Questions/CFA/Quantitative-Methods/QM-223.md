# Question

A spurious correlation most likely results from:

## Options

- **A**: Random chance or a common third variable affecting both variables
- **B**: Using too large a sample size
- **C**: Perfect multicollinearity

## Answer

A

## Explanation

A spurious correlation is a statistical relationship that does not reflect true causation. It can arise from: (1) pure coincidence, especially with data mining, (2) a confounding third variable that affects both measured variables, or (3) reverse causality. Discovering a correlation requires careful analysis to determine whether a causal relationship exists.

**Common mistakes:**
- Option B is incorrect; larger samples generally produce more reliable results
- Option C describes a different issue in multiple regression
