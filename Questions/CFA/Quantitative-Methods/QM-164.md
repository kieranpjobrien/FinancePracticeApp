# Question

Cohen's d is a measure of:

## Options

- **A**: The probability of rejecting the null hypothesis
- **B**: The standardized difference between two group means
- **C**: The degree of association between two variables

## Answer

B

## Explanation

Cohen's d measures effect size—the magnitude of the difference between groups in standardized units:

d = (Mean₁ - Mean₂) / Pooled Standard Deviation

Guidelines: d = 0.2 (small), d = 0.5 (medium), d = 0.8 (large)

Unlike p-values which depend on sample size, Cohen's d provides a standardized measure of practical significance that is independent of sample size.

**Common mistakes:**
- Option A describes statistical power or p-value concepts
- Option C describes correlation, not effect size between group means
