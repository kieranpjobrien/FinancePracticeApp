# Question

The Beneish M-Score is primarily used to:

## Options

- **A**: Predict future stock returns
- **B**: Detect potential earnings manipulation
- **C**: Calculate intrinsic value

## Answer

B

## Explanation

The Beneish M-Score uses eight financial ratios to identify potential earnings manipulation. Scores above -1.78 suggest higher manipulation probability. Key inputs include days sales in receivables index, gross margin index, asset quality index, and accruals. It's a forensic accounting tool, not a valuation model.

**Common mistakes:**
- Option A describes factor models or return predictors
- Option C describes DCF or comparable analysis
