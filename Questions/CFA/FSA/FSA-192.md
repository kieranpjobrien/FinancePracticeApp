# Question

A LIFO company reports inventory of $800,000 and a LIFO reserve of $150,000. The company's inventory on a FIFO basis would be:

## Options

- **A**: $650,000
- **B**: $800,000
- **C**: $950,000

## Answer

C

## Explanation

FIFO inventory = LIFO inventory + LIFO reserve = $800,000 + $150,000 = $950,000. The LIFO reserve represents the cumulative difference between LIFO and FIFO inventory values. Converting to FIFO allows comparison with IFRS companies and those using FIFO. LIFO COGS would be $150,000 higher than FIFO COGS.

**Common mistakes:**
- Option A subtracts instead of adds the LIFO reserve
- Option B ignores the LIFO reserve adjustment
