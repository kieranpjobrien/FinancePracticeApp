# Question

A zero-coupon bond is initially recorded at:

## Options

- **A**: Face value
- **B**: Present value of face value at market rate
- **C**: Present value of face value plus coupon payments

## Answer

B

## Explanation

Zero-coupon bonds pay no periodic interest—investors receive only face value at maturity. The bond is issued at a deep discount, recorded at its present value (proceeds received). Interest expense is recognized using the effective interest method as the discount amortizes, increasing carrying value toward face value.

**Common mistakes:**
- Option A ignores the discount at issuance
- Option C describes coupon-bearing bonds, not zero-coupon
