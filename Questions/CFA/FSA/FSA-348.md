# Question

A company has FCFF of $50 million, total debt of $200 million, and equity market value of $300 million. The FCFF yield is:

## Options

- **A**: 10%
- **B**: 16.7%
- **C**: 25%

## Answer

A

## Explanation

FCFF yield = FCFF / Enterprise value = FCFF / (Debt + Equity market value) = $50M / ($200M + $300M) = $50M / $500M = 10%. This measures cash return on total capital invested, useful for valuation and comparing companies with different capital structures.

**Common mistakes:**
- Option B uses only equity value: $50M / $300M = 16.7%
- Option C uses only debt: $50M / $200M = 25%
