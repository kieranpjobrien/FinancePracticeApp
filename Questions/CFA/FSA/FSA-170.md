# Question

In horizontal common-size analysis, each line item is expressed as a percentage of:

## Options

- **A**: Total assets or total revenue
- **B**: The same line item in a base year
- **C**: The industry average

## Answer

B

## Explanation

Horizontal (trend) analysis expresses each line item as a percentage of its base year value, showing growth or decline over time. For example, if Year 1 revenue is $100 and Year 3 is $130, Year 3 revenue = 130% of base. Vertical analysis uses total assets or revenue as the base.

**Common mistakes:**
- Option A describes vertical common-size analysis
- Option C describes benchmarking, not common-size analysis
