# Question

Cross-sectional analysis involves comparing:

## Options

- **A**: A company's performance across different time periods
- **B**: A company's ratios to industry peers at a point in time
- **C**: Different accounting methods for the same transaction

## Answer

B

## Explanation

Cross-sectional analysis compares a company's metrics to competitors or industry averages at a single point in time. It helps assess relative performance and competitive position. Time-series (trend) analysis compares the same company over time. Both are valuable and often used together.

**Common mistakes:**
- Option A describes time-series analysis
- Option C describes accounting policy comparison
