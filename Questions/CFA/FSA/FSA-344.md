# Question

Antidilutive securities are:

## Options

- **A**: Excluded from diluted EPS calculation
- **B**: Always included in diluted EPS calculation
- **C**: Only included if they increase net income

## Answer

A

## Explanation

Antidilutive securities would increase EPS if included (or produce EPS above basic). They're excluded from diluted EPS since diluted should be equal to or less than basic. Examples: out-of-the-money options, convertibles where interest savings exceed share dilution effect. Each security is tested for dilutive effect.

**Common mistakes:**
- Option B would overstate EPS
- Option C misunderstands the dilution concept
