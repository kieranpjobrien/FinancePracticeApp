# Question

Using the same company with current assets of $800,000 and current liabilities of $500,000, the current ratio is:

## Options

- **A**: 0.63
- **B**: 1.60
- **C**: 2.60

## Answer

B

## Explanation

Current ratio = Current assets / Current liabilities = $800,000 / $500,000 = 1.60. A current ratio above 1.0 indicates current assets exceed current liabilities. Generally, a ratio between 1.5 and 2.0 is considered healthy, though optimal levels vary by industry.

**Common mistakes:**
- Option A inverts the ratio (liabilities/assets)
- Option C adds 1 to the ratio incorrectly
