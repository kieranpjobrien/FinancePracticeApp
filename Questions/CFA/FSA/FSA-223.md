# Question

Classifying expenses by function rather than nature means reporting:

## Options

- **A**: Salaries, depreciation, and utilities as separate line items
- **B**: Cost of goods sold, selling expenses, and administrative expenses
- **C**: Fixed costs and variable costs separately

## Answer

B

## Explanation

Function classification groups expenses by purpose (COGS, selling, administrative, R&D). Nature classification shows expense types (salaries, depreciation, materials). US companies typically use function; IFRS allows either with nature disclosure if function is used. Function helps analyze gross and operating margins.

**Common mistakes:**
- Option A describes classification by nature
- Option C describes cost behavior classification, not financial statement presentation
