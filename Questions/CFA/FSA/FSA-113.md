# Question

Under IFRS, impairment losses on property, plant, and equipment can be:

## Options

- **A**: Never reversed under any circumstances
- **B**: Reversed if circumstances indicate recovery, up to original carrying value
- **C**: Reversed above original cost if market value increases

## Answer

B

## Explanation

IFRS allows impairment reversal for PP&E (except goodwill) if conditions improve, but only up to the carrying amount that would have existed without impairment (after normal depreciation). US GAAP prohibits reversals for assets held for use. Goodwill impairment cannot be reversed under either framework.

**Common mistakes:**
- Option A describes US GAAP treatment
- Option C would exceed the ceiling for reversal
