# Question

Historical cost is the primary measurement basis used for most assets because it is:

## Options

- **A**: Always more relevant than fair value
- **B**: More verifiable and reliable than estimates of current value
- **C**: Required for all assets under both IFRS and US GAAP

## Answer

B

## Explanation

Historical cost is based on actual transaction prices, making it verifiable and less subject to estimation. While fair value may be more relevant for some assets, it requires estimates that can be less reliable. Accounting standards balance relevance and reliability, using fair value where reliable measures exist and historical cost otherwise.

**Common mistakes:**
- Option A overstates—fair value is more relevant in some cases
- Option C is incorrect—many assets use fair value (investments, derivatives)
