# Question

Diluted EPS differs from basic EPS by:

## Options

- **A**: Excluding extraordinary items from net income
- **B**: Including the effect of potentially dilutive securities
- **C**: Using beginning-of-period shares instead of weighted average

## Answer

B

## Explanation

Diluted EPS adjusts for securities that could increase shares outstanding: stock options, convertible bonds, convertible preferred stock, and warrants. The calculation assumes these securities are converted/exercised if doing so would reduce EPS (be dilutive). Diluted EPS is always less than or equal to basic EPS.

**Common mistakes:**
- Option A is incorrect—both basic and diluted EPS use the same net income figure
- Option C is incorrect—both use weighted average shares
