# Question

A company has net income of $1,000,000, 500,000 common shares, and convertible bonds that would add $50,000 after-tax interest and 100,000 shares if converted. Diluted EPS is:

## Options

- **A**: $1.67
- **B**: $1.75
- **C**: $2.00

## Answer

B

## Explanation

Diluted EPS = (Net income + After-tax interest saved) / (Shares + Convertible shares) = ($1,000,000 + $50,000) / (500,000 + 100,000) = $1,050,000 / 600,000 = $1.75. Basic EPS = $1,000,000 / 500,000 = $2.00. Since diluted ($1.75) < basic ($2.00), the convertibles are dilutive and included.

**Common mistakes:**
- Option A results from calculation error
- Option C is basic EPS, not diluted
