# Question

If ROE exceeds ROA, the company:

## Options

- **A**: Has no debt
- **B**: Has profitable use of financial leverage
- **C**: Has declining sales

## Answer

B

## Explanation

ROE = ROA × Financial leverage multiplier. When ROE > ROA, financial leverage is positive—borrowed funds earn returns exceeding their cost, amplifying returns to equity. If ROA exceeds borrowing cost, debt increases ROE. If ROA < borrowing cost, leverage would reduce ROE below ROA.

**Common mistakes:**
- Option A would mean ROE = ROA (no leverage effect)
- Option C is unrelated to the ROE-ROA relationship
