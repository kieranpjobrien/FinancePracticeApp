# Question

An asset has a carrying value of $80,000 and a tax base of $60,000. If the tax rate is 25%, the deferred tax liability is:

## Options

- **A**: $5,000
- **B**: $15,000
- **C**: $20,000

## Answer

A

## Explanation

Taxable temporary difference = Carrying value - Tax base = $80,000 - $60,000 = $20,000. DTL = Temporary difference × Tax rate = $20,000 × 25% = $5,000. Carrying value exceeds tax base because tax depreciation has been faster, creating future taxable amounts when the book/tax difference reverses.

**Common mistakes:**
- Option B uses the wrong base ($60,000 × 25%)
- Option C uses the temporary difference without applying tax rate
