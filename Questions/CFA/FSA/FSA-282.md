# Question

Management Discussion and Analysis (MD&A) provides:

## Options

- **A**: Audited financial data only
- **B**: Management's perspective on financial condition, results, and future prospects
- **C**: A summary of accounting policies used

## Answer

B

## Explanation

MD&A offers management's view of operations, financial condition, liquidity, and known trends or uncertainties. It provides context for understanding financial statements, including explanations of changes between periods. It's not audited (though reviewed) and is required for public companies. Notes to statements contain accounting policy details.

**Common mistakes:**
- Option A is incorrect—MD&A is not audited
- Option C describes notes to financial statements
