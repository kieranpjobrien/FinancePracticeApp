# Question

Under IFRS, investment property can be measured using:

## Options

- **A**: Only the cost model
- **B**: Either the cost model or fair value model
- **C**: Only the fair value model

## Answer

B

## Explanation

IFRS allows a choice between cost model (historical cost less depreciation and impairment) and fair value model (remeasured each period with changes in profit or loss) for investment property. US GAAP generally requires the cost model. The choice significantly affects reported assets and earnings.

**Common mistakes:**
- Option A ignores the fair value alternative
- Option C ignores the cost model alternative
