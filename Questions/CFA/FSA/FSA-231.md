# Question

Under a periodic inventory system, COGS is calculated:

## Options

- **A**: With each sale as it occurs
- **B**: At the end of the accounting period using physical count
- **C**: Using real-time inventory tracking

## Answer

B

## Explanation

Periodic systems calculate COGS at period end: Beginning inventory + Purchases - Ending inventory (from physical count) = COGS. Perpetual systems (Options A and C) track inventory continuously, updating COGS with each sale. Periodic is simpler but provides less timely inventory information.

**Common mistakes:**
- Option A describes perpetual inventory system
- Option C describes perpetual system with technology
