# Question

Horizontal analysis compares financial data:

## Options

- **A**: Across different companies in the same period
- **B**: Across different time periods for the same company
- **C**: As percentages of a base item within the same period

## Answer

B

## Explanation

Horizontal analysis examines trends over time by comparing financial statement items across periods. It can show absolute changes or percentage changes from a base year. This reveals growth patterns, cyclicality, and changes in financial position. Vertical (common-size) analysis compares items within a period.

**Common mistakes:**
- Option A describes cross-sectional analysis
- Option C describes vertical/common-size analysis
