# Question

For a company with a long operating cycle (over one year), the current/non-current classification is based on:

## Options

- **A**: Strictly the one-year rule
- **B**: The length of the operating cycle
- **C**: Management's preference

## Answer

B

## Explanation

When the operating cycle exceeds one year (construction, distilleries), current assets/liabilities are those expected to be realized/settled within the operating cycle. This ensures working capital reflects the actual business cycle. Most businesses have cycles shorter than a year, making the distinction moot.

**Common mistakes:**
- Option A ignores the operating cycle exception
- Option C overstates management discretion in classification
