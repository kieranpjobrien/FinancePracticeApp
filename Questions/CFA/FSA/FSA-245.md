# Question

Fair value hierarchy Level 1 inputs are:

## Options

- **A**: Observable inputs other than quoted prices
- **B**: Quoted prices in active markets for identical assets/liabilities
- **C**: Unobservable inputs based on entity's own assumptions

## Answer

B

## Explanation

Fair value hierarchy: Level 1 = quoted prices in active markets for identical items (most reliable); Level 2 = observable inputs including quoted prices for similar items, interest rates, yield curves; Level 3 = unobservable inputs using entity assumptions (least reliable). Disclosure of levels is required.

**Common mistakes:**
- Option A describes Level 2 inputs
- Option C describes Level 3 inputs
