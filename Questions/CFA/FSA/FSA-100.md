# Question

The defensive interval ratio measures:

## Options

- **A**: How many times current assets cover current liabilities
- **B**: How many days a company can operate using only liquid assets
- **C**: The percentage of assets financed by equity

## Answer

B

## Explanation

Defensive interval = (Cash + Marketable securities + Receivables) / Daily cash expenditures. It estimates how long a company can pay operating expenses from liquid assets without additional cash inflows. Unlike the current or quick ratio, it converts liquidity into a time-based measure.

**Common mistakes:**
- Option A describes the current ratio
- Option C describes the equity ratio
