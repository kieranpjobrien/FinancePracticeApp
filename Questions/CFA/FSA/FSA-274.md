# Question

Equipment costs $500,000, has salvage value of $50,000, and total estimated production of 900,000 units. If 120,000 units are produced in Year 1, depreciation expense is:

## Options

- **A**: $50,000
- **B**: $60,000
- **C**: $66,667

## Answer

B

## Explanation

Depreciation per unit = (Cost - Salvage) / Total units = ($500,000 - $50,000) / 900,000 = $0.50 per unit. Year 1 depreciation = 120,000 units × $0.50 = $60,000. Units-of-production matches expense to actual usage, appropriate when asset wear correlates with production rather than time.

**Common mistakes:**
- Option A ignores cost and uses only salvage value
- Option C ignores salvage value: $500,000 / 900,000 × 120,000 = $66,667
