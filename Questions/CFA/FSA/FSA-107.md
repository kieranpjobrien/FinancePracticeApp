# Question

Management's Discussion and Analysis (MD&A) is useful because it provides:

## Options

- **A**: Audited financial information
- **B**: Management's perspective on financial results and future outlook
- **C**: A standardized format required by accounting standards

## Answer

B

## Explanation

MD&A provides management's narrative about operations, financial condition, liquidity, capital resources, and known trends affecting future performance. It complements quantitative statements with qualitative context. MD&A is required by securities regulators (like the SEC) but is not audited like the financial statements themselves.

**Common mistakes:**
- Option A is incorrect—MD&A is not audited
- Option C is incorrect—format varies and is regulatory, not accounting standard
