# Question

The inventory disclosure in notes typically includes:

## Options

- **A**: Daily sales projections
- **B**: Cost method used and breakdown by category
- **C**: Customer names and order details

## Answer

B

## Explanation

Inventory notes disclose: costing method (FIFO, LIFO, weighted average), amounts by category (raw materials, WIP, finished goods), any write-downs recognized, and LIFO reserve if applicable. This enables analysis of inventory composition, age, and comparability adjustments between companies.

**Common mistakes:**
- Option A is forward-looking, not disclosed in notes
- Option C is confidential business information, not disclosed
