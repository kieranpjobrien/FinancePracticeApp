# Question

When a company elects the fair value option for its debt:

## Options

- **A**: Fair value increases from credit deterioration reduce reported liabilities
- **B**: Fair value changes are ignored until maturity
- **C**: Only interest expense is affected

## Answer

A

## Explanation

Under the fair value option, own credit risk changes affect fair value. If credit deteriorates, debt fair value decreases (counterparty would pay less), creating a gain. This counterintuitive result shows struggling companies reporting gains from their own credit problems. OCI presentation now isolates this effect.

**Common mistakes:**
- Option B ignores the mark-to-market requirement
- Option C ignores fair value changes in the principal amount
