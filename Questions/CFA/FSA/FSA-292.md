# Question

To compare inventory turnover of a LIFO company to a FIFO company, an analyst should:

## Options

- **A**: Use reported figures directly since GAAP ensures comparability
- **B**: Convert LIFO inventory to FIFO using the LIFO reserve
- **C**: Only compare days inventory on hand, not turnover

## Answer

B

## Explanation

LIFO understates inventory during rising prices, artificially inflating turnover ratios. To compare with FIFO companies: FIFO inventory = LIFO inventory + LIFO reserve. FIFO COGS = LIFO COGS - Change in LIFO reserve. This creates comparable metrics. Days inventory would still differ without conversion.

**Common mistakes:**
- Option A ignores the distortion from different methods
- Option C doesn't solve the underlying comparability issue
