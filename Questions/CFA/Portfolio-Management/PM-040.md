# Question

A two-asset portfolio has equal weights. Asset 1 has variance of 0.04 and Asset 2 has variance of 0.09. The covariance between them is 0.02. The portfolio variance is closest to:

## Options

- **A**: 0.0325
- **B**: 0.0425
- **C**: 0.0650

## Answer

B

## Explanation

Portfolio variance formula for two assets:

σ²p = w₁²σ₁² + w₂²σ₂² + 2w₁w₂Cov(1,2)

Where:
- w₁ = w₂ = 0.50
- σ₁² = 0.04
- σ₂² = 0.09
- Cov(1,2) = 0.02

σ²p = (0.50)²(0.04) + (0.50)²(0.09) + 2(0.50)(0.50)(0.02)
σ²p = 0.25(0.04) + 0.25(0.09) + 0.50(0.02)
σ²p = 0.01 + 0.0225 + 0.01
σ²p = 0.0425

Portfolio standard deviation = √0.0425 = 20.6%

**Common mistakes:**
- Option A: May omit or miscalculate the covariance term
- Option C: May use simple average of variances (0.04 + 0.09)/2 = 0.065
