# Question

An investor buys a stock for $50 and sells it one year later for $55 after receiving a $2 dividend. The holding period return is:

## Options

- **A**: 10.0%
- **B**: 14.0%
- **C**: 16.0%

## Answer

B

## Explanation

Holding period return (HPR):

HPR = (Ending Value - Beginning Value + Income) / Beginning Value

Or equivalently:
HPR = (P₁ - P₀ + D) / P₀

Where:
- P₀ = Beginning price = $50
- P₁ = Ending price = $55
- D = Dividend = $2

HPR = ($55 - $50 + $2) / $50
HPR = $7 / $50
HPR = 0.14 = 14%

**Components:**
- Capital gain: ($55 - $50) / $50 = 10%
- Dividend yield: $2 / $50 = 4%
- Total return: 14%

**Common mistakes:**
- Option A: Only includes capital gain, ignores dividend
- Option C: May add percentages incorrectly
