# Question

A top-down investment approach starts with:

## Options

- **A**: Individual security analysis
- **B**: Macroeconomic analysis and asset allocation
- **C**: Technical chart patterns

## Answer

B

## Explanation

Top-down vs. bottom-up:

**Top-down:**
1. Macroeconomic outlook
2. Asset class allocation
3. Country/sector selection
4. Security selection

- Starts broad, narrows down
- Macro factors drive decisions

**Bottom-up:**
1. Individual security analysis
2. Company fundamentals
3. Build portfolio from best ideas
4. Less concerned with macro

- Starts with securities
- Fundamentals drive decisions

**Hybrid approaches:**
- Combine both perspectives
- Macro overlay with fundamental selection

**Common mistakes:**
- Option A: This is bottom-up
- Option C: Technical analysis is separate methodology
