# Question

Total return on an investment includes:

## Options

- **A**: Only price appreciation
- **B**: Price appreciation plus income (dividends or interest)
- **C**: Only dividend income

## Answer

B

## Explanation

Total return components:

**Formula:**
Total Return = (P₁ - P₀ + Income) / P₀

**Two components:**
1. **Capital gain/loss:** Price change
2. **Income:** Dividends (stocks) or interest (bonds)

**Why use total return:**
- Complete picture of investment performance
- Allows comparison across asset types
- Some investments focus on income, others on growth
- Reinvestment assumption matters

**Example:**
- Buy stock at $100
- Receive $3 dividend
- Sell at $105
- Total return = ($105 - $100 + $3) / $100 = 8%

**Common mistakes:**
- Option A: Ignores income, understates return
- Option C: Ignores capital gains, incomplete measure
