# Question

An investor's optimal portfolio is found where:

## Options

- **A**: The efficient frontier reaches its maximum point
- **B**: The investor's highest indifference curve is tangent to the efficient frontier
- **C**: All assets have equal weights

## Answer

B

## Explanation

Optimal portfolio selection:

**Process:**
1. Construct efficient frontier
2. Overlay investor's indifference curves
3. Find tangency point

**At tangency:**
- Highest achievable utility
- Best risk-return for this investor
- Different investors → different optimal points

**Indifference curves:**
- Represent constant utility
- Slope reflects risk aversion
- Higher curves = higher utility

**Result:**
- More risk-averse → point toward left
- Less risk-averse → point toward right
- With Rf → tangency portfolio + leverage

**Common mistakes:**
- Option A: Maximum frontier point may not suit investor
- Option C: Equal weighting rarely optimal
