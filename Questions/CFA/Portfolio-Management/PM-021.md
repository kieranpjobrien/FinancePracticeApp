# Question

A retiree has a $2,000,000 portfolio and needs $90,000 annually for living expenses. Inflation is expected to average 2.5% annually. The retiree's required real return is closest to:

## Options

- **A**: 2.0%
- **B**: 4.5%
- **C**: 7.0%

## Answer

B

## Explanation

The required real return is the return needed after accounting for inflation to maintain purchasing power while meeting spending needs.

Step 1: Calculate spending rate
Spending rate = Annual spending / Portfolio value
Spending rate = $90,000 / $2,000,000 = 4.5%

Step 2: Understand real vs. nominal returns
- Real return = return above inflation
- To maintain real portfolio value while spending 4.5% annually, the portfolio needs a 4.5% real return
- The nominal return needed = real return + inflation = 4.5% + 2.5% = 7.0%

The required **real return** is 4.5%.

**Common mistakes:**
- Option A (2.0%): May subtract inflation from spending rate incorrectly
- Option C (7.0%): This is the nominal return needed, not the real return
