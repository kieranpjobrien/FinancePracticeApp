# Question

For evaluating portfolio manager performance, time-weighted return is preferred over money-weighted return because:

## Options

- **A**: Time-weighted return gives more weight to recent performance
- **B**: Time-weighted return eliminates the impact of client cash flows
- **C**: Time-weighted return is easier to calculate

## Answer

B

## Explanation

Time-weighted vs. Money-weighted returns:

**Time-weighted return (TWR):**
- Geometrically links sub-period returns
- Eliminates effect of external cash flows
- Measures manager's investment decisions
- Standard for manager performance comparison

**Money-weighted return (MWR/IRR):**
- Internal rate of return considering all cash flows
- Affected by timing and size of client contributions/withdrawals
- Measures investor's actual experience
- Manager doesn't control when clients add/withdraw funds

**Why TWR for managers:**
- Clients control cash flow timing
- Large inflow before poor period penalizes MWR
- Fair comparison across managers with different cash flow patterns

**Common mistakes:**
- Option A: TWR weights each period equally, not recent periods more
- Option C: TWR requires more frequent valuations; actually harder to calculate
