# Question

An investor in the 30% tax bracket earns a 10% pretax return. The after-tax return is:

## Options

- **A**: 3%
- **B**: 7%
- **C**: 10%

## Answer

B

## Explanation

After-tax return:

**Formula:**
After-tax return = Pretax return × (1 - Tax rate)

**Calculation:**
After-tax = 10% × (1 - 0.30)
After-tax = 10% × 0.70
After-tax = 7%

**Considerations:**
- Taxes reduce effective returns
- Tax efficiency matters
- Different treatments for:
  - Ordinary income (interest)
  - Qualified dividends
  - Short-term vs. long-term capital gains

**Tax-advantaged accounts:**
- Defer or eliminate this drag
- Consider asset location

**Common mistakes:**
- Option A: Subtracts tax rate from return (10% - 30%)
- Option C: Ignores taxes entirely
