# Question

A stock has a beta of 1.3. The risk-free rate is 3% and the expected market return is 10%. According to CAPM, if the stock's expected return is 12%, the stock is:

## Options

- **A**: Undervalued (plots above the SML)
- **B**: Fairly valued (plots on the SML)
- **C**: Overvalued (plots below the SML)

## Answer

C

## Explanation

Using CAPM to find required return:
E(Ri) = Rf + βi × [E(Rm) - Rf]
E(Ri) = 3% + 1.3 × (10% - 3%)
E(Ri) = 3% + 1.3 × 7%
E(Ri) = 3% + 9.1%
E(Ri) = 12.1%

Comparison:
- CAPM required return: 12.1%
- Stock's expected return: 12.0%
- Alpha = 12.0% - 12.1% = -0.1%

Since the expected return (12%) is less than the CAPM required return (12.1%), the stock has negative alpha and plots below the Security Market Line. This means it is overvalued—it doesn't offer enough return for its level of systematic risk.

**Common mistakes:**
- Option A incorrectly assumes the stock is undervalued; undervaluation occurs when expected return exceeds required return
- Option B would require expected return to equal required return exactly
