# Question

An appropriate benchmark for evaluating portfolio performance should be:

## Options

- **A**: The portfolio manager's past performance
- **B**: Specified in advance, investable, and measurable
- **C**: Based on the best-performing portfolio in the peer group

## Answer

B

## Explanation

Benchmark characteristics:

**Key properties of a valid benchmark:**
1. **Specified in advance:** Known before evaluation period
2. **Appropriate:** Matches investment style and mandate
3. **Measurable:** Returns can be calculated
4. **Investable:** Manager could replicate it
5. **Unambiguous:** Clearly defined components
6. **Reflective of current investment opinions:** Not purely passive
7. **Owned:** Manager accepts it as appropriate

**Why these matter:**
- Prevents benchmark cherry-picking after the fact
- Enables fair comparison
- Provides meaningful information about manager skill

**Common mistakes:**
- Option A: Self-reference isn't meaningful comparison
- Option C: Best performer changes; creates survivorship bias
