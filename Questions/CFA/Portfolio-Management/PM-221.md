# Question

The Herfindahl-Hirschman Index (HHI) in portfolio analysis measures:

## Options

- **A**: Expected portfolio return
- **B**: Portfolio concentration based on position weights
- **C**: Correlation between assets

## Answer

B

## Explanation

Herfindahl-Hirschman Index (HHI):

**Formula:**
HHI = Σwi²

Where wi = weight of position i

**Interpretation:**
- Sum of squared weights
- Range: 1/n to 1.0
- Higher = more concentrated
- Lower = more diversified

**Examples:**
- Equal weighted (10 stocks): HHI = 10 × (0.1)² = 0.10
- Single stock: HHI = 1.0
- 50% in one, 50% other: HHI = 0.25 + 0.25 = 0.50

**Uses:**
- Measure portfolio concentration
- Compare diversification levels
- Risk management limits

**Effective number of positions:**
= 1/HHI

**Common mistakes:**
- Option A: HHI measures concentration, not return
- Option C: Correlation not part of HHI calculation
