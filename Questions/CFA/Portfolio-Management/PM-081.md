# Question

A stock's returns are regressed against market returns, yielding: Ri = 2% + 1.3Rm. If the market return is 8%, the stock's expected return is:

## Options

- **A**: 10.4%
- **B**: 12.4%
- **C**: 13.0%

## Answer

B

## Explanation

The regression equation represents the market model:

Ri = α + βRm

Where:
- α = 2% (intercept/alpha)
- β = 1.3 (slope/beta)
- Rm = 8% (market return)

Expected return:
Ri = 2% + 1.3(8%)
Ri = 2% + 10.4%
Ri = 12.4%

**Interpretation:**
- The stock has a beta of 1.3 (more volatile than market)
- Alpha of 2% suggests historical outperformance
- When market returns 8%, stock expected to return 12.4%

**Common mistakes:**
- Option A: Calculates only the beta component (1.3 × 8%)
- Option C: May use incorrect multiplication or addition
