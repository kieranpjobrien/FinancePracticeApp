# Question

A portfolio reached a peak value of $1,200,000, subsequently fell to $900,000, then recovered to $1,100,000. The maximum drawdown is:

## Options

- **A**: 8.3%
- **B**: 25.0%
- **C**: 33.3%

## Answer

B

## Explanation

Maximum drawdown:

**Definition:**
- Largest peak-to-trough decline in portfolio value
- Measures worst cumulative loss from any high point
- Expressed as percentage of peak value

**Calculation:**
Maximum Drawdown = (Peak - Trough) / Peak
Maximum Drawdown = ($1,200,000 - $900,000) / $1,200,000
Maximum Drawdown = $300,000 / $1,200,000
Maximum Drawdown = 0.25 = 25%

**Key points:**
- Subsequent recovery is irrelevant to drawdown calculation
- Measures historical worst-case experience
- Important for investor psychology and risk capacity

**Uses:**
- Risk assessment
- Manager evaluation
- Setting stop-loss levels

**Common mistakes:**
- Option A: May calculate from final value instead of trough
- Option C: May calculate trough relative to current value
