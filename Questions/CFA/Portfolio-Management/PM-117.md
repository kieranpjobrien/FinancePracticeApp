# Question

A portfolio has an expected return of 10% and standard deviation of 15%. Using Roy's Safety-First criterion, what is the probability of the return falling below a threshold of 4%? (z = 0.40 corresponds to probability of 34.5%)

## Options

- **A**: 15.5%
- **B**: 34.5%
- **C**: 65.5%

## Answer

B

## Explanation

Roy's Safety-First Ratio (SFRatio):

**Formula:**
SFRatio = (E(Rp) - RL) / σp

Where:
- E(Rp) = Expected return = 10%
- RL = Threshold/minimum acceptable return = 4%
- σp = Standard deviation = 15%

**Calculation:**
SFRatio = (10% - 4%) / 15%
SFRatio = 6% / 15%
SFRatio = 0.40

**Interpretation:**
- The threshold is 0.40 standard deviations below the mean
- z = 0.40 corresponds to approximately 34.5% probability in left tail
- 34.5% probability of falling below 4%

**Common mistakes:**
- Option A: May use wrong z-table lookup or calculation error
- Option C: This is probability of exceeding the threshold, not falling below
