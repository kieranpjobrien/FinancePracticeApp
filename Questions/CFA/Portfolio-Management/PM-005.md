# Question

A portfolio that lies on the efficient frontier is best described as one that:

## Options

- **A**: Has the highest expected return among all possible portfolios
- **B**: Offers the maximum expected return for a given level of risk
- **C**: Has zero correlation among all its constituent assets

## Answer

B

## Explanation

The efficient frontier represents the set of optimal portfolios that offer the highest expected return for a given level of risk (or equivalently, the lowest risk for a given expected return). Each point on the efficient frontier is "efficient" because:

- No other portfolio exists with the same risk but higher return
- No other portfolio exists with the same return but lower risk

This is the foundation of Markowitz mean-variance optimization.

**Common mistakes:**
- Option A: Only the single portfolio at the top of the efficient frontier has the absolute highest return; the frontier contains many portfolios at different risk levels
- Option C: While low correlations help achieve efficiency, zero correlation is neither required nor typically possible; efficient portfolios are defined by their risk-return characteristics, not correlation constraints
