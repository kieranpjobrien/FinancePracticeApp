# Question

The global minimum variance portfolio:

## Options

- **A**: Offers the highest expected return of all portfolios
- **B**: Has the lowest risk of any portfolio on the efficient frontier
- **C**: Contains only risk-free assets

## Answer

B

## Explanation

Global minimum variance portfolio:

**Definition:**
- Lowest possible portfolio variance
- Located at leftmost point of frontier
- Minimum risk achievable with risky assets

**Characteristics:**
- Lowest standard deviation
- Not the highest return
- Starting point of efficient frontier
- Determined by covariances

**Location:**
- Bottom of the Markowitz bullet
- Where efficient frontier begins
- Transition point to efficient portion

**Composition:**
- Depends on asset correlations
- May not be intuitive weights
- Low-volatility, low-correlation assets
- Contains only risky assets

**Trade-off:**
- Lower risk = lower expected return
- Not optimal for most investors
- Risk-averse investors may prefer

**Common mistakes:**
- Option A: Higher returns require more risk
- Option C: Contains risky assets, not risk-free
