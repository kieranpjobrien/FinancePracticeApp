# Question

A criticism of traditional mean-variance optimization is that:

## Options

- **A**: It always produces portfolios with too little diversification
- **B**: Small changes in inputs can produce dramatically different optimal portfolios
- **C**: It ignores expected returns in the optimization process

## Answer

B

## Explanation

Mean-variance optimization limitations:

**Input sensitivity:**
- Small changes in expected returns → large weight changes
- Optimization "maximizes" estimation errors
- Often produces extreme, concentrated portfolios

**Other criticisms:**
1. Assumes normal distribution (ignores skewness, kurtosis)
2. Single-period framework
3. Uses historical data for forward estimates
4. Ignores transaction costs, taxes
5. May produce impractical portfolios

**Solutions/improvements:**
- Constraints on weights
- Resampling/Monte Carlo methods
- Black-Litterman model
- Robust optimization
- Factor-based approaches

**Common mistakes:**
- Option A: MVO can produce overly concentrated, not diversified portfolios
- Option C: Expected returns are key inputs; it doesn't ignore them
