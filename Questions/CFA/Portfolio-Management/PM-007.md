# Question

An investor can choose between Stock A (expected return 12%, standard deviation 18%) and Stock B (expected return 8%, standard deviation 12%). The correlation between the stocks is 0.3. The weight of Stock A in the minimum variance portfolio is closest to:

## Options

- **A**: 22%
- **B**: 35%
- **C**: 50%

## Answer

A

## Explanation

The weight of Asset A in the minimum variance two-asset portfolio is:

wA = (σ²B - Cov(A,B)) / (σ²A + σ²B - 2×Cov(A,B))

First, calculate covariance:
Cov(A,B) = ρAB × σA × σB = 0.3 × 0.18 × 0.12 = 0.00648

Then:
σ²A = (0.18)² = 0.0324
σ²B = (0.12)² = 0.0144

wA = (0.0144 - 0.00648) / (0.0324 + 0.0144 - 2 × 0.00648)
wA = 0.00792 / (0.0468 - 0.01296)
wA = 0.00792 / 0.03384
wA = 0.234 ≈ 22%

The minimum variance portfolio places more weight on the lower-volatility asset (Stock B) to minimize overall portfolio risk.

**Common mistakes:**
- Option B: Calculation errors in the covariance or formula application
- Option C: Assumes equal weighting minimizes variance, which is only true when assets have equal standard deviations
