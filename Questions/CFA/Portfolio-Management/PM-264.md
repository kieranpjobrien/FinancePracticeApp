# Question

A portfolio manager with low tracking error but positive alpha is most likely:

## Options

- **A**: Taking large bets away from the benchmark
- **B**: Consistently adding small incremental value versus benchmark
- **C**: Running a pure passive index strategy

## Answer

B

## Explanation

Tracking error and alpha relationship:

**Low tracking error:**
- Returns close to benchmark
- Small active positions
- "Benchmark hugger" or enhanced indexer

**Positive alpha:**
- Outperforming benchmark
- Adding value

**Combined interpretation:**
- Small but consistent outperformance
- High information ratio potential
- Efficient use of active risk

**Information ratio:**
IR = Alpha / Tracking error
- High IR = consistent skill
- Low TE + positive alpha = efficient management

**Common mistakes:**
- Option A: Large bets create high tracking error
- Option C: Passive should have zero alpha
