# Question

A target-date fund's glide path refers to:

## Options

- **A**: The fund's benchmark index performance over time
- **B**: The predetermined change in asset allocation as the target date approaches
- **C**: The fund's historical returns since inception

## Answer

B

## Explanation

Target-date fund glide path:

**Definition:**
- Pre-set schedule of asset allocation changes over time
- Typically reduces equity exposure as target date nears
- Increases fixed income allocation approaching retirement

**Typical glide path:**
- Age 25: 90% equity, 10% fixed income
- Age 45: 70% equity, 30% fixed income
- Age 65: 50% equity, 50% fixed income
- Post-retirement: Continues adjusting (if "through" fund)

**Types:**
- "To" retirement: Allocation static at target date
- "Through" retirement: Continues adjusting after target date

**Benefits:**
- Automatic rebalancing
- Age-appropriate risk
- Simplified decision-making

**Common mistakes:**
- Option A: Glide path is about allocation, not benchmark performance
- Option C: Glide path is forward-looking allocation plan, not historical returns
