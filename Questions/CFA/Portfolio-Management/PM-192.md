# Question

Compared to CAPM, Arbitrage Pricing Theory (APT):

## Options

- **A**: Uses fewer factors to explain returns
- **B**: Does not specify which factors affect returns but allows for multiple factors
- **C**: Requires the market portfolio to be observable

## Answer

B

## Explanation

APT vs. CAPM:

**CAPM:**
- Single factor (market)
- Market portfolio must be specified
- More restrictive assumptions
- Clear theoretical foundation

**APT:**
- Multiple factors (unspecified)
- No market portfolio needed
- Less restrictive assumptions
- Factors determined empirically

**APT model:**
E(Ri) = Rf + β₁F₁ + β₂F₂ + ... + βₖFₖ

**APT factors (examples):**
- Industrial production
- Inflation
- Interest rates
- Credit spreads
- Market (can include)

**APT advantage:**
- More flexible
- Potentially more accurate
- Doesn't require market portfolio

**Common mistakes:**
- Option A: APT uses more factors, not fewer
- Option C: APT doesn't require market portfolio; CAPM does
