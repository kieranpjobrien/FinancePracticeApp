# Question

A stock has a beta of 1.4 and generated a return of 18% when the market returned 12% and the risk-free rate was 3%. The stock's alpha is closest to:

## Options

- **A**: 2.4%
- **B**: 3.0%
- **C**: 6.0%

## Answer

A

## Explanation

Alpha = Actual Return - CAPM Expected Return

First, calculate expected return using CAPM:
E(R) = Rf + β[E(Rm) - Rf]
E(R) = 3% + 1.4(12% - 3%)
E(R) = 3% + 1.4(9%)
E(R) = 3% + 12.6%
E(R) = 15.6%

Then calculate alpha:
α = Actual Return - Expected Return
α = 18% - 15.6%
α = 2.4%

Positive alpha indicates the stock outperformed its CAPM expected return.

**Common mistakes:**
- Option B: May use incorrect beta or market risk premium
- Option C: Calculates actual minus market return (18% - 12%) instead of actual minus expected
