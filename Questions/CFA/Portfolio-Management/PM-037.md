# Question

An investor has a risk aversion coefficient of 4. Portfolio X has an expected return of 12% and standard deviation of 18%. Portfolio Y has an expected return of 9% and standard deviation of 10%. Using U = E(R) - ½Aσ², the investor would prefer:

## Options

- **A**: Portfolio X with utility of 0.0552
- **B**: Portfolio Y with utility of 0.0700
- **C**: Portfolio X with utility of 0.1200

## Answer

B

## Explanation

Calculate utility for each portfolio using U = E(R) - ½Aσ²

Portfolio X:
U = 0.12 - ½(4)(0.18)²
U = 0.12 - 2(0.0324)
U = 0.12 - 0.0648
U = 0.0552

Portfolio Y:
U = 0.09 - ½(4)(0.10)²
U = 0.09 - 2(0.01)
U = 0.09 - 0.02
U = 0.0700

Portfolio Y has higher utility (0.0700 > 0.0552), so the investor prefers Portfolio Y despite its lower expected return.

**Common mistakes:**
- Option A: Correctly calculates X's utility but doesn't compare to Y
- Option C: Ignores the risk penalty entirely
