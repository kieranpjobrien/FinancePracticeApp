# Question

Constant-proportion portfolio insurance (CPPI) is a strategy that:

## Options

- **A**: Maintains fixed asset class weights regardless of market movements
- **B**: Adjusts equity exposure based on cushion above a floor value
- **C**: Buys put options to protect against downside risk

## Answer

B

## Explanation

CPPI strategy:

**Concept:**
- Protect minimum value (floor)
- Exposure to risky assets based on "cushion"

**Formula:**
Equity exposure = m × (Portfolio value - Floor)

Where:
- m = multiplier (e.g., 3-5)
- Cushion = Portfolio value - Floor

**Dynamics:**
- As portfolio rises → cushion grows → more equity
- As portfolio falls → cushion shrinks → less equity
- Protects floor (in theory)

**Characteristics:**
- Dynamic strategy requiring rebalancing
- Procyclical: Buy after gains, sell after losses
- Gap risk: Large sudden drops can breach floor

**Contrast with options:**
- CPPI: Dynamic replication
- Options: Direct hedging instrument

**Common mistakes:**
- Option A: CPPI actively adjusts weights
- Option C: This describes protective put strategy
