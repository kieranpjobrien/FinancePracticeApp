# Question

A portfolio manager generates 3% excess return with a tracking error of 6%. The information ratio is:

## Options

- **A**: 0.25
- **B**: 0.50
- **C**: 2.00

## Answer

B

## Explanation

Information ratio calculation:

**Formula:**
IR = Active Return / Tracking Error
IR = (Rp - Rb) / σ(Rp - Rb)

**Given:**
- Excess return = 3%
- Tracking error = 6%

**Calculation:**
IR = 3% / 6%
IR = 0.50

**Interpretation:**
- Measures active return per unit of active risk
- Higher is better
- 0.50 = good performance
- 1.0+ = exceptional

**Benchmarks:**
- IR > 0.50: Strong
- IR > 0.75: Very strong
- IR > 1.00: Exceptional

**Relationship:**
- Annualized for comparison
- Related to Sharpe but for active management

**Common mistakes:**
- Option A: Divided incorrectly
- Option C: Inverted the ratio
