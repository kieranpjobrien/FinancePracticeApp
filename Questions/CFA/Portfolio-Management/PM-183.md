# Question

Behavioral portfolio theory suggests that investors:

## Options

- **A**: Construct mean-variance efficient portfolios
- **B**: Construct portfolios in layers based on aspirations and goals
- **C**: Only hold risk-free assets

## Answer

B

## Explanation

Behavioral Portfolio Theory (BPT):

**Key concepts:**
- Portfolios constructed in mental account layers
- Each layer serves different goal/aspiration
- Not optimized as single portfolio

**Pyramid structure:**
1. **Bottom layer:** Security (minimal risk)
   - Protection from poverty
   - Safe assets

2. **Middle layers:** Moderate goals
   - Education, retirement
   - Balanced risk

3. **Top layer:** Aspirational
   - Wealth accumulation
   - Speculative investments

**Contrast with MPT:**
- MPT: Single optimal portfolio
- BPT: Multiple sub-portfolios
- BPT reflects how people actually behave

**Common mistakes:**
- Option A: This is Modern Portfolio Theory, not Behavioral
- Option C: BPT includes risky assets in upper layers
