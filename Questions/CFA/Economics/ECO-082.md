# Question

For a US investor, a direct quote for the euro would be expressed as:

## Options

- **A**: EUR/USD (euros per dollar)
- **B**: USD/EUR (dollars per euro)
- **C**: The euro's percentage change

## Answer

B

## Explanation

A direct quote expresses the exchange rate as units of domestic currency per one unit of foreign currency. For a US investor, a direct quote for the euro shows dollars per euro (USD/EUR = 1.10 means $1.10 buys €1). An indirect quote shows foreign currency per domestic unit. Quote conventions vary by market, so understanding the format is essential.

**Common mistakes:**
- Option A shows an indirect quote for a US investor
- Option C describes percentage change, not a quote format
