# Question

If the nominal interest rate is 8% and inflation is 3%, the approximate real interest rate is:

## Options

- **A**: 11%
- **B**: 5%
- **C**: 24%

## Answer

B

## Explanation

The Fisher equation relates nominal rates, real rates, and inflation:

Real interest rate ≈ Nominal rate - Inflation rate
Real rate ≈ 8% - 3% = 5%

The exact formula is (1 + r) = (1 + i)/(1 + π), giving r = 4.85%, but the approximation is commonly used. Real rates represent the actual increase in purchasing power from lending.

**Common mistakes:**
- Option A adds the rates instead of subtracting
- Option C multiplies instead of subtracting
