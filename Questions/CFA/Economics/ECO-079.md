# Question

A currency carry trade involves:

## Options

- **A**: Borrowing in a high-interest-rate currency and investing in a low-interest-rate currency
- **B**: Borrowing in a low-interest-rate currency and investing in a high-interest-rate currency
- **C**: Only trading currencies with the same interest rates

## Answer

B

## Explanation

A carry trade borrows in a low-yielding currency (funding currency) and invests in a high-yielding currency (target currency) to profit from the interest rate differential. The strategy assumes exchange rates remain stable or the target currency appreciates. Carry trades can be profitable but are risky—if the target currency depreciates sharply, losses can exceed interest gains.

**Common mistakes:**
- Option A reverses the strategy—this would generate negative carry
- Option C eliminates the interest differential that drives carry trade profits
