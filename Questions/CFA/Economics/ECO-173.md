# Question

If USD/EUR = 1.10, then EUR/USD equals:

## Options

- **A**: 0.91
- **B**: 1.10
- **C**: 2.20

## Answer

A

## Explanation

To find the inverse (reciprocal) rate:
EUR/USD = 1 / (USD/EUR) = 1 / 1.10 = 0.909 ≈ 0.91

If it takes 1.10 dollars to buy 1 euro, it takes 0.91 euros to buy 1 dollar.

**Common mistakes:**
- Option B assumes the rates are the same
- Option C doubles instead of inverting
