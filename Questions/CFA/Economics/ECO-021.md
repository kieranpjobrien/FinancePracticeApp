# Question

The Taylor rule suggests that central banks should raise policy rates when:

## Options

- **A**: Inflation is below target and output is below potential
- **B**: Inflation is above target or output is above potential
- **C**: The unemployment rate is rising

## Answer

B

## Explanation

The Taylor rule provides a formula for setting policy rates based on inflation and output gaps. It suggests raising rates when inflation exceeds the target (positive inflation gap) or when output exceeds potential (positive output gap). Conversely, rates should be lowered when inflation is below target or output is below potential. This rule has been influential in understanding and evaluating central bank behavior.

**Common mistakes:**
- Option A describes conditions warranting rate cuts, not increases
- Option C is partially related but incomplete—rising unemployment suggests output below potential
