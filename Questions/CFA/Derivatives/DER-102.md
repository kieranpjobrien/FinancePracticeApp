# Question

An interest rate swap can be viewed as a series of:

## Options

- **A**: Spot transactions at current interest rates
- **B**: Forward rate agreements (FRAs) at different forward rates
- **C**: Options on interest rates

## Answer

B

## Explanation

Swap as series of FRAs:

**Conceptual decomposition:**
- Swap = Portfolio of FRAs
- Each payment period = One FRA
- Different forward rates for each period

**Example (2-year quarterly swap):**
- 8 payment periods
- 8 implicit FRAs
- Each locks in rate for that period

**FRA connection:**
- FRA: Agreement on future interest rate
- Swap: Multiple such agreements
- Fixed rate vs. floating rate for each period

**Valuation implication:**
- Value each FRA component
- Sum of FRA values = Swap value
- Forward rates from yield curve

**At inception:**
- Fixed rate set so swap value = 0
- Weighted average of forward rates
- Par swap rate

**After inception:**
- Forward rates change
- Individual FRA values change
- Swap gains/loses value

**Common mistakes:**
- Option A: Uses forward rates, not spot rates
- Option C: Swaps are commitments, not options
