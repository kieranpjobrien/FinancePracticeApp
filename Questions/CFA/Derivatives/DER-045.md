# Question

A synthetic long stock position can be created by:

## Options

- **A**: Buying a call and buying a put with the same strike
- **B**: Buying a call and selling a put with the same strike, plus lending PV(K)
- **C**: Selling both a call and a put with the same strike

## Answer

B

## Explanation

Synthetic stock creation:

**From put-call parity:**
C + PV(K) = P + S

**Rearranging for S:**
S = C - P + PV(K)

**Synthetic long stock:**
- Long call (+C)
- Short put (-P)
- Plus bond/lending (+PV(K))
- Equals long stock position

**Payoff verification:**

*If S_T > K:*
- Call exercised: Buy at K
- Put expires worthless
- Bond matures: Receive K
- Net: Own stock

*If S_T < K:*
- Call expires worthless
- Put exercised: Buy at K
- Bond matures: Receive K
- Net: Own stock

**Uses:**
- When stock is hard to borrow
- Regulatory constraints
- Arbitrage strategies

**Common mistakes:**
- Option A: This is a long straddle, not synthetic stock
- Option C: This is a short straddle
