# Question

A synthetic short stock position can be created by:

## Options

- **A**: Buying a call and selling a put at the same strike
- **B**: Selling a call and buying a put at the same strike
- **C**: Buying both a call and a put at the same strike

## Answer

B

## Explanation

Synthetic short stock:

**From put-call parity:**
C - P = S - PV(K)

**Rearranging for -S:**
-S = P - C - PV(K)

**Synthetic short stock:**
- Buy put (+P)
- Sell call (-C)
- Same strike, same expiration

**Ignoring lending component:**
Short Stock ≈ Long Put + Short Call

**Payoff verification:**

*If S_T > K:*
- Call exercised: Sell at K, owe S_T
- Put expires worthless
- Net: K - S_T (short stock payoff)

*If S_T < K:*
- Call expires worthless
- Exercise put: Receive K, deliver stock equivalent
- Net: K - S_T (short stock payoff)

**Uses:**
- When can't short stock
- Avoid borrowing fees
- Regulatory constraints

**Common mistakes:**
- Option A: This is synthetic LONG stock
- Option C: This is a straddle
