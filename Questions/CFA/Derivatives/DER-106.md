# Question

The maximum loss for an option buyer is:

## Options

- **A**: Unlimited
- **B**: Limited to the premium paid
- **C**: Limited to the strike price

## Answer

B

## Explanation

Option buyer's risk:

**Maximum loss = Premium paid**
- Paid upfront when purchased
- Lost if option expires worthless
- No additional obligations

**Why loss is limited:**
- Buyer has rights, not obligations
- Can simply not exercise
- "Walk away" if unfavorable

**Call buyer:**
- Loses premium if stock stays below strike
- Worst case: Option expires worthless
- Loss = Premium paid

**Put buyer:**
- Loses premium if stock stays above strike
- Worst case: Option expires worthless
- Loss = Premium paid

**Contrast with seller:**
- Call seller: Potentially unlimited loss
- Put seller: Large loss if stock falls to zero
- Higher risk for premium received

**Common mistakes:**
- Option A: Unlimited loss applies to short calls
- Option C: Strike price is exercise price, not loss limit
