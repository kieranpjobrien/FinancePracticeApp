# Question

A protective put strategy has the same payoff as:

## Options

- **A**: A covered call
- **B**: A fiduciary call
- **C**: A naked call

## Answer

B

## Explanation

Put-call parity equivalence:

**Protective put:**
- Long stock + Long put
- Downside protection
- Unlimited upside (minus premium)

**Fiduciary call:**
- Long call + Bond paying strike at expiration
- Same payoff profile
- Put-call parity relationship

**Payoff comparison:**

*If S_T > K:*
- Protective put: S_T (stock value)
- Fiduciary call: S_T - K (call) + K (bond) = S_T

*If S_T < K:*
- Protective put: K (exercise put)
- Fiduciary call: 0 (call expires) + K (bond) = K

**Both payoffs:** Max(S_T, K)

**Put-call parity:**
S + P = C + PV(K)
Protective put = Fiduciary call

**Common mistakes:**
- Option A: Covered call has different payoff
- Option C: Naked call has unlimited risk
