# Question

If put-call parity is violated with C + PV(K) > P + S, an arbitrageur should:

## Options

- **A**: Buy the call and sell the put
- **B**: Sell the call and buy the put, stock, and borrow PV(K)
- **C**: Buy both the call and the put

## Answer

B

## Explanation

Put-call parity arbitrage:

**Violation:**
C + PV(K) > P + S
Left side overpriced relative to right side

**Arbitrage strategy:**
- Sell overpriced side
- Buy underpriced side
- Lock in profit

**Execution:**
1. Sell call (receive C)
2. Buy put (pay P)
3. Buy stock (pay S)
4. Borrow PV(K) (receive PV(K))

**Initial cash flow:**
C - P - S + PV(K) > 0 (profit)

**At expiration:**

*If S_T > K:*
- Call exercised: Deliver stock, receive K
- Put expires worthless
- Repay loan: K
- Net: 0

*If S_T < K:*
- Call expires worthless
- Exercise put: Deliver stock, receive K
- Repay loan: K
- Net: 0

**Risk-free profit = Initial positive cash flow**

**Common mistakes:**
- Option A: This adds to overpriced position
- Option C: This doesn't exploit the mispricing
