# Question

According to put-call parity for European options, a fiduciary call (call + bond) is equivalent to:

## Options

- **A**: Protective put (stock + put)
- **B**: Covered call (stock - call)
- **C**: Stock + bond

## Answer

A

## Explanation

Put-call parity states:
c + PV(X) = p + S

Where:
- c = Call option price
- PV(X) = Present value of exercise price (bond)
- p = Put option price
- S = Current stock price

Rearranging: Call + Bond = Put + Stock

Therefore:
- Fiduciary call (c + PV(X)) = Protective put (S + p)

This relationship ensures no arbitrage opportunity exists between the two equivalent positions.

**Common mistakes:**
- Option B (covered call) involves selling a call, not buying one
- Option C ignores the put option entirely
