# Question

Early exercise of an American put option is more likely when the option is:

## Options

- **A**: At the money with long time to expiration
- **B**: Deep in the money with little time to expiration
- **C**: Out of the money regardless of time

## Answer

B

## Explanation

American put early exercise:

**When early exercise optimal:**
- Deep in the money
- Low time value remaining
- High interest rates
- Short time to expiration

**Why deep ITM matters:**
- Intrinsic value > Continuation value
- Time value minimal
- Better to receive strike now

**Interest rate effect:**
- Exercising: Receive K now
- Can invest K at risk-free rate
- Higher rates = more incentive

**Time effect:**
- Near expiration: Less time value
- More likely exercise is optimal
- Time value approaches zero

**Example:**
- Put with $50 strike
- Stock at $30
- Intrinsic = $20
- If time value < Interest on $50, exercise

**Contrast with calls:**
- Rarely exercise non-dividend calls early
- For puts: Exercise to get money now

**Common mistakes:**
- Option A: ATM has significant time value
- Option C: OTM puts have no intrinsic value to capture
