# Question

In a credit default swap (CDS), the protection buyer:

## Options

- **A**: Receives periodic payments and pays if a credit event occurs
- **B**: Makes periodic payments and receives payment if a credit event occurs
- **C**: Owns the underlying bond that is being protected

## Answer

B

## Explanation

Credit default swap structure:

**Protection buyer:**
- Pays periodic premium (CDS spread)
- Receives payment if credit event
- Like buying insurance
- Hedging credit risk

**Protection seller:**
- Receives periodic premium
- Pays if credit event occurs
- Like selling insurance
- Taking credit risk

**Credit events:**
- Bankruptcy
- Failure to pay
- Restructuring
- Other defined events

**Settlement:**
- Physical: Deliver defaulted bond, receive par
- Cash: Receive par minus recovery value
- Auction-based settlement common

**CDS spread:**
- Annual premium in basis points
- Reflects credit risk
- Wider = Higher default probability

**Note:**
- Don't need to own underlying bond
- Can speculate on credit
- "Naked" CDS positions possible

**Common mistakes:**
- Option A: This describes protection seller
- Option C: Ownership not required
