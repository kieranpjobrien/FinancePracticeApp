# Question

If a stock has no dividends and interest rates are positive, an at-the-money call will be worth:

## Options

- **A**: Less than an at-the-money put with the same expiration
- **B**: More than an at-the-money put with the same expiration
- **C**: Exactly the same as an at-the-money put

## Answer

B

## Explanation

ATM call vs. put relationship:

**From put-call parity:**
C - P = S - PV(K)

**At the money (S = K):**
C - P = K - PV(K)
C - P = K - K/(1+r)^T
C - P = K × [1 - 1/(1+r)^T]

**Since (1+r)^T > 1:**
- PV(K) < K
- K - PV(K) > 0
- Therefore: C - P > 0
- Call > Put

**Intuition:**
- Call: Defer payment of strike
- Put: Defer receipt of strike
- With positive rates, deferring payment is valuable
- Deferring receipt is costly

**Example:**
- S = K = $100, r = 5%, T = 1 year
- C - P = $100 - $100/1.05 = $4.76
- Call worth $4.76 more than put

**Common mistakes:**
- Option A: Reverses the relationship
- Option C: Only true if rates are zero
