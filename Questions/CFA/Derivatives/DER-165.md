# Question

A stock index is at 5,000. The continuously compounded risk-free rate is 4% and the continuous dividend yield is 1.5%. The 3-month forward price is closest to:

## Options

- **A**: 4,969
- **B**: 5,031
- **C**: 5,050

## Answer

B

## Explanation

Index forward with continuous yields:

**Formula:**
F = S × e^((r - q) × T)

**Given:**
- S = 5,000
- r = 4% = 0.04
- q = 1.5% = 0.015
- T = 3 months = 0.25

**Calculation:**
F = 5,000 × e^((0.04 - 0.015) × 0.25)
F = 5,000 × e^(0.025 × 0.25)
F = 5,000 × e^0.00625
F = 5,000 × 1.00627
F = 5,031.35

**Interpretation:**
- Net cost of carry = 2.5% (r - q)
- Forward > Spot because r > q
- Dividend yield reduces forward

**Annualized basis:**
- If r < q: Forward < Spot
- If r = q: Forward = Spot
- If r > q: Forward > Spot

**Common mistakes:**
- Option A: Used q - r instead of r - q
- Option C: Used r only, ignored dividend yield
