# Question

Put-call parity establishes a relationship between:

## Options

- **A**: Call prices and futures prices
- **B**: Put and call prices on the same underlying with the same strike and expiration
- **C**: Option prices and historical volatility

## Answer

B

## Explanation

Put-call parity basics:

**The relationship:**
C + PV(K) = P + S

**Components:**
- C = Call option price
- P = Put option price
- S = Current stock price
- PV(K) = Present value of strike price
- Same strike, same expiration

**Interpretation:**
- Fiduciary call = Protective put
- Two ways to achieve same payoff
- Must have same price (no arbitrage)

**Fiduciary call:**
- Long call + Bond paying K
- Payoff: Max(S_T, K)

**Protective put:**
- Long put + Long stock
- Payoff: Max(S_T, K)

**Applications:**
- Price one option from the other
- Identify arbitrage opportunities
- Create synthetic positions

**Common mistakes:**
- Option A: Futures not part of put-call parity
- Option C: Volatility affects both but isn't the relationship
