# Question

In the binomial model, risk-neutral probabilities are used to:

## Options

- **A**: Estimate actual probabilities of price movements
- **B**: Discount expected payoffs at the risk-free rate
- **C**: Calculate historical volatility

## Answer

B

## Explanation

Risk-neutral probabilities:

**Purpose:**
- Price options without knowing risk preferences
- Discount expected payoffs at risk-free rate
- Simplified calculation approach

**Formula:**
π = (1 + r - d) / (u - d)

Where:
- π = risk-neutral probability of up move
- r = risk-free rate per period
- u = up factor
- d = down factor

**How it works:**
1. Calculate expected payoff using π
2. Discount at risk-free rate
3. Result = fair option price

**Important distinction:**
- NOT actual/real-world probabilities
- Mathematical construct for pricing
- Gives same answer as replication

**Example calculation:**
- u = 1.1, d = 0.9, r = 0.05
- π = (1.05 - 0.9) / (1.1 - 0.9)
- π = 0.15 / 0.20 = 0.75

**Common mistakes:**
- Option A: These are not real probabilities
- Option C: Volatility is input, not output
