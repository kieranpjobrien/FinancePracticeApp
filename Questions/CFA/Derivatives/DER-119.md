# Question

Holding other factors constant, an option with more time to expiration will:

## Options

- **A**: Always be worth less than one with less time
- **B**: Generally be worth more than one with less time
- **C**: Have the same value regardless of time

## Answer

B

## Explanation

Time to expiration effect:

**General rule:**
- More time = More value
- Greater probability of favorable moves
- Higher time value component

**Why time adds value:**
- More uncertainty about outcome
- More chance to move ITM
- Flexibility has value

**For calls and puts:**
- Both generally increase with time
- Time value positive for both
- Exception: Deep ITM puts

**Exception - Deep ITM puts:**
- American puts may have less value with more time
- Could exercise now and invest proceeds
- Rare case where more time = less value

**Time decay (theta):**
- Time value erodes
- Accelerates near expiration
- Works against option buyers

**Common mistakes:**
- Option A: More time generally increases value
- Option C: Time is significant factor
