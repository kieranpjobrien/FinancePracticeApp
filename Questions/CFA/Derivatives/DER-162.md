# Question

In a multi-period binomial model, the number of terminal stock prices after n periods is:

## Options

- **A**: 2^n
- **B**: n + 1
- **C**: n × 2

## Answer

B

## Explanation

Binomial tree nodes:

**Terminal nodes (recombining tree):**
- After n periods: n + 1 terminal prices
- Not 2^n due to recombining

**Why recombining:**
- Up then down = Down then up
- Sud = Sdu
- Paths merge

**Example counts:**
- 1 period: 2 terminal nodes
- 2 periods: 3 terminal nodes
- 3 periods: 4 terminal nodes
- n periods: n + 1 terminal nodes

**Total possible paths:**
- Still 2^n paths to terminal nodes
- But many paths lead to same price
- Combinations determine probabilities

**Computational advantage:**
- Without recombining: 2^n nodes
- With recombining: n + 1 nodes
- Much more efficient

**Probability at each terminal node:**
- Binomial distribution
- C(n,k) × π^k × (1-π)^(n-k)

**Common mistakes:**
- Option A: This is number of paths, not terminal prices
- Option C: Incorrect formula
