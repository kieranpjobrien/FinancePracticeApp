# Question

A box spread using options is designed to create:

## Options

- **A**: Leveraged exposure to the underlying asset
- **B**: A risk-free position equivalent to lending or borrowing
- **C**: Maximum exposure to volatility

## Answer

B

## Explanation

Box spread structure:

**Components:**
- Bull call spread (long K1 call, short K2 call)
- Bear put spread (long K2 put, short K1 put)
- K1 < K2

**Alternatively:**
- Long synthetic forward at K1
- Short synthetic forward at K2

**Payoff:**
- Always equals K2 - K1 at expiration
- Regardless of stock price
- Risk-free payoff

**Value:**
Box value = PV(K2 - K1)

**Uses:**
- Test put-call parity
- Arbitrage if mispriced
- Synthetic borrowing/lending

**Arbitrage example:**
- If box price < PV(K2-K1): Buy box (lending)
- If box price > PV(K2-K1): Sell box (borrowing)

**Risk-free nature:**
- No exposure to underlying
- Fixed payoff known
- Interest rate instrument

**Common mistakes:**
- Option A: No exposure to underlying
- Option C: Zero volatility exposure
