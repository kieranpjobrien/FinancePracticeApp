# Question

A call option has a strike price of $40 and the underlying stock trades at $47. The intrinsic value is:

## Options

- **A**: $0
- **B**: $7
- **C**: $40

## Answer

B

## Explanation

Call intrinsic value:

**Formula:**
Intrinsic Value = Max(0, S - K)

**Given:**
- Stock price (S) = $47
- Strike price (K) = $40

**Calculation:**
IV = Max(0, $47 - $40)
IV = Max(0, $7)
IV = $7

**Interpretation:**
- Call is in the money by $7
- Could exercise and immediately profit $7
- Minimum option value = $7

**Additional value:**
- Total value = Intrinsic + Time value
- If option trades at $9
- Time value = $9 - $7 = $2

**Out-of-the-money call:**
- If S < K
- Intrinsic = 0
- All value is time value

**Common mistakes:**
- Option A: Would be true if stock < strike
- Option C: Strike price, not intrinsic value
