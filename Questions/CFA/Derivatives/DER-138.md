# Question

Mark-to-market accounting for derivatives requires:

## Options

- **A**: Recording derivatives at historical cost
- **B**: Recording derivatives at current fair value with changes in income
- **C**: Ignoring derivatives until settlement

## Answer

B

## Explanation

Mark-to-market for derivatives:

**Requirement:**
- Report at fair value
- Update each reporting period
- Changes flow through earnings or OCI

**Fair value determination:**
- Market prices if available
- Model-based valuation
- Observable inputs preferred

**Income statement impact:**
- Unrealized gains/losses recognized
- Before settlement occurs
- Can create earnings volatility

**Hedge accounting exception:**
- Qualifying hedges may defer gains/losses
- Match with hedged item
- Reduces earnings volatility

**Challenges:**
- Illiquid derivatives harder to value
- Model risk
- Complexity of exotic products

**Benefits:**
- Transparency
- Current exposure reflected
- Timely information

**Common mistakes:**
- Option A: Historical cost not used for derivatives
- Option C: Must recognize before settlement
