# Question

An investor buys a call option with a strike of $50 for a premium of $3. The breakeven stock price at expiration is:

## Options

- **A**: $47
- **B**: $50
- **C**: $53

## Answer

C

## Explanation

Call option breakeven:

**Formula:**
Breakeven = Strike price + Premium paid

**Calculation:**
Breakeven = $50 + $3 = $53

**At breakeven ($53):**
- Intrinsic value = $53 - $50 = $3
- Less premium paid = $3
- Net profit = $0

**Profit analysis:**
- Below $50: Option expires worthless, lose $3
- At $50: No intrinsic value, lose $3
- At $53: Breakeven, zero profit
- Above $53: Profit = Stock - $53

**Example at $55:**
- Exercise call, pay $50
- Stock worth $55
- Gain = $5 - $3 = $2

**For put options:**
- Breakeven = Strike - Premium
- Below breakeven = profit

**Common mistakes:**
- Option A: This would be put breakeven
- Option B: Strike alone doesn't account for premium
