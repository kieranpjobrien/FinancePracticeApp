# Question

The cost-of-carry model for futures pricing includes all EXCEPT:

## Options

- **A**: Financing costs
- **B**: Storage and insurance costs
- **C**: Expected future price changes

## Answer

C

## Explanation

Cost-of-carry model components:

**Included costs:**

*Financing (interest):*
- Cost of borrowing to buy spot
- Risk-free rate applied
- Opportunity cost of capital

*Storage:*
- Physical storage for commodities
- Warehouse costs
- Handling fees

*Insurance:*
- Protection for stored goods
- Spoilage/damage coverage

**NOT included:**
- Expected future prices
- Forecasts or predictions
- Speculation about direction

**Why expectations excluded:**
- Model based on no-arbitrage
- Not on forecasting
- Replication costs, not predictions

**Formula:**
F = S × (1 + r + c - y)^T

Where:
- r = risk-free rate
- c = storage + insurance
- y = convenience yield (benefit)

**Key insight:**
- Forward price ≠ Expected spot
- Based on carrying costs
- Arbitrage-free relationship

**Common mistakes:**
- Option A: Financing is key component
- Option B: Storage is included for commodities
