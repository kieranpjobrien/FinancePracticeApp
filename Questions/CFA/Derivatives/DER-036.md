# Question

The fixed rate in a swap is set such that the swap has:

## Options

- **A**: A positive value to the fixed-rate payer
- **B**: Zero value to both parties at inception
- **C**: A positive value to the floating-rate payer

## Answer

B

## Explanation

Swap pricing at inception:

**Par swap rate:**
- Rate that makes swap value = 0
- Fair to both parties
- No upfront payment needed
- Market equilibrium rate

**How it's determined:**
- PV of fixed payments = PV of expected floating payments
- Use forward rates for floating
- Discount at appropriate rates
- Solve for fixed rate

**No arbitrage:**
- If swap had value at inception
- One party wouldn't agree
- Competition ensures fair pricing

**Value changes after inception:**
- Interest rates move
- Swap becomes valuable to one party
- Mark-to-market value changes

**Off-market swaps:**
- Non-zero value at inception
- Requires upfront payment
- Used for specific purposes

**Common mistakes:**
- Option A: Would require payment to fixed payer
- Option C: Would require payment to floating payer
