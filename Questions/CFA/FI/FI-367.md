# Question

A portfolio uses 2x leverage via repo financing. If the unlevered portfolio return is 4% and the repo rate is 2%, the levered return is:

## Options

- **A**: 4%
- **B**: 6%
- **C**: 8%

## Answer

B

## Explanation

Levered return = Unlevered return + Leverage × (Unlevered return - Financing cost). With 2x leverage (1x borrowed): Return = 4% + 1 × (4% - 2%) = 4% + 2% = 6%. Leverage amplifies returns but also amplifies losses. Repo common for bond portfolio leverage.

**Common mistakes:**
- Option A—ignores leverage benefit
- Option C—doesn't subtract financing cost
