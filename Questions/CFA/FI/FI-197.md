# Question

The quick ratio (acid-test ratio) differs from the current ratio by:

## Options

- **A**: Including all current assets
- **B**: Excluding inventory from current assets
- **C**: Using market values instead of book values

## Answer

B

## Explanation

Quick ratio = (Current assets - Inventory) / Current liabilities. Stricter than current ratio—inventory may not be quickly convertible to cash. Tests ability to meet short-term obligations without selling inventory. Higher ratio = better short-term liquidity.

**Common mistakes:**
- Option A describes current ratio
- Option C describes fair value accounting issue
