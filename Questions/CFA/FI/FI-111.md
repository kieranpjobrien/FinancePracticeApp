# Question

Matrix pricing is used to:

## Options

- **A**: Calculate the yield curve from Treasury prices
- **B**: Estimate the price of illiquid bonds using prices of comparable traded bonds
- **C**: Determine the coupon rate for new bond issues

## Answer

B

## Explanation

Matrix pricing: estimates value of infrequently traded bonds using comparable liquid bonds. Matches credit quality, maturity, coupon. Interpolates/extrapolates from known prices. Used for portfolio valuation when market prices unavailable.

**Common mistakes:**
- Option A describes yield curve construction
- Option C describes new issue pricing
