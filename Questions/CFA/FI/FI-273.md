# Question

For a single-liability immunization strategy to work, the portfolio must have:

## Options

- **A**: Higher duration than the liability
- **B**: Duration equal to the liability's time horizon
- **C**: Zero duration

## Answer

B

## Explanation

Immunization: match portfolio duration to liability horizon. When interest rates change, price effect offsets reinvestment effect. Also need portfolio present value ≥ liability present value. Works for parallel yield curve shifts. Requires rebalancing as time passes.

**Common mistakes:**
- Option A—mismatched duration exposes to interest rate risk
- Option C—zero duration means cash only, no yield
