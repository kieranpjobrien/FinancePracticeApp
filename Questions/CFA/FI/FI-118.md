# Question

The breakeven inflation rate represents:

## Options

- **A**: The minimum inflation rate for TIPS to outperform nominal bonds
- **B**: The inflation rate at which nominal and inflation-linked bonds provide equal returns
- **C**: The maximum inflation rate the economy can sustain

## Answer

B

## Explanation

Breakeven inflation = nominal yield - real yield (TIPS yield). If actual inflation exceeds breakeven, TIPS outperform. If inflation below breakeven, nominal bonds outperform. Market's implied inflation expectation embedded in pricing.

**Common mistakes:**
- Option A reverses the relationship slightly
- Option C describes macroeconomic concept, not bond math
