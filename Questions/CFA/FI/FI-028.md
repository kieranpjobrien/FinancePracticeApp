# Question

A forward rate represents:

## Options

- **A**: Today's yield on existing bonds
- **B**: The interest rate agreed today for a loan starting in the future
- **C**: Historical interest rates

## Answer

B

## Explanation

Forward rates: rates for future periods implied by current spot rates. Example: 1-year rate starting in 2 years (1y2y forward). Derived from spot curve using no-arbitrage. Important for: interest rate derivatives pricing, expectation analysis, relative value assessment.

**Common mistakes:**
- Option A describes current spot or yield
- Option C is backward-looking, not forward
