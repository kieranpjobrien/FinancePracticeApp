# Question

The discount margin on a floating-rate note represents:

## Options

- **A**: The margin fixed at issuance
- **B**: The constant spread over the reference rate that makes present value equal market price
- **C**: The current reference rate level

## Answer

B

## Explanation

Discount margin (required margin): spread that equates FRN cash flows to current price, assuming reference rate stays constant. If discount margin > quoted margin, FRN trades below par (credit deteriorated). Key for FRN relative value analysis.

**Common mistakes:**
- Option A describes quoted margin
- Option C describes reference rate, not margin
