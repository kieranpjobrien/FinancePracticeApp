# Question

Duration matching (immunization) aims to:

## Options

- **A**: Maximize portfolio returns regardless of rate changes
- **B**: Offset price risk and reinvestment risk to lock in a target return
- **C**: Eliminate all portfolio risk

## Answer

B

## Explanation

Immunization: match asset duration to liability duration. Rate rise: price falls but reinvestment income rises. Rate fall: opposite. At duration match point, effects offset. Locks in initial yield for horizon. Requires rebalancing as time passes.

**Common mistakes:**
- Option A—immunization targets specific return, not maximum
- Option C—only protects against parallel yield curve shifts
