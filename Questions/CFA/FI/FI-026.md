# Question

Convertible bonds allow holders to:

## Options

- **A**: Convert the bond into a specified number of common shares
- **B**: Convert the bond into a different bond
- **C**: Increase the coupon rate over time

## Answer

A

## Explanation

Convertible bonds: can be exchanged for common stock at conversion ratio. Combines bond features (fixed income, downside protection) with equity upside. Conversion value = stock price × conversion ratio. Typically lower coupon than straight bond due to equity option value.

**Common mistakes:**
- Option B is not standard convertible feature
- Option C describes step-up coupon bonds
