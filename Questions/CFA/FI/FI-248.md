# Question

For a putable bond, the relationship between values is:

## Options

- **A**: Value of putable = Value of straight bond + Value of put option
- **B**: Value of putable = Value of straight bond - Value of put option
- **C**: Value of putable = Value of straight bond only

## Answer

A

## Explanation

Putable bond value = Straight bond value + Put option value. Bondholder owns put (can sell back to issuer). Put has value to holder. Putable trades above otherwise identical straight bond. Put more valuable when rates high/credit weak.

**Common mistakes:**
- Option B subtracts (would be callable from holder perspective)
- Option C ignores embedded option value
