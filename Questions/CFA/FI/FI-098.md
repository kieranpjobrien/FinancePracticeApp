# Question

A bond has a 2% probability of default and a 60% loss given default. The expected loss is:

## Options

- **A**: 0.80%
- **B**: 1.20%
- **C**: 2.00%

## Answer

B

## Explanation

Expected loss = Probability of default × Loss given default = 2% × 60% = 1.20%. This is the average loss over many scenarios. Actual loss is binary—either 0% (no default) or 60% (default occurs).

**Common mistakes:**
- Option A calculates 2% × 40% (using recovery instead of loss)
- Option C uses only default probability
