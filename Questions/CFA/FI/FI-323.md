# Question

A mortgage IO (interest-only) strip:

## Options

- **A**: Benefits from faster prepayments
- **B**: Loses value when prepayments accelerate because interest payments decline
- **C**: Has no interest rate sensitivity

## Answer

B

## Explanation

IO strip receives only interest payments. When prepayments accelerate (rates fall), outstanding principal declines faster—less interest to receive. IO value falls. Negative duration to interest rates (unusual). PO strip (principal only) benefits from faster prepayments.

**Common mistakes:**
- Option A—faster prepayments hurt IO strips
- Option C—IOs have unusual (negative) duration
