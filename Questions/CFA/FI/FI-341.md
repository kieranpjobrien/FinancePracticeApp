# Question

The Nelson-Siegel model is used to:

## Options

- **A**: Value bonds with embedded options
- **B**: Fit a smooth yield curve using a mathematical function with level, slope, and curvature factors
- **C**: Calculate credit spreads

## Answer

B

## Explanation

Nelson-Siegel: parametric model fitting yield curve. Three factors: level (long-term rate), slope (short vs long), curvature (hump). Produces smooth curve from discrete bond data. Widely used by central banks and practitioners for curve construction and analysis.

**Common mistakes:**
- Option A—Nelson-Siegel is curve-fitting, not option valuation
- Option C—describes credit analysis, not term structure
