# Question

Treasury Inflation-Protected Securities (TIPS) protect investors by:

## Options

- **A**: Paying a higher fixed coupon rate
- **B**: Adjusting the principal value based on inflation
- **C**: Guaranteeing a minimum real return above inflation

## Answer

B

## Explanation

TIPS: principal adjusted by CPI. Coupon rate fixed but applied to inflation-adjusted principal, so coupon payments rise with inflation. At maturity, receive greater of adjusted principal or original par. Provides real return protection.

**Common mistakes:**
- Option A—coupon rate is typically lower than nominal bonds
- Option C—return depends on real yield at purchase, not guaranteed minimum
