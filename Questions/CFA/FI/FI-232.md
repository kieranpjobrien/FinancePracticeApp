# Question

For a floating-rate note, the quoted margin is:

## Options

- **A**: The spread over the reference rate set at issuance
- **B**: The current market spread over the reference rate
- **C**: The difference between bid and ask prices

## Answer

A

## Explanation

Quoted margin: fixed spread over reference rate (SOFR, formerly LIBOR) set at issuance. Coupon = Reference rate + Quoted margin. If credit improves after issuance, FRN trades above par. Discount margin = spread required by market currently.

**Common mistakes:**
- Option B describes discount margin (required margin)
- Option C describes bid-ask spread
