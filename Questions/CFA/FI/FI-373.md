# Question

A derivatives overlay strategy allows portfolio managers to:

## Options

- **A**: Only invest in physical bonds
- **B**: Adjust duration, credit, or curve exposure using derivatives without trading underlying bonds
- **C**: Eliminate all portfolio risk

## Answer

B

## Explanation

Overlay: use futures, swaps, options to modify portfolio characteristics. Change duration quickly without selling bonds (avoid transaction costs, tax consequences). Hedge currency. Adjust curve positioning. Efficient, capital-light. Common for large institutional portfolios.

**Common mistakes:**
- Option A—overlays add derivatives to physical holdings
- Option C—overlays modify risk, don't eliminate it
