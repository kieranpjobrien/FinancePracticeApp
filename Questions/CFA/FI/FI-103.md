# Question

Reduced-form credit models differ from structural models in that they:

## Options

- **A**: Use company balance sheet data to predict default
- **B**: Model default as a random event with probability estimated from market data
- **C**: Only apply to investment-grade bonds

## Answer

B

## Explanation

Reduced-form: default modeled as random event with hazard rate estimated from spreads/historical data. Don't require firm asset value (unobservable). More practical for pricing credit derivatives. Structural models need asset value/volatility estimates.

**Common mistakes:**
- Option A describes structural models
- Option C—reduced-form applies to all credit
