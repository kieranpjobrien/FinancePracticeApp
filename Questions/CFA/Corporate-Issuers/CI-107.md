# Question

The crossover rate for two mutually exclusive projects is:

## Options

- **A**: The discount rate at which both projects have zero NPV
- **B**: The discount rate at which both projects have equal NPV
- **C**: The average of the two projects' IRRs

## Answer

B

## Explanation

The crossover rate is where NPV profiles intersect—both projects have the same NPV. Below this rate, one project is preferred; above it, the other is preferred. It's found by calculating the IRR of the differential cash flows (Project A - Project B).

**Common mistakes:**
- Option A describes the IRR of each project
- Option C is not how crossover rate is calculated
