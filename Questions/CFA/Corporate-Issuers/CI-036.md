# Question

A company owns land it could sell for $2 million. If used for a new project, the opportunity cost is:

## Options

- **A**: Zero, since no cash is paid
- **B**: $2 million, the foregone sale proceeds
- **C**: The original purchase price of the land

## Answer

B

## Explanation

Opportunity cost is the value of the best alternative use. Using owned land for a project means forgoing the $2 million sale. This cost must be included in project analysis. The original purchase price is a sunk cost—irrelevant to the decision.

**Common mistakes:**
- Option A ignores that using the asset has a cost
- Option C uses the sunk cost instead of current market value
