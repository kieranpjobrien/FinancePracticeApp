# Question

When calculating WACC, the weights used should be based on:

## Options

- **A**: Historical book values of debt and equity
- **B**: Target capital structure using market values
- **C**: The proportion of financing for the specific project

## Answer

B

## Explanation

WACC weights should reflect target capital structure at market values. Market values represent current economic value; book values may be outdated. Target weights assume company will maintain its optimal structure over time, making WACC appropriate for project evaluation.

**Common mistakes:**
- Option A uses book values which don't reflect current market conditions
- Option C mixes project-specific financing with company cost of capital
