# Question

Scenario analysis differs from sensitivity analysis in that:

## Options

- **A**: It changes only one variable at a time
- **B**: It examines the effect of simultaneous changes in multiple variables
- **C**: It provides a single NPV estimate

## Answer

B

## Explanation

Scenario analysis evaluates combinations of variable changes (best case, worst case, base case). It recognizes that variables are often correlated—high sales may coincide with high prices. This provides a range of outcomes more realistic than one-at-a-time sensitivity analysis.

**Common mistakes:**
- Option A describes sensitivity analysis
- Option C describes single-point estimation
