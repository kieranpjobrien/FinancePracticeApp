# Question

The option to abandon a project is most valuable when:

## Options

- **A**: Project outcomes are highly certain
- **B**: There is significant uncertainty and valuable salvage or alternative use
- **C**: The project has no downside risk

## Answer

B

## Explanation

Abandonment options: if a project performs poorly, management can abandon and recover salvage value or redeploy assets. This limits downside loss. The option is more valuable with higher uncertainty (more scenarios where abandonment is attractive) and higher salvage value.

**Common mistakes:**
- Option A makes flexibility valueless
- Option C means abandonment would never be exercised
