# Question

A company has 60% equity (cost 12%) and 40% debt (pre-tax cost 7%, tax rate 30%). WACC is closest to:

## Options

- **A**: 9.0%
- **B**: 9.2%
- **C**: 11.0%

## Answer

B

## Explanation

WACC = (E/V × Re) + (D/V × Rd × (1-T)) = (0.60 × 12%) + (0.40 × 7% × 0.70) = 7.2% + 1.96% = 9.16% ≈ 9.2%. After-tax debt cost = 7% × 0.70 = 4.9%. Weight-average the costs using capital structure weights.

**Common mistakes:**
- Option A uses pre-tax debt cost
- Option C uses simple average instead of weighted average
