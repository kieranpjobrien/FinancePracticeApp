# Question

The Economic Order Quantity (EOQ) model determines:

## Options

- **A**: The maximum inventory level
- **B**: The order size that minimizes total inventory costs
- **C**: When inventory should be reordered

## Answer

B

## Explanation

EOQ minimizes total inventory costs (carrying costs + ordering costs). EOQ = √(2 × Demand × Order cost / Carrying cost per unit). Larger orders reduce ordering frequency but increase carrying costs; smaller orders do the opposite. EOQ balances these trade-offs.

**Common mistakes:**
- Option A describes inventory capacity
- Option C describes the reorder point, a separate decision
