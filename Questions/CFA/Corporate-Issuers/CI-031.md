# Question

Stakeholder theory differs from shareholder theory in that:

## Options

- **A**: Only shareholders matter in stakeholder theory
- **B**: It considers interests of all groups affected by the company
- **C**: It ignores long-term value creation

## Answer

B

## Explanation

Stakeholder theory: companies should balance interests of all stakeholders—shareholders, employees, customers, suppliers, communities, environment. It argues long-term value creation requires considering all affected parties, not just maximizing short-term shareholder returns. ESG considerations align with stakeholder theory.

**Common mistakes:**
- Option A reverses the definition
- Option C contradicts stakeholder theory's focus on sustainability
