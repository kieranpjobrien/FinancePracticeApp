# Question

In project valuation, terminal value captures:

## Options

- **A**: Only the explicit forecast period cash flows
- **B**: The value of cash flows beyond the explicit forecast period
- **C**: Sunk costs from prior investments

## Answer

B

## Explanation

Terminal value: estimates value at the end of the explicit forecast period, representing all subsequent cash flows. Methods include perpetuity growth models or exit multiples. Terminal value often comprises a significant portion of total value, making assumptions critical.

**Common mistakes:**
- Option A ignores the continuing value
- Option C describes irrelevant costs
