# Question

A stock pays a $2 dividend, is priced at $40, and dividends are expected to grow at 5%. Using the dividend growth model, the cost of equity is:

## Options

- **A**: 5%
- **B**: 10%
- **C**: 15%

## Answer

B

## Explanation

Dividend growth model: Cost of equity = (D1 / P0) + g = ($2 × 1.05) / $40 + 5% = $2.10 / $40 + 5% = 5.25% + 5% = 10.25% ≈ 10%. Or using D0: ($2 / $40) + 5% = 5% + 5% = 10%.

**Common mistakes:**
- Option A uses only growth rate
- Option C adds incorrectly
