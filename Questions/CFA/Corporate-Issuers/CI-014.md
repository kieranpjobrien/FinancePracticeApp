# Question

The risk-free rate is 3%, market risk premium is 6%, and a stock's beta is 1.2. Using CAPM, the cost of equity is:

## Options

- **A**: 9.0%
- **B**: 10.2%
- **C**: 10.8%

## Answer

B

## Explanation

CAPM: Cost of equity = Risk-free rate + Beta × Market risk premium = 3% + 1.2 × 6% = 3% + 7.2% = 10.2%. This represents the return shareholders require given the stock's systematic risk. Higher beta means higher required return.

**Common mistakes:**
- Option A ignores the beta multiplication
- Option C calculates incorrectly
