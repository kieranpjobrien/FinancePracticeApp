# Question

Equipment with book value of $20,000 is sold for $35,000. If the tax rate is 30%, the after-tax salvage value is:

## Options

- **A**: $24,500
- **B**: $30,500
- **C**: $35,000

## Answer

B

## Explanation

Gain on sale = Sale price - Book value = $35,000 - $20,000 = $15,000. Tax on gain = $15,000 × 0.30 = $4,500. After-tax salvage = Sale price - Tax = $35,000 - $4,500 = $30,500. Alternatively: Book value + After-tax gain = $20,000 + $10,500 = $30,500.

**Common mistakes:**
- Option A calculates incorrectly
- Option C ignores tax on the gain
