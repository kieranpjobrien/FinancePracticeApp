# Question

Project sequencing (staging) creates value by:

## Options

- **A**: Forcing immediate full commitment
- **B**: Allowing abandonment or modification between stages based on new information
- **C**: Reducing total project returns

## Answer

B

## Explanation

Staged investment: breaking projects into phases with decision points. This creates real options—management can continue, modify, or abandon based on results from earlier stages. Staging reduces risk by limiting commitment until uncertainty is resolved.

**Common mistakes:**
- Option A describes the opposite of staging
- Option C confuses risk reduction with return reduction
