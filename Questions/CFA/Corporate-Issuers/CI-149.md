# Question

When using WACC to discount project cash flows, the implicit assumption is:

## Options

- **A**: The project is financed entirely with equity
- **B**: The project is financed with the same mix of debt and equity as the company
- **C**: Project risk is higher than company average

## Answer

B

## Explanation

WACC assumes the project maintains the company's target capital structure—same debt/equity mix, same overall risk. If project risk differs significantly from company average, a risk-adjusted discount rate should be used. WACC is appropriate for projects similar to the firm's existing business.

**Common mistakes:**
- Option A would require using cost of equity, not WACC
- Option C would require a higher discount rate
