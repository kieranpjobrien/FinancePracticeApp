# Question

When choosing between mutually exclusive projects with different scales:

## Options

- **A**: Always choose the project with higher NPV
- **B**: Always choose the project with higher IRR
- **C**: Consider incremental analysis or use NPV decision

## Answer

A

## Explanation

For mutually exclusive projects, NPV provides the correct ranking—it measures absolute value creation. A larger project with higher NPV creates more wealth than a smaller project with higher IRR. Incremental IRR analysis can confirm, but NPV is the primary decision criterion.

**Common mistakes:**
- Option B may select lower-value project with higher percentage return
- Option C is partially correct but NPV alone is sufficient
