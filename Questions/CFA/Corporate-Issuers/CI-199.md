# Question

Post-audit analysis of capital projects serves to:

## Options

- **A**: Only evaluate whether the project achieved expected returns
- **B**: Improve future forecasting and identify systematic biases
- **C**: Eliminate all future project failures

## Answer

B

## Explanation

Post-audit: compare actual results to projections. Benefits: identify forecasting errors, discover systematic biases (optimism, pessimism), improve future estimation processes, create accountability. Should be constructive, not punitive. Limitations: cannot perfectly isolate project performance from external factors.

**Common mistakes:**
- Option A understates the learning benefits
- Option C is unrealistic—post-audits improve but don't guarantee success
