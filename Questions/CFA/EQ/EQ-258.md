# Question

To calculate probability-weighted expected value from scenarios:

## Options

- **A**: Simply average all scenario values equally
- **B**: Multiply each scenario value by its probability and sum the results
- **C**: Use only the highest probability scenario

## Answer

B

## Explanation

Probability-weighted value: E(V) = Σ[P(i) × V(i)] for all scenarios. Example: 30% bull × $50 + 50% base × $40 + 20% bear × $25 = $15 + $20 + $5 = $40. Incorporates likelihood of each outcome. Probabilities must sum to 100%. Represents expected value across scenarios.

**Common mistakes:**
- Option A assumes equal probabilities
- Option C ignores valuable information from other scenarios
