# Question

Model risk in equity valuation refers to:

## Options

- **A**: Risk that all models give identical values
- **B**: Risk that the model is inappropriate or inputs are wrong, leading to incorrect valuation
- **C**: Guarantee that models always produce accurate values

## Answer

B

## Explanation

Model risk: errors from model misspecification or incorrect inputs. Sources: wrong model for situation (DDM for non-dividend payer), incorrect assumptions (growth rate, discount rate), garbage-in-garbage-out. Mitigate through: multiple methods, sensitivity analysis, sanity checks, understanding model limitations.

**Common mistakes:**
- Option A—different models typically give different values
- Option C—no model guarantees accuracy
