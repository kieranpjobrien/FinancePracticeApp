# Question

A stock pays a $2 dividend expected to grow at 5% annually. If the required return is 10%, the stock's value using the Gordon Growth Model is:

## Options

- **A**: $40.00
- **B**: $42.00
- **C**: $20.00

## Answer

B

## Explanation

Gordon Growth Model: V₀ = D₁ / (r - g). D₁ = D₀ × (1 + g) = $2 × 1.05 = $2.10. V₀ = $2.10 / (0.10 - 0.05) = $2.10 / 0.05 = $42.00. The formula requires the next period's dividend (D₁), not the current dividend (D₀).

**Common mistakes:**
- Option A uses current dividend ($2.00) instead of D₁
- Option C uses only the required return as denominator
