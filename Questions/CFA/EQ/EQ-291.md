# Question

The size effect (small-cap premium) suggests that:

## Options

- **A**: Large-cap stocks consistently outperform small-caps
- **B**: Small-cap stocks have historically generated higher average returns
- **C**: Stock size has no relationship to returns

## Answer

B

## Explanation

Size effect: small-cap stocks historically outperformed large-caps on average. Fama-French included size as factor. Explanations: higher risk/illiquidity, less analyst coverage, behavioral factors. Premium has varied over time and across markets. Part of rationale for small-cap allocations.

**Common mistakes:**
- Option A reverses the historical relationship
- Option C ignores documented size-return relationship
