# Question

Algorithmic trading systems are primarily used to:

## Options

- **A**: Replace all human decision-making in portfolio management
- **B**: Execute orders efficiently by minimizing market impact and transaction costs
- **C**: Guarantee positive returns

## Answer

B

## Explanation

Algorithmic trading: computer programs execute orders using pre-defined rules. Goals: minimize market impact (breaking large orders), reduce timing risk, lower transaction costs, ensure best execution. Examples: VWAP, TWAP, implementation shortfall algorithms. Complements human strategy decisions.

**Common mistakes:**
- Option A overstates—algorithms execute, humans often decide what to trade
- Option C is unrealistic—no trading system guarantees returns
