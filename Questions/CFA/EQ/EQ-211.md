# Question

Trading comparables analysis values a company based on:

## Options

- **A**: The price multiples of similar publicly traded companies
- **B**: Historical book value only
- **C**: Replacement cost of assets

## Answer

A

## Explanation

Trading comparables: derives value from current market multiples of peer companies. Common multiples: P/E, EV/EBITDA, P/B, P/S. Assumes market prices peers correctly and target should trade at similar multiples. Adjustments for size, growth, risk differences. Quick, market-based valuation approach.

**Common mistakes:**
- Option B describes a single metric, not comparative analysis
- Option C describes asset-based valuation
