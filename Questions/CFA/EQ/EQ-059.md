# Question

An equity investor's required rate of return consists of:

## Options

- **A**: Only the dividend yield
- **B**: The risk-free rate plus a risk premium for equity investment
- **C**: The inflation rate only

## Answer

B

## Explanation

Required return = Risk-free rate + Equity risk premium. CAPM further specifies: r = Rf + β(Rm - Rf). Reflects time value of money (risk-free rate) and compensation for bearing equity risk. Higher risk stocks require higher returns. Used as discount rate in valuation models.

**Common mistakes:**
- Option A is only one return component
- Option C ignores risk premium requirement
