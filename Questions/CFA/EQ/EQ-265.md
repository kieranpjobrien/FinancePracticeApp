# Question

In a two-stage DDM, the terminal value represents:

## Options

- **A**: The value of dividends during the high-growth period only
- **B**: The present value of all dividends from the stable-growth period onward
- **C**: The company's book value at the end of the growth period

## Answer

B

## Explanation

Terminal value in two-stage DDM: uses Gordon model to value stable-growth phase. TV = D(n+1) / (r - g_stable), then discounted to present. Represents bulk of value for growth companies. Critical to get stable-growth assumptions right—small changes significantly impact total value.

**Common mistakes:**
- Option A ignores the terminal/stable period
- Option C confuses book value with intrinsic value concepts
