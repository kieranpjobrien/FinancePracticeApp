# Question

Buying on margin allows investors to:

## Options

- **A**: Purchase securities using only their own funds
- **B**: Amplify both potential gains and losses by borrowing funds
- **C**: Avoid paying any interest on borrowed funds

## Answer

B

## Explanation

Margin trading: borrow from broker to increase position size beyond available capital. Initial margin requirement sets minimum equity percentage. Leverage amplifies returns in both directions. Margin calls occur if equity falls below maintenance margin. Interest charged on borrowed funds.

**Common mistakes:**
- Option A describes cash account trading, not margin
- Option C is wrong—margin loans incur interest charges
