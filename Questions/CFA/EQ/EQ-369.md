# Question

When assessing competitive moat durability, analysts should consider:

## Options

- **A**: Only current profitability levels
- **B**: Whether competitive advantages can be sustained against erosion from competition and change
- **C**: Stock price momentum only

## Answer

B

## Explanation

Moat durability: assesses how long competitive advantages can persist. Threats: imitation, substitution, technology disruption, regulatory change. Wider moats from structural advantages (network effects, switching costs, patents). Narrow moats erode quickly. Long-term value depends on moat sustainability.

**Common mistakes:**
- Option A ignores forward-looking competitive analysis
- Option C focuses on price, not fundamental advantages
