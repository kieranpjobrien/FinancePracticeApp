# Question

Absolute valuation methods estimate value by:

## Options

- **A**: Comparing the company to peer multiples
- **B**: Discounting expected future cash flows or earnings
- **C**: Using only historical stock prices

## Answer

B

## Explanation

Absolute valuation: estimates intrinsic value based on fundamentals—discounted cash flows, DDM, residual income. Value derived from expected cash flows and required return, not comparison to others. Relative valuation uses multiples from comparable companies. Both approaches complement each other.

**Common mistakes:**
- Option A describes relative valuation
- Option C ignores fundamental value drivers
