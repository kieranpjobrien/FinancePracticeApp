# Question

Semi-strong form market efficiency implies that:

## Options

- **A**: Only technical analysis is useful
- **B**: All publicly available information is reflected in prices
- **C**: Markets are perfectly predictable

## Answer

B

## Explanation

Semi-strong efficiency: prices reflect all publicly available information including financial statements, news, and analyst reports. Neither technical nor fundamental analysis based on public information generates excess returns. Only non-public (insider) information could provide an edge. Event studies test this form.

**Common mistakes:**
- Option A is contradicted—technical analysis wouldn't help
- Option C overstates efficiency—prices still change with new information
