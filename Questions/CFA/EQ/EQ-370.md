# Question

Platform business models create value by:

## Options

- **A**: Manufacturing physical products
- **B**: Connecting different user groups and enabling transactions between them
- **C**: Owning all inventory they sell

## Answer

B

## Explanation

Platform economics: connects multiple participant groups (buyers/sellers, users/developers). Value from network effects—more participants increase value for all. Examples: marketplaces, social networks, payment systems. Often asset-light, scalable, winner-take-most dynamics. Key metrics: GMV, take rate, user engagement.

**Common mistakes:**
- Option A describes traditional manufacturing
- Option C describes traditional retail inventory model
