# Question

When using comparable transaction analysis for valuation:

## Options

- **A**: Only the target company's current stock price matters
- **B**: Recent M&A transactions for similar companies are analyzed to derive valuation multiples
- **C**: Future earnings are discounted at the risk-free rate

## Answer

B

## Explanation

Comparable transactions: uses multiples paid in actual M&A deals for similar companies. Includes control premiums paid. More relevant for acquisition valuation than trading multiples. Considerations: transaction timing (market conditions), deal structure, strategic vs. financial buyers, synergies expected.

**Common mistakes:**
- Option A ignores the "comparable" aspect of methodology
- Option C describes DCF with incorrect discount rate
