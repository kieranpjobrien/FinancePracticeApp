# Question

Event studies test market efficiency by examining:

## Options

- **A**: Long-term stock returns over multiple years
- **B**: Whether security prices adjust quickly and correctly to new information
- **C**: The average returns of all stocks in the market

## Answer

B

## Explanation

Event studies: measure abnormal returns around specific events (earnings announcements, mergers, splits). If prices adjust quickly and fully, little opportunity for profit. Tests semi-strong efficiency. Key elements: identifying event date, measuring expected vs. actual returns, calculating abnormal returns.

**Common mistakes:**
- Option A describes long-term performance studies
- Option C describes market return calculation, not event studies
