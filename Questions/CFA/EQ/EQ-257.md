# Question

Scenario analysis in equity valuation:

## Options

- **A**: Produces a single point estimate of value
- **B**: Evaluates value under different future states to understand the range of outcomes
- **C**: Only considers the most optimistic case

## Answer

B

## Explanation

Scenario analysis: develops multiple valuation cases (base, bull, bear) with different assumptions. Produces value range, identifies key value drivers, stress-tests assumptions. More informative than point estimate. Weight scenarios by probability for expected value. Complements sensitivity analysis.

**Common mistakes:**
- Option A describes point-estimate valuation only
- Option C is one-sided—proper analysis includes downside
