# Question

Under weak-form market efficiency, investors cannot earn abnormal returns using:

## Options

- **A**: Fundamental analysis of financial statements
- **B**: Technical analysis of past price and volume data
- **C**: Insider information

## Answer

B

## Explanation

Weak-form efficiency: current prices fully reflect all historical price and volume information. Technical analysis cannot generate excess returns because patterns are already incorporated in prices. Fundamental analysis and private information might still add value. Strongest evidence supports weak-form efficiency.

**Common mistakes:**
- Option A applies to semi-strong form efficiency
- Option C applies to strong-form efficiency
