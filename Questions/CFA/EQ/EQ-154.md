# Question

A stock pays an annual dividend of $2.00, which is expected to grow at 5% indefinitely. If the required return is 10%, the intrinsic value using the Gordon Growth Model is:

## Options

- **A**: $40.00
- **B**: $42.00
- **C**: $20.00

## Answer

B

## Explanation

Gordon Growth Model: V₀ = D₁ / (r - g) = D₀(1+g) / (r - g) = $2.00 × 1.05 / (0.10 - 0.05) = $2.10 / 0.05 = $42.00. Uses next year's dividend (D₁), not current dividend (D₀). The model values perpetual growing dividend stream.

**Common mistakes:**
- Option A uses D₀ instead of D₁ ($2.00 / 0.05 = $40)
- Option C incorrectly uses total required return as denominator
