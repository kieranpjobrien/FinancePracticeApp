# Question

Industry classification systems like GICS help investors by:

## Options

- **A**: Guaranteeing investment returns by sector
- **B**: Organizing companies into comparable groups for analysis and benchmarking
- **C**: Setting stock prices for each industry

## Answer

B

## Explanation

GICS (Global Industry Classification Standard): hierarchical system—sectors, industry groups, industries, sub-industries. Developed by MSCI and S&P. Enables consistent company grouping for peer comparison, sector analysis, portfolio construction, index creation. Other systems: ICB, NAICS.

**Common mistakes:**
- Option A—classification doesn't affect returns
- Option C—classification is analytical, doesn't set prices
