# Question

Return on invested capital (ROIC) measures:

## Options

- **A**: The dividend return to shareholders
- **B**: How efficiently a company generates returns from its invested capital
- **C**: Stock price appreciation only

## Answer

B

## Explanation

ROIC = NOPAT / Invested capital. Measures operating profitability relative to capital invested in the business. If ROIC > WACC, company creates value; if ROIC < WACC, destroys value. Key metric for assessing competitive advantage and management effectiveness. Comparable across capital structures.

**Common mistakes:**
- Option A describes dividend yield
- Option C ignores fundamental value creation
