# Question

Currency risk in international equity investing affects returns when:

## Options

- **A**: The investor's home currency strengthens against the foreign currency
- **B**: Exchange rates change, either enhancing or diminishing foreign investment returns
- **C**: All currencies maintain fixed exchange rates

## Answer

B

## Explanation

Currency impact: foreign investment return in home currency = local return × currency change. Local currency depreciation hurts, appreciation helps foreign investor. Unhedged international investing has currency exposure. Can be hedged with forward contracts but adds cost. Two-way risk affects volatility.

**Common mistakes:**
- Option A only describes negative scenario
- Option C—fixed rates would eliminate currency risk
