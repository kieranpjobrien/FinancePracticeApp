# Question

While facilitating a team to establish its ground rules, the project manager encounters four distinct situations in a single workshop. Each calls for a different facilitation response to keep the agreement both owned by the team and genuinely useful. Match each situation (left) to the most appropriate facilitator response (right).

## Items

- **1**: Two members are arguing from fixed positions on a meeting-time rule and it is turning personal.
- **2**: The team has happily proposed a vague rule: "everyone should communicate well."
- **3**: A quiet member privately tells you afterwards she disagreed with a rule the group adopted enthusiastically.
- **4**: The team asks you, the project manager, what the "correct" rule on code review is.

## Matches

- **A**: Ask the team to make the rule observable by defining the specific behaviour it expects.
- **B**: Surface the underlying interests each side is protecting and have them design a rule serving both.
- **C**: Reflect the question back, prompting the team to decide the rule themselves while offering options if asked.
- **D**: Create a safe way for dissent to be voiced and revisit the rule so quiet objections are heard before it sticks.

## Answer

1-B, 2-A, 3-D, 4-C

## Explanation

Situation 1 is a clash of positions, so the facilitator surfaces underlying interests to craft a rule serving both (B), which defuses the personal heat. Situation 2 is an unmeasurable aspiration, so the response is to make it observable by naming the concrete behaviour expected (A). Situation 3 reveals suppressed dissent, which threatens real ownership; the facilitator creates a safe channel and revisits the rule so the objection is heard before it hardens (D). Situation 4 is the team deferring authorship to the project manager; reflecting the question back keeps ownership with the team while still offering options if they ask (C). Each situation maps to exactly one response because mismatching them — for instance, treating the positional conflict as a wording problem, or simply answering the team's deferral with "the correct rule" — would either escalate the conflict or strip the team of ownership.
