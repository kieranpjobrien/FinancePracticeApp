# Question

HR has offered a personality assessment to help you select team members for a complex regulatory project. Some candidates feel strongly that such tests are intrusive and should not drive hiring. What should the project manager do?

## Options

- **A**: Use the assessment as the primary filter for selection since it provides a standardised comparison between candidates that is hard to produce any other way.
- **B**: Refuse to use the assessment entirely because the candidate pushback means it is not a legitimate tool for team-building decisions in this project.
- **C**: Take the approach HR recommends without further consideration, since HR owns the policy on selection tools and the PM should not second-guess it.
- **D**: Use the assessment only as one input among several alongside skills, experience, and team-fit evidence, with transparent communication to candidates about how results are used.

## Answer

D

## Explanation

PMI treats selection tools as inputs to judgement, used transparently, not as a single filter or a political lightning rod. (D) combines multiple inputs with candidate transparency. (A) over-weights a single tool whose predictive validity is contested. (B) rejects a useful input rather than positioning it properly. (C) defers a project-critical decision entirely without applying PM judgement.
