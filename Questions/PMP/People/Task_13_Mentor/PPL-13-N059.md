# Question

A project manager supports stakeholders in several distinct ways across a programme. Match each developmental relationship to the description that best characterises it.

## Items

- **1**: Mentoring
- **2**: Coaching
- **3**: Sponsoring

## Matches

- **A**: A focused, often short-term partnership aimed at improving specific performance or unlocking a particular goal, typically question-led.
- **B**: A broader, relationship-based partnership where an experienced person supports another's longer-term growth, sharing perspective and guidance.
- **C**: Actively advocating for a person's advancement and visibility, using one's own standing to create opportunities for them.

## Answer

1-B, 2-A, 3-C

## Explanation

Mentoring is the broad, longer-term developmental relationship built on shared experience and guidance (1-B). Coaching is more focused and goal- or performance-oriented, typically drawing answers out through questions over a shorter horizon (2-A). Sponsoring is distinct from both: it is using one's own influence to advocate for and create opportunities for someone (3-C). Conflating these — for example treating sponsoring as just intensive mentoring — leads a PM to misjudge what a given stakeholder actually needs.
