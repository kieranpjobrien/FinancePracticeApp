# Question

A project manager mentors a developer on an agile team and has built a trusting, confidential relationship. The PM is then asked by the resource manager to provide a formal performance assessment of that same developer that will feed into pay decisions. The PM is concerned this will compromise the mentoring. What should the PM do?

## Options

- **A**: Provide the assessment using insights gained during mentoring, since the PM now knows the developer's strengths and weaknesses best.
- **B**: Flag the conflict to the resource manager and base any input strictly on observable work performance, keeping confidential mentoring discussions out of it.
- **C**: Decline to give any input at all, telling the resource manager only that a conflict exists.
- **D**: Tell the developer that mentoring confidentiality is suspended for now because the PM must report on their performance.

## Answer

B

## Explanation

The mentor must protect the confidentiality that underpins the relationship while still meeting a legitimate organisational request, which means surfacing the conflict and drawing only on observable performance, not private mentoring disclosures. (A) raids confidential conversations for an evaluation, destroying trust and misusing the relationship. (C) refuses a reasonable request the PM can fulfil within proper bounds. (D) breaks the confidentiality the mentoring depends on rather than separating the two roles.
