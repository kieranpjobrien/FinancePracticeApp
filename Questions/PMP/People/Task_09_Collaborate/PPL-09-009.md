# Question

A project requires collaboration with an external partner whose organisation runs a predictive/waterfall process, while the PM's team runs Scrum. How should the PM structure the collaboration?

## Options

- **A**: Persuade the partner to adopt Scrum for the joint work so both sides operate the same way.
- **B**: Switch the PM's own team to predictive delivery to match the partner's cadence for the engagement.
- **C**: Minimise joint working and pass deliverables back and forth at agreed contractual gates only.
- **D**: Agree interface points — weekly sync, milestone calendar, handoff artefacts — while each team runs its own method.

## Answer

D

## Explanation

PMI's hybrid and tailoring guidance is explicit that cross-organisational work should respect each team's internal method and define clean interfaces between them. (A) and (B) each force a methodology change on one party and generate friction that damages delivery. (C) treats collaboration as transactional handoffs and loses the value of joint problem-solving.
