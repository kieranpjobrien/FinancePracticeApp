# Question

Two stakeholder groups report to executives of equal seniority and hold conflicting requirements: one group wants maximum security controls, the other wants minimal friction for end users. What should the project manager do FIRST?

## Options

- **A**: Document both requirements and escalate jointly to both executives for a binding decision immediately.
- **B**: Facilitate a working session with both groups on underlying interests and co-design an option set.
- **C**: Implement the security requirements fully and treat usability as secondary because security is non-negotiable.
- **D**: Propose a 50/50 compromise that partially satisfies each of the conflicting stakeholder requirements.

## Answer

B

## Explanation

PMI expects the PM to attempt facilitated, interest-based resolution before escalating; modern security design often produces options that satisfy both groups once interests are understood. (A) escalates before the PM has facilitated the conversation they own. (C) picks a side on the PM's personal judgement rather than on stakeholder dialogue. (D) jumps to compromise without exploring whether both interests can be more fully met.
