# Question

A PM on a hybrid data-migration project has several open impediments but can only personally drive one this morning. One developer is blocked on the production deployment of a story that is otherwise done. Separately, a pending security sign-off on the shared migration framework is holding nothing up today, but if it is not cleared in the next two days it will block four developers who are all due to start framework-dependent stories then. Both need the PM to act outside the team. Where should the PM focus first?

## Options

- **A**: Drive the security sign-off, because clearing it now prevents a near-term blockage that would stall four developers, whereas the single deployment can be picked up shortly after.
- **B**: Drive the blocked deployment first, because it is stopping finished work from delivering value today, then turn to the security sign-off well within its two-day window.
- **C**: Tackle whichever impediment has the responsible approver most readily available this morning, to make concrete progress rather than waiting.
- **D**: Split the morning between both so the deployment moves and the security review is at least initiated, keeping either from stalling.

## Answer

A

## Explanation

Triage is about protecting the flow of value across the team, not just clearing what is loudest now: an impediment about to block four people imminently outweighs one blocking a single finished story that can be released a little later, so A maximises future throughput by acting while there is still lead time. (B) is genuinely tempting — finished work sitting undelivered is real waste — but the deployment can wait briefly without compounding, whereas the sign-off has a hard two-day fuse that will multiply the damage if missed. (C) optimising for approver availability is a reasonable tie-breaker but should not override relative impact. (D) splitting attention across both risks doing neither well and may not secure the sign-off, which is the higher-leverage item. The fine margin is between unblocking value already earned (B) and preventing a larger imminent blockage (A); the lead time on the sign-off is what tips it.
