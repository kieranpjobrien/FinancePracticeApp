# Question

On a hybrid pharmaceutical packaging upgrade, the project manager has resolved the same class of impediment four sprints running: each time an engineering change is needed, the regulatory affairs reviewer is unavailable for two weeks because she supports the whole portfolio, and the project manager personally chases each one to completion. The team is no longer surprised by the delay; they simply pad estimates. What should the project manager do now?

## Options

- **A**: Continue personally expediting each regulatory review, since this has reliably kept the project moving so far.
- **B**: Ask the team to keep padding estimates to absorb the regulatory delay so the schedule stays predictable.
- **C**: Request that regulatory reviews for this project be reclassified as low-risk so they can proceed without the dedicated reviewer.
- **D**: Document the recurring two-week regulatory bottleneck and its cumulative schedule impact and take it to the sponsor as a resourcing constraint to be solved at the portfolio level.

## Answer

D

## Explanation

A blocker that recurs every sprint from a portfolio-level resourcing constraint is no longer a one-off to expedite; the durable fix is to quantify the cumulative cost and escalate it to the sponsor as a structural problem only the organisation can resolve (D). (A) personal expediting has worked tactically but treats the symptom indefinitely and keeps the project hostage to one overloaded reviewer. (B) institutionalising padded estimates hides the impediment in the schedule rather than removing it. (C) reclassifying regulatory work to dodge the reviewer is a compliance risk and substitutes a shortcut for a legitimate process. The hard distinction is recognising that repeated tactical fixes signal a systemic cause that must now be escalated with evidence (D), not absorbed (A, B) or circumvented (C).
