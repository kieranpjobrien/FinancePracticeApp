# Question

A project manager coaching an automotive software team faces four distinct understanding gaps in the same fortnight. Each could loosely be tackled by more than one technique, so match each gap to the technique that most directly addresses its specific cause.

## Items

- **1**: The team and the product owner repeatedly clash over whether a feature is genuinely complete and shippable
- **2**: Distributed stakeholders keep losing track of overall progress and ask the same status questions between meetings
- **3**: Business and engineering repeatedly talk past each other and ship mismatched work because "trip", "session" and "journey" carry a different meaning for each group
- **4**: A specific complex story is interpreted differently by two developers despite a written description

## Matches

- **A**: Maintain a shared information radiator everyone can see at any time
- **B**: Agree and publish a definition of done
- **C**: Build a shared glossary of domain terms with the business
- **D**: Co-create concrete worked examples for the story

## Answer

1-B, 2-A, 3-C, 4-D

## Explanation

Each gap has a distinct cause that selects one best technique even though neighbours look plausible. Recurring disputes over completeness are a team-wide standards gap, resolved by a definition of done (1-B), not by clarifying any single story. Stakeholders losing the thread between meetings need a continuously visible source of truth, which an information radiator provides (2-A). The same words meaning different things is a vocabulary gap that a shared glossary fixes directly (3-C); a radiator would still display ambiguous terms, and a definition of done addresses completeness, not language. One complex story read two ways despite a written description is a story-level intent gap, best closed by co-creating concrete examples that pin down behaviour (4-D); a glossary aligns terms but not the specific scenario, and a definition of done sets the bar for "finished" without disambiguating what this story should do. The discrimination is matching the technique to the precise kind of understanding that is missing.
