# Question

What does the communication technique "checking for understanding" involve?

## Options

- **A**: Asking questions or using techniques to verify the audience has received and interpreted the message correctly.
- **B**: Confirming that all attendees were physically present for the relevant portion of the meeting.
- **C**: Reviewing previously distributed documentation to make sure it has been read.
- **D**: Conducting a formal post-training assessment of the audience's knowledge of the topic.

## Answer

A

## Explanation

Checking for understanding is a real-time technique — open questions, asking the receiver to paraphrase, or prompting them to describe how they will apply the information — that confirms the message landed as intended. (B) checks attendance, not comprehension. (C) checks that documentation exists and has supposedly been read, which is not the same as confirming current understanding. (D) is formal assessment, useful in training but heavier and not what the technique describes in everyday communication.
