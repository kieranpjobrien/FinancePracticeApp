# Question

A project manager faces several distinct negotiation situations on a logistics-software programme. For each situation, identify the conflict-handling approach the project manager is best advised to adopt. Match each situation to the most appropriate approach.

## Items

- **1**: A regulator's audit lands in three days and the vendor proposes deferring a control fix that the auditors will check; the project manager has reviewed it and is satisfied the fix is mandatory for sign-off.
- **2**: Two senior stakeholders want different reporting cadences; each cadence reflects a real reporting obligation their own function carries, and both will live with whatever is set for the rest of the programme.
- **3**: A long-standing partner team objects to the colour scheme on a shared dashboard the project manager owns; the project manager has no strong view and the partner's continued goodwill matters more than the styling.
- **4**: A shared test environment is double-booked for the only free week before two separate go-lives, both teams have a genuine claim, and a workable split must be settled in today's call.

## Matches

- **A**: Collaborate / problem-solve to find a solution that meets both parties' interests
- **B**: Assert the project manager's position and hold firm
- **C**: Compromise, with each side conceding something to reach a timely settlement
- **D**: Accommodate, yielding to the other party's preference

## Answer

1-B, 2-A, 3-D, 4-C

## Explanation

The skill here is reading circumstances rather than reciting definitions. In (1) the project manager has independently judged the fix mandatory and the audit is imminent, so asserting the position and holding firm is warranted despite vendor pushback (1-B). In (2) both cadences serve legitimate obligations and the outcome is durable, so investing in a solution that satisfies both interests pays off (2-A). In (3) the project manager has no real stake and values the relationship, so yielding on styling costs nothing and preserves goodwill (3-D). In (4) both claims are genuine and the matter must close today with no time to engineer a fully integrative fix, so a give-and-take split is the pragmatic settlement (4-C). The fine margin is between (2) and (4): both involve two valid positions, but the deciding factors are time and reversibility — collaborate when the outcome is lasting and there is room to work the problem, compromise when a timely settlement is forced and each side can absorb some loss.
