# Question

What does "anchoring" mean in a negotiation context?

## Options

- **A**: Holding firm to a position throughout the negotiation regardless of any new information offered.
- **B**: Taking a strategic break mid-negotiation to reset the negotiating dynamic between the parties.
- **C**: Building personal rapport with the counterparty before discussing any substantive commercial issues.
- **D**: Using an opening offer that sets a reference point and biases subsequent discussion.

## Answer

D

## Explanation

Anchoring is a cognitive bias the first concrete number exploits — once stated, both sides tend to negotiate around it even if it is arbitrary. (A) describes positional rigidity, not anchoring. (B) is a tactical pause technique. (C) describes rapport-building, which is unrelated to the anchoring concept.
