# Question

A project manager is structuring three cost-reimbursable work packages on a research-heavy programme. None has a scope certain enough for a fixed-price deal, but the buyer wants each contractor motivated in the way that best fits the work. Match each work package to the incentive arrangement that best aligns the contractor's reward with the buyer's goal.

## Items

- **1**: Development work where costs will be fully auditable and the buyer wants the contractor to share directly in any savings against a target cost
- **2**: Early-stage research where the deliverables are hard to measure objectively and the buyer wants to reward overall quality and responsiveness as it judges them over time
- **3**: A package where the cost outturn is uncertain but the technical performance achieved (such as throughput) can be measured precisely, and the buyer wants reward tied to that performance

## Matches

- **A**: Cost-plus-award-fee, where a discretionary fee is granted on the buyer's periodic performance judgements
- **B**: Cost-plus-incentive-fee with a cost-sharing formula against a target cost
- **C**: Cost-plus-incentive-fee with the incentive tied to a measured technical performance target

## Answer

1-B, 2-A, 3-C

## Explanation

All three are cost-reimbursable, so the discrimination is which incentive mechanism fits the buyer's specific goal and what can actually be measured. Work package 1 has auditable costs and a desire to share savings against a target, which is exactly a cost-plus-incentive-fee with a cost-sharing formula (B). Work package 2 has deliverables that resist objective measurement and a buyer who wants to reward judged quality and responsiveness, which suits a cost-plus-award-fee where the fee rests on the buyer's periodic assessment rather than a formula (A). Work package 3 has measurable technical performance and a buyer who wants reward tied to that performance, fitting a cost-plus-incentive-fee whose incentive is linked to the performance target rather than to cost (C). The trap is treating "incentive fee" as a single answer; the right choice turns on whether the incentive should run off cost savings, off measured technical performance, or off subjective judgement when nothing objective is available.
