# Question

You are building a skills matrix for a games-studio team moving to a new real-time rendering engine. The matrix maps each person against the competencies the upcoming work needs. Used well, the matrix can do more than record current skills. Which uses of the skills matrix genuinely help you ensure the team becomes adequately trained for the work ahead? (Select THREE.)

## Options

- **A**: Highlight competencies held by only one person so you can plan cross-skilling and reduce single-person dependencies.
- **B**: Reveal where current capability falls short of what the upcoming rendering work requires, so development targets the real gaps.
- **C**: Treat the matrix as a one-off snapshot captured at the start of the work, since re-rating people as they develop would create ongoing overhead.
- **D**: Pair people with a competency against those who need it, planning on-the-job development through the actual work.
- **E**: Route every rendering task to whoever is already rated strongest in it, to keep quality high as the new engine beds in.

## Answer

A, B, D

## Explanation

A skills matrix is a living planning instrument for building capability, and these three uses serve that purpose. (A) exposes single-person dependencies so you can deliberately spread the skill before it becomes a bottleneck. (B) compares current capability against the work's requirements, pinpointing the gaps that development should close. (D) turns the matrix into a pairing plan, matching those who hold a competency with those who need it for on-the-job learning. (C) freezing it as a start-of-work snapshot defeats the point: the matrix has to be updated as people develop, or it stops reflecting reality and cannot guide the next development decision. (E) always assigning work to the current expert protects short-term quality but starves everyone else of the chance to build the competency, entrenching the very single-person dependencies the matrix is meant to help you remove. The matrix works when it drives development over time, not when it locks work to today's strengths.
