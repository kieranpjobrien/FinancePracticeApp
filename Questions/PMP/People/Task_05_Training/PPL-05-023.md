# Question

What is the primary purpose of a skills matrix on a project team?

## Options

- **A**: Rank team members so the strongest performers can be moved to the most critical work.
- **B**: Document each team member's proficiency across the skills the project needs, in order to plan resourcing and development.
- **C**: Provide input to the annual performance review and compensation process.
- **D**: Identify team members who should be replaced before the project starts.

## Answer

B

## Explanation

A skills matrix is a planning tool for resourcing and development, not a ranking or evaluation tool, which is what (B) describes. (A) misuses the matrix to rank people. (C) repurposes it for HR processes it was not built for. (D) treats a development tool as a basis for termination decisions, which damages trust and rarely informs good staffing.
