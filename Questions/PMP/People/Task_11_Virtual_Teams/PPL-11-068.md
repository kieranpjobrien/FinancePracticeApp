# Question

A PM supporting a distributed agile team encounters several recurring problems that each have a clear, distinct root cause. Match each symptom on the virtual team to the intervention that best addresses its underlying cause.

## Items

- **1**: Decisions made in the team's one daily overlap call are routinely forgotten or disputed a week later, with members recalling them differently.
- **2**: Two time zones never overlap during working hours, so a question from one zone waits a full day for an answer and work stalls.
- **3**: Team members feel like isolated individuals rather than a team and rarely interact beyond assigned hand-offs.
- **4**: A newcomer cannot find current project state because information is scattered inconsistently across chat, email and three different tools.

## Matches

- **A**: Establish a single source of truth and an agreed convention for where each type of information lives.
- **B**: Capture decisions in a shared, written decision log at the moment they are made.
- **C**: Create deliberate social and informal-interaction opportunities so relationships form beyond task hand-offs.
- **D**: Restructure work so each zone can make progress independently, reducing cross-zone blocking dependencies.

## Answer

1-B, 2-D, 3-C, 4-A

## Explanation

Each symptom has a specific cause that points to one intervention. Decisions remembered differently a week later are an artefact problem — they were never written down — so a decision log captures them in durable form (1-B). A full-day wait caused by zero overlap is a dependency-structure problem; reducing cross-zone blocking lets each zone progress without waiting (2-D). A team that feels like isolated individuals lacks the informal contact that builds cohesion, which deliberate social opportunities supply (3-C). Information scattered across many tools is a single-source-of-truth gap, resolved by agreeing where each type of information lives (4-A). The discriminator is matching the fix to the actual root cause rather than to a superficially adjacent remedy.
