# Question

A pharmaceutical company is introducing a new laboratory information management system that changes how technicians record results. The project manager is supporting the organisational change across the project's life. Match each situation to the change-support action that best fits it.

## Items

- **1**: Planning is starting and the project manager needs to know how much each lab's routine will change and how prepared each team is.
- **2**: A long-serving lab supervisor is openly negative, and junior technicians take their cue from that person.
- **3**: Technicians are trained and capable but keep using the old recording sheet because they trust it more.
- **4**: Adoption is solid and the project is closing; the quality function will run the system from now on.

## Matches

- **A**: Engage the influential supervisor and work to convert them into an advocate.
- **B**: Reinforce the new behaviour and show evidence that the new system is reliable.
- **C**: Conduct a change-impact and readiness assessment across the labs.
- **D**: Hand over the process, measures, and support materials to the operating function.

## Answer

1-C, 2-A, 3-B, 4-D

## Explanation

Each situation calls for a different change lever. Needing to know how deeply each lab is affected and how ready it is, before designing the rollout, is the purpose of an impact and readiness assessment (1-C). An openly negative, influential supervisor whom juniors follow is best addressed by engaging that person and turning them into an advocate (2-A). Capable technicians who distrust the new system and revert to the old sheet need reinforcement and proof of reliability, not more training (3-B). Solid adoption at closure calls for transferring the process, measures, and materials to the quality function so the change is sustained (4-D).
