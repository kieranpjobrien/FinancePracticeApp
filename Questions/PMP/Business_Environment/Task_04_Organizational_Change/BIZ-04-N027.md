# Question

A media company is moving its newsroom from a desktop publishing tool to a cloud-based collaborative platform, delivered in agile increments. Early sprints have shipped, but journalists who try the platform tend to revert to the old tool by the next day. One senior editor, widely respected across the newsroom, has quietly become a confident, enthusiastic user and has offered to help however is most useful. The project manager wants to convert that goodwill into broader adoption. How should the editor's influence best be deployed?

## Options

- **A**: Ask the editor to use the platform openly for their own live stories and to coach colleagues informally as they hit problems, so peers see a respected user succeeding with it day to day.
- **B**: Ask the editor to run a series of formal training sessions teaching the rest of the newsroom how to use the platform's features.
- **C**: Ask the editor to author the definitive how-to guide and feature reference for the platform, so colleagues have an authoritative source written by someone they trust.
- **D**: Ask the editor to gather and bring colleagues' problems and objections back to the project team each sprint, so the platform can be improved to win people over.

## Answer

A

## Explanation

The newsroom is reverting not because the platform is unknown but because no trusted peer is visibly succeeding with it; the editor's credibility does the most work when they model the new behaviour on real stories and coach colleagues in the moment, giving doubters a respected example to follow (A). Running formal training (B) recasts a peer advocate as an instructor and addresses know-how, which is not the gap — people can use it but choose not to. Writing the reference guide (C) is genuinely useful but converts personal influence into a static document, spending the editor's credibility where it persuades least. Channelling colleagues' objections back to the team (D) is a sound feedback mechanism and a fair use of the editor's reach, but it improves the product rather than directly shifting peer behaviour now; the strongest lever is the visible, modelling-and-coaching presence that makes adoption feel normal.
