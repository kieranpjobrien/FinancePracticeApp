# Question

Phase 1 of a phased rollout went live with a wave of usability complaints. Phase 2 is due to start in two weeks. What should the project manager do?

## Options

- **A**: Proceed with phase 2 on schedule because the original rollout plan was already approved.
- **B**: Push back phase 2 indefinitely until every last phase 1 complaint has been fully closed.
- **C**: Set lower expectations for phase 2 by warning incoming users about known limitations upfront.
- **D**: Address critical issues, communicate fixes to phase 1, and bring lessons into phase 2.

## Answer

D

## Explanation

Fix the most damaging issues, show responsiveness, and improve the next wave (D). (A) ignores the data and erodes trust. (B) over-corrects when fixes can run in parallel. (C) manages perception instead of solving the problem.
