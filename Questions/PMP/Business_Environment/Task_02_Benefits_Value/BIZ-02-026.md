# Question

Earned value reporting shows SPI 1.15 and CPI 1.10. UAT, however, indicates users are deeply dissatisfied with the solution's usability. How should the project manager interpret this?

## Options

- **A**: Treat the project as on track because EVM indicators are favourable.
- **B**: Treat usability as out-of-scope for project performance reporting.
- **C**: Note the EVM strength but flag a benefits risk; usability could undermine adoption and value.
- **D**: Push for early closure to capture the favourable cost and schedule position.

## Answer

C

## Explanation

EVM measures schedule and cost, not value (C); favourable indicators do not guarantee adoption. (A) confuses delivery efficiency with success. (B) excludes the very signal that predicts benefit realisation. (D) closes a project before its value risk is resolved.
