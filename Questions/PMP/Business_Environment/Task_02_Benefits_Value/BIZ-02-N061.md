# Question

A university is funding a project to redesign its student-services portal. Part of the justification is a tangible saving in call-centre volume, but the larger part of the sponsor's argument rests on "improved student satisfaction," which the business case states but does not quantify. The finance director is sceptical that an unquantified benefit should count towards the investment decision. As the PM, how should you handle the satisfaction benefit so it can legitimately inform the decision?

## Options

- **A**: Define a measurable proxy for satisfaction — such as a survey-based satisfaction score with a baseline and target — so the benefit can be tracked and weighed alongside the call-centre saving.
- **B**: Drop the satisfaction benefit from the case and justify the project on the call-centre saving alone, since only quantifiable benefits should drive the decision.
- **C**: Keep satisfaction as a stated qualitative benefit with no measure, trusting the sponsor's judgement that it is real and important.
- **D**: Assign a notional monetary value to the satisfaction improvement so it can be added directly to the call-centre saving in a single financial total.

## Answer

A

## Explanation

Intangible benefits can legitimately inform a decision provided they are made measurable through a defined proxy with a baseline and target — a satisfaction score lets the benefit be tracked and weighed credibly rather than asserted (A). B discards a benefit the sponsor considers the larger part of the value simply because it is currently unquantified, narrowing the case and potentially undervaluing the investment. C leaves the benefit as an unmeasured claim, which is exactly the finance director's objection — without a measure it cannot be tracked, verified, or defended. D over-reaches by inventing a monetary figure with no defensible basis and folding it into a hard financial total, lending false precision that is more misleading than leaving it as a tracked non-financial measure. The fine margin is A versus D: both try to make satisfaction count, but A makes it *measurable* without fabricating a currency value, whereas D manufactures a number that cannot withstand the scrutiny the finance director is already applying.
