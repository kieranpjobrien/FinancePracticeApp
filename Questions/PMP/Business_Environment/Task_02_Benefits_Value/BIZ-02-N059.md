# Question

A two-year transformation programme at a utility is at its midpoint gate. The PM is assembling the pack the governance board needs to decide whether continued funding is justified on value grounds. Which pieces of information are most relevant to that continuation decision? (Select all that apply.)

## Options

- **A**: The remaining cost to complete the programme from this point forward.
- **B**: The current best forecast of the benefits still to be realised.
- **C**: The total amount already spent to reach the midpoint gate.
- **D**: Any material changes to the assumptions the original benefit case relied on.
- **E**: The number of change requests processed since the programme began.

## Answer

A, B, D

## Explanation

A forward-looking value decision weighs the cost still to be incurred (A) against the benefits still to be realised (B), with explicit attention to whether the assumptions underpinning those benefits still hold (D) — these three frame whether finishing is worthwhile from here. The amount already spent (C) is sunk: it cannot be recovered by stopping or continuing, so importing it into a forward decision distorts it, which is the classic sunk-cost trap. The count of change requests (E) is a process-activity metric that says nothing about whether remaining value justifies remaining cost. The deliberate distractor is C, which feels material because it is large and real, but is precisely the figure a disciplined continuation decision must exclude.
