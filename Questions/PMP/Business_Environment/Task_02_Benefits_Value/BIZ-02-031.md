# Question

What is the primary purpose of a benefits realisation plan?

## Options

- **A**: To track project costs against the approved budget baseline.
- **B**: To document the work breakdown structure for delivery teams.
- **C**: To list project deliverables in the order they will be released.
- **D**: To define how, when, and by whom expected benefits will be measured and realised.

## Answer

D

## Explanation

A benefits realisation plan ties expected benefits to owners, measures, and timing so value is tracked beyond delivery (D). (A) is cost management. (B) is scope decomposition. (C) is a release plan, which delivers outputs but says nothing about outcomes.
