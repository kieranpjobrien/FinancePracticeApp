# Question

A delivered project achieved 90% of projected cost savings but cannot demonstrate any improvement in employee satisfaction because no baseline survey was ever taken. What lesson should the project manager carry to future projects?

## Options

- **A**: Drop intangible benefits from future business cases and focus only on financial metrics.
- **B**: Plan benefits measurement, including baseline data collection, for every benefit cited in the business case.
- **C**: Use industry benchmarks as a substitute when a project baseline is not available.
- **D**: Treat intangible benefits as inherently unmeasurable and report them only narratively.

## Answer

B

## Explanation

The fix is in planning: every benefit needs a measurement plan and a baseline before delivery starts (B). (A) discards legitimate value because it was poorly tracked, not because it doesn't exist. (C) compares against another organisation's data, which does not show your improvement. (D) accepts the planning failure as a feature.
