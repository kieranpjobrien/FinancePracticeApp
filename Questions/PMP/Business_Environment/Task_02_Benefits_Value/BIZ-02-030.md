# Question

After five Scrum sprints, the product owner asks how to tell whether the releases are actually delivering value. What should the project manager recommend?

## Options

- **A**: Use velocity as the indicator of value being delivered each sprint.
- **B**: Use story points completed as the value measure for each release.
- **C**: Wait until project closure when full value can be assessed in one report.
- **D**: Define outcome metrics tied to user behaviour and business results and track them release by release.

## Answer

D

## Explanation

Value lives in user outcomes and business results, not throughput (D); track those each release. (A) and (B) measure team output, not value. (C) defers feedback in a methodology designed to learn early.
