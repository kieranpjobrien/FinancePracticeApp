# Question

Mid-project, the sponsor asks the project manager to add a new feature that they believe will substantially increase business value. What should the project manager do FIRST?

## Options

- **A**: Add the feature immediately because the request comes from the sponsor.
- **B**: Decline the request because the feature was not in the approved baseline scope.
- **C**: Assess the feature's value, schedule, cost, and risk impact and submit a change request.
- **D**: Defer the request to the post-implementation phase as a future enhancement.

## Answer

C

## Explanation

Even sponsor-driven changes follow change control (C): impact assessment, then a change request. (A) bypasses the process and the baselines. (B) rejects without evaluation, ignoring potential value. (D) parks a potentially valuable change without analysis.
