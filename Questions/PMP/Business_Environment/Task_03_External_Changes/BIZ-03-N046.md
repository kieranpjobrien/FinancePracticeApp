# Question

A project manager runs a hybrid infrastructure programme and, in the same week, meets four external developments. Each could be met with monitoring, an options analysis, a backlog change, or a contractual review. Match each development to the response that best fits its specific circumstances.

## Items

- **1**: A regulator publishes a final, enacted rule the current design already meets, but signals a stricter second phase is being drafted with no fixed date.
- **2**: An enacted rule clearly applies and the design breaches it, the remedy is well understood, and only the sequencing of the fix within delivery is open.
- **3**: A competitor launch makes it likely, but not yet evidenced, that the market value of several planned features has changed.
- **4**: A currency move sharply raises a sole-source supplier's cost mid-contract, and the contract's wording on who absorbs such movements is ambiguous.

## Matches

- **A**: Pull the remedy into the team's plan and prioritise its sequencing against the rest of the work.
- **B**: Treat it as uncertainty, monitor for the trigger, and steer present design choices to stay low-regret.
- **C**: Run an options analysis with the product owner on what to re-order, validated against real demand signals.
- **D**: Quantify the exposure and review the contractual allocation of the cost with procurement.

## Answer

1-B, 2-A, 3-C, 4-D

## Explanation

Item 1's enacted rule is already satisfied, so the live concern is the undated future phase, an uncertain event managed by monitoring and keeping current choices low-regret rather than building to an unfinalised text (1-B). Item 2 is a confirmed breach with a known remedy where only timing is open, so it is no longer an analysis problem but a planning-and-sequencing one (2-A). Item 3's value shift is plausible but unevidenced, so the fit is an options analysis tied to real demand signals before committing a re-order, not an immediate backlog change on assumption (3-C). Item 4 couples a quantifiable cost move with ambiguous contract wording, making the first response to size the exposure and review the contractual allocation with procurement (4-D). The matches turn on how settled and how understood each change is, not on a keyword in the wording: items 1 and 2 are both enacted regulation yet call for opposite responses, and 3 and 4 both involve cost or value yet diverge on whether evidence or contract is the gating question.
