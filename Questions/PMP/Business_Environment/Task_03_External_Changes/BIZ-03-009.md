# Question

A core third-party API your project relies on will be discontinued in 18 months. Your project completes in 6 months. What should the project manager do?

## Options

- **A**: Open a contract negotiation with the provider asking them to extend the deprecation date.
- **B**: Switch to an alternative API now to remove the dependency before the planned launch.
- **C**: Continue as planned because the API will still be live throughout the delivery window.
- **D**: Assess implications, identify alternatives, and bring options to stakeholders for decision.

## Answer

D

## Explanation

The product will outlive the API even if the project does not, so stakeholders deserve the choice (D). (A) is unlikely to land and addresses only timing. (B) commits to migration without an impact analysis. (C) ships a known-end-of-life dependency without informing the buyer.
