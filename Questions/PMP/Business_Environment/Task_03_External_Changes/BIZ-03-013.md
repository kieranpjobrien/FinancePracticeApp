# Question

A pandemic forces the organisation into long-term remote working. The customer service platform you are building was designed for a centralised call centre. What should the project manager do?

## Options

- **A**: Pause the project until normal operations resume in a few months.
- **B**: Reassess requirements with stakeholders and re-prioritise the backlog around distributed working needs.
- **C**: Continue the original scope and treat remote work as a separate follow-on project.
- **D**: Press on with original plans because the disruption is expected to be temporary.

## Answer

B

## Explanation

A change of this magnitude in the operating model rewrites the requirements (B); re-prioritisation is the agile-aware response. (A) waits without basis. (C) delays needed value into a future project. (D) overrides current evidence with optimism.
