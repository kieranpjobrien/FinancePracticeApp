# Question

How should significant external environment changes be communicated to project stakeholders?

## Options

- **A**: Through formal change requests routed via the change control board.
- **B**: After the PM has decided on the response so the message is clear.
- **C**: Only when a change directly affects schedule or budget.
- **D**: Proactively, with impact analysis and a recommended response for stakeholder decision.

## Answer

D

## Explanation

Stakeholders deserve early visibility plus the analysis they need to decide (D). (A) confuses external change communication with internal change control. (B) bypasses input on decisions that often belong to the sponsor. (C) is too narrow; external change touches scope, value, and risk too.
