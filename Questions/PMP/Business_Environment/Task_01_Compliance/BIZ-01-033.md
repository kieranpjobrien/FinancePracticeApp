# Question

A vendor delivering a critical module has not provided the SOC 2 report required by your security policy. Integration testing is scheduled to start in two weeks. What should the project manager do?

## Options

- **A**: Proceed with integration testing now and request the SOC 2 report ahead of go-live.
- **B**: Replace the vendor immediately and restart procurement to avoid any further schedule slip.
- **C**: Substitute a security questionnaire for the SOC 2 and treat it as equivalent evidence.
- **D**: Engage procurement and the vendor to obtain the report before testing, escalating if blocked.

## Answer

D

## Explanation

Compliance evidence must be in place before exposing systems to vendor code (D): the PM works with procurement to enforce the contractual obligation. (A) accepts a known compliance gap during a phase that may already create risk. (C) substitutes self-attestation for independent assurance, lowering the standard. (B) is disproportionate without first attempting to resolve through normal vendor management.
