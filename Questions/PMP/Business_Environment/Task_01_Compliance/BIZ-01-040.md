# Question

Mid-execution, a regulator releases new guidance that clarifies (but does not change) an existing rule the project already addresses. The team's current design happens to align with the new guidance. What should the PM do?

## Options

- **A**: Submit a change request to formally adopt the guidance into the project baseline.
- **B**: Notify the regulator that the project is already compliant with the new guidance.
- **C**: Pause work until legal counsel confirms in writing that no action is needed.
- **D**: Document the alignment in the compliance register and note it in the next status report.

## Answer

D

## Explanation

When existing controls already satisfy new guidance, the proper action is to document the conformance and move on (D). (A) creates change-control overhead for a baseline that does not need to change. (C) halts work needlessly. (B) is unnecessary outreach; regulators do not need unsolicited compliance attestations.
