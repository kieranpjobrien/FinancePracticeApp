# Question

A consumer-electronics firm is finalising the quality budget for a new product line. Field-return data from the previous, very similar line shows that most warranty returns traced back to a small number of assembly-step errors that inspection had not been catching. The quality lead's draft plan keeps the same end-of-line inspection regime as before and adds a larger warranty reserve to cover the expected returns. The project manager is reviewing the draft before sign-off. Which change should the project manager press for first?

## Options

- **A**: Redirect part of the budget into mistake-proofing and operator training at the assembly steps the return data implicates, so the recurring errors are not produced.
- **B**: Increase the end-of-line inspection sampling rate and tighten the screening criteria so more of the defective units are caught before they ship.
- **C**: Add in-process inspection checkpoints after the implicated assembly steps so defects are detected closer to where they are introduced.
- **D**: Keep the reserve but commission an independent review of the warranty data to confirm the failure pattern before changing the production approach.

## Answer

A

## Explanation

The previous line already proved that inspection does not catch these particular assembly errors, and the return data points to specific, known steps. The most cost-effective move is to stop those errors being created at the implicated steps rather than spending more to detect or to absorb them. (A) targets prevention at the identified source. (B) and (C) both invest in better appraisal — sampling, screening, in-process checks — which detect defects but accept that they keep being produced, and inspection has already been shown to miss this failure mode. (D) is reasonable diligence, but the failure pattern is already evidenced by field returns from a near-identical line; delaying the prevention decision for further confirmation lets the next batches be built the same flawed way.
