# Question

A precision-machining project tracks the diameter of a turned shaft on a control chart. The last eight consecutive measurements all fall on the same side of the centre line, though every one is comfortably inside the upper and lower control limits and within the customer's specification limits. The operator wants to keep running because no point has breached a limit. What should the project manager conclude?

## Options

- **A**: The process is fine to continue because all points are within both the control limits and the specification limits.
- **B**: The run of eight points on one side of the centre line signals a non-random pattern, indicating the process has shifted and should be investigated even though no limit is breached.
- **C**: The control limits are set too wide and should be tightened to the specification limits so the shift would have triggered an out-of-limit point.
- **D**: The shift is acceptable because the parts still meet the customer's specification, which is the only requirement that matters.

## Answer

B

## Explanation

Control charts flag an out-of-control condition not only when a point crosses a control limit but also when a non-random pattern appears — a run of seven or more consecutive points on one side of the centre line being a classic rule. Eight points on the same side indicates an assignable cause has shifted the process, warranting investigation before defects occur. (B) is correct. (A) and (D) judge only against limit breaches and the spec, missing the pattern signal. (C) confuses control limits (the voice of the process) with specification limits (the voice of the customer); control limits are derived from process variation, not arbitrarily tightened to the spec.
