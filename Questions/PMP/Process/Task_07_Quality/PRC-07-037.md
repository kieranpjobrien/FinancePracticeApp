# Question

Which of the following is an example of an appraisal cost in PMI's cost of quality model?

## Options

- **A**: Training the team on the new test framework.
- **B**: Reworking a module that failed system test.
- **C**: Conducting code reviews and inspections of deliverables.
- **D**: Settling a customer dispute caused by a defect found in production.

## Answer

C

## Explanation

Appraisal costs are spent finding defects: testing, reviews, inspections, audits. (A) is a prevention cost. (B) is internal failure (defect found before delivery). (D) is external failure (defect found after delivery). All three are still costs of quality, but only (C) is appraisal.
