# Question

A scatter diagram shows a strong positive correlation between hours of testing performed on a module and the number of defects found in it. What is the most appropriate conclusion for the project manager?

## Options

- **A**: More testing causes more defects, so testing intensity should be reduced.
- **B**: The relationship warrants investigation because correlation does not establish causation.
- **C**: Testing is clearly effective, so current effort levels should be maintained.
- **D**: Modules with more testing are more defective and should be re-architected.

## Answer

B

## Explanation

A scatter diagram describes a relationship; it does not explain it. Heavily tested modules might be the riskier ones, the more complex ones, or simply the ones owned by a more diligent team. (A) inverts cause and effect. (C) accepts the pattern as confirmation without understanding it. (D) leaps to a structural conclusion based on a graph alone. Only (B) reflects sound analytical practice.
