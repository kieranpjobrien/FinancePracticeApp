# Question

A project manager is classifying quality-related costs on a software product into cost-of-quality categories for a steering report. Each cost below is easy to misfile. Match each cost to its correct category.

## Items

- **1**: Running the automated regression test suite each build to check whether new defects have been introduced
- **2**: Re-testing and re-releasing a build after the team fixed defects that those tests had found before release
- **3**: Paying contractual penalties to a customer after a defect that escaped to production breached an agreed service level

## Matches

- **A**: Internal failure cost
- **B**: External failure cost
- **C**: Appraisal cost

## Answer

1-C, 2-A, 3-B

## Explanation

Appraisal costs are incurred to examine and detect whether defects are present, so running the regression suite to find defects (1) is an appraisal cost (C) — it is inspection, not the rework itself. Internal failure costs arise from defects found before the product reaches the customer, so the re-testing and re-releasing effort caused by those defects (2) is an internal failure cost (A). External failure costs arise from defects that reach the customer, including consequential costs such as contractual penalties for the SLA breach (3), making it an external failure cost (B). The common traps are filing the test run as prevention (it detects, so it is appraisal) and filing the penalty as merely a fee rather than a failure cost of an escaped defect.
