# Question

A PM is choosing between two vendors. Vendor A bids $200,000 with a 20% chance of a $150,000 cost overrun. Vendor B bids $230,000 with a 10% chance of a $100,000 overrun. Using expected monetary value, which is the better choice and by how much?

## Options

- **A**: Vendor A is better by $10,000 because the base bid gap outweighs the higher expected overrun cost.
- **B**: Vendor B is better by $10,000 because its lower-risk profile beats Vendor A's cheaper base bid.
- **C**: Vendor A is better by $40,000 because the base bid gap is the only meaningful comparison here.
- **D**: Vendor B is better by $30,000 because expected overrun cost outweighs the cheaper Vendor A base bid.

## Answer

A

## Explanation

EMV(A) = 200,000 + (0.2 × 150,000) = 230,000. EMV(B) = 230,000 + (0.1 × 100,000) = 240,000. Vendor A is cheaper by $10,000 on EMV. (A) applies EMV correctly. (B) reverses the conclusion despite correct direction on risk profile. (C) ignores expected overrun entirely. (D) treats expected overrun cost as dominant when it isn't.
