# Question

A PM has identified five risks with expected monetary values totalling $120,000 (contingency reserve). The sponsor has also approved a $60,000 management reserve. The work package cost estimate is $800,000 before any reserves are added. What are the cost baseline and the total project budget?

## Options

- **A**: Cost baseline $860,000; total project budget $920,000, with management reserve included inside the cost baseline.
- **B**: Cost baseline $920,000; total project budget $980,000, with management reserve added above the cost baseline.
- **C**: Cost baseline $920,000; total project budget $920,000, with no additional management reserve added on top.
- **D**: Cost baseline $800,000; total project budget $980,000, with both reserves held above the cost baseline entirely.

## Answer

B

## Explanation

Contingency reserve for identified risks sits inside the cost baseline; management reserve for unknown-unknowns sits outside. Baseline = 800,000 + 120,000 = 920,000. Total budget = 920,000 + 60,000 = 980,000. (B) applies the definitions correctly. (A) misplaces both reserves. (C) omits management reserve from total budget. (D) incorrectly excludes contingency reserve from baseline.
