# Question

A project manager on a hybrid satellite-ground-station build is finalising responses for four open threats with the team. Each threat below has been assessed, and the project manager wants to label each chosen action with the threat response strategy it represents. Match each action to the strategy it best illustrates.

## Items

- **1**: For a low-impact risk that a minor reporting tool may occasionally be slow, the team decides to take no proactive action but to set aside a small time buffer in case it occurs.
- **2**: For the risk of damage during equipment transport, the team purchases specialist cargo insurance so a third party bears the financial loss if damage occurs.
- **3**: For the risk that a complex bespoke antenna design fails testing, the team removes the bespoke element and adopts a proven off-the-shelf antenna instead.
- **4**: For the risk of integration defects, the team adds extra automated regression testing each iteration to lower the chance that defects reach the customer.

## Matches

- **A**: Avoid
- **B**: Transfer
- **C**: Mitigate
- **D**: Accept (active)

## Answer

1-D, 2-B, 3-A, 4-C

## Explanation

Each action maps to a distinct threat strategy. Holding a buffer while taking no proactive action is active acceptance (1-D). Buying insurance shifts the financial consequence to a third party, which is transfer (2-B). Eliminating the bespoke design so the failure mode cannot occur is avoidance (3-A). Adding regression testing to reduce the probability of defects reaching the customer is mitigation (4-C). The traps are that all four reduce overall exposure in some sense, but only mitigation lowers probability/impact while the risk remains, only avoidance removes the cause, only transfer keeps the risk but reassigns its cost, and only acceptance takes no action to change the risk itself.
