# Question

A risk has a 30% probability of occurring and would cost the project $80,000 if it does. What is the Expected Monetary Value (EMV) of this risk?

## Options

- **A**: -$8,000
- **B**: -$16,000
- **C**: -$24,000
- **D**: -$56,000

## Answer

C

## Explanation

EMV = Probability x Impact = 0.30 x (-$80,000) = -$24,000. The negative sign indicates a threat (cost). (A) would come from using 10% probability. (B) uses 20% probability. (D) reflects the residual $56,000 (70% of $80,000) which represents the non-occurrence complement, not EMV.
