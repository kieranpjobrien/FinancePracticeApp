# Question

A change request marked "deferred" by the CCB means which of the following?

## Options

- **A**: The request has been formally rejected and will not be considered again.
- **B**: The request has been approved and is queued for later implementation.
- **C**: The request requires more information before any decision can be made.
- **D**: The request is set aside without approval or rejection and may be reconsidered later.

## Answer

D

## Explanation

Deferral is a distinct CCB disposition: the change is neither approved nor rejected, but parked for later reconsideration when conditions change. (A) conflates deferral with rejection. (B) conflates it with approval; deferred requests are not authorised for implementation. (C) describes a request returned for clarification, which is a different status.
