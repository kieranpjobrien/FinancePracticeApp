# Question

A predictive enterprise-software project is four days from go-live and has entered a planned change freeze under which only fixes essential to launch are accepted. A business stakeholder submits a change request to relabel several screens to match a brand guideline issued this week, arguing the launch communications and press demo will show the old labels. The relabelling is low risk and not required for the system to function at launch. How should the project manager handle it?

## Options

- **A**: Take the request to the change control board as a candidate exception, given the stakeholder's launch-communications rationale, and let the board weigh it against the freeze.
- **B**: Log the request, confirm with the stakeholder that it falls outside the launch-critical criteria, and schedule it for the first post-launch release.
- **C**: Approve the relabelling under the freeze, treating the brand alignment for the launch demo as effectively launch-critical given the executive visibility.
- **D**: Reject and close the request, since the change freeze admits only launch-essential fixes and this is cosmetic.

## Answer

B

## Explanation

A change freeze admits only launch-critical changes, and screen relabelling is not required for the launch to function, so the correct handling is to log it and schedule it for a near-term post-launch release rather than destabilise an imminent go-live. (B) honours the freeze and preserves the request. (A) escalates a request that plainly fails the launch-critical test as if its status were genuinely ambiguous, inviting the board to relitigate a freeze the project already set; the PM should apply the criteria, not punt a clear-cut case upward. (C) stretches "launch-critical" to cover a cosmetic preference because executives will see it, which is exactly the rationalisation a freeze exists to resist. (D) rejects rather than defers, discarding a legitimate request the business still wants delivered shortly after launch. Deferral to a post-launch release, not exception-seeking or rejection, is the disciplined response.
