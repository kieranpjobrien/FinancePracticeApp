# Question

A risk the team identified and for which they planned an active response has now materialised. The documented response will consume part of the contingency reserve, which sits within the project manager's control and inside the cost baseline. Carrying out the response will, however, also push out a milestone on the approved schedule baseline. What should the project manager do?

## Options

- **A**: Execute the documented response and draw on the contingency reserve right away, since both the response and the reserve were pre-approved for exactly this risk.
- **B**: Execute the response and draw on the contingency reserve, and raise a change request for the schedule-baseline revision the response forces.
- **C**: Raise a single change request covering both the contingency drawdown and the schedule-baseline change, and wait for approval before acting.
- **D**: Escalate to the sponsor for authority to use the contingency reserve before enacting the planned response.

## Answer

B

## Explanation

This question turns on a fine line between what the project manager may already do and what still needs change control. Drawing on contingency reserve to enact a pre-planned response to a known risk is within the project manager's authority and needs no fresh approval, but changing the approved schedule baseline always requires a change request regardless of how the slip arose. (B) draws that line correctly: act on the response and reserve now, and route only the baseline revision through change control. (A) gets the reserve half right but ignores that the schedule baseline cannot be quietly moved without an approved change. (C) is the most tempting distractor because bundling everything into one change request feels safe and thorough, yet it needlessly delays a response the project manager is already authorised to execute and subjects the contingency drawdown to an approval it does not require. (D) confuses contingency reserve, which the project manager controls, with management reserve, which the sponsor controls; escalating for contingency authority misreads who owns that reserve. Enacting the response while change-controlling the baseline impact is the precise move.
