# Question

What does a hybrid project management approach combine?

## Options

- **A**: Multiple agile frameworks layered together, such as Scrum with XP practices.
- **B**: Different delivery approaches for different stakeholder groups in the same project.
- **C**: Predictive documentation paired with the ceremonies of an agile framework.
- **D**: Elements of both predictive and adaptive approaches, tailored to the project context.

## Answer

D

## Explanation

PMI defines hybrid as combining predictive and adaptive elements as the project warrants. (D) is the textbook answer. (A) describes blending two agile frameworks, not hybrid. (B) is stakeholder-specific tailoring, not hybrid methodology. (C) is "agile theatre" — visible ceremonies without the discipline behind them — which is not what PMI means by hybrid.
