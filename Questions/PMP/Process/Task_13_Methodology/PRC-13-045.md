# Question

A project manager joins a media company that mandates a stage-gated predictive lifecycle for all projects through its PMO. The manager's project is a content-recommendation feature with rapidly evolving requirements that would clearly benefit from an iterative approach. What should the project manager do FIRST?

## Options

- **A**: Quietly run the project iteratively and present stage-gate documents retrospectively so the PMO sees what it expects.
- **B**: Comply fully with the mandated predictive lifecycle, since deviating from the organisational standard is not the project manager's call.
- **C**: Engage the PMO to understand the intent behind the standard and discuss tailoring it to suit the project's characteristics.
- **D**: Escalate to the sponsor to have the PMO's mandate overridden for this project.

## Answer

C

## Explanation

Tailoring is done with the governance body, not around it, and the first step is to understand why the standard exists and propose a fit-for-purpose adaptation. (C) does this and keeps the PMO as a partner. (A) deceives the governance function and erodes trust even if the delivery method is sound. (B) applies an ill-fitting method without even attempting the tailoring conversation the situation invites. (D) jumps to escalation and confrontation before any constructive discussion with the PMO has taken place, which is premature and damages the relationship the project manager has just begun to build.
