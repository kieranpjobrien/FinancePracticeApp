# Question

A PM notices that work in the organisation flows through eight handoffs before reaching production, each adding queue time. Individual teams are busy but end-to-end lead time is very long. What should the PM propose?

## Options

- **A**: Map the value stream, identify waste and queues, and redesign flow to reduce handoffs and work in process.
- **B**: Introduce tighter SLAs between teams so each handoff has a clear maximum waiting time set for incoming work.
- **C**: Give each team more capacity so they can process incoming work faster and reduce the queues that build up.
- **D**: Assign an expediter who chases priority items through the handoffs so the highest-value work gets done quickly.

## Answer

A

## Explanation

PMI's lean thinking treats system optimisation as addressing flow, not local capacity. (A) looks at the whole system. (C) adds capacity without removing waste and will only shorten queues temporarily. (B) manages symptoms inside a poorly designed flow. (D) creates a workaround for privileged work while the system stays broken.
