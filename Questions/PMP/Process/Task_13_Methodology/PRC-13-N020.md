# Question

A sponsor has read about agile and instructs the project manager to "run everything agile" on a new logistics-optimisation project. The project manager's own analysis of the requirements stability, team capability, and contractual constraints suggests a hybrid approach would deliver better outcomes. How should the project manager proceed?

## Options

- **A**: Comply with the sponsor's instruction and run the project fully agile, since the sponsor has the authority to set the approach.
- **B**: Present the analysis to the sponsor, explain how a hybrid approach meets the goals within the project's constraints, and agree the approach together.
- **C**: Quietly run the project as a hybrid as the analysis recommends, while reporting progress in agile terms to satisfy the sponsor.
- **D**: Ask the PMO to overrule the sponsor's instruction on the grounds that methodology selection is the project manager's remit, not the sponsor's.

## Answer

B

## Explanation

Approach selection should be evidence-based and agreed with the sponsor, who needs to understand the trade-offs rather than receive a label. (B) respects the sponsor's authority while bringing professional analysis to a shared decision. (A) implements a method the analysis shows is a poor fit, abdicating the PM's advisory duty. (C) deceives the sponsor and undermines trust, a clear integrity breach. (D) turns a conversation that should happen directly into an escalation battle and needlessly antagonises the sponsor before any dialogue has occurred.
