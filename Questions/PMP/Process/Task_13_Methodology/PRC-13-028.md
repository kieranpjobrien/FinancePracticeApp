# Question

A PMO oversees a portfolio of 15 projects spanning data centre infrastructure, software product development, and a major change-management initiative. What should the PMO standardise?

## Options

- **A**: A single delivery methodology to use across the portfolio so reporting is uniform.
- **B**: Methodology by project category — predictive for infrastructure, adaptive for software, hybrid for change.
- **C**: Selection criteria for delivery approach, plus common governance and portfolio reporting that all projects feed into.
- **D**: Allow each PM to pick their preferred methodology so long as they hit committed milestones.

## Answer

C

## Explanation

Portfolio governance needs common visibility, but project methodology should match the work. (C) does both — selection criteria drive fit, common governance enables roll-up. (A) forces a one-size-fits-all approach onto very different work. (B) hard-codes the choice and loses the nuance the criteria would surface. (D) optimises for PM preference and produces inconsistent governance information at portfolio level.
