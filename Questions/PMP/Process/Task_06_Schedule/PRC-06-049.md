# Question

Path A-B-C has durations 4, 6, and 5 days. Path A-D-E has durations 4, 3, and 9 days. Path A-F-G has durations 4, 7, and 3 days. All paths start at A and converge at the end of the network. What is the critical path duration and the total float on path A-D-E?

## Options

- **A**: Critical path is 16 days via A-D-E; total float on A-D-E is zero, since it is the longest path through the network.
- **B**: Critical path is 15 days via A-B-C; total float on A-D-E is 1 day, allowing that path to slip slightly without impact.
- **C**: Critical path is 14 days via A-F-G; total float on A-D-E is 2 days, because that path is two days shorter overall.
- **D**: Critical path is 16 days via A-D-E; total float on A-B-C is 1 day and total float on A-F-G is 2 days overall.

## Answer

A

## Explanation

A-B-C = 4+6+5 = 15; A-D-E = 4+3+9 = 16; A-F-G = 4+7+3 = 14. The critical path is A-D-E at 16 days with zero total float. (B) picks the wrong path. (C) picks the shortest path. (D) correctly names the critical path but asks about A-D-E float, which is zero, not about the non-critical paths.
