# Question

A predictive water-treatment build is four days behind on its critical path with a contractual completion date that cannot move. The critical path runs "pour foundation slab" then "erect steel frame", and the slab carries mandatory cure-time hold points before any load may be placed on it. The sponsor will release contingency funds if asked, and skilled steel crews are available locally. Which recovery option should the project manager assess first?

## Options

- **A**: Add a second concrete crew and a night shift to the slab pour so the slab finishes sooner.
- **B**: Overlap the activities by starting steel erection on the earliest-poured slab sections while later sections are still being placed.
- **C**: Bring forward downstream non-critical fit-out activities to run alongside the slab so the team makes visible progress.
- **D**: Re-sequence the steel erection to start before the slab cure hold points are cleared, recovering the four days directly.

## Answer

A

## Explanation

Both fast-tracking and crashing are on the table, so the deciding factor is which one the physical constraints actually permit. The slab's mandatory cure hold points govern when load may sit on it; erecting steel on a slab still curing (B) or before the hold points clear (D) violates an inherent engineering constraint and risks structural failure, so neither is a legitimate first lever here despite "overlap" normally being assessed before adding cost. (C) compresses non-critical work, which does not move the constrained finish. With funding available and the only safe compression being to shorten the slab pour itself, crashing the pour with added crew and a shift (A) is the recovery option to evaluate first.
