# Question

What is the purpose of the forward pass in the Critical Path Method?

## Options

- **A**: To identify which activities have already been completed so far in the project.
- **B**: To calculate the Early Start and Early Finish dates for each activity in the schedule network.
- **C**: To approve the schedule baseline before execution begins on the project.
- **D**: To compute the Late Start and Late Finish dates for each activity in the schedule network.

## Answer

B

## Explanation

(B) is the PMI definition: the forward pass works through the network from project start, computing Early Start and Early Finish for each activity to determine the minimum project duration. (A) describes status reporting, not CPM. (C) describes change control approval. (D) describes the backward pass, which is the complementary calculation.
