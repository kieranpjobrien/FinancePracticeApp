# Question

Halfway through a 40-week predictive project, the earned value report shows SPI steady at 0.80 across the last three reporting periods. Cost performance is on plan. The sponsor asks the project manager for a realistic finish-date view and what can be done about it. Assuming the pattern continues, what should the project manager present?

## Options

- **A**: A duration forecast extrapolated from the sustained 0.80 schedule performance, paired with recovery options such as fast-tracking or crashing the critical path.
- **B**: A duration forecast built from the critical-path schedule's own remaining-work estimate, using SPI only as a corroborating signal, alongside recovery options.
- **C**: A range of finish dates bounded by the optimistic case (the slip is recoverable) and the trend case (0.80 continues), so the sponsor sees best and likely outcomes.
- **D**: The trend-based late finish from SPI now, while commissioning a schedule risk analysis to firm up the forecast before committing to a date with the sponsor.

## Answer

A

## Explanation

Every option takes the 0.80 trend seriously — none offers the wishful "we'll catch up" answer — so the discriminator is what a competent PM presents *now*, this reporting cycle. (A) extrapolates the sustained schedule performance into a duration forecast and pairs it with concrete recovery levers, which is the direct, decision-ready response the sponsor asked for. (B) is technically the more rigorous practice — the critical-path network typically forecasts the finish better than SPI, which loses meaning as the project nears completion — but the scenario gives only EV data and a stable index, so reframing around a separate network estimate exceeds what is available and what was asked. (C) is reasonable, yet labelling the slip "recoverable" as an equally weighted bound is unsupported by three periods of flat 0.80 and risks understating a systemic problem. (D) appropriately reaches for deeper analysis, but deferring the headline forecast pending a risk analysis withholds the actionable view the sponsor needs today, when the trend is already clear enough to act on.
