# Question

A project network has these dependencies: A (3 days) -> B (5 days) -> D (4 days), and A (3 days) -> C (2 days) -> D (4 days). What is the critical path and the project duration?

## Options

- **A**: A-B-D, 12 days
- **B**: A-C-D, 9 days
- **C**: A-B-D, 9 days
- **D**: Both paths are critical at 12 days

## Answer

A

## Explanation

Path A-B-D = 3 + 5 + 4 = 12 days; Path A-C-D = 3 + 2 + 4 = 9 days. The critical path is the longest, so A-B-D at 12 days. (A) is correct. (B) names the shorter path, which is non-critical and has 3 days of float. (C) names the right path but the wrong duration. (D) is wrong because the two paths have different durations, so only one is critical.
