# Question

A network analysis shows the critical path running through three activities, all of which require the project's only licensed surveyor. After resource levelling to respect the single-surveyor constraint, the project finish slips nine days beyond the deadline. The sponsor wants the original date held. Hiring a second surveyor is possible but costly, and some non-critical activities have float. What should the project manager do first?

## Options

- **A**: Test whether resource smoothing, re-sequencing the surveyor's work within available float, can flatten the demand enough to remove the slip without extending the finish.
- **B**: Fast-track by overlapping two of the three surveyor activities so they run partly in parallel and recover the nine days.
- **C**: Crash the deadline-critical activity by hiring the second surveyor only for that activity, accepting the cost to hold the date.
- **D**: Submit a change request for the nine-day extension, since a single shared resource makes the slip unavoidable.

## Answer

A

## Explanation

After levelling forces a slip, the next move is to test resource smoothing, adjusting within float to relieve the conflict before spending money or accepting a date change. (A) exhausts the no-cost option first. (B) the three activities all need the same single surveyor, so overlapping them is physically impossible, not merely risky, fast-tracking cannot parallelise one person's work. (C) crashing with a second surveyor may be the eventual answer, but committing the cost before checking whether float can absorb the conflict skips the cheaper option. (D) accepting the extension treats the constraint as immovable when smoothing has not yet been attempted, and abandons the sponsor's firm date prematurely.
