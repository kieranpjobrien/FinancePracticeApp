# Question

A predictive project is eight days behind on the critical path and the sponsor has confirmed the end date is fixed. The critical path runs through detailed design, then a structural build that normally starts only once design is signed off. The design lead says about 60% of the design package is stable enough that the build team could begin foundations against it, with the remaining 40% still in review and liable to change. A second build crew is available at a premium, and the project carries no remaining schedule contingency. The project's cost position is already a concern for the sponsor. What should the project manager do first?

## Options

- **A**: Overlap the foundation work with the still-running design by releasing the stable 60% of the package early, accepting the rework exposure if the remaining design changes the foundations.
- **B**: Bring in the second build crew at the premium rate to run the structural build in parallel shifts and shorten its duration.
- **C**: Re-sequence the critical-path activities to shift their resource peaks within available float so the build can start sooner at no added cost.
- **D**: Reduce the design review effort on the remaining 40% so the full package signs off sooner and the build can start on the normal finish-to-start logic.

## Answer

A

## Explanation

With a fixed date, no contingency, and cost already under scrutiny, fast-tracking the genuinely stable portion of design into the build is the lowest-cost recovery available, so it is the right first move; crashing is held back for when the cheaper option proves insufficient. (A) overlaps only the part the design lead has confirmed is stable, which is how partial fast-tracking should be scoped, and accepts the residual rework risk knowingly. (B) crashing with the premium crew adds the very cost the sponsor is worried about and should follow, not precede, the no-extra-cost overlap. (C) is the trap: the critical path by definition has zero float, so there is nothing to re-sequence the critical activities within, and smoothing cannot pull the start forward here. (D) cutting review on design that is still liable to change trades schedule for engineering quality and risks defective foundations, which is a quality sacrifice rather than a legitimate compression technique.
