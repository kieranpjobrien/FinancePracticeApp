# Question

Which best describes the purpose of a RACI matrix?

## Options

- **A**: A risk assessment tool ranking each project task by likelihood, severity, and detectability.
- **B**: A reporting template that summarises status by role for distribution to stakeholders.
- **C**: A staffing model that maps team members to functional reporting lines on a matrix-managed project.
- **D**: A responsibility assignment matrix mapping deliverables or decisions to who is Responsible, Accountable, Consulted, and Informed.

## Answer

D

## Explanation

RACI is the standard responsibility assignment matrix — Responsible, Accountable, Consulted, Informed. (D) is the textbook definition. (A) describes a risk-scoring approach. (B) describes a status reporting template. (C) describes a matrix organisation structure, which is a different concept entirely.
