# Question

Which document specifies who receives which project information, when, in what format, and through which channel?

## Options

- **A**: Stakeholder register
- **B**: Project charter
- **C**: Project status report
- **D**: Communications management plan

## Answer

D

## Explanation

The communications management plan is the artefact that defines all of these — recipients, content, cadence, format, and channel. (D) is correct. (A) identifies stakeholders and their interests but doesn't prescribe how to communicate. (B) authorises the project; it doesn't define operational communication mechanics. (C) is a communication output, not the plan that defines outputs.
