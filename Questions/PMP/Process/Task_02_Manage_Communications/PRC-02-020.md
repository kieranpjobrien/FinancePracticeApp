# Question

Which statement best describes the difference between push and pull communication?

## Options

- **A**: Push communication is always verbal; pull communication is always written and archived formally.
- **B**: Push communication targets external stakeholders only; pull communication targets internal teams only.
- **C**: Push communication is informal and conversational; pull communication is formal and structured by default.
- **D**: Push communication is sent directly to recipients; pull communication is published for recipients to access on demand.

## Answer

D

## Explanation

The push/pull distinction is about how information reaches the recipient — sent or retrieved. (D) captures it correctly. (A) confuses delivery direction with format. (B) confuses delivery direction with audience. (C) confuses delivery direction with formality. Both push and pull can be verbal or written, internal or external, formal or informal.
