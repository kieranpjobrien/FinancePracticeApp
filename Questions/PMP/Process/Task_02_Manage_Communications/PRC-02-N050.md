# Question

A PM is building the communications management plan for a newly formed virtual team spread across four countries with mixed time zones and varying first languages. The PM wants the plan to genuinely support shared understanding rather than just schedule meetings. Which actions should the PM take when designing the approach? (Select THREE.)

## Options

- **A**: Confirm each stakeholder group's information needs, preferred channels, and language considerations before fixing the cadence.
- **B**: Favour asynchronous written channels for substantive decisions so members in any time zone can engage and respond on equal terms.
- **C**: Define how communication effectiveness will be measured and reviewed, so the plan can be adjusted if shared understanding proves weak.
- **D**: Set the recurring synchronous meeting at a fixed time that splits the inconvenience reasonably and require all four countries to attend it live for alignment.
- **E**: Translate every written communication into all the team's first languages before issuing it, to remove any language disadvantage.

## Answer

A, B, C

## Explanation

A sound plan for a dispersed, multilingual team starts from analysing needs and language factors (A), leans on asynchronous written communication so no time zone is disadvantaged for substantive decisions (B), and builds in a way to assess and adjust effectiveness (C). (D) is the closest wrong option: sharing the inconvenience fairly is right, but mandating live attendance for all four time zones still forces some members to make decisions at unsocial hours, which is exactly what moving substantive decisions to asynchronous channels (B) avoids. (E) sounds inclusive but blanket translation of everything is disproportionate, slows communication, and is not what equal participation requires; language considerations (A) are weighed per audience, not solved by translating every message wholesale.
