# Question

When choosing the right communication method for a particular message, which set of factors should the PM consider?

## Options

- **A**: The PM's own preference and the speed of the available delivery technology.
- **B**: Urgency, complexity, audience preference, sensitivity, and whether a documented record is needed.
- **C**: The seniority of the recipient and the size of the project budget allocated to communications.
- **D**: The geographic location of the team and the language spoken by the recipient.

## Answer

B

## Explanation

Method selection is multi-factor — sensitivity calls for secure channels, complexity for interactive ones, documentation needs for written, and so on. (B) covers the standard PMBOK set. (A) is too narrow and PM-centric. (C) treats seniority as the dominant input, which it isn't. (D) is relevant to channel mechanics but excludes the more important content-driven factors.
