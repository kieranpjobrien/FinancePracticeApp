# Question

A PM is diagnosing three recurring communication problems on a hybrid programme. Each could superficially look like another, so the corrective action must fit the actual failure mode. Match each problem (left) to the most appropriate corrective action (right).

## Items

- **1**: The fortnightly report reaches everyone on time and is accurate, yet stakeholders keep being blindsided by items it clearly contained.
- **2**: A regulator, an executive, and the delivery crews all get the identical 12-page report; it is too detailed for some and omits what others most need.
- **3**: The co-located members pick up design decisions in passing while the remote members consistently learn of them only when their work is rejected.

## Matches

- **A**: Build in confirmation of receipt and understanding to close the one-way loop.
- **B**: Segment the audience and tailor content, format, and frequency to each group.
- **C**: Capture the informal decisions to a shared channel visible to everyone regardless of location.

## Answer

1-A, 2-B, 3-C

## Explanation

Each symptom names a distinct failure that an adjacent remedy would not fix. Accurate, on-time messages that still do not register point to one-way communication with no comprehension check, so the remedy is a feedback loop (1-A), not tailoring or a radiator. A single uniform report that fits no audience is an audience-segmentation failure, resolved by tailoring (2-B), not by adding a confirmation step that would still send the wrong content. Location-based loss of informally shared decisions is an access-asymmetry failure, resolved by making those decisions visible to all (3-C), not by reformatting reports. The discrimination is that "stakeholders missing information" can stem from non-comprehension, poor targeting, or unequal access, and only matching the right cause to the right fix resolves it.
