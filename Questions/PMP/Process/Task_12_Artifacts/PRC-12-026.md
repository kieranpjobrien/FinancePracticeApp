# Question

Which artifact is created FIRST when initiating a new project?

## Options

- **A**: Work breakdown structure
- **B**: Project management plan
- **C**: Project charter
- **D**: Project schedule

## Answer

C

## Explanation

The project charter is produced in the Initiating process group, formally authorising the project and naming the PM. (A) and (D) are planning artifacts that depend on an approved charter. (B) is the consolidated output of planning, also downstream of the charter.
