# Question

A year-long ERP rollout has just closed out a difficult first phase. The phase surfaced several hard-won insights about vendor coordination and data migration that the next three phases will hit again. The team is tired and already rolling onto phase-two planning, and the next formal lessons-learned workshop is scheduled for project closure nearly a year away. What should the project manager do?

## Options

- **A**: Capture the phase's insights in the lessons-learned register now and feed the relevant ones into phase-two planning before it sets.
- **B**: Note the headline insights privately and table them for the project-closure lessons-learned workshop where they will be worked through properly with the full picture.
- **C**: Hold a full retrospective covering the whole project to date before recording anything, so the phase-one lessons are set in their broader context.
- **D**: Route the insights to the PMO so they can be added to the organisation's lessons-learned repository for future ERP programmes.

## Answer

A

## Explanation

The lessons-learned register is a living artifact, and its value here is double: capturing the insights now preserves them while accurate and lets the relevant ones improve phase-two planning before it locks in, which is where they will pay off on this very project (A). Holding them for the closure workshop (B) is the trap the scenario is built around; by then memory will have faded and, more damagingly, three phases that could have benefited will already have run, so the knowledge arrives too late to act on. Insisting on a whole-project retrospective before recording anything (C) over-engineers the moment, delays capture while the team disperses onto phase two, and risks losing the detail it is trying to contextualise. Forwarding everything to the PMO for the organisational repository (D) is worth doing for future programmes but serves other projects, not this one, and on its own does nothing to improve the phases still ahead. The fine margin is between capturing-and-applying now (A) and the several ways of deferring or redirecting the lessons (B, C, D), each of which forfeits the chance to benefit the remaining phases.
