# Question

A project manager maintains a stakeholder register that includes candid assessments of each stakeholder's attitude and influence, including notes that two executives are obstructive. A junior analyst asks for the full register so she can prepare a distribution list for the next newsletter. What should the project manager do?

## Options

- **A**: Share the full stakeholder register so the analyst has everything she needs in one place.
- **B**: Provide only the contact and communication details required for the distribution list, keeping the sensitive assessment fields restricted.
- **C**: Remove the candid assessments from the register entirely so it can be shared freely in future.
- **D**: Refuse the request and prepare the distribution list yourself to keep the register confidential.

## Answer

B

## Explanation

Artifacts often contain mixed-sensitivity information, and good information management means releasing only the fields appropriate to the recipient's need, so the analyst gets contact data without the candid assessments. (A) sharing the full register exposes confidential judgements about executives that could cause real harm if they circulate. (C) stripping the assessments destroys information the PM legitimately needs to manage engagement, sacrificing utility to solve an access problem. (D) doing the task yourself is unnecessary and fails to empower the analyst when a simple filtered extract solves it.
