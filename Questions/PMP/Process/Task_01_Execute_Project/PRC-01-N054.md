# Question

A government-services hybrid project is digitising a benefits-application process. The sponsor wants the whole end-to-end system delivered before any public release, citing reputational risk if citizens hit problems. The project manager believes an incremental release, starting with the most common application type, would deliver citizen value far sooner and surface real usability issues early. Both positions are reasonable and the two are in tension. What should the project manager do?

## Options

- **A**: Bring the sponsor a phased release plan that quantifies the earlier value, sets out the risk controls and rollback for a limited first increment, and seek agreement to release the highest-volume application type first.
- **B**: Propose a closed pilot of the highest-volume application type with a small invited cohort under a non-public banner, so real usage data is gathered before committing to any phased public release.
- **C**: Accept the sponsor's position and deliver the entire system before any release, since the sponsor carries the reputational accountability and has stated a clear preference.
- **D**: Build the system so that incremental release remains technically possible, but defer the release-strategy decision until the full system is closer to ready and the risks are clearer.

## Answer

A

## Explanation

Delivering value with urgency means advocating for the earlier release, but the sponsor's reputational accountability must be respected, so the project manager brings evidence, risk controls and a rollback path and seeks agreement on a controlled first increment (A). (B) is a credible step and a good way to de-risk, but a closed non-public pilot delays actual citizen value and, on its own, concedes the early public release rather than making the case for it. (C) abandons the early-value opportunity without testing whether the sponsor's concern can be met with controls, defaulting to the slowest path. (D) keeps the option open technically but postpones a foundational decision that should shape execution now, eroding the very urgency at stake.
