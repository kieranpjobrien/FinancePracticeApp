# Question

On a predictive construction fit-out, after six months the PM reports BAC $2,000,000, EV $600,000, AC $750,000, PV $800,000. The sponsor wants the original finish date held. Assuming the current CPI continues, what is the estimated cost at completion (EAC)?

## Options

- **A**: $2,000,000 — the original budget holds if corrective action is taken.
- **B**: $2,500,000 — computed as BAC divided by the current cost performance index.
- **C**: $2,666,667 — computed using BAC over SPI to reflect schedule slippage impact.
- **D**: $2,150,000 — computed as AC plus remaining BAC minus the earned value already.

## Answer

B

## Explanation

EAC using CPI is BAC/CPI = 2,000,000 / (600,000/750,000) = 2,000,000 / 0.8 = 2,500,000. (B) applies the standard typical-variances formula correctly. (A) ignores the cost overrun already incurred. (C) substitutes SPI for CPI, a common formula confusion. (D) is the atypical-variances formula (AC + BAC − EV) which assumes the overrun was one-off — not what the question specifies.
