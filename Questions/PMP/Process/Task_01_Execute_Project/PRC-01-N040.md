# Question

A project manager is steering a hybrid pharmaceutical supply-chain programme and must keep value flowing while respecting governance. Match each pressure to the most appropriate first response.

## Items

- **1**: A sponsor pushes to expedite one deliverable, which would delay another and change the agreed plan.
- **2**: A finished, validated increment is ready, but acceptance is scheduled weeks out only because the reviewer is unavailable.
- **3**: A team is accruing technical debt to hit a fixed regulatory date, and the cost falls on future releases.

## Matches

- **A**: Propose an earlier remote or delegated acceptance so benefit is realised sooner without bypassing governance.
- **B**: Assess the impact and route the request through change control so the trade-off is owned and documented.
- **C**: Make the cost explicit to the value owners, quantify it against the regulatory risk, and log the debt for repayment.

## Answer

1-B, 2-A, 3-C

## Explanation

Each first response delivers value with urgency while keeping decisions visible and owned. An expedite that alters the plan must be assessed and authorised through change control, not actioned unilaterally (1-B). An avoidable acceptance delay should be compressed via a legitimate faster route rather than tolerated or bypassed (2-A). A real technical-debt trade-off belongs to the accountable stakeholders, made explicit and logged for repayment (3-C). Swapping any pairing would either bypass governance or let a significant trade-off be made implicitly by the project manager.
