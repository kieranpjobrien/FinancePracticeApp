# Question

A predictive rail-signalling upgrade spans twelve stations and is mid-execution, with the team currently progressing all twelve in parallel. Funding for the second half of the programme is under review and a decision is several months away. A signalling upgrade only delivers operational benefit once a station is fully commissioned. The sponsor asks the project manager how to steer the work given the funding uncertainty. What should the project manager recommend?

## Options

- **A**: Continue advancing all twelve stations in parallel to maximise total progress, so that whatever the funding decision, the programme is as far along as possible everywhere.
- **B**: Resequence to fully commission stations cluster by cluster, prioritising the highest-benefit stations, so completed clusters are operational and benefit is banked before any decision lands.
- **C**: Bring forward only the stations whose works are cheapest and quickest to commission, completing the largest number of stations before funding is decided.
- **D**: Hold deep progress on the second-half stations and concentrate the crew on design and procurement across all twelve, so the programme can restart fast whichever way funding goes.

## Answer

B

## Explanation

Benefit accrues only at full commissioning and continuation is uncertain, so the move that secures value is to finish whole stations in priority order rather than spread effort that may strand everything half-done. (B) banks operational benefit cluster by cluster, weighting the most valuable stations first, so realised value survives a funding cut. (A) maximises generic progress but leaves all twelve short of commissioning, risking zero realised benefit if funding stops. (C) does complete stations early, which is far better than parallelism, but ordering by ease rather than benefit can bank the least valuable stations and leave high-value ones unfinished. (D) keeps options open for a restart yet deliberately defers the very commissioning that turns work into benefit, so it realises nothing while the decision is pending.
