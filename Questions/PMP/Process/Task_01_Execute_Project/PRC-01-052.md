# Question

A PM estimating a critical activity collects optimistic 8 days, most-likely 14 days, pessimistic 26 days. What is the PERT expected duration and the standard deviation for this activity?

## Options

- **A**: Expected 14 days, standard deviation 3 days, using the simple three-point average formula.
- **B**: Expected 15 days, standard deviation 3 days, using beta distribution weighting on most-likely.
- **C**: Expected 16 days, standard deviation 4.5 days, derived from adding optimistic and pessimistic only.
- **D**: Expected 12 days, standard deviation 2.5 days, using triangular distribution with equal weights.

## Answer

B

## Explanation

PERT beta: (O + 4M + P)/6 = (8 + 56 + 26)/6 = 90/6 = 15 days; SD = (P − O)/6 = 18/6 = 3 days. (B) applies both formulas correctly. (A) uses the simple three-point average (O+M+P)/3, which is the triangular method, not PERT. (C) and (D) apply incorrect weightings that produce common distractor values candidates pick when confusing the two distributions.
