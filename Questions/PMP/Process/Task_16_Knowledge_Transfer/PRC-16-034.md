# Question

What is the main benefit of conducting knowledge transfer continuously throughout a project, rather than only at closure?

## Options

- **A**: Information is captured while memory is fresh, the running project can apply early learnings, and capture stays sustainable.
- **B**: It allows the project sponsor to closely monitor the team's reflection activities at every major milestone across the project.
- **C**: It guarantees that the lessons-learned register will be longer than projects that only capture at closure and therefore more valuable.
- **D**: It produces a calendar of recurring meetings that helps with the overall project communications cadence and stakeholder touchpoints.

## Answer

A

## Explanation

(A) is the standard set of benefits: freshness, in-flight applicability, and sustainability. (B) reframes the practice as oversight rather than learning. (C) treats register length as the goal, which it is not. (D) confuses a side-effect (additional ceremonies) with the substantive benefit.
