# Question

Why should lessons learned capture both successes and failures?

## Options

- **A**: To produce a balanced report that protects team morale by avoiding an exclusive focus on what went wrong.
- **B**: Because external auditors typically require both categories to be evidenced for project closure.
- **C**: Because successes provide patterns to replicate while failures provide warnings to avoid; both improve future projects.
- **D**: Because the PMO will not accept a closure pack that contains only one category.

## Answer

C

## Explanation

(C) is the substantive learning answer: each category contributes something distinct. (A) reframes lessons as morale management. (B) overstates audit drivers. (D) treats it as a procedural compliance issue rather than a learning one.
