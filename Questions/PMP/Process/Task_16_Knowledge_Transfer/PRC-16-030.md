# Question

Which best describes "knowledge management" in a project context?

## Options

- **A**: The systematic process of creating, sharing, applying, and maintaining knowledge across the project and the wider organisation.
- **B**: The activity of storing project documents in a controlled folder structure with version control.
- **C**: The training programme that brings new project members up to a baseline level of competence.
- **D**: The function that decides who has read or write access to project information assets.

## Answer

A

## Explanation

(A) is the standard PMI definition: an end-to-end approach to knowledge as an organisational asset. (B) is one downstream activity (document storage). (C) is one transfer mechanism (training). (D) is access control, which is a sub-component.
