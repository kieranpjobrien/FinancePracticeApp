# Question

A predictive project is two-thirds complete. Several of the risks the contingency reserve was sized for have not occurred and their windows have now passed, leaving the reserve substantially under-consumed. A senior stakeholder asks the project manager to release the unused contingency to fund an attractive scope enhancement. What should the project manager do?

## Options

- **A**: Conduct a reserve analysis to confirm which risks have genuinely passed, then take any reserve release for the new scope through integrated change control.
- **B**: Release the surplus contingency directly to the scope enhancement, since the risks it covered did not materialise.
- **C**: Refuse to touch the contingency and return all of it to the organisation only at project closure.
- **D**: Move the surplus contingency into management reserve so it is available for the enhancement without a change request.

## Answer

A

## Explanation

The disciplined response is a reserve analysis to verify which risks have truly expired versus those still open, and then to route any reuse of freed funds for new scope through integrated change control. (B) diverting contingency to new scope without confirming the remaining risk exposure or running change control is unsanctioned and may leave live risks unfunded. (C) blanket refusal until closure ignores that genuinely freed reserve can be legitimately reconsidered through proper governance. (D) reclassifying contingency as management reserve to dodge change control is a governance workaround, not a control action.
