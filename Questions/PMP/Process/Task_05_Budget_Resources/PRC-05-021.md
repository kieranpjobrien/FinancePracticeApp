# Question

Which term describes adjusting a resource-constrained schedule so resource usage stays within defined limits, even if this extends duration?

## Options

- **A**: Resource smoothing.
- **B**: Fast tracking.
- **C**: Crashing.
- **D**: Resource levelling.

## Answer

D

## Explanation

Resource levelling adjusts activities based on resource availability, often extending duration to avoid exceeding resource limits. (A) resource smoothing adjusts within existing float so the end date isn't extended. (B) fast tracking runs activities in parallel to compress schedule. (C) crashing adds resources to reduce duration at extra cost.
