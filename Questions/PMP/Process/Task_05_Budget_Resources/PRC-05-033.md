# Question

What is resource smoothing?

## Options

- **A**: A scheduling technique that adjusts activities within float so resource demand stays within preset limits without changing the end date.
- **B**: A scheduling technique that delays activities beyond their late finish dates to keep resource usage flat across the project.
- **C**: A budgeting technique that averages monthly resource costs over the project to produce a flat spend profile.
- **D**: A staffing technique that swaps high-cost resources for lower-cost resources during peak demand periods.

## Answer

A

## Explanation

(A) is the PMI definition: resource smoothing uses available float to remove resource peaks without affecting the critical path or finish date. (B) describes resource levelling, which can extend the schedule because it is allowed to push activities past their late finish. (C) is a financial profiling concept, not a schedule technique. (D) describes resource substitution, which is unrelated to smoothing.
