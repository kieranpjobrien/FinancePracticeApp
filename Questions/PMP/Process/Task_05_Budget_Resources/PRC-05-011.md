# Question

Which estimation technique uses the three-point formula (optimistic + pessimistic + 4*most likely) / 6?

## Options

- **A**: Analogous estimating.
- **B**: Beta (PERT) estimating.
- **C**: Parametric estimating.
- **D**: Bottom-up estimating.

## Answer

B

## Explanation

The formula (O + 4M + P) / 6 is the beta distribution weighted average used in PERT. (A) analogous uses past similar projects as a reference point. (C) parametric uses unit rates multiplied by quantity. (D) bottom-up estimates each work package and rolls up; no three-point formula is implied.
