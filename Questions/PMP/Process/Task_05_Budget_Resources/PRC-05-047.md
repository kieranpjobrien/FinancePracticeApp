# Question

On a predictive ERP rollout, two functional managers are both claiming the same senior integration engineer for critical work in the same fortnight. The engineer is on the project's resource calendar for your sprint. Both managers outrank the project manager. What should you do first?

## Options

- **A**: Escalate immediately to the sponsor so the organisation's leadership can decide which project receives the engineer's time.
- **B**: Accept the other manager's priority for harmony and replan your work using a less experienced engineer from the bench.
- **C**: Meet both managers with the resource calendar and project priorities to negotiate a shared plan that protects committed dates.
- **D**: Direct the engineer to remain on your project since the resource calendar formally assigns them for this period.

## Answer

C

## Explanation

PMI expects the PM to negotiate and collaborate first, armed with documented assignments and priority data, before escalating. (C) is the correct first step. (A) is premature escalation. (B) surrenders a committed resource without negotiation. (D) is authoritarian and ignores the functional managers' legitimate claims in a matrix structure.
