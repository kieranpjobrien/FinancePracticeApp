# Question

A public-works project has BAC = $10,000,000, EV = $5,000,000, AC = $6,250,000, PV = $6,000,000. The PM wants an EAC that assumes future performance reflects both past cost and schedule performance. What value should the PM report?

## Options

- **A**: $11,250,000, calculated as AC plus BAC minus EV, reflecting only the existing cost overrun incurred to date.
- **B**: $13,750,000, calculated as AC plus BAC minus EV divided by the product of CPI and SPI going forward.
- **C**: $12,500,000, calculated as BAC divided by the cumulative cost performance index alone at this point.
- **D**: $10,416,000, calculated as BAC divided by the schedule performance index without any cost consideration.

## Answer

B

## Explanation

When both cost and schedule performance are assumed to continue, EAC = AC + (BAC - EV) / (CPI × SPI). CPI = 5,000/6,250 = 0.80, SPI = 5,000/6,000 = 0.833, product = 0.667. EAC = 6,250,000 + 5,000,000 / 0.667 ≈ 6,250,000 + 7,500,000 = $13,750,000. (A) applies atypical EAC. (C) uses CPI only. (D) uses SPI only, a non-standard formula for cost EAC.
