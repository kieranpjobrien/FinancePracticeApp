# Question

A project manager has activity-level cost estimates. During execution the sponsor wants to be able to see, the moment a variance appears, which single manager owns the slipping work and to track each manager's performance against scope, schedule, and cost together rather than as separate reports. Several managers each own a branch of the WBS. How should the project manager structure the budget to support this?

## Options

- **A**: Roll activities into work packages and group them into control accounts aligned to each accountable manager's WBS branch, then measure and report performance at the control-account level.
- **B**: Roll activities into work packages and measure and report performance at the work-package level, since the work package is the lowest point where scope, schedule, and cost already meet.
- **C**: Aggregate the budget into a time-phased S-curve and identify the owning manager from the activities active in the period a variance arises.
- **D**: Build a responsibility assignment matrix linking managers to deliverables, and report cost at the total-project level against it.

## Answer

A

## Explanation

The control account is the point in the WBS where a single manager is accountable and where scope, schedule, and cost performance are measured together, so aligning control accounts to each manager's WBS branch lets a variance be traced straight to its owner — exactly the sponsor's need (A). (B) is the strongest distractor: work packages also bring the three dimensions together, but a manager typically owns many of them, so reporting at that level scatters performance across numerous packages instead of consolidating it to the one accountable owner the control account gives. (C) an S-curve shows when money is spent, not who owns it; inferring ownership from whichever activities are active is indirect and breaks down when several managers' work overlaps in a period. (D) a responsibility matrix maps who is accountable but carries no cost measurement, and total-project reporting cannot localise a variance to a manager.
