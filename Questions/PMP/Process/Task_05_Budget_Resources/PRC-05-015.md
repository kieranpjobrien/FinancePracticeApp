# Question

A project has BAC = $400,000, EV = $200,000, AC = $240,000. What is the To-Complete Performance Index (TCPI) based on BAC?

## Options

- **A**: 1.10
- **B**: 1.25
- **C**: 1.20
- **D**: 0.83

## Answer

B

## Explanation

TCPI (based on BAC) = (BAC - EV) / (BAC - AC) = (400,000 - 200,000) / (400,000 - 240,000) = 200,000 / 160,000 = 1.25. This means remaining work must be performed at 125% of planned efficiency to stay within budget. (A) 1.10 doesn't match. (C) 1.20 doesn't match. (D) 0.83 is the inverted ratio.
