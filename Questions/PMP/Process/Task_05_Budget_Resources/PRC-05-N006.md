# Question

A project manager is preparing the cost estimate for a mid-sized building retrofit. The design is well advanced for the structural scope but still conceptual for the services fit-out, and the sponsor wants a figure that will support funding approval now and cost control later. Which actions best fit this situation? (Select THREE.)

## Options

- **A**: Document the basis of estimate, recording assumptions, exclusions, and the confidence range for each major component.
- **B**: Use bottom-up estimating for the well-defined structural work and analogous or parametric estimating for the still-conceptual services scope.
- **C**: Derive contingency from the quantified risk analysis rather than a single uniform percentage across all line items.
- **D**: Apply bottom-up estimating uniformly to every component so the whole estimate is built at the same level of detail.
- **E**: Reconcile the structural figure against the awarded subcontractor tender and the services figure against published unit-cost data.

## Answer

A, B, C

## Explanation

The estimate should document its basis (A), match the estimating method to how well each part is defined — bottom-up where the design is firm, analogous/parametric where it is still conceptual (B) — and derive contingency from the analysed risks rather than a blanket percentage (C). (D) forcing bottom-up onto conceptual services demands detail the design cannot yet support, producing false precision. (E) is plausible but mixes the benchmarks: the services scope is not yet at tender, so reconciling it against a tender is not possible, and a single benchmark source is weaker than method-appropriate validation — it is the least-right of the five.
