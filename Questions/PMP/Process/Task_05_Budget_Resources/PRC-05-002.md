# Question

A project has BAC = $200,000, AC = $80,000, EV = $60,000. Assuming current cost performance continues, what is the Estimate at Completion (EAC)?

## Options

- **A**: $240,000
- **B**: $266,667
- **C**: $220,000
- **D**: $280,000

## Answer

B

## Explanation

When current performance is expected to continue, EAC = BAC / CPI. CPI = 60,000 / 80,000 = 0.75. EAC = 200,000 / 0.75 = $266,667. (A) $240,000 = AC + (BAC - EV), which assumes remaining work at budgeted rates (different scenario). (C) $220,000 = AC + BAC - EV - some other variant. (D) $280,000 doesn't match a standard EAC formula for this scenario.
