# Question

Using beta (PERT) with O=8, M=12, P=22, what is the expected duration?

## Options

- **A**: 12 days
- **B**: 13 days
- **C**: 13.0 days (rounded)
- **D**: 14 days

## Answer

C

## Explanation

Expected duration = (O + 4M + P) / 6 = (8 + 48 + 22) / 6 = 78 / 6 = 13.0 days. (A) 12 is simply the most likely, ignoring optimistic and pessimistic. (B) 13 rounds the same answer but C is the precisely stated form. (D) 14 overweights the pessimistic.
