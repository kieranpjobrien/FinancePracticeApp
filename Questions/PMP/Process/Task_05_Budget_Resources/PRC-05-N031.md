# Question

At a monthly review of a predictive project, the cost engineer reports that the schedule performance index has recovered to 1.0 this period, but the cumulative cost performance index remains at 0.85 and has been flat for three periods. The overspend driver — a labour rate higher than budgeted — is contractual and will persist to the end. The sponsor asks the project manager for the most realistic statement about the project's cost outcome. Which response is best?

## Options

- **A**: The project is forecast to complete over budget; the estimate at completion should be revised using the cumulative cost efficiency, because the rate overrun is systemic and ongoing.
- **B**: With the schedule back on track, the cost performance should recover over the remaining periods, so the original budget at completion remains the best forecast.
- **C**: Because the cost index has been flat rather than worsening, the cost position is now stable and no change to the budget at completion is warranted.
- **D**: The estimate at completion should be revised using the combined schedule-and-cost index, since both indices are now driving the remaining forecast.

## Answer

A

## Explanation

A persistent, contractual rate overrun is systemic, so the remaining work will carry the same inefficiency: EAC = BAC / CPI gives the realistic over-budget forecast, and the recovered SPI does not undo a cost gap. (B) confuses schedule recovery with cost recovery; the cost driver is contractual and fixed, so it will not "recover", and assuming the original BAC ignores a real ongoing overrun. (C) a flat CPI means the overrun is steady, not absent — stable inefficiency still compounds across the remaining work and still pushes EAC above BAC. (D) the SPI×CPI combined index is for when both schedule pressure and cost inefficiency jointly drive the remainder, but schedule has recovered to 1.0, so only the cost index should shape the forecast.
