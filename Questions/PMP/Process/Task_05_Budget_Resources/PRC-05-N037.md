# Question

A hybrid programme runs an agile product stream funded incrementally and reviewed each quarter, and a predictive infrastructure stream funded against a cost baseline with formal change control, under one programme budget. Midway, the agile stream is delivering strong value and the infrastructure stream is forecasting an overrun. The project manager wants to manage the combined budget responsibly while respecting each stream's funding model. Which two actions are appropriate? (Choose two.)

## Options

- **A**: At the agile stream's quarterly gate, confirm delivered value against cost before authorising release of the next funding increment.
- **B**: Forecast the infrastructure overrun, identify its drivers, and bring corrective options or a change request to the body that governs the baseline.
- **C**: Propose to the programme governance body that some of the agile stream's later, lower-value increments be deferred and the freed budget be made available to the infrastructure stream.
- **D**: Re-baseline the infrastructure stream now to the forecast overrun so the cost baseline matches the latest estimate and variance returns to zero.
- **E**: Apply the agile stream's quarterly value-gate review to the infrastructure stream as the primary control, in place of its change-control process.

## Answer

A, B

## Explanation

Each stream keeps its own funding model: the agile increment is released only after the quarterly gate confirms value (A), and the infrastructure overrun is forecast, explained, and taken to its governing body through change control or corrective options (B). (C) reallocating funds between streams is a legitimate idea, but proposing to cut value-delivering agile increments to subsidise an infrastructure overrun pre-empts the corrective analysis in (B) and trades away value rather than first addressing the overrun's drivers — it is not the responsible action here. (D) re-baselining to the overrun erases the variance instead of managing it and bypasses the change control that governs the baseline. (E) replacing the infrastructure stream's change control with a value gate strips out the control its predictive funding model depends on.
