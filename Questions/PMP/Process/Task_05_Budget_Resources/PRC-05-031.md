# Question

Your monthly performance report shows CPI = 1.05 and SPI = 1.02, yet the business sponsor and end users report dissatisfaction with the features released so far. What does this most likely indicate?

## Options

- **A**: Stakeholders need education on earned value so they understand the project is performing well.
- **B**: The cost and schedule data must be inaccurate and should be re-baselined to match perceived reality.
- **C**: Earned value measures execution efficiency, not value delivery, so additional outcome metrics are needed.
- **D**: The project is on track and the sponsor's concerns can be deferred to a later steering committee.

## Answer

C

## Explanation

EVM tells you whether planned work is being completed on time and within budget — it does not tell you whether the right work is being done. (C) recognises that good CPI/SPI with poor stakeholder satisfaction signals a value problem requiring outcome measures (NPS, adoption, defect rates, benefit realisation). (A) blames stakeholders rather than examining a real signal. (B) discards valid metrics instead of complementing them. (D) ignores stakeholder feedback, violating PMI's stakeholder engagement principles.
