# Question

A predictive project includes one work package that uses a fabrication technique the organisation has never attempted, sitting alongside many packages closely resembling past projects. A vendor who has done similar first-of-a-kind work has offered a fixed-price quotation for the novel package, and one of the organisation's senior engineers has hands-on experience with the underlying process from a previous employer. The sponsor wants the overall estimate quickly, the novel package to be credible, and its uncertainty made visible to the steering group. How should the project manager best estimate the novel work package?

## Options

- **A**: Adopt the vendor's fixed-price quotation as the package estimate, since a supplier experienced in this first-of-a-kind work has already priced it.
- **B**: Decompose the package into its component activities and build a bottom-up estimate using the senior engineer's expert judgement, expressing each activity as a three-point range to surface the uncertainty.
- **C**: Apply a parametric estimate by scaling the familiar packages' established unit rates up by a complexity factor that reflects the novel technique.
- **D**: Build the estimate bottom-up from component activities using the engineer's judgement, and present it as a single most-likely figure to keep the baseline clean.

## Answer

B

## Explanation

First-of-a-kind work has no comparable history, so the credible estimate is built bottom-up from the actual activities using available expert judgement; expressing each activity three-point (optimistic/most-likely/pessimistic) is what makes the uncertainty visible to the steering group, which the sponsor explicitly asked for. (D) is the strongest distractor — it uses the same correct bottom-up method and the same expert, but collapsing genuinely uncertain novel work to a single point hides the very uncertainty the sponsor wants surfaced, so it loses to B on that one requirement. (A) a vendor quotation is a useful input and a cross-check, but adopting one external price as the estimate outsources the project's own analysis and inherits the vendor's margin and assumptions sight-unseen; it informs the estimate rather than constituting it. (C) parametric rates drawn from familiar work do not legitimately transfer to a fundamentally different technique, and a subjective "complexity factor" dresses up that broken comparison as a calculation.
