# Question

A project has BAC = $800,000. Current status: EV = $300,000, AC = $360,000. Management insists the project still finish within the original budget. The project manager calculates the To-Complete Performance Index against the BAC. What does the result tell the sponsor?

## Options

- **A**: TCPI ≈ 1.14, meaning the remaining work must run about 14% more efficiently than budgeted — a stretch from the current CPI of 0.83 that should be flagged as probably unrealistic.
- **B**: TCPI ≈ 0.88, taking (BAC − AC) / (BAC − EV) and reporting that remaining work can run below budgeted efficiency and still hit the target.
- **C**: TCPI ≈ 0.83, calculated against the estimate at completion instead of the BAC, suggesting the budget needs no efficiency gain.
- **D**: TCPI ≈ 1.00, meaning current performance, if sustained, lands the project exactly on budget.

## Answer

A

## Explanation

TCPI to BAC = (BAC − EV) / (BAC − AC) = (800,000 − 300,000) / (800,000 − 360,000) = 500,000 / 440,000 ≈ 1.14. Since current CPI is only 300,000 / 360,000 ≈ 0.83, requiring 1.14 going forward is a large efficiency jump that should be flagged as probably unrealistic. (B) inverts the numerator and denominator, giving 440,000 / 500,000 ≈ 0.88 — a value below 1.0 wrongly implies the team can ease off and still finish on budget. (C) targets the EAC (= BAC / CPI ≈ 960,000) instead of the BAC, giving 500,000 / (960,000 − 360,000) ≈ 0.83; that is TCPI to the forecast, not to the original budget the sponsor is asking about. (D) only TCPI = 1.0 would mean no change is needed, but at CPI 0.83 the project is already behind, so staying on budget demands better-than-budgeted efficiency, not the same.
