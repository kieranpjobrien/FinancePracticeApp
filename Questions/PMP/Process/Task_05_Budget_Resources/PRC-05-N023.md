# Question

A construction project's time-phased cost baseline is built and approved. Treasury then confirms that funds will be released in fixed quarterly tranches, and in two quarters the planned spend exceeds the tranche available. Total approved funding for the project is unchanged and adequate, and several work packages in those quarters have float. What should the project manager do first?

## Options

- **A**: Re-time the work in the affected quarters, within available float where possible, so planned spend in each period no longer exceeds the released tranche.
- **B**: Submit a change request to re-phase the funding tranches so the cash released each quarter matches the existing expenditure curve.
- **C**: Negotiate payment terms with suppliers so invoices in the two quarters fall due in later periods when funding is available.
- **D**: Draw on contingency reserve in the two quarters to bridge the gap between planned spend and the released tranche.

## Answer

A

## Explanation

The mismatch is between the project's own expenditure curve and the periodic funding limits; the project controls the curve, so the disciplined first move is funding limit reconciliation — re-time the work (using float first) so planned spend fits each tranche, since total funding is adequate and only the phasing is wrong. (B) re-phasing the tranches asks the funder to change a confirmed constraint to fit the plan, when the plan can be adjusted instead; it may be a fallback but is not the first move. (C) deferring supplier invoices manipulates payment timing rather than the work that drives the cost, and can attract penalty terms or strain relationships. (D) contingency is reserved for identified risks, not for absorbing a self-inflicted cash-phasing mismatch that re-timing resolves at no cost.
