# Question

In a weak matrix organisation, the senior database engineer the project manager scheduled for a critical migration is pulled by her functional manager onto urgent operational work for the same two weeks. The migration is on the critical path. The project manager has already met the functional manager and proposed a part-time split and a one-week shift; the functional manager maintains the operational incident must take the engineer full-time and will not release her in the window. A capable contractor could be onboarded but would need about a week to come up to speed on the environment. The resource assignments and the operational priority were both agreed under the same programme. What should the project manager do next?

## Options

- **A**: Continue negotiating with the functional manager, offering further scheduling concessions until an arrangement that frees the engineer is reached.
- **B**: Begin onboarding the contractor now so a qualified resource is ready, treating the engineer as unavailable for this window.
- **C**: Escalate to the programme sponsor with the schedule impact and the two competing priorities, so the resource conflict is resolved at the level that owns both.
- **D**: Re-sequence the migration to start after the two-week operational window using the engineer's float, accepting the knock-on to the critical path.

## Answer

C

## Explanation

Direct negotiation has already been tried and has reached a genuine deadlock between two priorities — the critical migration and an urgent operational incident — that sit under the same programme; that is precisely the situation escalation exists for, taking the conflict to the authority that owns both so it can be resolved on its merits (C). (A) negotiation is the correct *first* move and was made, but continuing to offer concessions after a flat refusal, with no new lever, spends time the critical path does not have and concedes the project's position rather than surfacing the trade-off. (B) onboarding the contractor is a reasonable parallel hedge, but committing to it unilaterally accepts cost and a week of ramp-up to work around a conflict the sponsor might resolve outright, and pre-empts the decision the programme owns. (D) the migration is on the critical path, so it has no float to absorb a two-week slip; "using float" that does not exist simply books a delay to the end date as if it were free.
