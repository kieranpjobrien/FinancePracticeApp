# Question

A predictive five-year research-facility programme is using rolling wave planning: year one is defined to work-package level and later years are held at a coarse, high-level scope to be elaborated as outcomes emerge. The sponsor is comfortable with this but wants confidence that, once approved, the programme baseline will not be eroded every time a later wave is detailed. A governance reviewer asks the project manager how the later, coarsely defined work should be controlled so that the baseline stays meaningful. What should the project manager put in place?

## Options

- **A**: Detail each later wave through the change control process as it is elaborated, so the added scope detail is reviewed and the baseline is updated under control at each wave.
- **B**: Set the later years' high-level scope as firmly fixed now, so that elaborating a wave can only fill in detail and can never alter the approved scope.
- **C**: Hold the later years outside the baseline entirely and bring each wave into the baseline for the first time only when it is fully detailed.
- **D**: Allow the team to elaborate and begin each later wave as information emerges, recording the detail in the baseline afterwards so progress is not held up.

## Answer

A

## Explanation

Rolling wave keeps later work coarse on purpose, so the control question is how elaboration enters the baseline without uncontrolled drift. Routing each wave's added detail through change control as it is elaborated keeps the baseline authoritative and reviewed at every step while still honouring progressive elaboration (A). (B) freezing the later high-level scope as firmly fixed denies the genuine uncertainty rolling wave exists to handle and will force constant change battles when reality diverges from a premature commitment. (C) keeping later years entirely outside the baseline leaves the programme with no approved high-level scope to control against until late, which is the opposite of giving the sponsor a meaningful baseline. (D) elaborating and starting work first and recording it afterwards is precisely the uncontrolled-change pattern that erodes the baseline the reviewer is worried about. The margin is between bringing elaborated detail in under control and three ways of either over-fixing, under-baselining, or bypassing control.
