# Question

Which statement most accurately distinguishes product scope from project scope?

## Options

- **A**: Product scope is owned by the customer while project scope is owned by the project manager throughout delivery.
- **B**: Product scope describes the features of the result; project scope describes the work needed to produce that result.
- **C**: Product scope is fixed once approved while project scope can be freely adjusted during execution as needed.
- **D**: Product scope is documented in detail while project scope is intentionally kept high-level throughout execution.

## Answer

B

## Explanation

Product scope = "what we're building"; project scope = "what we're doing to build it." (A) misallocates ownership; both involve many stakeholders. (C) is wrong because both are subject to change control. (D) is wrong; both can be detailed or rolling-wave depending on the lifecycle.
