# Question

The scope baseline is composed of which of the following?

## Options

- **A**: Project charter, stakeholder register, and requirements documentation.
- **B**: Project scope statement, WBS, and WBS dictionary.
- **C**: Requirements traceability matrix, scope statement, and project schedule.
- **D**: WBS, cost baseline, and schedule baseline.

## Answer

B

## Explanation

PMI defines the scope baseline as exactly three components: scope statement, WBS, and WBS dictionary. (A) lists scope inputs, not the baseline. (C) mixes a scope artefact with the schedule. (D) confuses the scope baseline with the broader performance measurement baseline.
