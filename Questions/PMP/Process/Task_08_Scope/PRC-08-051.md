# Question

During a regulatory audit on a pharmaceuticals project, the auditor asks how each implemented control maps back to the regulatory requirements. The PM finds the traceability matrix has not been kept current through the last three change cycles. What should the PM do FIRST?

## Options

- **A**: Update the requirements traceability matrix to reflect the current state of implemented controls.
- **B**: Ask the auditor for additional time before answering so the team can reconstruct traceability evidence.
- **C**: Commit to improving traceability practices on the next project and lessons learned capture the gap.
- **D**: Explain to the auditor that traceability has been maintained informally through team channels.

## Answer

A

## Explanation

A current requirements traceability matrix is a core scope artifact and a compliance obligation; bringing it current is the immediate corrective action. (B) stalls the audit without fixing the underlying gap. (C) defers the fix to a future project while leaving the current compliance exposure. (D) misrepresents the state of controls to an auditor, which is an ethics violation.
