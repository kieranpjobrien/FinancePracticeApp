# Question

A new sustainability policy requires every project above a threshold to report ESG metrics quarterly to a new sustainability committee. The PM's project was approved before the policy and has no baseline metrics. What should the PM do?

## Options

- **A**: Argue the project should be exempt since it was approved before the policy was published for new projects going forward.
- **B**: Start reporting immediately using whatever data exists and mark the gaps as known limitations to be resolved later.
- **C**: Engage the sustainability committee to agree a proportionate baseline and reporting scope and build it into governance.
- **D**: Wait until a baseline can be formally established before reporting anything, to avoid sending misleading data to the committee.

## Answer

C

## Explanation

PMI expects new compliance obligations to be engaged constructively with the control owner to define proportionate expectations. (C) does that. (A) seeks exemption rather than engagement. (B) reports with known gaps without agreeing with the committee how to treat them. (D) delays reporting indefinitely and ignores a live obligation.
