# Question

In a weak matrix, functional managers pull team members onto competing work without telling the PM. Delivery has slipped twice and the PM has no formal authority over resourcing decisions. What should the PM do FIRST?

## Options

- **A**: Raise each individual pull with the functional manager after it happens and request the person be returned urgently.
- **B**: Escalate the pattern directly to the sponsor and ask for formal authority over resources for the remaining project duration.
- **C**: Work with the sponsor and functional managers to document resource commitments and a conflict-resolution process up front.
- **D**: Keep a log of each interruption and present cumulative delivery impact to the steerco at the next scheduled meeting.

## Answer

C

## Explanation

PMI's governance guidance addresses matrix friction by agreeing resource governance and conflict-resolution before conflicts happen. (C) puts the structure in place. (A) manages incidents one at a time without fixing the system. (B) asks for a matrix reshape rather than agreeing governance rules. (D) documents harm instead of preventing it.
