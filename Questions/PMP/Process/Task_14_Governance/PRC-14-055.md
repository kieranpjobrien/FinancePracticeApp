# Question

A project manager on a predictive water-treatment upgrade is agreeing the governance arrangements with the sponsor. On the last project the sponsor was either dragged into trivial cost and schedule fluctuations the project manager could have handled, or learned of a serious overrun only after it had grown large. The sponsor asks the project manager to set up the arrangement so this does not recur. Which governance element should the project manager put in place to address both halves of the sponsor's concern?

## Options

- **A**: Defined cost and schedule tolerances within which the project manager manages alone, with escalation required once a tolerance is crossed.
- **B**: A reporting cadence under which the project manager sends the sponsor a status report at a fixed, agreed interval.
- **C**: A communications plan that sets out which stakeholders receive which information and how often.
- **D**: An issue and escalation log in which variances are recorded as they arise and reviewed at the next governance meeting.

## Answer

A

## Explanation

The sponsor is describing two failures — being pulled into minor matters, and hearing about serious ones too late — and tolerances are the single mechanism that addresses both. By bounding the project manager's delegated authority, agreed cost and schedule tolerances let the project manager absorb routine fluctuation without troubling the sponsor, while a breach is the defined trigger that surfaces a serious variance promptly. Each distractor handles only one half, or neither. (B) a fixed reporting interval keeps the sponsor informed but neither grants authority to act on small matters nor guarantees timely warning, since a serious overrun can erupt between reports. (C) governs who receives information, not how much variance the project manager may absorb before the sponsor must be involved. (D) records and reviews variances after the fact at the next meeting, which neither empowers independent action on the minor ones nor raises the serious ones at the moment they cross the line.
