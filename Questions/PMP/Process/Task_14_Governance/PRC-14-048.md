# Question

A project manager on a predictive aerospace component project has just stood up a steering committee. At its first meeting the senior members, keen to be useful, propose that the committee should add value by reviewing each engineering deliverable in technical detail, working through the delivery task board, and approving individual work assignments. The project manager wants to steer the committee toward the contribution it is actually constituted to make. Which use of the committee's time should the project manager encourage?

## Options

- **A**: Setting strategic direction, deciding matters that exceed the project manager's delegated authority, and clearing organisational obstacles the project manager cannot remove alone.
- **B**: Working through the delivery task board with the team each cycle and approving how individual work is assigned.
- **C**: Conducting detailed technical reviews of each engineering deliverable before it is released.
- **D**: Owning the minutes, status reports, and the project's documentation set so the record is authoritative.

## Answer

A

## Explanation

A steering committee is a governance body, and the project manager should direct its limited senior time to oversight and direction-setting: strategic guidance, decisions beyond the project manager's delegated authority, and removal of organisational impediments the project manager cannot clear alone. Each distractor is a genuine and necessary activity — which is what makes them tempting — but none is the committee's job. (B) is delivery management: a committee that worked the task board and assigned individual work would be doing the project manager's and team's job, not governing. (C) is technical assurance, which belongs with subject-matter experts or a technical review board; senior governance members reviewing every deliverable in detail both misuses their role and creates a bottleneck. (D) is administrative support; the committee may receive and rely on the record, but owning the minutes and documentation is not the reason the body exists.
