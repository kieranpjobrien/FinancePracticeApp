# Question

Which technique uses iterative "why" questioning to uncover the root cause of a project issue?

## Options

- **A**: Pareto analysis
- **B**: Five Whys
- **C**: Monte Carlo simulation
- **D**: Decision tree analysis

## Answer

B

## Explanation

The Five Whys is the classic root cause technique: ask "why" repeatedly until you reach a fundamental cause rather than a symptom. (B) is the textbook answer. (A) ranks problem sources by frequency or impact. (C) is a quantitative risk analysis technique. (D) is a structured decision-making technique.
