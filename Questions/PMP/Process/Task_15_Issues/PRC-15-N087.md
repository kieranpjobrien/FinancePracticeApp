# Question

A Scrum Master on a telecommunications product maintains a visible impediment board. The team has surfaced a blocker none of its members can clear: a shared integration server owned by a separate operations group is unavailable, and operations has not answered two requests from the team. The Scrum Master wants the board to actively drive this specific, stalled, cross-team item to closure. Which THREE practices most directly advance *this* impediment toward resolution? (Select THREE.)

## Options

- **A**: Assign a single named owner on the board accountable for chasing the server issue to closure.
- **B**: Escalate to the operations group's management now that the team's own two requests have gone unanswered.
- **C**: Record the impediment's age on the board so the stall becomes conspicuous and forces the next action.
- **D**: Capture the blocker in the impediment description with detail on which stories it is holding up.
- **E**: Raise the server outage as a discussion item in the upcoming retrospective so the team can agree a standing process.

## Answer

A, B, C

## Explanation

All five are reasonable board hygiene; the question asks which three most directly move *this* stalled, cross-team blocker forward. A named owner accountable for closure (A) stops it sitting ownerless; visible ageing (C) makes the stall impossible to ignore and triggers the next step; and escalation to operations' management (B) supplies the authority the team has already exhausted its own channel trying to reach — the live discriminator here. Documenting which stories it blocks (D) is genuinely good practice and strengthens the case for escalation, but on its own it only describes the impediment more fully; it does not itself drive operations to act, so it is the strongest distractor rather than a wrong statement. Taking the outage to the retrospective (E) is appropriate for designing a *future* process, but it defers action on a blocker that is stopping work now, when impediments should be removed as they arise. The set that actually advances closure is own it, age it, escalate it — A, B, C.
