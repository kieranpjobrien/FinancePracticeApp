# Question

A project manager on a predictive wind-farm construction project is sorting four newly raised items so each is routed to the correct process. Match each item to the artefact or process where it primarily belongs.

## Items

- **1**: A turbine component delivered today fails inspection and cannot be installed, stopping the affected erection crew.
- **2**: The client asks to add two extra met-masts not in the approved scope.
- **3**: A forecast suggests an unusually severe storm season next winter that might disrupt offshore lifts.
- **4**: A subcontractor's installed bolt torque does not meet the specified value and must be corrected.

## Matches

- **A**: Risk register.
- **B**: Issue log.
- **C**: Change control process.
- **D**: Defect log / quality records.

## Answer

1-B, 2-C, 3-A, 4-D

## Explanation

The distinguishing cues are whether the event has occurred, whether it alters scope, and whether it concerns conformance of delivered work. (1) is a present problem stopping work — a classic issue for the issue log (B). (2) is a request to add scope beyond the baseline, which must run through change control rather than be worked as a problem (C). (3) is a conditional future event that may or may not happen, the signature of a risk for the risk register (A). (4) is delivered work that fails to meet specification, a quality non-conformance recorded as a defect (D). The traps are mistaking (2)'s addition for an issue or (4)'s torque shortfall for a generic issue rather than a conformance defect, and treating (3)'s "might" as if it had already materialised.
