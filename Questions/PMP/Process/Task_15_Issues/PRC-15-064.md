# Question

On a predictive energy project, the PM notices the issue log contains inconsistent severity labels, missing owners, and vague descriptions. Stakeholders are starting to question report accuracy at the steering committee. What should the project manager do first?

## Options

- **A**: Rewrite the most problematic entries personally overnight so the next steering committee report looks clean and defensible to reviewers.
- **B**: Establish clear entry standards, coach the team on applying them, and remediate existing entries through a prioritised data-quality pass.
- **C**: Replace the current issue log tooling with a structured system that enforces mandatory fields before the next reporting cycle.
- **D**: Stop reporting issue data to the steering committee until full remediation has been completed across every existing log entry.

## Answer

B

## Explanation

PMI treats artefact quality as a team capability built through standards and coaching, not a one-off cleanup. (B) fixes both the current data and the upstream behaviour. (A) produces a cosmetic fix without changing how data is captured. (C) replaces tooling before addressing practice. (D) withholds governance data instead of improving it.
