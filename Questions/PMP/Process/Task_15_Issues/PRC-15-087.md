# Question

A project manager on a hybrid telecoms-network rollout is reviewing four open issues, each at a different point. For each, identify the next action that does the most to move it forward.

## Items

- **1**: A high-impact issue with a named owner and target date has stalled for two weeks; the owner has repeatedly asked a partner organisation's manager to approve access the fix needs, and been ignored.
- **2**: A field engineer mentions in passing that a faulty test handset has been slowing site commissioning for several days; it is not on the issue log and no one is tracking it.
- **3**: A blocked vendor portal login that halted billing configuration has been fixed by IT, the analyst confirms configuration is now complete, and the issue still shows "in progress" with two follow-up actions open.
- **4**: A cell-handover defect logged, fixed and closed last quarter has reappeared at another site, and the same fix has just been re-applied and the new instance closed.

## Matches

- **A**: Log it with an owner, priority and target date so it can be tracked and worked.
- **B**: Escalate it to a governance body that can authorise the decision the owner cannot obtain.
- **C**: Confirm the agreed outcome is met, clear the open actions, and close it.
- **D**: Investigate the common cause across both occurrences before relying on the latest fix.

## Answer

1-B, 2-A, 3-C, 4-D

## Explanation

Item 1 is already logged and owned; the blocker is an approval the owner has chased in vain and the PM cannot compel, so escalation to a body that can authorise it is the move with leverage (B). Item 2 is a live problem nobody is tracking, so it must first be logged and owned before it can be managed at all (A). Item 3 is resolved in substance but left open with stale actions, so the value is in verifying the outcome, clearing the actions and closing it (C). Item 4 has been "fixed and closed" twice, so re-applying the same patch treats the symptom; the action with leverage is investigating why it recurs before trusting the latest fix (D). The discrimination is in each issue's state, not its category — several could superficially read as "just log it" or "just close it", but the action that actually advances each one differs.
