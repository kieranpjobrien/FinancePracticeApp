# Question

A PM closing out the planning phase of a predictive rail infrastructure project has signed off schedule, cost, and scope baselines but has not defined a performance measurement baseline for earned value tracking. The sponsor expects EVM reports from month one. What should the PM do?

## Options

- **A**: Start execution now and build the performance measurement baseline during the first reporting cycle.
- **B**: Use cost baseline alone for earned value, as sophisticated PMB detail is unnecessary for most projects.
- **C**: Integrate scope, schedule, and cost into a formal performance measurement baseline before execution begins.
- **D**: Report execution status narratively until enough actuals exist to compute earned value meaningfully.

## Answer

C

## Explanation

A performance measurement baseline integrates scope, schedule, and cost into a measurable reference against which EVM is calculated; it must exist before execution starts. (A) defers baseline establishment into execution, undermining early variance detection. (B) conflates cost baseline with PMB and misses the scope and schedule integration. (D) waives EVM reporting the sponsor is expecting.
