# Question

In an agile delivery, what set of artefacts and agreements collectively serves the function of the project management plan?

## Options

- **A**: The product backlog alone, since it lists the work to be done.
- **B**: The current sprint backlog, since it commits the team for the iteration.
- **C**: There is no project management plan in agile because plans emerge during execution.
- **D**: The release plan, team working agreements, Definition of Done/Ready, and the cadence of ceremonies, taken together.

## Answer

D

## Explanation

Agile projects absolutely have integrated planning artefacts; they are just lighter weight and distributed. The release plan provides direction; team agreements and DoD/DoR define how work is done; ceremonies enforce the cadence. (A) and (B) are scope/work artefacts only. (C) is a misconception about agile.
