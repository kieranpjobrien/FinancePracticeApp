# Question

What is the primary role of the project manager in integration management?

## Options

- **A**: Ensuring plans, processes, and people work together as a coherent whole across the project.
- **B**: Personally writing each subsidiary management plan across all knowledge areas.
- **C**: Personally approving every change request raised on the project by any stakeholder.
- **D**: Producing all stakeholder-facing status communications and reporting updates directly.

## Answer

A

## Explanation

Integration is the unique, non-delegable PM responsibility: stitching the parts into a working whole. (B) is delegable to discipline leads. (C) is one input into integration but is usually a CCB function, not solely the PM's. (D) is communications management, distinct from integration.
