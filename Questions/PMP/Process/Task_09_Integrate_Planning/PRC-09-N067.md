# Question

A project manager is assembling the integrated project management plan for a renewable-energy retrofit and is mapping each planning conflict to the subsidiary plan that must drive its resolution. Match each integration conflict (left) to the subsidiary plan whose content should govern how it is reconciled (right).

## Items

- **1**: The agreed contingency reserves for identified risks are not represented in the time-phased budget.
- **2**: Two work packages that share a single specialist crew are scheduled for the same week.
- **3**: A signed customer requirement does not trace to any deliverable in the WBS.

## Matches

- **A**: Scope management plan
- **B**: Cost management plan
- **C**: Resource management plan

## Answer

1-B, 2-C, 3-A

## Explanation

Integration routes each conflict to the plan that owns the relevant baseline. (1) Contingency for identified risks is incorporated into the cost baseline, so the cost management plan governs how the reserve is sized and embedded (B). (2) A double-booked specialist crew is a resource over-allocation, reconciled through resource levelling or smoothing under the resource management plan (C). (3) A requirement with no corresponding deliverable is a scope completeness gap, resolved by updating the scope baseline and WBS under the scope management plan (A). Correctly assigning ownership prevents the PM from patching a symptom in the wrong plan and leaving the underlying baseline inconsistent.
