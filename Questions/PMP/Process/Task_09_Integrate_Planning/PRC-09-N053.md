# Question

A project manager is briefing a new team member on how the integrated project management plan fits together. Match each planning component on the left with the description that best identifies it.

## Items

- **1**: Scope baseline
- **2**: Schedule management plan
- **3**: Configuration management plan
- **4**: Performance measurement baseline

## Matches

- **A**: Establishes how project and product information versions are identified, controlled, and kept consistent across the team
- **B**: The integrated scope, schedule, and cost baselines used together to measure and manage execution performance
- **C**: The approved scope statement, WBS, and WBS dictionary against which scope is controlled
- **D**: Defines how the schedule will be developed, monitored, and controlled, including tools and reserve policy

## Answer

1-C, 2-D, 3-A, 4-B

## Explanation

The scope baseline (1) is the approved scope statement, WBS, and WBS dictionary (C). The schedule management plan (2) sets out how the schedule is developed and controlled (D). The configuration management plan (3) governs version identification and control of information (A). The performance measurement baseline (4) integrates the scope, schedule, and cost baselines for measuring execution (B). Distinguishing a baseline (what you measure against) from a management plan (how you do the work) is central to understanding how the integrated plan is assembled.
