# Question

A predictive clinical trial programme uses stage-gate funding. The PM has integrated scope, schedule, and cost for the full programme but the sponsor has only approved funding for the first stage. The team wants to plan all stages in detail now to save time later. What should the PM do?

## Options

- **A**: Plan all stages in detail now, since integrated planning is more efficient when completed in a single effort.
- **B**: Plan stage one in detail and use rolling wave for later stages, consistent with stage-gate funding reality.
- **C**: Plan only stage one and ignore later stages until each stage's funding approval is actually obtained.
- **D**: Ask the sponsor to release funding for all stages so integrated planning can be completed in one pass.

## Answer

B

## Explanation

Stage-gate governance aligns with rolling wave planning; integration is complete at the level of current authorisation and progressively elaborated as funding is confirmed. (A) ignores funding reality and produces detailed plans likely to be rewritten. (C) forgoes forward integration that supports sequencing decisions today. (D) tries to bend governance to planning preference rather than the other way around.
