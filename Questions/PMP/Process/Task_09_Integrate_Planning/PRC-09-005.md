# Question

What is the primary output of the Develop Project Management Plan process?

## Options

- **A**: The project charter.
- **B**: The work breakdown structure.
- **C**: The project schedule.
- **D**: The project management plan.

## Answer

D

## Explanation

The process is named for its output: the integrated project management plan, which contains all subsidiary plans and baselines. (A) is an input — the charter authorises the project. (B) and (C) are outputs of other planning processes (Create WBS, Develop Schedule) that become components inside the project management plan.
