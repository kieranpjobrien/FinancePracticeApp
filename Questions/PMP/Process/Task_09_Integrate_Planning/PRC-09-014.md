# Question

Which of the following is a subsidiary plan integrated into the project management plan?

## Options

- **A**: The project charter.
- **B**: The stakeholder register.
- **C**: The risk management plan.
- **D**: The work breakdown structure.

## Answer

C

## Explanation

Subsidiary plans (scope, schedule, cost, quality, resource, communications, risk, procurement, stakeholder engagement) are the building blocks of the integrated project management plan. (A) is an authorising input, not a subsidiary plan. (B) is a project document. (D) is part of the scope baseline but is not itself a subsidiary plan.
