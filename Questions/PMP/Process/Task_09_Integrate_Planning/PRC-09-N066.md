# Question

A project manager is performing a final consistency review across the subsidiary plans before establishing the integrated project management plan baseline for an airport baggage-handling upgrade. Which of the following findings represent genuine integration defects that must be reconciled before baselining? (Choose THREE.)

## Options

- **A**: The schedule shows commissioning starting in week 30, but the resource plan releases the commissioning engineers at the end of week 28.
- **B**: The cost baseline's time-phased spend curve peaks in a quarter where the schedule shows the lowest planned activity volume.
- **C**: The scope baseline includes a conveyor diversion feature that does not appear as a deliverable anywhere in the WBS or the schedule.
- **D**: The risk register rates two threats as high probability, while the issue log currently contains no open issues.
- **E**: The communications plan specifies weekly status reports, whereas the stakeholder engagement plan describes the sponsor as preferring monthly summaries.

## Answer

A, B, C

## Explanation

Integration defects are contradictions between plans that make the integrated baseline internally infeasible or inaccurate. (A) is a resource-versus-schedule conflict — the people needed for commissioning are released before commissioning starts, so the schedule cannot execute as drawn. (B) is a cost-versus-schedule inconsistency — spend cannot legitimately peak when almost no work is scheduled, signalling the cost baseline was not derived from the schedule. (C) is a scope-versus-WBS/schedule gap — a scoped feature with no corresponding deliverable or activity means it is unplanned and unfunded. (D) is not a defect: risks are future and uncertain while issues are current and realised, so an empty issue log is fully consistent with high-probability risks. (E) is a real misalignment but it sits within the communications/engagement design and is resolved by tailoring reporting cadence per stakeholder; it does not invalidate the integrated baseline the way A, B and C do.
