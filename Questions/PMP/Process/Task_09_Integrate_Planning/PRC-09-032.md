# Question

A complex R&D project has high uncertainty about later phases. Some stakeholders want detailed multi-year plans; others say detail is wasted given the unknowns. What is the best integrated planning approach?

## Options

- **A**: Produce detailed multi-year plans to satisfy the stakeholders who request that level of detail.
- **B**: Keep all plans high-level until execution reveals enough information to plan in detail later.
- **C**: Adopt rolling-wave planning — detailed near-term, lighter longer-term plans elaborated as uncertainty resolves.
- **D**: Provide each stakeholder group with the plan detail level that they happen to prefer.

## Answer

C

## Explanation

Rolling wave is PMI's standard answer to this exact tension: detail where it is meaningful, summary where it isn't. (A) creates plausible but unreliable far-term plans. (B) leaves near-term decisions under-supported. (D) produces multiple inconsistent plans that cannot be integrated.
