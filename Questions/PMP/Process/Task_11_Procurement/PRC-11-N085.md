# Question

During the solicitation period for a large public-sector infrastructure tender, one prospective bidder emails the project manager a detailed technical question. Answering it would resolve a genuine ambiguity in a requirement and would materially improve any bid submitted against it. The bid deadline is in ten days, and the published solicitation timetable includes a single scheduled cut-off for written queries that falls in four days, after which the buyer issues consolidated written responses to all bidders. The project manager intends to make the clarification available to every bidder. How should the project manager handle this question?

## Options

- **A**: Reply to the asking bidder now so they are not held up, and also include the question and answer in the consolidated written responses issued to all bidders after the query cut-off.
- **B**: Hold the question and answer it through the consolidated written responses issued to all bidders after the scheduled query cut-off, rather than responding to the bidder ahead of that point.
- **C**: Issue the clarification immediately as a written addendum to every bidder, and assess whether the change to the requirement warrants extending the bid deadline.
- **D**: Phone each shortlisted bidder in turn to explain the clarification verbally, so every bidder receives the same information directly and at the same level of detail.

## Answer

C

## Explanation

The clarification changes how bidders should interpret a requirement, so the buyer should put it in front of every bidder promptly and in a form that becomes part of the solicitation record — a written addendum — and then judge whether a substantive change to the requirement leaves bidders enough time, extending the deadline if not (C). Answering the asking bidder now and only later folding the response into the consolidated replies (A) gives that one firm an early-information advantage during the intervening days, which is exactly the asymmetry equitable treatment is meant to prevent, even though the information eventually reaches everyone. Waiting for the scheduled query cut-off (B) is defensible process and is the trap, because using the published mechanism feels correct — but it knowingly leaves a material ambiguity live for several more days and may compress the time bidders have to react to a changed requirement, when the buyer could act now. Phoning each bidder verbally (D) does reach everyone, but verbal clarifications on a public tender are inconsistent in wording, leave no auditable record, and are poor practice for a defensible procurement. The discriminator is not whether to share with all bidders — every option does — but choosing the channel and timing that keep the tender both fair and auditable.
