# Question

A project manager needs an external specialist to investigate and repair intermittent faults in a manufacturing control system. The scope cannot be defined in advance because nobody yet knows how deep the faults run, so the work will be paid on a time-and-materials basis. Finance is concerned about an open-ended bill. Which addition to the time-and-materials arrangement best addresses that concern while still allowing the diagnostic work to proceed?

## Options

- **A**: Convert the engagement to firm fixed price by estimating a likely repair effort and fixing a price against it.
- **B**: Cap the hourly rate the specialist may charge so the cost per hour cannot rise during the work.
- **C**: Add a not-to-exceed ceiling to the time-and-materials contract, beyond which work pauses for the buyer's authorisation.
- **D**: Require the specialist to submit a detailed fixed quotation before any diagnostic work begins.

## Answer

C

## Explanation

A not-to-exceed (cost ceiling) on a time-and-materials contract directly answers the open-ended-bill worry by capping total exposure and pausing work for buyer authorisation if the ceiling is approached, while still letting the undefined diagnostic work proceed on actuals (C). Converting to firm fixed price (A) is unworkable because the scope genuinely cannot be defined yet, so any fixed price would be guesswork that either gouges the buyer or exposes the seller. Capping the hourly rate (B) controls the price per hour but not the number of hours, so the total bill can still run away, which is the actual concern. Demanding a detailed fixed quotation before work begins (D) again presupposes a definable scope the situation does not have and would stall the very investigation needed to understand the faults. The fine margin is that A, B, and D each try to pin down something that cannot yet be pinned down or control the wrong variable, whereas only the not-to-exceed ceiling caps total spend without requiring scope certainty.
