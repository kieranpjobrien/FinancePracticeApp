# Question

The bidder conference for a large infrastructure tender has finished and written addenda have been issued. Two days before bids close, one prospective seller phones the project manager with a question whose answer would materially change how the work is priced; the seller asks for a quick verbal reply so it can finalise its bid in time. The other bidders are not on the call. What should the project manager do?

## Options

- **A**: Give the seller the same answer that is already in the issued addenda and decline to go beyond what every bidder has, so no one gains an advantage.
- **B**: Provide the verbal answer now and follow it the same day with a written addendum to all bidders so the record is consistent.
- **C**: Issue the answer as a written addendum to every prospective bidder, and extend the bid deadline if the late clarification leaves bidders too little time to respond.
- **D**: Take the question, answer no one yet, and rule the point out of scope because clarifications closed when the bidder conference ended.

## Answer

C

## Explanation

The answer materially affects pricing, so every bidder must receive it in the same written form, and because it arrives close to the deadline the buyer should also consider extending the close so the new information can actually be acted on (C). Simply repeating the existing addenda (A) withholds information the buyer now knows is material and pricing-relevant, leaving a known ambiguity in the competition. Giving the verbal answer first (B) hands that one seller the information ahead of the others even by a few hours, which is the advantage the process exists to prevent; the written addendum must lead, not trail. Refusing on the basis that clarifications "closed" with the conference (D) elevates process over fairness and lets a material pricing question go unanswered for everyone. Equal and timely access to material information governs the response.
