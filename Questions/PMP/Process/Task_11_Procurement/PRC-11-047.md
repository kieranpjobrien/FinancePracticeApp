# Question

Under a cost plus incentive fee contract for a predictive utility upgrade, the vendor is on track to earn the maximum incentive fee but has cut corners on documentation quality to do so. The documentation is a contractual deliverable. What should the PM do?

## Options

- **A**: Award the full incentive because the vendor is meeting the cost and schedule incentive parameters as required.
- **B**: Deduct the documentation shortfall from the incentive calculation based on contract criteria and inspection results.
- **C**: Accept weaker documentation now and request improvements on the next contract with this particular vendor later.
- **D**: Withhold the incentive entirely because any contractual shortfall disqualifies a vendor from incentive payments.

## Answer

B

## Explanation

CPIF incentive calculations follow the contract's specified performance measures; documentation shortfalls that are contractual deliverables must be reflected in the fee per the contract's criteria. (A) ignores contractual quality criteria. (C) accepts weaker performance and defers the consequence indefinitely. (D) overreacts without reference to the contract's actual incentive formula.
