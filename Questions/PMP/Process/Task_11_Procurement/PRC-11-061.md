# Question

A PM reviewing a draft vendor contract for a custom software build notices the default template is silent on intellectual property ownership. The organisation plans to reuse the code across multiple future projects. The vendor has already countersigned the template. What should the PM do?

## Options

- **A**: Proceed with the current signed contract and address IP terms at the start of each future reuse project.
- **B**: Rely on local copyright law defaults, which typically assign IP to the commissioning party in service contracts.
- **C**: Add IP reuse clauses by email addendum, since the vendor is already on board and will likely agree quickly.
- **D**: Negotiate a formal contract amendment specifying IP ownership, reuse rights, and any payment implications.

## Answer

D

## Explanation

IP terms are a material commercial matter; formal amendment is the proper mechanism and protects reuse rights against later dispute. (A) defers a legal issue that grows harder over time. (B) relies on jurisdiction-specific defaults that may not cover the planned reuse. (C) treats a material term informally, creating enforceability risk.
