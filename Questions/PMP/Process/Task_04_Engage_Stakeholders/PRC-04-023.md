# Question

Which is an input to the Identify Stakeholders process?

## Options

- **A**: Stakeholder register.
- **B**: Project charter.
- **C**: Stakeholder engagement plan.
- **D**: Issue log.

## Answer

B

## Explanation

The project charter names key stakeholders and feeds Identify Stakeholders. (A) the stakeholder register is the output of the process, not an input. (C) the stakeholder engagement plan depends on the register and is built after. (D) the issue log is an execution artefact, produced after stakeholders are identified and engagement begins.
