# Question

A PM is preparing for a mid-project steering committee. One recurring issue is that meetings run long because every topic is discussed in full detail. What should the PM change?

## Options

- **A**: Send a pre-read two days in advance and reserve the meeting for decisions that need the group.
- **B**: Shorten the meeting time slot so participants are forced to be brief on every topic.
- **C**: Reduce the number of topics by dropping items that haven't had material status changes this period.
- **D**: Invite fewer stakeholders so there are fewer voices in the room and less discussion time required.

## Answer

A

## Explanation

The issue is information density, not people. (A) uses asynchronous pre-reads so the meeting focuses on what genuinely requires group time. (B) squeezing the slot without changing the format produces rushed decisions. (C) dropping topics may skip important items that lack visible status change but still need steering. (D) cutting participants damages engagement when the real problem is meeting design.
