# Question

What is a key objective of the project closure phase?

## Options

- **A**: To plan the next project so the team can move directly into new work.
- **B**: To formally complete project activities, obtain acceptance, transition ownership, and capture knowledge for the organisation.
- **C**: To identify the team members responsible for any failures so performance can be addressed.
- **D**: To spend remaining contingency reserves before they revert to the funding pool.

## Answer

B

## Explanation

PMI defines closure objectives around completion, acceptance, transition, and organisational learning. (B) captures all four. (A) confuses portfolio activity with the closure of the current project. (C) is contrary to PMI's blameless lessons learned philosophy and is not a closure objective. (D) is improper financial behaviour; unspent reserves should return to the funding pool, not be consumed.
