# Question

A two-year predictive satellite-ground-station project is fourteen months in and 55% complete when the regulator withdraws the spectrum licence the system was designed around, with no route to reinstatement. The sponsor asks the project manager for a recommendation on how to proceed. What should the project manager advise?

## Options

- **A**: Recommend terminating the project now and following the early-closure process to capture status, salvage any reusable assets, and release resources.
- **B**: Recommend re-scoping the project to deliver only the subsystems already built as a standalone partial system that could show some return on the spend so far.
- **C**: Recommend pausing delivery and tasking the team to investigate whether the design can be adapted to a different licensed frequency band before deciding.
- **D**: Recommend completing the remaining 45% so the asset is finished and available should the regulatory position change in future.

## Answer

A

## Explanation

A project continues only while its business case holds; once the justification is removed, prior spend is sunk and the decision must be made on forward value alone. (A) is correct because the licence withdrawal leaves the planned solution unusable with no reinstatement route, so an orderly termination that preserves status, assets and knowledge is the responsible recommendation. (B) spends more to package components that have no viable end use under the withdrawn licence, chasing a return that cannot be realised. (C) commits further effort to a speculative redesign that is a different undertaking from the authorised project and is not supported by any current business case. (D) finishes an asset that cannot be operated, betting additional budget on a regulatory reversal that has been ruled out.
