# Question

Which of the following is a responsibility of Project Support?

## Options

- **A**: Approving exceptions raised by the Project Manager
- **B**: Providing administrative services, tools and guidance, such as maintaining records and the project's information
- **C**: Representing the supplier interest on the project board
- **D**: Owning the business case throughout the project

## Answer

B

## Explanation

Project Support provides administrative help to the Project Manager and project board, including configuration and information management, scheduling support and guidance on PRINCE2 tools and techniques. (A) is a project board responsibility; (C) is the Senior Supplier's role; (D) is the Executive's responsibility.
