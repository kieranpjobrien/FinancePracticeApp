# Question

Which PRINCE2 issue type describes a product that does not meet, or is forecast not to meet, its agreed specification?

## Options

- **A**: Request for change
- **B**: Problem or concern
- **C**: Off-specification
- **D**: Exception

## Answer

C

## Explanation

An off-specification is a product (or one forecast) that does not meet its agreed specification, including a missing product. (A) is a proposal to change an agreed baseline, not a shortfall against it. (B) is any other issue the project manager needs to resolve. (D) describes a situation forecast to exceed tolerances, which is a control concept rather than an issue type.
