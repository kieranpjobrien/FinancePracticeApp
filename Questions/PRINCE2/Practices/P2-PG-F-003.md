# Question

Which management product does the project manager use to provide the project board with a summary of stage progress at a frequency defined by the board?

## Options

- **A**: Exception report
- **B**: Highlight report
- **C**: End stage report
- **D**: Lessons report

## Answer

B

## Explanation

The highlight report is a time-driven control that gives the project board a regular summary of stage status at the cadence the board specifies. (A) an exception report is event-driven, raised only when a stage or project is forecast to exceed tolerance; (C) the end stage report is produced once at the end of a stage; (D) a lessons report communicates lessons rather than routine progress.
