# Question

On a bank's regulatory-reporting project, the executive asks the project manager to also act as project assurance to save cost, and separately the firm's enterprise standards office wants to verify the project complies with corporate methods. How should the project manager respond?

## Options

- **A**: Decline the assurance role because project assurance must be independent of the project manager, and recognise the standards office as providing quality assurance from outside the project
- **B**: Accept both roles personally, as the project manager is best placed to judge the project's quality
- **C**: Treat the standards office's review as project assurance and ask them to also cover the board's assurance duties
- **D**: Combine project assurance and quality assurance into a single activity led by the standards office to avoid duplication

## Answer

A

## Explanation

Project assurance must remain independent of the project manager and belongs to the project board, so the project manager cannot hold it, while the enterprise standards office sits outside the project and is performing quality assurance. (B) breaches the independence of project assurance; (C) conflates an external quality assurance function with the board's own project assurance; (D) wrongly merges two distinct mechanisms that differ in both ownership and source of independence.
