# Question

A project board is reviewing the project plan for a city-wide cycle-lane programme and asks why it does not contain the detailed week-by-week activities for the work starting next month. What is the BEST explanation for the project manager to give?

## Options

- **A**: The project plan should indeed contain that detail and was prepared incorrectly
- **B**: Such detail is only ever held in team plans, never in any plan the board sees
- **C**: The detailed near-term activities belong in the stage plan; the project plan deliberately remains high level across the whole project
- **D**: The detail will be added to the project plan once the board sets the stage tolerances

## Answer

C

## Explanation

The project plan is intentionally a high-level whole-project plan, while the detailed activities for imminent work are planned in the stage plan for the relevant management stage. (A) the project plan is correct in staying high level. (B) detailed near-term work is held in the stage plan, which the board approves, not only in optional team plans. (D) project-level detail is not deferred pending tolerances; the appropriate detail simply sits at stage level.
