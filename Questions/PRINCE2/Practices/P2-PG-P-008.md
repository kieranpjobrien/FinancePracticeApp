# Question

On a charity fundraising-platform project, the project manager notices a minor concern that needs no formal action but should be recorded so it is not forgotten. The stage remains comfortably within all tolerances. What is the BEST way to handle this under the Progress practice?

## Options

- **A**: Record it in the daily log and monitor it as part of ongoing progress control
- **B**: Raise an exception report so the project board is aware of the concern
- **C**: Open a formal entry in the risk register and convene the project board
- **D**: Add it to the next highlight report as a forecast tolerance breach

## Answer

A

## Explanation

Minor informal concerns that require no formal management product are captured in the daily log, an informal record the project manager keeps to support day-to-day control. (B) and (C) escalate a trivial matter and invoke the board unnecessarily; (D) mislabels a non-breach as a tolerance breach, which would misinform the board and distort the reporting.
