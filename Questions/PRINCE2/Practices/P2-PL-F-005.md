# Question

When is an exception plan produced?

## Options

- **A**: At the start of every management stage as a matter of routine
- **B**: Whenever a team manager needs to plan the work of a work package
- **C**: When the project board wishes to close the project prematurely
- **D**: When a stage or the project is forecast to exceed its agreed tolerances

## Answer

D

## Explanation

An exception plan is prepared to show the actions required to recover from or avoid a forecast deviation when a stage plan or project plan is predicted to breach its tolerances. (A) routine stage plans, not exception plans, are produced at the start of each stage. (B) describes a team plan. (C) premature closure is handled through the closing a project process, not an exception plan.
