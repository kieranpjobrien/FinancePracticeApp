# Question

Which statement BEST describes the "accept" threat response?

## Options

- **A**: The project takes no active measures, choosing to tolerate the threat should it occur
- **B**: A third party is paid to bear the financial impact of the threat
- **C**: Actions are planned in advance and held ready to use only if the threat materialises
- **D**: The project is changed so that the threat can no longer occur

## Answer

A

## Explanation

"Accept" means making a conscious and informed decision to tolerate a threat, taking no proactive action because, for example, response costs would outweigh the benefit. (B) describes transfer, (C) describes preparing contingent plans, and (D) describes avoid. Acceptance is still a deliberate choice recorded in the risk register, not simply ignoring the risk.
