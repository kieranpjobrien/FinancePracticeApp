# Question

What is the correct sequence of steps in the PRINCE2 product-based planning technique?

## Options

- **A**: Write the project product description, create the product breakdown structure, write the product descriptions, create the product flow diagram
- **B**: Create the product breakdown structure, write the project product description, write the product descriptions, create the product flow diagram
- **C**: Write the product descriptions, create the product flow diagram, create the product breakdown structure, write the project product description
- **D**: Create the product flow diagram, create the product breakdown structure, write the project product description, write the product descriptions

## Answer

A

## Explanation

Product-based planning starts by writing the project product description for the overall output, then creating the product breakdown structure, then writing product descriptions for the identified products, and finally creating the product flow diagram showing their sequence. (B), (C) and (D) all place these steps out of order; the project product description must come first and the flow diagram last.
