# Question

Who approves an exception plan for a management stage that is forecast to exceed its stage tolerance?

## Options

- **A**: The team manager
- **B**: The project manager alone
- **C**: The project board
- **D**: Project assurance

## Answer

C

## Explanation

When a stage is forecast to breach its tolerance, the project manager produces an exception plan, but it is the project board that reviews and approves it, since tolerance breaches are escalated above the project manager. (A) the team manager works within work packages. (B) the project manager prepares but cannot approve the plan that recovers a tolerance breach. (D) project assurance advises but does not approve plans.
