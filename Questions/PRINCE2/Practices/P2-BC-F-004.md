# Question

During which PRINCE2 process is the outline business case first produced?

## Options

- **A**: Starting Up a Project
- **B**: Initiating a Project
- **C**: Directing a Project
- **D**: Closing a Project

## Answer

A

## Explanation

The outline business case is created during Starting Up a Project to give the project board enough justification to decide whether initiation is worthwhile. (B) Initiating a Project develops the outline into the detailed business case; (C) Directing a Project approves it but does not produce it; (D) Closing a Project confirms benefits, it does not create the case. The outline keeps early effort proportionate before the project is fully defined.
