# Question

On a retail e-commerce replatforming project, the project manager is examining a request for change to the checkout flow. To assess it properly under PRINCE2, whose perspectives should the impact analysis cover?

## Options

- **A**: Only the supplier, since they will build the change
- **B**: Only the project manager, who must protect the plan
- **C**: The business, user and supplier perspectives
- **D**: The project board chair alone, as the ultimate decision maker

## Answer

C

## Explanation

PRINCE2 impact analysis should consider the business, user and supplier perspectives so the change authority sees the full effect on the business case, on those who will use the product, and on those who will build it. (A) ignores the business case and user needs. (B) confines the analysis to the plan and overlooks the three interests. (D) confuses who decides with whose impacts must be assessed.
