# Question

Who typically produces a checkpoint report and to whom is it sent?

## Options

- **A**: The project manager produces it for the project board
- **B**: The team manager produces it for the project manager
- **C**: The project board produces it for corporate management
- **D**: The project assurance role produces it for the project manager

## Answer

B

## Explanation

A checkpoint report is created by the team manager to report the status of a work package to the project manager at the frequency agreed in the work package. (A) and (C) misassign the report; (D) project assurance advises and checks but does not produce checkpoint reports, which originate from the team delivering the work.
