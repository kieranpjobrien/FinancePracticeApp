# Question

A pharmaceutical trial project is forecast to breach its overall project-level cost tolerance, not just the current stage tolerance. The project board has reviewed the project manager's exception report. What is the correct next step?

## Options

- **A**: The project board approves an exception plan and authorises the project manager to continue under it
- **B**: The project manager revises the project plan and continues without further reference
- **C**: The project board immediately closes the project
- **D**: The project board escalates the situation to corporate, programme or customer management, who hold authority over project-level tolerances

## Answer

D

## Explanation

Project-level tolerances are owned by corporate, programme or customer management, so a forecast breach at that level must be escalated by the project board beyond itself, not resolved within it. (A) an exception plan handles a stage-level breach the board can authorise, but it cannot self-approve a project-level overrun; (B) the project manager has no authority to absorb a project tolerance breach; (C) premature closure pre-empts the higher authority's decision.
