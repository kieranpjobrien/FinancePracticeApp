# Question

Midway through a stage of a bridge-refurbishment project, the project manager forecasts that the current stage will overrun its agreed time tolerance by three weeks. What is the BEST response?

## Options

- **A**: Continue working and absorb the delay, informing the board only at the next end-stage assessment
- **B**: Quietly re-plan the remaining work to claw back the time without telling anyone
- **C**: Raise an exception to the project board and, if requested, prepare an exception plan
- **D**: Reduce the scope of the final product to fit the original timescale on the project manager's own authority

## Answer

C

## Explanation

A forecast breach of stage tolerance must be escalated to the project board by raising an exception; the board may then request an exception plan setting out the recovery options. (A) and (B) conceal a tolerance breach, which is not permitted. (D) changing the agreed scope is a decision for the board, not the project manager acting alone.
