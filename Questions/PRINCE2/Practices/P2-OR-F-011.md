# Question

Which set of roles together make up the project board?

## Options

- **A**: Project Manager, Team Manager and Project Support
- **B**: Executive, Project Manager and Change Authority
- **C**: Executive, Senior User and Senior Supplier
- **D**: Senior User, Senior Supplier and Project Assurance

## Answer

C

## Explanation

The project board comprises the Executive, the Senior User(s) and the Senior Supplier(s), representing the business, user and supplier interests respectively, and is accountable for directing the project. (A) lists the project management and support levels below the board; (B) and (D) mix board and non-board roles — the Project Manager, Change Authority and Project Assurance are not board members in their own right.
