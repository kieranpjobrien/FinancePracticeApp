# Question

PRINCE2 controls progress by delegating authority from one management level to the level below. Which sequence correctly shows authority being delegated downwards?

## Options

- **A**: Project manager → corporate management → project board → team manager
- **B**: Project board → corporate management → team manager → project manager
- **C**: Corporate/programme management → project board → project manager → team manager
- **D**: Team manager → project manager → project board → corporate management

## Answer

C

## Explanation

Authority flows down from corporate, programme or customer management to the project board, then to the project manager, and finally to the team manager, with tolerances set at each handover. (A) and (B) place corporate management below the board; (D) describes the reporting direction upwards, not the delegation of authority downwards.
