# Question

In PRINCE2 7, what is a tolerance?

## Options

- **A**: The permissible deviation above and below a plan's target for a performance target without needing to refer the matter to the next level of management
- **B**: A record of the costs actually incurred against those that were forecast in the plan
- **C**: The fixed point against which progress is measured and reported
- **D**: A formal request to change a product that has already been approved as part of the baseline

## Answer

A

## Explanation

A tolerance is the permissible deviation above and below a plan's target before the deviation must be escalated to the next level of management; tolerances are set for the seven performance targets. (B) describes cost actuals; (C) describes a baseline; (D) describes a request for change handled under the Issues practice.
