# Question

A Change Authority on a software migration has been given a change budget and authority to approve changes costing up to £5,000. A requested change is assessed at £8,000 but is considered essential by the users. What should happen?

## Options

- **A**: The Change Authority approves it anyway, since the users consider it essential
- **B**: The Senior User overrules the limit and authorises the spend directly
- **C**: The Project Manager splits the change into two smaller requests to keep each under £5,000
- **D**: The Change Authority refers the request to the project board because it exceeds the delegated limit

## Answer

D

## Explanation

A Change Authority may only act within the limits delegated by the project board; a change exceeding those limits must be escalated to the board for a decision. (A) breaches the delegated authority and the principle of managing within agreed limits; (C) is a deceptive workaround that circumvents proper control; (B) wrongly lets a single board member unilaterally override an agreed delegated limit.
