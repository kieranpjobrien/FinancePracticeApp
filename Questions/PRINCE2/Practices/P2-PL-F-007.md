# Question

Which of the following would you expect to find in a product description rather than in the project product description?

## Options

- **A**: The customer's overall quality expectations for the project
- **B**: The acceptance criteria for the final project output
- **C**: The quality criteria and quality method for an individual specialist product
- **D**: The acceptance responsibilities for the project as a whole

## Answer

C

## Explanation

A product description defines an individual product, including its purpose, composition, derivation, quality criteria and the quality method used to check it. (A), (B) and (D) — overall quality expectations, project acceptance criteria and acceptance responsibilities — all belong in the project product description, which describes the project's single overall output.
