# Question

On a pharmaceutical packaging project, a printed leaflet must achieve a colour-match score of 95, with a quality tolerance allowing a minimum of 92. The completed batch scores 93. The project manager is mid-stage and within all stage tolerances. What is the BEST response?

## Options

- **A**: Escalate an off-specification to the project board, because the leaflet has not hit the target score of 95
- **B**: Accept the leaflet and record the passed check in the quality register, because the score is within the defined quality tolerance
- **C**: Raise a request for change so the colour-match criterion can be formally relaxed to 93
- **D**: Issue an exception report to the project board, since a quality criterion has been missed

## Answer

B

## Explanation

A score of 93 sits inside the agreed quality tolerance (minimum 92), so the product conforms and the check is simply recorded as a pass; quality tolerance exists precisely to permit acceptable variation around the target. (A) is the tempting near-miss, but missing the target value is not an off-specification while still inside tolerance; (C) no change is needed because the criterion is met; (D) an exception report concerns forecast breaches of stage or project tolerances, which are not in play here.
