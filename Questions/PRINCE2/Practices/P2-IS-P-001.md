# Question

On a hospital IT system project, a clinical user submits a request for change that, if approved, would add cost beyond the change budget delegated to the change authority. The project manager has assessed the impact. What should the project manager do?

## Options

- **A**: Approve the change personally, because impact analysis is complete
- **B**: Reject the change because it exceeds the change budget
- **C**: Escalate the request to the project board for a decision
- **D**: Ask the clinical user to resubmit a smaller version of the change

## Answer

C

## Explanation

When a request for change would breach the change budget delegated to the change authority, the decision exceeds delegated authority and must be escalated to the project board. (A) has the project manager acting beyond authority — the project manager cannot approve changes that breach delegated limits. (B) prematurely rejects a potentially valuable change rather than letting the board decide. (D) sidesteps the proper governance route and pre-empts the board's judgement.
