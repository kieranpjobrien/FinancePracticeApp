# Question

PRINCE2 describes a recommended issue and change control procedure. Which sequence correctly reflects its steps?

## Options

- **A**: Decide, capture, implement, examine, propose
- **B**: Propose, decide, capture, implement, examine
- **C**: Examine, capture, decide, propose, implement
- **D**: Capture, examine, propose, decide, implement

## Answer

D

## Explanation

The PRINCE2 issue and change control procedure runs: capture the issue, examine its impact, propose a course of action, decide (escalating where it exceeds tolerance), then implement the decision. (A), (B) and (C) all reorder these steps illogically — for instance deciding before capturing or implementing before examining — which breaks the principle that decisions follow assessment.
