# Question

Near the end of a stage on an offshore wind project, an off-specification is confirmed: a delivered product misses a minor acceptance criterion. The project manager has examined it and judges that correcting it is not worthwhile and the stage and project tolerances are not threatened. What is the BEST response?

## Options

- **A**: Authorise a concession personally and close the issue without informing the board
- **B**: Raise an exception report because an off-specification has occurred
- **C**: Halt the stage until the product is reworked to full specification
- **D**: Recommend that the project board grant a concession to accept the product as it is

## Answer

D

## Explanation

When correcting an off-specification is not justified, PRINCE2 allows the product to be accepted via a concession, but granting a concession is a project board decision, so the project manager should recommend it. (A) has the project manager exceeding authority by accepting a deviation from baseline alone. (B) is the tempting near-miss: an exception report is triggered by a forecast breach of tolerance, not by an off-specification that stays within tolerance. (C) wastes effort reworking a product the board may be content to accept.
