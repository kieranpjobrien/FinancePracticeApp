# Question

While managing a stage of a museum redevelopment, the project manager assesses an aggregated set of newly emerged threats. Even after planned responses, the residual exposure now clearly breaches the risk tolerance set by the project board. What is the correct action?

## Options

- **A**: Apply further responses quietly within the stage and avoid alarming the project board
- **B**: Raise an exception report to the project board, because the risk tolerance has been forecast to be exceeded
- **C**: Record the position in the risk register and continue, as risk is managed continually
- **D**: Wait until the end-stage assessment to discuss the position with the project board

## Answer

B

## Explanation

When the residual risk is forecast to exceed the agreed risk tolerance, the situation is an exception that the project manager must escalate to the project board, normally via an exception report. (A) concealing a tolerance breach violates the manage-by-exception principle. (C) merely recording it ignores that tolerance has been exceeded, and (D) delaying to the stage boundary leaves a known tolerance breach unescalated.
