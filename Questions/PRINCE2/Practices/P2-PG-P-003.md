# Question

A museum refurbishment project has reached the end of its design management stage. Before any further work or spending is committed, what should happen next under the Progress practice?

## Options

- **A**: The project manager authorises the next stage on their own initiative to avoid delay
- **B**: The team manager closes the project because the design is complete
- **C**: Corporate management takes over day-to-day control of the next stage
- **D**: The project manager produces an end stage report and the project board decides whether to authorise the next stage

## Answer

D

## Explanation

At a stage boundary the project manager produces an end stage report and the project board reviews it before deciding whether to authorise the next stage, applying the manage-by-stages principle. (A) authorising the next stage is a project board decision, not the project manager's; (B) completing one stage does not mean closing the project; (C) corporate management delegates control to the board, it does not run stages directly.
