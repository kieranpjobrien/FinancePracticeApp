# Question

A banking project to launch a new mobile payments feature is nearing handover. The Executive asks who should be accountable for confirming that the predicted customer-adoption benefits are actually realised after the product goes live. What is the correct PRINCE2 answer?

## Options

- **A**: The Project Manager, because they delivered the product
- **B**: The Senior Supplier, because they built the feature
- **C**: The Senior User, because the user interest is responsible for specifying and realising the expected benefits
- **D**: Project Assurance, because it independently verifies all outcomes

## Answer

C

## Explanation

The Senior User represents those who will use the product and is responsible for specifying the benefits and confirming they are realised, including post-project benefit reviews. (A) is a tempting near-miss, but the Project Manager is responsible for delivering products, not for realising long-term benefits; (B) covers product quality, not benefit realisation; (D) assures that the project is run properly but does not own the benefits.
