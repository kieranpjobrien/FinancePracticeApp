# Question

A project has several different external suppliers, each critical to delivery. How does PRINCE2 accommodate this on the project board?

## Options

- **A**: Each supplier must have its own separate project board
- **B**: The Executive role is split so each supplier has equal accountability
- **C**: The Project Manager represents all suppliers so the board stays small
- **D**: More than one person may share the Senior Supplier role to represent the multiple supplier interests

## Answer

D

## Explanation

PRINCE2 allows the Senior User and Senior Supplier roles to be shared among more than one person where multiple user or supplier interests must be represented, while the board remains a single decision-making body. (A) is wasteful and contrary to a single directing authority; (C) is wrong because suppliers are represented at board level, not by the Project Manager; (B) breaches the rule that there is only one Executive who cannot be shared.
