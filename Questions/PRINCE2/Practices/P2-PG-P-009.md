# Question

A project manager on a wind-farm project is delegating a work package to a team manager. The team is experienced and the work is routine, but the project manager wants assurance the package stays on track without micromanaging. What is the BEST approach under the Progress practice?

## Options

- **A**: Set tight tolerances and require the team manager to seek approval before every task
- **B**: Agree appropriate tolerances in the work package and a sensible frequency for checkpoint reports, escalating only on a forecast breach
- **C**: Remove all tolerances so the team manager is free to deliver however they see fit
- **D**: Ask the project board to receive the team's checkpoint reports directly

## Answer

B

## Explanation

Effective delegation sets clear work package tolerances and an agreed checkpoint reporting cadence, with management by exception so the team manager escalates only when a breach is forecast. (A) requiring approval for every task is micromanagement that defeats delegation; (C) removing tolerances eliminates control altogether; (D) routing checkpoint reports to the board breaks the line of authority, as those reports are addressed to the project manager.
