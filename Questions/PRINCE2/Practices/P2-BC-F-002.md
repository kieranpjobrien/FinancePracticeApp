# Question

Which role owns the business case in a PRINCE2 project?

## Options

- **A**: The Project Manager
- **B**: The Senior User
- **C**: The Executive
- **D**: The Project Assurance function

## Answer

C

## Explanation

The Executive is accountable for the project and owns the business case throughout the project's life, ensuring the project remains value for money. (A) the Project Manager prepares and maintains the business case but does not own it; (B) the Senior User is accountable for specifying and realising benefits, not owning the case; (D) Project Assurance reviews the business case independently. Ownership sits with the single individual accountable to corporate or programme management.
