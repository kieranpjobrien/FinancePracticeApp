# Question

On a large aerospace programme, the Project Manager is spending so much time maintaining records, version-controlling products and chasing report formats that managing the work is suffering. Several other projects in the programme face the same issue. What is the BEST organisational response?

## Options

- **A**: Ask the project board to reduce the level of reporting required
- **B**: Establish a shared Project Support function to provide administrative and configuration management services across the projects
- **C**: Transfer the record-keeping to the Team Managers within their work packages
- **D**: Hand the configuration management role to Project Assurance

## Answer

B

## Explanation

Project Support provides administrative, configuration and information-management services to the Project Manager and can be shared across multiple projects in a programme, freeing the Project Manager to manage delivery. (A) is a tempting near-miss but cutting governance reporting weakens control rather than addressing the administrative burden; (C) overloads delivery teams with project-level administration; (D) compromises the independence of assurance by giving it an operational support task.
