# Question

What is the key distinction between the Project Manager and the Team Manager?

## Options

- **A**: The Project Manager directs the project while the Team Manager funds it
- **B**: They are interchangeable titles for the same single role
- **C**: The Team Manager sits on the project board and the Project Manager reports to them
- **D**: The Project Manager manages the project on a day-to-day basis, while the Team Manager manages the production of products allocated through work packages

## Answer

D

## Explanation

The Project Manager runs the project day to day within the constraints set by the project board, while the Team Manager is an optional role responsible for producing the products in an authorised work package and reporting progress to the Project Manager. (A) is wrong — the project board directs and the Executive owns funding; (C) reverses the reporting line and the Team Manager does not sit on the board; (B) is incorrect because they are distinct roles operating at different management levels.
