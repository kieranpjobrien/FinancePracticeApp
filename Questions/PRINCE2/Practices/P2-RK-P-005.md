# Question

On a rail signalling upgrade, the project manager wants to fund a contingent action that would only be carried out if a specific supplier-delay threat materialises. A risk budget has been established in the risk management approach. How should this be funded?

## Options

- **A**: Draw on the change budget, since a delay alters the plan
- **B**: Absorb the cost within the current work package's cost tolerance
- **C**: Request additional project funding from the project board as an exception
- **D**: Draw on the risk budget, which exists to fund responses to threats and opportunities

## Answer

D

## Explanation

A risk budget is set aside specifically to pay for risk responses, including contingent actions, so it is the correct source for funding this planned response. (A) the change budget funds requests for change, not risk responses. (C) escalating as an exception is unnecessary when a risk budget already exists for exactly this purpose, and (B) work package tolerance is not intended to absorb funded risk responses.
