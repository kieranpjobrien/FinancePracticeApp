# Question

Which threat response aims to make the uncertainty disappear so that the threat can no longer occur?

## Options

- **A**: Reduce
- **B**: Transfer
- **C**: Avoid
- **D**: Accept

## Answer

C

## Explanation

The "avoid" response changes some aspect of the project so that the threat can no longer occur or can no longer affect the project. (A) "reduce" lowers probability or impact but does not eliminate the threat, (B) "transfer" passes part of the impact to a third party, and (D) "accept" makes no active change at all.
