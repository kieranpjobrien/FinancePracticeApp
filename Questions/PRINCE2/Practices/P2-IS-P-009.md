# Question

On a pharmaceutical packaging line project, the project manager captures a problem/concern and, after impact analysis, forecasts that resolving it will push the current stage beyond its agreed cost tolerance. What is the BEST course of action?

## Options

- **A**: Record the issue, then produce an exception report for the project board
- **B**: Absorb the extra cost from the change budget and continue the stage
- **C**: Reduce the stage scope without telling the board so tolerance is preserved
- **D**: Ask the team manager to find savings elsewhere to stay within tolerance

## Answer

A

## Explanation

When analysis shows a stage is forecast to exceed its agreed tolerance, the project manager must escalate by producing an exception report for the project board, as the deviation is beyond delegated authority. (B) misuses the change budget, which funds requests for change, not a tolerance breach from a problem/concern. (C) hides a tolerance breach and unilaterally cuts scope, violating the manage-by-exception principle. (D) is the near-miss — informal cost juggling does not address a forecast breach that, by definition, requires board escalation.
