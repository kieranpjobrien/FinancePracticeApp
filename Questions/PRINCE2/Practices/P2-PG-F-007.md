# Question

A stage is forecast to exceed its agreed tolerances. Which management product should the project manager produce to alert the project board?

## Options

- **A**: An issue report
- **B**: An end stage report
- **C**: An exception report
- **D**: A highlight report

## Answer

C

## Explanation

When a stage or the project is forecast to breach tolerance, the project manager raises an exception report so the project board can decide how to respond. (A) an issue report captures an individual issue but does not itself signal a tolerance breach to the board; (B) the end stage report is produced at a planned stage boundary; (D) a highlight report gives routine status only.
