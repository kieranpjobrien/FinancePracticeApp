# Question

For which set of performance targets can tolerances be set under PRINCE2 7?

## Options

- **A**: Time, cost, quality, scope, benefits, risk and sustainability
- **B**: Time and cost only
- **C**: Time, cost and scope only
- **D**: Quality, risk and benefits only

## Answer

A

## Explanation

PRINCE2 7 defines seven performance targets — time, cost, quality, scope, benefits, risk and sustainability — and tolerances can be set against each. (B), (C) and (D) are all incomplete subsets; sustainability is a target newly emphasised in PRINCE2 7 and must be included.
