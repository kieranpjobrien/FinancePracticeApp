# Question

On a railway-signalling upgrade, the project board has approved an exception plan to recover a forecast breach of project-level tolerance. Once the board approves it, what becomes of the original project plan?

## Options

- **A**: The original project plan continues to be the baseline and the exception plan is filed for reference only
- **B**: Both plans run in parallel, with the project manager choosing which to report against
- **C**: The exception plan applies only to the current stage and the project plan is unchanged
- **D**: The approved exception plan replaces the project plan and becomes the new baseline against which progress is measured

## Answer

D

## Explanation

When an exception plan is approved at project level, it replaces the plan that was forecast to breach tolerance and becomes the new baseline for measuring progress. (A) the original plan is superseded, not retained as the baseline. (B) running two competing baselines defeats the purpose of control. (C) this exception is at project level, so it replaces the project plan, not merely a stage plan.
