# Question

A project manager on a hospital records-digitisation project is starting to plan the next stage. The team is keen to jump straight into listing tasks and assigning people. What should the project manager do FIRST?

## Options

- **A**: Begin assigning team members to tasks so work can start quickly
- **B**: Produce the product breakdown structure and product descriptions for the stage before identifying activities
- **C**: Draft the stage's Gantt chart and circulate it for resourcing
- **D**: Ask the project board to set the stage tolerances before any planning begins

## Answer

B

## Explanation

PRINCE2 uses product-based planning: the products to be delivered are identified and described first, and the activities and resources are derived from them. (A) and (C) jump to activities and scheduling before the products are defined, risking gaps and rework. (D) tolerances are agreed when the plan is presented for approval, not before the planning work has been done.
