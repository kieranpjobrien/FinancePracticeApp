# Question

At which level of plan does the project manager plan the detailed activities, products and resources for a single management stage?

## Options

- **A**: Project plan
- **B**: Exception plan
- **C**: Team plan
- **D**: Stage plan

## Answer

D

## Explanation

A stage plan is produced for each management stage and provides the level of detail the project manager needs for day-to-day control of that stage. (A) the project plan is a high-level whole-project plan. (B) an exception plan replaces a plan that is forecast to exceed tolerance. (C) a team plan is optional and prepared by a team manager for the work in a work package.
