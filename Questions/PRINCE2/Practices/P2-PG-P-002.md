# Question

The project board of a banking software project has asked to be kept informed of stage progress every fortnight without attending meetings. Which control should the project manager use to satisfy this request?

## Options

- **A**: Schedule a fortnightly project board review meeting
- **B**: Send the board a checkpoint report every fortnight
- **C**: Provide the board with a highlight report every fortnight
- **D**: Issue an exception report every fortnight

## Answer

C

## Explanation

The highlight report is the time-driven control the project manager uses to give the board a regular written summary of stage progress at the board's chosen frequency. (A) the board explicitly does not want meetings; (B) checkpoint reports flow from team manager to project manager, not to the board; (D) exception reports are event-driven and used only for forecast tolerance breaches, not routine updates.
