# Question

A work package agreed between the project manager and a team manager is forecast to exceed the tolerances set within it. What is the correct response under the Progress practice?

## Options

- **A**: The team manager continues working and reports the overrun in the next checkpoint report only
- **B**: The team manager escalates the situation to the project manager, as the work package tolerance has been breached
- **C**: The team manager escalates directly to the project board
- **D**: The project manager immediately produces an exception report to the project board

## Answer

B

## Explanation

Tolerances delegated within a work package are owned by the project manager, so a forecast breach must be escalated by the team manager to the project manager. (A) ignores the breach until routine reporting; (C) bypasses the project manager and the proper line of authority; (D) is premature, because the project manager may still be able to keep the stage within its tolerances without troubling the board.
