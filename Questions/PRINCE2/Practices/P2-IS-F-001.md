# Question

In PRINCE2, which of the following is the correct definition of an issue?

## Options

- **A**: Any uncertain event that could affect the project's objectives if it were to occur
- **B**: A relevant event that has happened, was not planned, and requires management action
- **C**: A failure to meet an agreed acceptance criterion that is discovered at the end of the project
- **D**: A proposal to enhance the scope of an approved product baseline

## Answer

B

## Explanation

In PRINCE2 7, an issue is any relevant event that has happened, was not planned, and requires management action — it has already occurred, distinguishing it from a risk. (A) describes a risk, an uncertain future event. (C) describes only one specific issue type (an off-specification) and adds a false timing constraint. (D) describes a request for change, which is just one of the three issue types.
