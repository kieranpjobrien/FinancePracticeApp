# Question

In PRINCE2, what is the issue report and when is it created?

## Options

- **A**: A report sent to the project board at the end of every stage summarising all open issues
- **B**: A standing agenda item raised verbally at each project board meeting
- **C**: A description of an issue being handled formally, created when the issue is registered
- **D**: The document that captures lessons so they can be applied to future projects

## Answer

C

## Explanation

The issue report is a management product that describes, assesses and records a single issue being handled formally; it is created when the issue is entered in the issue register. (A) describes elements of an end stage report or highlight report, not the issue report. (B) is not a PRINCE2 management product. (D) describes the lessons log or lessons report.
