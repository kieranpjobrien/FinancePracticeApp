# Question

A Senior User on a council waste-collection project is short of time and asks the Project Manager to carry out the user assurance activities on her behalf. What should the Project Manager do?

## Options

- **A**: Accept, since the Project Manager is best placed to assure the work
- **B**: Decline, explaining that Project Assurance must stay independent of the Project Manager, and suggest the Senior User appoint someone else to perform user assurance
- **C**: Ask Project Support to take on the user assurance role
- **D**: Refer the request to the Senior Supplier to perform on the user's behalf

## Answer

B

## Explanation

Project Assurance is a project board responsibility that must remain independent of the Project Manager, so the Project Manager cannot assure their own work; the Senior User may instead delegate user assurance to a suitable independent person. (A) destroys the independence that makes assurance meaningful; (C) misassigns assurance to an administrative role; (D) confuses the supplier interest with the user interest, so the Senior Supplier cannot provide user assurance.
