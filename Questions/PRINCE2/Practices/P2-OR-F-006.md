# Question

Which statement about Project Assurance is correct?

## Options

- **A**: Project Assurance is carried out by the Project Manager as part of controlling a stage
- **B**: Project Assurance and Project Support are the same function under different names
- **C**: Project Assurance can be delegated to the Team Manager when the board is busy
- **D**: Project Assurance is the responsibility of the project board and must remain independent of the Project Manager

## Answer

D

## Explanation

Project Assurance belongs to the project board members, who assure that the project is being conducted properly on behalf of their respective interests; it must be independent of the Project Manager so it cannot be delegated to them. (A) is wrong because the Project Manager cannot assure their own work; (C) is wrong as assurance supports the board, not the delivery teams; (B) confuses assurance (a board responsibility) with support (an administrative function for the Project Manager).
