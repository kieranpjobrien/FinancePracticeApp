# Question

On a highway resurfacing project, the project manager realises mid-stage that the current stage will finish nine days late, breaching the agreed time tolerance of five days. What should the project manager do?

## Options

- **A**: Absorb the delay quietly by reducing the scope of a later stage without telling anyone
- **B**: Produce an exception report and submit it to the project board
- **C**: Wait until the end of the stage and explain the overrun in the end stage report
- **D**: Instruct the team manager to work unpaid overtime to recover the schedule

## Answer

B

## Explanation

A forecast breach of stage tolerance must be escalated to the project board via an exception report so the board can decide on the appropriate response. (A) hides the deviation and changes scope without authority; (C) delays escalation past the point where the board can act; (D) is an inappropriate unilateral action that still does not inform the board of the tolerance breach.
