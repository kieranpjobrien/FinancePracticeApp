# Question

In the context of the Progress practice, what is a baseline?

## Options

- **A**: A snapshot of the costs incurred so far on the project
- **B**: The earliest date by which a product can be delivered
- **C**: The set of tolerances delegated to a team manager
- **D**: A reference level against which an entity is monitored and controlled

## Answer

D

## Explanation

A baseline is a reference level against which an entity such as a plan or product is monitored and controlled; progress is measured by comparing actuals to the baseline. (A) describes cost actuals; (B) describes a schedule constraint, not a baseline; (C) describes work package or stage tolerances rather than a baseline reference.
