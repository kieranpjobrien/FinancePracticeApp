# Question

While scheduling a stage of a wind-farm project, the project manager realises that turbine installation cannot start until grid-connection approval is granted by the regional network operator, who is not part of the project. How should this be reflected in the plan?

## Options

- **A**: Ignore it, since anything outside the project is beyond the plan's scope
- **B**: Add it as an additional product in the product breakdown structure
- **C**: Record it as an external dependency in the schedule so the installation activity is sequenced correctly
- **D**: Treat it as a stage tolerance to be set by the project board

## Answer

C

## Explanation

A reliance on a deliverable or decision from outside the plan is an external dependency and must be captured in the schedule so activities are sequenced realistically. (A) external dependencies directly affect the plan and cannot be ignored. (B) the approval is a dependency, not a product the project creates. (D) it is a dependency, not a tolerance.
