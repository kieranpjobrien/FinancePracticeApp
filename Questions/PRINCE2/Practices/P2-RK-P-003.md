# Question

A software project faces a threat that an inexperienced development team may produce defects late in the build, raising costs. The project manager decides to add early peer code reviews and additional automated testing to lower the chance and severity of defects. Which threat response is being applied?

## Options

- **A**: Avoid
- **B**: Reduce
- **C**: Transfer
- **D**: Accept

## Answer

B

## Explanation

Adding peer reviews and extra testing is a proactive action that lowers both the probability and the impact of the threat, which is the "reduce" response. (A) "avoid" would require changing the project so the threat could not occur at all, for example removing the work entirely. (C) "transfer" would shift the financial impact to a third party such as a supplier, and (D) "accept" would take no action, which contradicts the steps being taken.
