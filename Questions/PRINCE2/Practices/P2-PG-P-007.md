# Question

After receiving an exception report on a hospital construction project, the project board decides the stage should continue but along a revised path. Which management product should it now ask the project manager to produce, and what happens to it?

## Options

- **A**: A new project initiation documentation, to be approved by corporate management
- **B**: An exception plan, which the project board reviews and may approve to replace the remainder of the current stage plan
- **C**: A highlight report, which is filed for the project record
- **D**: An end stage report, which triggers immediate closure of the stage

## Answer

B

## Explanation

When the board accepts that a stage should proceed despite a forecast breach, it requests an exception plan, which it then reviews and may approve to supersede the current stage plan for the rest of the stage. (A) the PID covers the whole project and is not reissued for a single stage exception; (C) a highlight report is routine and does not replan; (D) an end stage report is for a planned boundary, not for replanning after an exception.
