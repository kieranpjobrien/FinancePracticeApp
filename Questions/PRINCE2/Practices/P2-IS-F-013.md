# Question

In PRINCE2, where should a minor issue that can be handled informally by the project manager be recorded?

## Options

- **A**: In the issue register, with a corresponding issue report
- **B**: In the daily log
- **C**: In the risk register
- **D**: In an exception report to the project board

## Answer

B

## Explanation

PRINCE2 allows minor issues to be managed informally and recorded in the daily log, avoiding the overhead of the formal procedure. (A) is reserved for issues handled formally. (C) is for risks, not issues. (D) is used only when a stage or project is forecast to exceed its tolerances, which a minor informal issue does not warrant.
