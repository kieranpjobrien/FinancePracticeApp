# Question

What is the primary purpose of an end stage report?

## Options

- **A**: To give the project board the information needed to review the success of the current stage and approve progression to the next
- **B**: To request authorisation to start a new work package
- **C**: To record day-to-day informal events for the project manager
- **D**: To escalate a forecast tolerance breach to corporate management

## Answer

A

## Explanation

The end stage report gives the project board the information it needs to review how the stage performed and to decide whether to authorise the next stage. (B) work packages are authorised by the project manager, not via this report; (C) describes the daily log; (D) escalation of a tolerance breach uses an exception report.
