# Question

Which of the following is an example of an event-driven control rather than a time-driven control?

## Options

- **A**: Highlight report
- **B**: Checkpoint report
- **C**: End stage report
- **D**: A weekly progress meeting

## Answer

C

## Explanation

Event-driven controls take place when a specific event occurs; the end stage report is produced because a stage boundary has been reached. (A) highlight reports and (B) checkpoint reports are time-driven, produced at a defined frequency; (D) a weekly meeting is also time-driven because it recurs on a schedule.
