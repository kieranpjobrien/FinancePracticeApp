# Question

On a software-migration project, a team manager has been given a work package but is unsure how to organise the detailed work for her team of developers. What should she do?

## Options

- **A**: Ask the project manager to plan the detailed developer tasks for her
- **B**: Wait for the project board to issue a stage plan for her team
- **C**: Update the project plan with the developer-level detail
- **D**: Create a team plan to organise the work needed to deliver the work package

## Answer

D

## Explanation

A team manager produces a team plan to organise and control the detailed work required to complete the work packages assigned to her. (A) the project manager assigns work packages but does not plan the team's internal detail. (B) the project board does not issue stage plans, and stage plans are not team-level. (C) the project plan is a high-level whole-project plan and should not carry team task detail.
