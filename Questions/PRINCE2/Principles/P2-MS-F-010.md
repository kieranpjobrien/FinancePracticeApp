# Question

Which PRINCE2 process produces the next stage plan and updates the project plan in readiness for the project board's go/no-go decision?

## Options

- **A**: Starting up a project
- **B**: Initiating a project
- **C**: Managing a stage boundary
- **D**: Closing a project

## Answer

C

## Explanation

The "managing a stage boundary" process is performed near the end of a stage to plan the next stage, update the project plan and business case, and report performance so the board can decide whether to proceed. (A) Starting up a project precedes initiation and produces the project brief. (B) Initiating a project assembles the project initiation documentation for the whole project. (D) Closing a project ends the final stage rather than enabling the next one.
