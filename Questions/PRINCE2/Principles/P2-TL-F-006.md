# Question

In PRINCE2 7, what is the difference between "adopting" and "adapting" the method?

## Options

- **A**: Adopting is done by the project; adapting is done by the organisation
- **B**: Adopting refers to an organisation embedding PRINCE2 across its projects; adapting (tailoring) refers to applying it to suit a specific project
- **C**: Adopting means using every process; adapting means using only some processes
- **D**: Adopting and adapting are interchangeable terms for the same activity

## Answer

B

## Explanation

Adopting describes an organisation embedding PRINCE2 into its own way of working across many projects, while adapting (tailoring) is applying the method to the particular needs of a specific project. (A) reverses the typical ownership emphasis; (C) misdescribes both terms; (D) is wrong because the two terms refer to distinct activities at different levels.
