# Question

How does a PRINCE2 project plan typically differ from a stage plan?

## Options

- **A**: The project plan is detailed and the stage plan is high-level
- **B**: The project plan covers a single stage and the stage plan covers the whole project
- **C**: The project plan is high-level and covers the whole project, while the stage plan is detailed and covers one stage
- **D**: They are identical documents with different titles

## Answer

C

## Explanation

The project plan provides a high-level view of the whole project for the project board, whereas a stage plan provides the detail needed by the project manager to control a single management stage day to day. (A) The relationship is reversed. (B) The scopes are stated the wrong way round. (D) They serve different audiences and levels of detail.
