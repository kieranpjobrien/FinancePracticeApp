# Question

To "streamline" a manufacturing project, a project manager proposes tailoring by removing stage boundaries and management stages, so the project runs as one continuous flow with no end-stage assessments. What is the BEST response?

## Options

- **A**: Approve it, since reducing stages is a normal form of tailoring
- **B**: Reject it, because removing all management stages and decision points undermines the manage by stages principle, even though the number of stages can be tailored
- **C**: Approve it provided the project manager reports more frequently to the team manager instead
- **D**: Reject it and insist the project uses the maximum possible number of stages

## Answer

B

## Explanation

The number of management stages can be tailored, but a PRINCE2 project must have at least an initiation stage and one further management stage with board decision points, so eliminating stages and end-stage assessments altogether breaches the manage by stages principle. (A) is the tempting near-miss: adjusting the number of stages is valid tailoring, but abolishing the concept of stages is not; (C) keeps no board control and reports to the wrong role; (D) overcorrects by mandating maximum stages, which is equally untailored.
