# Question

A renewable-energy installation project will run for two years through unstable supply markets. The sponsor proposes a single delivery stage covering all the work after initiation to reduce administrative effort. As project manager, what is the BEST response?

## Options

- **A**: Agree, because fewer stages always make a project cheaper to run
- **B**: Refuse to run the project at all under these conditions
- **C**: Keep initiation only and start delivery without defining any further stages
- **D**: Recommend dividing the delivery work into several management stages so the board has regular go/no-go control points across the uncertain period

## Answer

D

## Explanation

Long, high-uncertainty projects warrant more management stages so the board can reassess viability and re-plan at frequent control points; the project manager should advise breaking the delivery work into several stages. (A) Minimising stages sacrifices control on a risky two-year project. (B) Refusing is disproportionate. (C) A project must have at least one delivery stage defined, and one giant stage would remove the control the principle exists to provide.
