# Question

On a hospital records digitisation project, the project board has an executive and a senior supplier, but no one represents the clinicians and administrators who will use the new system. What should the project manager do?

## Options

- **A**: Proceed, as the executive can speak for the user interest as well as the business interest
- **B**: Recommend appointing a senior user so the user interest is properly represented on the board
- **C**: Ask the senior supplier to gather user requirements on the board's behalf
- **D**: Add the most vocal clinician to the delivery team to capture their views

## Answer

B

## Explanation

The principle requires all three interests to be represented, so the gap should be closed by appointing a senior user to speak for those who will use the system. (A) Combining executive and senior user undermines balanced representation and is not appropriate when a distinct user community exists. (C) The supplier cannot represent the user interest, which would conflict with its own. (D) Placing a user in the delivery team does not give the user interest a voice in board-level decisions.
