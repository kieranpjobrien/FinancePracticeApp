# Question

A motorway upgrade project depends on a traffic-modelling report produced by a separate government agency outside the project's control. The project manager wants to manage this dependency well within product-based planning. What is the BEST action?

## Options

- **A**: Identify the report as an external product, document it in a product description, and reflect the dependency in the product flow diagram
- **B**: Exclude the report from the plan because the project cannot produce it
- **C**: Recreate the report internally so the project controls every product
- **D**: Treat the report as an off-specification because it is not yet available

## Answer

A

## Explanation

External products are still identified in product-based planning, given a product description so expectations are clear, and shown as dependencies in the product flow diagram even though the project does not create them. (B) Excluding the dependency hides a real risk to the plan. (C) Duplicating another body's work is wasteful and outside scope. (D) An unavailable external product is a dependency to manage, not an off-specification, which concerns failing to meet an agreed quality criterion.
