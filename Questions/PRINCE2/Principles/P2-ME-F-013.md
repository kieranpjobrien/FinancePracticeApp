# Question

After reviewing an exception report, the project board asks the project manager to prepare an exception plan. Who approves that exception plan before the stage continues?

## Options

- **A**: The team manager
- **B**: The project manager, using their delegated authority
- **C**: The project board
- **D**: Project assurance

## Answer

C

## Explanation

An exception plan replaces the remainder of the current stage plan, and because changing the agreed stage baseline is a directing decision, the project board approves it. (A) The team manager works to the plan but does not approve it. (B) Approving a new stage baseline lies above the project manager's delegated authority. (D) Project assurance advises the board but holds no approval authority.
