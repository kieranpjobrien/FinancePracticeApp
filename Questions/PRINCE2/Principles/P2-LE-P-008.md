# Question

A project manager has diligently recorded 40 lessons in the lessons log over six months of a manufacturing-line upgrade, but project performance has not improved and the same scheduling errors keep recurring. Which point of 'learn from experience' is MOST likely being neglected?

## Options

- **A**: Lessons are not being sought from enough external sources
- **B**: Lessons are being recorded but not acted upon to change how the project is run
- **C**: The lessons log was created too early in the project life cycle
- **D**: Too few people are permitted to add lessons to the log

## Answer

B

## Explanation

The principle requires lessons to be sought, recorded and acted upon; recurring errors despite a full log show the 'acting upon' element is missing. (A) the volume of recorded lessons suggests sourcing is not the gap; the repeated errors point to inaction. (C) creating the log early is correct PRINCE2 practice, not a fault. (D) 40 lessons indicate participation is not the constraint here.
