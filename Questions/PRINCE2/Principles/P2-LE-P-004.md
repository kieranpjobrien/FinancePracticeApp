# Question

A solar-farm construction project is closing. Throughout delivery the team logged numerous lessons, several of which would help the organisation's upcoming wind-farm project. What should the project manager ensure happens at closure?

## Options

- **A**: Archive the lessons log unread, as the project is finished
- **B**: Delete the lessons log to keep the project archive concise
- **C**: Keep the lessons confidential to the closing project team
- **D**: Review the lessons and pass relevant ones on to benefit future projects

## Answer

D

## Explanation

The principle requires that, as a project closes, lessons useful to future projects are reviewed and passed on, typically through a lessons report or the end project report. (A) archiving unread loses the value of the captured lessons. (B) deleting lessons removes exactly the learning future projects need. (C) keeping lessons confidential prevents the wind-farm project from benefiting.
