# Question

In the PRINCE2 four levels of management, between which two levels are stage tolerances set and exceptions reported?

## Options

- **A**: Corporate, programme or customer management and the project board
- **B**: The project board and the project manager
- **C**: The project manager and the team manager
- **D**: The team manager and the team members

## Answer

B

## Explanation

Stage tolerances are delegated by the project board to the project manager, and the project manager reports a stage-level exception back to the project board. (A) This boundary governs project-level tolerance, set by corporate, programme or customer management. (C) The project manager sets work package tolerances for the team manager, which is a different boundary. (D) Team members operate below the formal tolerance structure.
