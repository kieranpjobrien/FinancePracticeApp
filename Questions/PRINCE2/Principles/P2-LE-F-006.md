# Question

How does the lessons log differ from a lessons report in PRINCE2?

## Options

- **A**: The lessons log is a formal report to the project board, while the lessons report is an informal working document
- **B**: They are two names for the same management product
- **C**: The lessons log records risks, while the lessons report records issues
- **D**: The lessons log is an ongoing repository of lessons, while a lessons report communicates lessons that could benefit other projects

## Answer

D

## Explanation

The lessons log is an informal repository that accumulates lessons throughout the project, whereas a lessons report is used to pass on lessons that could be applied to other projects, typically at a stage boundary or at project closure. (A) reverses their nature. (B) they are distinct products. (C) confuses them with the risk register and issue register.
