# Question

According to the "Ensure continued business justification" principle, what should happen if a project's business justification ceases to be valid during execution?

## Options

- **A**: The project should continue to closure because cancelling wastes the investment already made
- **B**: The project board should reduce the project's tolerances and continue
- **C**: The project manager should rewrite the Business Case to make it valid again
- **D**: The project should be stopped (closed prematurely)

## Answer

D

## Explanation

If the justification disappears, the project should be stopped, because continuing to invest in a project with no valid reason wastes resources. (A) is the sunk-cost fallacy that the principle explicitly guards against. (B) does nothing to restore justification. (C) is dishonest — the Business Case reflects reality and must not be massaged simply to keep a project alive.
