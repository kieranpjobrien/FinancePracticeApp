# Question

A pharmaceutical firm's drug-trial software project is mid-stage and within all tolerances. The drug it supports has just failed clinical trials and been abandoned, so the software will never be used. The project manager confirms the team can still finish on time and budget. What is the BEST response?

## Options

- **A**: Continue to the end of the current stage, since the project is within tolerance and on schedule
- **B**: Finish the project so the software is available in case the drug is revived later
- **C**: Reduce the remaining scope and deliver a cut-down version to save money
- **D**: Raise the loss of justification to the project board immediately so it can decide whether to close the project prematurely

## Answer

D

## Explanation

Being within tolerance is irrelevant once the justification has vanished; the principle requires that an unjustified project be stopped, and only the project board can authorise premature closure, so it must be told now. (A) is the tempting near-miss — staying in tolerance does not justify continuing a project that delivers no benefit. (B) and (C) keep spending on a product with no valid use, which the principle exists to prevent.
