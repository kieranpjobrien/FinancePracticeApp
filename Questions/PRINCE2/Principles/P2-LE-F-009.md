# Question

A lesson has been recorded in the lessons log. To satisfy the 'learn from experience' principle, what else must happen?

## Options

- **A**: Nothing further; recording the lesson is sufficient to satisfy the principle
- **B**: The lesson must be deleted once the relevant stage has closed
- **C**: The lesson must be escalated to corporate management before any work continues
- **D**: The lesson must be acted upon where it can improve the project or future projects

## Answer

D

## Explanation

The principle requires lessons to be sought, recorded AND acted upon; merely logging a lesson is not enough if nothing changes as a result. (A) recording alone leaves the principle only partly met. (B) lessons are retained, not deleted, so they can benefit future projects. (C) escalation is not automatically required simply because a lesson is recorded.
