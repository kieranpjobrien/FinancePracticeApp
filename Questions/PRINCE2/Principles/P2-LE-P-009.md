# Question

A previous office-fit-out project breached its sustainability target by sending large volumes of waste to landfill, and this was captured as a lesson. A new fit-out project is now starting. What is the BEST way to apply this lesson?

## Options

- **A**: Note the lesson but treat sustainability as outside the scope of learning from experience
- **B**: Wait until the new project also breaches the target before consulting the earlier lesson
- **C**: Use the lesson when planning the new project so the sustainability performance target is better managed
- **D**: Assume the earlier lesson cannot apply because every fit-out is different

## Answer

C

## Explanation

Lessons from previous projects should be acted upon when planning a new project, and sustainability is one of the seven performance targets PRINCE2 manages, so the lesson should inform how this target is handled. (A) sustainability is squarely within the project's targets and the scope of learning. (B) waiting for a repeat breach defeats the purpose of the lesson. (D) dismissing transferable lessons as inapplicable contradicts the principle's intent.
