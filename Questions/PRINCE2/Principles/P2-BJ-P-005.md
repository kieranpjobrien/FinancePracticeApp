# Question

On a city-centre cycleway scheme, a feasibility review during initiation shows the benefits will only marginally exceed the costs and risks, leaving the project barely viable. What should the project manager do before the project is authorised?

## Options

- **A**: Proceed to authorisation without comment, as a marginal case is still positive
- **B**: Present the marginal justification clearly to the project board so it can make an informed authorisation decision
- **C**: Inflate the benefit estimates so the Business Case looks stronger
- **D**: Decide alone not to start the project because the margin is too small

## Answer

B

## Explanation

Whether a marginally viable project should proceed is a project board decision; the project manager's job is to present an honest Business Case showing the balance of benefits, costs and risks. (A) hides material information the board needs. (C) falsifies the justification and breaches the principle. (D) takes an authorisation decision that belongs to the board, not the project manager.
