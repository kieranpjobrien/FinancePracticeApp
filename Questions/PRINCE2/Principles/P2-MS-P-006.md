# Question

Three weeks before the planned end of stage two on an airport baggage-system upgrade, the project manager forecasts that stage costs will breach the stage cost tolerance, although the next stage boundary is not yet due. What is the BEST response?

## Options

- **A**: Wait until the scheduled stage boundary and report the overspend in the end stage report
- **B**: Raise an exception report to the project board now and, if asked, prepare an exception plan
- **C**: Quietly reallocate budget from the next stage to stay within this stage's figure
- **D**: Reduce the scope of the current work package without telling the board to protect the tolerance

## Answer

B

## Explanation

A forecast breach of stage tolerance is an exception, which under manage by exception must be escalated to the project board immediately via an exception report, not held until the boundary. (A) Waiting for the boundary delays a decision the board must make now. (C) Borrowing from a future stage's budget circumvents the board's control of resources. (D) Cutting scope unilaterally bypasses change control and the board's authority.
