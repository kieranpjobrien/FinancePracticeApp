# Question

How does the "manage by stages" principle relate to the "manage by exception" principle in PRINCE2?

## Options

- **A**: They are the same principle described in two different ways
- **B**: Manage by stages applies only to large projects, while manage by exception applies only to small ones
- **C**: Manage by exception removes the need to divide a project into stages
- **D**: Stages define the control points where tolerances are set, and management by exception governs escalation within each stage

## Answer

D

## Explanation

The two principles work together: each management stage is given tolerances by the project board, and manage by exception means the project manager escalates only when those tolerances are forecast to be exceeded within the stage. (A) They are distinct principles. (B) Both principles apply to all PRINCE2 projects regardless of size. (C) Management by exception operates within stages, so it does not replace staging.
