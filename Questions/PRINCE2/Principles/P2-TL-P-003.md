# Question

A construction firm is starting a large, high-risk, multi-supplier project. The project manager is considering using the same minimal controls that worked on a previous tiny internal project. What should the project manager do?

## Options

- **A**: Reuse the minimal controls to keep overheads low and stay consistent
- **B**: Abandon PRINCE2 because it is unsuitable for large projects
- **C**: Wait until problems occur and then add controls reactively
- **D**: Tailor PRINCE2 to add more rigorous controls appropriate to the greater size, complexity and risk

## Answer

D

## Explanation

Tailoring works in both directions; a large, high-risk, multi-supplier project warrants more rigorous controls scaled to its size, complexity and risk, not less. (A) ignores the project's characteristics and under-controls it; (B) misunderstands PRINCE2, which is designed to scale up as well as down; (C) abandons the proactive, planned nature of tailoring and risks losing control of the project.
