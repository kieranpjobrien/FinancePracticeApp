# Question

Which statement about tailoring PRINCE2 is correct?

## Options

- **A**: Tailoring may include omitting one or more of the seven principles where they add little value
- **B**: Tailoring applies only to large, complex projects
- **C**: Tailoring never means omitting any of the seven principles
- **D**: Tailoring is decided solely by the team manager

## Answer

C

## Explanation

A defining rule of the principle is that tailoring never means omitting any of the seven PRINCE2 principles; they always apply in full. (A) directly contradicts this rule; (B) is wrong because every project, large or small, should be tailored appropriately; (D) is wrong because tailoring is an organisational and project board responsibility supported by the project manager, not the team manager alone.
