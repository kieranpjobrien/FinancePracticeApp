# Question

A small charity wants a quick website rebuild and the chair suggests "just doing the build" without any initiation, to save time. As project manager applying PRINCE2, what should you advise?

## Options

- **A**: Run the project with an initiation stage followed by at least one delivery stage, even if both are short
- **B**: Skip initiation entirely because the project is small
- **C**: Treat the whole project as a single delivery stage with no initiation
- **D**: Run three delivery stages but no initiation stage

## Answer

A

## Explanation

Every PRINCE2 project needs a minimum of two management stages — the initiation stage plus at least one delivery stage — though each may be brief and the processes tailored for a small project. (B) and (C) Omitting initiation removes the foundation set out in the project initiation documentation. (D) A project cannot proceed to delivery stages without first completing an initiation stage.
