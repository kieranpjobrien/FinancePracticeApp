# Question

On a rail signalling upgrade, a team manager reports that a work package will breach the time tolerance the project manager set for it. The project manager confirms that absorbing the slippage keeps the overall stage comfortably within the stage tolerance delegated by the project board. What is the BEST response?

## Options

- **A**: Raise an exception report to the project board, because a tolerance has been forecast to be exceeded
- **B**: Agree a corrective action with the team manager and manage the work package breach within the stage, without escalating to the project board
- **C**: Instruct the team manager to deliver to the original work package dates regardless of feasibility
- **D**: Ask corporate management to authorise additional stage tolerance before acting

## Answer

B

## Explanation

A work package tolerance is delegated by the project manager, so a forecast breach at that level is the project manager's to resolve; because the stage tolerance from the board is unaffected, no exception is escalated. (A) is the tempting near-miss: not every forecast breach goes to the board — only a forecast breach of the board's stage tolerance does. (C) Demanding unfeasible dates ignores the forecast and resolves nothing. (D) Stage tolerance is intact, so there is no basis to seek more from corporate management.
