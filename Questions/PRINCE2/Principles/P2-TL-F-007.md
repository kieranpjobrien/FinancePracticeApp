# Question

Which statement correctly distinguishes embedding from tailoring in PRINCE2 7?

## Options

- **A**: Embedding is performed by an organisation; tailoring is performed for an individual project
- **B**: Embedding is performed by the project board; tailoring is performed by corporate management
- **C**: Embedding removes principles; tailoring removes practices
- **D**: Embedding applies only to programmes; tailoring applies only to projects within a portfolio

## Answer

A

## Explanation

Embedding is what an organisation does to incorporate PRINCE2 into its standard working practices, whereas tailoring is what is done for an individual project to suit its specific context. (B) misassigns the responsibilities; (C) is wrong because neither activity removes principles; (D) invents a scope restriction that does not exist.
