# Question

On a hospital records digitisation project, the project manager is approaching the end of the first delivery stage. The next stage's detailed plan is ready and the current stage finished within tolerance. What should the project manager do next?

## Options

- **A**: Begin the next stage's work immediately to keep momentum, informing the board afterwards
- **B**: Merge the remaining stages into one to avoid further boundaries
- **C**: Produce an end stage report and request the project board's authorisation to proceed to the next stage
- **D**: Ask the team managers to approve the next stage plan

## Answer

C

## Explanation

The "manage by stages" principle makes the stage boundary a go/no-go point, so the project manager presents the end stage report and next stage plan and waits for the project board's authorisation. (A) Starting without authorisation bypasses the board's control point. (B) Removing boundaries strips out planned governance. (D) Authorising a stage is the project board's decision, not the team managers'.
