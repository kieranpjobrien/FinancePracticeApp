# Question

According to the PRINCE2 principle "define roles, responsibilities and relationships", which three project interests must always be represented?

## Options

- **A**: Sponsor, customer and regulator
- **B**: Business, user and supplier
- **C**: Time, cost and quality
- **D**: Executive, senior user and senior supplier

## Answer

B

## Explanation

The principle requires that the three primary project interests — business, user and supplier — are represented so that decisions balance value, fitness for use and feasibility of delivery. (A) names roles and an external party that PRINCE2 does not define as the three interests. (C) lists performance targets, not interests. (D) names the three project board roles, which represent the interests but are not themselves the interests.
