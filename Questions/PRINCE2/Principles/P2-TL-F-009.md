# Question

Which of the following is NOT one of the project characteristics that PRINCE2 tailoring is intended to take into account?

## Options

- **A**: The size and complexity of the project
- **B**: The importance and risk of the project
- **C**: The capability of the people involved
- **D**: The personal preference of the project manager for less paperwork

## Answer

D

## Explanation

Tailoring should reflect the project's environment, size, complexity, importance, team capability and risk, all of which are objective project characteristics. (A), (B) and (C) are all legitimate tailoring factors; (D) is wrong because tailoring is a justified response to the project's needs, not a vehicle for the project manager's personal convenience.
