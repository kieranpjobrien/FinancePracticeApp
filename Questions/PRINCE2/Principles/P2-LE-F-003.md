# Question

According to the 'learn from experience' principle, when should lessons be sought on a PRINCE2 project?

## Options

- **A**: Only at the start of the project, before the business case is approved
- **B**: Only during the delivery of the project's specialist products
- **C**: When starting, while running, and when closing the project
- **D**: Only at the end of the project, as part of the end project report

## Answer

C

## Explanation

PRINCE2 requires lessons to be sought when starting a project (drawing on previous experience), as the project progresses, and as the project closes (passing lessons on to future projects). (A), (B) and (D) each restrict learning to a single stage, whereas the principle applies across the whole project life cycle.
