# Question

On a retail store refurbishment project, a senior user emails the project manager directly and instructs them to add a new shopfront feature that is outside the agreed stage tolerance. What should the project manager do?

## Options

- **A**: Implement the feature immediately because a senior user requested it
- **B**: Raise the matter through the proper route as a potential change for the project board to decide, rather than act on a single member's instruction
- **C**: Refuse the request outright and tell the senior user it is not allowed
- **D**: Quietly absorb the work within the team without telling the board

## Answer

B

## Explanation

A single board member cannot unilaterally direct work that breaches tolerance; the project manager should handle it as a change through the agreed route so the board decides collectively. (A) Acting on one member's instruction bypasses the board's collective authority and the change procedure. (C) An outright refusal is not the project manager's call either; the board decides. (D) Silently absorbing out-of-tolerance work conceals the situation from those accountable.
