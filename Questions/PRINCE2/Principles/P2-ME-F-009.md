# Question

If the project board forecasts that the overall project tolerances will be exceeded, what must it do?

## Options

- **A**: Absorb the deviation, since the board has unlimited authority over the project
- **B**: Refer the matter up to corporate, programme or customer management
- **C**: Instruct the project manager to ignore the forecast until it materialises
- **D**: Close the project immediately without further reference

## Answer

B

## Explanation

The project board operates within project tolerances delegated by corporate, programme or customer management, so a forecast breach of project tolerance must be escalated to that level. (A) The board's authority is bounded by the project tolerances it was given. (C) Acting only once a breach materialises defeats the early-warning intent of manage by exception. (D) Premature closure is itself a decision for the level above, not a unilateral board action.
