# Question

A project manager sets time and cost tolerances on a work package for a team manager. What does this delegation mean in practice?

## Options

- **A**: The team manager may complete the work package freely while forecasts stay within tolerance, escalating to the project manager only if a tolerance is forecast to be exceeded
- **B**: The team manager must obtain the project manager's approval before each activity
- **C**: The team manager may change the work package's quality criteria without reference back
- **D**: The project manager loses all visibility of progress until the work package is complete

## Answer

A

## Explanation

Delegated authority means the team manager works without intervention while forecasts remain within the agreed tolerances, raising an issue to the project manager only when a tolerance is forecast to be breached. (B) Routine per-activity approval contradicts the principle. (C) Quality criteria are fixed in the work package and are not a tolerance the team manager may relax. (D) The project manager retains visibility through checkpoint reports; delegation is not loss of oversight.
