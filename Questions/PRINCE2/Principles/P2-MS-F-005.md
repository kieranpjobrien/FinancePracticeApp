# Question

Which stage is always the first management stage of a PRINCE2 project?

## Options

- **A**: The first delivery stage
- **B**: The initiation stage
- **C**: The closing stage
- **D**: The benefits review stage

## Answer

B

## Explanation

The initiation stage is always the first management stage; during it the project is properly defined and the project initiation documentation is assembled before any delivery work is authorised. (A) Delivery stages follow initiation. (C) Closing activities form the final part of the last stage, not the first. (D) Benefits reviews largely occur after the project, often post-closure.
