# Question

At the start-up of an airport baggage-system upgrade, the executive asks the project manager how the project will avoid arguments about whether the final solution is acceptable. Which action best addresses this concern?

## Options

- **A**: Produce a detailed activity schedule for the whole project
- **B**: Appoint additional team managers to oversee delivery
- **C**: Define the project product description with the customer's acceptance criteria before detailed planning
- **D**: Increase the project's cost tolerance to allow for rework

## Answer

C

## Explanation

Defining the project product description, including acceptance criteria agreed with the customer, gives an objective standard for accepting the final product and prevents end-of-project disputes. (A) An activity schedule does not establish acceptance criteria. (B) More managers do not define what 'acceptable' means. (D) Extra cost tolerance funds rework but does not prevent the disagreement.
