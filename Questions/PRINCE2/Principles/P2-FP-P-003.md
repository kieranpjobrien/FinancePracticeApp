# Question

Midway through a council library refurbishment, a team member begins adding a self-service kiosk system that was never listed among the project's products, arguing it will 'add value'. What is the BEST response from the project manager?

## Options

- **A**: Welcome the addition and let the team build it, since it adds value
- **B**: Treat the new requirement as an issue and handle it through change control before any work proceeds
- **C**: Quietly absorb the extra work within the existing budget without recording it
- **D**: Cancel the affected work package and replace it with the kiosk system

## Answer

B

## Explanation

Because scope is defined by the agreed products, an unplanned addition is uncontrolled scope creep and should be raised as an issue and assessed through change control. (A) Building it unassessed bypasses governance. (C) Absorbing it silently hides a change to scope and risks tolerance breaches. (D) Replacing planned work on a team member's initiative is not the project manager's call to make unilaterally.
