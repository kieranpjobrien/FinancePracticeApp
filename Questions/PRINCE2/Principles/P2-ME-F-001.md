# Question

What does the PRINCE2 principle "manage by exception" establish for a project?

## Options

- **A**: Defined tolerances for each performance target and delegated authority to act within them
- **B**: A requirement that every decision is escalated to the project board
- **C**: That the project manager must seek sign-off before starting any work package
- **D**: That tolerances are set only for time and cost

## Answer

A

## Explanation

The "manage by exception" principle establishes defined tolerances for each project performance target, creating delegated authority so a level of management can act freely within those tolerances. (B) The whole purpose is to avoid escalating routine decisions. (C) Authorising work packages within stage tolerance is the project manager's delegated role, not something requiring board sign-off. (D) Tolerances apply to all seven performance targets, not just time and cost.
