# Question

Where are tailoring decisions for a PRINCE2 project typically documented?

## Options

- **A**: In the daily log only
- **B**: In the project initiation documentation (PID)
- **C**: They are agreed verbally and not recorded
- **D**: In the lessons log at the end of the project

## Answer

B

## Explanation

Tailoring decisions are documented so they are visible and agreed, typically captured in the project initiation documentation (PID), which records how PRINCE2 will be applied to the project. (A) the daily log holds informal day-to-day notes, not the agreed approach; (C) is wrong because tailoring must be recorded, not left informal; (D) the lessons log captures lessons rather than the planned tailoring decisions.
