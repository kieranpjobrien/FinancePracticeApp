# Question

In a PRINCE2 project, scope is most accurately defined by which of the following?

## Options

- **A**: The total budget approved by the project board
- **B**: The number of stages in the project plan
- **C**: The duration agreed for the project
- **D**: The set of products the project will deliver

## Answer

D

## Explanation

Under the 'focus on products' principle, the scope of a PRINCE2 project is defined by the products it will deliver, typically captured in a product breakdown structure. (A) Budget and (C) duration are tolerances or constraints, not the definition of scope. (B) The number of stages is a planning decision and does not determine what the project will produce.
