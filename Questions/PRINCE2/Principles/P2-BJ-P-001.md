# Question

On a hospital records-digitisation project, the project manager is preparing the End Stage Report and Stage Plan for board approval. A government funding change means the predicted benefits have fallen sharply. What should the project manager do?

## Options

- **A**: Update the Business Case with the revised benefits and present it to the project board at the stage boundary
- **B**: Continue into the next stage and address the benefits shortfall at project closure
- **C**: Quietly leave the original benefits figures unchanged so the project is not threatened
- **D**: Cancel the project immediately without consulting the project board

## Answer

A

## Explanation

The principle requires justification to be revalidated at each stage boundary, so the project manager updates the Business Case with the new benefits and lets the board make an informed authorise/stop decision. (B) defers a justification question that must be answered now. (C) corrupts the Business Case and breaches the principle. (D) usurps the project board, which alone authorises stage progression or premature closure.
