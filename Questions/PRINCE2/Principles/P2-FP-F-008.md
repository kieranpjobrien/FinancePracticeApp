# Question

Which management product defines the overall scope and acceptance criteria for the project as a whole?

## Options

- **A**: The work package
- **B**: The daily log
- **C**: The project product description
- **D**: The end stage report

## Answer

C

## Explanation

The project product description defines what the project must deliver overall, including the customer's quality expectations and acceptance criteria for the final product, and so frames the project's scope. (A) A work package describes work for a single team. (B) The daily log records informal notes. (D) The end stage report reviews performance, not overall acceptance criteria.
