# Question

When a new PRINCE2 project starts, how does the 'learn from experience' principle expect previous projects to be used?

## Options

- **A**: Previous projects are ignored, because each project is unique
- **B**: Relevant lessons from previous and similar projects are reviewed and recorded in the lessons log
- **C**: The new project must reuse the previous project's plans without amendment
- **D**: Previous projects are referenced only if the same project manager is appointed

## Answer

B

## Explanation

At the start, the principle expects relevant lessons from previous and similar projects (internal or external) to be sought and recorded so the new project benefits from them. (A) treating every project as unique throws away useful learning. (C) lessons inform planning but do not force wholesale reuse of old plans. (D) lessons are valuable regardless of who manages the project.
