# Question

What does the PRINCE2 principle "manage by stages" require for a project?

## Options

- **A**: The project is planned, monitored and controlled on a stage-by-stage basis
- **B**: Every project must be divided into exactly four management stages
- **C**: Each team member manages their own stage independently of the project manager
- **D**: The project is delivered in a single continuous stage to save time

## Answer

A

## Explanation

The "manage by stages" principle requires that a PRINCE2 project is planned, monitored and controlled on a stage-by-stage basis, giving the project board control at defined points. (B) PRINCE2 sets no fixed number of stages — only a minimum of two. (C) Stages are managed by the project manager, not individual team members. (D) A single-stage approach removes the control points the principle is designed to provide.
