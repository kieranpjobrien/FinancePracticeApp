# Question

A pharmaceutical trial project is two weeks from a planned stage boundary when the project manager forecasts the current stage will breach its time tolerance before that boundary is reached. A board member suggests simply waiting and covering it in the forthcoming end stage report. What is the correct course of action?

## Options

- **A**: Wait for the stage boundary and address the breach in the end stage report as suggested
- **B**: Bring the stage boundary forward by two weeks to avoid producing an exception report
- **C**: Extend the stage tolerance informally so the figures fit until the boundary
- **D**: Raise an exception report now, since the tolerance is forecast to be exceeded before the boundary

## Answer

D

## Explanation

A forecast tolerance breach is handled when it is forecast, through an exception report, irrespective of how close the planned stage boundary is. (A) is the tempting near-miss: the imminent boundary does not justify deferring an active forecast breach to an end stage report. (B) Re-engineering the boundary to dodge the exception manipulates the control rather than using it. (C) The project manager cannot informally widen a tolerance set by the board.
