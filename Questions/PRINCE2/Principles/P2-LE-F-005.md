# Question

From which sources does PRINCE2 expect a project to draw lessons?

## Options

- **A**: Previous projects, this project, and external organisations
- **B**: Only the current project, to keep lessons relevant
- **C**: Only completed projects within the same organisation
- **D**: Only published industry standards and frameworks

## Answer

A

## Explanation

The 'learn from experience' principle expects lessons to be sought from previous projects, from the current project as it runs, and from external sources such as other organisations. (B), (C) and (D) each ignore one or more of these sources; PRINCE2 deliberately draws on all three to improve the chances of success.
