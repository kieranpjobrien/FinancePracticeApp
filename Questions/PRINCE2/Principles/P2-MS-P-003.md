# Question

During initiation of a museum refurbishment project, an eager planner produces a fully detailed, task-level schedule for all four anticipated stages spanning eighteen months. What should the project manager do?

## Options

- **A**: Keep a high-level project plan for the whole project and produce a detailed plan only for the imminent stage
- **B**: Adopt the detailed eighteen-month schedule as the single plan and avoid stage plans
- **C**: Discard the project plan and rely solely on the detailed schedule
- **D**: Delay all planning until each stage actually begins

## Answer

A

## Explanation

PRINCE2 plans at two levels: a high-level project plan for the whole project and detailed stage plans produced just before each stage, because distant detail is unreliable. (B) Locking in eighteen months of task detail will quickly go stale and removes the staged review structure. (C) The project plan is still needed for the board's overall view. (D) Some forward planning is essential; the principle is about timing the detail, not avoiding planning.
