# Question

Who is primarily responsible for ensuring that PRINCE2 is tailored appropriately for a specific project, with the approach approved and recorded?

## Options

- **A**: The team manager, when accepting a work package
- **B**: The project manager, with the tailoring approach approved by the project board
- **C**: Project assurance, acting independently of the project board
- **D**: Corporate, programme or customer management, with no project board involvement

## Answer

B

## Explanation

The project manager is responsible for tailoring PRINCE2 to suit the project and recording how it will be applied, with the project board approving that approach as part of authorising the project. (A) the team manager works within the tailoring already set, not deciding it; (C) project assurance advises and checks but does not own the tailoring decision; (D) wrongly excludes the project board, which must approve the tailoring even though corporate or programme standards may influence it.
