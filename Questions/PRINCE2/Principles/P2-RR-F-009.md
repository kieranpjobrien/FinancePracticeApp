# Question

Which statement about accountability on a PRINCE2 project is correct?

## Options

- **A**: The project board shares ultimate accountability equally among all its members
- **B**: A single individual, the executive, is ultimately accountable for the success of the project
- **C**: The project manager is ultimately accountable for the project's success
- **D**: Accountability rotates between the senior user and senior supplier each stage

## Answer

B

## Explanation

PRINCE2 requires a single point of accountability, so the executive is ultimately accountable for the project's success and owns the business case throughout. (A) Accountability is not shared equally; one person must be answerable. (C) The project manager is responsible for day-to-day management but accountable to the board, not ultimately accountable. (D) Accountability does not rotate between roles.
