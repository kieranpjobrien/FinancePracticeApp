# Question

A team manager forecasts that a work package will finish three days late, but the project manager confirms the stage still has enough time tolerance to absorb the slippage. What does manage by exception indicate the project manager should do?

## Options

- **A**: Raise an exception report to the project board immediately
- **B**: Manage the situation within the stage and not escalate to the project board
- **C**: Request an exception plan from the team manager
- **D**: Close the stage early to lock in the remaining tolerance

## Answer

B

## Explanation

Because the deviation remains within the stage tolerance the project board delegated, the project manager handles it at the managing level and does not escalate. (A) An exception report is only warranted when a stage tolerance is forecast to be exceeded. (C) Exception plans are prepared by the project manager at the board's request, not demanded from a team manager. (D) Closing a stage early is not a tolerance-management action and is not the board's instruction here.
