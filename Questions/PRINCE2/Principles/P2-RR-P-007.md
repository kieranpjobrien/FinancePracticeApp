# Question

On a banking compliance project, the senior supplier suggests that, to reduce overhead, the project manager should also perform project assurance for the supplier interest. Why should the project manager push back?

## Options

- **A**: Because project assurance must be independent of the project manager, who cannot objectively assure their own management of the work
- **B**: Because only the executive is ever permitted to carry out any project assurance
- **C**: Because project assurance can only be performed by an external auditor
- **D**: Because the supplier interest does not require assurance, only the business interest does

## Answer

A

## Explanation

Project assurance must be independent of the project manager so that the board receives an objective check; a project manager cannot assure their own day-to-day management. (B) is the tempting near-miss: the board owns assurance and may delegate it, but it is not restricted to the executive performing it personally. (C) Assurance is the board's responsibility and need not be an external auditor. (D) Each of the three interests, including the supplier interest, is entitled to assurance.
