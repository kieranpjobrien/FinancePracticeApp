# Question

At what point in a PRINCE2 project is the lessons log first created?

## Options

- **A**: During 'starting up a project', so that earlier lessons can be captured from the outset
- **B**: During 'closing a project', once all lessons have been identified
- **C**: At the first stage boundary, after the initiation stage completes
- **D**: Only if the project board specifically requests one

## Answer

A

## Explanation

The lessons log is created during 'starting up a project' so that lessons from previous and similar projects can be recorded and applied from the very beginning. (B) creating it at closure would lose the chance to act on lessons during delivery. (C) is too late to capture early lessons. (D) the lessons log is a standard part of applying the principle, not optional on request.
