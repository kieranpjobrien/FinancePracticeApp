# Question

A project manager is starting up a new hospital records digitisation project. A very similar digitisation project at a neighbouring trust finished last year and produced a detailed lessons report. What should the project manager do?

## Options

- **A**: Ignore the other trust's report, since lessons from a different organisation do not apply
- **B**: Review the external lessons report and record relevant lessons in the lessons log
- **C**: Wait until the project closes, then compare both projects' lessons
- **D**: Forward the report to the project board and take no further action

## Answer

B

## Explanation

The 'learn from experience' principle expects lessons to be sought from external sources as well as previous internal projects, and recorded in the lessons log during start-up so they can be acted upon. (A) wrongly excludes external lessons, which PRINCE2 explicitly values. (C) waiting until closure forfeits the benefit during delivery. (D) merely forwarding the report does not satisfy the duty to record and act on the lessons.
