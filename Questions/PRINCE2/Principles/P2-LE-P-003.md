# Question

At a stage boundary on a retail loyalty-app project, the team identifies several lessons that other digital projects in the organisation would clearly benefit from. How should the project manager BEST share these?

## Options

- **A**: Leave them in the lessons log and assume other project managers will read it
- **B**: Email the lessons informally to a few colleagues the project manager knows
- **C**: Produce a lessons report so the lessons can be passed on to those who can use them
- **D**: Withhold the lessons until the whole project completes, to avoid duplication

## Answer

C

## Explanation

PRINCE2 uses a lessons report to communicate lessons that could benefit other projects, which can be issued at a stage boundary as well as at closure. (A) relying on others to stumble upon the lessons log does not actively pass lessons on. (B) ad-hoc emails are not a reliable route to those who can apply the lessons. (D) waiting until project end delays value other projects could gain now.
