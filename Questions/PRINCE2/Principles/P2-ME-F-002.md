# Question

In PRINCE2, what is a tolerance?

## Options

- **A**: The total contingency budget held by the project board
- **B**: A fixed buffer added to every activity estimate by the team manager
- **C**: The maximum number of management stages allowed in a project
- **D**: The permissible deviation from a plan's target before the deviation must be escalated

## Answer

D

## Explanation

A tolerance is the permissible deviation above and below a plan's target for a performance target, without escalation to the next level of management. (A) Contingency is a separate budgeted response to a risk, not the definition of a tolerance. (B) Tolerance is a control boundary set by the higher level of management, not a buffer added by the team manager to estimates. (C) The number of stages is unrelated to tolerance.
