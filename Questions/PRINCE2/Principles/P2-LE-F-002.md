# Question

What is the purpose of the lessons log in PRINCE2?

## Options

- **A**: To record formal requests to change the project's baselined products
- **B**: To list the products to be produced and their quality criteria
- **C**: To capture and track all project-level risks and their responses
- **D**: To provide an informal repository for lessons that apply to this project or future projects

## Answer

D

## Explanation

The lessons log is a project repository for lessons that apply to this project or future projects. (A) describes the issue register/change control records. (B) describes the product descriptions and the project product description. (C) describes the risk register. The lessons log supports the 'learn from experience' principle by giving captured lessons a home.
