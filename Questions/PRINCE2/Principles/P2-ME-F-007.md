# Question

Which statement correctly distinguishes an exception report from an exception plan?

## Options

- **A**: An exception report forecasts a tolerance breach and seeks direction; an exception plan replaces the plan for the remainder of the work the breach affects
- **B**: An exception report replaces the current stage plan; an exception plan simply warns the project board
- **C**: Both are produced by the project board to instruct the project manager
- **D**: An exception report is only used for cost tolerance; an exception plan is only used for time tolerance

## Answer

A

## Explanation

An exception report is produced by the project manager to alert the project board that a stage tolerance is forecast to be exceeded and to seek a decision; if the board asks for an exception plan, that plan replaces the remaining portion of the affected plan. (B) reverses the two products. (C) Both are produced for or by the project manager, not issued by the board. (D) Neither product is restricted to a single performance target.
