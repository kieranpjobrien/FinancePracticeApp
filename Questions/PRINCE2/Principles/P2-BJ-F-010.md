# Question

A project is 80% complete when a competitor releases a free product that removes the entire market need. Applying the "Ensure continued business justification" principle, which view is correct?

## Options

- **A**: The project should be completed because most of the money has already been spent
- **B**: The project should be reviewed and, if no longer justified, stopped despite the work already done
- **C**: The justification cannot change once the final stage has begun
- **D**: Only the project board can change the justification, and only at project closure

## Answer

B

## Explanation

The principle requires justification to remain valid throughout; if it disappears the project should be stopped even when nearly finished, since further spend adds no value. (A) is the sunk-cost trap. (C) is false — justification can change at any time and must be checked continually. (D) wrongly limits when justification can be reassessed.
