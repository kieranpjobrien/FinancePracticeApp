# Question

A short, low-risk office relocation project is being initiated. The project manager wants to combine several routine reports into one lightweight progress update to save effort. What should the project manager do?

## Options

- **A**: Apply the combined reporting and record this tailoring decision in the project initiation documentation for the project board to approve
- **B**: Introduce the combined reporting quietly without recording it, since it only affects reports
- **C**: Use every standard PRINCE2 report in full because tailoring is not permitted on small projects
- **D**: Ask the team manager to decide whether the reports can be combined

## Answer

A

## Explanation

Tailoring of management products such as combining reports is legitimate, but the approach must be documented (in the PID) and approved by the project board. (B) fails the requirement to record tailoring decisions; (C) wrongly claims small projects cannot be tailored, which contradicts the principle; (D) misplaces the decision with the team manager rather than the project manager and board.
