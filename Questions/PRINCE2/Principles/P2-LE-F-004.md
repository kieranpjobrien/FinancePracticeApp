# Question

Under the 'learn from experience' principle, who is responsible for seeking, recording and acting on lessons?

## Options

- **A**: The project board alone, as part of its directing role
- **B**: The project assurance function only
- **C**: Everyone involved with the project
- **D**: An external lessons specialist appointed for the project

## Answer

C

## Explanation

PRINCE2 states that everyone involved with the project is responsible for seeking, recording and acting on lessons, rather than leaving it to one person or group. (A) the project board directs but does not hold sole responsibility for lessons. (B) project assurance reviews but does not own the activity alone. (D) PRINCE2 does not require a dedicated external specialist for this.
