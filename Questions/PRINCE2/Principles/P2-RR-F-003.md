# Question

In PRINCE2, which management team represents the three project interests of business, user and supplier?

## Options

- **A**: The project team
- **B**: The change authority
- **C**: The project board
- **D**: The project assurance function

## Answer

C

## Explanation

The project board is structured so that the three project interests — business, user and supplier — are each represented, enabling balanced direction and decision-making. (A) The delivery team builds products but does not represent the interests at board level. (B) A change authority handles delegated change decisions only. (D) Project assurance is an independent check on each interest, not the body that represents them.
