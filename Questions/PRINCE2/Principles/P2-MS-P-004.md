# Question

At the boundary of stage two on a retail loyalty-app project, new market data shows a competitor has launched an identical free service, weakening the expected benefits. The next stage plan is otherwise ready. What should the project manager do?

## Options

- **A**: Authorise the next stage personally to avoid disappointing the team
- **B**: Continue and hide the market data so the project is not stopped
- **C**: Cancel the project immediately on the project manager's own authority
- **D**: Update the business case and present it with the end stage report so the project board can make an informed go/no-go decision

## Answer

D

## Explanation

A stage boundary is where continued business justification is confirmed, so the project manager updates the business case to reflect the weakened benefits and presents it with the end stage report for the board's decision. (A) Authorising a stage is the board's role, not the project manager's. (B) Concealing information breaches PRINCE2's continued business justification principle. (C) The decision to stop also rests with the project board.
