# Question

Which project interest is concerned with whether the project remains a worthwhile investment and delivers value for money?

## Options

- **A**: The user interest
- **B**: The supplier interest
- **C**: The business interest
- **D**: The assurance interest

## Answer

C

## Explanation

The business interest focuses on whether the investment is, and remains, worthwhile and delivers value for money, and is represented by the executive on the project board. (A) The user interest is concerned with those who will use the products. (B) The supplier interest provides the resources and skills to create the products. (D) "Assurance interest" is not one of the three PRINCE2 interests.
