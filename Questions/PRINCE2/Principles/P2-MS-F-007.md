# Question

Why does PRINCE2 recommend planning the next stage in detail only towards the end of the current stage rather than at the very start of the project?

## Options

- **A**: Because the project board forbids any planning until each stage begins
- **B**: Because stage plans must be written by the team members, not the project manager
- **C**: Because the project plan replaces the need for any stage plans
- **D**: Because detailed plans for distant stages quickly become inaccurate, so near-term detail is most reliable

## Answer

D

## Explanation

It is difficult to plan distant work accurately, so PRINCE2 uses a high-level project plan for the whole project and produces the detailed stage plan close to the time it is needed, when information is most reliable. (A) The project board does not forbid planning; stages are simply planned progressively. (B) Stage plans are owned by the project manager. (C) The project plan and stage plans coexist at different levels of detail.
