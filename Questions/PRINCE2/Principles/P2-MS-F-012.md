# Question

Which factor would most appropriately lead a project board to decide on a larger number of management stages?

## Options

- **A**: Higher project risk and a longer planning horizon, which call for more frequent control points
- **B**: A desire to reduce the project board's involvement to a single review
- **C**: A wish to avoid producing any end stage reports
- **D**: A requirement that every stage last exactly the same number of weeks

## Answer

A

## Explanation

The number of management stages is driven by factors such as project risk, complexity and how far ahead work can be planned reliably; higher risk and uncertainty justify more, shorter stages with more frequent go/no-go reviews. (B) More stages increase, not reduce, board control points. (C) Each stage boundary still produces an end stage report. (D) Stages need not be of equal length; they are sized to the control needs of the work.
