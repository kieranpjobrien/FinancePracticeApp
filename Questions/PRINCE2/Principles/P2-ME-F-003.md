# Question

Tolerances in PRINCE2 7 can be set against which set of performance targets?

## Options

- **A**: Time, cost, quality, scope, benefits, risk and sustainability
- **B**: Time and cost only
- **C**: Scope, quality and stakeholder satisfaction only
- **D**: Cost, risk and resource availability only

## Answer

A

## Explanation

PRINCE2 7 recognises seven performance targets — time, cost, quality, scope, benefits, risk and sustainability — and tolerance may be set against each of them. (B), (C) and (D) each list only a partial subset; restricting tolerance to a few targets ignores the full set that PRINCE2 7 uses to manage performance and exceptions.
