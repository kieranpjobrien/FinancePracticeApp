# Question

On a factory automation project, the project manager forecasts that the current stage will overspend by 15%, well beyond the cost tolerance set by the project board. What should the project manager do?

## Options

- **A**: Quietly reallocate budget from a later stage to cover the overspend
- **B**: Produce an exception report and submit it to the project board
- **C**: Reduce the stage's quality criteria to bring costs back within tolerance
- **D**: Continue the stage and report the overspend in the next end stage report

## Answer

B

## Explanation

A forecast breach of stage cost tolerance is exactly the trigger for manage by exception: the project manager produces an exception report to alert the project board and seek a decision. (A) Moving budget between stages without authority bypasses the board and breaches delegated authority. (C) The project manager cannot unilaterally relax quality criteria to mask a cost problem. (D) Waiting until the end stage report ignores the forecast and removes the board's chance to act early.
