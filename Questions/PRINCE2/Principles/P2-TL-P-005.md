# Question

On a two-week software upgrade with an experienced, co-located team, the project manager insists on holding the full set of formal meetings and producing lengthy reports identical to those on a year-long programme. The team complains it is pure bureaucracy. What is the BEST response?

## Options

- **A**: Continue exactly as planned, because following PRINCE2 fully is always correct
- **B**: Stop using PRINCE2 entirely for this project
- **C**: Tailor the approach so the controls and products are proportionate to the small, low-risk project, avoiding mechanical use of the method
- **D**: Keep the heavy process but ask the team to spend less time on it

## Answer

C

## Explanation

Tailoring exists precisely to avoid mechanical, robotic application; controls and management products should be proportionate to this small, low-risk project with a capable team. (A) treats PRINCE2 as a fixed checklist, which the tailoring principle warns against; (B) overreacts by dropping the method instead of tailoring it; (D) keeps the disproportionate process while merely rushing it, which does not address the lack of tailoring.
