# Question

A team manager on an offshore wind maintenance programme is about to authorise work on a product, but notices the product description supplied in the work package has no quality criteria and no defined quality method. What should the team manager do FIRST?

## Options

- **A**: Begin the work and agree the quality criteria with the team as the product takes shape
- **B**: Write suitable quality criteria themselves and proceed without informing the project manager
- **C**: Raise the gap with the project manager so the product description is completed before the work proceeds
- **D**: Reject the work package and escalate directly to the project board

## Answer

C

## Explanation

A product cannot be properly built or assessed without agreed quality criteria and a quality method, so the team manager should refer the incomplete product description back to the project manager before starting. (A) Inventing criteria mid-build defeats the purpose of agreeing them in advance. (B) Defining them unilaterally and silently undermines the agreed baseline and shared understanding. (D) Escalating to the board bypasses the project manager, who owns the work package.
