# Question

On a project where the products will be built by an external contractor, which project interest does that contractor primarily represent on the project board?

## Options

- **A**: The business interest
- **B**: The user interest
- **C**: The supplier interest
- **D**: The delivery interest

## Answer

C

## Explanation

The supplier interest represents those who provide the resources, skills and expertise to create the project's products, so an external contractor sits within this interest, represented by the senior supplier. (A) The business interest concerns the investment case and value for money. (B) The user interest concerns those who will operate or benefit from the products. (D) "Delivery interest" is not one of the three PRINCE2 interests.
