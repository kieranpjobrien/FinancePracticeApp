# Question

Following an exception report on a university campus refurbishment, the project board decides the stage should still go ahead but on a revised basis. What does manage by exception indicate happens next?

## Options

- **A**: The project board writes a new stage plan and hands it to the project manager
- **B**: The current stage plan stays in force unchanged and the exception is closed
- **C**: The project manager continues under the old tolerances and notes the variance in the lessons log
- **D**: The project manager prepares an exception plan for the board to approve, which then replaces the remainder of the current stage plan

## Answer

D

## Explanation

When the board wants the stage to continue on a revised basis, it requests an exception plan; the project manager prepares it, the board approves it, and it replaces the rest of the affected stage plan. (A) Producing plans is the project manager's responsibility, not the board's. (B) Leaving the plan unchanged ignores that a tolerance was forecast to be breached. (C) Carrying on under tolerances already known to be unachievable is not a valid response to an exception.
