# Question

On a road resurfacing project, a team manager realises that an unexpected ground condition will push their work package beyond both the time and cost tolerances agreed with the project manager. What should the team manager do first?

## Options

- **A**: Raise an issue to the project manager that the work package tolerances are forecast to be exceeded
- **B**: Send an exception report directly to the project board
- **C**: Absorb the overrun silently and try to recover it later in the work package
- **D**: Stop all work on the project until corporate management responds

## Answer

A

## Explanation

A team manager who forecasts a breach of work package tolerance escalates by raising an issue to the project manager, who decides how to respond. (B) Team managers report to the project manager, not directly to the project board. (C) Concealing a forecast breach removes the early warning the principle relies on. (D) Halting the whole project is disproportionate and is not the team manager's decision.
