# Question

At a stage boundary, which management product does the project manager produce to give the project board an account of how the current stage performed?

## Options

- **A**: The benefits management approach
- **B**: The end stage report
- **C**: The project brief
- **D**: The work package

## Answer

B

## Explanation

The end stage report is produced at a stage boundary so the project board can assess the current stage's performance against its plan before authorising the next stage. (A) The benefits management approach defines how benefits will be measured, not stage performance. (C) The project brief is an early product created before initiation. (D) A work package authorises team-level delivery work within a stage, not a review of the whole stage.
