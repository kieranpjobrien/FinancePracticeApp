# Question

On a local-government housing project, the Project Manager delegates the maintenance of the project's configuration records and the filing of quality records to Project Support. A board member objects that the Project Manager is "offloading accountability". What is the BEST response?

## Options

- **A**: Agree and take all the administrative work back, since delegating it transfers accountability away from the Project Manager
- **B**: Reassign the tasks to the Team Manager instead, as support activities belong within Work Package delivery
- **C**: Ask the project board to take on the record-keeping to preserve independence
- **D**: Explain that the responsibility for performing these tasks can be delegated to Project Support while the Project Manager remains accountable for the project's management

## Answer

D

## Explanation

PRINCE2 distinguishes responsibility from accountability: the Project Manager may delegate the responsibility for support tasks to Project Support while still being accountable for managing the project, so no accountability has been offloaded. (A) misunderstands the distinction — delegating the work does not transfer accountability; (B) misassigns administrative support to the Team Manager, whose remit is producing Work Package products; (C) confuses routine support with assurance independence and burdens the board inappropriately.
