# Question

Who is accountable for ensuring that the communication needs of stakeholders external to the project are met?

## Options

- **A**: The project manager alone, with no involvement from the board
- **B**: The project board, which has a duty to communicate with and represent stakeholder interests
- **C**: The team managers delivering the work packages
- **D**: Project assurance, which takes over all external communication

## Answer

B

## Explanation

The project board is accountable for engaging with the project's stakeholders and representing the business, user and supplier interests externally, supported by the project manager. (A) the project manager manages day-to-day communication but is not solely accountable for external stakeholder engagement. (C) team managers focus on delivering products, not external stakeholder relations. (D) project assurance checks that engagement is happening but does not own the communication itself.
