# Question

On a retail point-of-sale rollout, a Team Manager realises the assigned Work Package can no longer be delivered within its agreed tolerances. What should the Team Manager do?

## Options

- **A**: Quietly absorb the overrun and hope it can be recovered later in the Work Package
- **B**: Raise an issue to the project manager so the situation can be assessed and managed
- **C**: Take the decision directly to the project board, bypassing the project manager
- **D**: Reduce the quality of the products so the original tolerances can still be met

## Answer

B

## Explanation

When a Team Manager forecasts that Work Package tolerances will be exceeded, the correct route is to raise the matter to the project manager, who manages by exception and decides how to respond. (A) concealing the overrun breaches the agreed reporting expectations; (C) going straight to the board ignores the proper chain and the project manager's role; (D) silently cutting quality compromises the product and the quality target without authority.
