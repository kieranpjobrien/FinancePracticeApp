# Question

PRINCE2 7 describes the project manager as a leader. Which statement BEST reflects this view?

## Options

- **A**: The project manager's leadership is limited to administering plans and chasing progress reports
- **B**: The project manager leads the project team day to day, building commitment and enabling people to deliver, in addition to managing the work
- **C**: Leadership belongs solely to the executive, so the project manager need only manage
- **D**: The project manager leads only the suppliers, while the senior user leads the team

## Answer

B

## Explanation

PRINCE2 7 explicitly casts the project manager as a leader who motivates and enables the team day to day, not merely an administrator. (A) reduces the role to clerical management and ignores the leadership dimension. (C) wrongly confines leadership to the executive; leadership is exercised at every level. (D) misallocates team leadership; the project manager leads the project team, including across supplier and user interests.
