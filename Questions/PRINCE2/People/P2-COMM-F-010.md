# Question

When choosing communication methods and channels, which factor should MOST influence the project manager's choice for a given stakeholder?

## Options

- **A**: The method that is cheapest for the project to operate
- **B**: The method the project board personally finds most convenient
- **C**: Whichever channel the project management team used on the last project
- **D**: The stakeholder's information needs, location and preferred way of receiving information

## Answer

D

## Explanation

Channel choice should be driven by the recipient's information needs, geography, culture and communication preferences so that messages actually land. (A) cost is a consideration but should not override effectiveness for the audience. (B) the board's convenience is only one stakeholder group, not the deciding factor for all communication. (C) reusing past channels by habit ignores the specific stakeholders of the current project.
