# Question

Why does PRINCE2 7 emphasise collaboration across the business, user and supplier interests within the project team?

## Options

- **A**: So that decisions balance value for money, fitness for use and what is feasible to deliver
- **B**: So that the supplier always has the final say on the solution design
- **C**: So that the project can dispense with a single point of accountability
- **D**: So that the project manager no longer needs to report to the project board

## Answer

A

## Explanation

Bringing the three interests together lets the team weigh business value, user needs and supplier feasibility so that decisions are balanced rather than dominated by one viewpoint. (B) is wrong because no single interest should dominate; (C) contradicts PRINCE2, which retains a clear single point of accountability; (D) is unrelated — collaboration does not remove the project manager's reporting line to the board.
