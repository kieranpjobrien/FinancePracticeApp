# Question

Why does PRINCE2 7 emphasise that communication should include feedback loops rather than being one-directional?

## Options

- **A**: Because feedback loops remove the need for a communication management approach
- **B**: Because feedback transfers accountability for the project to the stakeholders
- **C**: Because receiving information back from stakeholders confirms messages are understood and surfaces concerns early
- **D**: Because one-way broadcasts are prohibited by the PRINCE2 principles

## Answer

C

## Explanation

Feedback loops make communication two-way, confirming that messages have been received and understood and allowing concerns, risks and misunderstandings to surface early. (A) the approach still defines how feedback is gathered, so it is not made redundant. (B) accountability remains with the defined project roles regardless of feedback. (D) broadcasts can be appropriate for some audiences; the point is to build in return channels, not ban one-way messages.
