# Question

Within the project team, who holds ultimate accountability for the success of the project?

## Options

- **A**: The Project Manager
- **B**: The Senior User
- **C**: The Executive
- **D**: The Team Manager

## Answer

C

## Explanation

PRINCE2 7 places ultimate accountability for the project with the Executive, who owns the business case and represents the business interest as the single decision-making authority. (A) the Project Manager is responsible for day-to-day management but not ultimately accountable; (B) the Senior User is accountable only for specifying user needs and realising benefits; (D) the Team Manager is accountable solely for delivering assigned Work Packages.
