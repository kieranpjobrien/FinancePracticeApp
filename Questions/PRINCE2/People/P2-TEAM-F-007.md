# Question

How do the responsibilities of the Project Manager and the Team Manager differ in PRINCE2 7?

## Options

- **A**: The Project Manager manages the project on a day-to-day basis within the constraints set by the board, while the Team Manager oversees the creation of products within a Work Package
- **B**: The Project Manager creates the products while the Team Manager directs the project on behalf of the board
- **C**: They share identical responsibilities, with the Team Manager simply deputising when the Project Manager is absent
- **D**: The Team Manager sets stage tolerances while the Project Manager builds the specialist products

## Answer

A

## Explanation

The Project Manager runs the project day to day within board-set constraints, and the Team Manager takes responsibility for producing the products defined in an assigned Work Package. (B) reverses both roles; (C) is wrong because the roles are distinct, not interchangeable deputies; (D) misassigns tolerance-setting, which is a board function, not a Team Manager one.
