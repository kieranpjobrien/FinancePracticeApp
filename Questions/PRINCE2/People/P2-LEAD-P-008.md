# Question

A construction project manager notices the team has stopped raising problems and is simply following instructions, even when they can see better approaches. Delivery metrics are on target and within tolerance. Which interpretation and response is BEST aligned with PRINCE2 7?

## Options

- **A**: Nothing is wrong, since the metrics are on target and the work is within tolerance
- **B**: This is a management success and the project manager should formalise the command-and-control style across the team
- **C**: This signals a leadership shortfall: the culture lacks openness and trust, so the project manager should encourage challenge, listen to ideas, and rebuild psychological safety
- **D**: The project manager should escalate the silence to the project board as an issue requiring their decision

## Answer

C

## Explanation

On-target metrics mask a cultural problem: a high-performing project culture in PRINCE2 7 depends on openness, trust and the freedom to challenge, all of which are now missing. (A) is the tempting near-miss: tolerance and metrics look healthy, but the suppressed input is a real leadership risk to quality and improvement. (B) celebrating command-and-control entrenches the very behaviour undermining the culture. (D) this is the project manager's leadership responsibility to address directly, not a board decision.
