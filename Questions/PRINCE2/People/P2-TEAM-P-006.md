# Question

On a small museum-exhibition project, the Executive suggests saving cost by having one person act as both Senior User and Senior Supplier. The project manager is asked to advise. What is the BEST response?

## Options

- **A**: Agree, because combining roles on a small project always streamlines decisions
- **B**: Advise against it, because the user and supplier interests can conflict and combining them in one person undermines the balance of interests on the board
- **C**: Advise that the Project Manager should take both roles instead to keep the board small
- **D**: Agree, provided the combined role also takes on Project Assurance to save further cost

## Answer

B

## Explanation

PRINCE2 allows some roles to be combined, but the user and supplier interests can pull in opposite directions, so vesting both in one person removes the very tension the board exists to balance — making this combination unsuitable. (A) overgeneralises; not all roles can safely merge. (C) is wrong because the Project Manager cannot hold project board roles. (D) compounds the error by also collapsing the independent assurance function into the conflicted role.
