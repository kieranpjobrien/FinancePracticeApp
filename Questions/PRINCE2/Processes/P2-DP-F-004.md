# Question

Which of the following is one of the activities of the Directing a Project process?

## Options

- **A**: Authorise initiation
- **B**: Plan the next management stage
- **C**: Accept a work package
- **D**: Capture and examine issues and risks

## Answer

A

## Explanation

"Authorise initiation" is one of the five activities of Directing a Project performed by the project board. (B) Planning the next stage is part of Managing a Stage Boundary, a project manager activity. (C) Accepting a work package belongs to Managing Product Delivery and is done by the team. (D) Capturing and examining issues and risks is part of Controlling a Stage.
