# Question

What is the purpose of a checkpoint report in PRINCE2 7?

## Options

- **A**: To report stage-level progress from the project manager to the project board
- **B**: To report team-level progress from the team manager to the project manager at a frequency agreed in the work package
- **C**: To request authorisation for the next management stage
- **D**: To record lessons at the end of the project

## Answer

B

## Explanation

A checkpoint report is produced by the team manager to provide the project manager with progress information for a work package at the frequency agreed in that work package. (A) describes a highlight report. (C) is achieved through an end stage report and the board's decision. (D) describes the lessons section of the end project report.
