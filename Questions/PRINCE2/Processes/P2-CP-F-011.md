# Question

Which statement correctly describes the relationship between Closing a Project and Directing a Project?

## Options

- **A**: The project manager performs the closure activities and prepares the products, but the project board authorises the closure through the "authorise project closure" activity in Directing a Project
- **B**: The project board performs the closure activities, and the project manager merely files the records
- **C**: Closing a Project authorises its own closure without any project board involvement
- **D**: Closure can only be authorised by corporate, programme management or the customer, bypassing the project board

## Answer

A

## Explanation

The project manager carries out Closing a Project to prepare for the end, but the decision to close rests with the project board via "authorise project closure" within Directing a Project. (B) The board does not perform the closure activities themselves. (C) The board's authorisation is always required; the project cannot close itself. (D) The project board, not corporate or programme management, authorises closure, though it notifies them.
