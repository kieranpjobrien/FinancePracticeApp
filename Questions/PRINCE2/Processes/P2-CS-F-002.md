# Question

Which role is primarily responsible for the Controlling a Stage process?

## Options

- **A**: The project manager
- **B**: The project board
- **C**: The team manager
- **D**: Project assurance

## Answer

A

## Explanation

Controlling a Stage describes the work of the project manager during a delivery stage, covering the day-to-day management activities that keep the stage on track. (A) The project board operates Directing a Project and only becomes involved when escalation or a stage boundary occurs. (C) The team manager works within Managing Product Delivery to build products. (D) Project assurance supports and checks but does not own the process.
