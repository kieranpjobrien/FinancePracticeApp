# Question

The "authorise the project" activity within Directing a Project results in which key decision?

## Options

- **A**: Whether to commit resources to the initiation stage so the project can be planned
- **B**: Whether the project, as defined by the initiation documentation and business case, should proceed
- **C**: Whether the current management stage should continue to its scheduled end
- **D**: Whether the project's final products can be accepted and the project closed

## Answer

B

## Explanation

"Authorise the project" reviews the project initiation documentation, including the detailed business case and project plan, and decides whether the project as a whole is justified and should proceed beyond initiation. (A) is the "authorise initiation" decision, made earlier. (C) describes "authorise a stage or exception plan". (D) describes "authorise closure". The distinction is that authorising the project is the board's commitment to the whole project once it has been properly defined.
