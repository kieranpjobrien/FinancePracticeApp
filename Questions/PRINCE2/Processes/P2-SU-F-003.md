# Question

During Starting Up a Project, which two roles are appointed first?

## Options

- **A**: The Executive and the Project Manager
- **B**: The Senior User and the Senior Supplier
- **C**: The Project Manager and the Team Manager
- **D**: The Project Assurance and the Project Support roles

## Answer

A

## Explanation

The first action in Starting Up a Project is to appoint the Executive and the Project Manager, because someone must be accountable for the project (the Executive) and someone must run the start-up work day to day (the Project Manager). (B) The Senior User and Senior Supplier are appointed slightly later, when the full project management team is designed. (C) Team Managers are appointed during stages as needed, not in start-up. (D) Project Assurance and Project Support are defined when the team is designed, after the Executive and Project Manager.
