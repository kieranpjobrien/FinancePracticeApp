# Question

Which management product summarises how the project performed against the version of the project initiation documentation used to authorise it?

## Options

- **A**: End stage report
- **B**: Highlight report
- **C**: End project report
- **D**: Lessons report

## Answer

C

## Explanation

The end project report is produced during Closing a Project to review how the project performed against the PID baseline that authorised it. (A) The end stage report reviews a single stage at its boundary. (B) The highlight report is a routine progress update during a stage. (D) The lessons report captures lessons to pass on, not overall project performance against the PID.
