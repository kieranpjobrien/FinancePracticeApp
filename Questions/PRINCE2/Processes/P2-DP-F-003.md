# Question

At what point in the project lifecycle does the Directing a Project process begin and end?

## Options

- **A**: It begins at the start of Starting Up a Project and ends at the start of the initiation stage
- **B**: It begins on completion of Starting Up a Project and continues until the project is closed
- **C**: It begins when the project plan is approved and ends at the final stage boundary
- **D**: It runs only during the initiation stage and hands over to the project manager thereafter

## Answer

B

## Explanation

Directing a Project starts on completion of Starting Up a Project (triggered by the request to initiate a project) and runs until the project closes, spanning every management stage. (A) understates its duration; it continues well beyond the start of initiation. (C) omits the authorisation of initiation, which happens before the project plan exists. (D) is wrong because the board directs throughout, not only during initiation.
