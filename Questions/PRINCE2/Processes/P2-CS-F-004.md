# Question

Which management product does the project manager use in Controlling a Stage to report stage progress to the project board?

## Options

- **A**: Checkpoint report
- **B**: End stage report
- **C**: Highlight report
- **D**: Exception report

## Answer

C

## Explanation

The highlight report is the project manager's regular, time-driven progress report from the project manager to the project board during a stage. (A) The checkpoint report flows the other way, from team manager to project manager. (B) The end stage report is produced at a stage boundary in Managing a Stage Boundary, not as a regular update. (D) The exception report is event-driven, raised only when tolerance is forecast to be breached.
