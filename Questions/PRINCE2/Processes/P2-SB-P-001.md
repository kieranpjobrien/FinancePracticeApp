# Question

On a hospital IT migration project, the current management stage is two weeks from completion and on track. The project manager has drafted the end stage report. What should the project manager do next within Managing a Stage Boundary?

## Options

- **A**: Begin the work of the next stage immediately, since the current stage is on track
- **B**: Produce a detailed plan for the next management stage and seek the project board's authorisation
- **C**: Close the project because the stage is nearly finished
- **D**: Ask the team managers to authorise the next stage at their checkpoint meeting

## Answer

B

## Explanation

Within Managing a Stage Boundary the project manager prepares the next stage plan alongside the end stage report and submits both for the board's go/no-go decision. (A) bypasses authorisation, breaching the manage-by-stages principle; (C) confuses a stage boundary with project closure; (D) wrongly delegates an authorisation that only the project board can give.
