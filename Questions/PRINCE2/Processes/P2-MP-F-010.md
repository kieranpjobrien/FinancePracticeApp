# Question

The work package is the key interface between which two processes in PRINCE2 7?

## Options

- **A**: Starting up a Project and Initiating a Project
- **B**: Controlling a Stage and Managing Product Delivery
- **C**: Directing a Project and Managing a Stage Boundary
- **D**: Initiating a Project and Closing a Project

## Answer

B

## Explanation

Work packages are authorised within Controlling a Stage (by the project manager) and accepted, executed and delivered within Managing Product Delivery (by the team manager), making the work package the formal interface between these two processes. (A), (C) and (D) pair processes that are connected by other products such as the project brief, project mandate or PID, not by the work package.
