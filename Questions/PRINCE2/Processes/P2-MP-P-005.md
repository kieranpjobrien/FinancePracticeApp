# Question

A team manager leading a road-resurfacing crew has accepted a sizeable work package covering several kilometres of carriageway across multiple sites. To organise the work effectively, what is the BEST thing for the team manager to produce?

## Options

- **A**: A new stage plan to replace the project manager's existing one
- **B**: A team plan detailing how the team will deliver the products in the work package
- **C**: A revised project plan covering the remainder of the project
- **D**: An exception plan, because the work package is large

## Answer

B

## Explanation

When the work warrants it, the team manager produces a team plan to schedule and organise delivery of the work package's products. (A) the stage plan belongs to the project manager and is not replaced by the team. (C) the project plan covers the whole project and is maintained by the project manager. (D) an exception plan is only produced when tolerances have been or are forecast to be exceeded, which is not the case here.
