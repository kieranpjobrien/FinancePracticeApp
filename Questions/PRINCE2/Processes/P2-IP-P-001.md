# Question

A hospital is initiating a project to roll out a new electronic patient-records system. Keen to show progress, the project manager begins assigning build work to a supplier team before the risk, quality, change and communication management approaches have been drafted. What should the project manager do?

## Options

- **A**: Continue assigning the build work, as creating the approaches can wait until the first delivery stage
- **B**: Complete the four management approaches as part of establishing solid foundations before delivery work is authorised
- **C**: Ask the project board to approve the PID immediately so build work is covered
- **D**: Delegate creation of the four approaches to the supplier team manager

## Answer

B

## Explanation

Initiating a Project establishes solid foundations, which includes creating the risk, quality, change and communication management approaches before the project commits to delivery. (A) Starting build work prematurely bypasses the foundations and the board's authorisation to proceed. (C) The PID cannot be sensibly approved while key components — the four approaches — are still missing. (D) The project manager owns creation of the approaches; they are not delegated to a supplier.
