# Question

During a software migration, a minor defect is found that will cost two days to fix. The stage still has eight days of time tolerance and ample cost tolerance remaining. What should the project manager do?

## Options

- **A**: Take corrective action within the stage, log the issue, and continue managing the stage
- **B**: Raise an exception report so the project board can decide
- **C**: Escalate to corporate management for a decision
- **D**: Halt all work packages until the project board meets

## Answer

A

## Explanation

Because the fix stays within the delegated stage tolerances, the project manager simply takes corrective action, records the issue and continues day-to-day control under Controlling a Stage. (B) and (C) Escalation is reserved for forecast tolerance breaches, which is not the case here. (D) Stopping all work for a minor, in-tolerance issue is a disproportionate over-reaction that disrupts delivery unnecessarily.
