# Question

In Controlling a Stage, what is a work package used for?

## Options

- **A**: To set out the information needed to create one or more products and to authorise and control the work given to a team
- **B**: To record the overall business justification for the project
- **C**: To capture lessons identified during the stage for future projects
- **D**: To request that the project board approves an exception to stage tolerance

## Answer

A

## Explanation

A work package is the set of information about one or more required products that the project manager uses to authorise and control the work passed to a team manager or team. (B) describes the business case. (C) describes the lessons log or lessons report. (D) describes an exception report, a separate management product raised when tolerance is forecast to be exceeded.
