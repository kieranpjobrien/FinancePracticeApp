# Question

On a wind-farm construction project, the current stage is forecast to exceed its cost tolerance by 18%. The project manager has raised an exception report and the project board has asked for an exception plan. Which process produces that plan?

## Options

- **A**: Controlling a Stage
- **B**: Managing a Stage Boundary
- **C**: Closing a Project
- **D**: Starting Up a Project

## Answer

B

## Explanation

When the board requests an exception plan, it is produced within Managing a Stage Boundary, which is specifically triggered for this purpose as well as at normal stage ends. (A) handles in-tolerance day-to-day control and raising the exception, not producing the exception plan; (C) and (D) belong to project closure and start-up respectively.
