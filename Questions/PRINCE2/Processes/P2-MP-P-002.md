# Question

A team manager on a wind-farm construction project has agreed to provide weekly checkpoint reports. Two weeks in, the project manager has received nothing and is unsure how the foundations work is progressing. What is the BEST response?

## Options

- **A**: The project manager should wait until the work package is delivered before expecting any update
- **B**: The team manager should issue a highlight report instead of a checkpoint report
- **C**: The team manager should provide the agreed weekly checkpoint reports so the project manager has the progress information needed to control the stage
- **D**: The project manager should escalate immediately to the project board for an exception assessment

## Answer

C

## Explanation

Checkpoint reports are the team manager's agreed mechanism for feeding work-package progress to the project manager at the frequency set in the work package; the team manager simply needs to honour that commitment. (A) waiting until delivery defeats the purpose of progress monitoring. (B) highlight reports are produced by the project manager for the board, not by the team manager. (D) escalation to the board is premature when no tolerance is forecast to be breached.
