# Question

A software migration project has reached the end of its second management stage. The project manager has prepared the end stage report and the next stage plan. What should happen next in line with Directing a Project?

## Options

- **A**: The project manager approves the next stage plan and continues without involving the board
- **B**: The project board reviews the end stage report and next stage plan and decides whether to authorise the next stage
- **C**: The board issues ad hoc direction because a stage boundary has been reached
- **D**: The project is closed because a management stage has finished

## Answer

B

## Explanation

At a stage boundary the board performs "authorise a stage or exception plan", using the end stage report and next stage plan to decide whether the next stage should proceed. (A) usurps a board decision the project manager is not authorised to make. (C) misclassifies a planned stage boundary as an ad hoc event. (D) confuses the end of a stage with the end of the project; closure occurs only at the final stage.
