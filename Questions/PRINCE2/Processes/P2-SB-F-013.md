# Question

During which management stage is Managing a Stage Boundary normally NOT performed at the stage's end?

## Options

- **A**: The initiation stage
- **B**: The first delivery stage
- **C**: The final management stage
- **D**: Any stage that contains a risk

## Answer

C

## Explanation

Managing a Stage Boundary is run near the end of each stage except the final one, because at the end of the last stage the project moves into Closing a Project rather than planning a further stage. (A) is followed by a stage boundary into the first delivery stage; (B) is also followed by a boundary; (D) is irrelevant because the presence of risk does not determine whether the process runs.
