# Question

Which management role is responsible for the Directing a Project process?

## Options

- **A**: The project manager
- **B**: The team manager
- **C**: The project board
- **D**: Project assurance

## Answer

C

## Explanation

Directing a Project is the project board's process; it covers the activities the board carries out to direct and control the project throughout its life. (A) The project manager runs Controlling a Stage and Managing a Stage Boundary, not Directing a Project. (B) The team manager runs Managing Product Delivery. (D) Project assurance supports the board independently but does not own the process or make its decisions.
