# Question

A work package is to be delivered by an external contractor under a commercial agreement. The contractor's team manager notices that the checkpoint reporting frequency in the work package conflicts with the reporting cycle written into the contract. What should happen before the work begins?

## Options

- **A**: The team manager should ignore the work package frequency and follow only the contract
- **B**: The team manager should agree the reporting arrangements with the project manager so the work package and the commercial agreement are reconciled before acceptance
- **C**: The project manager should rewrite the contract without involving the supplier
- **D**: The team manager should accept the work package and resolve the conflict later if it causes problems

## Answer

B

## Explanation

Accept a work package is the point at which constraints, including reporting arrangements and any commercial agreement, are confirmed as workable; conflicts must be resolved with the project manager before the team manager commits. (A) unilaterally following only the contract ignores the project manager's information needs. (C) changing a contract without the supplier is neither practical nor appropriate. (D) accepting a known conflict and hoping it does not bite contradicts the principle of agreed, achievable acceptance.
