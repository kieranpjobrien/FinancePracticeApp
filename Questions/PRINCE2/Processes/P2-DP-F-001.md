# Question

What is the purpose of the Directing a Project process in PRINCE2 7?

## Options

- **A**: To enable the project board to be accountable for the project's success by making key decisions and exercising overall control
- **B**: To enable the project manager to plan and control each management stage on a day-to-day basis
- **C**: To establish whether the project is viable and worthwhile before any work is authorised
- **D**: To deliver the project's specialist products to the required quality standards

## Answer

A

## Explanation

The Directing a Project process is owned by the project board and exists so the board can be accountable for the project by making the key decisions and exercising overall control, while delegating day-to-day management to the project manager. (B) describes Controlling a Stage, the project manager's day-to-day process. (C) describes the purpose of Starting Up a Project. (D) is delivery work done in Managing Product Delivery by the team.
