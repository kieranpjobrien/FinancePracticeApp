# Question

The plan for which period is produced during Starting Up a Project?

## Options

- **A**: The Project Plan for the whole project
- **B**: The Stage Plan for the first delivery stage
- **C**: The Stage Plan for the initiation stage
- **D**: An Exception Plan to recover the project to tolerance

## Answer

C

## Explanation

A key output of Starting Up a Project is the Stage Plan for the initiation stage, so the project board can approve the resources needed to carry out Initiating a Project. (A) The Project Plan for the whole project is produced during initiation, not start-up. (B) The plan for the first delivery stage is created near the end of initiation, during Managing a Stage Boundary. (D) An Exception Plan is only created when a stage or project is forecast to exceed tolerance, which cannot apply before the project has even begun.
