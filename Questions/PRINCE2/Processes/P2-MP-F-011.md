# Question

What should the team manager do if, while executing a work package, the work is forecast to exceed the agreed work package tolerances?

## Options

- **A**: Continue working and report the overrun only in the next checkpoint report
- **B**: Adjust the team plan and absorb the overrun without informing anyone
- **C**: Raise the issue to the project manager rather than taking corrective action that breaches the agreed tolerances
- **D**: Escalate directly to the project board for a decision

## Answer

C

## Explanation

Work package tolerances define the team manager's delegated authority; if they are forecast to be exceeded, the team manager must notify the project manager, who decides on corrective action. (A) waiting risks the breach being acted on too late and outside authority. (B) silently absorbing a breach undermines control. (D) the team manager escalates to the project manager, who in turn escalates to the board only if stage tolerances are threatened.
