# Question

What is the team manager's main objective during deliver a work package?

## Options

- **A**: To notify the project manager that the work package has been completed and approved, and hand the products back
- **B**: To obtain authorisation from the project board to start the next work package
- **C**: To create the project initiation documentation for the completed products
- **D**: To raise an exception report because tolerances have been exceeded

## Answer

A

## Explanation

In deliver a work package the team manager confirms the products have been completed and gained the required approvals, then notifies the project manager that the work package is finished. (B) authorising further work is the project manager's responsibility, not the board's at this level. (C) the PID is produced in Initiating a Project. (D) raising an exception report is only triggered when tolerances are forecast to be exceeded, not on normal completion.
