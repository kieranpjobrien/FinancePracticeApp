# Question

What triggers the Starting Up a Project process?

## Options

- **A**: The Project Brief
- **B**: The Project Initiation Documentation
- **C**: The project mandate
- **D**: An approved Stage Plan

## Answer

C

## Explanation

The project mandate is the trigger for Starting Up a Project; it is supplied by the commissioning organisation (corporate, programme management or the customer) and provides the terms of reference for the prospective project. (A) The Project Brief is an output of Starting Up, assembled during the process, not the trigger. (B) The Project Initiation Documentation is produced later, in Initiating a Project. (D) A Stage Plan is created for each management stage during initiation and later stages, not before start-up.
