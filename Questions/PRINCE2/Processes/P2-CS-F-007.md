# Question

What is the main purpose of the "review the stage status" activity in Controlling a Stage?

## Options

- **A**: To decide whether the next stage should be authorised
- **B**: To compare actual progress against the stage plan so the project manager can decide what action is needed
- **C**: To allow the project board to confirm the business case remains viable
- **D**: To hand over completed products to the customer at the end of the project

## Answer

B

## Explanation

Reviewing the stage status gives the project manager a regular check of actual progress against the stage plan, drawing on checkpoint reports, registers and logs, so that timely decisions and corrective actions can be made. (A) Authorising the next stage is the project board's decision in Directing a Project. (C) Confirming viability sits with the project board. (D) Handover occurs in Closing a Project.
