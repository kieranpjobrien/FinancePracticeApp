# Question

A team manager reports that a work package will overrun its own agreed tolerance, but the slippage is small enough that the overall stage will still finish within stage tolerance. What is the BEST response by the project manager?

## Options

- **A**: Raise an exception report to the project board because a tolerance has been breached
- **B**: Tell the team manager to absorb the overrun without informing anyone
- **C**: Replace the team manager for failing to deliver within tolerance
- **D**: Review the situation, take corrective action and adjust the work package as needed, keeping the project board informed via the highlight report since the stage remains in tolerance

## Answer

D

## Explanation

A work package tolerance breach is escalated by the team manager to the project manager, who handles it within the stage; because stage tolerance is not threatened, no exception report to the board is needed and progress is communicated through the normal highlight report. (A) Exception reports to the board are triggered by stage or project tolerance breaches, not a single work package. (B) Suppressing the deviation removes the project manager's control. (C) Replacing the team manager is a disproportionate personnel reaction, not a project control.
