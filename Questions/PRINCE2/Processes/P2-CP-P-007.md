# Question

Near the end of a manufacturing plant's automation project, the project manager identifies a recurring integration problem that the operations team will keep hitting and a separate insight about supplier onboarding that would help future projects. Both are recorded at closure. Which statement reflects the correct PRINCE2 7 treatment?

## Options

- **A**: Both items go into the lessons report so future projects benefit
- **B**: Both items go into the follow-on action recommendations for the operations team
- **C**: The integration problem becomes a follow-on action recommendation for operations, while the supplier-onboarding insight goes into the lessons report
- **D**: Neither item is recorded, as the project is closing and ownership has transferred

## Answer

C

## Explanation

Follow-on action recommendations carry forward outstanding work and issues affecting this project's products to those who will support them, while the lessons report carries forward knowledge to benefit future projects. The ongoing integration problem is an operational follow-on, whereas the supplier-onboarding insight is a lesson. (A) and (B) wrongly funnel both items into a single product. (D) Failing to record either breaches the learn-from-experience principle and abandons a known operational issue.
