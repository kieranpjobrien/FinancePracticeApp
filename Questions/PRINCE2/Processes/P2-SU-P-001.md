# Question

A regional water utility's senior leadership has issued a one-page document instructing that a project be set up to replace ageing pumping stations, naming a likely sponsor and a target completion date. No Executive or Project Manager has yet been named. What should happen next?

## Options

- **A**: Starting Up a Project should be triggered, beginning with appointing the Executive and the Project Manager
- **B**: The prospective Project Manager should begin preparing the Project Initiation Documentation
- **C**: The leadership team should convene a project board to authorise the project
- **D**: A delivery supplier should be contracted so design work can begin immediately

## Answer

A

## Explanation

The one-page instruction is the project mandate, which triggers Starting Up a Project; the first step is appointing the Executive and Project Manager so the start-up work has accountable ownership. (B) The PID belongs to Initiating a Project, which cannot begin until start-up has produced a Project Brief and the board has authorised initiation. (C) The project cannot be authorised before it has even been shown to be viable through start-up. (D) Contracting a supplier and starting design pre-empts the project approach decision and the board's authorisation.
