# Question

The initiation stage of a city-wide cycleway scheme is complete and the project initiation documentation is ready for the project board. One board member proposes approving the first delivery stage plan at the same meeting to save time, before the board has decided whether the project as a whole should proceed. What is the BEST response?

## Options

- **A**: Approve both the project and the first delivery stage in a single combined decision to save a meeting
- **B**: Have the board first authorise the project, and then authorise the first delivery stage, recognising these are two separate decisions
- **C**: Authorise the first delivery stage now and defer the decision to authorise the project until later
- **D**: Skip authorising the project, since completing initiation already commits the board to proceed

## Answer

B

## Explanation

"Authorise the project" and "authorise a stage or exception plan" are distinct decisions: the board must first commit to the project as defined by the project initiation documentation, then separately approve the first delivery stage plan. (A) is the near-miss: although both can be considered at one meeting, they remain two decisions and the project commitment logically precedes the stage approval. (C) reverses the proper order. (D) wrongly treats completing initiation as automatic project approval.
