# Question

After Managing a Stage Boundary produces the end stage report and next stage plan, which process does the project board use to authorise the next stage?

## Options

- **A**: Controlling a Stage
- **B**: Directing a Project
- **C**: Managing Product Delivery
- **D**: Initiating a Project

## Answer

B

## Explanation

The request for authorisation goes to the project board, which makes the go/no-go decision within the Directing a Project process (the "authorise a stage or exception plan" activity). (A) is the project manager's day-to-day control process; (C) governs delegated work packages; (D) sets up the project rather than authorising subsequent stages.
