# Question

On a hospital IT rollout, the project manager forecasts that the current stage will finish three weeks late, breaching the agreed time tolerance. What should the project manager do?

## Options

- **A**: Quietly reduce the scope of the remaining work packages to claw back the time
- **B**: Continue and explain the slippage in the next routine highlight report
- **C**: Produce an exception report for the project board describing the forecast breach, its impact, options and a recommendation
- **D**: Instruct the team manager to work overtime until the schedule is recovered

## Answer

C

## Explanation

A forecast breach of stage tolerance must be escalated to the project board, and the correct PRINCE2 mechanism is the exception report setting out the situation, impact, options and a recommendation. (A) Cutting scope without board agreement breaches scope tolerance and bypasses governance. (B) The highlight report is for in-tolerance progress, not escalation. (D) Mandating overtime is a unilateral fix that may still breach tolerance and ignores the manage by exception principle.
