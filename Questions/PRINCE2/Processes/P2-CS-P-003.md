# Question

A project manager on a bridge-refurbishment project is ready to hand a discrete package of inspection work to a specialist team. What is the most appropriate action within Controlling a Stage?

## Options

- **A**: Authorise a work package agreeing the products, tolerances, constraints and reporting with the team manager
- **B**: Send the team a copy of the project plan and ask them to start
- **C**: Ask the project board to brief the team directly
- **D**: Wait until the stage boundary before assigning any work

## Answer

A

## Explanation

Assigning work in Controlling a Stage is done by authorising a work package, which records the products required and the agreed tolerances, constraints, quality criteria and reporting arrangements with the team manager. (B) Handing over the project plan without a work package gives no agreed controls or acceptance criteria. (C) The project board directs, it does not brief teams. (D) Work is assigned during the stage, not deferred to its boundary.
