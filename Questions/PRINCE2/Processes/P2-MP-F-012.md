# Question

How does a checkpoint report differ from a highlight report in PRINCE2 7?

## Options

- **A**: A checkpoint report goes from team manager to project manager; a highlight report goes from project manager to project board
- **B**: A checkpoint report goes from project manager to project board; a highlight report goes from team manager to project manager
- **C**: Both reports are produced by the project manager for the project board
- **D**: Both reports are produced by the team manager for the project board

## Answer

A

## Explanation

A checkpoint report communicates team-level progress on a work package from the team manager up to the project manager, whereas a highlight report communicates stage-level progress from the project manager up to the project board. (B) reverses the senders. (C) and (D) wrongly assign both reports to a single role and recipient; the two reports operate at different levels of the management hierarchy.
