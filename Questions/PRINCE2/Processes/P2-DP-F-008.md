# Question

Which situation would typically trigger the project board's "give ad hoc direction" activity?

## Options

- **A**: The project manager reaches the planned end of a management stage and submits an end stage report
- **B**: The project manager raises an exception report because a stage is forecast to exceed its tolerances
- **C**: A team member completes a work package and updates the configuration records
- **D**: The project board needs to authorise the initiation stage for the first time

## Answer

B

## Explanation

"Give ad hoc direction" covers the board's responses between formal decision points, including reacting to an exception report when a forecast breach of tolerance is escalated, as well as answering requests for advice and acting on issues. (A) triggers "authorise a stage or exception plan" at the end-stage assessment. (C) is routine delivery work that does not need board direction. (D) triggers "authorise initiation", a separate activity.
