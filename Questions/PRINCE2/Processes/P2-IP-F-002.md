# Question

In PRINCE2 7, what is the main management product assembled during the Initiating a Project process?

## Options

- **A**: The Project Brief
- **B**: The Project Mandate
- **C**: The End Project Report
- **D**: The Project Initiation Documentation (PID)

## Answer

D

## Explanation

Initiating a Project assembles the Project Initiation Documentation, the collection of the baselined products that defines the project and forms the basis for its management and for the board's decision to authorise delivery. (A) The Project Brief is created during Starting Up a Project and is an input to initiation. (B) The Project Mandate is the external trigger that precedes Starting Up. (C) The End Project Report is produced during Closing a Project.
