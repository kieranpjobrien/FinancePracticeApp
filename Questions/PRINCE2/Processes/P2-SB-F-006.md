# Question

What happens to the project plan during Managing a Stage Boundary?

## Options

- **A**: It is created for the first time at the end of each stage
- **B**: It is archived because each stage plan supersedes it
- **C**: It is updated to reflect actual progress and the latest forecasts
- **D**: It is replaced by the next stage plan

## Answer

C

## Explanation

The project plan is updated during Managing a Stage Boundary with actuals from the completed stage and revised forecasts for the remainder of the project, keeping the board's whole-project view current. (A) is wrong because the project plan is first created in Initiating a Project; (B) and (D) are wrong because the project plan coexists with and is informed by stage plans rather than being archived or replaced.
