# Question

Before authorising a work package in Controlling a Stage, what should the project manager agree with the team manager?

## Options

- **A**: The tolerances, reporting requirements and acceptance criteria for the work
- **B**: The contents of the end stage report
- **C**: The membership of the project board
- **D**: The closure recommendation for the project

## Answer

A

## Explanation

Authorising a work package involves agreeing what is to be produced, along with the constraints such as tolerances, the reporting (for example checkpoint frequency) and the quality and acceptance criteria, so the team manager understands exactly what is required. (B) and (D) belong to stage boundary and closure activities. (C) Project board membership is set during start-up and initiation, not when authorising work.
