# Question

How does the lessons report produced at closure differ from the follow-on action recommendations?

## Options

- **A**: The lessons report passes on knowledge to benefit future projects, while follow-on action recommendations pass on unfinished work and open items for this project's products
- **B**: The lessons report lists outstanding risks, while follow-on action recommendations capture what went well
- **C**: Both documents serve the identical purpose and are interchangeable
- **D**: The lessons report is for the project board only, while follow-on action recommendations are internal to the project team

## Answer

A

## Explanation

The lessons report captures lessons learned during the project so future projects can benefit, whereas follow-on action recommendations hand over unfinished work, open risks and open issues relating to this project's products. (B) reverses the two purposes. (C) They are distinct management products. (D) Distribution does not define the difference; their content and purpose do.
