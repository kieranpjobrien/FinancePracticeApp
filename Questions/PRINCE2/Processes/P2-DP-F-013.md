# Question

Which statement about the project board in the Directing a Project process is correct?

## Options

- **A**: The board produces the detailed stage plans that the project manager then executes
- **B**: The board is accountable for the project and provides unified direction, having delegated day-to-day management to the project manager
- **C**: The board attends daily checkpoint meetings to maintain close control of the team
- **D**: The board hands all of its decision-making authority to project assurance once initiation is authorised

## Answer

B

## Explanation

The board holds overall accountability for the project and gives it unified direction, while delegating day-to-day management to the project manager and intervening only by exception. (A) is wrong because the project manager produces stage plans in Managing a Stage Boundary. (C) contradicts manage by exception. (D) is wrong because project assurance advises the board independently but the board retains its decision-making authority throughout.
