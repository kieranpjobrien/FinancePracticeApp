# Question

Which statement best describes the exception report produced in Controlling a Stage?

## Options

- **A**: A regular progress report sent to the project board at an agreed frequency
- **B**: A report from the team manager describing progress on a work package
- **C**: A report prepared by the project manager for the project board describing a forecast tolerance breach, its impact and options
- **D**: A report that closes the current stage and seeks authorisation for the next

## Answer

C

## Explanation

An exception report is created by the project manager for the project board when a stage (or the project) is forecast to exceed its tolerances; it describes the exception, its impact, the available options and a recommendation. (A) describes the highlight report. (B) describes the checkpoint report. (D) describes the end stage report from Managing a Stage Boundary.
