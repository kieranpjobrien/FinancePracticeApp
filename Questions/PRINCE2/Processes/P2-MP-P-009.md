# Question

While executing a work package to develop a customer feedback form for an airline, a team manager realises she could add an attractive analytics dashboard that is not in the work package. She has spare time within tolerance. What should she do?

## Options

- **A**: Build the dashboard anyway, since there is time available and it would impress the customer
- **B**: Deliver only the products defined in the work package and, if the dashboard seems worthwhile, raise it to the project manager rather than adding unrequested scope
- **C**: Quietly add the dashboard and mention it in the next checkpoint report
- **D**: Reduce the form's quality criteria to free up budget for the dashboard

## Answer

B

## Explanation

The team manager must deliver what the work package specifies and avoid gold-plating; an idea outside the agreed scope should be raised to the project manager for proper consideration through change control. (A) and (C) adding unrequested products is gold-plating and bypasses the project manager's authority over scope, even when tolerance allows time. (D) lowering agreed quality criteria to fund extra work breaches the work package's acceptance criteria. Staying within the agreed scope protects the project's controlled delivery.
