# Question

A team manager on a museum exhibition project has finished building an interactive display and confirmed it has passed all the quality checks and gained the approvals specified in the work package. What should the team manager do?

## Options

- **A**: Begin the next work package straight away without telling the project manager
- **B**: Notify the project manager that the work package is complete and hand back the approved products
- **C**: Archive the display and close the project
- **D**: Update the business case to record the benefits the display will deliver

## Answer

B

## Explanation

Deliver a work package concludes with the team manager confirming completion and required approvals, then notifying the project manager that the work package is finished. (A) starting new work without authorisation breaks the controlled link with the project manager. (C) closing the project is the project manager's activity in Closing a Project, not the team manager's. (D) maintaining the business case is the executive's responsibility, not part of delivering a work package.
