# Question

While initiating a renewable-energy installation project, the project manager has just completed the project plan, which reveals costs noticeably higher than the rough figures in the outline business case. What is the BEST response?

## Options

- **A**: Leave the outline business case unchanged so the board is not alarmed by the higher figures
- **B**: Refine the detailed business case using the project plan's costs and timescales so the justification reflects reality
- **C**: Reduce the scope informally to bring costs back to the original estimate
- **D**: Escalate the cost increase to the board as an exception before the PID is assembled

## Answer

B

## Explanation

Refining the business case in light of the project plan's costs and timescales is exactly what Initiating a Project requires, so the board can judge continued viability on accurate information. (A) Concealing the figures undermines business justification. (C) Quietly cutting scope changes the project without board agreement. (D) There is no baseline yet to exceed, so this is not an exception; the project manager simply completes initiation and lets the board decide when authorising the project.
