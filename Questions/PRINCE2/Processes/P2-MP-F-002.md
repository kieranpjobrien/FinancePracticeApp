# Question

Which role is primarily responsible for the activities within the Managing Product Delivery process?

## Options

- **A**: The project board
- **B**: The project manager
- **C**: The team manager
- **D**: The project assurance function

## Answer

C

## Explanation

Managing Product Delivery is the process owned by the team manager (or by team members reporting directly to the project manager where no team manager is appointed). (A) the project board operates at the direction level. (B) the project manager owns Controlling a Stage and interfaces with the team manager via work packages. (D) project assurance supports the board but does not run the process.
