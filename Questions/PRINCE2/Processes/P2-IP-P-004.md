# Question

During initiation of a regional rail-signalling upgrade, the project manager is setting up project controls and must decide how much latitude each delivery stage will have before the project board needs to be involved. Which action reflects correct PRINCE2 practice?

## Options

- **A**: Agree tolerances for the performance targets with the project board and record them as part of the project controls
- **B**: Allow each team manager to set their own tolerances during delivery
- **C**: Leave tolerances undefined so any deviation is escalated to the board
- **D**: Set tolerances only for time and cost, ignoring the other performance targets

## Answer

A

## Explanation

Establishing tolerances for the seven performance targets — agreed with the project board and built into the project controls during initiation — defines the limits of delegated authority and when escalation is needed. (B) Tolerances are set by the board and delegated downward, not chosen by team managers. (C) Undefined tolerances would force every minor deviation to the board, defeating management by exception. (D) Tolerances should consider all relevant performance targets, not just time and cost.
