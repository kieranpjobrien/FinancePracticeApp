# Question

In the Directing a Project process, how is the project board expected to operate in relation to day-to-day work?

## Options

- **A**: By directing every work package and reviewing each daily log entry
- **B**: By managing by exception, leaving day-to-day management to the project manager
- **C**: By attending all team checkpoint meetings to monitor progress
- **D**: By approving each product description before any product is built

## Answer

B

## Explanation

A core idea of Directing a Project is that the board manages by exception: it sets tolerances and delegates day-to-day management to the project manager, becoming involved only at defined decision points or when tolerances are forecast to be exceeded. (A), (C) and (D) all describe the board immersing itself in day-to-day management, which contradicts manage by exception and the board's directing role.
