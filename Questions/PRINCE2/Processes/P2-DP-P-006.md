# Question

After an exception report on an offshore wind survey project, the project board asked the project manager to prepare an exception plan to recover the current stage. The project manager has now finished it. Which action correctly reflects Directing a Project?

## Options

- **A**: The project manager begins executing the exception plan as soon as it is written, since the board already requested it
- **B**: The project board reviews and authorises the exception plan under "authorise a stage or exception plan" before work continues against it
- **C**: The project manager folds the exception plan into the next stage plan and presents both at the end stage assessment
- **D**: The board treats the finished exception plan as ad hoc direction and no further authorisation is needed

## Answer

B

## Explanation

Although the board requested the exception plan as ad hoc direction, the plan itself must still be formally approved through "authorise a stage or exception plan" before the project manager works to it. (A) confuses the request to prepare a plan with authorisation to execute it. (C) wrongly defers a live recovery to a future boundary and merges two distinct plans. (D) is the tempting near-miss: requesting the plan was ad hoc direction, but authorising it is a separate, explicit board decision.
