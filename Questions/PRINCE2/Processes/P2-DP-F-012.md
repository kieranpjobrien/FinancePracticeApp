# Question

How does "authorise the project" differ from "authorise a stage or exception plan"?

## Options

- **A**: Authorising the project commits to the whole project once, whereas authorising a stage commits to the next management stage's work at each boundary
- **B**: Authorising the project is done by the project manager, whereas authorising a stage is done by the project board
- **C**: Authorising the project happens at every stage boundary, whereas authorising a stage happens only once
- **D**: Both decisions are identical and simply use different names for the same end-stage assessment

## Answer

A

## Explanation

"Authorise the project" is a one-off commitment to the project as a whole, made after initiation; "authorise a stage or exception plan" is a repeated commitment to each subsequent stage's plan at its boundary. (B) is wrong because both are project board decisions. (C) reverses the frequency of the two activities. (D) is wrong because they are distinct decisions with different inputs and timing.
