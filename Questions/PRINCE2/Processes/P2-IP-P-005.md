# Question

A university has finished assembling the Project Initiation Documentation for a new student-services platform. The project manager is eager to start the first delivery stage the next morning. What should happen before delivery begins?

## Options

- **A**: The project manager authorises the start of the delivery stage, since the PID is complete
- **B**: Project assurance signs off the PID on the board's behalf
- **C**: The supplier confirms it is ready, which is sufficient to begin
- **D**: The project board reviews the PID and decides whether to authorise the project through Directing a Project

## Answer

D

## Explanation

The completed PID is submitted to the project board, which decides whether to authorise the project — a Directing a Project responsibility — before any delivery stage starts. (A) The project manager cannot authorise the project; that authority rests with the board. (B) Assurance advises and reviews but does not authorise on the board's behalf. (C) Supplier readiness does not substitute for the board's go/no-go decision.
