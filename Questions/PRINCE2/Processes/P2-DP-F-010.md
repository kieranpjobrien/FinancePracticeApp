# Question

Before the project board performs "authorise initiation", which management product does it primarily use to make the decision?

## Options

- **A**: The project initiation documentation
- **B**: The project brief and the outline business case
- **C**: The end project report
- **D**: The benefits management approach

## Answer

B

## Explanation

"Authorise initiation" is based on the project brief, which contains the outline business case produced during Starting Up a Project, allowing the board to decide whether it is worth investing in initiation. (A) The project initiation documentation does not yet exist; it is produced during the initiation stage and used later to authorise the project. (C) The end project report is a closure product. (D) The benefits management approach is created during initiation, after this decision.
