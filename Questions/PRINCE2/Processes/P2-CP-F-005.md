# Question

In Closing a Project, who recommends to the project board that the project should be closed?

## Options

- **A**: The team manager
- **B**: The project manager
- **C**: Project assurance
- **D**: The change authority

## Answer

B

## Explanation

The project manager carries out the Closing a Project activities and then recommends project closure to the project board, which makes the actual decision to authorise closure in Directing a Project. (A) The team manager delivers products, not closure recommendations. (C) Project assurance advises but does not own this recommendation. (D) The change authority handles changes within delegated limits, not project closure.
