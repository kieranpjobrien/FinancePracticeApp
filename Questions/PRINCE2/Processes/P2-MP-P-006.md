# Question

Midway through a pharmaceutical packaging line upgrade, a supplier delay means the team manager now forecasts the work package will finish nine days late, breaching its agreed time tolerance. The cost tolerance remains fine. What should the team manager do?

## Options

- **A**: Raise the forecast tolerance breach to the project manager and let the project manager decide on corrective action
- **B**: Use the unspent cost tolerance to fund overtime and recover the time without informing the project manager
- **C**: Produce an exception report and send it directly to the project board
- **D**: Continue and simply note the late finish in the final checkpoint report

## Answer

A

## Explanation

A forecast breach of work package tolerance is beyond the team manager's delegated authority, so the correct step is to notify the project manager, who then decides how to respond. (B) tolerances are set per target; the team manager cannot trade cost tolerance for time and must not act outside authority unannounced. (C) producing an exception report and going to the board is the project manager's escalation route for stage-level breaches, not the team manager's. (D) merely recording the breach after the fact removes any chance of timely corrective action.
