# Question

Which statement about the team plan is correct in PRINCE2 7?

## Options

- **A**: A team plan must always be produced for every work package
- **B**: The team plan is produced by the project manager and handed to the team manager
- **C**: The team plan is optional and is produced by the team manager to plan the execution of one or more work packages
- **D**: The team plan replaces the stage plan once specialist work begins

## Answer

C

## Explanation

A team plan is an optional management product, created by the team manager to plan the work needed to deliver the products in a work package; its formality and even existence depend on the situation. (A) it is not mandatory. (B) the team manager, not the project manager, produces it. (D) it supports, but never replaces, the stage plan the project manager uses to control the stage.
