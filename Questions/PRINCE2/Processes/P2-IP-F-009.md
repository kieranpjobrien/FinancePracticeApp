# Question

In PRINCE2 7, what is the main aim of setting up the project controls during Initiating a Project?

## Options

- **A**: To ensure the project can be governed by establishing decision points, tolerances and reporting arrangements
- **B**: To allocate detailed work packages to the delivery teams
- **C**: To define the acceptance criteria for each of the project's products
- **D**: To record lessons from previous projects for future reference

## Answer

A

## Explanation

Project controls give the project board and project manager the means to govern the project — establishing decision points, the levels of authority, tolerances for the seven performance targets, and how progress will be monitored and reported. (B) Work packages are assigned later in Controlling a Stage. (C) Acceptance criteria sit in product descriptions and the project product description. (D) Capturing lessons is a separate activity, not the aim of project controls.
