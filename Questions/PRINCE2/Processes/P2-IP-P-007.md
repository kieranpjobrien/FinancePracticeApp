# Question

A construction firm is partway through Initiating a Project for a new bridge. The initiation stage plan is close to running out, and the project manager realises the PID will need another two weeks to complete properly. What is the BEST response?

## Options

- **A**: Cut the remaining initiation work short and submit an incomplete PID on time
- **B**: Quietly extend initiation and absorb the slip without telling the board
- **C**: Notify the project board that the initiation stage plan's tolerance is forecast to be exceeded so it can give direction
- **D**: Begin the first delivery stage in parallel to make up the lost time

## Answer

C

## Explanation

If the initiation stage is forecast to breach its agreed tolerance, the project manager must raise this to the project board, which gives ad hoc direction on how to proceed — applying management by exception even during initiation. (A) Submitting a deliberately incomplete PID undermines the foundations the process exists to establish. (B) Concealing a tolerance breach bypasses the board and breaches the manage-by-exception principle. (D) Starting delivery before the PID is approved pre-empts the board's authorisation of the project.
