# Question

On a hospital records digitisation project, the project manager forecasts that the current management stage will overrun its agreed time tolerance by three weeks because of an unexpected data-cleansing problem. What should the project manager do?

## Options

- **A**: Quietly extend the stage by reallocating slack from later stages without informing the board
- **B**: Produce an exception report and escalate the forecast breach to the project board
- **C**: Continue working and report the overrun in the next end stage report
- **D**: Authorise an exception plan and proceed with the recovery approach immediately

## Answer

B

## Explanation

A forecast breach of stage tolerance must be escalated to the project board via an exception report, which is exactly the trigger for the board's "give ad hoc direction" activity. (A) breaches the manage-by-exception principle by hiding the situation and acting outside authority. (C) defers escalation past the point where the board can intervene. (D) is wrong because the project manager cannot authorise an exception plan; only the board can request and then authorise one.
