# Question

During delivery of a museum-exhibition fit-out, a stakeholder requests a design change that the project manager assesses as within the stage's cost and time tolerances but which the change authority's delegated limit does not cover. What should the project manager do?

## Options

- **A**: Approve the change because it stays within stage tolerance
- **B**: Refer the request for change to the project board for a decision through "give ad hoc direction"
- **C**: Reject the change outright since it exceeds the change authority's limit
- **D**: Implement the change and record it for review at the next end stage assessment

## Answer

B

## Explanation

When a change falls outside the delegated change authority's limit, the decision rests with the project board, which handles it as ad hoc direction. (A) is the near-miss: staying within stage tolerance does not grant the project manager authority over a change beyond the delegated limit. (C) pre-empts a decision that is not the project manager's to make. (D) implements an unauthorised change and defers the necessary board decision.
