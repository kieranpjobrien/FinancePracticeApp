# Question

In PRINCE2 7, what is a work package?

## Options

- **A**: A report sent by the team manager to the project manager at an agreed frequency
- **B**: The set of information relevant to the creation of one or more products, forming the agreement between the project manager and the team manager
- **C**: The plan that the team manager uses to schedule the work of their team
- **D**: A request from the team to change an agreed baseline product

## Answer

B

## Explanation

A work package is the set of information about one or more required products collated by the project manager to formally pass responsibility for work to a team manager. (A) describes a checkpoint report. (C) describes a team plan. (D) describes a request for change. The work package is the central agreement underpinning Managing Product Delivery.
