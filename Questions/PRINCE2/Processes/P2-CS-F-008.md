# Question

In Controlling a Stage, where does the information in a checkpoint report come from and who provides it?

## Options

- **A**: From the project board, as a decision to the project manager
- **B**: From the project manager, reporting to the project board
- **C**: From project assurance, reporting to corporate management
- **D**: From the team manager, reporting progress on a work package to the project manager

## Answer

D

## Explanation

A checkpoint report is sent by the team manager to the project manager at the frequency agreed in the work package, reporting progress on the assigned work. It is a key input to the project manager's review of stage status. (A) reverses the direction of project board communication. (B) describes the highlight report. (C) misdescribes both the source and the audience.
