# Question

How does the end stage report differ from the lessons report?

## Options

- **A**: The end stage report records only financial figures, while the lessons report records only risks
- **B**: The end stage report summarises the stage's performance for the board's authorisation decision, while the lessons report captures lessons for wider use
- **C**: The end stage report is produced by the team manager, while the lessons report is produced by the board
- **D**: They are identical products with different names

## Answer

B

## Explanation

The end stage report gives the board a performance summary of the completed stage to inform authorisation of the next, whereas the lessons report is a vehicle for passing on lessons that can benefit other projects or future stages. (A) misstates their content; (C) misassigns authorship (both are owned by the project manager); (D) is wrong because they serve distinct purposes.
