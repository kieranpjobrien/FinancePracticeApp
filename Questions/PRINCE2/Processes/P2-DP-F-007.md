# Question

When does the project board carry out the "authorise a stage or exception plan" activity?

## Options

- **A**: Continuously, in response to every issue raised during a stage
- **B**: At the end of each management stage, and when an exception plan is required, to approve work for the next stage or to recover from an exception
- **C**: Only once, immediately after Starting Up a Project is complete
- **D**: At the request of the team manager when a work package is finished

## Answer

B

## Explanation

The board authorises a stage or exception plan at each stage boundary (the end-stage assessment) to approve the next stage's plan, and additionally whenever an exception plan has been produced to replace a plan forecast to breach tolerance. (A) describes ongoing ad hoc direction, not stage authorisation. (C) confuses it with "authorise initiation". (D) is a Managing Product Delivery interaction handled by the project manager, not a board authorisation.
