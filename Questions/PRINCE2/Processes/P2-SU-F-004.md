# Question

Why are previous lessons captured during Starting Up a Project?

## Options

- **A**: To create the Lessons Report that closes the project
- **B**: To satisfy the audit requirements of the host organisation
- **C**: To record lessons learned by the current project team during start-up
- **D**: So that relevant lessons from earlier projects can inform how this project is set up and approached

## Answer

D

## Explanation

Capturing previous lessons applies the "learn from experience" principle: lessons from earlier or similar projects (and from corporate or external sources) are reviewed so the project can be designed, planned and approached more effectively. (A) The Lessons Report is produced at the end of a stage or project, not in start-up. (B) Lessons are sought to improve delivery, not principally for audit compliance. (C) In start-up there is no delivery experience yet to capture; the focus is on prior, external lessons feeding into the new project.
