# Question

A retailer is starting up a project to introduce a new point-of-sale system across its stores. The Project Manager is weighing whether to commission a bespoke build, buy a configurable off-the-shelf package, or extend the current system. Within Starting Up a Project, how is this best handled?

## Options

- **A**: Defer the decision until delivery, choosing whichever supplier responds fastest
- **B**: Leave it to the project board to settle when they authorise the project
- **C**: Document each option in detail in the full Project Plan during start-up
- **D**: Record it as the project approach, deciding how the solution will be delivered before assembling the Project Brief

## Answer

D

## Explanation

Deciding between bespoke, off-the-shelf or extending the existing system is exactly the project approach — how the project's products will be delivered — and it is selected during start-up so it can inform the Project Brief and later planning. (A) Letting supplier speed dictate the approach abandons deliberate selection and ignores fit with the operating environment. (B) The approach is chosen by the project team in start-up; the board authorises, it does not select the delivery method. (C) The full Project Plan is produced during initiation, not start-up, and the approach is a higher-level choice than detailed planning.
