# Question

Which statement about taking corrective action in Controlling a Stage is correct?

## Options

- **A**: The project manager may take corrective action only after the project board has approved it
- **B**: The project manager takes corrective action to resolve issues and risks while keeping the stage within its tolerances
- **C**: Corrective action always requires a new stage plan to be produced
- **D**: Corrective action is the responsibility of the team manager, not the project manager

## Answer

B

## Explanation

Taking corrective action is the project manager's activity to address deviations and emerging issues or risks while remaining within the delegated stage tolerances. (A) Approval is needed only if the action would breach tolerance, in which case it becomes an exception. (C) Many corrective actions are handled without a new plan, for example by adjusting work packages. (D) The team manager executes work but the project manager decides corrective action at stage level.
