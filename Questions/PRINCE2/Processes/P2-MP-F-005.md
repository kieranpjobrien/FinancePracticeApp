# Question

Which three activities make up the Managing Product Delivery process?

## Options

- **A**: Authorise a work package, review a work package, close a work package
- **B**: Accept a work package, execute a work package, deliver a work package
- **C**: Plan a work package, monitor a work package, report a work package
- **D**: Initiate a work package, control a work package, escalate a work package

## Answer

B

## Explanation

The three activities of Managing Product Delivery are accept a work package, execute a work package and deliver a work package. (A), (C) and (D) use plausible PRINCE2-sounding verbs but are not the defined activities; the project manager authorises work packages within Controlling a Stage, while the team manager accepts, executes and delivers them.
