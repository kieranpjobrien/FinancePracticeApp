# Question

A hospital records project, designed against current HIPAA rules, learns that a stricter state amendment will take effect six weeks before go-live. What should the project manager do FIRST?

## Options

- **A**: Continue with the original design because the project predates the amendment.
- **B**: Cancel the project because the new rules make compliance unrealistic in time.
- **C**: Hold the new requirements for the next release and document the gap as a known risk.
- **D**: Work with the compliance lead to assess the delta, then route a change request through the CCB.

## Answer

D

## Explanation

Compliance changes that take effect before go-live must be addressed before launch. (D) starts with an impact assessment and follows formal change control, which is the PMI sequence. (A) ignores a regulation that will be active at launch. (B) is premature; the impact has not been sized. (C) accepts a known compliance gap rather than acting on it.
