# Question

A project manager discovers that the organization has been operating under an incorrect interpretation of a regulation for years. Correcting the interpretation would require significant changes to the current project and previous projects. What should the project manager do?

## Options

- **A**: Continue with the current interpretation since it's been accepted practice
- **B**: Immediately report the finding to compliance and legal, then work with them to determine the appropriate response
- **C**: Quietly correct the interpretation for the current project without raising the historical issue
- **D**: Wait for a regulator to identify the misinterpretation

## Answer

B

## Explanation

The correct answer is B. Regulatory misinterpretation is a significant organizational risk that requires proper expertise to assess and address. Compliance and legal need to evaluate the actual risk, determine whether self-disclosure is appropriate, and plan remediation. This is beyond PM authority to handle alone.

Option A perpetuates known non-compliance, which is ethically and legally problematic.

Option C addresses the current project but leaves organizational exposure unaddressed. The PM has a responsibility to raise the broader issue.

Option D waits for discovery rather than proactively addressing the risk. Self-identified issues often receive more favorable treatment than regulator-discovered violations.
