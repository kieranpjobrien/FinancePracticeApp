# Question

Two equally feasible projects compete for a single shared resource pool. The PM is asked which to recommend. What is the most useful basis for the recommendation?

## Options

- **A**: The project with the most senior sponsor.
- **B**: The project that started first.
- **C**: The project with the more detailed plan.
- **D**: The project with the higher net value relative to its cost and risk.

## Answer

D

## Explanation

Resource allocation should follow value-adjusted-for-risk (D). (A) is politics. (B) is order of arrival, not merit. (C) is plan quality, not value.
