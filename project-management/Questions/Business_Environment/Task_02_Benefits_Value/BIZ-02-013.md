# Question

The business case for a project included both tangible benefits (cost savings) and intangible benefits (improved employee satisfaction). After delivery, cost savings can be measured at 90% of target, but there's no way to measure satisfaction improvement because baseline data was never collected. What should the project manager recommend for future projects?

## Options

- **A**: Focus business cases only on tangible, measurable benefits
- **B**: Ensure benefits measurement plans, including baseline data collection, are part of project planning for all significant benefits
- **C**: Accept that intangible benefits cannot be measured
- **D**: Use industry benchmarks instead of baseline data

## Answer

B

## Explanation

The correct answer is B. Intangible benefits can often be measured (surveys, turnover rates, engagement scores) if planned for in advance. The lesson learned is that benefits measurement planning - including baseline collection - should be part of project initiation for all benefits included in the business case.

Option A discards valuable but measurable intangible benefits. Employee satisfaction can be measured with proper planning.

Option C accepts an avoidable measurement failure. With forethought, baselines could have been established.

Option D uses external data instead of addressing the planning gap. Benchmarks don't show improvement; baselines do.
