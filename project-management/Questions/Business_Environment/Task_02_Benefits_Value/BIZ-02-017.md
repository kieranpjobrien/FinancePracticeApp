# Question

Three months after CRM go-live, sales have not lifted as projected. Investigation shows reps still use the legacy system because the new one demands more data entry. What should the project manager do?

## Options

- **A**: Disable the legacy system to force adoption of the new CRM.
- **B**: Recommend decommissioning the new CRM and treating the project as unsuccessful.
- **C**: Diagnose the adoption barriers, refine the system to cut friction, and run a structured change effort.
- **D**: Update the business case so that projected sales lift matches current reality.

## Answer

C

## Explanation

Low adoption is the root cause; address it (C) by removing friction and supporting users through the change. (A) substitutes coercion for design; reps will work around it. (B) discards the investment without trying. (D) hides the gap by re-baselining the target.
