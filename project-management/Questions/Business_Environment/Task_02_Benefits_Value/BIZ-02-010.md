# Question

A project has delivered 80% of planned features and exhausted its budget. The sponsor asks whether to fund an extension to complete the remaining features or close the project. Analysis shows the delivered features will realize 95% of the projected benefits. What should the project manager recommend?

## Options

- **A**: Complete all features since the project should deliver what was promised
- **B**: Present the value analysis showing 95% of benefits achieved, recommend closing the project, and propose evaluating the remaining features as a separate initiative with its own business case
- **C**: Request the minimum budget needed to complete remaining features
- **D**: Reduce quality on remaining features to fit within budget

## Answer

B

## Explanation

The correct answer is B. With 95% of benefits achieved from 80% of features, the remaining 20% of features deliver diminishing returns. The PM should present this value analysis objectively. Closing now may be the right business decision; the remaining features can be evaluated on their own merits as a potential follow-on project.

Option A focuses on completing scope rather than delivering value. The project exists to deliver benefits, not features.

Option C continues to invest without value justification. The remaining features deliver only 5% of benefits for 20% of work.

Option D compromises quality, which may reduce realized benefits and create technical debt.
