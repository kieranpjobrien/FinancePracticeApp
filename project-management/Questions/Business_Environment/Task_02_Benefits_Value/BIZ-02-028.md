# Question

A project has a negative NPV (Net Present Value) based on quantifiable benefits alone. However, the sponsor argues that strategic benefits (market positioning, competitive response) justify the investment. What should the project manager do?

## Options

- **A**: Reject the project since NPV is negative
- **B**: Document both quantifiable and strategic benefits, ensure stakeholders understand the investment is justified by strategic rather than financial returns, and proceed if governance approves
- **C**: Inflate the quantifiable benefits to show positive NPV
- **D**: Ignore NPV since it doesn't capture all value

## Answer

B

## Explanation

The correct answer is B. Some projects are strategically valuable even with negative financial NPV - defensive moves, market entry, capability building. The PM should ensure transparent documentation of both financial and strategic justification, so decision-makers understand they're making a strategic investment, not expecting financial returns.

Option A applies financial metrics too rigidly, ignoring legitimate strategic value.

Option C is dishonest and creates false expectations about financial returns.

Option D dismisses financial analysis entirely rather than contextualizing it appropriately.
