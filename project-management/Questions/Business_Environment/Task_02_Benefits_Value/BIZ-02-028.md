# Question

A proposed project shows a negative NPV when only quantifiable benefits are considered, but the sponsor argues that strategic positioning makes the investment worthwhile. What should the project manager do?

## Options

- **A**: Reject the project because NPV does not clear zero.
- **B**: Adjust the financial assumptions until the NPV is positive on paper.
- **C**: Document financial and strategic benefits transparently and let governance decide.
- **D**: Drop NPV from the analysis since it does not capture strategic value.

## Answer

C

## Explanation

A strategic-but-unprofitable project can be a valid choice — but only with clear-eyed disclosure to governance (C). (A) is too rigid; some defensive or capability-building investments are right despite negative NPV. (B) is fraud. (D) is over-correction; NPV is still useful context.
