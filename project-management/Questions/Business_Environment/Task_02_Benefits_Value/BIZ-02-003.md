# Question

When should benefits realization be measured for a project?

## Options

- **A**: Only at project completion
- **B**: Throughout the project and continuing after project closure until benefits are fully realized
- **C**: Only during project execution
- **D**: Only when requested by the sponsor

## Answer

B

## Explanation

The correct answer is B. Benefits often materialize over time - some during the project, others only after deliverables are in use. Continuous measurement from project start through post-implementation ensures benefits are tracked to realization and enables early action if benefits aren't materializing as expected.

Option A measures too late for course correction and misses benefits that continue accruing after closure.

Option C misses pre-project baseline and post-project realization.

Option D makes measurement reactive rather than systematic.
