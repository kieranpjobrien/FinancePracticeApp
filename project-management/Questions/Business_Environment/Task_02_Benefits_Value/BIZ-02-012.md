# Question

After the third incremental release of an internal tool, users say the features work but the workflow is clumsy and the promised efficiency gains have not materialised. What should the project manager do?

## Options

- **A**: Continue the planned roadmap and address the workflow concerns in a later release.
- **B**: Add a training program to help users adapt to the new workflow.
- **C**: Restate the benefits target downward to reflect current adoption.
- **D**: Investigate the workflow feedback and adjust upcoming releases to improve usability and efficiency.

## Answer

D

## Explanation

Incremental delivery exists precisely to absorb feedback like this (D): investigate, then adjust the next sprint. (A) keeps building features users already say miss the point. (B) treats a design problem with training. (C) hides the gap by moving the target.
