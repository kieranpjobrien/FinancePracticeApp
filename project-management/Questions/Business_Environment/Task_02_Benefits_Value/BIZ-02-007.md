# Question

How are project outputs different from project benefits?

## Options

- **A**: Outputs are intangible deliverables; benefits are always tangible, financial returns to the organisation.
- **B**: Outputs are measured at project closure; benefits are measured only during the execution phase of delivery.
- **C**: They are different labels for the same underlying concept of value delivery in a project context.
- **D**: Outputs are what the project produces; benefits are the value the organisation gains from using them.

## Answer

D

## Explanation

A project ships outputs; the organisation gains benefits when those outputs are adopted and used (D). (A) inverts and over-narrows both terms. (B) reverses when each is measured. (C) ignores the distinction PMI relies on for benefits realisation.
