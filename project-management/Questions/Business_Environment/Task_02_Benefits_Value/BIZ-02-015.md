# Question

What document typically defines the expected benefits and value a project should deliver?

## Options

- **A**: The project schedule
- **B**: The business case
- **C**: The risk register
- **D**: The stakeholder register

## Answer

B

## Explanation

The correct answer is B. The business case justifies the project by articulating expected benefits, costs, and value. It answers "why should we do this project?" by comparing expected value to required investment and establishing the success criteria by which the project will be judged.

Option A defines when work happens, not why it's valuable.

Option C documents threats and opportunities, not the fundamental value proposition.

Option D identifies who has an interest in the project, not the expected benefits.
