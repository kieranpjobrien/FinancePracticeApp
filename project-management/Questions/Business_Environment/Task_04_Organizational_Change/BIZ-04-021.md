# Question

Phase 1 of a phased rollout went live with a wave of usability complaints. Phase 2 is due to start in two weeks. What should the project manager do?

## Options

- **A**: Proceed with phase 2 because the schedule was already approved.
- **B**: Push back phase 2 indefinitely until every phase 1 complaint is closed.
- **C**: Set lower expectations for phase 2 by warning users about known limitations.
- **D**: Address the critical usability issues, communicate the fixes back to phase 1 users, and bring lessons learned into phase 2.

## Answer

D

## Explanation

Fix the most damaging issues, show responsiveness, and improve the next wave (D). (A) ignores the data and erodes trust. (B) over-corrects when fixes can run in parallel. (C) manages perception instead of solving the problem.
