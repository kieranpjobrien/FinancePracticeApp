# Question

Six weeks after deployment, only 30% of target users are actively using the new system. What should the project manager do?

## Options

- **A**: Diagnose specific adoption barriers, talk to non-users, and target interventions to the actual blockers.
- **B**: Run another round of training for the entire user base.
- **C**: Mandate use of the new system and disable workarounds in the legacy environment.
- **D**: Close the project as delivered because the technical deployment was successful.

## Answer

A

## Explanation

Adoption diagnosis comes before treatment (A): is it awareness, ability, or willingness? (B) assumes a knowledge gap without evidence. (C) replaces engagement with force. (D) closes on outputs while outcomes are missing.
