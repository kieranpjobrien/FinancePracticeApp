# Question

A project successfully implemented a new system, and the change management effort achieved strong initial adoption. The project is now closing. The project manager is concerned that without ongoing attention, users may revert to old habits. What should the project manager do before closing?

## Options

- **A**: Document the project as successful and close
- **B**: Extend the project indefinitely to monitor adoption
- **C**: Establish transition arrangements with operations including adoption monitoring, reinforcement mechanisms, support structures, and clear ownership for sustaining the change
- **D**: Trust that users will maintain new behaviors since initial adoption was strong

## Answer

C

## Explanation

The correct answer is C. Sustaining change requires ongoing reinforcement that extends beyond project closure. The PM should ensure operations has clear ownership, monitoring mechanisms to detect regression, support structures for users, and reinforcement approaches. This transition planning is essential for benefits realization.

Option A ignores the risk of adoption decay after project attention ends.

Option B keeps the project open indefinitely rather than properly transitioning to operations.

Option D assumes initial adoption guarantees sustained adoption, which research shows is not the case without reinforcement.
