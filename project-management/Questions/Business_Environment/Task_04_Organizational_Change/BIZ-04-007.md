# Question

What is the PRIMARY goal of organisational change management in the context of a project?

## Options

- **A**: To document all changes to project scope
- **B**: To help people successfully adopt and sustain new ways of working introduced by the project
- **C**: To control the project budget
- **D**: To manage configuration changes to project deliverables

## Answer

B

## Explanation

The correct answer is B. Organisational change management focuses on the people side of change - helping individuals and groups transition from current state to the desired future state. This includes communication, training, stakeholder engagement, and addressing resistance. Without successful adoption, project deliverables don't create value.

Option A (document scope changes) describes project scope management or change control.

Option C (control budget) describes cost management.

Option D (configuration management) is a technical discipline, not organisational change management.
