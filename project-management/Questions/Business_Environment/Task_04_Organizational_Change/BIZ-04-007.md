# Question

What is the primary goal of organisational change management on a project?

## Options

- **A**: Documenting all changes to project scope through formal change requests.
- **B**: Helping people adopt and sustain the new ways of working introduced by the project.
- **C**: Controlling the project budget to remain within the approved baseline.
- **D**: Managing configuration changes to deliverables across environments.

## Answer

B

## Explanation

OCM is about people moving from current state to future state and staying there (B). (A) is project change control. (C) is cost management. (D) is configuration management.
