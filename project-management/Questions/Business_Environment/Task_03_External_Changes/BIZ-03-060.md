# Question

Mid-execution, a new accessibility regulation will apply to all public sector products in your country starting in nine months. Your hybrid project ships its first public release in eight months. What should the PM do?

## Options

- **A**: Continue the current release plan and add accessibility work to a future release.
- **B**: Pause the project until the accessibility scope is fully defined.
- **C**: Submit a change request and add accessibility deliverables to the release backlog with priority appropriate to the deadline.
- **D**: Strip the release scope to accessibility work alone.

## Answer

C

## Explanation

The PM updates the baseline through change control and integrates the new requirements with appropriate priority (C). (A) ships a non-compliant product. (B) pauses unnecessarily; the release can absorb the work. (D) over-corrects and discards other planned value.
