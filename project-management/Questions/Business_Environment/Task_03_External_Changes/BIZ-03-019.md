# Question

PESTLE analysis is a tool used to evaluate external factors. What does PESTLE stand for?

## Options

- **A**: Project, Environment, Schedule, Timeline, Leadership, Evaluation
- **B**: Political, Economic, Social, Technological, Legal, Environmental
- **C**: Planning, Execution, Strategy, Tracking, Learning, Evaluation
- **D**: Process, Efficiency, Standards, Technology, Labor, Expenses

## Answer

B

## Explanation

The correct answer is B. PESTLE analysis examines Political, Economic, Social, Technological, Legal, and Environmental factors that could affect a project or organization. It provides a framework for systematically scanning the external environment for opportunities and threats.

Options A, C, and D are not standard frameworks and don't represent recognized analytical tools for external environment assessment.
