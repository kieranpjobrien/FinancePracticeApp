# Question

Which tool is MOST useful for analyzing external factors that may affect a project?

## Options

- **A**: Work Breakdown Structure (WBS)
- **B**: PESTLE analysis (Political, Economic, Social, Technological, Legal, Environmental)
- **C**: Gantt chart
- **D**: RACI matrix

## Answer

B

## Explanation

The correct answer is B. PESTLE analysis is specifically designed to examine external macro-environmental factors that may affect an organization or project. It systematically considers Political, Economic, Social, Technological, Legal, and Environmental factors.

Option A (WBS) is for decomposing project scope, not analyzing external environment.

Option C (Gantt chart) is for schedule visualization.

Option D (RACI) is for clarifying roles and responsibilities.
