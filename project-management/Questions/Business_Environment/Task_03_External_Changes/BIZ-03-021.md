# Question

Mid-build of a new warehouse system, the organisation announces it will acquire a competitor with a different warehouse system and processes. What should the project manager do?

## Options

- **A**: Engage the integration team, understand how the acquisition affects requirements, and decide with stakeholders whether to continue, expand, or pause.
- **B**: Continue building for current operations only and ignore the announcement.
- **C**: Pause the project until the integration strategy is fully defined.
- **D**: Expand scope now to cover both companies' warehouses preemptively.

## Answer

A

## Explanation

The integration strategy will determine which system, scope, and timeline make sense (A); the PM has to plug into that work first. (B) builds for an obsolete future. (C) stops without need; useful contributions can usually be made in parallel. (D) expands scope before the strategy is set.
