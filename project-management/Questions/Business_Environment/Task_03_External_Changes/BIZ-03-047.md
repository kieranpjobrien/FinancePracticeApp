# Question

An external change has been assessed and the team has agreed an approach to absorb it. Schedule and budget will both shift. What governance step does the PM take next?

## Options

- **A**: Apply the changes to the project plan and notify stakeholders in the next status report.
- **B**: Update the baselines silently because the team has already agreed.
- **C**: Wait for the next baseline review to incorporate the changes.
- **D**: Submit a change request to formally update the schedule and cost baselines through change control.

## Answer

D

## Explanation

Any baseline change goes through formal change control regardless of how the change originated (D). (A) updates the plan without governance approval. (B) silently changes the baseline, which destroys variance reporting. (C) leaves the project running against an out-of-date baseline.
