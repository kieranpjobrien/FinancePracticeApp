# Question

A project is implementing a customer service platform. A pandemic suddenly forces the organization into remote work. The project scope was designed for call center operations, but now the organization needs distributed, work-from-home capabilities. What should the project manager do?

## Options

- **A**: Complete the original scope and implement remote capabilities as a separate project
- **B**: Assess how the changed environment affects requirements, reprioritize features to address remote work needs, and work with stakeholders to adjust scope and timeline
- **C**: Pause the project until normal operations resume
- **D**: Continue with original scope since the pandemic is temporary

## Answer

B

## Explanation

The correct answer is B. The business environment has fundamentally changed what the organization needs. The PM should work with stakeholders to reprioritize - features supporting distributed work become more critical; call-center-specific features may be deprioritized. The project should deliver what the organization needs now.

Option A delivers a product that doesn't meet current needs, deferring value.

Option C assumes normal operations will resume soon, which may not happen, and abandons opportunity to help.

Option D ignores a significant change in requirements driven by external circumstances.
