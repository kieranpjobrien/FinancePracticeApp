# Question

Which of the following best describes how external environment changes should be communicated to project stakeholders?

## Options

- **A**: Only when changes directly impact the project schedule
- **B**: Proactively, with analysis of potential impacts and recommendations for response
- **C**: After the project manager has already decided how to respond
- **D**: Through formal change requests only

## Answer

B

## Explanation

The correct answer is B. Stakeholders should be informed proactively about external changes that could affect the project, along with analysis of potential impacts and recommended responses. This enables informed decision-making and demonstrates the PM's environmental awareness.

Option A is too narrow - changes can affect scope, cost, risk, and value, not just schedule.

Option C bypasses stakeholder input on potentially significant decisions.

Option D conflates communication about external changes with internal change control.
