# Question

A project is building a new manufacturing facility. Midway through construction, a trade war begins that significantly increases the cost of imported equipment and materials. The project budget is now projected to exceed approved funding by 25%. What should the project manager do?

## Options

- **A**: Continue construction and address the budget issue later
- **B**: Stop construction until the trade situation resolves
- **C**: Analyze the cost impact, explore alternatives (domestic suppliers, value engineering, phased delivery), present options and recommendations to stakeholders for decision
- **D**: Reduce scope to fit within the original budget

## Answer

C

## Explanation

The correct answer is C. A 25% cost increase from external factors requires stakeholder decision-making with good analysis. Options might include: accepting increased costs if value still justifies it, finding alternative suppliers, redesigning to use different materials, phasing the project, or pausing construction. Each option has different implications that stakeholders must weigh.

Option A continues spending without authorization for increased costs.

Option B may be appropriate but is a decision for stakeholders, not the PM alone.

Option D arbitrarily reduces scope without understanding if that's the best response.
