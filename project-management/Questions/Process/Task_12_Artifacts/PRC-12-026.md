# Question

Which of the following artifacts would be created FIRST when starting a new project?

## Options

- **A**: Project schedule
- **B**: Project charter
- **C**: Work breakdown structure
- **D**: Status report

## Answer

B

## Explanation

The correct answer is B. The project charter formally authorizes the project and is created during project initiation, before detailed planning begins. It precedes the schedule, WBS, and other planning artifacts.

Option A (schedule) is created during planning after scope is defined.

Option C (WBS) is created during planning to decompose scope into manageable components.

Option D (status report) is created during execution to report on progress.
