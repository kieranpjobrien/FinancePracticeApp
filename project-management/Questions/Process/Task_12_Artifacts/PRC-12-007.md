# Question

Which of the following is a core activity within configuration management?

## Options

- **A**: Identifying configuration items and documenting their characteristics so changes can be controlled.
- **B**: Calculating the overall project budget and managing cost variances across the delivery period.
- **C**: Scheduling regular team meetings and capturing minutes for distribution to all stakeholders.
- **D**: Performing stakeholder analysis and updating the project's stakeholder register as engagement evolves.

## Answer

A

## Explanation

Configuration management identifies configuration items, controls changes to their characteristics, and reports change status; identification and documentation of CIs is foundational. (B) is cost management. (C) is communications activity. (D) is stakeholder management. None of these is a configuration management activity.
