# Question

Which of the following is a key component of configuration management?

## Options

- **A**: Identifying and documenting the functional and physical characteristics of a product or component
- **B**: Calculating the project budget
- **C**: Scheduling team meetings
- **D**: Conducting stakeholder analysis

## Answer

A

## Explanation

The correct answer is A. Configuration management involves identifying and documenting the characteristics of configuration items (products, results, components), controlling changes to those characteristics, and recording and reporting change status. This ensures product integrity throughout the project lifecycle.

Option B relates to cost management, not configuration management.

Option C relates to communications management.

Option D relates to stakeholder management.
