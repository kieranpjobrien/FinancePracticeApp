# Question

A project is using both predictive and agile elements (hybrid approach). The predictive governance board requires formal documentation, while the agile team prefers minimal documentation. How should the project manager handle artifact management?

## Options

- **A**: Choose one approach and apply it consistently
- **B**: Create two separate documentation systems for each methodology
- **C**: Tailor documentation practices to satisfy governance requirements while keeping team documentation lean and useful
- **D**: Request exemption from governance documentation requirements

## Answer

C

## Explanation

The correct answer is C. Hybrid projects require balancing different needs. The PM should identify which documentation is truly required for governance (and create it in the required format) while allowing the agile team to maintain their preferred lightweight documentation for day-to-day work. Some documentation may serve both purposes.

Option A ignores the reality that different project elements have different needs.

Option B creates unnecessary duplication and potential inconsistencies.

Option D may not be possible and doesn't address the legitimate need for governance oversight.
