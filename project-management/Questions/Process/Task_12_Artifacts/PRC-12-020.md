# Question

Which artifact captures the cumulative record of all changes requested during a project?

## Options

- **A**: Risk register
- **B**: Change log
- **C**: Project charter
- **D**: Work breakdown structure

## Answer

B

## Explanation

The correct answer is B. The change log (or change register) records all change requests submitted during the project, including their current status (pending, approved, rejected, deferred). It provides a complete history of requested changes for tracking and audit purposes.

Option A records identified risks and their status.

Option C authorizes the project and documents initial scope, but doesn't track ongoing changes.

Option D decomposes project work into manageable components but doesn't track changes.
