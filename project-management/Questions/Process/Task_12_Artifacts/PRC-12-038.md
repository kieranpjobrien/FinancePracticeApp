# Question

A project team works on sensitive government contracts requiring different classification levels for various documents. Some team members have clearance for all documents while others have limited access. How should the project manager structure artifact management?

## Options

- **A**: Give all team members access to all documents to improve collaboration
- **B**: Restrict all documents to only those with highest clearance
- **C**: Implement access controls based on classification levels, ensuring team members can access documents appropriate to their clearance while maintaining collaboration within allowed boundaries
- **D**: Keep all classified documents offline in physical storage only

## Answer

C

## Explanation

The correct answer is C. Access controls based on classification levels comply with security requirements while enabling appropriate collaboration. Modern document management systems support granular access controls. This balances security (need-to-know principle) with operational efficiency (enabling work within allowed boundaries).

Option A violates security requirements and could result in serious consequences.

Option B overly restricts access, potentially preventing team members from doing their work effectively.

Option D may be impractical for modern project work and creates significant inefficiency.
