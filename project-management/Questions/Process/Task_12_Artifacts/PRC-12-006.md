# Question

In a hybrid co-located/remote team, members regularly work from outdated specs because they cannot reliably find the latest versions. What should the PM put in place?

## Options

- **A**: Require everyone to return to the office to improve information flow.
- **B**: Stand up a centralised document repository with naming conventions, version control, and access protocols, and migrate the project artifacts into it.
- **C**: Email document updates to the full team every time anything changes.
- **D**: Assign one team member to be the keeper of all project documents.

## Answer

B

## Explanation

The root cause is the absence of a single authoritative location with version control; that is what the PM should build. (A) does not solve document location problems and is not a feasible response in many organisations. (C) creates an inbox flood and lets people work from outdated attachments. (D) creates a bottleneck and a single point of failure.
