# Question

A project team is working in a hybrid environment with some members co-located and others remote. Team members report difficulty finding the latest versions of project documents, and there have been instances of work being done on outdated specifications. What should the project manager implement?

## Options

- **A**: Require all team members to work from the office to improve communication
- **B**: Establish a centralized document repository with clear naming conventions, version control, and access protocols
- **C**: Email all document updates to the entire team
- **D**: Assign one team member to be responsible for all documents

## Answer

B

## Explanation

The correct answer is B. A centralized repository with proper version control solves the core problem of multiple versions and accessibility. Clear naming conventions help team members find documents, version control ensures everyone works from the latest version, and access protocols ensure appropriate permissions.

Option A doesn't solve the document management problem and may not be feasible or desirable.

Option C creates email overload and doesn't solve version control issues - team members may still work from outdated attachments.

Option D creates a bottleneck and single point of failure. Document management should be systematized, not dependent on one person.
