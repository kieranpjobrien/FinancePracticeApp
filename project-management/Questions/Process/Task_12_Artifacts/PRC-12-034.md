# Question

A project manager inherits a project where the previous PM created extensive documentation but used non-standard terminology and organization. New team members struggle to understand the existing documents. What should the project manager do?

## Options

- **A**: Rewrite all documentation using standard terminology immediately
- **B**: Continue using the existing terminology to maintain consistency
- **C**: Create a glossary mapping existing terminology to standards, update critical documents first, and establish standards for new documentation
- **D**: Abandon the existing documentation and start fresh

## Answer

C

## Explanation

The correct answer is C. A glossary provides a bridge between existing and standard terminology, enabling immediate use of existing documents. Prioritizing critical documents for updates ensures the most important artifacts become accessible first. New documentation follows standards going forward.

Option A requires significant effort and may not be the best use of time if documents are functional.

Option B perpetuates confusion for anyone unfamiliar with the non-standard terminology.

Option D discards potentially valuable work and requires recreating information.
