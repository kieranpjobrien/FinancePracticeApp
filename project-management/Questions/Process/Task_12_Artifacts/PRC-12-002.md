# Question

A project manager discovers that team members are editing the project schedule directly without going through the change control process. This has caused confusion about the current baseline. What should the PM do?

## Options

- **A**: Lock the schedule file so only the PM can edit it
- **B**: Determine the correct current state, re-establish the baseline, and reinforce the change control process with the team
- **C**: Accept the current schedule as the new baseline
- **D**: Revert to the original baseline and redo all changes properly

## Answer

B

## Explanation

The correct answer is B. The immediate need is to establish a known-correct baseline. Then reinforce the process to prevent recurrence. This addresses both the artifact problem (unclear baseline) and the process problem (bypassing change control).

Option A treats symptoms (unauthorized edits) without fixing the baseline confusion or addressing why the team bypassed process.

Option C legitimizes unauthorized changes without understanding what changed.

Option D may be excessive if changes were valid - the issue is process, not necessarily the changes themselves.
