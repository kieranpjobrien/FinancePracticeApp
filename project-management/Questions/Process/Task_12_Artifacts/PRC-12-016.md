# Question

The corporate PMIS suffers a confirmed security breach overnight. Project artifacts may have been exfiltrated. What should the project manager do FIRST?

## Options

- **A**: Notify the sponsor and key stakeholders, and engage IT security to follow the organisation's incident response plan.
- **B**: Lock down PMIS access for the team and pause all work pending forensic analysis.
- **C**: Move active artifacts into a secondary tool so the team can keep working while investigation continues.
- **D**: Audit the project's risk register and update the response for information-security risks.

## Answer

A

## Explanation

A confirmed breach is an organisational incident, not a project decision. (A) routes the response through the people who own it: leadership communication and the IT security incident response process. (B) is a unilateral action that may interfere with forensic preservation and is normally driven by IT security, not the PM. (C) keeps the team productive but may copy compromised data and creates shadow artifacts that will be hard to reconcile. (D) is good follow-up housekeeping but is not the FIRST action when a live incident is in progress.
