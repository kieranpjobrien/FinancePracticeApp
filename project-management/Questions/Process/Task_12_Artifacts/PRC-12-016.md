# Question

A project manager discovers that the project management information system (PMIS) used by the organization has been compromised by a security breach. Project artifacts may have been accessed by unauthorized parties. What should the project manager do FIRST?

## Options

- **A**: Continue working while IT investigates
- **B**: Switch to manual documentation methods permanently
- **C**: Notify appropriate stakeholders, follow organizational incident response procedures, and assess the impact on project artifacts and information security
- **D**: Delete all project artifacts to prevent further exposure

## Answer

C

## Explanation

The correct answer is C. A security breach requires immediate response following established procedures. The PM should notify stakeholders (including the sponsor and relevant governance bodies), work with IT security, and assess what artifacts were potentially compromised. This enables appropriate remediation and stakeholder communication.

Option A ignores a potentially serious breach that could have legal, contractual, or reputational implications.

Option B is an overreaction that abandons efficient tools without proper assessment.

Option D destroys evidence and project information, potentially making the situation worse and hampering investigation.
