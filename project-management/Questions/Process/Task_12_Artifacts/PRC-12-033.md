# Question

A Project Management Information System (PMIS) is BEST described as:

## Options

- **A**: A software tool that only tracks project schedules
- **B**: A system of tools and techniques used to gather, integrate, and disseminate project management information
- **C**: A database for storing historical project data only
- **D**: A communication tool for sending emails to stakeholders

## Answer

B

## Explanation

The correct answer is B. A PMIS encompasses the tools, techniques, and systems used to collect, store, and distribute project information. This may include scheduling tools, document management, reporting capabilities, and collaboration features - it's an integrated system, not just one function.

Option A is too narrow - PMIS includes more than just scheduling.

Option C describes only the archival function, not the active management of current project information.

Option D describes only one communication mechanism, not the full information management system.
