# Question

A Project Management Information System (PMIS) is BEST described as:

## Options

- **A**: The scheduling software used to maintain the project's baseline schedule.
- **B**: The document repository used to archive deliverables after closeout.
- **C**: An integrated set of tools, techniques, and systems for gathering and distributing project information.
- **D**: The communication system used by the PM to send status updates to stakeholders.

## Answer

C

## Explanation

A PMIS is integrated information infrastructure that covers scheduling, document management, reporting, collaboration, and communication — not any single tool. (A) names only the scheduling component. (B) describes one archival function. (D) names the communication channel only. (C) captures the integrated nature defined in PMBOK.
