# Question

A Project Management Information System (PMIS) is BEST described as:

## Options

- **A**: The scheduling software used by the team to maintain and update the project's baseline schedule.
- **B**: The document repository used by the PM to archive project deliverables after formal closeout activities.
- **C**: An integrated set of tools and systems for gathering and distributing project information across the team.
- **D**: The communication system used by the PM to send formal status updates to stakeholders on a cadence.

## Answer

C

## Explanation

A PMIS is integrated information infrastructure that covers scheduling, document management, reporting, collaboration, and communication — not any single tool. (A) names only the scheduling component. (B) describes one archival function. (D) names the communication channel only. (C) captures the integrated nature defined in PMBOK.
