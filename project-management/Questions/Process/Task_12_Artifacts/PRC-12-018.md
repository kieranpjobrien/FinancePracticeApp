# Question

A project team is geographically distributed across four time zones. Team members frequently make conflicting edits to shared documents because they don't realize others are working on the same files. What solution should the project manager implement?

## Options

- **A**: Restrict editing to specific time windows for each region
- **B**: Use a collaboration platform that supports real-time co-editing or check-in/check-out functionality to prevent conflicts
- **C**: Assign document ownership so only one person can edit each document
- **D**: Require all edits to go through the project manager

## Answer

B

## Explanation

The correct answer is B. Modern collaboration platforms solve this problem technically - either through real-time co-editing (multiple people can edit simultaneously with changes merged automatically) or check-in/check-out (one person locks the document while editing). This allows distributed work without conflicts.

Option A is impractical and limits productivity to narrow windows.

Option C creates bottlenecks when multiple people need to contribute to the same document.

Option D creates a severe bottleneck at the PM level and doesn't scale.
