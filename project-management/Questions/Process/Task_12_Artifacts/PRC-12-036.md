# Question

A regulated project may be audited at any time without notice. The PM is deciding how to manage artifacts so audits do not derail delivery. What is the BEST approach?

## Options

- **A**: Embed completeness, currency, and traceability checks into the team's normal cadence so artifacts are always audit-ready.
- **B**: Stand up a quarterly internal pre-audit to find and fix gaps before any external audit.
- **C**: Assign a project administrator to maintain a separate audit binder that mirrors the working artifacts.
- **D**: Build a runbook that lists what to gather quickly if an audit is announced.

## Answer

A

## Explanation

Audit-readiness is a property of the way you work, not an event you prepare for. (A) bakes the controls into the day-to-day, which both passes audits and improves project management. (B) helps but still treats audits as periodic events; in this scenario audits can happen with no notice. (C) creates a parallel record that will drift from reality. (D) is reactive and assumes you will be told the audit is coming, which contradicts the scenario.
