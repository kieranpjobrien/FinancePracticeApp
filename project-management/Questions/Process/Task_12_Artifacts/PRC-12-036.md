# Question

A project manager must prepare for a potential audit that could occur at any time during the project. The auditors will examine project artifacts to verify compliance with organizational standards and regulatory requirements. What artifact management strategy should the project manager implement?

## Options

- **A**: Prepare documentation specifically when audit is announced
- **B**: Maintain audit-ready documentation continuously by ensuring artifacts are complete, current, organized, and traceable throughout the project
- **C**: Assign a team member to prepare for audits as a side task
- **D**: Request advance notice of all audits to allow preparation time

## Answer

B

## Explanation

The correct answer is B. Continuous audit-readiness is achieved by maintaining proper documentation practices throughout the project. This means artifacts are always complete, current, properly organized, and demonstrate traceability. This approach also improves day-to-day project management, not just audit response.

Option A creates risk if audits occur unexpectedly and requires significant effort to catch up.

Option C may not provide sufficient attention if it's a side task, and creates a single point of failure.

Option D may not be possible for all audit types and doesn't address the underlying documentation quality.
