# Question

What is the term for an approved version of a configuration item that can only be changed through formal change control?

## Options

- **A**: Template
- **B**: Draft
- **C**: Archive
- **D**: Baseline

## Answer

D

## Explanation

A baseline is a formally approved version of scope, schedule, cost, or other configuration item, and any change requires a change request through CCB. (A) is a reusable pattern with no approval status. (B) is a working version that has not yet been approved. (C) refers to historical storage of completed records, not an active reference point.
