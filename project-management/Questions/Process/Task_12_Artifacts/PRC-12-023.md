# Question

What is the term for an approved version of a configuration item that serves as the basis for further development and can only be changed through formal change control?

## Options

- **A**: Draft
- **B**: Baseline
- **C**: Template
- **D**: Archive

## Answer

B

## Explanation

The correct answer is B. A baseline is a formally approved version that serves as a reference point. Once baselined, a configuration item can only be modified through the formal change control process. Baselines enable performance measurement by comparing actual results to the approved plan.

Option A is a preliminary version, not yet approved.

Option C is a reusable pattern or format, not a specific approved version.

Option D refers to stored historical documents, not an active reference point for development.
