# Question

A project is being cancelled before completion due to strategic changes in the organization. The project manager must ensure knowledge from the project is preserved even though it won't be completed. What is the MOST important artifact management action?

## Options

- **A**: Delete all project artifacts since the project is cancelled
- **B**: Archive all artifacts as-is without any additional documentation
- **C**: Document the project's current state, lessons learned, and decisions made, then properly archive all artifacts for potential future reference
- **D**: Transfer all artifacts immediately to the PMO without review

## Answer

C

## Explanation

The correct answer is C. Even cancelled projects contain valuable knowledge - what was learned, decisions made, work completed, and why it was cancelled. Documenting the current state and lessons learned before archiving ensures this knowledge is accessible if the project is restarted or similar projects are undertaken. Proper archival enables future retrieval.

Option A destroys potentially valuable information and organizational knowledge.

Option B preserves raw materials but loses context about the project's state and learnings.

Option D transfers artifacts but doesn't ensure they're organized or contextualized for future use.
