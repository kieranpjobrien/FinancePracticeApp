# Question

Integrated change control is performed:

## Options

- **A**: Only during the execution phase
- **B**: From project initiation through closing
- **C**: Only when changes are requested
- **D**: At scheduled change control meetings

## Answer

B

## Explanation

The correct answer is B. Integrated change control is performed throughout the project lifecycle, from initiation through closing. Changes can occur at any time and all need to be managed. Even during closing, changes may be needed to address final deliverable adjustments.

Option A is too narrow. Changes can occur during any phase.

Option C is reactive. Change control also involves proactive management of configuration and baselines.

Option D describes one mechanism but change control isn't limited to meetings.
