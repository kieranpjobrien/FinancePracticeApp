# Question

A change request proposes reducing scope to bring the project back on schedule. The reduction would remove features that one stakeholder considers essential. How should the PM proceed?

## Options

- **A**: Reject the change since it removes essential features
- **B**: Approve the change since schedule recovery is critical
- **C**: Evaluate the change with input from affected stakeholders before the CCB decision
- **D**: Let the CCB decide without additional stakeholder input

## Answer

C

## Explanation

The correct answer is C. Impact analysis should include stakeholder perspectives. If a feature is considered essential by some stakeholders, that's relevant information for the decision. The CCB needs to understand the full impact, including stakeholder concerns, to make an informed choice between schedule and scope.

Option A and B pre-judge without full analysis.

Option D excludes relevant input from the decision process.
