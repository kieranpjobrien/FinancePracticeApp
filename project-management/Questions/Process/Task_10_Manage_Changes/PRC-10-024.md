# Question

A team member identifies a defect in a deliverable that passed quality inspection. Fixing the defect requires changing the approved design. What type of change request is needed?

## Options

- **A**: Corrective action to fix the defect
- **B**: Preventive action to avoid similar defects
- **C**: Defect repair plus potentially a scope change for the design modification
- **D**: No change request since defect fixes are part of normal work

## Answer

C

## Explanation

The correct answer is C. Defect repair addresses the specific defect, but changing the approved design is a scope change that should go through change control. The combined request should address both the immediate fix and the design change, with appropriate impact analysis on the design modification.

Option A doesn't address the design change component.

Option B addresses future prevention but not the current situation.

Option D incorrectly assumes design changes don't need change control.
