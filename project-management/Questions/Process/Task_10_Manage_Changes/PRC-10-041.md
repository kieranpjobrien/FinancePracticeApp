# Question

A complete impact analysis for a change request should evaluate effects on which areas?

## Options

- **A**: Only the baseline that the change directly modifies.
- **B**: Cost and schedule, which are the financial and time dimensions.
- **C**: Scope, schedule, cost, quality, resources, risk, and other plan elements.
- **D**: Whatever areas the requester identifies as relevant to the change.

## Answer

C

## Explanation

A change to one element typically ripples to others (a scope addition affects schedule, cost, resources, quality, and risk). Comprehensive analysis surfaces those ripples so the CCB can decide with full information. (A) and (B) provide incomplete pictures. (D) defers analysis scope to the requester, who often does not see the full project impact.
