# Question

Impact analysis for a change request should evaluate effects on:

## Options

- **A**: Only the baseline that's directly affected
- **B**: Scope, schedule, cost, quality, risk, and other project areas
- **C**: Only cost and schedule
- **D**: Whatever the CCB requests

## Answer

B

## Explanation

The correct answer is B. Comprehensive impact analysis considers all potentially affected areas. A scope change might affect schedule (more work), cost (more resources), quality (more testing needed), and risk (new uncertainties). Partial analysis leads to incomplete decision-making.

Option A and C provide incomplete analysis.

Option D is reactive rather than applying standard thorough analysis.
