# Question

Version control primarily ensures:

## Options

- **A**: All team members use the same tools
- **B**: Changes to items are tracked and previous versions can be retrieved
- **C**: Only approved changes are implemented
- **D**: The project stays on schedule

## Answer

B

## Explanation

The correct answer is B. Version control tracks changes to items (documents, code, configurations) over time, maintains history of who changed what and when, and enables retrieval of previous versions. This supports configuration management and audit requirements.

Option A is about tool standardisation, not version control.

Option C describes change control, which is related but distinct from version control.

Option D is about schedule management, not version control.
