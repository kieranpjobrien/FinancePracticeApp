# Question

What is the PRIMARY input to the Perform Integrated Change Control process?

## Options

- **A**: Change requests
- **B**: The lessons learned register
- **C**: The project charter
- **D**: The stakeholder register

## Answer

A

## Explanation

The process exists to evaluate and decide on change requests; without them, the process has nothing to act on. (B) the lessons learned register informs decisions but is not the trigger. (C) the project charter is an enterprise input created in initiation, rarely changed during execution. (D) the stakeholder register supports analysis but does not initiate the change control process.
