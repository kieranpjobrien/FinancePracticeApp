# Question

The change log records:

## Options

- **A**: Only approved changes
- **B**: Only rejected changes
- **C**: All change requests and their current status
- **D**: Only changes that impact the baseline

## Answer

C

## Explanation

The correct answer is C. The change log is a comprehensive list of all changes submitted during the project along with their current status (submitted, under review, approved, rejected, deferred, implemented). It provides traceability and helps understand the history of change requests.

Option A and B are incomplete - both approved and rejected changes should be tracked.

Option D misses changes that don't impact the baseline but are still worth tracking.
