# Question

What is the PRIMARY purpose of a change control board (CCB)?

## Options

- **A**: To generate ideas for project improvements
- **B**: To review, evaluate, approve, defer, or reject changes to the project
- **C**: To implement approved changes
- **D**: To communicate changes to stakeholders

## Answer

B

## Explanation

The correct answer is B. The CCB is a formally chartered group responsible for reviewing, evaluating, approving, deferring, or rejecting changes to the project, and for recording and communicating such decisions. It ensures changes are properly evaluated before implementation.

Option A describes brainstorming or continuous improvement, not change control.

Option C describes execution of changes, not the CCB's decision-making role.

Option D describes communications management, though the CCB does communicate decisions.
