# Question

Two changes approved in separate CCB meetings turn out to be technically incompatible; implementing both would break the integration. The PM only realised this during implementation planning. What should the PM do?

## Options

- **A**: Implement the older change and unilaterally treat the newer one as superseded.
- **B**: Implement the newer change because it represents the CCB's most recent intent.
- **C**: Engineer a workaround that lets both changes coexist and proceed without going back to the CCB.
- **D**: Return both changes to the CCB with a conflict analysis and options (implement A only, implement B only, alternative approach, or revoke one) so the body can decide with full information.

## Answer

D

## Explanation

The CCB approved each change without knowing about the conflict. The decision rights still belong to the CCB and must be exercised with the new information. (A) and (B) make arbitrary choices the PM is not authorised to make. (C) assumes a workaround exists and bypasses the governance body that needs to decide which change stands.
