# Question

Two approved changes conflict with each other - implementing both would cause technical problems. The changes were approved in separate CCB meetings without recognising the conflict. What should the PM do?

## Options

- **A**: Implement the first change approved and reject the second
- **B**: Implement the second change since it's more recent
- **C**: Bring both changes back to the CCB with analysis of the conflict and options
- **D**: Find a technical workaround that implements both changes

## Answer

C

## Explanation

The correct answer is C. The CCB made decisions without complete information about the conflict. Rather than arbitrarily choosing, the PM should return to the CCB with analysis: what the conflict is, what the options are (implement A, implement B, implement neither, find alternative approach), and the implications of each. The CCB can then make an informed decision.

Options A and B make arbitrary choices without CCB input.

Option D assumes a workaround exists and is appropriate without analysis.
