# Question

An emergency occurs in production requiring immediate changes to the project deliverable. The change control board doesn't meet for another week. What should the PM do?

## Options

- **A**: Wait for the CCB meeting since all changes must go through formal approval
- **B**: Implement the emergency fix following the pre-defined emergency change procedure
- **C**: Get verbal approval from the sponsor and implement immediately
- **D**: Implement the change and submit the change request afterwards for documentation

## Answer

B

## Explanation

The correct answer is B. Well-designed change control processes include emergency procedures for time-critical situations. These typically define what constitutes an emergency, who can authorise emergency changes, and how they're documented afterwards. Following the emergency procedure maintains control while enabling necessary responsiveness.

Option A prioritises process over urgent business need.

Option C bypasses the defined emergency process if one exists.

Option D implements without any prior approval, which may not be authorised even for emergencies.
