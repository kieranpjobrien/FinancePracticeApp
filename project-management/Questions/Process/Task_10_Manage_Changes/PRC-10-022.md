# Question

Which document is MOST likely to be updated as a result of an approved change?

## Options

- **A**: Project charter
- **B**: Business case
- **C**: Project management plan or baselines
- **D**: Stakeholder register

## Answer

C

## Explanation

The correct answer is C. Approved changes that affect scope, schedule, or cost typically require updates to the project management plan and/or baselines. The plan documents how the project is executed; approved changes may modify that approach or the performance baseline against which progress is measured.

Option A is rarely changed after initiation.

Option B may be updated for significant changes but not for routine changes.

Option D captures stakeholder information, not project plans and baselines.
