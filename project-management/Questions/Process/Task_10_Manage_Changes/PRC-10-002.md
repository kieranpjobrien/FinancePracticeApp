# Question

A change control board (CCB) approves a change that the project manager believes will harm the project. The PM has presented concerns, but the CCB (which includes the sponsor) has decided to proceed. What should the PM do?

## Options

- **A**: Refuse to implement the change
- **B**: Implement the change while documenting their concerns and the CCB's decision
- **C**: Escalate above the sponsor to protect the project
- **D**: Resign from the project rather than implement a harmful change

## Answer

B

## Explanation

The correct answer is B. The PM advised, the CCB decided. Unless the change is unethical or illegal, the PM should implement while documenting their concerns. The decision authority rests with the CCB. Documentation protects the PM and provides a record if issues arise later.

Option A refuses legitimate governance authority. The PM advises; the CCB decides.

Option C escalates above the decision-making body, which undermines governance and likely damages the PM's relationship with leadership.

Option D is extreme for a disagreement within normal project governance.
