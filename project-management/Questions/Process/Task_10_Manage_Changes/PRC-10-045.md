# Question

A project has implemented numerous approved changes over 18 months. The original business case may no longer be valid given how much the project has evolved. What should the PM recommend?

## Options

- **A**: Continue since all changes were individually approved
- **B**: Commission a business case review to validate the project still delivers expected value
- **C**: Roll back changes to return to original scope
- **D**: Document the evolution as lessons learned for future projects

## Answer

B

## Explanation

The correct answer is B. Cumulative changes can fundamentally alter a project's value proposition. While each change was approved individually, the CCB may not have assessed whether the transformed project still aligns with original strategic objectives. A business case review ensures continued investment is justified.

Option A assumes individual approvals guarantee overall validity.

Option C is likely impractical and may not restore original value.

Option D captures lessons but doesn't address whether to continue the project.
