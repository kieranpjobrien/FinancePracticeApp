# Question

A project has multiple parallel work streams. A change to one work stream would require changes to three others. The PM of one affected work stream refuses to accept the cascading change. What should the PM do?

## Options

- **A**: Reject the original change since it can't be fully implemented
- **B**: Implement the change only in the originating work stream
- **C**: Escalate to program management for cross-work stream coordination
- **D**: Override the objecting PM's decision

## Answer

C

## Explanation

The correct answer is C. Cross-work stream changes require program-level coordination. Individual PMs may not have authority over each other's work streams. Program management can assess the overall benefit, mediate between work streams, and make decisions that balance program-wide objectives.

Option A rejects potentially valuable changes without proper escalation.

Option B creates inconsistency across related work streams.

Option D assumes authority the PM may not have.
