# Question

A project's quality costs are: Prevention $50,000, Appraisal $30,000, Internal Failure $40,000, External Failure $80,000. What is the cost of poor quality (COPQ)?

## Options

- **A**: $80,000
- **B**: $120,000
- **C**: $150,000
- **D**: $200,000

## Answer

B

## Explanation

The correct answer is B. Cost of Poor Quality (COPQ) includes only failure costs - both internal failure ($40,000) and external failure ($80,000). COPQ = $40,000 + $80,000 = $120,000.

Option A only includes external failure costs, missing internal failures.

Option C incorrectly adds appraisal costs. Appraisal (inspection, testing) is a cost of conformance, not a failure cost.

Option D represents total Cost of Quality (COQ), which includes all four categories. COPQ is a subset focusing only on failure costs.

Cost of Conformance = Prevention + Appraisal = $80,000
Cost of Non-Conformance (COPQ) = Internal Failure + External Failure = $120,000
Total COQ = $200,000
