# Question

A manufacturing project produces widgets. Control chart analysis shows the process is stable (in control) but 3% of output falls outside specification limits. The customer requires less than 1% defects. What type of action is needed?

## Options

- **A**: Identify and remove the special cause of variation
- **B**: Implement process changes to reduce common cause variation
- **C**: Tighten control limits to catch defects earlier
- **D**: Increase inspection frequency to catch more defects

## Answer

B

## Explanation

The correct answer is B. A stable process producing consistent defects indicates the problem is common cause variation - inherent to the process design. Meeting the 1% requirement requires fundamental process improvement (better equipment, different materials, improved methods) not just tighter controls.

Option A addresses special causes, but the process is stable (in control), meaning no special causes are present. The variation is built into the process.

Option C misunderstands control limits. They're calculated from the process, not set arbitrarily. Tightening them would just create false alarms.

Option D catches defects but doesn't reduce them. You'd still produce 3% defects; you'd just find more of them.
