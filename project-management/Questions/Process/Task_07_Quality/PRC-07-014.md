# Question

Which quality tool would be MOST appropriate for identifying the vital few causes responsible for the majority of defects?

## Options

- **A**: Control chart
- **B**: Fishbone diagram
- **C**: Pareto chart
- **D**: Scatter diagram

## Answer

C

## Explanation

The correct answer is C. A Pareto chart applies the 80/20 principle to identify the vital few causes that produce the majority of effects. It displays defect categories in descending order with a cumulative line, making it easy to identify which issues to address first for maximum impact.

Option A (control chart) monitors process stability over time but doesn't prioritise causes.

Option B (fishbone/Ishikawa diagram) helps brainstorm potential causes but doesn't quantify their relative impact.

Option D (scatter diagram) shows correlation between two variables but doesn't identify or prioritise defect causes.
