# Question

You want to capture lessons learned throughout the project rather than only at closure. How should the project manager implement this?

## Options

- **A**: Run a brief lessons-learned session after each major milestone or phase gate, capture insights while they are fresh, and feed them back into the current project's planning.
- **B**: Designate a single observer to maintain a running lessons document so the rest of the team can stay focused on delivery.
- **C**: Hold a single comprehensive lessons-learned session at project closure to ensure consistency and avoid duplication.
- **D**: Capture only negative lessons throughout the project, since positive practices are already understood and do not need recording.

## Answer

A

## Explanation

(A) reflects PMI's continuous learning view: capture early, capture often, and use the lessons in-flight. (B) limits lessons to one perspective. (C) loses detail and the chance to improve the running project. (D) discards "what worked" lessons that are equally valuable for replication.
