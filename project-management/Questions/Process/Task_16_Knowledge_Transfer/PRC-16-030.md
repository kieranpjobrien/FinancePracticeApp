# Question

What is "knowledge management" in project context?

## Options

- **A**: Filing project documents
- **B**: The systematic process of creating, sharing, using, and managing knowledge and information across the project and organization
- **C**: Controlling who can access information
- **D**: Training team members only

## Answer

B

## Explanation

The correct answer is B. Knowledge management encompasses the full lifecycle: creating knowledge (through work and learning), sharing it (documentation, training, collaboration), using it (applying to decisions and work), and managing it (maintaining, updating, archiving). It's a systematic approach to treating knowledge as a valuable organizational asset.

Option A describes only one aspect (documentation/filing).

Option C describes access control, which is related but not comprehensive.

Option D describes only one transfer method.
