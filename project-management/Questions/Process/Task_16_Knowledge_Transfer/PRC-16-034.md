# Question

What is the main benefit of conducting knowledge transfer continuously throughout a project, rather than only at closure?

## Options

- **A**: Information is captured while memory is fresh, the running project can use early learnings, and gradual capture is more sustainable than an end-of-project rush.
- **B**: It allows the project sponsor to closely monitor the team's reflection activities at every milestone.
- **C**: It guarantees that the lessons-learned register will be longer than projects that only capture at closure.
- **D**: It produces a calendar of recurring meetings that helps with project communications cadence.

## Answer

A

## Explanation

(A) is the standard set of benefits: freshness, in-flight applicability, and sustainability. (B) reframes the practice as oversight rather than learning. (C) treats register length as the goal, which it is not. (D) confuses a side-effect (additional ceremonies) with the substantive benefit.
