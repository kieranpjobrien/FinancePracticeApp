# Question

An organization wants to implement a mentoring program to transfer knowledge from senior project managers to junior ones. What elements would make this program effective?

## Options

- **A**: Pair people randomly and let them figure it out
- **B**: Have senior PMs write down everything they know
- **C**: Design the program with clear objectives, thoughtful mentor-mentee matching, structured interactions with flexibility, support for mentors, and mechanisms to capture learnings that benefit beyond individual pairs
- **D**: Make mentoring mandatory for promotion

## Answer

C

## Explanation

The correct answer is C. Effective mentoring programs need: clear objectives (what should be transferred), thoughtful matching (compatible pairs), structure with flexibility (regular touchpoints with room for organic interaction), mentor support (training, time allocation), and broader capture (sharing patterns and insights organizationally).

Option A provides no structure for success.

Option B captures explicit knowledge but not mentoring's relationship-based tacit transfer.

Option D creates compliance rather than genuine knowledge transfer.
