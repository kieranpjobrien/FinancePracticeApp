# Question

The project has sensitive information that should only be shared with certain stakeholders. However, team members are inadvertently sharing information too broadly in open channels. What should the PM do?

## Options

- **A**: Stop all communications to prevent leaks
- **B**: Establish clear information classification guidelines, train the team on what can be shared where, and implement appropriate communication channels for different sensitivity levels
- **C**: Blame team members publicly for mistakes
- **D**: Assume everyone can handle all information

## Answer

B

## Explanation

The correct answer is B. Establish governance for sensitive information. Classification: define categories (confidential, internal, public). Training: ensure team understands what falls in each category. Channels: create appropriate spaces for different types of communication (secure channels for sensitive discussions). This enables appropriate sharing while protecting sensitive information.

Option A prevents necessary communication.

Option C creates fear without providing guidance.

Option D ignores legitimate confidentiality requirements.
