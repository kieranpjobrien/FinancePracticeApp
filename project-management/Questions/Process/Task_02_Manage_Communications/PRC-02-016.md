# Question

Using the communication channels formula n(n-1)/2, how many channels exist for a team of 8?

## Options

- **A**: 16
- **B**: 28
- **C**: 36
- **D**: 64

## Answer

B

## Explanation

8 × (8 - 1) / 2 = 8 × 7 / 2 = 56 / 2 = 28. (B) is correct. (A) is 2n. (C) is n(n+1)/2, the sum-of-integers formula. (D) is n^2, ordered pairs including self-pairs.
