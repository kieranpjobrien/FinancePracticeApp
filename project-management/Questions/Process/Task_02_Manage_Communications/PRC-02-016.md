# Question

What is the formula for calculating potential communication channels in a project?

## Options

- **A**: n + 1
- **B**: n(n-1)/2 where n is the number of people
- **C**: n × 2
- **D**: n squared

## Answer

B

## Explanation

The correct answer is B. The formula n(n-1)/2 calculates the number of potential communication channels between n people. For example, with 5 people: 5(4)/2 = 10 channels. This demonstrates how communication complexity grows rapidly as team size increases, which is why larger teams require more structured communication approaches.

Options A, C, and D are incorrect formulas.
