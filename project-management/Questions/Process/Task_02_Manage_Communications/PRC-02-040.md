# Question

What is a project kickoff meeting primarily used for?

## Options

- **A**: To celebrate project completion
- **B**: To introduce the project, team, and stakeholders, establish expectations, and create shared understanding of objectives and approach
- **C**: To resolve project conflicts
- **D**: To conduct performance reviews

## Answer

B

## Explanation

The correct answer is B. The kickoff meeting launches the project formally. It typically covers: project objectives and scope, team introductions and roles, key stakeholders, timeline and milestones, communication approach, and initial expectations. It's an opportunity to build relationships and establish shared understanding at the project's start.

Option A describes a project celebration or closure event.

Option C would be addressed in issue resolution meetings.

Option D is an HR function, not a project meeting purpose.
