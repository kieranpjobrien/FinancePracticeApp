# Question

What is the formula for calculating the number of potential communication channels in a project?

## Options

- **A**: n + 1 (where n is the number of stakeholders)
- **B**: n(n-1)/2 (where n is the number of stakeholders)
- **C**: n x 2 (where n is the number of stakeholders)
- **D**: n squared (where n is the number of stakeholders)

## Answer

B

## Explanation

The correct answer is B. The formula n(n-1)/2 calculates the number of potential communication channels between n people. For example, with 5 stakeholders: 5(5-1)/2 = 5(4)/2 = 10 potential channels. This demonstrates why communication complexity grows rapidly as team/stakeholder size increases.

Option A would give 6 channels for 5 people, which is too low.

Option C would give 10 channels for 5 people (coincidentally correct for n=5, but wrong formula - it gives 20 for n=10 when the actual answer is 45).

Option D would give 25 channels for 5 people, which is too high and represents all possible ordered pairs rather than unique communication channels.
