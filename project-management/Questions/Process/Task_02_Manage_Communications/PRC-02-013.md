# Question

What is a stakeholder communication matrix?

## Options

- **A**: A mathematical formula for calculating communication frequency
- **B**: A tool that documents stakeholder communication requirements including who, what, when, and how they should receive project information
- **C**: A list of all project emails
- **D**: A phone directory

## Answer

B

## Explanation

The correct answer is B. A stakeholder communication matrix maps out communication requirements for each stakeholder or stakeholder group: what information they need, how often, in what format, through what channel, and who is responsible for providing it. It ensures consistent, appropriate communication to all stakeholders.

Option A misunderstands the "matrix" term.

Options C and D are specific tools, not a communication requirements framework.
