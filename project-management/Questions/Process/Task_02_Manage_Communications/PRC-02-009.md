# Question

Which document defines who receives what project information, when, and in what format?

## Options

- **A**: Project charter
- **B**: Communications management plan
- **C**: Stakeholder register
- **D**: Project status report

## Answer

B

## Explanation

The correct answer is B. The communications management plan defines the project's communication requirements - who needs information, what information they need, when they need it, how it will be delivered, and who is responsible for delivering it. It's a subsidiary plan of the project management plan.

Option A (project charter) authorises the project and identifies high-level stakeholders but doesn't detail communication requirements.

Option C (stakeholder register) identifies stakeholders and their interests but doesn't define the communication approach.

Option D (project status report) is an output of communications, not the plan that defines how communications work.
