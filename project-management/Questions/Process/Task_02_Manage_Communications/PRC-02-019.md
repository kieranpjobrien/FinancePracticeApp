# Question

The team uses multiple communication tools (email, chat, project management software, shared drives). Important information is scattered and hard to find. Team members waste time searching for information. What should the PM do?

## Options

- **A**: Add another tool to consolidate
- **B**: Establish clear guidelines on what information goes where, designate authoritative sources for different information types, and simplify the toolset where possible
- **C**: Let each person decide where to put information
- **D**: Move everything to email only

## Answer

B

## Explanation

The correct answer is B. Tool sprawl creates information fragmentation. Establish guidelines: decisions go here, documentation goes there, quick questions use chat. Designate authoritative sources: where is the "truth" for schedules, requirements, etc.? Simplify: perhaps some tools overlap and can be consolidated. Clear structure reduces search time and ensures information is findable.

Option A adds complexity without addressing the underlying structure problem.

Option C perpetuates the fragmentation.

Option D eliminates useful tools and doesn't solve the organisation problem.
