# Question

Which guideline best describes when to use formal versus informal communication?

## Options

- **A**: Always use formal communication with executives and informal communication with team members.
- **B**: Use formal communication for external parties only and informal for everyone inside the organisation.
- **C**: Use formal communication for the first message of any topic and informal for follow-ups.
- **D**: Use formal communication for decisions and audit trails; use informal for daily coordination and relationships.

## Answer

D

## Explanation

Formality should match purpose, not seniority or organisational boundary. (D) captures the standard guidance. (A) bases formality on seniority, which is a poor proxy. (B) treats internal/external as the dividing line, but internal contracts and audit items still need formal records. (C) treats formality as a function of message order rather than content type.
