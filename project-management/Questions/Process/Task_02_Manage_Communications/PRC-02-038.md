# Question

The project spans multiple organisations, each with different communication cultures and tools. Some use Slack, others Microsoft Teams, others email only. Integration is challenging. What approach should the PM take?

## Options

- **A**: Force everyone onto one platform
- **B**: Establish a communication hub approach with agreed touchpoints, designate bridge roles for cross-organisational communication, and respect that each organisation maintains their internal tools
- **C**: Let each organisation communicate independently
- **D**: Avoid cross-organisational communication

## Answer

B

## Explanation

The correct answer is B. Cross-organisational projects require pragmatic approaches to tool diversity. Hub approach: agree on shared touchpoints (perhaps a project-specific channel or regular meetings). Bridge roles: designate people responsible for ensuring information flows across organisational boundaries. Respect autonomy: each organisation may continue internal tools while connecting at agreed integration points.

Option A may not be possible across organisational boundaries.

Option C creates information silos and coordination failures.

Option D ignores a critical project need.
