# Question

A vendor responsible for a critical deliverable is repeatedly missing the contractually agreed 48-hour response window for project queries. Project work is being delayed. What should the PM do FIRST?

## Options

- **A**: Stop sending the vendor any further queries until they catch up on the existing backlog of unanswered messages.
- **B**: Begin parallel work in-house to build a backup capability that can replace the vendor if responsiveness doesn't improve.
- **C**: Issue a formal default notice citing breach of the contractual response time and copy the vendor's account executive.
- **D**: Document the missed responses, raise the issue through the vendor relationship channel defined in the contract, and seek to understand the cause before invoking penalties.

## Answer

D

## Explanation

The PMI sequence on vendor non-performance is to document, escalate through agreed channels, and seek understanding before invoking penalties. (D) does this. (A) blocks the project for a vendor problem. (B) builds parallel capability before exhausting contractual remedies. (C) jumps straight to formal default, which is usually a later step and damages the relationship if a remediable issue exists.
