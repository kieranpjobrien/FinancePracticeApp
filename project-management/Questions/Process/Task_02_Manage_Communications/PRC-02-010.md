# Question

What is the primary purpose of the communications management plan?

## Options

- **A**: To define how project information is created, distributed, stored, and disposed of throughout the project.
- **B**: To list every meeting on the project calendar with attendees and agendas.
- **C**: To track stakeholder engagement levels over time so the PM can update the engagement strategy.
- **D**: To establish the approval workflow for changes to project documents.

## Answer

A

## Explanation

The communications management plan defines the full lifecycle of project information — generation, distribution, storage, retrieval, and disposal. (A) is correct. (B) describes a meeting calendar, an output not a plan. (C) describes the stakeholder engagement assessment matrix, a different artefact. (D) describes change control procedures, which sit in the change management plan.
