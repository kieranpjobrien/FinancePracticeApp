# Question

What factors should be considered when determining the appropriate communication method for a message?

## Options

- **A**: Only the sender's preference
- **B**: Urgency, complexity, need for interaction, audience preferences, sensitivity, and need for documentation
- **C**: Only the technology available
- **D**: Only the message length

## Answer

B

## Explanation

The correct answer is B. Communication method should match the message needs. Urgency: time-sensitive information may need synchronous channels. Complexity: complex topics may need interactive discussion. Audience: their preferences and access. Sensitivity: confidential information needs secure channels. Documentation: some information needs a written record.

Options A, C, and D consider only single factors when multiple factors matter.
