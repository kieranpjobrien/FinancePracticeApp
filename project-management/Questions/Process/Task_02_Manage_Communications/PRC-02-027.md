# Question

Cost performance has slipped: the project is now 12% over budget at 60% complete. The PM is preparing the next sponsor briefing. What is the right approach?

## Options

- **A**: Withhold the variance until the team has had one more cycle to attempt recovery from within the existing budget.
- **B**: Present the variance with root-cause analysis, recovery options (scope, schedule, additional funding), the PM's recommendation, and a request for a decision.
- **C**: Brief the variance verbally only so it stays out of formal records and protects the project's reputation.
- **D**: Frame the report around delivery achievements with the variance discussed only if the sponsor specifically asks.

## Answer

B

## Explanation

PMI's communication ethic is timely, factual, and decision-supporting — variances above thresholds go to the sponsor with options and a recommendation, not as a hidden update. (B) is correct. (A) delays disclosure of material status. (C) keeps it off the record, which is a governance violation. (D) buries the lede.
