# Question

What is the difference between "push" and "pull" communication?

## Options

- **A**: Push is verbal, pull is written
- **B**: Push sends information to recipients directly; pull makes information available for recipients to access when needed
- **C**: Push is for external stakeholders; pull is for internal
- **D**: There is no meaningful difference

## Answer

B

## Explanation

The correct answer is B. Push communication delivers information to recipients (emails, reports, presentations). Pull communication makes information available for recipients to access when they choose (wikis, shared drives, dashboards). Both have appropriate uses: push for time-sensitive information requiring action, pull for reference material and ongoing status visibility.

Option A relates to format, not push/pull distinction.

Option C relates to audience, not delivery method.

Option D ignores meaningful distinctions.
