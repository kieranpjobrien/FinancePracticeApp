# Question

A project has multiple contracts with different vendors whose work is interdependent. One vendor's delay is affecting others. What procurement management challenge does this represent?

## Options

- **A**: Poor vendor selection
- **B**: Insufficient contract terms
- **C**: Multi-vendor coordination complexity
- **D**: Inadequate procurement planning

## Answer

C

## Explanation

The correct answer is C. When multiple vendors have interdependent work, coordination becomes a critical PM responsibility. Delays in one contract cascade to others. This requires clear interfaces, coordinated schedules, and often contract terms that address multi-vendor scenarios. It's an inherent complexity of multi-vendor environments.

Options A, B, and D may contribute to the situation but don't name the core challenge.
