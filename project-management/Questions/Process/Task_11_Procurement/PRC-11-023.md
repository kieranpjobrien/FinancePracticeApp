# Question

What is the purpose of source selection criteria?

## Options

- **A**: To list all vendors who are qualified to bid
- **B**: To define how proposals will be evaluated and scored
- **C**: To describe what the buyer wants to procure
- **D**: To document the selected vendor's qualifications

## Answer

B

## Explanation

The correct answer is B. Source selection criteria define the factors that will be used to evaluate proposals and their relative importance. Criteria might include price, technical approach, past performance, qualifications, and schedule. Having pre-defined criteria ensures objective, defensible selection decisions.

Option A describes a qualified sellers list.

Option C describes the procurement SOW.

Option D describes documentation that comes after selection.
