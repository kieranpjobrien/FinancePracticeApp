# Question

The project needs a capability the organisation has never delivered before and likely will not need again. Which factor most strongly favours buying rather than building in the make-or-buy analysis?

## Options

- **A**: The capability is core to the organisation's competitive advantage.
- **B**: The internal team has spare capacity in the relevant disciplines.
- **C**: The capability is a one-time need the organisation has no skills for and would not reuse.
- **D**: The project's timeline is generous and allows for skill development.

## Answer

C

## Explanation

A one-time, non-strategic capability with no internal skills is the textbook buy case: outsourcing avoids investing in skills the organisation will not exercise again. (A) is a make argument because core capabilities should not be outsourced. (B) and (D) are make arguments because internal capacity and time are available to build the skill.
