# Question

A vendor requests a scope change to the contract. The change would benefit the vendor but has no clear benefit to the project. The contract allows for changes with mutual agreement. What should the PM do?

## Options

- **A**: Reject the change since it only benefits the vendor
- **B**: Accept the change to maintain a positive relationship
- **C**: Evaluate the change's impact on the project and negotiate for something in return if proceeding
- **D**: Refer to legal for interpretation of the change clause

## Answer

C

## Explanation

The correct answer is C. Contract changes should provide value to both parties. If the vendor wants something that doesn't inherently benefit the buyer, the PM should negotiate. Perhaps the vendor offers schedule improvement, price reduction, or other value in exchange. This maintains a balanced commercial relationship.

Option A rejects without exploring possibilities.

Option B gives something for nothing.

Option D isn't necessary for straightforward commercial negotiation.
