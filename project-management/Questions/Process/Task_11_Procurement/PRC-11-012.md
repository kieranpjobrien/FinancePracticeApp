# Question

Three vendors have submitted proposals. Vendor A is cheapest but has weak references; Vendor B has strong references but the highest price; Vendor C is mid-range on both. Members of the procurement evaluation team are arguing for different vendors. What should the PM do?

## Options

- **A**: Apply the source selection criteria and weightings agreed before the RFP, score each proposal, and recommend accordingly.
- **B**: Recommend Vendor A because procurement is ultimately about achieving the lowest defensible cost outcome.
- **C**: Recommend Vendor B because strong reference quality materially reduces downstream execution risk.
- **D**: Recommend Vendor C as a defensible middle path that placates both camps and avoids conflict.

## Answer

A

## Explanation

The pre-defined source selection criteria exist precisely so vendor decisions are objective and traceable. The PM should apply them rather than letting subjective preferences decide. (B), (C), and (D) make subjective choices that bypass the agreed evaluation framework, exposing the procurement to challenge and bias.
