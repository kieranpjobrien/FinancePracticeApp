# Question

Three vendors submitted proposals. Vendor A has the lowest price but questionable references. Vendor B has excellent references but highest price. Vendor C is mid-range on both factors. The procurement team is split. How should the PM guide the selection?

## Options

- **A**: Select Vendor A since procurement should prioritise cost
- **B**: Select Vendor B since quality and reliability outweigh cost
- **C**: Apply the pre-defined source selection criteria weightings to score all proposals objectively
- **D**: Select Vendor C as the compromise choice

## Answer

C

## Explanation

The correct answer is C. Source selection criteria should be defined before proposals are received, with weightings that reflect project priorities. Applying these criteria objectively removes bias and ensures the selection aligns with stated priorities. If the criteria weren't pre-defined, that's a process improvement for future procurements.

Options A, B, and D make subjective choices without applying agreed criteria.
