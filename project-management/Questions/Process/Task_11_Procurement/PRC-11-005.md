# Question

In a fixed-price contract, who bears the risk of cost overruns?

## Options

- **A**: The buyer
- **B**: The seller
- **C**: Risk is shared equally
- **D**: The project manager

## Answer

B

## Explanation

The correct answer is B. In a fixed-price (or firm fixed-price) contract, the seller agrees to deliver specified work for a set price. If costs exceed the agreed amount, the seller absorbs the loss. This is why sellers build contingency into fixed-price bids.

Option A (buyer) describes cost-reimbursable contracts, where the buyer pays actual costs plus a fee.

Option C (shared equally) describes some contract types (e.g., cost-plus-incentive-fee with shared savings/overruns) but not pure fixed-price.

Option D (project manager) is not a contract party and doesn't bear financial risk in this sense.
