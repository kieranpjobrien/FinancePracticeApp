# Question

A project has been running for six months with significant investment, but market conditions have changed dramatically. The original business case is no longer valid, and continuing would deliver negative value. What should the PM recommend?

## Options

- **A**: Continue since significant investment has been made (sunk cost)
- **B**: Present the changed circumstances to sponsors and recommend terminating or pivoting the project to avoid further investment in negative value
- **C**: Hide the situation and hope conditions improve
- **D**: Speed up delivery to finish before anyone notices

## Answer

B

## Explanation

The correct answer is B. Sunk cost fallacy should not drive project decisions. If a project will deliver negative value, the responsible action is to present facts to decision-makers and recommend appropriate action - either termination or significant pivot. Continuing to invest in negative value destroys organisational resources.

Option A falls for the sunk cost fallacy.

Option C is unethical and will eventually be discovered.

Option D delivers something of negative value faster.
