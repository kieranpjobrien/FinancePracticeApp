# Question

A portfolio manager asks the PM to slow delivery for two months to free engineers for a higher-priority initiative. The current project is on schedule and producing measurable benefits. How should the PM respond?

## Options

- **A**: Comply immediately and adjust the schedule baseline; portfolio decisions outrank the project.
- **B**: Decline politely and continue at current pace until the sponsor formally redirects the team.
- **C**: Quantify the cost-of-delay (deferred benefits, ramp-up costs, risks) and bring it to the portfolio manager so the trade-off is decided with full information.
- **D**: Negotiate to release only the most junior engineers so the critical path stays protected.

## Answer

C

## Explanation

Portfolio-level decisions sit above the PM, but the PM owes the decision-maker the data they don't have. (C) supports the portfolio decision with evidence and is the PMI-aligned move. (A) accepts impact without surfacing trade-offs the portfolio manager may not have considered. (B) puts the project ahead of the portfolio, which is the wrong stance. (D) negotiates a specific solution before the trade-off has even been quantified — the portfolio manager may pick a different lever once they see the data.
