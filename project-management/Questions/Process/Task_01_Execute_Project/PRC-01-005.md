# Question

A project manager notices the team is spending significant time perfecting a feature that meets requirements but could be improved further. The feature is not on the critical path, and there are other deliverables waiting. What should the PM do?

## Options

- **A**: Allow the team to continue since quality is important
- **B**: Encourage the team to move on to other deliverables once requirements are met
- **C**: Assign additional resources to speed up the perfection work
- **D**: Remove the feature from scope entirely

## Answer

B

## Explanation

The correct answer is B. When a feature meets requirements, additional refinement has diminishing returns. The PM should guide the team to focus on work that delivers more value - other waiting deliverables. This is about executing with appropriate urgency and avoiding gold-plating.

Option A (continue perfecting) misallocates effort. While quality matters, exceeding requirements on one item while others wait isn't optimal value delivery.

Option C (add resources) wastes resources on diminishing returns work when they could be used on higher-value deliverables.

Option D (remove feature) is too extreme. The feature is done and meets requirements - there's no reason to remove it.
