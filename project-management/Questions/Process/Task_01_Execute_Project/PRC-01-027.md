# Question

A project is using a hybrid approach with waterfall phases for regulatory deliverables and agile sprints for feature development. The regulatory phase is blocked waiting for external approvals. How should the PM handle this to maintain value delivery?

## Options

- **A**: Stop all work until regulatory approval is received
- **B**: Continue agile feature development on items not dependent on regulatory approval while managing the regulatory track separately
- **C**: Abandon the hybrid approach
- **D**: Pressure the regulator for faster approval

## Answer

B

## Explanation

The correct answer is B. Hybrid approaches allow different workstreams to proceed at appropriate paces. Continue agile development on features that don't depend on the blocked regulatory work. Manage the regulatory track separately, maintaining communication with the approving body. This maximises value delivery despite the block.

Option A unnecessarily stops productive work.

Option C abandons an approach that may still be appropriate.

Option D is unlikely to be effective and may damage relationships.
