# Question

A project delivered a system that requires ongoing operational support. The project has completed its closure activities, but the operations team is still calling project team members with questions daily, effectively keeping them from fully transitioning to new assignments. How should this be managed?

## Options

- **A**: Tell operations to stop calling
- **B**: Keep project team members available indefinitely
- **C**: Establish a formal transition support period with defined scope and duration, after which operations assumes full responsibility with defined escalation paths for exceptional issues
- **D**: Have the project manager personally answer all questions

## Answer

C

## Explanation

The correct answer is C. Transitions need structure. A formal support period (with clear scope and end date) sets expectations. After that period, operations owns the system fully. Escalation paths for exceptional issues provide a safety net without keeping project resources permanently available. This balances support with enabling team transition.

Option A cuts off support abruptly without transition.

Option B never achieves full transition.

Option D isn't sustainable and may not address technical questions.
