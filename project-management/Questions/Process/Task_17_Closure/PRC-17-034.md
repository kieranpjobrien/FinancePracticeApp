# Question

Why is it important to update organizational process assets at project closure?

## Options

- **A**: It is not important
- **B**: To make project information available for future projects - templates, lessons learned, and historical data that can improve future project performance
- **C**: To satisfy auditors only
- **D**: To create work for the PMO

## Answer

B

## Explanation

The correct answer is B. Organizational process assets (templates, lessons learned, historical data) improve over time as projects contribute learnings. Updated templates reflect improvements discovered during the project. Lessons learned help future projects avoid mistakes and replicate successes. Historical data improves estimation and planning.

Option A ignores the value of organizational learning.

Option C understates the operational value.

Option D views it as make-work rather than value creation.
