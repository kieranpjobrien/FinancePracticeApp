# Question

A project is completing and transitioning to operations. The operations team has received training, but during the handover period, they frequently contact the project team with questions. The project team is eager to move on to new projects. What should the project manager ensure?

## Options

- **A**: Immediately transition all responsibility to operations to free the project team
- **B**: Keep the project team assigned indefinitely until operations is fully comfortable
- **C**: Establish a defined transition support period with clear escalation paths, then formally close the project
- **D**: Document all questions and answers for future reference

## Answer

C

## Explanation

The correct answer is C. Transitions need a defined support period where the project team remains available for questions, followed by formal closure. This balances operations' need for support with the project team's need to move on. Clear escalation paths ensure operations knows where to go for help after the project closes.

Option A abandons operations too quickly, risking operational problems and damage to the project team's reputation.

Option B leaves the project indefinitely open, preventing team members from focusing on new work and never achieving closure.

Option D is useful but doesn't address the transition structure. Documentation supports knowledge transfer but doesn't replace it.
