# Question

A project involved multiple vendors with different contract completion timelines. Some vendors completed their work months ago, others are just finishing. How should contract closure be managed?

## Options

- **A**: Wait until all vendors are complete to close any contracts
- **B**: Close each contract as it completes, regardless of overall project status
- **C**: Close each vendor contract as their work completes (verifying deliverables, settling payments, confirming warranty terms), while maintaining project oversight until all contracts are closed
- **D**: Let the procurement department handle all contracts without PM involvement

## Answer

C

## Explanation

The correct answer is C. Each contract should be closed as the vendor's work completes - this provides timely closure for the vendor and clean books for the project. The PM maintains oversight to ensure all contracts are properly closed as part of project closure. Waiting delays vendor closure unnecessarily; rushing loses oversight.

Option A delays closure unnecessarily for completed work.

Option B may close contracts without proper PM verification.

Option D removes PM oversight from project-related contracts.
