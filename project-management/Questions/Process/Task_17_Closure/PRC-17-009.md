# Question

During project closure, the project manager discovers that several contracts with vendors are still open with unresolved terms (pending payments, unreturned materials, outstanding warranties). What should be done?

## Options

- **A**: Close the project and let procurement handle the contracts later
- **B**: Keep the project open indefinitely until all contracts are naturally resolved
- **C**: Work with procurement to resolve or formally close each contract - settle payments, return materials, transfer warranties to operations - as part of project closure activities
- **D**: Ignore the contracts since deliverables are complete

## Answer

C

## Explanation

The correct answer is C. Contract closure is an essential part of project closure. Each contract should be formally closed: final payments made, materials returned or transferred, warranties documented and transferred to whoever will manage them going forward. This prevents financial exposure and ensures proper accountability transfer.

Option A leaves unresolved financial and contractual obligations.

Option B delays closure for potentially minor issues.

Option D creates risk from unmanaged contractual obligations.
