# Question

A project manager is preparing the final project report. Which elements are MOST important to include?

## Options

- **A**: Only the positive outcomes to maintain team reputation
- **B**: Detailed documentation of every project activity
- **C**: Summary of what was delivered versus planned, key metrics (schedule, cost, quality), major challenges and how they were addressed, lessons learned, and recommendations for future projects
- **D**: List of team member names only

## Answer

C

## Explanation

The correct answer is C. An effective final report captures the project's story: what was accomplished versus plans, key performance metrics, significant challenges and their resolution, and forward-looking elements (lessons learned, recommendations). This provides value for organizational learning and stakeholder communication without excessive detail.

Option A omits valuable learning from challenges.

Option B is too detailed for an executive summary report.

Option D lacks substantive content about project outcomes.
