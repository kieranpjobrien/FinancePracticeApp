# Question

A project manager is planning closure for a project that involved significant security-sensitive work. What special considerations apply to closure?

## Options

- **A**: No special considerations - close like any project
- **B**: Keep all documents indefinitely without disposing of anything
- **C**: Ensure secure handling of sensitive data, proper access control during archiving, compliance with security requirements for document retention and disposal, and appropriate clearance verification for anyone involved in closure activities
- **D**: Delete all security-sensitive documents immediately

## Answer

C

## Explanation

The correct answer is C. Security-sensitive projects require secure closure. Data must be handled securely even during closure. Archived documents need appropriate access controls. Retention and disposal must follow security requirements. Personnel involved in closure may need appropriate clearances. These considerations extend standard closure practices.

Option A ignores security requirements.

Option B doesn't address secure handling and may violate retention policies.

Option D may violate retention requirements and lose valuable information.
