# Question

What is a key objective of the project closure phase?

## Options

- **A**: To start planning the next project
- **B**: To formally complete all project activities, verify deliverables are accepted, and transition ownership while capturing knowledge for organizational benefit
- **C**: To assign blame for any problems
- **D**: To spend remaining budget

## Answer

B

## Explanation

The correct answer is B. Closure objectives include: completing all activities (not leaving loose ends), obtaining formal acceptance (confirming customer/sponsor agreement), transitioning ownership (to operations/support), and capturing knowledge (lessons learned, documentation) for organizational benefit.

Option A is a separate activity, not a closure objective.

Option C focuses on blame rather than proper completion and learning.

Option D is inappropriate financial management.
