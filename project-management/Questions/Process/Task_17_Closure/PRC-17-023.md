# Question

A project has been ongoing for three years and team composition has changed significantly - many original members have left and new members joined. The project manager is concerned about capturing comprehensive lessons learned when many people with early project knowledge are no longer available. What approach would help?

## Options

- **A**: Skip lessons learned since key people are gone
- **B**: Only gather lessons from current team members
- **C**: Capture lessons from current team, review any documented learnings from earlier phases, and if practical, reach out to key former team members for input on significant early-project experiences
- **D**: Fabricate lessons learned from documentation

## Answer

C

## Explanation

The correct answer is C. Comprehensive lessons learned requires creative approaches when team composition has changed. Current team provides recent learnings. Earlier documentation may capture some historical insights. Former key team members may be reachable for important early experiences. This multi-source approach maximizes coverage given the constraints.

Option A loses the opportunity to capture available knowledge.

Option B misses earlier project learnings.

Option D isn't authentic lessons learned.
