# Question

Which document formally authorises the completion of a project and the release of project resources?

## Options

- **A**: Project charter
- **B**: Final project report
- **C**: Project closure report or sign-off
- **D**: Lessons learned register

## Answer

C

## Explanation

The correct answer is C. The project closure report (or formal sign-off document) confirms that project objectives have been met (or the project is otherwise concluded), all deliverables have been accepted, and resources can be released. It's the formal end of the project.

Option A (project charter) authorises project start, not end. The charter initiates the project.

Option B (final project report) documents what happened but doesn't necessarily constitute formal authorisation to close.

Option D (lessons learned register) captures knowledge but doesn't authorise closure or resource release.
