# Question

A project team has worked intensively for a year to deliver a successful project. At closure, the project manager is focused on administrative closeout activities. What important activity might be overlooked?

## Options

- **A**: Creating more documentation
- **B**: Starting the next project immediately
- **C**: Celebrating success and recognizing team contributions - this supports morale, reinforces positive behaviors, and provides closure for the team
- **D**: Finding problems to fix

## Answer

C

## Explanation

The correct answer is C. Celebration and recognition are often overlooked closure activities, but they matter. Acknowledging team effort and success provides psychological closure, reinforces positive behaviors, strengthens relationships for future collaboration, and supports morale before team members move on.

Option A adds work without clear value at closure.

Option B doesn't allow for proper closure of the current project.

Option D focuses on negatives rather than celebrating accomplishment.
