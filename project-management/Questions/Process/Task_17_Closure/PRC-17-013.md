# Question

Between project phases, the project manager needs to ensure smooth transition while maintaining momentum. What is the BEST approach?

## Options

- **A**: Stop all work during the phase transition
- **B**: Continue without any transition activities
- **C**: Conduct a phase gate review to confirm phase completion, update plans based on learnings, ensure resources and information needed for the next phase are ready, and obtain approval to proceed
- **D**: Start the next phase before completing the current one

## Answer

C

## Explanation

The correct answer is C. Phase transitions are controlled checkpoints. The gate review confirms the previous phase met its objectives. Plan updates incorporate learnings. Resource and information readiness ensures the next phase can start effectively. Approval ensures appropriate oversight before proceeding. This maintains both control and momentum.

Option A loses momentum unnecessarily.

Option B misses opportunities to adjust based on phase learnings.

Option D starts before proper phase completion, risking rework.
