# Question

A project's closure is delayed because key stakeholders are busy and unavailable for acceptance meetings. The project team is being reassigned. What should the project manager do?

## Options

- **A**: Close without stakeholder acceptance
- **B**: Keep the entire team waiting indefinitely
- **C**: Maintain minimal resources for closure activities, escalate the urgency to stakeholder management, offer flexible acceptance methods (written approval, brief virtual meetings), and complete what can be done while waiting
- **D**: Reassign all team members and return for closure later

## Answer

C

## Explanation

The correct answer is C. Closure delays need management. Escalation raises awareness of the urgency. Flexible acceptance methods accommodate busy schedules. Minimal resources keep closure moving without tying up the full team. Completing other closure activities (documentation, archiving) uses wait time productively.

Option A completes without proper acceptance.

Option B wastes team resources.

Option D loses focus and may never achieve proper closure.
