# Question

A project is being closed, but several lessons learned have been identified that could benefit future projects. What is the MOST effective way to ensure these lessons are used?

## Options

- **A**: Document lessons learned in the project archive
- **B**: Email the lessons to all project managers in the organization
- **C**: Document lessons, present key findings to relevant groups, and work with PMO to integrate into organizational processes and training
- **D**: Add lessons to the project management methodology document

## Answer

C

## Explanation

The correct answer is C. Lessons learned have value only if they're actually used. Multiple transfer mechanisms increase the likelihood: documentation preserves the knowledge, presentations reach current practitioners, and integration into processes and training embeds the learning for future projects.

Option A creates documentation that may never be read.

Option B may be ignored amid email volume.

Option D adds to a document but doesn't ensure people read or apply it.
