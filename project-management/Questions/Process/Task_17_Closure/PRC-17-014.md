# Question

What is the primary purpose of a project closure checklist?

## Options

- **A**: To document evidence that the project complied with audit requirements.
- **B**: To assign each closure task to a named owner before the project ends.
- **C**: To ensure all required closure activities are completed systematically and none are missed in the rush to wrap up.
- **D**: To estimate how long the closure phase will take to plan resource availability.

## Answer

C

## Explanation

Closure spans many activities such as acceptance, contracts, archiving, lessons learned, resource release, and benefits handoff, and a checklist is the practical tool to ensure none are missed. (A) audit evidence is a downstream benefit, not the primary purpose. (B) assignment is useful but is a separate activity from the checklist itself. (D) estimating duration is part of closure planning but is not what the checklist is for.
