# Question

A project network has Path A = 15 days, Path B = 18 days, Path C = 12 days, and Path D = 18 days. Paths B and D share no activities. Which statement BEST describes this schedule?

## Options

- **A**: The project will complete in 63 days because all path durations must be added together.
- **B**: Path B is the only critical path because it appears first in the network analysis.
- **C**: Path C has 6 days of total float because the project end date is set by Paths B and D.
- **D**: Path A is the critical path because it lies between the longest and shortest paths.

## Answer

C

## Explanation

The critical path is the longest path; here, Paths B and D are tied at 18 days, so both are critical and the project duration is 18 days. Path C has 18 - 12 = 6 days of total float. (C) is correct. (A) wrongly adds path durations rather than taking the longest. (B) ignores that Path D is equally long and therefore equally critical. (D) misunderstands criticality — it is determined by length, not position.
