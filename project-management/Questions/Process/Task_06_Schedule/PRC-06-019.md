# Question

A project network has Path A (15 days), Path B (18 days), Path C (12 days), and Path D (18 days). Paths B and D share no activities. What is true about this schedule?

## Options

- **A**: Path B is the only critical path
- **B**: The project duration is 18 days, and both Path B and Path D are critical paths
- **C**: The project duration is 63 days
- **D**: Path A has 3 days of float

## Answer

B

## Explanation

The correct answer is B. The critical path is the longest path(s). Both B and D are 18 days, making them both critical. The project cannot complete faster than 18 days. Having multiple critical paths increases schedule risk since delays on either path impact the project.

Option A ignores that D is equally long.

Option C incorrectly adds all paths.

Option D: Path A's float is 18-15 = 3 days, which is correct, but B is the better answer as it addresses the critical path question.
