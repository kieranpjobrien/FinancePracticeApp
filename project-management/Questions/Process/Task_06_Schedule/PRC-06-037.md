# Question

What is the purpose of the forward pass in critical path method?

## Options

- **A**: To move the project forward
- **B**: To calculate the early start and early finish dates for each activity
- **C**: To identify late activities
- **D**: To approve the schedule

## Answer

B

## Explanation

The correct answer is B. The forward pass moves through the network from start to finish, calculating the earliest each activity can start (Early Start) and finish (Early Finish) based on dependencies. This determines the minimum project duration.

Option A misunderstands "forward pass."

Option C is related but not the purpose - late dates come from backward pass.

Option D is a governance activity, not a calculation.
