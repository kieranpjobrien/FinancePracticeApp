# Question

A project has the following activity dependencies: A(3 days) → B(5 days) → D(4 days), and A(3 days) → C(2 days) → D(4 days). What is the critical path and project duration?

## Options

- **A**: A-B-D, 12 days
- **B**: A-C-D, 9 days
- **C**: A-B-D, 9 days
- **D**: Both paths are critical, 12 days

## Answer

A

## Explanation

The correct answer is A. To find the critical path, calculate the duration of each path:

**Path A-B-D:** 3 + 5 + 4 = **12 days**
**Path A-C-D:** 3 + 2 + 4 = **9 days**

The critical path is the longest path through the network. A-B-D takes 12 days, making it the critical path. The project duration is determined by the critical path: **12 days**.

Path A-C-D has 3 days of float (12 - 9 = 3 days slack).

Option B identifies the shorter path, which is not critical.

Option C has the right path but wrong duration.

Option D is incorrect - only the longest path is critical.
