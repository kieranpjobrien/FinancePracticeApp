# Question

A critical-path activity has finished two days late. The project deadline is contractually fixed. What should the project manager do FIRST?

## Options

- **A**: Submit a change request to extend the project deadline by two days to reflect the slip.
- **B**: Analyse remaining critical-path activities for crashing and fast-tracking, then present recovery options with impact.
- **C**: Add resources across the entire project to recover the lost time as quickly as is possible.
- **D**: Inform the customer that a two-day slip is now baked into the project end date going forward.

## Answer

B

## Explanation

PMI's first response to a critical-path slip is to analyse recovery options before changing the baseline or notifying customers. (B) does that. (A) goes to formal change control before any recovery has been attempted. (C) wastes effort on non-critical work and may create new bottlenecks; only critical-path activities affect the end date. (D) communicates a slip the PM has not yet tried to recover, which is premature and undermines the contractual commitment.
