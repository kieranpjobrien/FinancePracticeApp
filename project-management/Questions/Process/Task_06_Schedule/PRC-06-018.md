# Question

The team's estimates are consistently optimistic - tasks regularly take longer than estimated. This pattern is causing schedule slippage. How should the PM address this?

## Options

- **A**: Punish the team for inaccurate estimates
- **B**: Analyse historical data on estimate accuracy, apply appropriate contingency factors, work with the team on improving estimation practices, and use ranges rather than single-point estimates
- **C**: Double all estimates arbitrarily
- **D**: Stop estimating

## Answer

B

## Explanation

The correct answer is B. Systematic optimism requires systematic correction. Analyse: how much are estimates typically off? Contingency: apply data-driven factors to adjust. Improve practices: use techniques like planning poker, historical comparison, or expert review. Ranges: acknowledge uncertainty with optimistic/likely/pessimistic estimates.

Option A discourages future estimation efforts.

Option C applies arbitrary correction without data.

Option D loses valuable planning information.
