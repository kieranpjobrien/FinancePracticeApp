# Question

The project schedule was developed using rolling wave planning. Detailed planning exists for the next phase; later phases have only high-level estimates. A stakeholder demands detailed schedules for all phases now. How should the PM respond?

## Options

- **A**: Create detailed schedules for all phases immediately
- **B**: Explain that rolling wave planning provides appropriate detail as knowledge increases - detailed planning for uncertain future phases creates false precision. Offer to update as each phase approaches.
- **C**: Refuse to provide any future timeline
- **D**: Inflate the schedule to cover uncertainty

## Answer

B

## Explanation

The correct answer is B. Rolling wave planning is a deliberate approach to progressive elaboration. Explain: detailed schedules for poorly understood future work creates false precision. Offer: as each phase approaches and understanding improves, provide detailed schedules. This manages expectations while maintaining schedule integrity.

Option A creates unreliable detail.

Option C doesn't meet legitimate planning needs.

Option D hides uncertainty rather than managing it.
