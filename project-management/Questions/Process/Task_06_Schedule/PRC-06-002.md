# Question

Which of the following BEST describes the critical path of a project schedule network?

## Options

- **A**: The sequence of activities with the highest concentration of project resources.
- **B**: The longest path through the network, which determines the minimum project duration and has zero total float.
- **C**: The set of activities flagged by stakeholders as the highest priority for delivery.
- **D**: The path through the network most likely to encounter risks during execution.

## Answer

B

## Explanation

(B) is the PMI definition: the critical path is the longest path through the network and therefore sets the shortest duration in which the project can finish; activities on it have zero total float. (A) confuses resource intensity with schedule criticality. (C) confuses business priority with schedule criticality. (D) confuses "critical" in the schedule sense with project risk exposure.
