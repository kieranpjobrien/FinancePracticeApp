# Question

A mega-project has hundreds of activities and the critical path calculation takes hours to run. During execution, frequent recalculation is needed but causes delays in decision-making. What approach could help?

## Options

- **A**: Stop calculating critical path
- **B**: Use sub-networks with interface milestones, focus detailed analysis on high-risk areas, use approximate methods for quick assessments, and run full calculations periodically
- **C**: Make all decisions without schedule analysis
- **D**: Simplify the project to have fewer activities

## Answer

B

## Explanation

The correct answer is B. Large project schedules require practical approaches. Sub-networks: break into manageable pieces linked by milestones. Focus: detailed analysis where risk is highest. Approximate methods: quick assessments for routine decisions. Full calculation: periodically run complete analysis for baseline updates. This balances analytical rigor with practical decision-making speed.

Option A loses critical information.

Option C makes uninformed decisions.

Option D isn't feasible for genuinely complex projects.
