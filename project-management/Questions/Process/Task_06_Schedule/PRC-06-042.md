# Question

A hybrid project uses waterfall for infrastructure and agile for application development. The agile stream depends on infrastructure milestones. How should the PM integrate these schedules?

## Options

- **A**: Maintain completely separate schedules
- **B**: Create an integrated schedule with waterfall milestones as constraints for agile sprints, ensure clear handoff points, and coordinate planning across both streams
- **C**: Force agile onto waterfall cadence
- **D**: Force waterfall onto agile cadence

## Answer

B

## Explanation

The correct answer is B. Hybrid requires integrated scheduling. Milestone constraints: infrastructure milestones become inputs for agile planning. Clear handoffs: define what's needed, when. Coordinated planning: both streams should understand dependencies. This allows each approach to work appropriately while managing interdependencies.

Option A ignores critical dependencies.

Options C and D force inappropriate methodologies on work that requires different approaches.
