# Question

A project has a contractual penalty of $10,000 per day for late delivery. The project is forecast to be 5 days late. Crashing the schedule would cost $8,000 per day saved. What should the PM recommend?

## Options

- **A**: Accept the penalties - it's cheaper than crashing
- **B**: Crash to recover the schedule since the cost ($8,000/day) is less than the penalty ($10,000/day), saving $2,000 per day recovered
- **C**: Crash some days and accept penalties for others based on detailed cost analysis
- **D**: Ignore the financial implications

## Answer

B

## Explanation

The correct answer is B. Financial analysis: penalty = $10,000/day, crash cost = $8,000/day. Crashing saves $2,000 per day recovered. For 5 days: penalty would be $50,000; crashing costs $40,000, saving $10,000. Beyond pure math, consider: reputation damage from late delivery, relationship impacts, and whether crashing is actually achievable.

Option A accepts higher cost.

Option C could apply if crashing costs vary by day.

Option D ignores material project implications.
