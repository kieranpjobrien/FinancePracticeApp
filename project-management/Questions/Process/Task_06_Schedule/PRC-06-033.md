# Question

What is a schedule baseline?

## Options

- **A**: The first draft of the schedule
- **B**: The approved version of the schedule against which performance is measured, which can only be changed through formal change control
- **C**: The most optimistic schedule
- **D**: A template for schedules

## Answer

B

## Explanation

The correct answer is B. The schedule baseline is the approved schedule plan against which actual progress is measured. It includes planned start and finish dates for all activities and milestones. Changes to the baseline require formal change control approval - it's the reference point for performance measurement.

Option A is preliminary, not approved baseline.

Option C describes one scenario, not the approved plan.

Option D is a different concept.
