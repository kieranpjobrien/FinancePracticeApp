# Question

Which of the following BEST describes a project milestone?

## Options

- **A**: A major work package consisting of several activities, with its own duration and resource requirements.
- **B**: A significant point or event in the project, such as a phase gate or deliverable acceptance, with zero duration.
- **C**: A scheduled checkpoint at which earned value is recalculated and reported to stakeholders.
- **D**: A risk-related event that requires the project manager to invoke a contingency response.

## Answer

B

## Explanation

(B) matches the PMBOK definition: milestones are zero-duration markers used to denote significant events, such as phase completions, deliverable approvals, or contractual gates. (A) describes a work package, which has duration and resources. (C) describes an EVM reporting checkpoint, which is a process step rather than a milestone. (D) describes a risk trigger, not a milestone.
