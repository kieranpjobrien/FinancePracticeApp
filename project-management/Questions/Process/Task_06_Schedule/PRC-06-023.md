# Question

A construction project uses critical chain method with buffers. The feeding buffer for one chain is 50% consumed, but the work on that chain is only 30% complete. What does this indicate and what action should be taken?

## Options

- **A**: Everything is fine - buffers are meant to be consumed
- **B**: The buffer consumption rate exceeds work completion rate, indicating the chain may overrun - investigate causes and consider recovery actions before the buffer is exhausted
- **C**: Add more buffer immediately
- **D**: Remove the feeding buffer

## Answer

B

## Explanation

The correct answer is B. In critical chain, buffer consumption is monitored against work completion. If buffer consumption outpaces work completion (50% buffer used for 30% work), the chain is trending toward overrun. Investigate: why is work slower than planned? Act: before buffer is exhausted, implement recovery actions to protect the project buffer.

Option A ignores the warning signal.

Option C adds buffer without addressing root cause.

Option D removes protection.
