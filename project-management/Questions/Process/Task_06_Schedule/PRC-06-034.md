# Question

A waterfall project is transitioning from design to development phase. Some design activities are running late, but development resources are scheduled to start next week. What should the PM do?

## Options

- **A**: Start development anyway with incomplete design
- **B**: Assess which development work can proceed with available design, potentially start non-dependent work, adjust the schedule for dependent work, and communicate impacts
- **C**: Wait for all design to complete before starting any development
- **D**: Cancel the development phase

## Answer

B

## Explanation

The correct answer is B. Phase transitions rarely align perfectly. Assess: which development work depends on late design? Which can proceed? Adjust: start independent work, resequence dependent work. Communicate: stakeholders should understand schedule impact. This balances progress with dependency management.

Option A risks rework from building on incomplete design.

Option C may be overly conservative if some work can proceed.

Option D is excessive for a phase timing issue.
