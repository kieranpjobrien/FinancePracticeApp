# Question

A risk workshop using brainstorming has identified 50 risks. The team needs to prioritise them quickly. What technique should be used?

## Options

- **A**: Detailed quantitative analysis of all 50 risks
- **B**: Qualitative assessment using probability/impact ratings to quickly categorise risks into high, medium, and low priority
- **C**: Random selection of risks to focus on
- **D**: Vote for the most interesting risks

## Answer

B

## Explanation

The correct answer is B. Qualitative assessment enables rapid prioritisation. Rate each risk for probability (High/Medium/Low) and impact (High/Medium/Low). Plot on a matrix to identify priorities. High probability/high impact risks get immediate attention. This can be done quickly, saving detailed quantitative analysis for the highest-priority risks.

Option A is too time-consuming for initial triage.

Option C isn't systematic.

Option D prioritises interest over importance.
