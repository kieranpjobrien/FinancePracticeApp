# Question

A project in execution has encountered three unrelated risks in two weeks, each of which triggered its mitigation. The contingency reserve has been drawn down to 20%. What should the PM do?

## Options

- **A**: Request the sponsor replenish the contingency reserve to its original level as a precaution.
- **B**: Continue executing, since 20% reserve is still available and other risks can be covered later.
- **C**: Stop non-critical work to preserve the remaining reserve for any future risk events in the project.
- **D**: Reassess the risk register and re-run quantitative analysis, then request a reserve change if needed.

## Answer

D

## Explanation

When reserve is consumed rapidly, PMI expects the PM to reassess exposure against remaining reserve before acting. (D) reanalyses the real position and then takes a data-backed request through change control if needed. (A) asks for money without justifying exposure — the sponsor will reasonably ask what's left to face. (B) complacent; ignoring a 3-in-2-weeks pattern invites another draw that leaves the project uncovered. (C) halting work is a drastic response that hurts value without addressing the actual question of whether remaining reserve matches remaining exposure.
