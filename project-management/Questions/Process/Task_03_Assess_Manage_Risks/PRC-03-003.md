# Question

A project identifies a risk with a 30% probability of occurring. If it occurs, it will cause a $50,000 cost overrun. A risk response that eliminates this risk will cost $12,000 to implement. Should the project manager implement the response?

## Options

- **A**: Yes, because $12,000 is less than $50,000
- **B**: No, because the Expected Monetary Value of the risk is $15,000, which is greater than the $12,000 response cost
- **C**: Yes, because the Expected Monetary Value of the risk is $15,000, which is greater than the $12,000 response cost
- **D**: No, because the risk may not occur

## Answer

C

## Explanation

The correct answer is C. Expected Monetary Value (EMV) = Probability x Impact = 30% x $50,000 = $15,000. Since the response cost ($12,000) is less than the EMV ($15,000), implementing the response provides net positive value. The decision should be based on comparing response cost to EMV, not to the full impact.

Option A reaches the right conclusion with wrong reasoning - comparing $12,000 to the full impact ($50,000) ignores probability.

Option B has the math right but the conclusion wrong - if response cost is LESS than EMV, implementing IS worthwhile.

Option D ignores expected value analysis. The fact that risk may not occur is already factored into the EMV calculation through probability.
