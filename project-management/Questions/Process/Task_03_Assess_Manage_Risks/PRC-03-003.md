# Question

A risk has a 30% probability of occurring. If it occurs, the project will incur a $50,000 cost overrun. A response that fully eliminates the risk costs $12,000 to implement. Should the PM implement the response, and why?

## Options

- **A**: No — the response cost ($12,000) exceeds the project's residual risk tolerance threshold of $10,000.
- **B**: No — at 30% probability the risk is below the impact-only review threshold for this project size.
- **C**: Yes — the risk's expected monetary value is $15,000, which exceeds the $12,000 response cost, so the response is net-positive.
- **D**: Yes — the response cost ($12,000) is much smaller than the potential loss ($50,000), so eliminating the risk is clearly worth it.

## Answer

C

## Explanation

EMV = probability × impact = 0.30 × $50,000 = $15,000. Compare EMV to response cost: $15,000 > $12,000, so the response is net-positive. (C) uses the correct comparison. (A) and (B) invent thresholds not given in the problem. (D) reaches the right action but with the wrong reasoning — comparing response cost to raw impact (rather than EMV) ignores probability and would justify expensive responses for unlikely risks.
