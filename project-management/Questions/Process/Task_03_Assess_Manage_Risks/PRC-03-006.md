# Question

A project team identifies that a key vendor might not deliver a critical component on time. The probability is assessed as medium (40%) and the impact is high ($200,000 in delays if it occurs). What is the Expected Monetary Value (EMV) of this risk?

## Options

- **A**: $200,000
- **B**: $80,000
- **C**: $120,000
- **D**: $500,000

## Answer

B

## Explanation

The correct answer is B. Expected Monetary Value is calculated by multiplying probability by impact.

**Formula:** EMV = Probability × Impact

**Calculation:** EMV = 0.40 × $200,000 = **$80,000**

This EMV can be used to prioritise risks, determine appropriate contingency reserves, and evaluate whether risk response investments are cost-effective.

Option A ($200,000) is the raw impact, not risk-adjusted.

Option C ($120,000) might be calculated using (1 - 0.40) × $200,000, which isn't the correct formula.

Option D ($500,000) appears to be Impact / Probability, which isn't a valid calculation.
