# Question

A risk has a 30% probability of occurring and would cost $100,000 if it occurred. What is the Expected Monetary Value (EMV) of this risk?

## Options

- **A**: $100,000
- **B**: $30,000
- **C**: $70,000
- **D**: $130,000

## Answer

B

## Explanation

The correct answer is B. EMV = Probability × Impact. EMV = 0.30 × $100,000 = $30,000. This represents the weighted average value of the risk, useful for comparing risks and making decisions about risk response investments.

Option A is the full impact, not EMV.

Option C incorrectly uses the complement (70%).

Option D adds the values rather than multiplying.
