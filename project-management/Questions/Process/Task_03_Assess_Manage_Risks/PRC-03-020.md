# Question

A project has three risks with the following EMVs: Risk A = -$50,000, Risk B = -$30,000, Risk C = +$20,000 (opportunity). What is the overall risk exposure?

## Options

- **A**: $100,000
- **B**: -$60,000
- **C**: -$100,000
- **D**: $80,000

## Answer

B

## Explanation

The correct answer is B. Overall risk exposure sums all EMVs including both threats (negative) and opportunities (positive). -$50,000 + (-$30,000) + $20,000 = -$60,000. This represents the net expected value of all identified risks and is useful for determining contingency reserves.

Option A ignores the signs.

Option C doesn't include the opportunity.

Option D calculates incorrectly.
