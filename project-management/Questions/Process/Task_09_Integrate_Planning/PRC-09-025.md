# Question

Which statement best distinguishes the project management plan from project documents?

## Options

- **A**: The plan is always approved formally; documents are always informal.
- **B**: The plan defines how the project is executed, monitored, and controlled; project documents are working artefacts that support those activities.
- **C**: The plan is produced first; documents are produced after baselining.
- **D**: The plan belongs to the project manager; documents belong to the wider team.

## Answer

B

## Explanation

The PMBOK distinction is by purpose: the plan is the integrated guidance for managing the project; documents (risk register, issue log, lessons learned, etc.) support and feed that management. (A) is too absolute — both can be formal. (C) is incorrect because some documents (like the stakeholder register) precede parts of the plan. (D) misallocates ownership.
