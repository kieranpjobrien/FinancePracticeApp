# Question

A project serves multiple stakeholder groups with competing priorities. The marketing team wants maximum features; finance wants minimum cost; operations wants earliest delivery. How should the PM integrate these into the plan?

## Options

- **A**: Prioritise the most powerful stakeholder's needs
- **B**: Find the mathematically optimal balance across all constraints
- **C**: Facilitate stakeholder alignment on project priorities before finalising the plan
- **D**: Present three alternative plans - one optimised for each stakeholder group

## Answer

C

## Explanation

The correct answer is C. Integration requires resolving competing demands before baselining. The PM should facilitate discussion where stakeholders understand trade-offs and agree on priorities. A plan everyone had input into has better buy-in than one that favours some over others.

Option A uses politics rather than collaboration to resolve conflict.

Option B assumes there's an objective "optimal" when priorities are value judgements that stakeholders must make.

Option D presents options without facilitating decision-making. The PM should drive toward one integrated plan.
