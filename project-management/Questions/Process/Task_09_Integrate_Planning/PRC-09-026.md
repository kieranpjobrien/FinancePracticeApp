# Question

Three powerful stakeholder groups have incompatible top priorities for the same project: marketing wants the broadest feature set, finance wants the lowest cost, operations wants the earliest delivery. How should the project manager produce an integrated plan?

## Options

- **A**: Facilitate a structured prioritisation session — using techniques like trade-off analysis or weighted scoring — that ends in a single agreed priority order before the plan is baselined.
- **B**: Build the plan around whichever group has the most organisational power.
- **C**: Compute a mathematical optimum across the constraints and present it as the integrated plan.
- **D**: Issue three plans — one optimised per stakeholder — and let governance pick.

## Answer

A

## Explanation

Integration depends on agreement, and agreement depends on structured facilitation. Stakeholders making the trade-off explicitly is what enables a single coherent plan. (B) substitutes politics for analysis. (C) treats values as math; trade-offs are judgement calls. (D) shifts the integration work to governance and produces three orphan plans.
