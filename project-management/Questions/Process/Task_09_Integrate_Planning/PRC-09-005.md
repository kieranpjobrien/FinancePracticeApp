# Question

What is the PRIMARY output of the Develop Project Management Plan process?

## Options

- **A**: The project charter
- **B**: The project management plan
- **C**: The work breakdown structure
- **D**: The project schedule

## Answer

B

## Explanation

The correct answer is B. The Develop Project Management Plan process produces the project management plan - an integrated document that defines how the project will be executed, monitored, controlled, and closed. It consolidates all subsidiary plans (scope, schedule, cost, quality, resource, communications, risk, procurement, stakeholder).

Option A (project charter) is an input to this process, not an output. The charter authorises the project and is developed earlier.

Option C (WBS) is an output of the Create WBS process, which is part of scope planning. The WBS becomes part of the project management plan.

Option D (project schedule) is an output of the Develop Schedule process. Like the WBS, it becomes a component of the project management plan.
