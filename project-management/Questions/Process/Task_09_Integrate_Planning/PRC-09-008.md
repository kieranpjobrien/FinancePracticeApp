# Question

In a final sweep before approving the project management plan, the PM finds the quality plan calls for two QA engineers full-time during testing — but the resource plan only assigns one. What should the PM do?

## Options

- **A**: Add the second QA engineer to the resource plan and adjust the cost baseline accordingly.
- **B**: Reduce planned quality activities to fit the existing resource allocation.
- **C**: Run an integrated cross-plan review to find this and any related inconsistencies, then reconcile all subsidiary plans before re-baselining.
- **D**: Escalate the resource shortfall to the sponsor for additional funding immediately.

## Answer

C

## Explanation

A single inconsistency between subsidiary plans is rarely the only one; the integration discipline is to do the systematic sweep, not just patch the visible defect. (A) fixes one symptom and may miss others. (B) trades quality without exploring alternatives. (D) escalates before the PM has done the analytical work the sponsor will want.
