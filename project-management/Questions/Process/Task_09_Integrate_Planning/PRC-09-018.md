# Question

What is the primary role of the project manager in integration management?

## Options

- **A**: Ensuring all the project's elements — plans, processes, and people — work together as a coherent whole.
- **B**: Personally writing each subsidiary management plan.
- **C**: Approving every change request raised on the project.
- **D**: Producing all stakeholder-facing status communications.

## Answer

A

## Explanation

Integration is the unique, non-delegable PM responsibility: stitching the parts into a working whole. (B) is delegable to discipline leads. (C) is one input into integration but is usually a CCB function, not solely the PM's. (D) is communications management, distinct from integration.
