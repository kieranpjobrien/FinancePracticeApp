# Question

What is the PRIMARY role of the project manager in integration management?

## Options

- **A**: Creating each subsidiary plan
- **B**: Ensuring all project elements work together coherently
- **C**: Approving all project changes
- **D**: Communicating status to stakeholders

## Answer

B

## Explanation

The correct answer is B. Integration management is uniquely the PM's responsibility. While specialists create their respective plans, the PM ensures all elements combine into a coherent whole - that schedules align with resources, quality requirements inform testing, risks inform contingencies, etc.

Option A may be delegated to specialists or team members for their areas.

Option C is a part of integration (change control) but not the primary role.

Option D is important but falls under communications management, not specifically integration.
