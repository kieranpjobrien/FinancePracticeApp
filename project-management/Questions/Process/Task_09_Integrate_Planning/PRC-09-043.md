# Question

A PM inherits a project with separate, conflicting plans created by different predecessors. The schedule baseline assumes different scope than the scope baseline. What should the PM do FIRST?

## Options

- **A**: Choose the most recent version of each document as the baseline
- **B**: Conduct a comprehensive review to understand the current true state of the project
- **C**: Create new plans from scratch
- **D**: Escalate to the sponsor about the plan quality issues

## Answer

B

## Explanation

The correct answer is B. Before fixing the plans, the PM needs to understand reality. What has actually been delivered? What has actually been spent? What do stakeholders actually expect? Understanding the true current state enables meaningful re-planning. Fixing documents without understanding reality just creates different wrong documents.

Option A perpetuates inconsistency without resolving it.

Option C may be needed but first understand the current state to know what plans should reflect.

Option D escalates before understanding the situation. Come with analysis and recommendations, not just problems.
