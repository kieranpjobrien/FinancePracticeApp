# Question

In PMI terminology, configuration management primarily exists to ensure:

## Options

- **A**: Standardised tooling is consistently used by every team member across the project.
- **B**: Project documents are stored in a single centralised repository for easy access.
- **C**: Integrity of product specifications is maintained, with current state identifiable and traceable.
- **D**: All change requests automatically follow the formal change control process without exceptions.

## Answer

C

## Explanation

Configuration management is about controlling, identifying, and tracking the configuration items that make up the product. (A) is a tooling decision unrelated to configuration management. (B) is document management. (D) is change control, a related but distinct discipline that operates on top of configuration management.
