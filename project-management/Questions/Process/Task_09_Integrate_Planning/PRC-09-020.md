# Question

Configuration management in a project primarily ensures:

## Options

- **A**: All team members use the same software tools
- **B**: Product specifications and deliverables are controlled and traceable
- **C**: Project documents are stored in a single location
- **D**: Changes follow the change control process

## Answer

B

## Explanation

The correct answer is B. Configuration management ensures that the specifications of deliverables and processes are controlled, that changes are properly identified and documented, and that the current state of any configuration item is known. It maintains product integrity throughout the project.

Option A is tool standardisation, not configuration management.

Option C describes document management, which is related but not the primary purpose of configuration management.

Option D describes change control, which is one aspect of configuration management but not its complete definition.
