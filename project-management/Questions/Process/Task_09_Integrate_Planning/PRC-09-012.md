# Question

The performance measurement baseline (PMB) is composed of which of the following?

## Options

- **A**: Risk register, issue log, and change log.
- **B**: Project charter, work breakdown structure, and project schedule.
- **C**: Schedule baseline, cost baseline, and quality baseline.
- **D**: Scope baseline, schedule baseline, and cost baseline.

## Answer

D

## Explanation

The PMB is the integrated triad of scope, schedule, and cost baselines, against which actual performance (including earned value) is measured. (A) lists project documents, not baselines. (B) lists planning artefacts; the WBS is part of the scope baseline but not the PMB itself. (C) substitutes "quality" for "scope," which is incorrect.
