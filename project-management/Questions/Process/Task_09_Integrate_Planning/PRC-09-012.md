# Question

What is the performance measurement baseline composed of?

## Options

- **A**: Schedule baseline, cost baseline, and quality baseline
- **B**: Scope baseline, schedule baseline, and cost baseline
- **C**: Project charter, WBS, and project schedule
- **D**: Risk register, issue log, and change log

## Answer

B

## Explanation

The correct answer is B. The performance measurement baseline integrates the scope baseline, schedule baseline, and cost baseline. It's used to compare actual execution against planned performance, enabling earned value analysis and variance reporting.

Option A substitutes quality for scope. Quality standards are important but the performance measurement baseline specifically combines scope, schedule, and cost.

Option C lists planning inputs and outputs, not the baselines that form the performance measurement baseline.

Option D lists project documents used during execution, not baselines.
