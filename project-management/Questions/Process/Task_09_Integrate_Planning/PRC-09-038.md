# Question

A PM is using expert judgement and historical data to develop the project plan. The experts disagree with what the historical data suggests. What should the PM do?

## Options

- **A**: Trust the experts since they have current context
- **B**: Trust the data since it's objective
- **C**: Investigate why they differ and use the insight to improve the plan
- **D**: Average the expert opinion and data-based estimate

## Answer

C

## Explanation

The correct answer is C. Disagreement between expert judgement and historical data is valuable information. Perhaps the project is genuinely different from historical projects (experts are right), or perhaps experts have optimistic biases (data is right). Understanding the difference improves planning accuracy.

Option A and B each dismiss valuable input without analysis.

Option D mathematically combines different inputs without understanding why they differ.
