# Question

While developing the project plan, the PM finds that expert judgement and historical organisational data point to materially different effort estimates. What is the best course of action?

## Options

- **A**: Trust the experts because they have direct knowledge of current project context.
- **B**: Trust the historical data because it is objective and pre-existing.
- **C**: Investigate the source of the disagreement, treat it as planning information, and use the diagnosis to refine the estimate and the underlying assumptions.
- **D**: Take the simple average of the expert and data-based numbers.

## Answer

C

## Explanation

A divergence between experts and data is signal, not noise — possibly the project really is different, possibly the experts are anchored, possibly the data set is biased. Diagnosis improves the plan; the others discard half the information. (A) and (B) each pre-emptively discard one source. (D) blends incompatible inputs without understanding them.
