# Question

Which of the following is a subsidiary plan that becomes part of the project management plan?

## Options

- **A**: Project charter
- **B**: Stakeholder register
- **C**: Risk management plan
- **D**: Work breakdown structure

## Answer

C

## Explanation

The correct answer is C. The risk management plan is one of several subsidiary plans that are integrated into the project management plan. Other subsidiary plans include scope, schedule, cost, quality, resource, communications, procurement, and stakeholder engagement management plans.

Option A is an input to the project management plan, not a component of it. The charter authorises the project.

Option B is a project document, not a subsidiary plan. It lists stakeholders but doesn't define how they're managed.

Option D is part of the scope baseline, not itself a subsidiary plan.
