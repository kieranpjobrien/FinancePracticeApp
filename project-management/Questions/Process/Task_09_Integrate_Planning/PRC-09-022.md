# Question

When is the project management plan baselined?

## Options

- **A**: When the project charter is approved
- **B**: After detailed planning is complete and before execution begins
- **C**: At the end of the first phase or iteration
- **D**: When the project sponsor requests it

## Answer

B

## Explanation

The correct answer is B. The project management plan is baselined when planning is sufficiently complete to begin controlled execution. The baseline becomes the reference point for measuring performance. In predictive projects, this is after detailed planning; in adaptive projects, it may be after initial release planning.

Option A is too early. The charter precedes detailed planning.

Option C may be too late. Execution should measure against a baseline from the start.

Option D makes baselining arbitrary rather than methodology-driven.
