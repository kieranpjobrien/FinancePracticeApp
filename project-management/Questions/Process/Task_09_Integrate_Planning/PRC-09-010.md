# Question

A project team wants to start execution immediately because the deadline is tight. They argue that detailed planning wastes valuable time. How should the PM respond?

## Options

- **A**: Allow immediate start since meeting the deadline is critical
- **B**: Insist on complete planning before any execution begins
- **C**: Begin with sufficient planning for immediate work while continuing to plan in parallel
- **D**: Escalate to the sponsor to decide between planning and execution

## Answer

C

## Explanation

The correct answer is C. Rolling wave or iterative planning balances the need for planning with time pressure. Plan enough to start the first work packages confidently while continuing to develop detailed plans for later work. This maintains planning quality without unnecessary delay.

Option A risks executing without understanding, leading to rework and delays that exceed any time "saved."

Option B may be unnecessarily rigid. Not all projects need complete upfront planning before any execution.

Option D abdicates professional responsibility. The PM should recommend an approach, not ask the sponsor to choose.
