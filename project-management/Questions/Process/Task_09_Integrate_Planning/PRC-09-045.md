# Question

A project plan has been through multiple revisions. Team members are confused about which version is current, and someone recently worked from an outdated version causing rework. What practice should the PM implement?

## Options

- **A**: Email the current version to all team members after each update
- **B**: Maintain a single source of truth with version control and clear communication of updates
- **C**: Lock the plan to prevent further changes
- **D**: Assign a team member to track which version each person is using

## Answer

B

## Explanation

The correct answer is B. Configuration management principles apply to the project plan itself. A single source of truth (central repository), version control (tracking changes with clear versioning), and communication of updates prevents confusion about currency. Team members should always know where to find the authoritative current version.

Option A creates multiple copies that can become out of sync.

Option C prevents necessary updates to keep the plan relevant.

Option D adds overhead without solving the root cause of version proliferation.
