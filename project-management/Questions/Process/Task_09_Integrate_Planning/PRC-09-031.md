# Question

Which planning activity typically occurs LAST when developing the project management plan?

## Options

- **A**: Creating the WBS
- **B**: Identifying stakeholders
- **C**: Integrating subsidiary plans into the overall project management plan
- **D**: Defining project scope

## Answer

C

## Explanation

The correct answer is C. Integration of subsidiary plans happens after individual plans are developed. You need scope, schedule, cost, quality, resource, communications, risk, procurement, and stakeholder plans before you can integrate them into a coherent project management plan.

Option A happens early to define the work to be done.

Option B happens during initiation, before detailed planning.

Option D happens before the WBS and other detailed planning.
