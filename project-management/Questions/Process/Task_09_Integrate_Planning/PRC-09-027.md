# Question

Which statement BEST describes tailoring in project management?

## Options

- **A**: Reducing documentation to save time
- **B**: Adapting processes and approaches to fit project characteristics
- **C**: Using only agile or only predictive methods
- **D**: Simplifying plans for smaller projects

## Answer

B

## Explanation

The correct answer is B. Tailoring means selecting and adapting processes, tools, techniques, and approaches based on project characteristics such as size, complexity, risk, and organisational culture. It's about fit-for-purpose application of project management practices.

Option A focuses only on documentation reduction, which is too narrow.

Option C limits to methodology choice, not the broader concept of adaptation.

Option D focuses only on project size, missing other factors like complexity, risk, and stakeholder needs.
