# Question

What is the relationship between the project charter and the project management plan?

## Options

- **A**: They are the same document with different names
- **B**: The charter authorises the project; the plan defines how to execute it
- **C**: The charter is created by the PM; the plan is created by the sponsor
- **D**: The charter replaces the plan in agile projects

## Answer

B

## Explanation

The correct answer is B. The project charter formally authorises the project and provides the PM with authority to apply resources. The project management plan defines how the project will be executed, monitored, controlled, and closed. The charter is an input to plan development.

Option A conflates two different documents with different purposes.

Option C reverses ownership. The sponsor authorises the charter; the PM develops the plan.

Option D is incorrect. Agile projects still have charters (or similar authorisation) and planning, though the form may differ.
