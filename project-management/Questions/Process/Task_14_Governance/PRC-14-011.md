# Question

A project manager discovers that two functional managers are providing conflicting direction to shared team members, causing confusion and rework. The project charter doesn't clearly address this situation. What should the project manager do?

## Options

- **A**: Let team members choose which manager to follow
- **B**: Escalate immediately to executive management
- **C**: Clarify roles, responsibilities, and decision authorities with both functional managers, document the agreements, and if needed, update governance documentation
- **D**: Remove the shared team members from the project

## Answer

C

## Explanation

The correct answer is C. The root cause is unclear governance regarding decision authority for shared resources. The PM should facilitate clarification between the functional managers, establish clear decision rights for different types of direction, and document the agreement. This prevents recurrence and provides guidance for team members.

Option A puts team members in an impossible position.

Option B may be premature if the managers can resolve it themselves.

Option D loses resources without solving the underlying governance issue.
