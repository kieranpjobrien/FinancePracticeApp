# Question

What is the purpose of a project charter in governance?

## Options

- **A**: To detail all project tasks
- **B**: To formally authorize the project and document the project manager's authority
- **C**: To track project issues
- **D**: To record lessons learned

## Answer

B

## Explanation

The correct answer is B. The project charter formally authorizes the project to exist, names the project manager, and documents their authority level. It establishes the governance foundation by defining project objectives, high-level requirements, success criteria, and key stakeholders. It's the foundational governance document.

Option A describes the WBS or schedule.

Option C describes the issue log.

Option D describes the lessons learned register.
