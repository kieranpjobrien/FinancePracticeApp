# Question

A project manager notices that steering committee meetings have become rubber-stamp sessions where all recommendations are approved without discussion. The PM is concerned that this isn't effective governance. What should the project manager do?

## Options

- **A**: Continue as is since it makes approvals easy
- **B**: Intentionally include items that will generate debate
- **C**: Discuss with the sponsor whether the current governance level is appropriate, and consider adjusting meeting content to focus on matters genuinely requiring committee input or restructuring governance tiers
- **D**: Skip steering committee meetings since they're not adding value

## Answer

C

## Explanation

The correct answer is C. Rubber-stamp governance suggests either the right decisions are being made at lower levels (governance may need restructuring) or the committee isn't engaging (content may need adjustment). The PM should raise this with the sponsor to determine if governance is appropriately structured or if changes would improve effectiveness.

Option A accepts ineffective use of senior stakeholders' time.

Option B artificially creates conflict rather than addressing root cause.

Option D abandons governance without resolution.
