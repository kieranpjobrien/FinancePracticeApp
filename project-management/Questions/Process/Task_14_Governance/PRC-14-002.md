# Question

What is the PRIMARY purpose of project governance?

## Options

- **A**: To create documentation for audits
- **B**: To provide a framework for decision-making, oversight, and control that ensures project alignment with organizational objectives
- **C**: To limit the project manager's authority
- **D**: To satisfy regulatory requirements

## Answer

B

## Explanation

The correct answer is B. Project governance establishes how decisions are made, who has authority for what, how oversight is provided, and how the project connects to organizational strategy. It provides structure and clarity for project direction and control.

Option A describes a byproduct, not the purpose.

Option C mischaracterizes governance - it clarifies authority rather than limiting it.

Option D may be relevant in some contexts but isn't the primary purpose.
