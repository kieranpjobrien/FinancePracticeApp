# Question

What is the PRIMARY purpose of project governance?

## Options

- **A**: To produce documentation that satisfies internal audit and external regulators.
- **B**: To set boundaries on the project manager's authority so the team cannot exceed them.
- **C**: To provide a framework for decision-making, oversight, and alignment with organisational objectives.
- **D**: To enforce compliance with the organisation's chosen project methodology.

## Answer

C

## Explanation

PMI defines governance as the framework that supports decision rights, oversight, and alignment with strategy. (C) is the textbook definition. (A) is a by-product, not the purpose. (B) misframes governance as a restraint when it actually clarifies and enables authority. (D) is methodology compliance, which sits inside governance but is not its purpose.
