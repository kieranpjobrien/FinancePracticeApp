# Question

What is a key benefit of documenting the governance structure?

## Options

- **A**: It creates more work for the project team
- **B**: It provides clarity on how decisions are made and who has authority, enabling consistent and efficient decision-making
- **C**: It satisfies auditors only
- **D**: It is only useful for large projects

## Answer

B

## Explanation

The correct answer is B. Documented governance provides clarity that enables the project to function efficiently. When everyone understands who makes what decisions, issues are routed correctly, decisions happen at appropriate levels, and confusion is minimized. This benefits projects of all sizes.

Option A views documentation as overhead rather than enabler.

Option C is one benefit but not the primary purpose.

Option D is incorrect - clear governance helps any size project.
