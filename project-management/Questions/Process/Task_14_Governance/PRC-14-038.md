# Question

Which statement best describes the typical role of a PMO in project governance?

## Options

- **A**: The PMO is the ultimate approver of every scope, schedule, and budget change across the portfolio.
- **B**: The PMO performs the day-to-day project work and reports outcomes to the sponsor.
- **C**: The PMO provides standards, templates, and oversight that support consistent governance across projects.
- **D**: The PMO serves as sponsor for every organisational project by default.

## Answer

C

## Explanation

PMI describes PMOs primarily as enabling functions that set standards and provide oversight, while project-specific governance (sponsor, steering committee, CCB) owns decisions. (C) matches that. (A) overstates PMO authority; decision rights typically sit with project governance. (B) confuses the PMO with the delivery team. (D) conflates PMO leadership with individual project sponsorship.
