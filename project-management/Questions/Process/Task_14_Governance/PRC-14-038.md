# Question

What role does the PMO typically play in project governance?

## Options

- **A**: Making all project decisions
- **B**: Providing standards, templates, oversight, and support for project governance practices across the organization
- **C**: Performing all project work
- **D**: Serving as the project sponsor

## Answer

B

## Explanation

The correct answer is B. The PMO (Project Management Office) typically establishes governance standards, provides templates and tools, offers oversight and quality assurance, and supports project managers in implementing governance. They ensure consistency across projects while individual projects maintain their own governance structures.

Option A overstates PMO authority - decisions typically rest with project-specific governance.

Option C describes the project team's role.

Option D describes executive sponsorship, not PMO function.
