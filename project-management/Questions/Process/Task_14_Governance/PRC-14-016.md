# Question

An organization is running multiple related projects as a program. Each project has its own governance, but cross-project issues are falling through the cracks. Resources are being contested between projects, and priority conflicts aren't being resolved. What governance enhancement is needed?

## Options

- **A**: Strengthen each project's individual governance
- **B**: Implement program-level governance that addresses cross-project priorities, resource allocation, and issue resolution while maintaining project-level governance for project-specific matters
- **C**: Combine all projects into one with a single governance structure
- **D**: Let project managers resolve cross-project issues among themselves

## Answer

B

## Explanation

The correct answer is B. Program governance provides a layer above project governance to address cross-project matters: priority decisions when projects compete, resource allocation across projects, and resolution of cross-project issues. Project governance handles project-specific matters. Both layers are needed.

Option A doesn't address cross-project issues.

Option C may create an unwieldy single project and lose benefits of separate project focus.

Option D lacks authority to make binding cross-project decisions.
