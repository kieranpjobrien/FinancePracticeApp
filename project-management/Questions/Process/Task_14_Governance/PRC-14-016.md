# Question

Five projects in a programme each have their own governance, but cross-project issues — shared resources, conflicting priorities, dependency slips — keep falling through the cracks. What governance enhancement is needed?

## Options

- **A**: Strengthen the governance of each individual project so they can resolve cross-project issues themselves.
- **B**: Add a programme-level governance layer that owns cross-project priority, resource allocation, and dependency resolution, while the project layers continue to handle local matters.
- **C**: Merge the five projects into a single project with one governance structure and one PM.
- **D**: Set up a weekly meeting of the five project managers to resolve cross-project issues by consensus.

## Answer

B

## Explanation

PMI distinguishes project from programme governance precisely for this case — there are decisions no single project owns. (B) puts those decisions where they belong. (A) cannot give projects authority they do not have. (C) loses focus and creates an unwieldy single project. (D) is a coordination forum without authority and tends to produce stalemates.
