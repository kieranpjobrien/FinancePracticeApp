# Question

A change control board has grown to fourteen members and any moderate change request now takes three weeks to approve. Some members rarely contribute or even attend. What should the PM propose?

## Options

- **A**: Leave the CCB composition alone and instead speed up pre-meeting analysis so decisions reach the meeting already evaluated.
- **B**: Escalate to the sponsor and ask them to pick which members should be removed from the change control board themselves.
- **C**: Keep all members but split the CCB into two subgroups that each handle half of the change requests each week.
- **D**: Review the CCB charter with the sponsor: define essential members, add thresholds for delegated approvals, and reset the cadence.

## Answer

D

## Explanation

PMI treats CCB structure as a deliberate design, not a standing list, and expects periodic redesign. (D) makes that redesign explicit with the sponsor. (A) treats a symptom without addressing the bloated body. (B) asks the sponsor to make a decision the PM should first structure and recommend. (C) creates parallel bodies without rightsizing the composition.
