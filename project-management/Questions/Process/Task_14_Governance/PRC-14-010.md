# Question

What does RACI stand for in a responsibility assignment matrix?

## Options

- **A**: Risk, Analysis, Control, Inspection
- **B**: Responsible, Accountable, Consulted, Informed
- **C**: Review, Approve, Communicate, Implement
- **D**: Requirements, Activities, Constraints, Issues

## Answer

B

## Explanation

The correct answer is B. RACI is a responsibility assignment matrix that clarifies roles:
- Responsible: Does the work
- Accountable: Ultimately answerable for completion (only one per task)
- Consulted: Provides input before decisions
- Informed: Kept updated on progress/decisions

This clarifies who does what and prevents confusion about responsibilities and decision authority.
