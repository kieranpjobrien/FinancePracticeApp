# Question

Which statement best describes the primary purpose of a project governance framework?

## Options

- **A**: To enforce consistent reporting templates across all projects in the portfolio.
- **B**: To define decision rights, accountability, and escalation paths so the project can deliver value within organisational boundaries.
- **C**: To document the methodology (predictive, agile, or hybrid) the project will use throughout its lifecycle.
- **D**: To assign individual deliverables to named team members at the start of the project.

## Answer

B

## Explanation

PMI defines governance as the framework of authority, accountability, and decision-making that aligns a project with the sponsoring organisation. (B) captures that intent. (A) is one possible artefact of governance but is far narrower than its purpose. (C) describes the development approach, not governance. (D) is a resource assignment activity belonging to the project plan, not governance.
