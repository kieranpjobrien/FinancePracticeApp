# Question

A project is operating in a matrix organization where team members report to both the project manager and their functional managers. Functional managers are assigning their staff to operational work during project hours, causing project delays. What governance approach would help?

## Options

- **A**: Escalate to the PMO every time this happens
- **B**: Accept that functional work takes priority
- **C**: Establish clear agreements with functional managers about resource commitments, documented in the project charter or resource management plan, with an escalation process for conflicts
- **D**: Convert to a dedicated team structure

## Answer

C

## Explanation

The correct answer is C. Matrix governance requires clear resource agreements upfront, documented formally. These agreements should specify time commitments, how conflicts will be resolved, and escalation paths when agreements aren't honored. This creates accountability and a mechanism for resolution.

Option A treats symptoms without establishing preventive agreements.

Option B cedes all priority to functional work, making the project impossible to manage.

Option D may not be feasible in the organization's structure.
