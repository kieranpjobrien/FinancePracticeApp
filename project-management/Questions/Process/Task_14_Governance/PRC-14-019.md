# Question

In a matrix organisation, functional managers keep pulling project team members onto operational work during agreed project hours, and the project is slipping. What governance approach would help?

## Options

- **A**: Document a resource agreement signed by each functional manager covering committed hours, conflict resolution, and escalation.
- **B**: Escalate every individual breach to the PMO so they can intervene with the functional manager and restore commitments.
- **C**: Convert the project to a fully dedicated team structure outside the matrix so resources sit with the PM directly.
- **D**: Accept that operational work takes priority and replan the project with reduced capacity and a later delivery date.

## Answer

A

## Explanation

PMI's recommended approach in a matrix is to make resource commitments explicit up front and use them as the basis for conflict resolution. (A) does that and gives the PM a credible reference point. (B) treats every conflict as an escalation, which does not scale. (C) is rarely available and would not address the underlying agreement gap. (D) gives up on the project's commitments without addressing the structural issue.
