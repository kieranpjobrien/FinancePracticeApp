# Question

A project is transitioning from initiation to planning phase. The governance structure established during initiation included only the sponsor and PM. As planning reveals the project's complexity, it's clear that more robust governance is needed. What should the project manager do?

## Options

- **A**: Continue with the current governance since changing it is disruptive
- **B**: Propose an enhanced governance structure appropriate for the project's actual complexity, including additional stakeholders, clearer decision authorities, and expanded oversight
- **C**: Wait until problems occur before changing governance
- **D**: Ask the sponsor to handle all additional governance needs

## Answer

B

## Explanation

The correct answer is B. Governance should be right-sized for the project. As understanding of complexity grows, governance should evolve. Proposing enhanced structure proactively (before problems occur) demonstrates good project management. The PM should assess what's needed and propose appropriate changes to the sponsor.

Option A accepts inadequate governance for the project's actual needs.

Option C is reactive rather than proactive.

Option D overloads the sponsor rather than establishing proper structure.
