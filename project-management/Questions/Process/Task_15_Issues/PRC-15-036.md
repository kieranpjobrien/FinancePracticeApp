# Question

Issue log analysis shows that 60% of issues come from one work package. The team running that work package is now feeling singled out and their morale is dropping. What should the project manager do?

## Options

- **A**: Investigate why that work package generates more issues (complexity, resourcing, requirements clarity, process), address the root causes, and reframe the conversation as joint process improvement rather than performance criticism.
- **B**: Reduce the level of scrutiny applied to that work package's issues to give the team some breathing room and rebuild morale.
- **C**: Replace the team running that work package because the data clearly shows they cannot manage their work effectively.
- **D**: Stop tagging issues by work package so the concentration is no longer visible in reports.

## Answer

A

## Explanation

A concentration of issues is information, not condemnation. (A) treats it as a diagnostic and engages the team in fixing the system. (B) hides a real signal and removes the chance to improve. (C) blames people without understanding the cause and damages trust. (D) suppresses data the project needs to manage itself.
