# Question

Multiple interdependent issues are creating a complex situation where fixing one issue may worsen others. The team is struggling to find a path forward. What approach should the project manager take?

## Options

- **A**: Fix issues in the order they were reported
- **B**: Address the easiest issues first to show progress
- **C**: Map the interdependencies between issues, identify if there's an optimal sequence or if a holistic solution addresses multiple issues, and involve relevant expertise in developing a coordinated resolution approach
- **D**: Escalate the entire situation as too complex for the team

## Answer

C

## Explanation

The correct answer is C. Complex interdependent issues require systems thinking. Mapping relationships between issues may reveal: an optimal sequence where some fixes enable others, a common root cause that addresses multiple issues, or trade-offs that need to be made consciously. This analysis, possibly with additional expertise, enables a coordinated approach.

Option A doesn't consider interdependencies.

Option B may make harder issues more difficult.

Option D gives up without attempting analysis.
