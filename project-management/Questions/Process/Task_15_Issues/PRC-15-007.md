# Question

What is the PRIMARY difference between a risk and an issue?

## Options

- **A**: Risks are more serious than issues
- **B**: A risk is an uncertain event that may occur; an issue is a current problem that has already occurred
- **C**: Issues require more documentation than risks
- **D**: Only the project manager can identify risks, while anyone can identify issues

## Answer

B

## Explanation

The correct answer is B. Risks are potential future events with uncertainty about occurrence. Issues are current, actual problems requiring immediate attention. A risk may become an issue if it occurs. Risk management is proactive (addressing uncertainty), while issue management is reactive (addressing current problems).

Option A is incorrect - severity varies independently for both.

Option C is incorrect - documentation needs depend on significance, not category.

Option D is incorrect - anyone can identify either risks or issues.
