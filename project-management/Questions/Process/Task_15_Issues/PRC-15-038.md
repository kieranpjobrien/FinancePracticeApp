# Question

What is the benefit of categorizing issues?

## Options

- **A**: It makes the issue log longer
- **B**: It enables pattern analysis, appropriate routing, and targeted process improvements
- **C**: It slows down issue logging
- **D**: It is only required for audits

## Answer

B

## Explanation

The correct answer is B. Categorization (by type, area, cause, severity) enables valuable analysis: are there patterns in certain categories? Should issues of certain types be routed to specific experts? Which categories need process improvements? This insight improves both issue resolution and prevention.

Option A is a disadvantage description, not a benefit.

Option C describes a potential cost, not a benefit.

Option D understates the operational value of categorization.
