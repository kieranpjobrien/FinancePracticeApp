# Question

An external regulatory change creates an issue requiring immediate project changes. The change affects scope, schedule, and potentially budget. The project team needs guidance quickly but the steering committee doesn't meet for two weeks. What should the project manager do?

## Options

- **A**: Wait for the steering committee meeting
- **B**: Make all decisions independently to keep the project moving
- **C**: Assess the changes required, prepare options and recommendations, and request an emergency governance session or delegated authority to make time-sensitive decisions
- **D**: Continue as if the regulatory change hasn't happened

## Answer

C

## Explanation

The correct answer is C. Regulatory changes often require immediate response, but changes of this magnitude may exceed PM authority. The PM should prepare analysis (what's required, options, recommendations) and request expedited governance - either an emergency meeting or temporary delegated authority. This maintains governance while addressing urgency.

Option A risks regulatory non-compliance.

Option B may exceed PM authority for significant changes.

Option D ignores regulatory requirements, creating serious risk.
