# Question

A significant issue is identified that will impact the project timeline by two weeks if not resolved immediately. However, the proposed resolution requires changes that will affect other projects. The project manager has authority over this project but not the others. What is the BEST approach?

## Options

- **A**: Implement the resolution since this project's timeline is at risk
- **B**: Accept the two-week delay to avoid affecting other projects
- **C**: Escalate to program or portfolio level where cross-project decisions can be made, providing impact analysis for both options
- **D**: Negotiate directly with the other project managers

## Answer

C

## Explanation

The correct answer is C. Decisions affecting multiple projects require authority that spans those projects - typically program or portfolio management. The PM should escalate with clear information: impact to this project without resolution, impact to other projects with resolution. The appropriate authority can then make an informed cross-project decision.

Option A exceeds authority and affects other projects without proper authorization.

Option B may accept unnecessary delay without exploring options.

Option D may work informally but lacks proper authority for binding decisions.
