# Question

Which of the following best describes scope creep?

## Options

- **A**: Approved changes to scope made via the integrated change control process.
- **B**: Replanning the backlog at the start of each sprint based on new priorities.
- **C**: Adding agreed quality enhancements to a deliverable to lift customer satisfaction.
- **D**: Uncontrolled expansion of project scope without corresponding adjustments to time, cost, or resources.

## Answer

D

## Explanation

The defining word in "scope creep" is "uncontrolled." (A) is the formal change control process, the antidote. (B) is normal agile re-planning within an existing scope envelope. (C) describes either an approved change or gold plating depending on whether it was authorised — but is not the textbook definition of creep.
