# Question

What is the key difference between Validate Scope and Control Scope?

## Options

- **A**: Validate Scope happens during planning; Control Scope happens during execution.
- **B**: Validate Scope obtains formal acceptance of completed deliverables; Control Scope monitors and manages changes to the scope baseline.
- **C**: Validate Scope is led by the project team; Control Scope is led by the customer.
- **D**: Validate Scope prevents scope creep; Control Scope confirms deliverables meet requirements.

## Answer

B

## Explanation

Validate Scope = acceptance, Control Scope = baseline management. (A) is wrong because both are monitoring & controlling processes that occur during execution. (C) reverses the actors. (D) reverses the definitions outright.
