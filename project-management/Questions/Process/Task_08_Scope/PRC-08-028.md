# Question

What is the scope baseline composed of?

## Options

- **A**: Project charter, stakeholder register, and requirements documentation
- **B**: Scope statement, WBS, and WBS dictionary
- **C**: Requirements traceability matrix, scope statement, and project schedule
- **D**: WBS, cost baseline, and schedule baseline

## Answer

B

## Explanation

The correct answer is B. The scope baseline consists of three components: the project scope statement (what's included and excluded), the WBS (hierarchical decomposition of the work), and the WBS dictionary (detailed descriptions of WBS elements). Together, these define the approved project scope.

Option A includes inputs to scope planning but not the baseline itself.

Option C mixes scope elements with schedule elements and misses the WBS and dictionary.

Option D combines scope with other baselines. The scope baseline is one of three baselines (scope, schedule, cost) that together form the project baseline.
