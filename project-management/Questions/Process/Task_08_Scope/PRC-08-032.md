# Question

A multi-year programme uses rolling-wave planning: detailed plans for the next two release cycles, summary plans beyond that. A new sponsor demands fully detailed plans for all five years before approving continued funding. What should the project manager do?

## Options

- **A**: Produce detailed plans for every phase to satisfy the sponsor's request quickly.
- **B**: Switch the entire programme to a fully predictive lifecycle to align with the sponsor's expectation.
- **C**: Provide detailed near-term plans, summary plans for distant work with explicit cost/schedule ranges, and a written rationale for rolling-wave appropriate to the work.
- **D**: Decline the request on the grounds that rolling-wave planning is current PMI good practice.

## Answer

C

## Explanation

The sponsor needs visibility; the PM owes them an honest, defensible picture. Detailed near-term plans plus ranged estimates for far-term work meets that need without inventing false precision. (A) creates plausible-looking but unreliable plans. (B) abandons an appropriate methodology for one stakeholder's preference. (D) refuses the legitimate need without offering an alternative.
