# Question

A project using rolling wave planning has detailed near-term work but only high-level descriptions of later phases. A new project sponsor wants complete WBS detail for all phases before approving continued funding. How should the PM respond?

## Options

- **A**: Create detailed estimates for all phases to satisfy the sponsor's requirement
- **B**: Explain rolling wave planning and why detailed planning for distant work is wasteful and potentially inaccurate
- **C**: Provide detail for immediate phases with progressively higher contingency ranges for later phases
- **D**: Switch to a fully predictive approach to meet sponsor expectations

## Answer

C

## Explanation

The correct answer is C. This balances the sponsor's need for visibility with planning realities. Detailed near-term planning with contingency ranges for later phases acknowledges uncertainty honestly. The sponsor gets the comprehensive view they need while understanding that later estimates will be refined as more is learned.

Option A creates false precision. Detailed estimates for poorly understood future work are unreliable and may set incorrect expectations.

Option B is educational but doesn't meet the sponsor's legitimate need for a complete picture to make funding decisions.

Option D abandons an appropriate methodology to satisfy one stakeholder's preference.
