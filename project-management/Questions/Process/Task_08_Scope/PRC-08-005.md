# Question

What is the PRIMARY purpose of a Work Breakdown Structure (WBS)?

## Options

- **A**: To assign resources to project tasks
- **B**: To decompose project scope into manageable deliverables
- **C**: To create the project schedule
- **D**: To identify project risks

## Answer

B

## Explanation

The correct answer is B. The WBS decomposes the total scope of work into smaller, more manageable components (work packages). It's a hierarchical representation of all deliverables, providing a foundation for planning, estimating, and controlling the project.

Option A (resource assignment) happens after the WBS is created. Resources are assigned to work packages, but that's not the WBS's primary purpose.

Option C (schedule creation) uses the WBS as input but is a separate process. The schedule adds duration and sequencing to work packages.

Option D (risk identification) may use the WBS to ensure comprehensive risk identification, but that's not the WBS's primary purpose.
