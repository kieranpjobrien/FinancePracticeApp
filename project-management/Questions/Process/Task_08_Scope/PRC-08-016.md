# Question

A project scope statement should include which of the following?

## Options

- **A**: Detailed task assignments for team members
- **B**: Project deliverables, acceptance criteria, and exclusions
- **C**: Complete project budget breakdown
- **D**: Risk register and mitigation strategies

## Answer

B

## Explanation

The correct answer is B. The project scope statement describes the project's deliverables and the work required to create them. Key elements include deliverables, acceptance criteria, exclusions (what's NOT included), constraints, and assumptions.

Option A belongs in the resource management plan or project schedule, not the scope statement.

Option C belongs in the cost management plan and cost baseline, not the scope statement.

Option D belongs in the risk register and risk management plan, not the scope statement.
