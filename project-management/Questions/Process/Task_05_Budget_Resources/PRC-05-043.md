# Question

The project has a negative Cost Variance (CV) but the team insists the variance is due to timing differences in invoicing rather than actual overruns. How should the PM investigate?

## Options

- **A**: Accept the team's explanation without verification
- **B**: Analyse the variance in detail: are costs truly higher or just recorded earlier? Review work completed vs planned. Verify with accounting. If timing, document for stakeholders; if real, address root cause.
- **C**: Report the variance without investigation
- **D**: Hide the variance until it corrects

## Answer

B

## Explanation

The correct answer is B. Variance investigation is essential. Analyse: is this an accounting timing issue (accruals, invoice timing) or a real overrun? Verify: check with accounting, review work completion. Report accurately: if timing, explain that to stakeholders; if real, address the cause. Understanding variance nature enables appropriate response.

Option A accepts an explanation that may be incorrect.

Option C reports without understanding.

Option D hides information from stakeholders.
