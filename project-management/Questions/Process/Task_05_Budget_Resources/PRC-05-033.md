# Question

What is resource smoothing?

## Options

- **A**: Making resources physically smooth
- **B**: A technique that adjusts activities within float to optimise resource usage without extending the project end date
- **C**: Removing difficult resources from the project
- **D**: Averaging resource costs

## Answer

B

## Explanation

The correct answer is B. Resource smoothing adjusts the schedule within available float to reduce resource demand peaks while keeping the same project end date. Unlike resource levelling, it maintains the deadline. The trade-off is that it can only address resource issues where float exists.

Options A, C, and D misunderstand the concept.
