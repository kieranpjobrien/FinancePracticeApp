# Question

Multiple projects in a portfolio are competing for the same scarce resources. Each PM advocates for their project. How should resource allocation be decided?

## Options

- **A**: First-come, first-served
- **B**: Portfolio-level governance should prioritise based on strategic alignment, value delivery, resource efficiency, and dependencies - individual projects may need to adjust based on portfolio priorities
- **C**: Loudest PM wins
- **D**: Each PM allocates independently

## Answer

B

## Explanation

The correct answer is B. Portfolio-level resource decisions should optimise for organisational value, not individual project advocacy. Strategic alignment: which projects matter most? Value: where does the resource create most value? Efficiency: how can resources be shared? Dependencies: what sequence makes sense? Individual PMs should input but not decide portfolio allocation.

Option A doesn't optimise for value.

Option C rewards behaviour, not value.

Option D leads to conflicts and suboptimal allocation.
