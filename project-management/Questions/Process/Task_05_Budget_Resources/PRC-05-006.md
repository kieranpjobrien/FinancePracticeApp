# Question

A project manager is creating resource assignments for a 12-month project. One critical team member is needed full-time for months 1-3, then 50% for months 4-12. However, the functional manager says this person can only be assigned 50% for the entire project. What should the PM do?

## Options

- **A**: Accept the 50% allocation and adjust the schedule accordingly
- **B**: Escalate to the sponsor to override the functional manager
- **C**: Negotiate with the functional manager to find alternatives, such as temporary full allocation or additional resources
- **D**: Plan the project assuming full allocation and address conflicts when they arise

## Answer

C

## Explanation

The correct answer is C. Resource conflicts are common in matrix organisations. The PM should negotiate solutions - perhaps the functional manager can provide full allocation for months 1-3 with reduced allocation later, or provide an additional resource to cover the gap. Negotiation respects both project needs and functional constraints.

Option A (accept and adjust) may significantly impact the project without exploring alternatives first.

Option B (escalate immediately) damages the relationship with the functional manager. Escalation should follow failed negotiation, not precede it.

Option D (plan and hope) creates a known resource conflict that will materialise during execution, causing preventable problems.
