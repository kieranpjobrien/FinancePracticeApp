# Question

A project has BAC = $100,000, EV = $40,000, AC = $50,000, and PV = $45,000. What is the Cost Performance Index (CPI)?

## Options

- **A**: 0.80
- **B**: 0.89
- **C**: 1.11
- **D**: 1.25

## Answer

A

## Explanation

CPI = EV / AC = 40,000 / 50,000 = 0.80. A CPI below 1.0 means the project is earning less value per dollar spent than planned — over budget. (B) 0.89 is SPI = EV/PV = 40/45. (C) 1.11 is PV/EV, an inverted SPI not typically reported. (D) 1.25 is AC/EV, the inverted CPI.
