# Question

A project has a Budget at Completion (BAC) of $100,000. At a status point, the Earned Value (EV) is $40,000, Actual Cost (AC) is $50,000, and Planned Value (PV) is $45,000. What is the Cost Performance Index (CPI)?

## Options

- **A**: 0.80
- **B**: 0.89
- **C**: 1.11
- **D**: 1.25

## Answer

A

## Explanation

The correct answer is A. Cost Performance Index (CPI) = Earned Value (EV) / Actual Cost (AC) = $40,000 / $50,000 = 0.80. A CPI of 0.80 means the project is getting $0.80 of value for every $1.00 spent - it's over budget.

Option B (0.89) would be EV/PV, which is the Schedule Performance Index (SPI), not CPI.

Option C (1.11) would be PV/EV, which is the inverse of SPI.

Option D (1.25) would be AC/EV, which is the inverse of CPI.
