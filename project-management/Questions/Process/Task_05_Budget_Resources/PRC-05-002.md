# Question

A project has a BAC of $200,000. Current status shows EV=$80,000, AC=$100,000, and the project is 40% complete. Assuming current performance continues, what is the Estimate at Completion (EAC)?

## Options

- **A**: $200,000
- **B**: $220,000
- **C**: $250,000
- **D**: $280,000

## Answer

C

## Explanation

The correct answer is C. When assuming current cost performance continues: EAC = BAC / CPI.

First, calculate CPI = EV / AC = $80,000 / $100,000 = 0.80

Then, EAC = BAC / CPI = $200,000 / 0.80 = $250,000

This means if the project continues at 80% cost efficiency, the final cost will be $250,000.

Option A assumes costs magically return to plan (EAC = BAC).

Option B is incorrect calculation.

Option D might result from using an incorrect formula.
