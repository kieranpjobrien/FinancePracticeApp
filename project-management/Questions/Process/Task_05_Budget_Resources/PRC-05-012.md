# Question

A project has a Budget at Completion (BAC) of $100,000. After spending $40,000, only $30,000 worth of work has been completed. What is the Cost Performance Index (CPI)?

## Options

- **A**: 0.75
- **B**: 1.33
- **C**: 0.30
- **D**: 1.00

## Answer

A

## Explanation

The correct answer is A. CPI = EV / AC = $30,000 / $40,000 = 0.75. This means the project is getting 75 cents of value for every dollar spent, indicating cost overrun. CPI below 1.0 indicates over budget; above 1.0 indicates under budget.

Option B (1.33) reverses the formula.

Option C doesn't use the correct values.

Option D would indicate on-budget, which isn't the case.
