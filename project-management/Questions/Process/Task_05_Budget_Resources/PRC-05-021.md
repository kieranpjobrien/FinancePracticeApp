# Question

What is a resource histogram?

## Options

- **A**: A graph of historical resource costs
- **B**: A bar chart showing resource usage or requirements over time, helping identify periods of over or under-allocation
- **C**: A list of all project resources
- **D**: A pie chart of budget allocation

## Answer

B

## Explanation

The correct answer is B. A resource histogram displays resource requirements over time as a bar chart. The height of each bar shows resource demand for that period. This visualisation helps identify peak demand periods, over-allocation that needs to be resolved, and periods when resources might be available for other work.

Options A, C, and D describe different tools.
