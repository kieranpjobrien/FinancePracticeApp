# Question

Budget tracking shows the project spent exactly what was planned each month. However, deliverables are behind schedule. What does this indicate?

## Options

- **A**: The project is on track since budget is on target
- **B**: The project is behind schedule but spending at the planned rate - less work is being completed than planned. This may indicate future cost overruns as catch-up effort is needed.
- **C**: The budget forecast is wrong
- **D**: This situation is impossible

## Answer

B

## Explanation

The correct answer is B. Spending to plan while delivering behind schedule indicates poor schedule performance (low SPI). The project is getting less value for the money being spent. If the full scope must be delivered, either future costs will increase (catch-up), or the schedule will slip further. This is a warning sign requiring investigation.

Option A ignores the schedule issue.

Option C assumes incorrectly.

Option D doesn't recognize this common pattern.
