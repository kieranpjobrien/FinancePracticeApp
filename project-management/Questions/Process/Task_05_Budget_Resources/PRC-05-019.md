# Question

The PM is managing a hybrid project with a predictive infrastructure stream and an agile software stream. The organisation requires a single consolidated budget. What is the BEST approach?

## Options

- **A**: Force both streams into the same estimation method to produce a single unified budget view.
- **B**: Estimate each stream using appropriate methods, then consolidate with documented assumptions and confidence levels.
- **C**: Only report the infrastructure stream's budget since it has better-defined costs and higher certainty.
- **D**: Provide only a rough consolidated figure without breaking it down by stream or estimating method.

## Answer

B

## Explanation

Tailoring: each stream uses the method that fits its nature, then consolidation makes the differences explicit. (B) is the PMI hybrid approach. (A) forcing a single method produces bad estimates in one stream or the other. (C) reporting only the predictive stream hides real costs and misleads the sponsor. (D) a rough number with no decomposition prevents meaningful change management later.
