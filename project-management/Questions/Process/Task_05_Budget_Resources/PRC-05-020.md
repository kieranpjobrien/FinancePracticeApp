# Question

A project has EV = $60,000 and PV = $80,000. What is the Schedule Variance (SV)?

## Options

- **A**: $20,000
- **B**: -$20,000
- **C**: $140,000
- **D**: 0.75

## Answer

B

## Explanation

The correct answer is B. SV = EV - PV = $60,000 - $80,000 = -$20,000. A negative SV indicates the project is behind schedule - less work has been completed than planned. The project has earned $20,000 less value than scheduled.

Option A has the wrong sign.

Option C adds instead of subtracts.

Option D is SPI (EV/PV), not SV.
