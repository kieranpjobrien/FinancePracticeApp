# Question

A key resource is consistently over-allocated across multiple projects. Their functional manager can't provide additional time. How should the PM address this resource constraint?

## Options

- **A**: Demand more time from the resource
- **B**: Analyse which tasks truly require this resource, explore alternatives (other resources, different approaches), adjust schedule if needed, and escalate through governance if the constraint impacts critical objectives
- **C**: Accept project delays without analysis
- **D**: Replace the resource with someone unqualified

## Answer

B

## Explanation

The correct answer is B. Resource constraints require analysis and creative problem-solving. Analyse: does every task truly need this specific resource? Alternatives: can others be trained? Can approaches change? Schedule: can work be resequenced to fit their availability? Escalate: if the constraint significantly impacts objectives, decision-makers need to know.

Option A won't succeed if the manager can't provide more time.

Option C accepts impact without exploring solutions.

Option D trades one problem for another (quality).
