# Question

A project has BAC = $200,000, EV = $80,000, AC = $100,000, and PV = $90,000. If current performance continues, what is the Estimate at Completion (EAC)?

## Options

- **A**: $200,000
- **B**: $250,000
- **C**: $220,000
- **D**: $280,000

## Answer

B

## Explanation

The correct answer is B. EAC = BAC / CPI. First calculate CPI = EV / AC = $80,000 / $100,000 = 0.8. Then EAC = $200,000 / 0.8 = $250,000. This forecasts the project will cost $250,000 if current cost performance continues.

Option A assumes no variance continues.

Option C and D use incorrect calculations.
