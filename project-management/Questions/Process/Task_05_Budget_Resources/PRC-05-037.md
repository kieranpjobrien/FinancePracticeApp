# Question

What is the purpose of a resource breakdown structure (RBS)?

## Options

- **A**: To break resources into smaller pieces
- **B**: To hierarchically categorise project resources by type, enabling resource planning, assignment, and tracking
- **C**: To fire underperforming resources
- **D**: To rank resources by cost

## Answer

B

## Explanation

The correct answer is B. The RBS is a hierarchical structure that categorises resources. Common categories include: labour (by skill/role), materials, equipment, and services. It helps with resource planning (what types are needed), assignment (matching resources to work), and tracking (where resources are being used).

Options A, C, and D misunderstand the purpose of the RBS.
