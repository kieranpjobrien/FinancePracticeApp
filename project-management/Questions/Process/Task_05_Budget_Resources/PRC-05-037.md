# Question

What is the primary purpose of a Resource Breakdown Structure (RBS)?

## Options

- **A**: To rank individual team members by performance for end-of-project appraisal.
- **B**: To allocate the project budget to cost accounts for earned value reporting.
- **C**: To schedule activities so resource peaks are smoothed within available float.
- **D**: To hierarchically categorise project resources by type and category for planning, assignment, and tracking.

## Answer

D

## Explanation

(D) matches the PMBOK definition: the RBS is a hierarchical list of resources organised by category and type (labour, equipment, material, etc.), used as an input to estimating, scheduling, and reporting. (A) is performance management, not RBS. (B) describes the cost baseline / control account structure derived from the WBS, not the RBS. (C) describes resource smoothing, a scheduling technique that uses the RBS as an input but is not its purpose.
