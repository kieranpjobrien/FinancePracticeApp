# Question

What is the cost baseline in project management?

## Options

- **A**: The original project estimate before changes
- **B**: The approved time-phased project budget excluding management reserves, used to measure cost performance
- **C**: The maximum budget the organisation can afford
- **D**: The minimum acceptable project cost

## Answer

B

## Explanation

The correct answer is B. The cost baseline is the approved, time-phased project budget against which actual spending is compared to measure performance. It includes contingency reserves but excludes management reserves. Changes to the baseline require formal change control approval.

Options A, C, and D don't accurately describe the cost baseline concept.
