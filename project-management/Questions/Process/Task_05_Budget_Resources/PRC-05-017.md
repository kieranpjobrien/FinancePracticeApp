# Question

What is resource levelling?

## Options

- **A**: Making all resources equally skilled
- **B**: A technique that adjusts the schedule to address resource constraints, typically extending the schedule to avoid over-allocation
- **C**: Levelling a physical workspace
- **D**: Reducing all resource costs equally

## Answer

B

## Explanation

The correct answer is B. Resource levelling adjusts the project schedule to resolve resource over-allocation while keeping resource usage within limits. This typically extends the project duration but ensures resources aren't double-booked. It's a trade-off: schedule for resource availability.

Option A relates to skills, not levelling.

Option C is literal, not project management terminology.

Option D relates to cost, not resource scheduling.
