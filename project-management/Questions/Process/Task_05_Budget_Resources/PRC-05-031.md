# Question

The PM notices that earned value metrics show good performance (CPI = 1.05, SPI = 1.02), but stakeholders are unhappy because delivered features don't meet their needs. What does this indicate?

## Options

- **A**: The metrics are wrong
- **B**: Earned value measures delivery efficiency but not value delivery - the project may be efficiently building the wrong things. Quality and stakeholder satisfaction metrics are also needed.
- **C**: Stakeholders don't understand the metrics
- **D**: Everything is fine because metrics are positive

## Answer

B

## Explanation

The correct answer is B. EVM measures whether work is being completed on time and budget, not whether it's the right work. A project can be efficient (good CPI/SPI) while delivering low value (wrong features, poor quality). Complement EVM with outcome metrics: customer satisfaction, defect rates, benefit realisation.

Option A dismisses valid metrics without analysis.

Option C shifts blame instead of addressing the issue.

Option D ignores the stakeholder dissatisfaction signal.
