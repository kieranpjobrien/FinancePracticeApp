# Question

What does a CPI of 1.2 indicate?

## Options

- **A**: The project is 20% over budget
- **B**: The project is performing 20% better than budget - getting $1.20 of value for every dollar spent
- **C**: 20% of the project is complete
- **D**: The project will be 20% late

## Answer

B

## Explanation

The correct answer is B. CPI = EV / AC. A CPI of 1.2 means for every dollar spent (AC), the project has earned $1.20 of value (EV). This indicates efficient cost performance - the project is under budget. CPI > 1.0 is favourable; CPI < 1.0 is unfavourable.

Option A reverses the interpretation - that would be CPI of about 0.83.

Option C confuses CPI with percent complete.

Option D relates to schedule, not cost.
