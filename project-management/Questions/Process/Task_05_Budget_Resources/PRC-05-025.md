# Question

A project reports a Cost Performance Index (CPI) of 1.2 at the end of the reporting period. What does this indicate about cost performance?

## Options

- **A**: The project has spent 20% of its total budget so far.
- **B**: The project is forecast to finish 20% over its approved budget.
- **C**: The project will be delivered approximately 20% later than planned.
- **D**: The project is earning $1.20 of value for every $1.00 spent, indicating favourable cost performance.

## Answer

D

## Explanation

CPI = EV / AC. A CPI greater than 1.0 means earned value exceeds actual cost, so the project is under budget for the work completed. (D) correctly states the interpretation. (A) confuses CPI with percent spent, which is a different metric. (B) reverses the meaning — that interpretation would correspond to a CPI of about 0.83. (C) refers to schedule performance (SPI), not cost.
