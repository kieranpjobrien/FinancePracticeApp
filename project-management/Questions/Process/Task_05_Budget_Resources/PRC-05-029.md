# Question

What is the difference between analogous estimating and parametric estimating?

## Options

- **A**: They are the same technique
- **B**: Analogous uses historical data from similar projects; parametric uses statistical relationships between variables and parameters
- **C**: Analogous is more accurate
- **D**: Parametric can only be used for schedules

## Answer

B

## Explanation

The correct answer is B. Analogous (top-down) estimating uses actual data from previous similar projects as a basis. Parametric estimating uses statistical models based on project parameters (e.g., cost per square metre, hours per feature). Both use historical data but in different ways.

Option A misses the distinction.

Option C isn't universally true - accuracy depends on data quality and applicability.

Option D is incorrect - parametric applies to both cost and schedule.
