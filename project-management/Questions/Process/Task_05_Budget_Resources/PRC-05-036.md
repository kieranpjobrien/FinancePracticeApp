# Question

A project has Actual Cost (AC) to date of $40,000 and a bottom-up Estimate to Complete (ETC) of $60,000 for the remaining work. What is the Estimate at Completion (EAC)?

## Options

- **A**: $100,000
- **B**: $60,000
- **C**: $40,000
- **D**: $20,000

## Answer

A

## Explanation

EAC = AC + ETC = $40,000 + $60,000 = $100,000. (A) is correct. This is the bottom-up EAC formula used when the remaining work has been re-estimated independently rather than extrapolated from past performance. (B) reports only the remaining work. (C) reports only the actual cost to date. (D) is the absolute difference between AC and ETC, which has no meaning as an EAC.
