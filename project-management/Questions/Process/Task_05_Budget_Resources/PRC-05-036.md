# Question

The Estimate to Complete (ETC) is $60,000 and Actual Cost (AC) to date is $40,000. What is the Estimate at Completion (EAC)?

## Options

- **A**: $60,000
- **B**: $100,000
- **C**: $40,000
- **D**: $20,000

## Answer

B

## Explanation

The correct answer is B. EAC = AC + ETC = $40,000 + $60,000 = $100,000. This is the bottom-up EAC calculation when the remaining work can be estimated independently. EAC represents the total expected cost when the project is complete.

Options A, C, and D use only one component or calculate incorrectly.
