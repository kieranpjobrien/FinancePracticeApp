# Question

A project has BAC = $500,000, CPI = 0.9, and SPI = 0.8. Using the formula that accounts for both cost and schedule performance, what is the EAC?

## Options

- **A**: $555,556
- **B**: $625,000
- **C**: $694,444
- **D**: $500,000

## Answer

C

## Explanation

The correct answer is C. EAC = BAC / (CPI × SPI) = $500,000 / (0.9 × 0.8) = $500,000 / 0.72 = $694,444. This formula accounts for both cost and schedule performance degradation, providing a more comprehensive forecast when both factors are affecting the project.

Option A uses only CPI.

Option B uses only SPI.

Option D assumes original budget holds.
