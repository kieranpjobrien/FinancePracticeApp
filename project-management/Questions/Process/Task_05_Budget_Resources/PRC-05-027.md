# Question

A project has BAC = $500,000, CPI = 0.9, and SPI = 0.8. Using the EAC formula that assumes both cost and schedule performance will continue at their current rates for the remaining work, what is the EAC?

## Options

- **A**: $555,556
- **B**: $625,000
- **C**: $694,444
- **D**: $720,000

## Answer

C

## Explanation

The EAC formula that accounts for both cost and schedule performance is EAC = BAC / (CPI x SPI) = 500,000 / (0.9 x 0.8) = 500,000 / 0.72 = 694,444. (C) is correct. (A) uses only CPI: 500,000 / 0.9 = 555,556. (B) uses only SPI: 500,000 / 0.8 = 625,000. (D) is a plausible-looking distractor that does not match any standard EAC formula.
