# Question

A project manager receives a request to add a feature that would cost $25,000. The project has a $50,000 management reserve. Should the PM use the management reserve for this request?

## Options

- **A**: Yes, management reserve is available for any project needs
- **B**: No, management reserve is for unknown risks, not scope changes - this requires a change request
- **C**: Yes, if the sponsor approves
- **D**: No, management reserve can only be used at project end

## Answer

B

## Explanation

The correct answer is B. Management reserve is budget set aside for "unknown unknowns" - unforeseen risks that weren't identified during planning. New features are scope changes, not unforeseen risks. Scope changes should follow the change control process, potentially adjusting the budget baseline if approved.

Option A confuses management reserve with discretionary funds.

Option C partially addresses approval but doesn't address the fundamental misuse of reserve purpose.

Option D incorrectly limits when reserve can be used. Reserve can be used when unknown risks materialise, regardless of timing.
