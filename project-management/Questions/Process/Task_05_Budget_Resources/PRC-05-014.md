# Question

What is the difference between Actual Cost (AC) and Earned Value (EV)?

## Options

- **A**: They are the same thing
- **B**: AC is money spent; EV is the budgeted value of work actually completed
- **C**: AC is future costs; EV is past costs
- **D**: AC is for labour; EV is for materials

## Answer

B

## Explanation

The correct answer is B. Actual Cost (AC) is the money actually spent on work performed. Earned Value (EV) is the budgeted value (not actual cost) of work that has been completed. Comparing these shows whether you're getting value for money: if AC > EV, you've spent more than the work was worth.

Option A misses the key distinction.

Option C and D mischaracterise both terms.
