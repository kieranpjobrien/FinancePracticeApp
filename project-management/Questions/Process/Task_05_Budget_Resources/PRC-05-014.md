# Question

What does 'rolling wave planning' mean?

## Options

- **A**: Planning near-term work in detail while leaving later work at a higher level, then elaborating later work as it approaches.
- **B**: Delivering the project in waves of deployment releases.
- **C**: Rotating PM responsibility across team members each phase.
- **D**: Re-planning the full project on a rolling monthly basis.

## Answer

A

## Explanation

Rolling wave is progressive elaboration — detailed planning for the near term, higher-level planning for later work that gets detailed as the project progresses. (B) describes phased delivery, a different concept. (C) is about PM role, unrelated. (D) full re-planning monthly is heavy and not what rolling wave means.
