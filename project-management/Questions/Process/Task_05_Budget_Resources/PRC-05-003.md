# Question

What does a Cost Performance Index (CPI) of 0.85 indicate?

## Options

- **A**: The project is 15% under budget
- **B**: The project is getting 85 cents of work for every dollar spent
- **C**: The project has completed 85% of the work
- **D**: The project will finish 15% early

## Answer

B

## Explanation

The correct answer is B. CPI measures cost efficiency - how much value (EV) you're getting for money spent (AC). A CPI of 0.85 means for every $1.00 spent, only $0.85 of value is earned. The project is over budget - you're paying more than planned for the work completed.

Option A has the direction wrong - CPI < 1 means over budget, not under.

Option C confuses CPI with percent complete.

Option D confuses cost performance with schedule performance.
