# Question

Which component is included in the cost baseline?

## Options

- **A**: Management reserve.
- **B**: Funding limit reconciliation.
- **C**: Contingency reserve.
- **D**: Escalation provision.

## Answer

C

## Explanation

Cost baseline = project cost estimates + contingency reserve. Management reserve sits outside the baseline. (A) management reserve is excluded from the baseline by definition. (B) funding limit reconciliation is an analysis technique, not a baseline component. (D) 'escalation provision' isn't a standard PMI cost baseline component — inflation adjustments are handled within estimates.
