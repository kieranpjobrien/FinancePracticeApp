# Question

A 24-month defence programme reports at month 10: BAC = $6,000,000, EV = $2,200,000, AC = $2,000,000, PV = $2,500,000. What do the performance indices tell the project manager about programme status?

## Options

- **A**: CPI is 1.10 and SPI is 0.88, so the programme is under budget but behind schedule and needs recovery actions.
- **B**: CPI is 0.91 and SPI is 1.14, so the programme is over budget but ahead of schedule and needs cost actions.
- **C**: CPI is 1.10 and SPI is 1.14, so the programme is performing well on both axes and needs no management action.
- **D**: CPI is 0.91 and SPI is 0.88, so the programme is over budget and behind schedule and needs intervention.

## Answer

A

## Explanation

CPI = EV/AC = 2200/2000 = 1.10; SPI = EV/PV = 2200/2500 = 0.88. The project is earning more value per dollar than planned (under budget) but has not earned as much value as was scheduled (behind schedule). (B) swaps the ratios. (C) inverts SPI. (D) inverts CPI.
