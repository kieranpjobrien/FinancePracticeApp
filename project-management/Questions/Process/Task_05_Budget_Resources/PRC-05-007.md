# Question

A project has BAC = $800,000, EV = $300,000, AC = $360,000. Assuming future work will be performed at the budgeted rate (not the current rate), what is the EAC?

## Options

- **A**: $920,000
- **B**: $960,000
- **C**: $860,000
- **D**: $1,066,667

## Answer

C

## Explanation

When future work is expected to revert to budgeted rate, EAC = AC + (BAC - EV) = 360,000 + (800,000 - 300,000) = 360,000 + 500,000 = $860,000. (A) $920,000 doesn't match a standard formula. (B) $960,000 doesn't match. (D) $1,066,667 = BAC / CPI, which assumes current performance continues — the other scenario.
