# Question

A project has BAC = $400,000, EV = $150,000, AC = $180,000, and PV = $200,000. Calculate the Schedule Variance (SV) and Cost Variance (CV).

## Options

- **A**: SV = -$50,000, CV = -$30,000
- **B**: SV = $50,000, CV = $30,000
- **C**: SV = -$30,000, CV = -$50,000
- **D**: SV = $30,000, CV = $50,000

## Answer

A

## Explanation

The correct answer is A.

**Schedule Variance (SV) = EV - PV**
SV = $150,000 - $200,000 = **-$50,000**

**Cost Variance (CV) = EV - AC**
CV = $150,000 - $180,000 = **-$30,000**

Both variances are negative, meaning:
- The project is **behind schedule** (delivered less value than planned)
- The project is **over budget** (spent more than the value earned)

Negative variances indicate unfavourable performance in EVM.
