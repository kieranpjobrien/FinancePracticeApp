# Question

A project has a Budget at Completion (BAC) of $500,000. At month 6, the project has an Earned Value (EV) of $200,000 and Actual Cost (AC) of $250,000. What is the Cost Performance Index (CPI)?

## Options

- **A**: 0.80
- **B**: 1.25
- **C**: 0.40
- **D**: 2.50

## Answer

A

## Explanation

The correct answer is A. The Cost Performance Index (CPI) measures cost efficiency.

**Formula:** CPI = EV / AC

**Calculation:** CPI = $200,000 / $250,000 = **0.80**

A CPI of 0.80 means for every dollar spent, only $0.80 of value is earned. This indicates the project is over budget - it's spending more than planned to achieve the work completed.

Option B (1.25) would be AC/EV, which is the inverse.

Option C (0.40) might be EV/BAC, which isn't a standard EVM metric.

Option D (2.50) is AC/EV rounded differently.
