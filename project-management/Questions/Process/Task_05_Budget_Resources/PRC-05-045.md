# Question

What is the To-Complete Performance Index (TCPI)?

## Options

- **A**: A measure of past performance
- **B**: The cost performance that must be achieved on remaining work to meet a target (BAC or EAC)
- **C**: The team's productivity index
- **D**: A schedule metric

## Answer

B

## Explanation

The correct answer is B. TCPI = (BAC - EV) / (BAC - AC) for BAC target, or (BAC - EV) / (EAC - AC) for EAC target. It shows the efficiency that must be achieved on remaining work to meet the target. A TCPI significantly above 1.0 indicates the target may be unachievable with current approaches.

Option A describes CPI or SPI.

Option C is not a standard EVM metric.

Option D relates to schedule performance.
