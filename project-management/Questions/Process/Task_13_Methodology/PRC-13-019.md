# Question

A project involves integrating software from three different vendors. Each vendor uses their own development methodology - one uses waterfall, one uses Scrum, and one uses Kanban. The integration project needs to coordinate deliveries from all three. What approach should the project manager use?

## Options

- **A**: Require all vendors to adopt a common methodology
- **B**: Use an integration methodology that coordinates across different vendor approaches, focusing on interface points and milestone alignment
- **C**: Let each vendor work independently and integrate at the end
- **D**: Assign different project managers for each vendor relationship

## Answer

B

## Explanation

The correct answer is B. The integration project needs to coordinate deliverables regardless of how each vendor produces them. Focusing on interface points (where systems connect), milestone alignment (when deliverables need to be ready), and clear communication bridges the methodology differences.

Option A may not be contractually possible and disrupts vendor teams.

Option C risks discovering integration issues too late.

Option D fragments leadership when coordination is the primary need.
