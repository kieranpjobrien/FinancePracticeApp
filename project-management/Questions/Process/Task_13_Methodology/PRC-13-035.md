# Question

A project manager is asked to lead a project in an organization that has never done formal project management. There are no templates, no PMO, and team members have varied expectations about how projects work. What approach should the project manager take for methodology?

## Options

- **A**: Implement a comprehensive methodology immediately
- **B**: Work without any methodology since the organization has no standards
- **C**: Start with lightweight, essential practices appropriate to the project, then evolve based on what works in this organizational context
- **D**: Request transfer to a more mature organization

## Answer

C

## Explanation

The correct answer is C. In organizations without PM maturity, implementing heavy methodology creates resistance and overhead. Starting with essential practices (clear goals, basic planning, regular status) that provide value, then evolving based on organizational learning, builds PM capability sustainably. This is methodology tailoring to organizational context.

Option A may overwhelm an organization without PM culture.

Option B misses the opportunity to bring value through good practices.

Option D abandons an opportunity to help an organization develop.
