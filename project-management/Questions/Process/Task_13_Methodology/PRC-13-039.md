# Question

A distributed team spans multiple countries with different holiday schedules, cultural approaches to meetings, and time zone differences of up to 12 hours. What methodology considerations are important?

## Options

- **A**: Require all team members to work the same hours
- **B**: Choose practices that accommodate asynchronous communication and reduce dependency on real-time interaction, while establishing core collaboration windows where possible
- **C**: Only use predictive methodology since daily standups are difficult
- **D**: Assign all work individually to avoid coordination needs

## Answer

B

## Explanation

The correct answer is B. Distributed teams need methodology adaptation. Asynchronous communication tools and practices reduce dependency on synchronous meetings. Where real-time interaction is valuable, identifying core overlap windows that work for all time zones enables coordination without unreasonable hour requirements.

Option A is impractical and potentially illegal (forced night work).

Option C incorrectly assumes agile requires daily synchronous standups - async updates can achieve similar goals.

Option D eliminates collaboration benefits and may not be feasible for complex work.
