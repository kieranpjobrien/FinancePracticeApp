# Question

What is "tailoring" in project management methodology selection?

## Options

- **A**: Choosing between agile and waterfall
- **B**: Adapting the methodology, processes, and practices to fit the specific project's needs and context
- **C**: Creating a completely new methodology for each project
- **D**: Using the organization's standard methodology without changes

## Answer

B

## Explanation

The correct answer is B. Tailoring means adapting established methodologies, processes, tools, and techniques to suit the specific project's characteristics, constraints, and environment. No single approach fits all projects, so methodologies should be adjusted appropriately.

Option A is methodology selection, not tailoring.

Option C describes creating new methodologies, which is different from adapting existing ones.

Option D is the opposite of tailoring - it's applying a standard without adaptation.
