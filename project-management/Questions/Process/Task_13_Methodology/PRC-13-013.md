# Question

A project manager is evaluating agile frameworks for a software development project. The team is new to agile but the organization wants to adopt it. The project has complex dependencies with other systems. Which framework would be MOST appropriate to start with?

## Options

- **A**: Large-Scale Scrum (LeSS)
- **B**: Scaled Agile Framework (SAFe)
- **C**: Basic Scrum with appropriate adaptation for the team's learning curve
- **D**: Extreme Programming (XP) with all practices implemented immediately

## Answer

C

## Explanation

The correct answer is C. For a team new to agile, starting with basic Scrum provides a solid foundation without overwhelming complexity. Scrum's defined roles, events, and artifacts create structure while being flexible enough to adapt as the team learns. More advanced frameworks can be considered as maturity grows.

Option A and B are scaling frameworks designed for multiple teams, which adds unnecessary complexity for a new-to-agile team.

Option D implements many technical practices at once, which may overwhelm a team still learning basic agile concepts.
