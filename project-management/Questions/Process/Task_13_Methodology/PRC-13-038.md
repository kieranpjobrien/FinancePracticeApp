# Question

What is "rolling wave planning" in project management?

## Options

- **A**: Planning only one day at a time
- **B**: Detailed planning for near-term work with higher-level planning for future work, progressively elaborating as the project proceeds
- **C**: Planning everything in detail at the project start
- **D**: Never planning, just reacting to events

## Answer

B

## Explanation

The correct answer is B. Rolling wave planning recognizes that near-term work can be planned in detail while distant work is only known at a high level. As the project progresses, future work is elaborated into detail. This approach is a middle ground between comprehensive upfront planning and no planning at all.

Option A is too reactive with no forward visibility.

Option C describes comprehensive planning which may not be possible when future work is uncertain.

Option D describes no planning, which is not rolling wave.
