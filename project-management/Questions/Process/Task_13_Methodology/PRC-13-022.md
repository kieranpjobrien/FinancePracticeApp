# Question

Which approach is characterised by detailed upfront planning, sequential phases, and formal phase gates?

## Options

- **A**: Predictive (waterfall)
- **B**: Scrum
- **C**: Kanban
- **D**: Lean startup

## Answer

A

## Explanation

Predictive delivery is the textbook description in the question — define everything up front, work through phases, gate between them. (B) is a timeboxed iterative framework with no big upfront design. (C) is flow-based with no phases. (D) is a discovery model built around build-measure-learn loops.
