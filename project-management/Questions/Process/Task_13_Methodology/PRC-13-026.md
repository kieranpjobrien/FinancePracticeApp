# Question

What does a hybrid project management approach combine?

## Options

- **A**: Multiple agile frameworks only
- **B**: Elements from both predictive and adaptive/agile approaches tailored to project needs
- **C**: Only the documentation from predictive with agile ceremonies
- **D**: Using different approaches for different stakeholders

## Answer

B

## Explanation

The correct answer is B. A hybrid approach combines elements from predictive (waterfall) and adaptive (agile) methodologies based on what fits the project's characteristics. For example, a project might use predictive planning for known requirements while using agile iterations for uncertain components.

Option A describes combining multiple agile frameworks, not a predictive-agile hybrid.

Option C is too narrow - hybrid approaches consider all elements, not just documentation and ceremonies.

Option D describes stakeholder-based variation, which isn't the definition of hybrid methodology.
