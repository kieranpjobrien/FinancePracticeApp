# Question

An organization's PMO mandates that all projects use their standard predictive methodology. A project manager is assigned a research and development project where outcomes are highly uncertain. What should the project manager do?

## Options

- **A**: Follow the PMO mandate strictly without question
- **B**: Ignore the mandate and use agile
- **C**: Discuss with the PMO the project's characteristics and propose appropriate tailoring or methodology modifications while respecting organizational governance
- **D**: Decline the project assignment

## Answer

C

## Explanation

The correct answer is C. The PM should advocate for appropriate approaches based on project characteristics while working within organizational governance. This means explaining why the standard approach may not fit, proposing alternatives, and seeking approval for tailoring. Most PMOs allow exceptions when properly justified.

Option A may set the project up for failure by using an inappropriate approach.

Option B violates organizational governance and could create problems for the PM.

Option D abandons the opportunity to influence methodology selection appropriately.
