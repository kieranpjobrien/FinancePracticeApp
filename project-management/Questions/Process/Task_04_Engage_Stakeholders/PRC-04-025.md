# Question

The project sponsor is also the functional manager of several team members. They're using their dual role to direct team members outside of project governance. This creates confusion and scope creep. How should the PM address this?

## Options

- **A**: Accept it since the sponsor is powerful
- **B**: Have a private, respectful conversation with the sponsor about the impact, clarify governance roles, and propose ways to handle their input through appropriate channels
- **C**: Complain to the sponsor's manager
- **D**: Ignore the problem

## Answer

B

## Explanation

The correct answer is B. This is a governance issue requiring diplomatic resolution. Private conversation: address it directly but respectfully with the sponsor. Impact: explain how bypassing governance causes confusion and scope issues. Clarify roles: sponsor provides direction through governance; team receives direction through PM. Propose solutions: how can their input be channelled appropriately?

Option A allows dysfunction to continue.

Option C escalates before attempting direct resolution.

Option D ignores governance breakdown.
