# Question

Stakeholders have different preferred communication styles - some want formal reports, others prefer informal chats, some want email, others want meetings. Managing all these preferences is becoming overwhelming. What should the PM do?

## Options

- **A**: Use one communication style for everyone
- **B**: Prioritise high-power/high-interest stakeholders for customised communication, establish standard channels for broader groups, and be flexible where practical without overextending
- **C**: Stop all communication until consensus on style
- **D**: Delegate all stakeholder communication to others

## Answer

B

## Explanation

The correct answer is B. Balance customisation with sustainability. Prioritise: key stakeholders get tailored communication that matches their preferences. Standard channels: broader groups get consistent, efficient communication (status reports, newsletters). Flexibility: accommodate preferences where practical. This serves important relationships while maintaining manageable overhead.

Option A ignores stakeholder needs.

Option C stops necessary communication.

Option D may lose PM relationships with key stakeholders.
