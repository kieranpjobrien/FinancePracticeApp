# Question

In Plan Stakeholder Engagement, which tool helps compare current vs desired engagement levels?

## Options

- **A**: Power/interest grid.
- **B**: Stakeholder engagement assessment matrix.
- **C**: Salience model.
- **D**: Stakeholder cube.

## Answer

B

## Explanation

The stakeholder engagement assessment matrix lists each stakeholder's current engagement level (unaware/resistant/neutral/supportive/leading) against the desired level — gaps drive engagement actions. (A), (C), and (D) are classification tools for importance, not tracking engagement-level movement.
