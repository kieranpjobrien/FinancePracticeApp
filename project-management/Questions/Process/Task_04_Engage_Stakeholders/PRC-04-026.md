# Question

Stakeholders are complaining that they don't feel heard despite regular status meetings. They attend meetings but feel like they're only receiving information, not influencing decisions. What should the PM change?

## Options

- **A**: Provide more detailed status information
- **B**: Restructure engagement to include genuine consultation opportunities - ask for input before decisions, create feedback loops, and demonstrate how stakeholder input influences project direction
- **C**: Reduce meeting frequency
- **D**: Tell stakeholders they're being unreasonable

## Answer

B

## Explanation

The correct answer is B. "Being heard" requires genuine influence opportunities. Consultation: ask for input on decisions before they're made. Feedback loops: show how input was used (or explain why it couldn't be). Demonstrate influence: "Based on your feedback, we're adjusting X." This creates genuine two-way engagement rather than one-way broadcasting.

Option A provides more of the same one-way communication.

Option C reduces engagement without improving it.

Option D dismisses legitimate stakeholder needs.
