# Question

What does the stakeholder engagement assessment matrix compare?

## Options

- **A**: Each stakeholder's power and interest levels on a 2x2 grid.
- **B**: Each stakeholder's communication preferences and frequency requirements.
- **C**: Each stakeholder's current engagement level (unaware, resistant, neutral, supportive, leading) against their desired engagement level.
- **D**: The influence of each stakeholder on each project phase.

## Answer

C

## Explanation

The stakeholder engagement assessment matrix maps current vs desired engagement across five levels: unaware, resistant, neutral, supportive, leading — used to plan engagement actions that close the gap. (A) describes a power/interest grid, a different tool. (B) describes the communications management plan inputs. (D) describes phase-based influence analysis rather than the engagement assessment matrix.
