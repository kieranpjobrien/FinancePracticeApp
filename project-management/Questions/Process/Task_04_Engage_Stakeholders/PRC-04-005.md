# Question

What is the PRIMARY purpose of a stakeholder engagement assessment matrix?

## Options

- **A**: To identify which stakeholders have the most influence over the project
- **B**: To compare current engagement levels against desired engagement levels for each stakeholder
- **C**: To document stakeholder contact information and communication preferences
- **D**: To rank stakeholders by their financial contribution to the project

## Answer

B

## Explanation

The correct answer is B. A stakeholder engagement assessment matrix plots each stakeholder's current engagement level (Unaware, Resistant, Neutral, Supportive, Leading) against the desired level. This gap analysis helps the PM identify where engagement efforts should focus.

Option A (identify influence) describes the power/interest grid or salience model, not the engagement assessment matrix.

Option C (contact information) describes a stakeholder register, not the engagement assessment matrix.

Option D (financial contribution) is not a standard stakeholder analysis tool and conflates stakeholder engagement with funding.
