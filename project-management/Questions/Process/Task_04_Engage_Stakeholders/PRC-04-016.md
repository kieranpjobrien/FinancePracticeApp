# Question

During project execution, a stakeholder consistently requests more detail than the standard status report provides. Other stakeholders are satisfied with the current level. What should the PM do?

## Options

- **A**: Expand the standard report to match the stakeholder's detail preference.
- **B**: Direct the stakeholder to raise specific questions instead of expecting them in the report.
- **C**: Provide a supplementary briefing tailored to this stakeholder's needs, while keeping the standard report for others.
- **D**: Explain that the report format is fixed and cannot be changed for individual stakeholders.

## Answer

C

## Explanation

Tailoring communication is a core PMI principle. (C) serves one stakeholder's needs without imposing cost on the rest. (A) expanding the standard report creates overload for other stakeholders who are satisfied. (B) directing the stakeholder to file ad-hoc queries is reactive and inefficient when their need is predictable. (D) refusing to tailor contradicts PMI's emphasis on fit-for-purpose communication.
