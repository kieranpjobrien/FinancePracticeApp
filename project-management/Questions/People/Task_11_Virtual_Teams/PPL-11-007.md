# Question

What is a key challenge unique to virtual teams that project managers must address?

## Options

- **A**: Technical skill requirements
- **B**: Reduced informal communication and relationship building opportunities
- **C**: Budget constraints
- **D**: Scope management

## Answer

B

## Explanation

The correct answer is B. Virtual teams miss the informal interactions that happen naturally in co-located settings - hallway conversations, lunch discussions, spontaneous problem-solving. These interactions build relationships and share tacit knowledge. PMs must intentionally create opportunities for informal communication in virtual environments.

Option A (technical skills) applies to all teams, not uniquely to virtual teams.

Option C (budget) is a general project constraint, not specific to virtual teams.

Option D (scope management) applies equally to co-located and virtual teams.
