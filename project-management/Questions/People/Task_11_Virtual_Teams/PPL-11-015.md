# Question

What is the PRIMARY advantage of asynchronous communication for virtual teams spanning multiple time zones?

## Options

- **A**: It reduces the need for documentation
- **B**: It allows team members to contribute during their own working hours
- **C**: It eliminates the need for video conferencing
- **D**: It speeds up decision-making

## Answer

B

## Explanation

The correct answer is B. Asynchronous communication (email, recorded videos, shared documents, collaboration tools) allows team members to participate meaningfully without requiring everyone to be online simultaneously. This respects different time zones and personal schedules.

Option A is incorrect - async communication typically requires MORE documentation, not less, since context must be captured in writing.

Option C is incorrect - video conferencing remains valuable for relationship building and complex discussions; async doesn't eliminate it but complements it.

Option D is incorrect - async communication typically slows down decision-making compared to real-time discussion, which is a trade-off for inclusivity.
