# Question

What is a key challenge that virtual teams face compared to co-located teams?

## Options

- **A**: Inability to use project management software
- **B**: Reduced opportunities for spontaneous communication and relationship building
- **C**: Higher project costs
- **D**: Inability to use agile methods

## Answer

B

## Explanation

The correct answer is B. Co-located teams benefit from informal interactions - hallway conversations, lunch chats, overhearing relevant discussions - that build relationships and share information naturally. Virtual teams must intentionally create these opportunities since they don't happen spontaneously.

Option A is incorrect - virtual teams often make excellent use of project management software.

Option C is not inherently true - virtual teams can actually reduce costs (travel, facilities).

Option D is incorrect - agile methods work well for virtual teams with appropriate adaptations.
