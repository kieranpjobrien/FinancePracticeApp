# Question

A virtual team has members in Australia, India, and the UK. The team is professional in meetings but barely knows each other; nobody can name a personal detail about anyone outside their region. Cohesion is suffering and ad-hoc collaboration is rare. What should the PM do?

## Options

- **A**: Add a recurring 30-minute "team huddle" to the existing meeting series so the team has more contact time.
- **B**: Run a one-off in-person off-site so the team has the relationship base they currently lack.
- **C**: Create lightweight informal touchpoints — virtual coffee pairings, a non-work chat channel, short personal intros at the top of each ceremony.
- **D**: Reorganise the work so most collaboration happens within a region and cross-region work is minimised.

## Answer

C

## Explanation

PMI's virtual-team guidance and modern remote practice both stress that informal moments are what build cohesion, and they have to be designed in deliberately when there is no shared office. (C) does this at low cost. (A) adds more formal meeting time, which doesn't replicate the kind of contact people actually bond over. (B) is valuable when feasible but is a one-off and may not be possible in budget or time; it shouldn't be the first move. (D) reduces the symptom by reducing the collaboration the project depends on, which is the wrong trade-off.
