# Question

Which virtual team practice BEST supports building trust among team members who have never met in person?

## Options

- **A**: Limiting communication to essential project updates only
- **B**: Regular video calls that include time for informal conversation
- **C**: Using only written communication to ensure documentation
- **D**: Assigning individual tasks to minimise the need for collaboration

## Answer

B

## Explanation

The correct answer is B. Trust in virtual teams is built through repeated positive interactions that include both professional and personal connection. Video calls allow team members to see facial expressions and body language, making communication more human. Informal conversation builds personal relationships that support professional collaboration.

Option A reduces opportunities for trust-building interaction.

Option C removes the human elements that help build rapport and can lead to misinterpretation of tone.

Option D avoids the collaboration that builds trust through shared work experiences.
