# Question

Which communication approach is MOST important for managing virtual teams effectively?

## Options

- **A**: Using the most advanced collaboration technology available
- **B**: Over-communicating compared to co-located teams
- **C**: Relying primarily on written communication for documentation
- **D**: Minimising meetings to respect people's time

## Answer

B

## Explanation

The correct answer is B. Virtual teams lack the informal communication (hallway conversations, overhearing discussions, visual cues) that co-located teams have. Deliberate over-communication ensures information reaches everyone, reduces assumptions, and prevents isolation. This includes status updates, context sharing, and personal connection.

Option A - technology is a tool, not a strategy; the best technology poorly used is ineffective.

Option C - while documentation is important, it shouldn't replace richer forms of communication.

Option D - minimising meetings may increase isolation; the key is having effective meetings, not fewer meetings.
