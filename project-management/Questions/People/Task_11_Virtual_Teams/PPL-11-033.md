# Question

A virtual team uses multiple communication tools (email, instant messaging, video calls, project management software). Team members are frustrated because they can't find information and miss important updates. What should the PM do?

## Options

- **A**: Reduce to a single communication tool
- **B**: Establish clear guidelines for which tool to use for what purpose
- **C**: Hire an administrator to manage communications
- **D**: Create a daily email summary of all communications

## Answer

B

## Explanation

The correct answer is B. Multiple tools serve different purposes (async vs sync, formal vs informal, reference vs discussion). The problem isn't the tools but lack of clarity about their use. Guidelines like "project decisions in the project tool, quick questions in chat, complex discussions in video calls" reduce confusion.

Option A loses the benefits of purpose-appropriate tools.

Option C doesn't scale and creates dependency on one person.

Option D adds noise rather than reducing it and doesn't address the root cause.
