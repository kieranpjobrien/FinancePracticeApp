# Question

A project manager is building a team for a six-month project. They can either get experienced resources who are available for only four months, or less experienced resources available for the full duration. Which option is generally BETTER and why?

## Options

- **A**: Experienced resources - their productivity will offset the handover costs
- **B**: Less experienced resources - continuity is more valuable than experience
- **C**: It depends on factors like task complexity, knowledge transfer feasibility, and whether the experienced resources' exit point aligns with natural project phases
- **D**: Split the team between experienced and less experienced resources

## Answer

C

## Explanation

The correct answer is C. This decision depends on context. For simple work, less experienced continuous resources may be better. For complex work requiring specialized expertise, experienced resources might deliver more value despite handover. If the four-month point aligns with a natural phase boundary, handover is easier. No universal answer exists.

Option A assumes productivity always offsets handover, which isn't true for all projects.

Option B assumes continuity always wins, which isn't true for complex specialized work.

Option D may create coordination overhead without clearly addressing the core trade-off.
