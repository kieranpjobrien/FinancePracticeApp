# Question

What is the primary purpose of a team charter?

## Options

- **A**: To assign individual performance ratings
- **B**: To document how the team will work together, including values, agreements, and decision-making processes
- **C**: To list all project deliverables
- **D**: To track project budget

## Answer

B

## Explanation

The correct answer is B. A team charter (or working agreement) establishes shared expectations for how the team operates. It typically includes communication norms, meeting protocols, decision-making processes, and shared values. This creates clarity and reduces friction.

Option A relates to performance management, not team chartering.

Option C describes scope documentation.

Option D describes financial planning.
