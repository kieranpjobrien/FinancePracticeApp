# Question

During team formation, one member dominates discussions while others remain quiet. The quiet members have valuable perspectives that aren't being heard. How should the PM address this during team building?

## Options

- **A**: Let the dynamic sort itself out naturally
- **B**: Privately ask the dominant member to step back
- **C**: Use facilitation techniques to ensure all voices are heard - round-robins, written input before discussion, direct invitations to quiet members
- **D**: Remove the dominant personality from the team

## Answer

C

## Explanation

The correct answer is C. Facilitation techniques create space for different communication styles. Round-robin gives everyone a turn. Written input before discussion lets introverts compose thoughts. Direct invitations ("What do you think?") bring in quiet voices. This is a process intervention, not a personality problem.

Option A allows valuable perspectives to be lost.

Option B makes it personal rather than structural.

Option D is extreme when facilitation can solve the issue.
