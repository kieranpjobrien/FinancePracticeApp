# Question

A functional manager is reluctant to release their best performer to a new project team, citing critical departmental needs. The PM needs this person's unique expertise. What is the best approach?

## Options

- **A**: Escalate immediately to the PMO
- **B**: Accept a less qualified team member
- **C**: Work with the functional manager to find a solution that addresses both needs, such as part-time allocation, knowledge transfer to a backup, or phased transition
- **D**: Start the project without the required expertise

## Answer

C

## Explanation

The correct answer is C. Matrix organisations require negotiation and creative problem-solving. Understanding the functional manager's constraints enables collaborative solutions. Part-time allocation, overlapping transitions, or developing internal backups can satisfy both parties. Building relationships with functional managers is essential for future resource negotiations.

Option A damages the relationship without attempting resolution.

Option B may compromise project quality.

Option D creates significant project risk.
