# Question

A PM discovers that a team member was assigned to the project without the necessary security clearance required for some project work. The clearance process takes 3 months. What should the PM do?

## Options

- **A**: Allow the team member to access secure materials anyway
- **B**: Remove the team member from the project entirely
- **C**: Initiate the clearance process immediately, assign the team member to non-sensitive work in the interim, and adjust the project plan to account for the constraint
- **D**: Delay all secure work until clearance is obtained

## Answer

C

## Explanation

The correct answer is C. This balances compliance, team utilisation, and project progress. Starting clearance immediately minimises delay. The team member can contribute to non-sensitive work while waiting, maintaining their integration with the team. Planning adjustments account for the constraint realistically.

Option A violates security requirements and creates serious risk.

Option B loses a resource that will eventually be valuable.

Option D unnecessarily delays the project when parallel work is possible.
