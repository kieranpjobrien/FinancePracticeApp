# Question

When should training needs be identified for a project?

## Options

- **A**: Only when team members fail to perform tasks correctly
- **B**: During project planning when resource requirements are assessed
- **C**: After the project is complete as part of lessons learned
- **D**: Training is only an HR responsibility, not a project management concern

## Answer

B

## Explanation

The correct answer is B. Training needs should be identified proactively during planning when the project manager assesses what skills are required versus what skills are available. This allows time to address gaps before they impact project execution. Waiting until problems occur is reactive and costly.

Option A is reactive - addressing training only after failures means accepting preventable problems. Proactive identification is more effective.

Option C is too late to help the current project. Lessons learned may identify training for future projects, but the current project needs training identified during planning.

Option D incorrectly separates training from project management. While HR may administer training programs, the PM is responsible for ensuring the project team has needed skills. This is part of resource management.
