# Question

The project's training budget will not cover external courses, but the team genuinely needs new skills. What should the PM do FIRST?

## Options

- **A**: Defer the training need to the next phase when additional training budget may become available.
- **B**: Drop the project features that depend on the new skills so formal training is no longer needed.
- **C**: Run targeted internal sessions to test whether the team can build the required skills without external input.
- **D**: Map the skill gap and source it from a mix of low-cost options: internal experts, free materials, pairing.

## Answer

D

## Explanation

When formal training is not affordable, the PM should still address the gap through a portfolio of low-cost mechanisms, which is what (D) does. (A) leaves the gap open through delivery and is rarely workable on a live project. (B) sacrifices project value to avoid solving the training problem. (C) jumps to one mechanism (internal sessions) without considering the full set of options.
