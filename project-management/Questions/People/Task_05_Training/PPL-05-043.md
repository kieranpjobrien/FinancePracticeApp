# Question

What role does written documentation play in supporting team training?

## Options

- **A**: It replaces the need for live training in most cases because content is searchable.
- **B**: It is mainly a compliance artefact for audit and accreditation purposes.
- **C**: It provides reference material that supports formal training, enables self-service learning, and preserves knowledge for future team members.
- **D**: It is a legacy practice that has been superseded by short-form video.

## Answer

C

## Explanation

PMI sees documentation as one of several complementary learning aids, not as a substitute or a compliance artefact. (C) reflects how it functions in practice. (A) overstates its role and ignores limits to self-paced learning. (B) reduces it to compliance. (D) is a fashion claim, not a project planning rationale.
