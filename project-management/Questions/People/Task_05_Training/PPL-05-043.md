# Question

What role does documentation play in team training?

## Options

- **A**: Documentation replaces the need for any other training
- **B**: Documentation is irrelevant in the age of online search
- **C**: Documentation provides reference material that supports training, enables self-service learning, and preserves knowledge for future team members
- **D**: Documentation is only for compliance purposes

## Answer

C

## Explanation

The correct answer is C. Documentation is a valuable training support tool. It provides reference material during and after formal training. It enables self-service learning for those who prefer reading. It preserves knowledge for future team members who weren't present for original training. Good documentation complements other training methods.

Option A overestimates documentation's ability to fully train people.

Option B dismisses documentation's role in contextual, project-specific knowledge.

Option D views documentation narrowly when it has broader training value.
