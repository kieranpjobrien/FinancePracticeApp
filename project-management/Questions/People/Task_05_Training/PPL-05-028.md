# Question

The end users who will operate the system after deployment have been largely uninvolved during the project. Deployment is approaching, and they'll need training. What's the best approach?

## Options

- **A**: Provide documentation and assume they'll figure it out
- **B**: Delay deployment until users are fully trained
- **C**: Develop a comprehensive training program with varied methods (classroom, hands-on, documentation), involve users in UAT for early exposure, and provide post-deployment support
- **D**: Let the IT help desk handle all user questions after deployment

## Answer

C

## Explanation

The correct answer is C. End user training is critical for adoption and realisation of project benefits. Use varied methods to accommodate different learning styles. Involve users in UAT - it provides early exposure and builds familiarity. Comprehensive documentation supports ongoing reference. Plan post-deployment support for questions and issues that emerge with real use.

Option A assumes users can learn a new system from documentation alone.

Option B may unnecessarily delay value realisation.

Option D overloads help desk and doesn't address the fundamental training need.
