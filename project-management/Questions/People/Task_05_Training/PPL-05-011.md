# Question

What should a training needs assessment identify?

## Options

- **A**: Only technical skill gaps
- **B**: The gap between current capabilities and required capabilities for project success
- **C**: Who should be fired for lack of skills
- **D**: External training vendors only

## Answer

B

## Explanation

The correct answer is B. A training needs assessment compares current capabilities (what people know/can do now) against required capabilities (what they need to know/do for the project). The gap identifies training needs. This includes technical skills, soft skills, tool proficiency, process knowledge, and domain understanding.

Option A is too narrow - training needs include more than technical skills.

Option C misunderstands the purpose - it's about development, not termination.

Option D is about training delivery, not needs assessment.
