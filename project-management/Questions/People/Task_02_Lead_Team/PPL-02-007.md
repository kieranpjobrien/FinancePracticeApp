# Question

A project manager leads a cross-functional team with members from different departments who have competing priorities. Each team member's functional manager evaluates their performance based on departmental goals, not project success. Team members consistently prioritize departmental work over project tasks. What is the MOST effective approach?

## Options

- **A**: Escalate to the project sponsor to mandate project priority over departmental work
- **B**: Work with functional managers to align on shared goals and ensure project contributions are recognized in performance evaluations
- **C**: Create detailed task assignments with deadlines to hold team members accountable
- **D**: Request dedicated resources who report only to the project manager

## Answer

B

## Explanation

The correct answer is B. This addresses the root cause - the misaligned incentive structure. When functional managers evaluate based on departmental goals only, team members rationally prioritize those. By engaging functional managers to include project performance in evaluations and establishing shared goals, the PM creates aligned incentives. This is systems thinking - changing the structure that drives behavior.

Option A uses authority but doesn't change the incentive structure. Even with mandated priority, team members still face conflicting performance evaluations. Executive mandates without structural change create compliance without commitment.

Option C (detailed assignments) treats symptoms, not causes. More detailed tracking doesn't resolve competing priorities - it just documents the conflict.

Option D may not be feasible in a matrix organization and doesn't solve the underlying organizational challenge. Even if possible, it avoids rather than solves the cross-functional collaboration issue that many projects face.
