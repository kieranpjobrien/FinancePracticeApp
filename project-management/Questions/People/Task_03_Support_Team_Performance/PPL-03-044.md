# Question

The team completed a challenging project successfully, but everyone is exhausted. Another demanding project is starting immediately. How should the PM support sustainable performance?

## Options

- **A**: Expect the same intensity since the last project was successful
- **B**: Advocate for a transition period, acknowledge the team's effort, ensure proper handoff of the previous project, and set realistic initial expectations for the new project
- **C**: Replace tired team members with fresh ones
- **D**: Start the new project and hope the team recovers while working

## Answer

B

## Explanation

The correct answer is B. Sustainable performance requires recovery between intense efforts. Advocate for the team: request transition time, reduced initial commitments, or staggered start. Acknowledge their effort explicitly. Ensure clean handoff of the previous project to avoid lingering obligations. Set realistic early expectations to allow rebuilding of energy and focus.

Option A leads to burnout and declining performance.

Option C loses project knowledge and doesn't respect the team's contributions.

Option D ignores team needs and risks quality and health issues.
