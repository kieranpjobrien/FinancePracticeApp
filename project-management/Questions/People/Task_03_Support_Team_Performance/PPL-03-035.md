# Question

What is a Work Breakdown Structure (WBS)?

## Options

- **A**: A list of team members and their assignments
- **B**: A hierarchical decomposition of project scope into manageable work packages
- **C**: A chart showing project timeline
- **D**: A document listing project risks

## Answer

B

## Explanation

The correct answer is B. The WBS is a deliverable-oriented hierarchical decomposition of project scope. It breaks the total scope into smaller, manageable pieces called work packages. This supports team performance by making work tangible, estimable, and assignable. It ensures all required work is identified and nothing is missed.

Option A describes a responsibility assignment matrix, not WBS.

Option C describes a schedule or Gantt chart.

Option D describes a risk register.
