# Question

Which of the following BEST describes a team's Definition of Done (DoD)?

## Options

- **A**: A shared team agreement on criteria every increment must meet to be complete.
- **B**: A list of features the team plans to complete in the current sprint cycle.
- **C**: The product-level acceptance criteria as defined by the external customer.
- **D**: A checklist used only by the QA function to verify deliverables before release.

## Answer

A

## Explanation

The DoD is a team-owned quality bar that applies to every work item — coding standards, tests, documentation, etc. (B) describes the sprint backlog. (C) describes story-level acceptance criteria. (D) wrongly limits DoD ownership to QA when it belongs to the whole team.
