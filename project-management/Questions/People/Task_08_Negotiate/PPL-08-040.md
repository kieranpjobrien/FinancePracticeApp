# Question

What is "anchoring" in negotiation?

## Options

- **A**: Establishing a strong physical presence
- **B**: Making the first offer to set a reference point that influences subsequent negotiation
- **C**: Refusing to move from your position
- **D**: Building trust through shared experiences

## Answer

B

## Explanation

The correct answer is B. Anchoring is a cognitive bias where initial information disproportionately influences subsequent judgments. In negotiation, the first offer sets an anchor that shapes expectations. Research shows first offers significantly affect final outcomes. Understanding anchoring helps you both set favourable anchors and recognise when you're being anchored.

Option A describes body language.

Option C describes positional stubbornness.

Option D describes rapport building.
