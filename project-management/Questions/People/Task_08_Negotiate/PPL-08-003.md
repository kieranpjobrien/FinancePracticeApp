# Question

A project manager is negotiating resource allocation with a functional manager who is reluctant to release a key team member full-time. The functional manager has their own department goals to meet. What negotiation approach would be MOST effective?

## Options

- **A**: Escalate to senior management to mandate the resource allocation
- **B**: Accept whatever allocation the functional manager offers
- **C**: Explore options that address both parties' needs, such as phased allocation, specific deliverable commitments, or backfill support
- **D**: Offer to reduce the project scope to need fewer resources

## Answer

C

## Explanation

The correct answer is C. Interest-based negotiation seeks solutions that address both parties' underlying needs. The functional manager needs to meet department goals; the project needs the resource. Creative options (phased allocation, specific commitments, backfill) can satisfy both. This approach builds relationships and often finds better solutions than positional bargaining.

Option A uses authority but damages the relationship and doesn't address the functional manager's legitimate concerns.

Option B yields without exploring options that might serve both parties better.

Option D sacrifices project value without exploring whether better solutions exist.
