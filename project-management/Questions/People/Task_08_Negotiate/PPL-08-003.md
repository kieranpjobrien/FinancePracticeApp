# Question

A project manager needs a senior analyst full-time for the next phase, but the analyst's functional manager has departmental commitments that depend on her. What should the PM do?

## Options

- **A**: Take the resource issue to the PMO director so she can decide allocation.
- **B**: Meet with the functional manager, share the project's needs, and explore options like phased allocation, named deliverables, or a backfill arrangement.
- **C**: Accept whatever percentage the functional manager offers to preserve the working relationship.
- **D**: Reduce the phase scope so the analyst is no longer required full-time.

## Answer

B

## Explanation

PMI's interest-based approach starts with a peer conversation that surfaces both parties' real needs and looks for creative options that satisfy them. (A) escalates before negotiating, which damages the relationship and skips a step the PM owns. (C) caves without testing whether better arrangements exist, and may set a precedent. (D) sacrifices project value before any conversation has been had.
