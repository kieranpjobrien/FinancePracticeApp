# Question

Which statement BEST describes an empowered project team?

## Options

- **A**: The team operates independently of the project manager and reports outcomes only at milestones.
- **B**: The team has authority to make decisions within agreed boundaries and is accountable for the results.
- **C**: The team has unrestricted freedom to set its own scope, schedule, and budget.
- **D**: The team escalates significant decisions to the project manager but executes routine work itself.

## Answer

B

## Explanation

Empowerment pairs decision authority within defined boundaries with accountability for outcomes, which is what (B) describes. (A) confuses empowerment with isolation; empowered teams still work closely with the PM and stakeholders. (C) describes unrestricted autonomy, which conflicts with project constraints and governance. (D) describes a delegated-execution model rather than true empowerment, because the meaningful decisions still belong to the PM.
