# Question

A project team is blocked waiting for a vendor to deliver a component. The vendor is unresponsive to the PM's communications. What escalation path should the PM follow?

## Options

- **A**: Wait for the vendor to respond
- **B**: Immediately terminate the vendor contract
- **C**: Use the contractual communication channels, escalate to vendor management, involve your own procurement/legal team if needed, and document all interactions
- **D**: Have the team build the component themselves

## Answer

C

## Explanation

The correct answer is C. Vendor issues require structured escalation using contractual mechanisms. Most contracts specify communication protocols, escalation contacts, and remedies for non-performance. Involving procurement/legal ensures proper documentation and options. This protects the project while maintaining professional relationships.

Option A accepts delay without action.

Option B may be premature and costly.

Option D may not be feasible, cost-effective, or contractually permitted.
