# Question

Critical hardware for your smart-building project has been stuck in customs for ten days because a classification code on the import paperwork is disputed. The installation window starts in three weeks and later slots are fully booked. What should the project manager do?

## Options

- **A**: Re-baseline the installation plan for three weeks later and accept the knock-on delay, given customs disputes can be unpredictable and hard to forecast reliably.
- **B**: Offer an informal side payment through the local agent to speed the clearance, on the basis that logistics sometimes benefits from this kind of expediting.
- **C**: Engage a licensed customs broker and the supplier's trade team to challenge the classification with correct documentation and surface the delivery impact with procurement.
- **D**: Request that the supplier re-ship a duplicate set of hardware by air freight from the origin country, regardless of cost, so the original installation window is preserved.

## Answer

C

## Explanation

PMI treats a customs dispute as a documentation and process issue handled through legitimate specialists, with delivery impact made visible to governance. (C) is that approach. (A) accepts a preventable slip. (B) is an ethics violation that also exposes the project to serious legal risk. (D) is a disproportionate duplicate cost before proper resolution has been attempted.
