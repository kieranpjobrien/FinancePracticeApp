# Question

What is the difference between an impediment and a blocker?

## Options

- **A**: They are exactly the same thing
- **B**: An impediment slows progress while a blocker completely stops work from continuing
- **C**: A blocker is less severe than an impediment
- **D**: Impediments are technical; blockers are organisational

## Answer

B

## Explanation

The correct answer is B. While often used interchangeably, the distinction is useful. Impediments slow progress (inefficient processes, missing information, skill gaps). Blockers prevent any progress (system down, critical approval missing, dependency not delivered). Blockers typically require more urgent action as work has stopped completely.

Option A misses the useful severity distinction.

Option C has the severity reversed.

Option D incorrectly categorises by type rather than impact.
