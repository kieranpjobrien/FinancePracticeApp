# Question

A stakeholder asks the project manager to help them prepare for a board presentation about the project. This is outside the project manager's formal duties. What should the project manager do?

## Options

- **A**: Help by reviewing the draft, sharing relevant project data, and offering coaching on key messages.
- **B**: Decline because preparing executive presentations is not part of the project manager role.
- **C**: Take responsibility for the entire presentation and deliver a finished deck for the stakeholder.
- **D**: Refer the stakeholder to the corporate communications team for help with the deck.

## Answer

A

## Explanation

Supporting stakeholders strengthens both relationships and the project's representation. (A) provides meaningful help while keeping the stakeholder accountable for their own delivery. (B) is unhelpful and damages the relationship. (C) does the work for them and cedes the project's narrative. (D) outsources a moment where the project manager can add direct value.
