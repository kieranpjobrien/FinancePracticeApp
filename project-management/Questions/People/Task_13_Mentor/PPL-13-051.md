# Question

A mentee arrives at each session with a specific problem and wants the mentor to tell them what to do. The mentor has started answering directly to be helpful, but now notices the mentee is not building decision-making capability and returns each week with similar problems. What should the mentor do?

## Options

- **A**: Continue giving direct answers, since the mentee is getting what they asked for and clearly finds the advice useful in their day-to-day work.
- **B**: Refuse to give direct answers any more and redirect every question to a coaching-style inquiry so the mentee is pushed to reason things through.
- **C**: Shift toward coaching questions as the default while still offering direct input when genuinely specialised knowledge is required rather than judgement.
- **D**: Raise the concern with the mentee's line manager so the line manager can set expectations about independent decision-making in the mentee's day job.

## Answer

C

## Explanation

PMI's mentoring balance is coaching-first with expert input where it adds genuine value — this builds judgement without withholding specialised knowledge the mentee reasonably needs. (A) optimises for comfort and reinforces dependence. (B) is too rigid — sometimes a direct answer genuinely is what serves the mentee best. (D) goes behind the mentee's back to fix a mentoring pattern.
