# Question

A functional manager who provides developers to the project regularly pulls them away mid-sprint for unrelated departmental work. The project manager has explained agile principles to this manager more than once with no change. What should the project manager try NEXT?

## Options

- **A**: Accept the disruptions because functional managers ultimately control how their resources are used.
- **B**: Escalate to the project sponsor and ask them to enforce the team's agreed sprint commitments.
- **C**: Invite the functional manager to attend a sprint review and retrospective to see the impact.
- **D**: Request replacement developers who report to a different, more cooperative functional manager instead.

## Answer

C

## Explanation

When abstract explanation has not worked, experiential learning often does. (C) invites the manager into the team's rhythm so they can see the cost of mid-sprint disruption directly. (A) accepts a dysfunctional pattern. (B) escalates before relationship-building options have been exhausted, and may harden the manager's resistance. (D) avoids the underlying organisational issue and is rarely feasible at short notice.
