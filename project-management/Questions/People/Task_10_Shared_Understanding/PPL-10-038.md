# Question

A PM notices that despite extensive documentation and meetings, the same questions keep being asked by different stakeholders. What does this pattern suggest about how to improve shared understanding?

## Options

- **A**: The stakeholders aren't paying attention
- **B**: The documentation needs to be longer
- **C**: The current communication methods aren't creating lasting understanding; consider more interactive, visual, or contextual approaches and make key information more accessible
- **D**: Reduce the number of stakeholders

## Answer

C

## Explanation

The correct answer is C. Repeated questions indicate communication methods aren't effective for this audience. Perhaps information is buried in long documents, presented without sufficient context, or communicated once without reinforcement. Interactive approaches, visual communication, just-in-time information delivery, or more accessible formats might work better.

Option A blames stakeholders without examining communication effectiveness.

Option B more of the same approach unlikely to succeed.

Option D doesn't address the communication problem.
