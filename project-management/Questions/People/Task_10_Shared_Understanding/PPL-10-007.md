# Question

What is a project vision statement?

## Options

- **A**: A detailed schedule of project milestones
- **B**: A concise description of the project's purpose, goals, and desired end state that creates shared understanding
- **C**: A list of project risks
- **D**: The project budget summary

## Answer

B

## Explanation

The correct answer is B. A vision statement describes what success looks like - the purpose of the project, the value it delivers, and the desired end state. It provides a north star that guides decisions and creates shared understanding across stakeholders. A good vision is inspiring, memorable, and helps people understand "why" the project matters.

Option A describes a schedule or milestone plan.

Option C describes a risk register.

Option D describes budget documentation.
