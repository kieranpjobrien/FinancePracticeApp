# Question

What is a project vision statement?

## Options

- **A**: A concise description of the project's purpose, intended outcomes, and what success looks like.
- **B**: A high-level timeline showing the major project milestones and dependencies.
- **C**: The list of project objectives translated into measurable acceptance criteria.
- **D**: A formal authorisation that establishes the project manager's authority to use resources.

## Answer

A

## Explanation

A vision statement is a short, inspirational description of why the project exists and what success looks like; it serves as a north star that aligns decisions across the team and stakeholders. (B) describes a milestone schedule, which sits inside the schedule baseline. (C) describes objectives or acceptance criteria, which derive from the vision but are not the same thing. (D) describes the project charter — related to the vision but performing a different function (formal authorisation and PM authority).
