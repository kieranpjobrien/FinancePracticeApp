# Question

What is the primary purpose of a project vision statement?

## Options

- **A**: To list all project deliverables
- **B**: To create shared understanding of the project's purpose, goals, and expected value
- **C**: To assign team responsibilities
- **D**: To document the project budget

## Answer

B

## Explanation

The correct answer is B. A vision statement articulates why the project exists, what it aims to achieve, and what value it delivers. This creates alignment and shared understanding among all stakeholders about the project's fundamental purpose. It guides decision-making and helps team members connect their work to meaningful outcomes.

Option A relates to scope documentation.

Option C relates to responsibility assignment.

Option D relates to financial planning.
