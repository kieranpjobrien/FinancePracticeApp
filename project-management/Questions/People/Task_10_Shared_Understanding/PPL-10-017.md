# Question

What is the primary purpose of a project vision statement?

## Options

- **A**: To provide an authoritative list of every deliverable the project will produce.
- **B**: To create shared understanding of the project's purpose, intent, and the value it will deliver.
- **C**: To document the formal RACI assignments for project roles and responsibilities.
- **D**: To establish the approved cost and schedule baselines for the project.

## Answer

B

## Explanation

A vision statement aligns stakeholders on why the project exists and what good looks like; it's an inspiration and decision filter, not a control document. (A) describes a deliverables list or scope statement, which derives from but is distinct from the vision. (C) describes a responsibility assignment matrix. (D) describes the cost and schedule baselines. Each of those exists for a different purpose; only the vision statement's purpose is to create shared understanding of intent.
