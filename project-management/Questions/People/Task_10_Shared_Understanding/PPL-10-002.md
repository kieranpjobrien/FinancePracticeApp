# Question

What is the PRIMARY purpose of a project kickoff meeting?

## Options

- **A**: To begin project execution work
- **B**: To establish shared understanding of project objectives, approach, roles, and expectations among stakeholders
- **C**: To present the project schedule to management
- **D**: To assign all project tasks to team members

## Answer

B

## Explanation

The correct answer is B. The kickoff meeting's primary purpose is building shared understanding among stakeholders about what the project will accomplish, how it will work, who is responsible for what, and what success looks like. This alignment at the start prevents misunderstandings and conflicts later.

Option A confuses kickoff with execution start. Kickoff is about alignment, not doing project work.

Option C is too narrow - presenting the schedule may be part of kickoff, but the broader purpose is shared understanding across many dimensions.

Option D describes detailed planning activity, not the kickoff purpose. Detailed task assignments happen during project execution.
