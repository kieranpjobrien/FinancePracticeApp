# Question

A project involves stakeholders from different countries with different communication styles. Direct stakeholders interpret questions as challenges; indirect stakeholders interpret silence as agreement. Misunderstandings are frequent. What approach would BEST build shared understanding?

## Options

- **A**: Establish a single communication standard that all must follow
- **B**: Develop explicit communication norms acknowledging different styles, use multiple confirmation methods, and create space for different approaches
- **C**: Have all communications go through the project manager to filter for style
- **D**: Conduct cultural awareness training for all stakeholders

## Answer

B

## Explanation

The correct answer is B. Cross-cultural communication requires acknowledging differences, not eliminating them. Explicit norms help people understand how to interpret each other. Multiple confirmation methods (written summaries, verbal check-ins) catch misunderstandings. Creating space for different approaches respects diversity while ensuring understanding.

Option A imposes one style, which may alienate stakeholders whose natural style differs. It's also hard to enforce.

Option C creates a bottleneck and removes direct stakeholder interaction, which can build relationships.

Option D might help long-term but doesn't immediately address the communication issues.
