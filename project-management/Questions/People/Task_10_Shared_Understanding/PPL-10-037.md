# Question

What does the communication technique "check for understanding" involve?

## Options

- **A**: Testing participants on material covered
- **B**: Asking questions or using techniques to verify that your message has been received and interpreted correctly
- **C**: Checking that everyone is paying attention
- **D**: Reviewing written documentation

## Answer

B

## Explanation

The correct answer is B. Checking for understanding validates that communication has succeeded. This might involve asking "What questions do you have?" (better than "Any questions?"), requesting paraphrasing, or having recipients explain how they'll apply the information. It's proactive verification rather than assuming communication occurred.

Option A is more formal testing than verification.

Option C focuses on attention, not understanding.

Option D is documentation review, not real-time verification.
