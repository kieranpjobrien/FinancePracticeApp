# Question

A remote team has been working together for months but team members still frequently misunderstand each other's messages. Written communication seems to cause more confusion than clarity. What should the PM do to improve shared understanding?

## Options

- **A**: Write more detailed messages
- **B**: Increase synchronous communication (video calls, real-time chat) to reduce misinterpretation and build relationship context
- **C**: Document every conversation in detail
- **D**: Accept miscommunication as inevitable in remote work

## Answer

B

## Explanation

The correct answer is B. Written communication lacks tone, facial expressions, and immediate clarification. Synchronous communication (video preferred) provides richer context and allows real-time clarification. Building stronger relationships helps people interpret messages more charitably. Balance asynchronous efficiency with synchronous relationship-building.

Option A may just create more content to misinterpret.

Option C adds overhead without improving understanding.

Option D accepts avoidable dysfunction.
