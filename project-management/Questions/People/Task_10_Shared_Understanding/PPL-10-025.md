# Question

What is a user story mapping session designed to achieve?

## Options

- **A**: Assign stories to specific developers
- **B**: Build shared understanding of the user journey and how features fit together
- **C**: Estimate the project budget
- **D**: Create detailed technical specifications

## Answer

B

## Explanation

The correct answer is B. User story mapping creates a visual representation of the user journey with features organised along it. This builds shared understanding of how individual stories contribute to user value, how features relate to each other, and what constitutes a minimal viable product. It's a collaborative activity that aligns the whole team.

Option A relates to work assignment.

Option C relates to financial planning.

Option D relates to technical design.
