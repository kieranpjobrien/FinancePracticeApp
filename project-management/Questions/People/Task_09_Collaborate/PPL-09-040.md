# Question

A major project sponsor has been replaced mid-project. The new sponsor has different priorities than the original and questions existing decisions. How should the PM navigate this transition?

## Options

- **A**: Insist that all previous decisions remain unchanged
- **B**: Immediately change everything to match the new sponsor's preferences
- **C**: Brief the new sponsor thoroughly, understand their priorities, evaluate which previous decisions might need revision, and facilitate appropriate governance for significant changes
- **D**: Escalate the sponsor's questioning as interference

## Answer

C

## Explanation

The correct answer is C. Sponsor transitions require careful navigation. Thorough briefing builds understanding of project context and rationale for decisions. Understanding new priorities enables alignment. Some decisions may legitimately need revision given new direction—this isn't failure but appropriate adaptation. Proper governance (change control) ensures changes are deliberate.

Option A ignores legitimate authority of the new sponsor.

Option B may undo valid decisions and create instability.

Option D frames appropriate oversight as interference.
