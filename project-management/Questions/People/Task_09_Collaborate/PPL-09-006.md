# Question

A cross-departmental project involves groups with a history of poor cooperation; past projects have failed on this exact interaction. What should the project manager do to set collaboration up for success?

## Options

- **A**: Structure the work so each department's scope is largely independent and only integrate at the end.
- **B**: Address each non-cooperation incident by escalating promptly to senior management for visibility.
- **C**: Co-create shared goals, run cross-functional working sessions, and visibly celebrate joint wins to build new norms.
- **D**: Ask the sponsor to set mandatory cooperation expectations before kickoff.

## Answer

C

## Explanation

Breaking through ingrained interdepartmental friction requires the PM to build new relationships intentionally — shared goals align incentives, joint sessions build trust, and visible wins reset the narrative. (A) papers over the collaboration problem with a structure that creates severe late-stage integration risk. (B) treats every incident as an escalation, which positions the PM as adversarial and punishes any attempt to engage. (D) relies on top-down mandate to fix what is fundamentally a relationship problem at the working level.
