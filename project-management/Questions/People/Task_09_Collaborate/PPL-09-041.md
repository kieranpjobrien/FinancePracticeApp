# Question

A PM discovers that a stakeholder has been sharing confidential project information outside the authorised communication channels. How should the PM address this?

## Options

- **A**: Ignore it if no obvious harm has occurred
- **B**: Immediately restrict all the stakeholder's access
- **C**: Privately discuss the breach with the stakeholder, understand if it was inadvertent or intentional, reinforce confidentiality requirements, and escalate to appropriate authorities if necessary
- **D**: Publicly address the breach in a stakeholder meeting

## Answer

C

## Explanation

The correct answer is C. Confidentiality breaches require direct but measured response. Private discussion maintains dignity while addressing the issue. Understanding circumstances (accident vs. intentional) informs appropriate response. Reinforcing requirements provides clarity. Escalation is warranted for serious or repeated breaches but should follow initial private discussion.

Option A ignores a governance concern.

Option B may be disproportionate for inadvertent breaches.

Option D embarrasses the stakeholder without adding value.
