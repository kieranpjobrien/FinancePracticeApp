# Question

How often should stakeholder analysis be performed on a project?

## Options

- **A**: Only at project initiation
- **B**: At regular intervals throughout the project and when significant changes occur
- **C**: Only when problems with stakeholders arise
- **D**: Only at project closure

## Answer

B

## Explanation

The correct answer is B. Stakeholder dynamics change throughout a project. New stakeholders emerge, existing ones change roles or interests, engagement levels shift. Regular review ensures the stakeholder register remains current. Significant changes (scope changes, organisational restructuring, phase transitions) warrant specific stakeholder analysis updates.

Option A misses changing dynamics.

Option C is reactive rather than proactive.

Option D is too late to inform project execution.
