# Question

How frequently should stakeholder analysis be revisited during a project?

## Options

- **A**: Only at project initiation, since the stakeholder map is set up front.
- **B**: Only at project closure as part of lessons learned.
- **C**: Regularly throughout the project and whenever significant changes occur.
- **D**: Only when a problem with a stakeholder arises.

## Answer

C

## Explanation

PMI treats the stakeholder register and engagement assessment as living documents that must be revisited regularly and at any meaningful change (org restructure, scope change, phase transition). (A) misses the dynamic nature of stakeholder engagement. (B) is too late to do anything with the insight. (D) is reactive and forfeits the chance to engage stakeholders before issues arise.
