# Question

End users have been reluctant to participate in user acceptance testing, claiming they're too busy with their regular jobs. UAT is critical for project success. How should the PM improve end user collaboration?

## Options

- **A**: Skip UAT and proceed to deployment
- **B**: Force users to participate through their managers
- **C**: Work with user managers to allocate dedicated time, make testing sessions efficient and well-organised, show appreciation for participation, and communicate how their input improves the final product
- **D**: Have the project team conduct UAT instead

## Answer

C

## Explanation

The correct answer is C. End user participation requires making it feasible and valuable. Working with managers creates formal allocation. Well-organised sessions respect busy schedules. Showing appreciation and demonstrating impact builds willingness. When users see their input influencing outcomes, engagement typically improves.

Option A risks deploying a product that doesn't meet user needs.

Option B may create resentment and poor-quality feedback.

Option D defeats the purpose of user acceptance testing.
