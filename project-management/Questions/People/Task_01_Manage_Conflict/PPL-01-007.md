# Question

A project manager finds that the CFO and the CTO have fundamentally different visions: the CFO wants aggressive automation for cost savings; the CTO wants a scalable platform for future innovation. Both have equal authority and the charter does not prioritise between them. What should the PM do FIRST?

## Options

- **A**: Run a joint working session with both stakeholders to define shared success criteria and test whether both objectives can be met.
- **B**: Compromise by designing a solution that partially addresses each vision.
- **C**: Ask the project sponsor to decide which objective takes priority.
- **D**: Proceed with the CFO's direction since cost reduction delivers quicker measurable value.

## Answer

A

## Explanation

(A) is collaborating — before compromising or escalating, the PM should see whether the two visions can align on shared criteria; automation and scalability are often complementary. (B) locks in a lose-lose compromise without exploring win-win. (C) escalates prematurely, putting the sponsor between two peers the PM has not yet tried to reconcile. (D) picks a winner on the PM's own judgment, which is not the PM's call.
