# Question

The Regulated Asset Base (RAB) in utility regulation represents:

## Options

- **A**: The company's market capitalization
- **B**: The value of assets on which the utility earns a regulated return
- **C**: The maximum debt the utility can hold

## Answer

B

## Explanation

RAB concept:

**Definition:**
- Book value of regulated assets
- Base for allowed returns
- Grows with investment
- Key to utility valuation

**Return calculation:**
Return = RAB × Allowed Rate of Return

**RAB components:**
- Original capital investment
- Plus: New capex additions
- Less: Depreciation
- Inflation adjustments (some regimes)

**Allowed return:**
- Set by regulator
- Cost of capital based
- Periodic reviews
- Equity + debt weighted

**Why RAB matters:**
- Drives profitability
- Investment incentive
- Regulatory compact
- Valuation foundation

**Growth in RAB:**
- Through new investment
- Capex programs
- Network expansion
- Value creation for equity

**Common mistakes:**
- Option A: Regulatory value, not market value
- Option C: Separate from debt limits
