# Question

Flash loans in DeFi allow users to:

## Options

- **A**: Borrow with traditional credit checks and collateral
- **B**: Borrow large amounts without collateral if repaid within the same transaction
- **C**: Obtain long-term loans with fixed interest rates

## Answer

B

## Explanation

Flash loans explained:

**How they work:**
- Borrow within single transaction
- No collateral required
- Must repay in same block
- Atomicity ensures safety

**Technical mechanism:**
- Smart contract executes
- Loan, use, repay in one tx
- If not repaid, tx reverts
- Never actually at risk

**Use cases:**
- Arbitrage opportunities
- Collateral swaps
- Self-liquidation
- Yield strategies

**Attack vector concerns:**
- Protocol exploits
- Oracle manipulation
- Governance attacks
- Market manipulation

**Example:**
- Borrow $10M
- Execute arbitrage
- Repay $10M + fee
- Keep profit

**Risk considerations:**
- Protocol vulnerability
- Complexity
- Gas costs

**Common mistakes:**
- Option A: No collateral or credit check
- Option C: Single transaction, not long-term
