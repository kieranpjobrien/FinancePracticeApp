# Question

Gas fees on Ethereum represent:

## Options

- **A**: A subscription cost for using the network
- **B**: Transaction fees paid to validators for processing and securing transactions
- **C**: The price of Ethereum tokens

## Answer

B

## Explanation

Ethereum gas fees:

**What gas is:**
- Unit measuring computation
- Required for transactions
- Paid to validators
- Varies by complexity

**Fee components:**
- Gas units × Gas price
- Base fee (burned)
- Priority fee (to validator)
- Total = Transaction cost

**Why fees vary:**
- Network congestion
- Transaction complexity
- Smart contract execution
- Supply and demand

**High fees when:**
- Network congested
- NFT mints
- DeFi activity high
- Popular events

**Fee burning (EIP-1559):**
- Base fee destroyed
- Deflationary mechanism
- Reduces supply
- Implemented 2021

**User impact:**
- Cost of using dApps
- Barrier for small transactions
- Layer 2 solutions help

**Common mistakes:**
- Option A: Per-transaction, not subscription
- Option C: Gas separate from ETH price
