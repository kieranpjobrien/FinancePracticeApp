# Question

Oracles in blockchain and DeFi serve to:

## Options

- **A**: Mine new cryptocurrency tokens
- **B**: Provide external data like prices to smart contracts
- **C**: Store all blockchain transaction history

## Answer

B

## Explanation

Blockchain oracles:

**Purpose:**
- Connect blockchain to real world
- Provide external data on-chain
- Enable smart contract execution
- Price feeds most common

**Why needed:**
- Blockchains isolated by design
- Can't access off-chain data
- Smart contracts need triggers
- Real-world information required

**Data types:**
- Asset prices
- Weather data
- Sports scores
- Interest rates
- Any external data

**Oracle providers:**
- Chainlink (leading)
- Band Protocol
- Pyth Network
- API3

**Risks:**
- Oracle manipulation
- Single point of failure
- Data accuracy
- Latency issues

**DeFi dependence:**
- Lending protocol liquidations
- DEX pricing
- Derivatives settlement

**Common mistakes:**
- Option A: Mining is different function
- Option C: Nodes store transaction history
