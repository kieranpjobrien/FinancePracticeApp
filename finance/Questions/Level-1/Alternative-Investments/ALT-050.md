# Question

The capitalization rate (cap rate) in real estate represents:

## Options

- **A**: The total return including appreciation
- **B**: Net operating income divided by property value
- **C**: The interest rate on property financing

## Answer

B

## Explanation

Cap rate explained:

**Formula:**
Cap Rate = NOI / Property Value

**What it measures:**
- Current income yield
- Unlevered return
- Property pricing metric

**Example:**
- Property value: $10 million
- Annual NOI: $600,000
- Cap rate: 6%

**Uses:**
- Property valuation
- Market comparison
- Investment analysis
- Yield benchmarking

**Relationship to value:**
- Higher cap rate = lower price
- Lower cap rate = higher price
- Inverse relationship

**Factors affecting cap rates:**
- Property quality
- Location
- Tenant credit
- Market conditions
- Interest rates

**Common mistakes:**
- Option A: Cap rate is income only, not total return
- Option C: Mortgage rate is separate from cap rate
