# Question

A hurdle rate in alternative investment fee structures represents:

## Options

- **A**: The maximum return the fund can achieve
- **B**: The minimum return before performance fees are charged
- **C**: The management fee percentage

## Answer

B

## Explanation

Hurdle rate explained:

**Definition:**
- Minimum return threshold
- Must be exceeded before carry/performance fee
- Protects investor interests
- Aligns manager incentives

**Common hurdle rates:**
- Private equity: 8% preferred return
- Hedge funds: Often LIBOR/SOFR based
- Real estate: 6-10% typical

**How it works:**
- Fund returns 12%, hurdle is 8%
- Carry only on returns above 8%
- Some structures have catch-up provisions

**Types:**
- Hard hurdle: Carry only on excess
- Soft hurdle: Carry on all if exceeded
- Catch-up: GP gets larger share until equal

**Investor benefit:**
- Ensures minimum return priority
- Reduces fee drag on modest returns
- Better alignment of interests

**Common mistakes:**
- Option A: No maximum return limit
- Option C: Hurdle is separate from management fee
