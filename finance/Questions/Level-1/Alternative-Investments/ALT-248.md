# Question

Maximal Extractable Value (MEV) in blockchain refers to:

## Options

- **A**: The total market cap of a cryptocurrency
- **B**: Profit from transaction ordering, inclusion, or exclusion by block producers
- **C**: The minimum value of tokens in a wallet

## Answer

B

## Explanation

MEV explained:

**Definition:**
- Value from transaction manipulation
- By validators/miners
- Through ordering power
- Front-running, sandwich attacks

**How it works:**
- Block producers see pending txs
- Can reorder transactions
- Extract value from users
- Profit from information

**Common MEV types:**
- Front-running: Trade ahead
- Sandwich attacks: Buy before, sell after
- Arbitrage: Price discrepancies
- Liquidations: Priority on liquidating

**Impact on users:**
- Worse execution prices
- Higher transaction costs
- Invisible "tax"
- DeFi friction

**Mitigation efforts:**
- Flashbots
- Private mempools
- Fair ordering solutions
- Protocol-level designs

**Common mistakes:**
- Option A: Market cap is different concept
- Option C: Not about minimum values
