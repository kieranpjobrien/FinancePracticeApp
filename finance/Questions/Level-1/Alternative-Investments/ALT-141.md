# Question

Assets Under Management (AUM) for hedge funds represents:

## Options

- **A**: Only the personal wealth of the fund manager
- **B**: Total market value of investments managed by the fund
- **C**: The fund's annual revenue

## Answer

B

## Explanation

AUM defined:

**Definition:**
- Total value of assets managed
- Market value basis
- Includes all investor capital
- Changes with performance and flows

**Components:**
- Investor capital contributions
- Accumulated returns
- Minus withdrawals
- Market value changes

**Why it matters:**
- Size indication
- Management fee base
- Capacity constraints
- Operational scale

**AUM growth from:**
- Positive returns (appreciation)
- New investor subscriptions
- Existing investor additions

**AUM decline from:**
- Negative returns
- Investor redemptions
- Distributions

**Industry context:**
- Ranges from $10M to $100B+
- Largest funds very diversified
- Smaller funds more nimble

**Common mistakes:**
- Option A: Investor capital, not manager wealth
- Option C: Revenue is separate from AUM
