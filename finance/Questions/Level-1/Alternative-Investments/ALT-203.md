# Question

Asset recycling in infrastructure investment refers to:

## Options

- **A**: Decommissioning old assets for scrap value
- **B**: Selling mature assets to reinvest proceeds in new development or acquisitions
- **C**: Rotating assets between different portfolios

## Answer

B

## Explanation

Asset recycling strategy:

**Concept:**
- Sell stabilized assets
- At premium valuations
- Reinvest in new opportunities
- Continuous portfolio refresh

**Why recycle:**
- Capture value creation
- Access new opportunities
- Portfolio optimization
- Return capital for reinvestment

**Lifecycle approach:**
- Develop/acquire greenfield
- Stabilize operations
- Sell to core investors
- Redeploy capital

**Buyer profiles differ:**
- Sellers: Value-add/development focus
- Buyers: Core income focus
- Different return requirements
- Natural market segmentation

**Government asset recycling:**
- Sell state-owned assets
- Fund new infrastructure
- Private capital participation
- Policy mechanism

**Common mistakes:**
- Option A: Not decommissioning
- Option C: Strategic sale/reinvestment process
