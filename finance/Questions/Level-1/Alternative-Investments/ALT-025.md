# Question

The Sharpe ratio may be misleading for alternative investments because:

## Options

- **A**: Alternative investments always have higher Sharpe ratios
- **B**: Smoothed returns understate volatility, inflating the ratio
- **C**: The ratio cannot be calculated for illiquid investments

## Answer

B

## Explanation

Sharpe ratio limitations for alternatives:

**The problem:**
- Sharpe = (Return - Rf) / Volatility
- Lower volatility → higher Sharpe
- Smoothed returns reduce measured volatility
- Artificially inflates Sharpe ratio

**Why returns are smoothed:**
- Appraisal-based valuations
- Infrequent transactions
- Manager discretion in marking
- Stale prices

**Impact:**
- Volatility understated significantly
- Correlations appear lower
- Risk-adjusted returns overstated
- Misleading portfolio optimization

**Better approaches:**
- Unsmooth returns statistically
- Use transaction-based values
- Adjust for autocorrelation
- Consider drawdown measures

**Common mistakes:**
- Option A: Higher ratios may be artificial
- Option C: Sharpe can be calculated, just misleading
