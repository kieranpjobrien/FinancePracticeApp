# Question

The residual income model values equity based on:

## Options

- **A**: Only future dividends
- **B**: Book value plus the present value of expected future residual income
- **C**: Market value of assets minus liabilities

## Answer

B

## Explanation

Residual income model: V₀ = B₀ + Σ(RI_t / (1+r)^t). RI = Earnings - (Equity charge) = NI - (r × Beginning book value). Values firm based on whether it earns returns above cost of equity. Useful when dividends are unpredictable or company retains most earnings.

**Common mistakes:**
- Option A describes the dividend discount model
- Option C describes liquidation value, not going concern value
