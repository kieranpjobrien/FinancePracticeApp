# Question

A "football field" chart in equity valuation displays:

## Options

- **A**: Historical stock price movements only
- **B**: Value ranges from multiple valuation methodologies for comparison
- **C**: Quarterly earnings estimates

## Answer

B

## Explanation

Football field: horizontal bar chart showing value ranges from different methods (DCF, trading comps, transaction comps, LBO). Visualizes where current price sits relative to various value estimates. Helps identify over/undervaluation, assess methodology sensitivity. Common in M&A and equity research presentations.

**Common mistakes:**
- Option A describes price charts
- Option C describes earnings projections
