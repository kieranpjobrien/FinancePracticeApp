# Question

An investor buys a stock for $40, receives $2 in dividends, and sells for $45. The holding period return is:

## Options

- **A**: 12.5%
- **B**: 17.5%
- **C**: 5.0%

## Answer

B

## Explanation

Holding period return = (Ending price - Beginning price + Dividends) / Beginning price = ($45 - $40 + $2) / $40 = $7 / $40 = 17.5%. Captures both capital appreciation (12.5%) and dividend income (5.0%). Total return measure for evaluating investment performance.

**Common mistakes:**
- Option A only includes capital gain, ignores dividend
- Option C only includes dividend yield
