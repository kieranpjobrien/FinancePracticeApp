# Question

DCF valuations are typically most sensitive to changes in:

## Options

- **A**: The number of decimal places used
- **B**: The discount rate and terminal growth rate assumptions
- **C**: The company's office location

## Answer

B

## Explanation

DCF sensitivity: small changes in discount rate or terminal growth have large value impact. Terminal value often represents majority of DCF value. 1% change in either can change value by 20-30%. Analysts must carefully support these assumptions and show sensitivity analysis.

**Common mistakes:**
- Option A—decimal precision is immaterial
- Option C is irrelevant to valuation math
