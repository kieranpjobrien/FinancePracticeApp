# Question

If markets are weak-form efficient, which investment strategy should NOT generate abnormal returns?

## Options

- **A**: Fundamental analysis using financial statements
- **B**: Technical analysis using historical price patterns
- **C**: Insider trading using material non-public information

## Answer

B

## Explanation

Weak-form efficiency: prices reflect all historical market data (prices, volume, returns). Technical analysis using past price patterns should not generate excess returns. Fundamental analysis (semi-strong) and insider information (strong form) could still work under weak-form efficiency.

**Common mistakes:**
- Option A could work in weak-form markets (semi-strong addresses this)
- Option C could work in weak and semi-strong (only strong-form addresses insider info)
