# Question

Using multiple valuation methods (triangulation) is recommended because:

## Options

- **A**: Only one method can ever be correct
- **B**: Different methods provide checks and balances, increasing confidence if results converge
- **C**: It guarantees an accurate valuation

## Answer

B

## Explanation

Triangulation: using multiple approaches (DCF, multiples, asset-based) to cross-check value estimates. Convergence across methods increases confidence. Divergence prompts investigation of assumptions. Each method has strengths and weaknesses—triangulation leverages multiple perspectives for more robust conclusion.

**Common mistakes:**
- Option A ignores value of multiple perspectives
- Option C—no method or combination guarantees accuracy
