# Question

The size effect anomaly suggests that:

## Options

- **A**: Large-cap stocks consistently outperform small-cap stocks
- **B**: Small-cap stocks have historically generated higher returns than large-cap stocks
- **C**: Stock size has no impact on returns

## Answer

B

## Explanation

Size effect: small-cap stocks historically earn higher returns than large caps on average. May reflect higher risk (less liquidity, higher volatility, greater business risk) or persistent mispricing. Effect weakened in recent decades. Included in Fama-French factor models.

**Common mistakes:**
- Option A is opposite to the size effect
- Option C contradicts documented evidence
