# Question

The residual income model values a company based on:

## Options

- **A**: Projected free cash flows discounted at WACC
- **B**: Book value plus the present value of expected future residual income
- **C**: Replacement cost of assets less liabilities

## Answer

B

## Explanation

Residual income model: V₀ = BV₀ + Σ(RI_t / (1+r)^t), where RI = Net Income - (r × Beginning Book Value). Values company as book value plus PV of earnings above required return on equity. Useful when cash flows are negative or unpredictable. Links accounting values to intrinsic value.

**Common mistakes:**
- Option A describes DCF/FCFE model
- Option C describes asset-based valuation
