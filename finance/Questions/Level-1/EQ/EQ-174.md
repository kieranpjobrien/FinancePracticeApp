# Question

Value chain analysis helps identify competitive advantage by examining:

## Options

- **A**: Only the company's marketing activities
- **B**: All activities from input procurement through customer delivery and support
- **C**: Exclusively financial ratios and metrics

## Answer

B

## Explanation

Value chain (Porter): primary activities (inbound logistics, operations, outbound logistics, marketing/sales, service) and support activities (infrastructure, HR, technology, procurement). Analyzes where value is created and costs incurred. Competitive advantage from performing activities better or differently than rivals.

**Common mistakes:**
- Option A is only one primary activity in the chain
- Option C is financial analysis, not value chain analysis
