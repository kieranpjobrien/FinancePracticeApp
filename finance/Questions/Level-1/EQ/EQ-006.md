# Question

An investor buys 100 shares at $50 per share using 50% initial margin. If the stock falls to $35, the investor's equity percentage is approximately:

## Options

- **A**: 50%
- **B**: 29%
- **C**: 71%

## Answer

B

## Explanation

Initial position: 100 × $50 = $5,000. Borrowed: $2,500 (50%). Equity: $2,500. At $35: Position value = $3,500. Debt unchanged = $2,500. Equity = $3,500 - $2,500 = $1,000. Equity percentage = $1,000 / $3,500 = 28.6% ≈ 29%. Leverage amplifies losses.

**Common mistakes:**
- Option A is the initial margin, not current equity
- Option C may result from calculation errors
