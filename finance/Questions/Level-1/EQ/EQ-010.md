# Question

An equal-weighted index requires regular rebalancing because:

## Options

- **A**: Stock prices never change
- **B**: Price changes cause weights to drift from equal proportions
- **C**: Companies frequently merge

## Answer

B

## Explanation

Equal-weighted index: each stock initially gets same weight. As prices change, weights drift—outperformers become overweighted, underperformers underweighted. Periodic rebalancing (selling winners, buying losers) restores equal weights. Higher turnover and transaction costs than cap-weighted indices.

**Common mistakes:**
- Option A is factually incorrect
- Option C is not the primary reason for rebalancing
