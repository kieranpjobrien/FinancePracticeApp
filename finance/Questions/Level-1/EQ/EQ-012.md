# Question

A total return index differs from a price return index by:

## Options

- **A**: Excluding capital gains
- **B**: Including reinvested dividends in the return calculation
- **C**: Only including the largest stocks

## Answer

B

## Explanation

Total return index: assumes dividends are reinvested in the index. Captures both price appreciation and income. Price return index only tracks price changes, excluding dividends. Total return provides better measure of actual investment performance. Most academic studies use total return indices.

**Common mistakes:**
- Option A is backwards—total return includes everything
- Option C describes index construction, not return type
