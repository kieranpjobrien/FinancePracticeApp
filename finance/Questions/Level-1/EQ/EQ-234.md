# Question

Using the Gordon Growth Model, the justified P/B ratio equals:

## Options

- **A**: (ROE - g) / (r - g)
- **B**: (ROE × Payout) / (r - g)
- **C**: D₁ / Book Value

## Answer

A

## Explanation

Justified P/B from DDM: P₀/B₀ = (ROE - g) / (r - g). Derived by dividing intrinsic value by book value. When ROE > r, justified P/B > 1. Higher ROE and growth increase justified P/B. Links fundamental drivers (ROE, growth, required return) to price-to-book multiple.

**Common mistakes:**
- Option B incorrectly structures the numerator
- Option C doesn't produce a ratio that links P/B to fundamentals
