# Question

According to the semi-strong form of the efficient market hypothesis, an investor cannot consistently achieve abnormal returns using:

## Options

- **A**: Technical analysis only
- **B**: Fundamental analysis and technical analysis
- **C**: Inside information only

## Answer

B

## Explanation

The three forms of the Efficient Market Hypothesis are:

- **Weak form**: Prices reflect all past market data (price and volume). Technical analysis cannot generate abnormal returns.
- **Semi-strong form**: Prices reflect all publicly available information. Neither technical analysis nor fundamental analysis can generate abnormal returns.
- **Strong form**: Prices reflect all information, including private/inside information. No strategy can generate abnormal returns.

Under semi-strong efficiency, both technical and fundamental analysis are ineffective because all public information is already reflected in prices.

**Common mistakes:**
- Option A only describes weak-form efficiency
- Option C describes strong-form efficiency
