# Question

Intrinsic value of a stock represents:

## Options

- **A**: The current market price
- **B**: The analyst's estimate of the stock's true worth based on fundamentals
- **C**: The book value per share

## Answer

B

## Explanation

Intrinsic value: estimated true value based on analysis of fundamentals—earnings, cash flows, growth, risk. Compare to market price: if intrinsic > market, stock is undervalued (buy); if intrinsic < market, overvalued (sell/avoid). Different analysts may derive different intrinsic values.

**Common mistakes:**
- Option A is market price, which may differ from intrinsic value
- Option C is an accounting measure, not intrinsic value
