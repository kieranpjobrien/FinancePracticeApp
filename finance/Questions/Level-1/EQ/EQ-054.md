# Question

The value effect suggests that stocks with:

## Options

- **A**: High P/E ratios outperform low P/E stocks
- **B**: Low valuation multiples (low P/E, high book-to-market) tend to outperform
- **C**: Highest market caps always generate the best returns

## Answer

B

## Explanation

Value effect: value stocks (low P/E, P/B; high dividend yield, B/M) historically outperform growth stocks. Possible explanations: compensation for distress risk, behavioral overreaction creating mispricing. Fama-French model includes value factor. Effect varies over time and across markets.

**Common mistakes:**
- Option A is opposite—high P/E (growth) historically underperforms
- Option C describes size effect incorrectly
