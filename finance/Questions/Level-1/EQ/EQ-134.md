# Question

Index rebalancing refers to:

## Options

- **A**: Adding new companies to the index
- **B**: Adjusting constituent weights back to target weights periodically
- **C**: Removing companies that no longer meet criteria

## Answer

B

## Explanation

Rebalancing: periodic adjustment of index weights back to target weights. Price-weighted indices don't require rebalancing. Equal-weighted indices need frequent rebalancing as price changes alter weights. Fundamental-weighted indices rebalance to reflect updated fundamental values. Reconstitution (adding/removing) is different from rebalancing.

**Common mistakes:**
- Option A describes reconstitution, not rebalancing
- Option C also describes reconstitution
