# Question

A key limitation of relative valuation is that:

## Options

- **A**: It requires no peer companies
- **B**: If the entire sector is overvalued, relative value may not indicate absolute mispricing
- **C**: It never uses financial ratios

## Answer

B

## Explanation

Relative valuation limitation: a stock can be cheap relative to peers but still overvalued in absolute terms if entire sector is mispriced. Technology bubble example—cheap vs. peers but all overvalued. Need absolute valuation check. Also requires truly comparable peers.

**Common mistakes:**
- Option A—relative valuation requires peers for comparison
- Option C—relative valuation is based on ratios/multiples
