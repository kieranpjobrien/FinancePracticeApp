# Question

Event studies testing semi-strong form efficiency examine:

## Options

- **A**: Long-term autocorrelation in stock returns
- **B**: The speed and accuracy with which prices adjust to public announcements
- **C**: Whether insiders can profit from material non-public information

## Answer

B

## Explanation

Event study methodology: measures abnormal returns around public announcements (earnings, M&A, dividends). If prices adjust quickly and completely, supports semi-strong efficiency. Delayed or drift in abnormal returns suggests inefficiency. Common events studied: earnings surprises, stock splits, dividend changes.

**Common mistakes:**
- Option A tests weak-form efficiency (technical analysis)
- Option C tests strong-form efficiency (insider trading)
