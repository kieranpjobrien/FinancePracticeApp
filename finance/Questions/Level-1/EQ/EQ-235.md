# Question

Valuing private companies compared to public companies typically requires:

## Options

- **A**: The same methods with no adjustments
- **B**: Additional adjustments for lack of marketability and possibly control premiums
- **C**: Only using the book value approach

## Answer

B

## Explanation

Private company valuation adjustments: lack of marketability discount (DLOM)—harder to sell shares, control premium/minority discount depending on stake valued, size premium, key person risk. Same basic methods (DCF, comparables) but with adjustments for private company characteristics.

**Common mistakes:**
- Option A ignores important differences between public/private
- Option C is unnecessarily restrictive—many methods apply
