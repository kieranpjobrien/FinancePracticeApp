# Question

An investor buys $10,000 of stock with 50% initial margin and 30% maintenance margin. A margin call will occur if the stock value falls below:

## Options

- **A**: $7,143
- **B**: $5,000
- **C**: $3,000

## Answer

A

## Explanation

Margin call trigger: Equity/Stock Value = Maintenance margin. Initial equity = $5,000 (50% × $10,000). Loan = $5,000. At margin call: (Stock Value - $5,000) / Stock Value = 30%. Solving: 0.70 × Stock Value = $5,000, so Stock Value = $5,000/0.70 = $7,143. Below this triggers margin call.

**Common mistakes:**
- Option B confuses initial investment with trigger level
- Option C miscalculates the relationship between equity and loan
