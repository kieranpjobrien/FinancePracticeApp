# Question

A stock is at $100. Over each of two periods, it can go up 10% or down 10%. The risk-free rate is 5% per period. A two-period European call with strike $100 has an initial value closest to:

## Options

- **A**: $10.71
- **B**: $12.47
- **C**: $21.00

## Answer

A

## Explanation

Two-period binomial:

**Build the stock tree:**
- S = $100
- Suu = $100 × 1.10 × 1.10 = $121
- Sud = Sdu = $100 × 1.10 × 0.90 = $99
- Sdd = $100 × 0.90 × 0.90 = $81

**Option payoffs at expiration:**
- Cuu = Max(121 - 100, 0) = $21
- Cud = Max(99 - 100, 0) = $0
- Cdd = Max(81 - 100, 0) = $0

**Risk-neutral probability:**
π = (1.05 - 0.90) / (1.10 - 0.90)
π = 0.15 / 0.20 = 0.75

**Work backwards - Period 1:**
Cu = [0.75 × 21 + 0.25 × 0] / 1.05 = $15
Cd = [0.75 × 0 + 0.25 × 0] / 1.05 = $0

**Period 0 (initial value):**
C = [0.75 × 15 + 0.25 × 0] / 1.05
C = 11.25 / 1.05 = $10.71

**Common mistakes:**
- Option B: Calculation error in discounting
- Option C: Used final payoff without discounting
