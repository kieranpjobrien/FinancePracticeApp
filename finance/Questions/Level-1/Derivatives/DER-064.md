# Question

At inception, the value of a forward contract to both parties is:

## Options

- **A**: Positive for the long and negative for the short
- **B**: Zero for both parties
- **C**: Equal to the current spot price

## Answer

B

## Explanation

Forward value at inception:

**Zero value principle:**
- No money exchanged upfront
- Fair contract for both sides
- Forward price set by no-arbitrage

**Why zero value:**
- Forward price = Fair price
- PV of expected payoff = 0
- Neither party advantaged

**Forward price determination:**
F = S × (1 + r)^T

This price makes contract value = 0

**After inception:**
- Market prices change
- Contract gains value for one party
- Loses value for the other
- Zero-sum game

**Example:**
- Agree to forward at $105
- If spot rises, long has positive value
- Short has negative value
- Values are mirror images

**Common mistakes:**
- Option A: Only true after prices move
- Option C: Forward price, not spot price, is relevant
