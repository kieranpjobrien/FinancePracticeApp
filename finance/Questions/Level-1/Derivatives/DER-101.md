# Question

A stock at $50 can go to $60 or $45 in one period. The risk-free rate is 4%. A put option with strike $50 is worth closest to:

## Options

- **A**: $1.28
- **B**: $2.40
- **C**: $5.00

## Answer

A

## Explanation

One-period put calculation:

**Parameters:**
- S = $50
- Su = $60, u = 1.20
- Sd = $45, d = 0.90
- K = $50
- r = 4%

**Put payoffs:**
- Pu = Max(50 - 60, 0) = $0
- Pd = Max(50 - 45, 0) = $5

**Risk-neutral probability:**
π = (1 + r - d) / (u - d)
π = (1.04 - 0.90) / (1.20 - 0.90)
π = 0.14 / 0.30
π = 0.467

**Expected payoff:**
E[P] = π × Pu + (1-π) × Pd
E[P] = 0.467 × 0 + 0.533 × 5
E[P] = $2.67

**Discount to present:**
P = $2.67 / 1.04
P = $2.57

Wait, let me recalculate:
π = (1.04 - 0.90)/(1.20 - 0.90) = 0.14/0.30 = 0.4667
1 - π = 0.5333
E[P] = 0 + 0.5333 × 5 = 2.667
P = 2.667/1.04 = $2.56

Hmm, closest to $2.40.

- Option A: Calculation variations
- Option C: This is intrinsic value at down node
