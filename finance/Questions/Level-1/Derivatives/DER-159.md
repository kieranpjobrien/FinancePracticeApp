# Question

A synthetic risk-free bond can be created using options by:

## Options

- **A**: Buying a call and a put at the same strike
- **B**: Buying stock, buying a put, and selling a call at the same strike
- **C**: Selling both a call and a put at the same strike

## Answer

B

## Explanation

Synthetic bond creation:

**From put-call parity:**
C + PV(K) = P + S

**Rearranging for bond:**
PV(K) = P + S - C

**Synthetic risk-free bond:**
- Long put (+P)
- Long stock (+S)
- Short call (-C)
- Equals bond paying K at expiration

**Verification at expiration:**

*If S_T > K:*
- Stock worth S_T
- Put expires worthless
- Call exercised: Sell at K
- Net: K

*If S_T < K:*
- Stock worth S_T
- Exercise put: Sell at K
- Call expires worthless
- Net: K

**Always receive K at expiration:**
- Risk-free payoff
- Like a zero-coupon bond
- Value today = PV(K)

**Common mistakes:**
- Option A: This is a straddle, not risk-free
- Option C: This is a short straddle, has risk
