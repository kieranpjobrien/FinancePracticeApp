# Question

A swap can be thought of as a series of forward contracts because:

## Options

- **A**: Both require physical delivery of assets
- **B**: Each swap payment date represents a forward commitment at a fixed rate
- **C**: Both have identical tax treatment

## Answer

B

## Explanation

Swap as forward series:

**Conceptual decomposition:**
- Each payment period = One forward
- Swap = Bundle of forwards
- All at same fixed rate

**Interest rate swap:**
- 5-year quarterly swap
- = 20 forward rate agreements
- Each locks in fixed vs. floating

**Key insight:**
- Forward on interest rate for each period
- Fixed rate determined at inception
- Settlement on each payment date

**Difference from actual forwards:**
- Individual forwards: Different forward rates
- Swap: Single fixed rate for all periods
- Swap rate = Weighted average

**Valuation connection:**
- Value swap as sum of forwards
- Each forward valued separately
- Sum = Total swap value

**Common mistakes:**
- Option A: Many swaps are cash settled
- Option C: Tax treatment varies by jurisdiction
