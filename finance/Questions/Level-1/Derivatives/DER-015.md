# Question

A forward rate agreement (FRA) is used to:

## Options

- **A**: Lock in an exchange rate for a future currency transaction
- **B**: Lock in an interest rate for a future borrowing or lending period
- **C**: Lock in a commodity price for future delivery

## Answer

B

## Explanation

Forward Rate Agreement basics:

**Definition:**
- OTC derivative
- Locks in interest rate
- For a future period
- Cash settled

**Structure:**
- Notional principal (not exchanged)
- Contract rate (locked-in rate)
- Reference rate (e.g., SOFR)
- Settlement based on rate difference

**FRA notation (e.g., 3×6 FRA):**
- Contract starts in 3 months
- Underlying rate is 3-month rate
- Total period = 6 months

**Uses:**
- Hedge future borrowing costs
- Hedge future investment rates
- Speculate on rate movements

**Settlement:**
- If reference rate > contract rate: buyer receives
- If reference rate < contract rate: seller receives

**Common mistakes:**
- Option A: Describes currency forward
- Option C: Describes commodity forward
