# Question

The maximum value of a call option is:

## Options

- **A**: The strike price
- **B**: The present value of the strike price
- **C**: The stock price

## Answer

C

## Explanation

Call option upper bound:

**Maximum call value:**
C ≤ S

**Reasoning:**
- Call gives right to buy stock
- Can't be worth more than what it buys
- At most, pays stock value

**Extreme case:**
- Strike = $0
- Call = Right to get stock for free
- Call value = Stock value exactly

**Non-zero strike:**
- Must pay strike to exercise
- Call worth less than stock
- C < S

**Practical upper bound:**
- Very deep ITM call
- Approaches stock value
- But always less (must pay K)

**Contrast with put:**
- Put maximum = PV(K) for European
- Put maximum = K for American
- Different upper bounds

**Example:**
- Stock at $100
- $10 strike call
- Maximum value = $100
- Likely close to $90 (intrinsic)

**Common mistakes:**
- Option A: Strike is exercise price, not call maximum
- Option B: This relates to put maximum
