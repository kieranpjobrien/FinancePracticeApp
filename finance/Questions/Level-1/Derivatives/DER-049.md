# Question

A stock is at $100. In one period, it can go to $120 (up) or $85 (down). The risk-free rate is 5% per period. A call option with strike $100 is worth closest to:

## Options

- **A**: $12.86
- **B**: $15.24
- **C**: $20.00

## Answer

B

## Explanation

One-period binomial calculation:

**Step 1: Identify parameters**
- S = $100
- Su = $120, so u = 1.20
- Sd = $85, so d = 0.85
- K = $100
- r = 5%

**Step 2: Calculate option payoffs**
- Cu (up state) = Max(120 - 100, 0) = $20
- Cd (down state) = Max(85 - 100, 0) = $0

**Step 3: Risk-neutral probability**
π = (1 + r - d) / (u - d)
π = (1.05 - 0.85) / (1.20 - 0.85)
π = 0.20 / 0.35
π = 0.5714

**Step 4: Expected payoff**
E[C] = π × Cu + (1-π) × Cd
E[C] = 0.5714 × $20 + 0.4286 × $0
E[C] = $11.43

**Step 5: Discount to present**
C = E[C] / (1 + r)
C = $11.43 / 1.05
C = $15.24

**Common mistakes:**
- Option A: Used wrong probability
- Option C: Didn't discount to present value
