# Question

For a dividend-paying stock, put-call parity becomes:

## Options

- **A**: C + PV(K) = P + S - PV(Dividends)
- **B**: C + PV(K) = P + S + PV(Dividends)
- **C**: Put-call parity does not hold for dividend-paying stocks

## Answer

A

## Explanation

Put-call parity with dividends:

**Standard parity:**
C + PV(K) = P + S

**With dividends:**
C + PV(K) = P + S - PV(D)

**Why subtract dividends:**
- Option holder doesn't receive dividends
- Stock holder does receive dividends
- Must adjust stock value for dividends

**Rearranged:**
C - P = S - PV(D) - PV(K)

**Interpretation:**
- Right side = Forward value
- Dividends reduce effective stock price
- Lower call values, higher put values

**Example:**
- Stock at $50, dividend PV = $2
- Effective stock = $48 for parity
- Call worth less than no-dividend case

**Common mistakes:**
- Option B: Dividends are subtracted, not added
- Option C: Parity holds with adjustment
