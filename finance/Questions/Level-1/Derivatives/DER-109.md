# Question

A synthetic long put can be created by:

## Options

- **A**: Buying a call and buying the stock
- **B**: Buying a call, selling the stock, and lending the present value of the strike
- **C**: Selling a call and selling the stock

## Answer

B

## Explanation

Synthetic put creation:

**From put-call parity:**
C + PV(K) = P + S

**Solving for P:**
P = C - S + PV(K)

**Synthetic long put:**
- Buy call (+C)
- Sell stock (-S)
- Lend PV(K) (+PV(K))

**Verification at expiration:**

*If S_T > K:*
- Exercise call: Buy at K, receive stock
- Stock sold: Owe S_T
- Loan matures: Receive K
- Net: K - S_T (same as put payoff = $0)

Wait, let me reconsider:
*If S_T > K:*
- Call worth S_T - K
- Short stock: -S_T
- Bond: +K
- Net: S_T - K - S_T + K = 0 ✓

*If S_T < K:*
- Call worth 0
- Short stock: -S_T
- Bond: +K
- Net: 0 - S_T + K = K - S_T ✓ (put payoff)

**Common mistakes:**
- Option A: This creates call + stock exposure
- Option C: Different payoff profile
