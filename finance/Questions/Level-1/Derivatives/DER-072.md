# Question

When pricing an American option using the binomial model, at each node the option value is:

## Options

- **A**: Always the discounted expected value of future payoffs
- **B**: The maximum of the exercise value and the discounted expected value
- **C**: Always equal to the intrinsic value

## Answer

B

## Explanation

American option in binomial model:

**European option:**
- Value = Discounted expected value
- No early exercise consideration
- Work backward from expiration

**American option:**
- At each node, check if early exercise optimal
- Value = Max(Exercise value, Holding value)
- May exercise early if beneficial

**Decision at each node:**
1. Calculate intrinsic value (exercise now)
2. Calculate continuation value (hold option)
3. Option value = Maximum of the two

**When early exercise matters:**
- Deep ITM American puts
- Calls on dividend-paying stocks
- Time value < Immediate value

**Example:**
- Intrinsic value = $8
- Continuation value = $7
- Exercise now (American value = $8)
- European would wait (value = $7)

**Common mistakes:**
- Option A: Applies to European, not American
- Option C: Only if early exercise optimal everywhere
