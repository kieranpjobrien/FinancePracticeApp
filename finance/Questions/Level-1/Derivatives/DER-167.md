# Question

Put-call parity holds exactly for:

## Options

- **A**: Only American-style options
- **B**: Only European-style options
- **C**: Both American and European options equally

## Answer

B

## Explanation

Put-call parity and option style:

**European options:**
- Parity holds exactly
- C + PV(K) = P + S
- No early exercise complicates it

**American options:**
- Parity is an inequality, not equality
- Early exercise premium exists
- American call ≥ European call
- American put ≥ European put

**For American options:**
C + PV(K) ≤ P + S ≤ C + K

**Why parity breaks for American:**
- Early exercise creates extra value
- Put-call symmetry disrupted
- Replication doesn't work exactly

**American calls (no dividend):**
- Usually equal to European
- Parity nearly holds

**American puts:**
- Often worth more than European
- Early exercise for deep ITM
- Greater deviation from parity

**Common mistakes:**
- Option A: American options don't satisfy exact parity
- Option C: Parity is exact only for European
