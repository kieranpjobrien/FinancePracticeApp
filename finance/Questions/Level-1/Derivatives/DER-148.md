# Question

At expiration, an option's value equals:

## Options

- **A**: Its time value only
- **B**: Its intrinsic value only
- **C**: The premium originally paid

## Answer

B

## Explanation

Option value at expiration:

**At expiration:**
- Time value = 0
- No time left for favorable moves
- Value = Intrinsic value only

**For calls:**
Value = Max(0, S - K)

**For puts:**
Value = Max(0, K - S)

**Why time value disappears:**
- No uncertainty remains
- Outcome is known
- No optionality left

**Before expiration:**
Value = Intrinsic + Time value

**At expiration:**
Value = Intrinsic + 0 = Intrinsic

**Examples at expiration:**
- Stock $55, Call strike $50: Value = $5
- Stock $45, Call strike $50: Value = $0
- Stock $45, Put strike $50: Value = $5

**Common mistakes:**
- Option A: Time value is zero at expiration
- Option C: Original premium is sunk cost
