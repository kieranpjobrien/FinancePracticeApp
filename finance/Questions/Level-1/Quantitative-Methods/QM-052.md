# Question

A stock price increases from $100 to $110. The continuously compounded (logarithmic) return is closest to:

## Options

- **A**: 9.53%
- **B**: 10.00%
- **C**: 10.52%

## Answer

A

## Explanation

Continuously compounded return = ln(Ending Price / Beginning Price)

r = ln($110 / $100)
r = ln(1.10)
r = 0.0953 = 9.53%

The continuously compounded return is slightly less than the simple return of 10% because of the mathematical relationship between logarithmic and arithmetic returns. Continuous returns are additive over time, making them useful for multi-period analysis.

**Common mistakes:**
- Option B is the simple (arithmetic) return, not the continuously compounded return
- Option C results from incorrectly applying the exponential function instead of the natural logarithm
