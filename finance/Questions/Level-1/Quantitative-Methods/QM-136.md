# Question

The information ratio measures:

## Options

- **A**: The total return of a portfolio relative to its risk
- **B**: The active return per unit of active risk (tracking error)
- **C**: The portfolio's sensitivity to market movements

## Answer

B

## Explanation

Information Ratio = (Portfolio Return - Benchmark Return) / Tracking Error
                  = Active Return / Tracking Error

The information ratio measures how much excess return over the benchmark an active manager generates per unit of active risk taken. A higher information ratio indicates more consistent value added through active management. It is commonly used to evaluate the skill of active portfolio managers.

**Common mistakes:**
- Option A describes the Sharpe ratio, which uses total return and total risk
- Option C describes beta, which measures systematic risk sensitivity
