# Question

When the p-value is less than the significance level, the analyst should:

## Options

- **A**: Fail to reject the null hypothesis
- **B**: Reject the null hypothesis
- **C**: Increase the sample size

## Answer

B

## Explanation

The decision rule for hypothesis testing using p-values is: reject the null hypothesis if p-value ≤ significance level (α). A small p-value indicates the observed data is unlikely under the null hypothesis, providing evidence against it. When p-value < α, the result is statistically significant and we reject the null hypothesis.

**Common mistakes:**
- Option A reverses the decision rule
- Option C is not a standard response to a significant result
