# Question

A point estimate is best described as:

## Options

- **A**: A range of values within which a parameter is expected to fall
- **B**: A single value used to estimate an unknown population parameter
- **C**: The exact value of a population parameter

## Answer

B

## Explanation

A point estimate is a single numerical value calculated from sample data that serves as the best estimate of an unknown population parameter. For example, the sample mean is a point estimate of the population mean. Point estimates are useful but provide no information about the uncertainty or precision of the estimate, which is why confidence intervals are often used alongside them.

**Common mistakes:**
- Option A describes a confidence interval, not a point estimate
- Option C is incorrect because point estimates are calculated from samples and are unlikely to exactly equal the true population parameter
