# Question

When comparing the risk of two investments with different expected returns, the most appropriate measure to use is:

## Options

- **A**: Standard deviation
- **B**: Coefficient of variation
- **C**: Variance

## Answer

B

## Explanation

The coefficient of variation (CV) measures risk per unit of expected return: CV = Standard Deviation / Mean. This standardized measure allows comparison of relative risk across investments with different return levels. When expected returns differ, CV provides a fairer comparison than standard deviation alone.

**Common mistakes:**
- Option A (standard deviation) doesn't account for differing return levels
- Option C (variance) has the same limitation as standard deviation and is in squared units
