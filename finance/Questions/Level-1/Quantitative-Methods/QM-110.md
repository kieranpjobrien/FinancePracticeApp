# Question

An analyst must rank 5 stocks from best to worst. How many different rankings are possible?

## Options

- **A**: 25
- **B**: 120
- **C**: 3,125

## Answer

B

## Explanation

This is a permutation problem where order matters. The number of ways to arrange n items is n! (n factorial).

5! = 5 × 4 × 3 × 2 × 1 = 120

There are 120 different possible rankings for the 5 stocks.

**Common mistakes:**
- Option A calculates 5² = 25, confusing with combinations or powers
- Option C calculates 5⁵ = 3,125, which would apply if stocks could be assigned to the same rank
