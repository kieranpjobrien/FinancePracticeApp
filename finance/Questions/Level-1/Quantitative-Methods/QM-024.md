# Question

An investor allocates $300,000 to Stock A (expected return 12%) and $200,000 to Stock B (expected return 8%). The expected return of the portfolio is:

## Options

- **A**: 9.6%
- **B**: 10.0%
- **C**: 10.4%

## Answer

C

## Explanation

First, calculate portfolio weights:
- Weight of A = $300,000 / $500,000 = 0.60 (60%)
- Weight of B = $200,000 / $500,000 = 0.40 (40%)

Expected portfolio return = w₁E(R₁) + w₂E(R₂)
= (0.60)(12%) + (0.40)(8%)
= 7.2% + 3.2%
= 10.4%

**Common mistakes:**
- Option A results from reversing the weights (putting 40% in A and 60% in B)
- Option B results from taking the simple average of returns without considering the different investment amounts
