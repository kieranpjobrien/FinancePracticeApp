# Question

First-order autocorrelation in a return series refers to:

## Options

- **A**: The correlation between all pairs of returns
- **B**: The correlation between consecutive returns (t and t-1)
- **C**: The correlation between returns and trading volume

## Answer

B

## Explanation

First-order (lag-1) autocorrelation measures the correlation between each observation and the immediately preceding observation. If returns at time t are correlated with returns at time t-1, the series exhibits first-order autocorrelation. Positive autocorrelation suggests momentum; negative suggests mean reversion. Higher-order autocorrelation examines correlations at longer lags.

**Common mistakes:**
- Option A is too broad; autocorrelation is measured at specific lags
- Option C describes cross-correlation between different variables, not autocorrelation
