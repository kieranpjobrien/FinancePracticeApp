# Question

A regression equation is: Y = 2 + 3X. If X equals 5, the predicted value of Y is:

## Options

- **A**: 10
- **B**: 15
- **C**: 17

## Answer

C

## Explanation

Substituting X = 5 into the regression equation:

Ŷ = 2 + 3(5)
Ŷ = 2 + 15
Ŷ = 17

The predicted value (fitted value) is 17.

**Common mistakes:**
- Option A multiplies 2 × 5 = 10 instead of using the correct formula
- Option B only calculates the 3X term (3 × 5 = 15) without adding the intercept
