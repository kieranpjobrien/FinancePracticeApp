# Question

If the probability of an event is 0.25, the odds for the event are:

## Options

- **A**: 1 to 3
- **B**: 1 to 4
- **C**: 3 to 1

## Answer

A

## Explanation

Odds for an event = P(event) / P(not event)
Odds for = 0.25 / 0.75 = 1/3

This is expressed as "1 to 3" odds for the event occurring (1 chance of success for every 3 chances of failure).

**Common mistakes:**
- Option B expresses the probability as a ratio (1 out of 4) rather than odds
- Option C gives the odds against the event (3 to 1 against), not odds for
