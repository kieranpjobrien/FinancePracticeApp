# Question

When conducting multiple hypothesis tests simultaneously, the Bonferroni correction:

## Options

- **A**: Increases the significance level for each test to maximize power
- **B**: Divides the overall significance level by the number of tests to control Type I error
- **C**: Eliminates the need for hypothesis testing

## Answer

B

## Explanation

The Bonferroni correction addresses the multiple comparisons problem. When performing n tests at significance level α, the probability of at least one false positive increases. Bonferroni uses α/n as the significance level for each test, maintaining the overall (family-wise) Type I error rate at α. This is conservative and reduces power but controls false positives.

**Common mistakes:**
- Option A would increase Type I error, the opposite of the correction's purpose
- Option C is incorrect; the correction adjusts hypothesis testing, doesn't replace it
