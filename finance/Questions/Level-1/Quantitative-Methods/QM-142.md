# Question

A portfolio has 40% in stocks with 12% expected return and 60% in bonds with 5% expected return. The portfolio's expected return is:

## Options

- **A**: 7.8%
- **B**: 8.5%
- **C**: 10.2%

## Answer

A

## Explanation

Portfolio expected return = Σ(weight × return)

E(Rp) = (0.40 × 12%) + (0.60 × 5%)
E(Rp) = 4.8% + 3.0%
E(Rp) = 7.8%

**Common mistakes:**
- Option B uses a simple average of returns: (12% + 5%)/2 = 8.5%
- Option C reverses the weights, applying 60% to stocks and 40% to bonds
