# Question

A stock pays a current dividend of $2.00, which is expected to grow at 4% annually forever. If the required return is 10%, the stock's value is closest to:

## Options

- **A**: $33.33
- **B**: $34.67
- **C**: $50.00

## Answer

B

## Explanation

For a growing perpetuity (Gordon Growth Model):

PV = D₁ / (r - g)

Where D₁ = D₀ × (1 + g) = $2.00 × 1.04 = $2.08

PV = $2.08 / (0.10 - 0.04)
PV = $2.08 / 0.06
PV = $34.67

**Common mistakes:**
- Option A uses the current dividend D₀ instead of next year's dividend D₁ in the numerator
- Option C ignores the growth rate and treats it as a simple perpetuity ($2.00 / 0.04 or similar error)
