# Question

A portfolio consists of 60% Asset A and 40% Asset B. Asset A has a standard deviation of 20% and Asset B has a standard deviation of 30%. If the correlation between the assets is 0.5, the portfolio variance is closest to:

## Options

- **A**: 0.0360
- **B**: 0.0432
- **C**: 0.0576

## Answer

B

## Explanation

Portfolio Variance = w²ₐσ²ₐ + w²ᵦσ²ᵦ + 2wₐwᵦσₐσᵦρₐᵦ

σ²ₚ = (0.60)²(0.20)² + (0.40)²(0.30)² + 2(0.60)(0.40)(0.20)(0.30)(0.50)
σ²ₚ = 0.36(0.04) + 0.16(0.09) + 2(0.60)(0.40)(0.06)(0.50)
σ²ₚ = 0.0144 + 0.0144 + 0.0144
σ²ₚ = 0.0432

Portfolio standard deviation = √0.0432 = 20.8%

**Common mistakes:**
- Option A ignores the covariance term
- Option C may result from using variances incorrectly
