# Question

Tracking error measures:

## Options

- **A**: The difference between portfolio return and benchmark return at a point in time
- **B**: The standard deviation of the differences between portfolio and benchmark returns over time
- **C**: The correlation between portfolio returns and benchmark returns

## Answer

B

## Explanation

Tracking error is the standard deviation of active returns (portfolio return minus benchmark return). It measures the consistency of the portfolio's performance relative to its benchmark. A higher tracking error indicates the portfolio deviates more from the benchmark, either outperforming or underperforming by varying amounts. Active managers typically have higher tracking error than passive managers.

**Common mistakes:**
- Option A describes active return at a single point, not tracking error
- Option C describes a correlation measure, not tracking error
