# Question

The standard error of the estimate (SEE) in regression measures:

## Options

- **A**: The precision of the slope coefficient estimate
- **B**: The typical distance between observed values and the regression line
- **C**: The correlation between the independent and dependent variables

## Answer

B

## Explanation

The standard error of the estimate (SEE) measures the dispersion of the observed Y values around the regression line. It represents the standard deviation of the residuals and indicates the typical prediction error. A smaller SEE indicates a better fit, with observed values clustering more closely around the predicted values.

**Common mistakes:**
- Option A describes the standard error of the slope coefficient, not the SEE
- Option C describes correlation or R-squared concepts, not the SEE
