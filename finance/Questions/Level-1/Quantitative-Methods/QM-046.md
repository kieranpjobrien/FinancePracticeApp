# Question

The assumption of homoskedasticity in regression analysis means that:

## Options

- **A**: The independent variable has constant mean across observations
- **B**: The variance of the error term is constant across all values of X
- **C**: The error terms are normally distributed

## Answer

B

## Explanation

Homoskedasticity requires that the variance of the regression residuals (error terms) remains constant across all levels of the independent variable. When this assumption is violated (heteroskedasticity), the variance of errors changes with X—for example, prediction errors might be larger for higher values of X. Heteroskedasticity does not bias coefficient estimates but does affect the validity of standard errors and hypothesis tests.

**Common mistakes:**
- Option A is not an assumption of regression; the independent variable is treated as fixed or given
- Option C describes the normality assumption, which is separate from homoskedasticity
