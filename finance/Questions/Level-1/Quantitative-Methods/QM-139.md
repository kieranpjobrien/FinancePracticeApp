# Question

An estimator is efficient if it:

## Options

- **A**: Always equals the true population parameter
- **B**: Has the smallest variance among all unbiased estimators
- **C**: Converges to the true value as sample size increases

## Answer

B

## Explanation

Efficiency refers to an estimator having the minimum variance among all unbiased estimators of a parameter. An efficient estimator makes the best use of information in the sample, producing estimates that cluster most tightly around the true value. The sample mean, for example, is an efficient estimator of the population mean for normally distributed data.

**Common mistakes:**
- Option A describes an exact estimator, which is impossible in practice
- Option C describes consistency, not efficiency
