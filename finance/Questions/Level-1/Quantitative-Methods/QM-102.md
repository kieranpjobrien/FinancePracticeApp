# Question

If the probability that a bond defaults is 3%, the probability that it does not default is:

## Options

- **A**: 3%
- **B**: 97%
- **C**: 0.03%

## Answer

B

## Explanation

The complement rule states that P(A) + P(not A) = 1

P(no default) = 1 - P(default)
P(no default) = 1 - 0.03
P(no default) = 0.97 = 97%

The probability of an event and its complement must sum to 1 (or 100%).

**Common mistakes:**
- Option A confuses the probability of default with non-default
- Option C incorrectly converts the percentage to a decimal twice
