# Question

A stock has a beta of 1.3. The risk-free rate is 3% and the expected market return is 10%. According to CAPM, the required return is:

## Options

- **A**: 10.0%
- **B**: 12.1%
- **C**: 13.0%

## Answer

B

## Explanation

Using CAPM: E(Ri) = Rf + βi[E(Rm) - Rf]

E(Ri) = 3% + 1.3(10% - 3%)
E(Ri) = 3% + 1.3(7%)
E(Ri) = 3% + 9.1%
E(Ri) = 12.1%

**Common mistakes:**
- Option A uses the market return instead of calculating the required return
- Option C multiplies beta by market return directly: 1.3 × 10% = 13%
