# Question

A paired t-test is most appropriate when:

## Options

- **A**: Comparing means from two independent populations
- **B**: Comparing means from the same subjects before and after a treatment
- **C**: Testing whether a sample variance equals a hypothesized value

## Answer

B

## Explanation

A paired (or dependent) t-test is used when observations are naturally paired or matched, such as before/after measurements on the same subjects, or matched pairs in an experiment. The test analyzes the differences within each pair, accounting for the correlation between paired observations. This is more powerful than an independent t-test when the pairing removes variability between subjects.

**Common mistakes:**
- Option A describes an independent samples t-test, which is appropriate for unrelated groups
- Option C describes a chi-square test for variance, not a t-test
