# Question

What is the present value of $1,000 to be received in 3 years if the discount rate is 6% per year?

## Options

- **A**: $839.62
- **B**: $840.00
- **C**: $943.40

## Answer

A

## Explanation

Present Value = Future Value / (1 + r)ⁿ

PV = $1,000 / (1.06)³
PV = $1,000 / 1.191016
PV = $839.62

The present value represents what a future sum is worth today, accounting for the time value of money.

**Common mistakes:**
- Option B results from rounding errors or approximation
- Option C uses n = 1 instead of n = 3, discounting for only one year
