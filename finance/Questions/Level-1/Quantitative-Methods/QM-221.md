# Question

A 95% confidence interval for the mean that does not include zero is equivalent to:

## Options

- **A**: Rejecting the null hypothesis that μ = 0 at the 5% significance level
- **B**: Failing to reject the null hypothesis that μ = 0 at the 5% level
- **C**: Rejecting at the 1% significance level

## Answer

A

## Explanation

There is a direct relationship between confidence intervals and hypothesis tests. If a (1-α)% confidence interval does not include the hypothesized value, we reject the null hypothesis at the α significance level. A 95% CI excluding zero means we reject H₀: μ = 0 at the 5% level because zero is not a plausible value for the parameter.

**Common mistakes:**
- Option B is the opposite; exclusion means rejection
- Option C is incorrect; the 95% CI corresponds to a 5% test, not 1%
