# Question

The linearity assumption in regression means that:

## Options

- **A**: The dependent variable must be linearly increasing over time
- **B**: The relationship between the independent and dependent variables is linear
- **C**: The residuals must follow a straight line

## Answer

B

## Explanation

The linearity assumption states that the expected value of the dependent variable is a linear function of the independent variable(s). The model specifies that Y changes by a constant amount (β) for each unit change in X. If the true relationship is nonlinear (curved), a linear regression will be misspecified and may produce biased estimates.

**Common mistakes:**
- Option A confuses time trends with the linearity assumption
- Option C misinterprets the assumption; residuals should have zero mean but are not expected to form a line
