# Question

An investment earns 10% before taxes. If the tax rate on investment income is 25%, the after-tax return is:

## Options

- **A**: 2.5%
- **B**: 7.5%
- **C**: 12.5%

## Answer

B

## Explanation

After-tax return = Pre-tax return × (1 - Tax rate)

After-tax return = 10% × (1 - 0.25)
After-tax return = 10% × 0.75
After-tax return = 7.5%

The investor keeps 75% of the return after paying 25% in taxes.

**Common mistakes:**
- Option A calculates the tax amount (10% × 25% = 2.5%) rather than the after-tax return
- Option C adds rather than subtracts the tax effect
