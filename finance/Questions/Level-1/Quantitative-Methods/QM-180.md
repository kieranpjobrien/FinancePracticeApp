# Question

An investment grows from $1,000 to $1,331 over 3 years. The annualized return is closest to:

## Options

- **A**: 10%
- **B**: 11%
- **C**: 33.1%

## Answer

A

## Explanation

Total return = ($1,331 - $1,000) / $1,000 = 33.1%

To annualize a multi-period return:
Annualized return = (Ending Value / Beginning Value)^(1/n) - 1
= ($1,331 / $1,000)^(1/3) - 1
= (1.331)^0.333 - 1
= 1.10 - 1
= 10%

**Common mistakes:**
- Option B results from calculation errors
- Option C uses the total return without annualizing
