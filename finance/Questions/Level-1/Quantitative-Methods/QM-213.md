# Question

Using simple interest, $1,000 invested at 6% per year for 3 years grows to:

## Options

- **A**: $1,060
- **B**: $1,180
- **C**: $1,191

## Answer

B

## Explanation

With simple interest, interest is calculated only on the original principal:

Interest = Principal × Rate × Time
Interest = $1,000 × 0.06 × 3 = $180

Future Value = Principal + Interest = $1,000 + $180 = $1,180

**Common mistakes:**
- Option A calculates interest for only one year
- Option C calculates compound interest: $1,000 × (1.06)³ = $1,191
