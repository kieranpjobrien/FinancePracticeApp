# Question

The Durbin-Watson statistic is used to test for:

## Options

- **A**: Heteroskedasticity in the residuals
- **B**: Serial correlation in the residuals
- **C**: Multicollinearity among independent variables

## Answer

B

## Explanation

The Durbin-Watson (DW) statistic tests for first-order serial correlation (autocorrelation) in regression residuals. It ranges from 0 to 4, with values near 2 indicating no serial correlation, values near 0 indicating positive serial correlation, and values near 4 indicating negative serial correlation. Serial correlation violates regression assumptions and affects the reliability of standard errors.

**Common mistakes:**
- Option A is tested using other methods like the Breusch-Pagan test
- Option C applies to multiple regression and is assessed using variance inflation factors (VIF)
