# Question

When calculating the variance of a three-asset portfolio, the total number of unique terms (including variances and covariances) in the formula is:

## Options

- **A**: 3
- **B**: 6
- **C**: 9

## Answer

B

## Explanation

For an n-asset portfolio, the variance formula includes:
- n variance terms (one for each asset)
- n(n-1)/2 unique covariance terms (or n(n-1)/2 pairs)

For n = 3:
- 3 variance terms: σ₁², σ₂², σ₃²
- 3 unique covariance terms: Cov(1,2), Cov(1,3), Cov(2,3)
- Total: 3 + 3 = 6 unique terms

Note: Each covariance appears twice in the full formula (2×w₁w₂Cov(1,2)), but they represent the same 3 unique pairwise relationships.

**Common mistakes:**
- Option A only counts variance terms, ignoring covariances
- Option C counts each covariance twice (as if Cov(1,2) and Cov(2,1) were different)
