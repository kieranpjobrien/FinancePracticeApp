# Question

An estimator is consistent if:

## Options

- **A**: Its expected value equals the true population parameter
- **B**: It converges to the true population parameter as sample size increases
- **C**: It has the smallest variance among all unbiased estimators

## Answer

B

## Explanation

Consistency is a large-sample property. An estimator is consistent if it converges in probability to the true parameter value as the sample size approaches infinity. This means with a large enough sample, the estimator will be arbitrarily close to the true value. Consistency does not require the estimator to be unbiased for any particular sample size—even a biased estimator can be consistent if the bias diminishes with sample size.

**Common mistakes:**
- Option A describes unbiasedness, not consistency; an estimator can be biased but consistent
- Option C describes efficiency (specifically, BLUE - Best Linear Unbiased Estimator), not consistency
