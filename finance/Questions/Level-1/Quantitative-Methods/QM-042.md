# Question

In the simple linear regression equation Y = a + bX, the coefficient 'b' represents:

## Options

- **A**: The predicted value of Y when X equals zero
- **B**: The expected change in Y for a one-unit change in X
- **C**: The correlation between X and Y

## Answer

B

## Explanation

In simple linear regression (Y = a + bX), the slope coefficient 'b' measures the expected change in the dependent variable (Y) for each one-unit increase in the independent variable (X). It represents the marginal effect of X on Y. The intercept 'a' represents the predicted value of Y when X equals zero.

**Common mistakes:**
- Option A describes the intercept 'a', not the slope 'b'
- Option C confuses slope with correlation; while related, they are different—correlation measures the strength of the linear relationship (standardized), while the slope measures the magnitude of change in original units
