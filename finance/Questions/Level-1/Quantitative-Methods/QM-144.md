# Question

A tornado diagram is used to:

## Options

- **A**: Display the distribution of Monte Carlo simulation results
- **B**: Rank input variables by their impact on the output
- **C**: Show the time series of returns

## Answer

B

## Explanation

A tornado diagram visualizes sensitivity analysis results by ranking input variables based on their impact on the output. Variables with the largest impact are shown at the top, creating a tornado shape. Each bar shows how much the output changes when that input varies from low to high values. This helps identify which assumptions most critically affect results.

**Common mistakes:**
- Option A describes a histogram or probability distribution of simulation outputs
- Option C describes a time series chart, not a tornado diagram
