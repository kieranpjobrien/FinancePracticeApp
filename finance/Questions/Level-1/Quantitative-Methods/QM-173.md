# Question

Antithetic variates is a variance reduction technique in Monte Carlo simulation that:

## Options

- **A**: Uses only the average of all simulated values
- **B**: Pairs each random draw with its complement to reduce sampling variance
- **C**: Eliminates the need for random number generation

## Answer

B

## Explanation

Antithetic variates creates negatively correlated pairs of simulated paths. For each random number U from a uniform distribution, the technique also uses (1-U), creating a mirror path. Since these paths are negatively correlated, their average has lower variance than independent paths. This improves precision without increasing the number of simulations.

**Common mistakes:**
- Option A describes simple averaging, not a specific variance reduction technique
- Option C is incorrect; the technique still uses random numbers but more efficiently
