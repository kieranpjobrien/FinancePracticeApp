# Question

In Monte Carlo simulation, a "trial" refers to:

## Options

- **A**: The first run used to calibrate the model
- **B**: A single complete run through the simulation generating one outcome
- **C**: The average of all simulation results

## Answer

B

## Explanation

Each trial (or iteration) in a Monte Carlo simulation generates one complete set of random values from the specified distributions and calculates the resulting output. Running thousands of trials produces a distribution of outcomes. Each trial is independent and contributes one data point to the final analysis.

**Common mistakes:**
- Option A describes calibration, which is separate from the simulation trials
- Option C describes summary statistics calculated after trials are complete
