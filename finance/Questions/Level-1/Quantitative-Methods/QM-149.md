# Question

Semivariance measures:

## Options

- **A**: Half of the total variance
- **B**: The variance of returns below the mean or a target return
- **C**: The average squared return

## Answer

B

## Explanation

Semivariance (or semi-deviation squared) calculates dispersion using only returns below a threshold—typically the mean or a minimum acceptable return. It focuses on downside risk by ignoring upside volatility. This is useful for investors who are primarily concerned with the risk of underperformance rather than total volatility.

Target Semivariance = Σ[min(Ri - Target, 0)]² / n

**Common mistakes:**
- Option A incorrectly interprets "semi" as meaning half of total variance
- Option C describes mean squared return, which is different from semivariance
