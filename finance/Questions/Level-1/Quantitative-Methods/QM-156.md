# Question

The symbol σ (sigma) typically represents:

## Options

- **A**: Sample standard deviation
- **B**: Population standard deviation
- **C**: Sample variance

## Answer

B

## Explanation

In statistics, Greek letters typically denote population parameters:
- σ (sigma) = population standard deviation
- μ (mu) = population mean
- σ² = population variance

Latin letters denote sample statistics:
- s = sample standard deviation
- x̄ = sample mean
- s² = sample variance

**Common mistakes:**
- Option A uses the sample notation 's' for sample standard deviation
- Option C describes variance, which would be σ² for population variance
