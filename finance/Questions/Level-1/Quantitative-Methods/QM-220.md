# Question

Mean absolute deviation (MAD) differs from standard deviation in that MAD:

## Options

- **A**: Squares the deviations before averaging
- **B**: Uses absolute values instead of squared deviations
- **C**: Is always larger than standard deviation

## Answer

B

## Explanation

MAD = Σ|Xi - mean| / n

MAD uses absolute values of deviations from the mean, while standard deviation squares the deviations. MAD is more intuitive (average distance from mean) and less sensitive to outliers because it doesn't square large deviations. However, standard deviation has better mathematical properties and is more commonly used.

**Common mistakes:**
- Option A describes standard deviation's approach
- Option C is incorrect; standard deviation is typically larger due to squaring
