# Question

A project requires an initial investment of $50,000 and has a present value of future cash flows of $60,000. The profitability index (PI) is:

## Options

- **A**: 0.20
- **B**: 1.00
- **C**: 1.20

## Answer

C

## Explanation

Profitability Index = PV of future cash flows / Initial investment

PI = $60,000 / $50,000 = 1.20

A PI greater than 1 indicates the project adds value (positive NPV). The PI of 1.20 means every dollar invested generates $1.20 in present value, with $0.20 in value creation.

**Common mistakes:**
- Option A calculates only the ratio of NPV to investment ($10,000 / $50,000)
- Option B would indicate a zero NPV project
