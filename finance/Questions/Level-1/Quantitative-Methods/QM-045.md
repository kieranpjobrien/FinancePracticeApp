# Question

In regression analysis, a residual represents the:

## Options

- **A**: Predicted value of Y from the regression equation
- **B**: Difference between the actual value of Y and the predicted value
- **C**: Average value of Y across all observations

## Answer

B

## Explanation

A residual (or error) is the difference between the actual observed value of Y and the value predicted by the regression equation: e = Y - Ŷ. Residuals measure how well the regression line fits individual data points. Positive residuals indicate the actual value exceeds the prediction, while negative residuals indicate the prediction exceeds the actual value. The sum of residuals in an OLS regression equals zero.

**Common mistakes:**
- Option A describes the fitted or predicted value (Ŷ), not the residual
- Option C describes the sample mean of Y, which is unrelated to residuals
