# Question

When using historical data to estimate future risk, rebalancing bias may occur if:

## Options

- **A**: The sample period is too short
- **B**: Historical weights differ from current or intended portfolio weights
- **C**: The data contains missing values

## Answer

B

## Explanation

Rebalancing bias arises when historical data is analyzed assuming portfolio weights were constant, but actual portfolios are periodically rebalanced. If using historical returns with assumed static weights that differ from how the portfolio was or will be managed, the resulting risk estimates may be inaccurate. This is particularly relevant when backtesting strategies that involve dynamic rebalancing.

**Common mistakes:**
- Option A describes time-period bias, not rebalancing bias
- Option C describes a data quality issue, not rebalancing bias
