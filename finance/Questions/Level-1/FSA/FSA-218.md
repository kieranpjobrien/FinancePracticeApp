# Question

A company repurchases $1,000,000 face value bonds for $960,000. The bonds have a carrying value of $980,000. The gain or loss on extinguishment is:

## Options

- **A**: $20,000 gain
- **B**: $40,000 gain
- **C**: $20,000 loss

## Answer

A

## Explanation

Gain/loss on extinguishment = Carrying value - Repurchase price = $980,000 - $960,000 = $20,000 gain. The carrying value (face value adjusted for unamortized discount/premium) is compared to the cash paid. Paying less than carrying value creates a gain; paying more creates a loss.

**Common mistakes:**
- Option B incorrectly uses face value instead of carrying value
- Option C reverses the gain/loss calculation
