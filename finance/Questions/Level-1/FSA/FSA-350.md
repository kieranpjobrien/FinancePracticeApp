# Question

Return on invested capital (ROIC) is preferred over ROA because it:

## Options

- **A**: Uses market values instead of book values
- **B**: Focuses on operating returns and invested capital, excluding non-operating items
- **C**: Always produces higher values than ROA

## Answer

B

## Explanation

ROIC = NOPAT / Invested capital, measuring operating profitability relative to operating capital employed. It excludes non-operating assets (excess cash, investments) and related income, providing a cleaner view of core operating efficiency. ROA includes all assets regardless of operating relevance.

**Common mistakes:**
- Option A is incorrect—ROIC typically uses book values
- Option C depends on company specifics—not always true
