# Question

A company has total assets of $500 million, total liabilities of $300 million, and 20 million shares outstanding. Book value per share is:

## Options

- **A**: $10.00
- **B**: $15.00
- **C**: $25.00

## Answer

A

## Explanation

Book value per share = (Total assets - Total liabilities) / Shares outstanding = ($500M - $300M) / 20M = $200M / 20M = $10.00 per share. Book value represents the accounting value of shareholders' equity allocated to each share.

**Common mistakes:**
- Option B divides liabilities by shares
- Option C divides assets by shares
