# Question

Under IFRS, the expected return on pension plan assets:

## Options

- **A**: Is a separate component of pension expense based on management estimates
- **B**: Equals the discount rate multiplied by plan assets (net interest approach)
- **C**: Is based on historical returns achieved

## Answer

B

## Explanation

IFRS uses net interest approach: expected return = Discount rate × Plan assets. The difference between actual and expected return is an actuarial gain/loss in OCI. US GAAP allows a separate expected return assumption, which may differ from the discount rate. This is a key IFRS/GAAP difference.

**Common mistakes:**
- Option A describes the US GAAP approach
- Option C is not used for expense calculation under either standard
