# Question

Off-balance-sheet financing arrangements are concerning because they may:

## Options

- **A**: Improve a company's actual financial position
- **B**: Understate liabilities and overstate financial ratios
- **C**: Increase reported assets and equity

## Answer

B

## Explanation

Off-balance-sheet arrangements (like certain operating leases pre-2019, special purpose entities, or synthetic leases) kept obligations off the balance sheet, understating liabilities. This inflated ratios like debt-to-equity and return on assets. Current standards have reduced but not eliminated off-balance-sheet concerns.

**Common mistakes:**
- Option A confuses appearance with reality
- Option C describes on-balance-sheet effects
