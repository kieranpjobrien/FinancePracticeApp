# Question

A company has current assets of $500,000, inventory of $150,000, and current liabilities of $250,000. The quick ratio is:

## Options

- **A**: 1.4
- **B**: 2.0
- **C**: 2.6

## Answer

A

## Explanation

Quick ratio = (Current assets - Inventory) / Current liabilities = ($500,000 - $150,000) / $250,000 = $350,000 / $250,000 = 1.4. This acid-test ratio excludes inventory (less liquid) from current assets, providing a more stringent liquidity measure than the current ratio (2.0 here).

**Common mistakes:**
- Option B is the current ratio: $500,000 / $250,000 = 2.0
- Option C results from calculation error
