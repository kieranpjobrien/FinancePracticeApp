# Question

Stock options granted to employees are measured for compensation expense at:

## Options

- **A**: Intrinsic value at grant date
- **B**: Fair value at grant date
- **C**: Fair value at exercise date

## Answer

B

## Explanation

Under both IFRS and US GAAP, employee stock options are measured at fair value on the grant date. This value is expensed over the vesting period. Fair value is typically estimated using option pricing models (Black-Scholes, binomial). Subsequent changes in fair value don't affect expense for equity-settled awards.

**Common mistakes:**
- Option A was the old APB 25 approach, now superseded
- Option C would result in expense volatility from market fluctuations
