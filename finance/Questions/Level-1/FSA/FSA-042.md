# Question

The LIFO reserve represents:

## Options

- **A**: The difference between LIFO inventory and FIFO inventory
- **B**: Inventory set aside for future production
- **C**: The cumulative write-down for obsolete inventory

## Answer

A

## Explanation

LIFO reserve = FIFO inventory - LIFO inventory. It's disclosed by companies using LIFO and represents the cumulative difference between the two methods. Analysts use it to convert LIFO financials to FIFO for comparison: FIFO inventory = LIFO inventory + LIFO reserve. The reserve generally grows when prices rise.

**Common mistakes:**
- Option B misinterprets "reserve" as physical stock
- Option C describes inventory impairment, not LIFO reserve
