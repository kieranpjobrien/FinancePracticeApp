# Question

Tracking error is most useful for evaluating:

## Options

- **A**: Absolute return strategies with no benchmark
- **B**: Index funds and strategies with benchmark targets
- **C**: Market timing strategies

## Answer

B

## Explanation

Tracking error application:

**Definition:**
- Standard deviation of excess returns
- Measures deviation from benchmark
- Active risk measure

**Most useful for:**
- Index funds (low tracking error target)
- Enhanced index strategies
- Benchmark-aware active management
- Core-satellite portfolios

**Interpretation:**
- Index fund: < 0.5% tracking error
- Enhanced index: 0.5-2%
- Active: 2-8%+

**Less relevant for:**
- Absolute return funds (no benchmark)
- Hedge funds (may have unique mandates)
- Market timing (benchmark changes)

**Calculation:**
TE = σ(Rp - Rb)

**Common mistakes:**
- Option A: Absolute return has no benchmark to track
- Option C: Timing strategies don't track consistently
