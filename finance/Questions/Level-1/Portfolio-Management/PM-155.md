# Question

The characteristic line in the single-index model plots:

## Options

- **A**: Expected return against standard deviation
- **B**: Security returns against market returns
- **C**: Portfolio weights against risk levels

## Answer

B

## Explanation

Characteristic line:

**Definition:**
- Regression line of security returns on market returns
- Used in single-index (market) model

**Equation:**
Ri = αi + βiRm + εi

**Components:**
- Slope = Beta (systematic risk)
- Intercept = Alpha (abnormal return)
- Residuals = Firm-specific returns

**Construction:**
- Y-axis: Security returns
- X-axis: Market returns
- Plot historical observations
- Fit regression line

**Uses:**
- Estimate beta
- Decompose returns
- Risk analysis

**Common mistakes:**
- Option A: This describes CML or efficient frontier
- Option C: Not related to characteristic line
