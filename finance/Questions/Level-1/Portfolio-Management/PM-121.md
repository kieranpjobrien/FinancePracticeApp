# Question

The feedback phase of portfolio management includes:

## Options

- **A**: Developing the investment policy statement
- **B**: Selecting individual securities for the portfolio
- **C**: Monitoring the portfolio and evaluating performance

## Answer

C

## Explanation

Portfolio management process - Feedback phase:

**Key activities:**
1. **Monitoring:**
   - Track investor circumstances
   - Monitor capital market conditions
   - Review portfolio holdings and weights

2. **Rebalancing:**
   - Compare actual vs. target allocation
   - Execute trades to restore targets
   - Consider transaction costs and taxes

3. **Performance evaluation:**
   - Calculate returns
   - Compare to benchmarks
   - Attribution analysis

4. **IPS review:**
   - Update for changed circumstances
   - Revise as needed

**Common mistakes:**
- Option A: Developing IPS is part of Planning phase
- Option C: Security selection is part of Execution phase
