# Question

The investment universe for a portfolio is best described as:

## Options

- **A**: All securities the manager currently holds
- **B**: All investable assets that the manager may consider for the portfolio
- **C**: Only the benchmark constituents

## Answer

B

## Explanation

Investment universe:

**Definition:**
- Set of all securities/assets eligible for investment
- Defined by IPS constraints and manager mandate
- Broader than current holdings

**Determined by:**
- Asset class restrictions
- Geographic limitations
- Quality/rating requirements
- Liquidity minimums
- Ethical/ESG screens
- Regulatory constraints

**Examples:**
- "All U.S. large-cap equities"
- "Investment-grade corporate bonds"
- "Global developed market stocks excluding tobacco"

**Importance:**
- Defines opportunity set
- Limits manager discretion appropriately
- Basis for benchmark selection

**Common mistakes:**
- Option A: Current holdings are subset of universe
- Option C: Universe often broader than benchmark
