# Question

A portfolio consists of 40% Stock A with an expected return of 10% and 60% Stock B with an expected return of 14%. The expected return of the portfolio is closest to:

## Options

- **A**: 12.0%
- **B**: 12.4%
- **C**: 13.6%

## Answer

B

## Explanation

Portfolio expected return is the weighted average of individual asset returns:

E(Rp) = wA × E(RA) + wB × E(RB)

Where:
- wA = 0.40, E(RA) = 10%
- wB = 0.60, E(RB) = 14%

E(Rp) = (0.40 × 10%) + (0.60 × 14%)
E(Rp) = 4% + 8.4%
E(Rp) = 12.4%

**Common mistakes:**
- Option A: Simple average of returns, ignoring weights
- Option C: Reversed the weights (60% × 10% + 40% × 14%)
