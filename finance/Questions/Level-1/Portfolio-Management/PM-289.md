# Question

Model risk arises from:

## Options

- **A**: Using too few securities in a portfolio
- **B**: Errors or limitations in quantitative models used for decisions
- **C**: Trading too frequently

## Answer

B

## Explanation

Model risk:

**Definition:**
- Risk from model deficiencies
- Incorrect assumptions
- Implementation errors
- Misapplication of models

**Sources:**
- Wrong model specification
- Parameter estimation errors
- Data quality issues
- Coding mistakes
- Changed market conditions

**Examples:**
- VaR underestimating tail risk
- Black-Scholes during volatility spikes
- Correlation assumptions failing
- Factor models missing risks

**Consequences:**
- Underestimated risk
- Mispriced securities
- Unexpected losses
- Failed hedges

**Mitigation:**
- Model validation
- Stress testing
- Multiple models
- Expert judgment overlay

**Common mistakes:**
- Option A: This is concentration risk
- Option C: This is transaction cost issue
