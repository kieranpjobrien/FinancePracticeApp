# Question

M-squared (M²) is a risk-adjusted performance measure that:

## Options

- **A**: Measures excess return per unit of systematic risk
- **B**: Expresses performance as the difference from the market return at the same total risk level
- **C**: Calculates the geometric mean return over multiple periods

## Answer

B

## Explanation

M-squared (Modigliani-Modigliani measure):

- Adjusts portfolio to have same total risk as market
- Compares the adjusted portfolio's return to market return
- Result expressed in percentage terms (easy to interpret)

Calculation:
1. Leverage or de-leverage portfolio to match market's standard deviation
2. Calculate return of this risk-adjusted portfolio
3. M² = Risk-adjusted return - Market return

Interpretation:
- Positive M²: Outperformed on risk-adjusted basis
- Related to Sharpe ratio: M² = (Sharpe ratio difference) × σmarket

**Common mistakes:**
- Option A: This describes the Treynor ratio
- Option C: This describes geometric mean return, not a risk-adjusted measure
