# Question

In CAPM, required return represents:

## Options

- **A**: The return an investor hopes to achieve
- **B**: The return that compensates for the security's systematic risk
- **C**: Historical average return

## Answer

B

## Explanation

Required return in CAPM:

**Definition:**
- Return demanded given security's beta
- Compensation for systematic risk
- Fair return based on risk level

**CAPM formula:**
Required Return = Rf + β(Rm - Rf)

**Components:**
- Rf: Time value of money
- β(Rm - Rf): Risk premium for systematic risk

**Uses:**
- Valuation (discount rate)
- Investment evaluation
- Cost of equity

**Relationship to expected return:**
- If Expected > Required: Undervalued
- If Expected < Required: Overvalued

**Common mistakes:**
- Option A: "Hopes" is desired, not required
- Option C: Required is forward-looking
