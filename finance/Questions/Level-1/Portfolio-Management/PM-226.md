# Question

A security that plots above the Security Market Line is:

## Options

- **A**: Overvalued with negative alpha
- **B**: Undervalued with positive alpha
- **C**: Fairly valued on the SML

## Answer

B

## Explanation

SML and valuation:

**Above SML:**
- Expected return > CAPM required return
- Positive alpha
- Undervalued (price too low)
- Buy signal

**Below SML:**
- Expected return < CAPM required return
- Negative alpha
- Overvalued (price too high)
- Sell/avoid signal

**On SML:**
- Expected return = Required return
- Zero alpha
- Fairly valued

**Logic:**
- Above SML means more return than justified by beta
- In equilibrium, price would rise, pushing return down to SML
- Currently underpriced → opportunity

**Common mistakes:**
- Option A: Above SML means undervalued, not overvalued
- Option C: On SML is fair value, not above
