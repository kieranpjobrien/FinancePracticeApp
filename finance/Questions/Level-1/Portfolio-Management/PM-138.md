# Question

A portfolio has a return of 12%, a risk-free rate of 3%, and a downside deviation (using 3% as minimum acceptable return) of 10%. The Sortino ratio is:

## Options

- **A**: 0.45
- **B**: 0.90
- **C**: 1.20

## Answer

B

## Explanation

Sortino ratio:

**Formula:**
Sortino Ratio = (Rp - MAR) / Downside Deviation

Where:
- Rp = Portfolio return = 12%
- MAR = Minimum Acceptable Return = 3% (often Rf)
- Downside deviation = 10%

**Calculation:**
Sortino = (12% - 3%) / 10%
Sortino = 9% / 10%
Sortino = 0.90

**Comparison to Sharpe:**
- Sharpe uses total standard deviation
- Sortino uses only downside deviation
- Sortino better for asymmetric return distributions
- Sortino doesn't penalize upside volatility

**Common mistakes:**
- Option A: May use wrong inputs or calculation
- Option C: May use different MAR or deviation
