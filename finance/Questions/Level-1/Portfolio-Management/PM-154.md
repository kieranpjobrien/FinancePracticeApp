# Question

CAPM is an equilibrium model that describes:

## Options

- **A**: How to time the market for maximum returns
- **B**: The relationship between systematic risk and expected return
- **C**: How to identify undervalued securities using technical analysis

## Answer

B

## Explanation

CAPM fundamentals:

**Key insight:**
- Only systematic (market) risk is priced
- Expected return is linear function of beta
- Unsystematic risk earns no premium

**The equation:**
E(Ri) = Rf + βi[E(Rm) - Rf]

**Equilibrium meaning:**
- In equilibrium, all securities plot on SML
- Mispricings are quickly corrected
- Prices adjust until expected returns match required returns

**Implications:**
- Higher beta → higher expected return
- No reward for diversifiable risk
- Market portfolio is efficient

**Common mistakes:**
- Option A: CAPM doesn't provide timing signals
- Option C: CAPM is fundamental/equilibrium, not technical analysis
