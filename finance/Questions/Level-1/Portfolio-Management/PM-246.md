# Question

Multi-factor models extend CAPM by:

## Options

- **A**: Using only beta as the risk measure
- **B**: Incorporating additional sources of systematic risk
- **C**: Eliminating the need for any risk factors

## Answer

B

## Explanation

Multi-factor model extensions:

**CAPM limitation:**
- Only market factor
- Doesn't explain all return variation
- Anomalies unexplained

**Multi-factor approach:**
- Multiple systematic risk sources
- Each factor has risk premium
- Better explains returns

**Common factors:**
- Market (CAPM)
- Size (SMB)
- Value (HML)
- Momentum
- Quality
- Volatility

**Model form:**
E(R) = Rf + β₁F₁ + β₂F₂ + ... + βₖFₖ

**Benefits:**
- Better risk attribution
- Explains more return variation
- Identifies factor exposures

**Common mistakes:**
- Option A: That's single-factor CAPM
- Option C: Models still need factors
