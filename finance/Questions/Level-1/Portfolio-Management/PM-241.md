# Question

Security selection in portfolio management refers to:

## Options

- **A**: Determining overall asset class weights
- **B**: Choosing specific securities within each asset class
- **C**: Setting the investment time horizon

## Answer

B

## Explanation

Security selection vs. asset allocation:

**Asset allocation:**
- Determines asset class weights
- Strategic (long-term) and tactical (short-term)
- Stocks vs. bonds vs. alternatives
- Primary driver of portfolio risk/return

**Security selection:**
- Choosing individual securities
- Stocks within equity allocation
- Bonds within fixed income
- Aims to outperform asset class benchmark

**Two-step process:**
1. Allocate across asset classes (top-down)
2. Select securities within classes (bottom-up)

**Common mistakes:**
- Option A: This is asset allocation
- Option C: Time horizon is an IPS constraint
