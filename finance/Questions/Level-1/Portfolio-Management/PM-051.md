# Question

A portfolio manager generated returns of 12% versus a benchmark return of 10%. The standard deviation of the excess returns (tracking error) was 4%. The information ratio is closest to:

## Options

- **A**: 0.30
- **B**: 0.50
- **C**: 3.00

## Answer

B

## Explanation

The Information Ratio measures active return per unit of active risk:

IR = (Rp - Rb) / σ(Rp - Rb)

Where:
- Rp - Rb = Active return = 12% - 10% = 2%
- σ(Rp - Rb) = Tracking error = 4%

IR = 2% / 4% = 0.50

Interpretation:
- Measures manager skill in generating excess returns
- IR of 0.50 is generally considered good
- IR > 0.75 is excellent
- Higher IR = more consistent outperformance

**Common mistakes:**
- Option A: May use incorrect values in calculation
- Option C: Divides returns instead of using the formula correctly
