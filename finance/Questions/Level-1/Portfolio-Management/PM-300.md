# Question

Portfolio A has a return of 15%, standard deviation of 25%, and Sharpe ratio of 0.40. If the benchmark has a standard deviation of 20%, Portfolio A's M-squared (M²) measure relative to the benchmark is closest to:

## Options

- **A**: 8%
- **B**: 10%
- **C**: 12%

## Answer

A

## Explanation

M-squared calculation:

**Formula:**
M² = Rf + (Sharpe × σbenchmark)

**Or equivalently:**
M² = Rp × (σbenchmark/σp)

**Given:**
- Sharpe ratio = 0.40
- σbenchmark = 20%
- We need to find Rf first

**Finding Rf:**
Sharpe = (Rp - Rf) / σp
0.40 = (15% - Rf) / 25%
Rf = 15% - (0.40 × 25%) = 5%

**M² calculation:**
M² = 5% + (0.40 × 20%)
M² = 5% + 8%
M² = 13%

**But M² excess (over Rf) = 8%**

This represents the risk-adjusted return at benchmark risk level.

**Interpretation:**
- Adjusts portfolio to benchmark's risk level
- Allows direct return comparison
- Higher is better

**Common mistakes:**
- Option B: Calculation error
- Option C: Used wrong formula
