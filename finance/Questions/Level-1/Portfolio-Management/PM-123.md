# Question

Performance attribution analysis that breaks returns into allocation and selection effects is designed to:

## Options

- **A**: Predict future portfolio returns
- **B**: Identify sources of active return relative to benchmark
- **C**: Calculate the portfolio's absolute return

## Answer

B

## Explanation

Performance attribution:

**Purpose:**
- Decompose active return (portfolio minus benchmark)
- Identify sources of outperformance or underperformance
- Evaluate manager skill in different areas

**Main effects:**
1. **Allocation effect:**
   - Value added from overweighting outperforming sectors
   - Underweighting underperforming sectors
   - Measures sector/asset class timing skill

2. **Selection effect:**
   - Value added from picking securities within sectors
   - Compares portfolio holdings vs. benchmark holdings
   - Measures security selection skill

3. **Interaction effect:**
   - Combined impact of allocation and selection
   - May be included or distributed

**Common mistakes:**
- Option A: Attribution is backward-looking, not predictive
- Option C: Attribution explains relative, not absolute returns
