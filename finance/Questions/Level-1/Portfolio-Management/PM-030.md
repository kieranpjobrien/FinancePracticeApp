# Question

A portfolio has a value of $10 million, an expected annual return of 8%, and an annual standard deviation of 15%. Using the parametric method, the 5% annual VaR is closest to (z-score for 5% = 1.65):

## Options

- **A**: $1.68 million
- **B**: $1.98 million
- **C**: $2.48 million

## Answer

A

## Explanation

The parametric (variance-covariance) VaR formula is:

VaR = Portfolio Value × (Expected Return - z × Standard Deviation)

For 5% VaR (95% confidence):
- z = 1.65 (left-tail critical value)
- Expected return = 8% = 0.08
- Standard deviation = 15% = 0.15

VaR = $10,000,000 × (0.08 - 1.65 × 0.15)
VaR = $10,000,000 × (0.08 - 0.2475)
VaR = $10,000,000 × (-0.1675)
VaR = -$1,675,000

The VaR is approximately $1.68 million (reported as absolute value).

This means there is a 5% probability of losing at least $1.68 million over one year.

**Common mistakes:**
- Option B: May use incorrect z-score or calculation errors
- Option C: May ignore the expected return component
