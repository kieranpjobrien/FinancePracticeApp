# Question

Portfolio rebalancing is the process of:

## Options

- **A**: Selecting new securities to replace underperforming ones
- **B**: Realigning portfolio weights back to strategic target allocations
- **C**: Increasing portfolio risk to enhance expected returns

## Answer

B

## Explanation

Portfolio rebalancing:

**Definition:**
- Returning portfolio weights to strategic asset allocation targets
- Counteracts drift caused by differential asset returns

**Process:**
1. Market movements cause drift from targets
2. Compare current weights to policy weights
3. Buy underweight assets, sell overweight assets
4. Restore target allocation

**Benefits:**
- Maintains intended risk profile
- Enforces disciplined "buy low, sell high"
- Prevents unintended style drift

**Costs:**
- Transaction costs
- Tax consequences
- Opportunity cost if momentum persists

**Common mistakes:**
- Option A: This describes security selection, not rebalancing
- Option C: Rebalancing maintains, not increases, target risk level
