# Question

Standard deviation is preferred to variance as a risk measure because:

## Options

- **A**: Standard deviation is always larger than variance
- **B**: Standard deviation is in the same units as returns
- **C**: Standard deviation ignores outliers

## Answer

B

## Explanation

Standard deviation vs. variance:

**Variance (σ²):**
- Average of squared deviations from mean
- Units are squared (e.g., %²)
- Not directly interpretable

**Standard deviation (σ):**
- Square root of variance
- Same units as returns (e.g., %)
- Directly interpretable

**Example:**
- Variance = 0.04 (or 4%²) - hard to interpret
- Standard deviation = 0.20 (or 20%) - meaningful

**Interpretation:**
- Approximately 68% of returns within ±1 std dev
- Approximately 95% of returns within ±2 std dev (normal distribution)

**Common mistakes:**
- Option A: Standard deviation < variance when variance > 1 (in decimals)
- Option C: Neither ignores outliers; both are affected by extreme values
