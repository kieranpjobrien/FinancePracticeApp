# Question

Implementation shortfall measures:

## Options

- **A**: The difference between portfolio return and benchmark return
- **B**: The total cost of implementing an investment decision from idea to execution
- **C**: The management fee charged by portfolio managers

## Answer

B

## Explanation

Implementation shortfall:

**Definition:**
- Difference between theoretical paper return and actual return
- Total cost of execution including all frictions
- Also called "slippage"

**Components:**
1. Explicit costs: Commissions, fees
2. Realized profit/loss: Execution price vs. decision price
3. Delay costs: Price drift during execution
4. Missed trade opportunity cost: Unfilled orders

**Formula:**
IS = Paper Return - Actual Return

**Uses:**
- Comprehensive trading cost measure
- Evaluate execution quality
- Compare brokers/algorithms

**Common mistakes:**
- Option A: This describes active return, not IS
- Option C: Management fees are separate from execution costs
