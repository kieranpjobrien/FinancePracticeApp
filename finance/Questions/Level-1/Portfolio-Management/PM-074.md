# Question

A portfolio has the following quarterly excess returns vs. its benchmark: +2%, -1%, +3%, +1%. The annualized tracking error is closest to:

## Options

- **A**: 1.71%
- **B**: 3.42%
- **C**: 6.83%

## Answer

B

## Explanation

Tracking error = Standard deviation of excess returns, annualized

Step 1: Calculate mean excess return
Mean = (2% - 1% + 3% + 1%) / 4 = 1.25%

Step 2: Calculate variance
Deviations: (2-1.25)², (-1-1.25)², (3-1.25)², (1-1.25)²
= 0.5625 + 5.0625 + 3.0625 + 0.0625 = 8.75
Variance = 8.75 / (4-1) = 2.917
Std Dev (quarterly) = √2.917 = 1.71%

Step 3: Annualize (multiply by √4 for quarterly data)
Tracking error = 1.71% × 2 = 3.42%

**Common mistakes:**
- Option A: Quarterly tracking error, not annualized
- Option C: May have incorrectly annualized or used population std dev
