# Question

On a graph with expected return on the y-axis and risk on the x-axis, an indifference curve slopes:

## Options

- **A**: Downward because investors prefer less return
- **B**: Upward because investors require more return for more risk
- **C**: Horizontal because risk doesn't matter

## Answer

B

## Explanation

Indifference curves:

**Shape:**
- Upward sloping
- Convex (curved) to x-axis
- Higher curves = higher utility

**Interpretation:**
- Points on same curve give equal utility
- Moving up: more return, same risk = higher utility
- Moving right: more risk, same return = lower utility
- To stay indifferent, need more return for more risk

**Risk aversion impact:**
- More risk-averse = steeper curves
- Less risk-averse = flatter curves
- Risk-neutral = horizontal curves

**Optimal portfolio:**
- Where indifference curve tangent to efficient frontier
- Highest utility achievable

**Common mistakes:**
- Option A: Downward slope would imply dislike of return
- Option C: Horizontal implies risk neutrality
