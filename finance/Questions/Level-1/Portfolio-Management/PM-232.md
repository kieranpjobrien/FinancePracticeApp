# Question

Over multiple periods, compound (geometric) average return is typically:

## Options

- **A**: Higher than arithmetic average return
- **B**: Equal to arithmetic average return
- **C**: Lower than arithmetic average return

## Answer

C

## Explanation

Arithmetic vs. geometric returns:

**Relationship:**
Geometric ≤ Arithmetic

Equal only when all returns are identical.

**Example:**
- Year 1: +50%, Year 2: -50%
- Arithmetic: (50% - 50%) / 2 = 0%
- Geometric: √(1.5 × 0.5) - 1 = √0.75 - 1 = -13.4%

**Why geometric is lower:**
- Volatility drag
- Losses hurt more than gains help
- Starting point matters

**When to use each:**
- Geometric: Actual wealth accumulation
- Arithmetic: Expected single-period return

**Common mistakes:**
- Option A: Geometric is always ≤ arithmetic
- Option B: Only equal when no volatility
