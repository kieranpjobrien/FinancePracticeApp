# Question

Survivorship bias in mutual fund performance data:

## Options

- **A**: Understates average historical returns
- **B**: Overstates average historical returns by excluding failed funds
- **C**: Has no effect on performance analysis

## Answer

B

## Explanation

Survivorship bias:

**Definition:**
- Only surviving funds included in analysis
- Failed/merged funds excluded
- Creates upward bias in results

**Mechanism:**
- Poor performers close or merge
- Only successful funds remain
- Historical averages based on survivors only

**Impact:**
- Average returns overstated
- Risk understated
- Manager skill appears better
- May be 1-2% per year effect

**Mitigation:**
- Use survivorship-bias-free databases
- Include dead funds
- Adjust for known bias

**Common mistakes:**
- Option A: Bias overstates, not understates returns
- Option C: Bias significantly affects analysis
