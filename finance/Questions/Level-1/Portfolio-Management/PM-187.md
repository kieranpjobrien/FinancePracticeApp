# Question

An inefficient portfolio is one that:

## Options

- **A**: Lies on the efficient frontier
- **B**: Can be improved by either increasing return for same risk or decreasing risk for same return
- **C**: Has the highest possible Sharpe ratio

## Answer

B

## Explanation

Inefficient portfolios:

**Definition:**
- Not on the efficient frontier
- "Dominated" by efficient portfolios
- Suboptimal risk-return trade-off

**How to identify:**
- Same risk, but lower return exists on frontier
- Same return, but lower risk exists on frontier
- Both conditions indicate inefficiency

**Causes:**
- Poor diversification
- Suboptimal weights
- Holding redundant positions
- Ignoring correlations

**Improvement paths:**
- Move up: Same risk, higher return
- Move left: Same return, lower risk
- Move northwest: Better risk-return

**Common mistakes:**
- Option A: Efficient frontier portfolios are efficient by definition
- Option C: Highest Sharpe ratio is optimal (tangency portfolio)
