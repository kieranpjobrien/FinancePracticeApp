# Question

The profitability index is most useful when:

## Options

- **A**: Projects are mutually exclusive
- **B**: Capital is rationed and projects must be ranked by efficiency
- **C**: IRR is unavailable

## Answer

B

## Explanation

Profitability Index = PV of future cash flows / Initial investment. It measures value per dollar invested, useful when limited capital requires prioritizing projects by efficiency rather than total NPV. For mutually exclusive projects, NPV is preferred. PI > 1.0 indicates positive NPV.

**Common mistakes:**
- Option A is when NPV should be used for absolute value comparison
- Option C is unrelated—both metrics are available from the same data
