# Question

Real GDP differs from nominal GDP in that real GDP:

## Options

- **A**: Includes the effects of price changes
- **B**: Adjusts for inflation to measure actual output changes
- **C**: Is always higher than nominal GDP

## Answer

B

## Explanation

Real GDP measures economic output in constant prices, removing the effect of inflation to show actual changes in production volume. Nominal GDP measures output in current prices, which can increase due to either higher output or higher prices. Comparing real GDP over time shows genuine economic growth, while nominal GDP changes can be misleading during inflationary periods.

**Common mistakes:**
- Option A describes nominal GDP
- Option C is only true during deflation; normally real GDP is lower
