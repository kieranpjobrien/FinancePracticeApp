# Question

A trade-weighted exchange rate index measures:

## Options

- **A**: The exchange rate against only the US dollar
- **B**: A currency's value against a basket of trading partner currencies, weighted by trade shares
- **C**: The nominal interest rate differential between countries

## Answer

B

## Explanation

A trade-weighted (or effective) exchange rate measures a currency's value against multiple trading partners, weighted by their share of trade. This provides a more comprehensive view of currency competitiveness than any bilateral rate. If a currency appreciates against major trading partners, the trade-weighted index rises, indicating reduced export competitiveness overall.

**Common mistakes:**
- Option A describes a single bilateral rate
- Option C describes an interest rate concept, not an exchange rate measure
