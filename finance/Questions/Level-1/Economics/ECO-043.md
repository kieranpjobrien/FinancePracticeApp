# Question

Given: USD/EUR = 1.10 and USD/GBP = 1.30. The GBP/EUR cross rate is closest to:

## Options

- **A**: 0.85
- **B**: 1.18
- **C**: 1.43

## Answer

A

## Explanation

To find GBP/EUR (how many GBP per EUR):

GBP/EUR = (USD/EUR) / (USD/GBP)
GBP/EUR = 1.10 / 1.30
GBP/EUR = 0.846 ≈ 0.85

This means one euro buys 0.85 British pounds.

**Common mistakes:**
- Option B results from dividing in the wrong direction
- Option C results from multiplying instead of dividing correctly
