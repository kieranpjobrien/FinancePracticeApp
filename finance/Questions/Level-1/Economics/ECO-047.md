# Question

The USD/JPY rate moves from 110 to 121. The percentage change in the yen's value is closest to:

## Options

- **A**: The yen appreciated by 10%
- **B**: The yen depreciated by 10%
- **C**: The yen depreciated by 9.1%

## Answer

C

## Explanation

When USD/JPY rises, it takes more yen to buy a dollar, meaning the yen has depreciated.

To calculate yen depreciation from its perspective:
% change = (1/121 - 1/110) / (1/110)
= (0.00826 - 0.00909) / 0.00909
= -0.0913 = -9.1%

The yen depreciated by approximately 9.1%.

**Common mistakes:**
- Option A reverses the direction of change
- Option B uses the wrong calculation method (110/121 - 1 = -9.1%, not -10%)
