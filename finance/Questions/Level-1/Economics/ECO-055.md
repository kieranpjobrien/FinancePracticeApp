# Question

In the short run, a firm should shut down production if:

## Options

- **A**: Price is below average total cost
- **B**: Price is below average variable cost
- **C**: The firm is making any loss

## Answer

B

## Explanation

A firm should shut down in the short run only if price falls below average variable cost (AVC). When P < AVC, the firm cannot even cover its variable costs, and continuing to operate increases losses beyond fixed costs. If P ≥ AVC but P < ATC, the firm should continue operating because revenue covers variable costs and contributes toward fixed costs, minimizing losses.

**Common mistakes:**
- Option A would cause premature shutdown—firms can operate at a loss if covering variable costs
- Option C ignores that operating losses may be smaller than shutdown losses
