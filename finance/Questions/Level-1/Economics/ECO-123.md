# Question

The Lerner Index measures market power as (P - MC)/P. If a firm charges $50 and marginal cost is $30, the Lerner Index is:

## Options

- **A**: 0.40
- **B**: 0.60
- **C**: 1.67

## Answer

A

## Explanation

Lerner Index = (P - MC) / P = ($50 - $30) / $50 = $20 / $50 = 0.40

The index ranges from 0 (perfect competition, P = MC) to approaching 1 (high market power). A value of 0.40 indicates significant pricing power—the firm marks up 40% above marginal cost. The Lerner Index relates inversely to demand elasticity.

**Common mistakes:**
- Option B reverses the calculation (MC/P or similar error)
- Option C calculates P/MC rather than the Lerner Index formula
