# Question

An analyst changes a recommendation from buy to hold. Standard V(B) requires:

## Options

- **A**: Explaining the reasons for the change to clients
- **B**: No explanation needed if the current recommendation is clear
- **C**: Only explaining if the client asks

## Answer

A

## Explanation

Standard V(B) requires communication of factors that may affect investment values, including changes in recommendations. Clients should understand why the analyst's view has shifted to make informed decisions about their holdings.

**Common mistakes:**
- Option B ignores the need to explain recommendation changes
- Option C makes transparency contingent on client inquiry
