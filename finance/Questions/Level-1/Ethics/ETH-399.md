# Question

A manager allocates shares from an oversubscribed IPO to accounts based on account size, giving larger accounts proportionally more shares. This practice:

## Options

- **A**: Violates Standard III(B) because all clients should receive equal shares
- **B**: Is consistent with Standard III(B) if the policy is disclosed and applied fairly
- **C**: Is prohibited because IPO allocations must be random

## Answer

B

## Explanation

Standard III(B) requires fair, not equal, treatment. Pro-rata allocation based on account size can be fair if the policy is consistently applied and disclosed. The key is that the methodology does not favor certain clients over others inappropriately.

**Common mistakes:**
- Option A confuses fair with equal treatment
- Option C imposes a non-existent requirement for random allocation
