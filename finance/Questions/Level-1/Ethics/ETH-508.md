# Question

Documentation of suitability analysis is important because:

## Options

- **A**: It is only required by regulators
- **B**: It demonstrates due diligence and supports the basis for recommendations
- **C**: It protects the analyst from all liability

## Answer

B

## Explanation

Standard III(C) suitability requires reasonable basis for recommendations. Documentation demonstrates the analytical process, supports that recommendations were appropriate, and provides evidence of due diligence.

**Common mistakes:**
- Option A understates the ethical importance
- Option C overstates the legal protection documentation provides
