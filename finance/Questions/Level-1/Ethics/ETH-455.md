# Question

When presenting performance, a manager shows different composites to different prospects based on which performed best. This practice:

## Options

- **A**: Is good marketing by showing relevant success stories
- **B**: Violates Standard III(D) by selectively presenting misleading information
- **C**: Is acceptable if all shown composites are real

## Answer

B

## Explanation

Standard III(D) requires fair, accurate, and complete performance presentation. Selecting composites to show based on performance rather than relevance to the prospect's investment needs creates a misleading picture.

**Common mistakes:**
- Option A confuses selective presentation with legitimate marketing
- Option C ignores that true but selectively presented data can mislead
