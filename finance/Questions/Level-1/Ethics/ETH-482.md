# Question

A manager compares fund performance to an inappropriate benchmark that makes results look favorable. Standard III(D) requires:

## Options

- **A**: Using any benchmark as long as it is disclosed
- **B**: Using appropriate benchmarks that allow fair evaluation of performance
- **C**: No benchmark comparison is required

## Answer

B

## Explanation

Standard III(D) requires fair and accurate performance presentation. Benchmark selection should enable meaningful comparison. Choosing inappropriate benchmarks to flatter results creates a misleading impression.

**Common mistakes:**
- Option A accepts any disclosed benchmark regardless of appropriateness
- Option C ignores the value of benchmark comparisons
