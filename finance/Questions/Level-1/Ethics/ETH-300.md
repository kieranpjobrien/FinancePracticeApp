# Question

A firm wants to use a client's name in marketing materials. Under Standard III(E):

## Options

- **A**: Client names can always be used if performance was positive
- **B**: Client permission should be obtained before using their name
- **C**: Only institutional client names require permission

## Answer

B

## Explanation

Standard III(E) protects client confidentiality, including their identity as clients. Using client names in marketing requires their permission, regardless of performance or client type.

**Common mistakes:**
- Option A ties permission to performance outcomes
- Option C creates an unjustified distinction between client types
