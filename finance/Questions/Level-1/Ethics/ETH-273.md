# Question

A client's investment objectives typically include:

## Options

- **A**: Only desired return
- **B**: Both return requirements and risk tolerance
- **C**: Only risk avoidance

## Answer

B

## Explanation

Standard III(C) requires understanding clients' complete investment profile, including return requirements and risk tolerance. Objectives must consider both dimensions, as they together determine appropriate investments.

**Common mistakes:**
- Option A ignores risk considerations
- Option C ignores return requirements
