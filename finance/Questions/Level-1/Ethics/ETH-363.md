# Question

A GIPS-compliant firm discovers it has incorrectly classified a portfolio in the wrong composite for several years. The firm should:

## Options

- **A**: Ignore the error if performance was similar between composites
- **B**: Correct the historical composite performance and disclose the correction
- **C**: Only correct going forward without adjusting history

## Answer

B

## Explanation

GIPS requires firms to correct material errors and disclose that corrections were made. Historical performance must be restated to accurately reflect what actually occurred in each composite.

**Common mistakes:**
- Option A ignores the need for accurate reporting
- Option C fails to correct historical misclassification
