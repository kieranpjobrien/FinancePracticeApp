# Question

Standard V(B) Communication requires disclosure of changes in:

## Options

- **A**: Only the final recommendation
- **B**: The investment process or methodology used
- **C**: Only negative developments

## Answer

B

## Explanation

Standard V(B) requires disclosure of changes to the investment decision-making process, not just final recommendations. Clients need to understand methodology changes that could affect future performance.

**Common mistakes:**
- Option A focuses only on outputs, ignoring process transparency
- Option C selectively discloses only unfavorable information
