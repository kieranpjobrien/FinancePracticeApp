# Question

When the Standards do not directly address a situation, members should:

## Options

- **A**: Assume the conduct is permitted if not explicitly prohibited
- **B**: Apply the spirit and intent of the Code and Standards using professional judgment
- **C**: Wait for explicit guidance before proceeding

## Answer

B

## Explanation

Ethical conduct requires applying principles beyond explicit rules. When facing situations not directly covered, members should consider what the Standards intend to achieve and act consistently with those underlying principles.

**Common mistakes:**
- Option A takes a legalistic approach ignoring ethical principles
- Option C is impractical for real-time decision-making
