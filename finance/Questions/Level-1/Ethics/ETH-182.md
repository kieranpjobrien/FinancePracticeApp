# Question

An analyst upgrades a stock from "hold" to "buy" but fails to notify clients who received the original "hold" recommendation. This:

## Options

- **A**: Is acceptable since clients should monitor reports independently
- **B**: May violate Standard V(B) if prior recipients should reasonably receive updates
- **C**: Is only problematic if the stock price changes significantly

## Answer

B

## Explanation

Standard V(B) requires appropriate communication of recommendation changes. Clients who received initial recommendations should be notified of changes so they can make informed decisions. The significance of price movement is not the determining factor.

**Common mistakes:**
- Option A ignores the member's duty to communicate material changes
- Option C applies an incorrect standard for when notification is required
