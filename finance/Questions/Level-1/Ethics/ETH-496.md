# Question

GIPS requires composite returns to be calculated using:

## Options

- **A**: Simple average of portfolio returns
- **B**: Asset-weighted average of portfolio returns
- **C**: Returns of the largest portfolio only

## Answer

B

## Explanation

GIPS requires asset-weighted composite returns. This approach ensures larger portfolios appropriately influence composite results, providing a more accurate representation of the firm's management of the strategy.

**Common mistakes:**
- Option A gives equal weight regardless of portfolio size
- Option C ignores all but one portfolio
