# Question

A GIPS-compliant presentation shows only gross-of-fees returns. This:

## Options

- **A**: Fully complies with GIPS requirements
- **B**: Must be supplemented with net-of-fees returns or sufficient fee disclosure
- **C**: Is prohibited under GIPS

## Answer

B

## Explanation

GIPS allows presentation of gross-of-fees returns but requires either net-of-fees returns or sufficient fee disclosure for prospects to calculate net returns. This enables meaningful comparison of actual investor outcomes.

**Common mistakes:**
- Option A ignores the net return or fee disclosure requirement
- Option C incorrectly prohibits gross-of-fees presentation
