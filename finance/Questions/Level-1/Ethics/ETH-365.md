# Question

The Code of Ethics focuses on:

## Options

- **A**: Detailed rules for every possible situation
- **B**: Broad principles that guide ethical conduct
- **C**: Only technical investment standards

## Answer

B

## Explanation

The Code of Ethics establishes broad ethical principles rather than specific rules for every situation. These principles guide members in navigating novel situations where specific rules may not exist.

**Common mistakes:**
- Option A mischaracterizes the Code as detailed rules (that's the Standards)
- Option C ignores the ethical rather than technical focus
