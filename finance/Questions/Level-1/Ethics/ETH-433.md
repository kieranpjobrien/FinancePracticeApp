# Question

When using quantitative models in analysis, Standard V(A) requires:

## Options

- **A**: Blind reliance on model outputs
- **B**: Understanding of model limitations and appropriate skepticism about outputs
- **C**: Using only models approved by CFA Institute

## Answer

B

## Explanation

Standard V(A) requires reasonable basis for recommendations. Members using quantitative models must understand their construction, assumptions, and limitations. Model outputs should be evaluated critically, not accepted blindly.

**Common mistakes:**
- Option A abdicates analytical responsibility
- Option C imposes a non-existent approval requirement
