# Question

Presenting only gross returns without disclosing fee information:

## Options

- **A**: Is always acceptable since gross returns are more accurate
- **B**: May mislead clients about actual performance they would receive
- **C**: Is required under GIPS

## Answer

B

## Explanation

Standard III(D) requires fair and complete performance presentation. Showing only gross returns without fee context may mislead clients about the returns they would actually experience, especially if fees are substantial.

**Common mistakes:**
- Option A ignores that net returns matter to clients
- Option C misrepresents GIPS requirements, which actually require both
