# Question

GIPS-compliant presentations typically show returns both gross and net of fees because:

## Options

- **A**: Gross returns allow comparison of investment skill while net returns show client experience
- **B**: Regulators require both presentations for compliance
- **C**: Gross returns are more accurate measures of true performance

## Answer

A

## Explanation

Gross returns (before fees) enable comparison of pure investment skill across managers with different fee structures. Net returns (after fees) show what clients actually experience. Presenting both gives a complete picture—gross for comparing investment ability, and net for understanding actual client outcomes.

**Common mistakes:**
- Option B misattributes the requirement to regulators rather than GIPS standards
- Option C ignores the value of net returns in showing client experience
