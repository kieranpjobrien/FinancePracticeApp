# Question

According to the Code of Ethics, members must maintain independence and objectivity in their:

## Options

- **A**: Personal investment decisions only
- **B**: Professional activities
- **C**: Interactions with regulators only

## Answer

B

## Explanation

The Code of Ethics requires members to use reasonable care and exercise independent professional judgment in all professional activities. This broad requirement applies to research, recommendations, investment actions, and other professional functions.

**Common mistakes:**
- Option A too narrowly limits the scope to personal investments
- Option C incorrectly restricts the requirement to only regulatory interactions
