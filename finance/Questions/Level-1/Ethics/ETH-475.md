# Question

Misrepresentation under Standard I(C) can occur through:

## Options

- **A**: Only active false statements
- **B**: Both false statements and material omissions
- **C**: Only written communications

## Answer

B

## Explanation

Standard I(C) covers misrepresentation through action (false statements) and omission (failing to disclose material information). Creating false impressions by withholding relevant facts violates the Standard.

**Common mistakes:**
- Option A ignores misrepresentation through omission
- Option C excludes oral and other communication forms
