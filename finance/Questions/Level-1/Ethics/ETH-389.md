# Question

GIPS requires firms to calculate portfolio returns using:

## Options

- **A**: Money-weighted returns to reflect actual investor experience
- **B**: Time-weighted returns to eliminate the impact of external cash flows
- **C**: Simple arithmetic returns for ease of comparison

## Answer

B

## Explanation

GIPS requires time-weighted returns because they eliminate the distorting effects of client-directed cash flows, allowing fair comparison of investment manager skill across different portfolios and firms.

**Common mistakes:**
- Option A reflects investor experience but not manager skill
- Option C oversimplifies the calculation requirement
