# Question

A code of ethics for a profession serves primarily to:

## Options

- **A**: Protect members of the profession from liability by establishing minimum standards of behavior
- **B**: Communicate the shared principles and expected behaviors that define the profession's commitment to ethical conduct
- **C**: Create barriers to entry that limit competition and increase compensation for existing members

## Answer

B

## Explanation

A code of ethics communicates the values, principles, and expected behaviors of a profession. It defines the profession's commitment to act ethically and serves as a public declaration of what clients, employers, and society can expect from members. The code helps build trust by establishing common standards that all members are expected to uphold.

**Common mistakes:**
- Option A focuses on liability protection, which is not the primary purpose; a code of ethics sets aspirational standards, not just minimum legal requirements
- Option C describes anticompetitive behavior that would actually undermine professional trust; codes of ethics serve the public interest, not member self-interest
