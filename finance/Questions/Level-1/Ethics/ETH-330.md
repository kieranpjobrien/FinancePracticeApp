# Question

A firm changes its investment process from fundamental to quantitative analysis. Under Standard V(B):

## Options

- **A**: No communication is required if returns remain positive
- **B**: Material changes to investment process should be disclosed to clients
- **C**: Only regulatory filings need updating

## Answer

B

## Explanation

Standard V(B) requires disclosure of changes to the investment process. Clients who selected the manager based on fundamental analysis should be informed of the shift to quantitative methods, regardless of performance.

**Common mistakes:**
- Option A ties disclosure to performance rather than material changes
- Option C ignores direct client communication needs
