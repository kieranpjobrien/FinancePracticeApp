# Question

"Best execution" under Standard III(A) requires portfolio managers to:

## Options

- **A**: Always use the broker with the lowest commission
- **B**: Seek the most favorable terms for client transactions considering all relevant factors
- **C**: Execute all trades through a single broker for consistency

## Answer

B

## Explanation

Best execution requires considering the overall value of the transaction, including price, speed, likelihood of execution, and settlement costs—not just commissions. The lowest-cost broker may not provide best execution if other factors are inferior.

**Common mistakes:**
- Option A focuses only on explicit costs, ignoring execution quality
- Option C ignores the need to evaluate execution quality across venues
