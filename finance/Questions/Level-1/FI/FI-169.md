# Question

A bond's holding period return includes:

## Options

- **A**: Coupon income only
- **B**: Coupon income plus price change over the holding period
- **C**: Price change only

## Answer

B

## Explanation

Holding period return = (End price - Begin price + Coupon) / Begin price. Both income return (coupons) and capital return (price change). For total return analysis, also consider reinvestment of coupons. Measures actual investor experience.

**Common mistakes:**
- Option A ignores capital gains/losses
- Option C ignores income component
