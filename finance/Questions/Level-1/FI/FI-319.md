# Question

A cross-default clause in a bond indenture:

## Options

- **A**: Limits the issuer's ability to issue additional debt
- **B**: Triggers default on the bond if the issuer defaults on any other debt obligation
- **C**: Requires the issuer to maintain certain financial ratios

## Answer

B

## Explanation

Cross-default: default on any obligation triggers default on this bond. Prevents issuer from selectively defaulting on some obligations while servicing others. Protects bondholders from subordination in distress. Accelerates all debt if any defaults.

**Common mistakes:**
- Option A—describes debt incurrence covenant
- Option C—describes financial maintenance covenant
