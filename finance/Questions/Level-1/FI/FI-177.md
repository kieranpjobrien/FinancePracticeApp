# Question

A credit default swap (CDS) provides the protection buyer with:

## Options

- **A**: Insurance against interest rate increases
- **B**: Compensation if a reference entity experiences a credit event
- **C**: Guaranteed returns regardless of credit conditions

## Answer

B

## Explanation

CDS: protection buyer pays periodic premium (spread) to seller. If credit event occurs (default, bankruptcy, restructuring), seller compensates buyer for loss. Credit events defined in contract. CDS spread reflects market's view of default risk.

**Common mistakes:**
- Option A describes interest rate derivatives
- Option C—CDS only protects against credit events, not guarantee
