# Question

The value of a convertible bond equals:

## Options

- **A**: The greater of straight bond value or conversion value
- **B**: Straight bond value plus conversion option value
- **C**: Stock price times conversion ratio

## Answer

B

## Explanation

Convertible value = straight bond value + conversion option value. Trades above both floor (straight bond) and parity (conversion value). When stock low: bond-like behavior. Stock high: equity-like. Option value highest when near conversion price.

**Common mistakes:**
- Option A describes minimum value (floor), not total value
- Option C describes conversion value only
