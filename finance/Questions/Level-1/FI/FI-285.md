# Question

In the Merton structural model of credit risk:

## Options

- **A**: Default occurs when asset value exceeds debt at maturity
- **B**: Equity is modeled as a call option on the firm's assets with strike equal to debt value
- **C**: Recovery rates are assumed to be zero

## Answer

B

## Explanation

Merton model: equity = call option on assets, strike = debt face value. If assets > debt at maturity, equity has value (in the money). If assets < debt, default (option expires worthless). Uses Black-Scholes framework. Asset volatility and leverage drive credit risk.

**Common mistakes:**
- Option A—default occurs when assets fall BELOW debt
- Option C—recovery can be positive if assets > 0
