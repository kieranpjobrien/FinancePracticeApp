# Question

Empirical duration is estimated by:

## Options

- **A**: Calculating the weighted average of cash flow timing
- **B**: Regressing historical bond price changes against yield changes
- **C**: Using the bond's modified duration formula

## Answer

B

## Explanation

Empirical duration: estimated from historical price/yield relationship via regression. Useful when analytical duration unreliable (complex structures, credit-sensitive bonds). May differ from analytical duration if spread changes affect price. Captures actual price behavior.

**Common mistakes:**
- Option A describes Macaulay duration calculation
- Option C describes analytical (formula-based) duration
