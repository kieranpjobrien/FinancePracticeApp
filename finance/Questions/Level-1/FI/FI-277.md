# Question

A bond has a 2% probability of default and an expected loss given default of 60%. The expected credit loss is:

## Options

- **A**: 0.8%
- **B**: 1.2%
- **C**: 2.0%

## Answer

B

## Explanation

Expected credit loss = Probability of default × Loss given default = 2% × 60% = 1.2%. This represents the expected percentage loss due to credit risk. Used in pricing credit spreads and provisions. Recovery rate = 1 - LGD = 40%.

**Common mistakes:**
- Option A—uses recovery rate instead of LGD
- Option C—ignores LGD, uses only PD
