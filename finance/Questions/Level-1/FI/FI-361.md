# Question

Tracking error in fixed-income portfolios measures:

## Options

- **A**: The absolute return of the portfolio
- **B**: The standard deviation of portfolio returns minus benchmark returns
- **C**: The portfolio's duration

## Answer

B

## Explanation

Tracking error = Std dev of (Portfolio return - Benchmark return). Measures active risk—how closely portfolio follows benchmark. Lower tracking error = more passive. Active managers accept higher tracking error seeking alpha. Risk budgeting allocates active risk across strategies.

**Common mistakes:**
- Option A—tracking error is relative, not absolute
- Option C—duration is sensitivity, not return deviation
