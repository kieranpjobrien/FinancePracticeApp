# Question

Effective duration is more appropriate than modified duration for bonds with embedded options because:

## Options

- **A**: It ignores cash flow changes
- **B**: It captures how cash flows change as rates change
- **C**: It assumes constant cash flows

## Answer

B

## Explanation

Effective duration: uses valuation model to capture how option exercise affects cash flows. Modified duration assumes fixed cash flows—wrong for callables (may be called), putables, MBS (prepayments). Effective duration calculated by shifting yield curve up/down.

**Common mistakes:**
- Option A—effective duration specifically accounts for changes
- Option C describes modified duration's limitation
