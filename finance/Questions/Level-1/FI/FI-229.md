# Question

An asset swap spread (ASW) measures:

## Options

- **A**: The spread over Treasuries
- **B**: The spread earned by converting a fixed-rate bond to floating rate via swap
- **C**: The option-adjusted spread

## Answer

B

## Explanation

ASW: combines bond with interest rate swap to create synthetic floater. ASW spread = floating rate received (LIBOR/SOFR + spread). Adjusts for premium/discount to par. Used for relative value—compare ASW across similar credits.

**Common mistakes:**
- Option A describes G-spread
- Option C describes OAS (different methodology)
