# Question

A portfolio has DV01 of $5,000. To hedge using a futures contract with DV01 of $100, approximately how many contracts are needed?

## Options

- **A**: 5 contracts
- **B**: 50 contracts
- **C**: 500 contracts

## Answer

B

## Explanation

Hedge ratio = Portfolio DV01 / Hedge instrument DV01 = $5,000 / $100 = 50 contracts. Short 50 contracts to offset long duration. If rates rise 1bp, portfolio loses $5,000, futures gain ~$5,000 (before basis risk).

**Common mistakes:**
- Option A divides wrong way
- Option C multiplies instead of divides
