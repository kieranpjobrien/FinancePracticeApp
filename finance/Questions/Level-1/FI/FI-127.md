# Question

Overcollateralization as credit enhancement means:

## Options

- **A**: The collateral value exceeds the securities issued
- **B**: The servicer guarantees payments
- **C**: Multiple asset types back the securities

## Answer

A

## Explanation

Overcollateralization: collateral pool value > securities issued. Excess provides cushion for losses. Example: $105M loans backing $100M securities. Internal enhancement—builds equity cushion. Often combined with subordination.

**Common mistakes:**
- Option B describes servicer advancing, different enhancement
- Option C describes diversification, not overcollateralization
