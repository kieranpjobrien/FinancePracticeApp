# Question

An investor buys a bond at 98, receives $5 in coupon payments, and sells at 101. The holding period return is approximately:

## Options

- **A**: 5.1%
- **B**: 7.1%
- **C**: 8.2%

## Answer

C

## Explanation

HPR = (Ending price - Beginning price + Income) / Beginning price = (101 - 98 + 5) / 98 = 8 / 98 = 8.16% ≈ 8.2%. Includes both capital gain (101 - 98 = 3) and income (5). Simple return measure over holding period.

**Common mistakes:**
- Option A—omits capital gain
- Option B—calculation error
