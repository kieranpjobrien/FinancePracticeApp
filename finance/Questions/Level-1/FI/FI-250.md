# Question

A bond quoted at "98.50" is trading at:

## Options

- **A**: $98.50 per $100 face value (discount)
- **B**: $98.50 per $1,000 face value
- **C**: A 98.50% premium over par

## Answer

A

## Explanation

Bond prices quoted as percentage of par. 98.50 = $98.50 per $100 face value = discount (below par). For $1,000 par bond, price is $985. Above 100 = premium. Price does not include accrued interest (clean price).

**Common mistakes:**
- Option B confuses quote convention
- Option C—98.50 is below 100, so discount not premium
