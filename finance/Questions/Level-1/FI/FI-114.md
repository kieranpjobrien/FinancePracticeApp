# Question

Spread duration measures a bond's sensitivity to:

## Options

- **A**: Changes in the risk-free rate
- **B**: Changes in the credit spread
- **C**: Changes in inflation expectations

## Answer

B

## Explanation

Spread duration: price sensitivity to changes in credit spread, holding benchmark rate constant. Important for corporate bonds where spread changes drive returns. A bond with spread duration of 5 loses ~5% if spread widens 100 bps.

**Common mistakes:**
- Option A describes interest rate duration
- Option C describes inflation sensitivity
