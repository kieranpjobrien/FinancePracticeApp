# Question

The 2-year spot rate is 4% and the 3-year spot rate is 5%. The 1-year forward rate starting in year 2 (f2,1) is closest to:

## Options

- **A**: 5.0%
- **B**: 6.0%
- **C**: 7.0%

## Answer

C

## Explanation

(1 + S₃)³ = (1 + S₂)² × (1 + f₂,₁). So (1.05)³ = (1.04)² × (1 + f₂,₁). 1.157625 = 1.0816 × (1 + f₂,₁). f₂,₁ = (1.157625/1.0816) - 1 = 1.0703 - 1 = 7.03% ≈ 7.0%. Forward rates extract implied future rates from spot curve.

**Common mistakes:**
- Option A—confuses forward with spot rate
- Option B—arithmetic error in calculation
