# Question

Scenario analysis for bond portfolios involves:

## Options

- **A**: Using only historical returns to predict future performance
- **B**: Projecting returns under different interest rate and spread environments
- **C**: Ignoring reinvestment income

## Answer

B

## Explanation

Scenario analysis: project portfolio value under different rate/spread scenarios. Test outcomes for rate rises, falls, curve shifts, spread changes. Stress testing. More comprehensive than single-point estimates. Used for risk management and strategy.

**Common mistakes:**
- Option A describes historical analysis only
- Option C—reinvestment is part of total return analysis
