# Question

The Z-spread is the constant spread added to:

## Options

- **A**: The Treasury yield to maturity
- **B**: Each point on the spot rate curve to price the bond correctly
- **C**: The coupon rate to calculate yield

## Answer

B

## Explanation

Z-spread (zero-volatility spread): constant spread added to each Treasury spot rate to make PV of cash flows equal market price. More accurate than nominal spread which uses single yield. Accounts for term structure shape.

**Common mistakes:**
- Option A describes nominal/G-spread using single yield
- Option C confuses spread with yield calculation
