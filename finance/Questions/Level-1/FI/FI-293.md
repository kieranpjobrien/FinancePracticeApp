# Question

When valuing a callable bond using a binomial interest rate tree:

## Options

- **A**: The call option is exercised at every node
- **B**: At each node, the bond value is the minimum of the call price and the calculated value
- **C**: Interest rate volatility has no impact on value

## Answer

B

## Explanation

Callable bond valuation: at each node, issuer calls if calculated value > call price. So bond value = min(call price, calculated PV). Higher volatility increases option value, reducing callable bond price. Backward induction through tree.

**Common mistakes:**
- Option A—call exercised only when advantageous to issuer
- Option C—volatility directly affects option value
