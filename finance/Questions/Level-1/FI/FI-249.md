# Question

Higher interest rate volatility will:

## Options

- **A**: Increase the value of callable bonds
- **B**: Increase the value of embedded options (calls and puts)
- **C**: Have no effect on bond values

## Answer

B

## Explanation

Higher volatility increases option value (more chance of favorable rate moves). Callable: call option worth more, so callable bond worth less to holder. Putable: put option worth more, so putable bond worth more to holder. Volatility key OAS input.

**Common mistakes:**
- Option A—higher volatility hurts callable bond holders
- Option C—volatility significantly impacts bonds with options
