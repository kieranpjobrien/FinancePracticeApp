# Question

An inverse floating-rate note has a coupon that:

## Options

- **A**: Moves in the same direction as the reference rate
- **B**: Moves in the opposite direction of the reference rate
- **C**: Remains fixed regardless of reference rate changes

## Answer

B

## Explanation

Inverse floater: coupon = fixed rate - reference rate. When rates rise, coupon falls; when rates fall, coupon rises. Higher duration than fixed-rate bond of same maturity. Often leveraged—small rate change causes large coupon change.

**Common mistakes:**
- Option A describes regular floater
- Option C describes fixed-rate bond
