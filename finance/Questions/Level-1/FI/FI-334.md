# Question

In reduced-form credit models, the hazard rate represents:

## Options

- **A**: The recovery rate in default
- **B**: The instantaneous probability of default given no prior default
- **C**: The bond's yield to maturity

## Answer

B

## Explanation

Hazard rate (λ): conditional probability of default in next instant, given survival to now. Reduced-form models treat default as random event with this intensity. Higher hazard rate = higher credit risk. Used to price credit derivatives and risky bonds.

**Common mistakes:**
- Option A—hazard rate is probability, not recovery
- Option C—hazard rate feeds into spread, not YTM directly
