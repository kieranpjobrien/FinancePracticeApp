# Question

An investor in the 35% tax bracket comparing a 4% municipal bond yield to a taxable bond should use a tax-equivalent yield of:

## Options

- **A**: 5.40%
- **B**: 6.15%
- **C**: 2.60%

## Answer

B

## Explanation

Tax-equivalent yield = Municipal yield / (1 - tax rate) = 4% / (1 - 0.35) = 4% / 0.65 = 6.15%. The muni's 4% tax-free equals 6.15% pre-tax. If taxable yields >6.15%, taxable bond better; if <6.15%, muni better.

**Common mistakes:**
- Option A multiplies by (1 + tax rate) incorrectly
- Option C multiplies yield by (1 - tax rate)
