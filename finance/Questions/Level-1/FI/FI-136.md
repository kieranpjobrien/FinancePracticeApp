# Question

Bootstrapping is a technique used to:

## Options

- **A**: Estimate credit spreads from market prices
- **B**: Derive spot rates from par bond prices
- **C**: Calculate duration from yield changes

## Answer

B

## Explanation

Bootstrapping: extracts spot rates from coupon bond prices sequentially. Start with shortest maturity, solve for spot rate, use that to solve for next maturity's spot rate. Builds zero curve from observed par curve.

**Common mistakes:**
- Option A describes spread analysis
- Option C describes duration estimation
