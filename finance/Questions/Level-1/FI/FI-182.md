# Question

A convertible bond with $1,000 par, conversion ratio of 25, and stock price of $35 has a conversion value of:

## Options

- **A**: $875
- **B**: $1,000
- **C**: $1,400

## Answer

A

## Explanation

Conversion value = conversion ratio × stock price = 25 × $35 = $875. This is the value if converted immediately. Conversion premium = (bond price - conversion value) / conversion value. Bond typically trades above conversion value due to option value.

**Common mistakes:**
- Option B uses par value
- Option C may multiply incorrectly
