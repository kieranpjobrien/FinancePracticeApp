# Question

Loss given default (LGD) represents:

## Options

- **A**: The probability that a bond will default
- **B**: The percentage of value lost if default occurs (1 minus recovery rate)
- **C**: The total face value of defaulted bonds

## Answer

B

## Explanation

LGD = 1 - Recovery rate. If recovery is 40%, LGD is 60%. Expected loss = Probability of default × LGD. Seniority affects recovery—senior secured typically recovers more than subordinated. LGD varies by industry, collateral, economic conditions.

**Common mistakes:**
- Option A describes probability of default (PD)
- Option C describes exposure at default
