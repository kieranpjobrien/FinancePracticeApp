# Question

A floating-rate note (FRN) has:

## Options

- **A**: Fixed coupon payments throughout its life
- **B**: Coupon payments that reset periodically based on a reference rate
- **C**: Coupons that increase by a fixed amount each period

## Answer

B

## Explanation

FRN coupon = reference rate + spread. Reference rate (SOFR, formerly LIBOR) resets periodically (quarterly, semiannually). Spread fixed at issuance reflects credit risk. Price stays near par because coupon adjusts to market rates.

**Common mistakes:**
- Option A describes fixed-rate bonds
- Option C describes step-up bonds
